繼續戳，這次戳出一個相當重要的修正：我上一輪說 1/30,1/42,1/70 小範圍沒立刻出現，這不是題目真的難在那裏，而是我當時搜索方式太笨。它們其實有一個極簡公式。

  

設 b 是 squarefree，素因子集合為 P(b)。若存在一個素數

  

R=\sum_{p\mid b}\frac bp,

  

且 R 不等於任何 p\mid b，那麼

  

\boxed{ \frac1b=\sum_{p\mid b}\frac1{pR} }

  

因為

  

\sum_{p\mid b}\frac1{pR} =\frac1R\sum_{p\mid b}\frac1p =\frac1R\cdot \frac1b\sum_{p\mid b}\frac bp =\frac1R\cdot\frac{R}{b} =\frac1b.

  

而每個分母 pR 都是兩個不同素數之積。這就是一個「星形構造」：中心是新素數 R，外圍是 b 的所有素因子。

  

例如：

  

b=30=2\cdot3\cdot5,\qquad R=15+10+6=31,

  

所以

  

\boxed{ \frac1{30} = \frac1{2\cdot31} +\frac1{3\cdot31} +\frac1{5\cdot31} }

  

即

  

\frac1{30}=\frac1{62}+\frac1{93}+\frac1{155}.

  

同理：

  

b=42=2\cdot3\cdot7,\qquad R=21+14+6=41,

  

故

  

\frac1{42} = \frac1{82}+\frac1{123}+\frac1{287}.

  

b=70=2\cdot5\cdot7,\qquad R=35+14+10=59,

  

故

  

\frac1{70} = \frac1{118}+\frac1{295}+\frac1{413}.

  

b=66=2\cdot3\cdot11,\qquad R=33+22+6=61,

  

故

  

\frac1{66} = \frac1{122}+\frac1{183}+\frac1{671}.

  

b=78=2\cdot3\cdot13,\qquad R=39+26+6=71,

  

故

  

\frac1{78} = \frac1{142}+\frac1{213}+\frac1{923}.

  

這一招很漂亮，因為它直接處理了我剛才以為是第一道關卡的三素因子分母。也就是說，至少很多

  

\frac1{pqr}

  

並不難。

  

真正的壞例子會從何處來？看

  

b=210=2\cdot3\cdot5\cdot7.

  

此時

  

R=105+70+42+30=247=13\cdot19,

  

不是素數。星形公式變成

  

\frac1{210} = \frac1{2\cdot247} +\frac1{3\cdot247} +\frac1{5\cdot247} +\frac1{7\cdot247},

  

但分母如 2\cdot247=2\cdot13\cdot19，有三個素因子，不合題目規定。這才是較真實的障礙。

  

所以問題可以重新理解為：

  

\boxed{ \text{當 } R=\sum_{p\mid b}\frac bp \text{ 不是素數時，如何把這個星形公式「拆細」成 }pq\text{ 分母？} }

  

這比原題清楚多了。

  

再看一層。如果 R 合數，例如 R=13\cdot19，我們需要把

  

\frac1{pR}

  

這種三素因子分母再拆成 pq-型單位分數。也就是說，原題會自然遞歸到

  

\frac1{p\cdot13\cdot19}

  

這類問題。

  

例如

  

\frac1{2\cdot13\cdot19}

  

可以再用星形公式，因為

  

2\cdot13\cdot19=494,

  

而

  

\frac{494}{2}+\frac{494}{13}+\frac{494}{19} = 247+38+26 = 311,

  

且 311 是素數。因此

  

\frac1{494} = \frac1{2\cdot311} +\frac1{13\cdot311} +\frac1{19\cdot311}.

  

也就是

  

\frac1{2\cdot13\cdot19} = \frac1{622} +\frac1{4043} +\frac1{5909}.

  

這一項拆成功了。

  

但對

  

\frac1{3\cdot13\cdot19}

  

星形數是

  

\frac{741}{3}+\frac{741}{13}+\frac{741}{19} = 247+57+39 = 343=7^3,

  

不是素數；甚至不是 squarefree。這就不再能直接用同一招。

  

所以 1/210 附近已經顯示出原題的核心困難：**有些局部項可以靠星形公式拆開，有些會撞上壞的合數。**

  

不過，這仍不是反例。因為我們沒有義務先寫成

  

\frac1{210} = \frac1{2\cdot247} +\frac1{3\cdot247} +\frac1{5\cdot247} +\frac1{7\cdot247}.

  

那只是其中一條路。或許有完全不同的邊集表示。

  

把這件事抽象一下，對 b=p_1\cdots p_m，若只允許分母形如「一個 p_i 乘一個外部素數 r」，那麼要找的是

  

\frac1b = \sum_{i,r}\frac{\varepsilon_{i,r}}{p_i r}, \qquad \varepsilon_{i,r}\in\{0,1\}.

  

兩邊乘以 b，得到

  

1= \sum_{i,r}\varepsilon_{i,r}\frac{b/p_i}{r}.

  

這就變成一個「帶係數的素數倒數子集和」問題。例如 b=210 時係數是

  

105,\ 70,\ 42,\ 30,

  

要找

  

1 = 105\sum_{r\in A_2}\frac1r +70\sum_{r\in A_3}\frac1r +42\sum_{r\in A_5}\frac1r +30\sum_{r\in A_7}\frac1r.

  

星形公式只是最特殊情形：

  

A_2=A_3=A_5=A_7=\{247\},

  

但 247 不是素數，所以失敗。更自由的構造則允許不同 A_i 用不同素數。這看起來比原問題更可攻，因為它把問題壓縮成了「素數倒數的線性組合」。

  

目前戳出的階段性結論如下：

  

第一，對很多 b，尤其三素因子小例子，原題有非常直接的星形構造。

  

第二，真正難點不是 b 有三個素因子，而是

  

R_b=\sum_{p\mid b}\frac bp

  

合成數時如何拆解。

  

第三，b=210 很可能是值得重點研究的第一個非平凡測試點，因為

  

R_{210}=247=13\cdot19

  

剛好讓星形公式失效。

  

第四，若能證明一個「拆分引理」：

  

對任意 squarefree B，\frac1B 可拆成 pq-型分母；

  

那麼一般 \frac ab 還要再處理「倍數 a」與「分母互異」問題，但主幹已經會清楚很多。

  

我現在更覺得原題有希望為真。它不像亂撞，而是有明顯的構造骨架：

  

\frac1b = \sum_{p\mid b}\frac1{pR_b}

  

是一個「理想公式」；問題只是在 R_b 不是素數時，把其中的非法分母再合法化。這種情形很像許多埃及分數問題：先有一個不合規但結構正確的表示，再用替換恆等式逐步消除壞分母。

  

如果繼續攻，我會把目標收窄為：

  

\boxed{ \text{先專門解 } \frac1{210}. }

  

不是因為它本身重要，而是因為它是星形公式第一次明顯失效的自然案例。若 1/210 能找到漂亮表示，觀察它的圖形結構，可能會看見真正的一般方法。
