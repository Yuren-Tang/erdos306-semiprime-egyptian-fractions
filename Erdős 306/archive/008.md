好，推進到可以明確寫成「若證此引理，原題即解」的程度。這一步的價值在於：我們不再只是說「也許可用隨機圖」，而是把需要證的 Fourier 估計寫出來。

原問題可歸約為：

[  
\forall b\ \text{squarefree},\quad \frac1b  
=\sum_i\frac1{p_iq_i}  
]

且可避開任意有限禁用素數集合。若此成立，則 (a/b) 只需取 (a) 份互不相交的 (1/b) 表示即可。

---

## 1. 有限圖映射

取一批輔助素數

[  
C={c_1,\dots,c_N},  
]

暫時只考慮在這些輔助素數之間選邊。對一張簡單圖 (G)，定義

# [  
\Phi(G)_j

\sum_{i\sim j}\frac1{c_i}\pmod {c_j}.  
]

也就是：

[  
\Phi:{0,1}^{\binom N2}\longrightarrow  
\prod_{j=1}^N\mathbb F_{c_j}.  
]

若這個映射能達到任意指定殘差向量

[  
(\beta_1,\dots,\beta_N),  
]

那麼我們就可以把一批尚未平衡的端口一次性封口。

所以核心問題變成：

[  
\boxed{  
\Phi\text{ 是否滿射，或至少是否能命中我們需要的殘差？}  
}  
]

這是正面戰場。

---

## 2. Fourier 展開

令

[  
H=\prod_{j=1}^N\mathbb F_{c_j}.  
]

隨機選一張圖 (G)，每條邊獨立以概率 (1/2) 選取。這給出 (H) 上的一個分佈 (\mu)。

若 (\mu) 接近均勻分佈，則 (\Phi) 幾乎必然滿射。更精確地，對任意目標 (y\in H)，

# [  
\mathbb P(\Phi(G)=y)

\frac1{|H|}  
\sum_{\lambda\in H}  
\widehat\mu(\lambda)\chi_\lambda(-y).  
]

其中 (\lambda=(\lambda_1,\dots,\lambda_N))，(\lambda_j\in\mathbb F_{c_j})，而

# [  
\chi_\lambda(x)

\exp\left(2\pi i\sum_j\frac{\lambda_jx_j}{c_j}\right).  
]

主項是 (\lambda=0)，等於 (1/|H|)。只要能證明

[  
\sum_{\lambda\ne0}|\widehat\mu(\lambda)|<1,  
]

就有

[  
\mathbb P(\Phi(G)=y)>0  
]

對所有 (y) 成立。因此 (\Phi) 滿射。

---

## 3. 每條邊給出的 Fourier 因子

一條邊 ({i,j}) 若被選，會對第 (i) 個坐標貢獻

[  
\frac1{c_j}\pmod {c_i},  
]

對第 (j) 個坐標貢獻

[  
\frac1{c_i}\pmod {c_j}.  
]

所以對 character (\lambda)，這條邊的相位是

# [  
\theta_{ij}(\lambda)

\frac{\lambda_i c_j^{-1}}{c_i}  
+  
\frac{\lambda_j c_i^{-1}}{c_j}  
\pmod 1.  
]

因為邊獨立，

# [  
\widehat\mu(\lambda)

\prod_{1\le i<j\le N}  
\frac{1+e^{2\pi i\theta_{ij}(\lambda)}}2.  
]

取絕對值：

# [  
|\widehat\mu(\lambda)|

\prod_{i<j}  
\left|\cos \pi\theta_{ij}(\lambda)\right|.  
]

利用估計

[  
|\cos \pi x|\le \exp(-2|x|^2),  
]

其中 (|x|) 是到最近整數的距離，得到

[  
|\widehat\mu(\lambda)|  
\le  
\exp\left(  
-2\sum_{i<j}|\theta_{ij}(\lambda)|^2  
\right).  
]

所以我們真正需要的是一個**相位分散估計**：

[  
\boxed{  
\sum_{i<j}|\theta_{ij}(\lambda)|^2  
\text{ 對所有非零 }\lambda\text{ 足夠大。}  
}  
]

## 4. 關鍵引理：相位分散

現在可以提出一個清楚的技術引理。

**相位分散引理。**  
給定任意有限禁用素數集合 (F)，存在任意大的素數集合

[  
C={c_1,\dots,c_N},\qquad C\cap F=\varnothing,  
]

使得對所有非零

[  
\lambda=(\lambda_1,\dots,\lambda_N)\in \prod_j\mathbb F_{c_j},  
]

都有足夠強的估計：

[  
\sum_{i<j}  
\left|  
\frac{\lambda_i c_j^{-1}}{c_i}  
+  
\frac{\lambda_j c_i^{-1}}{c_j}  
\right|^2  
\gg  
|\operatorname{supp}\lambda|\cdot N  
]

至少在平均意義上足以推出

[  
\sum_{\lambda\ne0}|\widehat\mu(\lambda)|<1.  
]

若這個引理成立，則 (\Phi) 滿射。

這就是目前最核心的技術命題。

---

## 5. 為什麼這個估計看起來合理？

若 (\lambda) 只在一個坐標 (i) 上非零，則

# [  
\theta_{ij}(\lambda)

\frac{\lambda_i c_j^{-1}}{c_i}  
]

對所有 (j\ne i)。如果 (c_j^{-1}\pmod {c_i}) 在 (\mathbb F_{c_i}^\times) 中分佈得足夠散，那麼這些相位不會都接近整數。於是

[  
\sum_{j\ne i}|\theta_{ij}|^2  
\asymp N.  
]

若 (\lambda) 有 (t) 個非零坐標，則至少從支撐集與外部的邊可得到約

[  
t(N-t)  
]

個分散貢獻。當 (t\le N/2)，這很好。

當 (t) 很大時，主要依靠支撐集內部的邊。這裏可能有相位抵消：

[  
\frac{\lambda_i c_j^{-1}}{c_i}  
+  
\frac{\lambda_j c_i^{-1}}{c_j}  
\approx 0.  
]

但要讓大量邊同時抵消，應該極不可能。若能證明「不存在大規模近似抵消」，Fourier 估計就成立。

所以技術核心是：

[  
\boxed{  
\text{素數倒數矩陣 }(c_i^{-1}\bmod c_j)\text{ 沒有大規模低秩或近零相位結構。}  
}  
]

這很像隨機矩陣分散性問題，只是矩陣來自素數互逆模。

## 6. 若 Fourier 滿射成立，如何推出原題？

現在把鏈條補完整。

給定 squarefree (b)，令

[  
B={p:p\mid b}.  
]

我們要造圖 (E)，使

[  
\sum_{{u,v}\in E}\frac1{uv}=\frac1b.  
]

對每個邊界點 (p\in B)，需要的局部模值是

[  
c_p=\frac1{b/p}\pmod p.  
]

先選一批輔助素數 (C)，避開 (B) 及所有已用過的素數。通過 CRT 與 Dirichlet 定理，可以讓若干邊界—輔助邊先在 (B) 上產生指定殘差；這一步會在輔助頂點 (C) 上留下某些殘差 (\beta_j)。

然後用 Fourier 滿射結果，在 (C) 內部選邊，使

[  
\Phi(G)_j=-\beta_j  
]

對每個 (c_j\in C) 成立。於是所有輔助頂點平衡，邊界頂點有指定值。

令

[  
L=b\prod_{c\in C}c.  
]

局部模條件對 (L) 的每個素因子都成立，所以

[  
\sum_{{u,v}\in E}\frac{L}{uv}  
\equiv  
\frac Lb  
\pmod L.  
]

若所有輔助素數取足夠大，則可保證

[  
0<  
\sum_{{u,v}\in E}\frac1{uv}<1.  
]

因此

[  
0<  
\sum_{{u,v}\in E}\frac{L}{uv}<L.  
]

同時

[  
0<\frac Lb<L.  
]

兩個介於 (0) 與 (L) 的整數模 (L) 同餘，所以相等：

# [  
\sum_{{u,v}\in E}\frac{L}{uv}

\frac Lb.  
]

除以 (L)，得

[  
\sum_{{u,v}\in E}\frac1{uv}=\frac1b.  
]

於是 (1/b) 表示完成。

若要表示 (a/b)，反覆構造 (a) 次，每次把已用素數加入禁用集合，即可得到互不重複的分母。

所以我們有：

[  
\boxed{  
\text{相位分散引理}  
\Rightarrow  
\text{Fourier 滿射}  
\Rightarrow  
\text{封口引理}  
\Rightarrow  
\frac1b\text{ 表示}  
\Rightarrow  
\frac ab\text{ 表示}.  
}  
]

## 7. 現在真正卡在哪裏？

卡在相位分散引理。

也就是要證明：可以選一批輔助素數 (c_1,\dots,c_N)，使得對任意非零 (\lambda)，大量邊 ((i,j)) 的相位

[  
\frac{\lambda_i c_j^{-1}}{c_i}  
+  
\frac{\lambda_j c_i^{-1}}{c_j}  
]

不會同時接近整數。

這不是顯然的，但它比原題清楚得多。它把一個埃及分數問題變成了「素數互逆矩陣的 Fourier 分散性」問題。

這裏有兩種可能。

第一種，這個相位分散引理是真的，而且可由素數在算術級數中的分佈、CRT、隨機選素數、加性組合工具證出。若如此，原 Erdős 問題很可能能被這條路打開。

第二種，這個引理太強，存在某些非零 (\lambda) 造成大規模相位接近零。那麼單純隨機圖滿射法失敗，但失敗本身也會揭示一種隱藏代數結構，可能反過來指向原題中的真正障礙。

## 8. 這一步的實質進展

我們現在不是在猜「應該很多自由度所以可行」，而是得到了一個具體、可攻、可反駁的核心命題：

[  
\boxed{  
\forall \lambda\ne0,\quad  
\sum_{i<j}  
\left|  
\frac{\lambda_i c_j^{-1}}{c_i}  
+  
\frac{\lambda_j c_i^{-1}}{c_j}  
\right|^2  
\text{ 足夠大。}  
}  
]

這就是下一個真正要打的靶子。

若我是要繼續研究，我會把下一步定為：

先不碰所有 (\lambda)，而證三個層級：

1. (\lambda) 只有一個非零坐標；
    
2. (\lambda) 有小支撐；
    
3. (\lambda) 有大支撐。
    

第一、第二層看起來很可能可用素數剩餘類分佈證明；第三層最難，需要排除大規模成對抵消。

至此，原題已被推到一個相當正面的研究前沿：  
**證明素數互逆矩陣對所有非零 character 有 Fourier 分散性。**
