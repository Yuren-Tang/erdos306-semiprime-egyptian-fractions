好。現在正面研究「二部封口引理」。這一步我先不急於證它，而是檢查它是否本身過強；結果是：**原始二部封口引理確實過強，至少不能無條件成立**。但它失敗的方式很有啟發，會逼出更正確的終端問題。

---

我們有一層舊素數

[  
Y={y_1,\dots,y_m},  
]

每個 (y\in Y) 有一個待補殘差

[  
\gamma_y\in\mathbb F_y.  
]

想選一層新素數 (Z)，只用跨邊 (Y-Z)，使：

對舊頂點 (y)：

[  
\sum_{z:,yz\in E}\frac1z\equiv \gamma_y\pmod y;  
]

對新頂點 (z)：

[  
\sum_{y:,yz\in E}\frac1y\equiv0\pmod z.  
]

第二條是「新頂點立即平衡」，所以整個封口一次完成。

現在看一個新頂點 (z)。它連到某個子集

[  
I\subseteq Y.  
]

平衡條件是

[  
\sum_{y\in I}\frac1y\equiv0\pmod z.  
]

令

[  
Y_I=\prod_{y\in I}y.  
]

則

# [  
\sum_{y\in I}\frac1y

\frac{N_I}{Y_I},  
\qquad  
N_I=\sum_{y\in I}\frac{Y_I}{y}.  
]

因為 (z) 不等於任何 (y\in Y)，所以 (z\nmid Y_I)。因此

[  
\sum_{y\in I}\frac1y\equiv0\pmod z  
]

等價於

[  
z\mid N_I.  
]

所以每個可用的新頂點 (z) 不是自由選的；它必須是某個數

[  
N_I=\sum_{y\in I}\frac{Y_I}{y}  
]

的素因子。

這是非常強的限制。

例如若 (|I|=1)，則

[  
N_I=1.  
]

沒有任何新素數 (z) 可用。所以一個新頂點不能只連到一個舊頂點。

若 (|Y|=1)，二部封口完全不可能。  
若 (|Y|=2)，唯一非平凡子集是 (I={y_1,y_2})，此時

[  
N_I=y_1+y_2.  
]

所以可用的新素數只能是 (y_1+y_2) 的素因子；數量極少。顯然不可能補任意殘差。

因此：

[  
\boxed{  
\text{原始二部封口引理作為「任意 }Y,\gamma\text{」命題是假的或至少過強。}  
}  
]

這不是壞消息。它告訴我們：**終端封口不能只靠一個任意的新層 (Z)；末層 (Y) 必須先被製造成一個很「富」的集合。**

---

## 正確問題：富末層封口

對固定 (Y)，所有可用的平衡列由以下資料給出：

選一個非空子集 (I\subseteq Y)，計算

[  
N_I=\sum_{y\in I}\frac{Y_I}{y}.  
]

若 (z) 是 (N_I) 的素因子，則可以加入一個新頂點 (z)，連到 (I)，且 (z) 自身平衡。

它對舊層 (Y) 的貢獻向量是

[  
v(I,z)_y=  
\begin{cases}  
z^{-1}\pmod y,& y\in I,\  
0,& y\notin I.  
\end{cases}  
]

所以二部封口問題等價於：

[  
\boxed{  
\text{這些向量 }v(I,z)\text{ 的子集和，是否生成整個 }  
\prod_{y\in Y}\mathbb F_y？  
}  
]

也就是：

[  
\forall \gamma=(\gamma_y)_{y\in Y},  
\quad  
\gamma=\sum v(I,z)  
]

是否可解。

這個表述很乾淨。現在真正的問題不是「任意 (Y) 可封口」，而是：

[  
\boxed{  
\text{能否選擇一個足夠富的素數層 }Y，  
\text{使上述平衡列向量生成全部殘差空間？}  
}  
]

這比原引理更合理。

---

## 為什麼「富末層」有希望？

因為子集 (I\subseteq Y) 有

[  
2^{|Y|}  
]

個。當 (|Y|) 大時，數量非常多。每個 (N_I) 又可能有若干素因子。於是可用列向量的總數可能極大。

目標空間大小是

[  
\prod_{y\in Y} y.  
]

若 (Y) 中素數大小控制在大約 (e^{\eta |Y|}) 以下，則目標空間的對數大小約為

[  
\sum_{y\in Y}\log y  
\asymp \eta |Y|^2.  
]

而可用子集 (I) 的數量是

[  
2^{|Y|},  
]

其對數是線性的，但可用列數本身是指數級。這在計數上完全可能有足夠多的自由度。

當然，計數不是證明。真正要知道的是：這些 (N_I) 的素因子是否足夠多、模各個 (y) 的倒數是否足夠分散。這是新核心。

---

## 新核心命題：富層封口命題

我們可以把終端問題重寫成：

**富層封口命題。**  
存在任意大的有限素數集合 (Y)，避開任何預先指定有限集合，滿足：由所有 pairs

[  
(I,z),\qquad I\subseteq Y,\ |I|\ge2,\quad z\mid N_I  
]

產生的向量

[  
v(I,z)\in\prod_{y\in Y}\mathbb F_y  
]

的子集和覆蓋整個殘差空間

[  
\prod_{y\in Y}\mathbb F_y.  
]

若這個命題成立，終端封口解決；再配合前面的多層二部傳遞，原 Erdős 問題就很接近了。

---

## 但這個命題仍然很硬

因為它要求知道許多整數

[  
N_I=\sum_{y\in I}\frac{Y_I}{y}  
]

的素因子分佈。這不是普通素數在算術級數中的分佈，而是大量「子集倒數和分子」的素因子問題。這可能相當深。

所以我會再作一個策略性選擇：**不要先證富層封口命題的最強形式，而是尋找較小的可控封口列。**

---

## 更可控的封口列：兩點列與三點列

先看 (|I|=2)。若

[  
I={u,v},  
]

則

[  
N_I=u+v.  
]

任何素因子

[  
z\mid u+v  
]

都給出一個平衡列，連到 (u,v)。

它的向量是

# [  
v_{{u,v},z}

z^{-1}e_u+z^{-1}e_v.  
]

這非常簡單。它相當於在 (Y) 上的一條邊 ({u,v})，但權重由 (z\mid u+v) 決定。

若我們能選 (Y)，使很多 pair-sums

[  
u+v  
]

有豐富素因子，那麼只用兩點列也許就能封口大量殘差。

但這有一個限制：每個列同時改變兩個坐標。這像是在 (Y) 上做加權邊流。可能存在總和類的障礙，但因為各坐標模數不同，未必有簡單守恆律。

再看 (|I|=3)。若

[  
I={u,v,w},  
]

則

[  
N_I=uv+uw+vw.  
]

任何素因子

[  
z\mid uv+uw+vw  
]

給出一個連到三個舊頂點的平衡列。這比兩點列自由得多。

所以一個更現實的封口命題是：

[  
\boxed{  
\text{只用 }|I|=2,3\text{ 的列，是否已能生成全部殘差？}  
}  
]

如果能，封口會大幅簡化。

---

## 這裏有一個非常重要的修正

我們不必要求「任意末層 (Y)」可封口。  
在分層傳遞過程中，末層 (Y) 是我們自己選的。更妙的是，上一層傳遞到 (Y) 的殘差 (\gamma_y) 也不是完全外來；它取決於我們在上一層選邊的方式。

也就是說，終端任務不一定是：

[  
\text{給定任意 }\gamma,\text{ 封掉它。}  
]

更弱、更合理的是：

[  
\boxed{  
\text{選擇上一層傳遞方式，使末層殘差落入可封口子空間。}  
}  
]

這非常重要。前面的二部滿射每一步我們其實有很多選擇。平衡上一層的方案不唯一；不同方案會在下一層造成不同殘差。只要能選一個方案，使最後殘差可被富層列封口，就夠了。

因此整體策略應該是：

1. 用二部子集和把原始殘差推到末層；
    
2. 但不是任意推，而是推到末層可封口殘差集合；
    
3. 用平衡列封口。
    

這比「二部封口要覆蓋所有殘差」弱得多，也合理得多。

---

## 新的整體框架

現在我們可以把完整路線修正為：

### 第一段：殘差傳遞

用若干二部層

[  
A_0\to A_1\to\cdots\to A_r=Y  
]

把原始邊界殘差從 (A_0) 傳到末層 (Y)。

每一步只要求上一層被清零；下一層殘差可變。

### 第二段：終端吸收

末層 (Y) 有一個「可封口殘差集合」

[  
H_Y\subseteq \prod_{y\in Y}\mathbb F_y,  
]

由所有平衡列向量 (v(I,z)) 的子集和生成。

只需讓傳遞後的末層殘差落在 (H_Y)。

若傳遞映射的可能殘差集合與 (H_Y) 相交非空，即可完成。

這看起來像 additive combinatorics：

[  
\boxed{  
\text{傳遞可達集 }R_Y  
\quad\text{與}\quad  
\text{吸收集 }H_Y  
\quad\text{相交。}  
}  
]

若兩者都很大，交集應該非空。

這比要求 (H_Y) 等於全空間弱很多。

---

## 真正應攻的下一步

現在我認為最應證的是一個「吸收集很大」的命題，而不是「吸收集等於全空間」。

形式如下：

令

[  
G_Y=\prod_{y\in Y}\mathbb F_y.  
]

由所有平衡列向量生成的子集和集合記作

[  
H_Y\subseteq G_Y.  
]

若能證明

[  
|H_Y|\ge |G_Y|^{\alpha}  
]

某個固定 (\alpha>0)，甚至更強地

[  
|H_Y|>|G_Y|/2,  
]

那麼終端吸收就非常有希望。

尤其若上一層傳遞可達集 (R_Y) 也有大小

[  
|R_Y|>|G_Y|/2,  
]

則必有

[  
R_Y\cap(-H_Y)\ne\varnothing.  
]

這是很標準的加法組合思路。

所以終端問題不必立刻證滿射；只要證兩個大集合相交。

---

## 小結：這一步真正推進了什麼？

我們發現：

[  
\boxed{  
\text{原始二部封口引理過強。}  
}  
]

原因是新頂點 (z) 若要立即平衡，必須滿足

[  
z\mid N_I=\sum_{y\in I}\frac{Y_I}{y},  
]

所以可用 (z) 只能來自大量子集倒數和分子的素因子。

但這沒有殺死路線，而是把終端問題精確化為：

[  
\boxed{  
\text{末層 }Y\text{ 的「平衡列吸收集」 }H_Y\text{ 有多大？}  
}  
]

下一步最應攻的是：

[  
\boxed{  
\text{證明對適當選取的富末層 }Y,\quad H_Y\text{ 很大。}  
}  
]

我認為這比「封口滿射」更正確，也比回到小例子更前進。它把終端困難壓成一個明確的有限加法組合問題：

[  
\text{由 }z\mid N_I\text{ 產生的向量 }v(I,z)  
\text{ 的子集和集是否大？}  
]

若這個吸收集大，前面的二部傳遞就有機會把殘差送進去；原問題也就有了真正的全局路線。
