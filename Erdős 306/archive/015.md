繼續。這一步可以真正證出一個工具，而不是只提出框架。核心是：

[  
K_z=\left{x\in{0,1}^Y:\sum_{y\in Y}x_y y^{-1}\equiv0\pmod z\right}  
]

這個「列核」在許多坐標上的投影近似均勻。這是終端封口的第一塊硬磚。

---

## 一、有限域子集和混合引理

令 (z) 為素數，令 (U) 是一組不被 (z) 整除的整數。考慮隨機子集和

[  
S_U=\sum_{u\in U}\varepsilon_u u^{-1}\pmod z,  
]

其中 (\varepsilon_u\in{0,1}) 獨立且各以概率 (1/2) 取 (1)。

假設存在 (\kappa>0)，使對所有非零 (\mu\in\mathbb F_z)，

[  
\sum_{u\in U}  
\left|  
\frac{\mu u^{-1}}{z}  
\right|^2  
\ge  
\kappa |U|.  
]

那麼 (S_U) 在 (\mathbb F_z) 上接近均勻：

[  
\left|  
\mathbb P(S_U=r)-\frac1z  
\right|  
\le  
e^{-2\kappa |U|}  
]

對所有 (r\in\mathbb F_z) 成立。

證明很短。Fourier 展開給出

# [  
\mathbb P(S_U=r)-\frac1z

\frac1z  
\sum_{\mu\ne0}  
e\left(-\frac{\mu r}{z}\right)  
\prod_{u\in U}  
\frac{1+e(\mu u^{-1}/z)}2.  
]

而

# [  
\left|  
\frac{1+e(t)}2  
\right|

|\cos \pi t|  
\le  
e^{-2|t|^2}.  
]

所以每個非零 Fourier 係數至多

[  
\exp\left(  
-2\sum_{u\in U}  
\left|  
\frac{\mu u^{-1}}z  
\right|^2  
\right)  
\le  
e^{-2\kappa |U|}.  
]

求和後得到上述估計。

這個引理很重要：它把「倒數分散」直接轉化成「子集和均勻」。

## 二、列核 (K_z) 的投影近似均勻

現在取末層

[  
Y={y_1,\dots,y_m}.  
]

固定一個終端素數 (z)。列核是

[  
K_z=  
\left{  
x\in{0,1}^Y:  
\sum_{y\in Y}x_y y^{-1}\equiv0\pmod z  
\right}.  
]

取一個坐標子集 (T\subseteq Y)，想看 (K_z) 在 (T) 上的投影是否近似均勻。

給定 (T) 上的一個取值 (x_T)，設

[  
s_T=\sum_{y\in T}x_y y^{-1}\pmod z.  
]

要把它補成 (K_z) 中的一列，就要在補集

[  
U=Y\setminus T  
]

上選 (x_U)，使

[  
\sum_{u\in U}x_u u^{-1}\equiv -s_T\pmod z.  
]

若 (U) 滿足剛才的子集和混合條件，則可補方式數為

[  
2^{|U|}  
\left(  
\frac1z+O(e^{-2\kappa |U|})  
\right).  
]

因此只要

[  
e^{-2\kappa |U|}\ll \frac1z,  
]

也就是

[  
|U|\gg \log z,  
]

那麼所有 (x_T) 的補成數幾乎相同。於是：

[  
\boxed{  
K_z\text{ 在 }T\text{ 上的投影近似均勻，只要 }Y\setminus T\text{ 足夠大且倒數分散。}  
}  
]

這是真工具。它說：列約束雖然存在，但只要還留下一大批自由坐標，這個約束幾乎不影響已觀察坐標的分佈。

## 三、用它控制終端吸收的稀疏 character

終端吸收中，對每個 (z\in Z)，我們從 (K_z) 中選一列 (x^{(z)})。它對 (Y) 上殘差的貢獻是

[  
R_y=\sum_{z\in Z}x^{(z)}_y z^{-1}\pmod y.  
]

我們希望 (R=(R_y)_{y\in Y}) 在

[  
G_Y=\prod_{y\in Y}\mathbb F_y  
]

中近似均勻。

取一個 character

[  
\lambda=(\lambda_y)_{y\in Y}\in G_Y.  
]

其支撐是

[  
T={y:\lambda_y\ne0}.  
]

單列 Fourier 因子為

# [  
F_z(\lambda)

\mathbb E_{x\in K_z}  
e\left(  
\sum_{y\in T}  
x_y\frac{\lambda_y z^{-1}}y  
\right).  
]

若 (|T|\le m/2)，則 (Y\setminus T) 很大。由投影近似均勻，(x_T) 幾乎像獨立均勻的 (0/1) 變量。所以

[  
F_z(\lambda)  
\approx  
\prod_{y\in T}  
\frac{1+e(\lambda_y z^{-1}/y)}2.  
]

因此

[  
|F_z(\lambda)|  
\lesssim  
\exp\left(  
-2\sum_{y\in T}  
\left|  
\frac{\lambda_y z^{-1}}y  
\right|^2  
\right)  
+\text{小誤差}.  
]

再對所有 (z\in Z) 相乘，得到

# [  
|\widehat\nu(\lambda)|

\prod_{z\in Z}|F_z(\lambda)|  
\lesssim  
\exp\left(  
-2\sum_{z\in Z}\sum_{y\in T}  
\left|  
\frac{\lambda_y z^{-1}}y  
\right|^2  
\right).  
]

所以若 (Z) 對每個 (y\in Y) 的倒數模 (y) 分散，即

[  
\sum_{z\in Z}  
\left|  
\frac{\lambda_y z^{-1}}y  
\right|^2  
\ge  
\rho |Z|  
]

對所有 (y) 和所有 (\lambda_y\ne0) 成立，則

[  
|\widehat\nu(\lambda)|  
\lesssim  
e^{-2\rho |Z||T|}.  
]

而支撐大小為 (t=|T|) 的 character 數大約是

[  
\binom mt \prod_{y\in T}y.  
]

若 (y\le e^{\eta |Z|}) 且 (\eta) 足夠小，則

[  
\binom mt \prod_{y\in T}y  
\le  
\exp(O(t\log m)+\eta t|Z|),  
]

被

[  
e^{-2\rho t|Z|}  
]

壓住。

所以我們得到：

[  
\boxed{  
\text{終端吸收中的稀疏 character 可以被消掉。}  
}  
]

這一步與前面「全圖 Fourier」的稀疏層相似，但更乾淨，因為列獨立，且列約束可以用投影均勻引理處理。

## 四、真正剩下的是稠密 character

若

[  
|T|>m/2,  
]

則上面的投影論證不夠。因為 (Y\setminus T) 太小，沒有足夠自由坐標去把列約束洗平。這是目前真正的終端難點。

對稠密 (\lambda)，需要直接估計

# [  
F_z(\lambda)

\mathbb E_{x\in K_z}  
e\left(  
\sum_{y\in Y}  
x_y\frac{\lambda_y z^{-1}}y  
\right).  
]

用列約束的 Fourier 展開：

# [  
1_{K_z}(x)

\frac1z\sum_{\mu\in\mathbb F_z}  
e\left(  
\frac{\mu}{z}  
\sum_{y\in Y}x_y y^{-1}  
\right).  
]

分子變成

[  
\frac1z  
\sum_{\mu\in\mathbb F_z}  
\prod_{y\in Y}  
\left(  
1+  
e\left(  
\frac{\lambda_y z^{-1}}y  
+  
\frac{\mu y^{-1}}z  
\right)  
\right).  
]

分母是

[  
|K_z|\approx \frac{2^m}{z}.  
]

因此若對每個 (\mu\in\mathbb F_z) 都有

[  
\sum_{y\in Y}  
\left|  
\frac{\lambda_y z^{-1}}y  
+  
\frac{\mu y^{-1}}z  
\right|^2  
\ge  
\kappa m,  
]

那麼

[  
|F_z(\lambda)|  
\le  
z,e^{-2\kappa m}  
]

至常數誤差。

若再有

[  
\log z\le \kappa m,  
]

則 (F_z(\lambda)) 會很小。這就是稠密 character 的正確技術條件。

所以稠密層的核心不是一般「倒數分散」，而是更強的反聚集：

[  
\boxed{  
\text{對稠密 }\lambda,\text{ 不存在 }\mu  
\text{ 使大量相位 }  
\frac{\lambda_y z^{-1}}y+\frac{\mu y^{-1}}z  
\text{ 同時接近整數。}  
}  
]

這是終端吸收真正的硬核。

## 五、現在整個問題被壓縮成兩個一維分散命題

終端吸收目前需要兩類輸入：

### 1. 列核混合

對每個 (z\in Z)，集合

[  
{y^{-1}\pmod z:y\in Y}  
]

的 (0/1) 子集和在 (\mathbb F_z) 中混合。這由

[  
\sum_{y\in U}  
\left|  
\frac{\mu y^{-1}}z  
\right|^2  
\gg |U|  
]

推出。

### 2. 稠密反聚集

對稠密 (\lambda)，要求

[  
\sum_{y\in Y}  
\left|  
\frac{\lambda_y z^{-1}}y  
+  
\frac{\mu y^{-1}}z  
\right|^2  
\gg m  
]

對所有 (\mu) 成立，至少對足夠多的 (z\in Z) 成立。

第一個像普通子集和混合，應該相對可攻。  
第二個是目前真正深處。

## 六、這一步的實質進展

我們現在已經不是泛泛地說「用隨機矩陣封口」。我們證出了：

[  
\boxed{  
\text{若 }Y\setminus T\text{ 的倒數子集和混合，則 }K_z\text{ 在 }T\text{ 上近似均勻。}  
}  
]

由此推出：

[  
\boxed{  
\text{終端吸收的稀疏 Fourier 項可控。}  
}  
]

剩餘困難非常明確：

[  
\boxed{  
\text{終端吸收的稠密 character 反聚集。}  
}  
]

我認為下一步應該直接攻這個稠密反聚集。這是目前最集中的核心；若它能證成，終端封口大體可完成，整條「分層傳遞 + 終端吸收」路線就真正有望推回原 Erdős 問題。
