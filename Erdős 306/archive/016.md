繼續。這一步很關鍵，因爲我們會發現上一輪的「稠密反聚集」其實**不可能照原樣成立**。但這不是死路，而是暴露出一個真正的結構性障礙：**對角 character**。

先回到終端封口。末層 (Y)，終端素數 (z)。列核

[  
K_z=\left{x\in{0,1}^Y:\sum_{y\in Y}x_y y^{-1}\equiv0\pmod z\right}.  
]

對 character (\lambda=(\lambda_y)_{y\in Y})，單列 Fourier 因子是

[  
F_z(\lambda)=  
\mathbb E_{x\in K_z}  
e\left(  
\sum_{y\in Y}x_y\frac{\lambda_y z^{-1}}y  
\right).  
]

我們本來希望：只要 (\lambda\ne0)，許多 (z) 上都有 (|F_z(\lambda)|<1-\epsilon)。這對稀疏 (\lambda) 可能成立；但對某些稠密 (\lambda) 明顯失敗。

看最危險的例子：

[  
\lambda_y=h\quad\text{對所有 }y\in Y,  
]

其中 (h) 是一個小整數，且 (h<\min Y)。這是「對角 character」。

對每個 (y,z)，有一個簡單 CRT 恆等式：

[  
\frac{z^{-1}\bmod y}{y}  
+  
\frac{y^{-1}\bmod z}{z}  
\equiv  
\frac1{yz}  
\pmod 1.  
]

乘以 (h)，得

## [  
\frac{h(z^{-1}\bmod y)}y  
\equiv  
\frac{h}{yz}

\frac{h(y^{-1}\bmod z)}z  
\pmod1.  
]

因此對 (x\in K_z)，

## [  
\sum_{y\in Y}x_y\frac{h(z^{-1}\bmod y)}y  
\equiv  
\frac hz\sum_{y\in Y}\frac{x_y}{y}

\frac hz\sum_{y\in Y}x_y(y^{-1}\bmod z)  
\pmod1.  
]

而 (x\in K_z) 正表示

[  
\sum_{y\in Y}x_y(y^{-1}\bmod z)\equiv0\pmod z.  
]

所以第二項是整數。於是

[  
\boxed{  
\sum_{y\in Y}x_y\frac{h(z^{-1}\bmod y)}y  
\equiv  
\frac hz\sum_{y\in Y}\frac{x_y}{y}  
\pmod1.  
}  
]

這就糟了。因爲右邊是一個很小的實數，只要

[  
\frac{h}{z}\sum_{y\in Y}\frac1y  
\ll1,  
]

那麼對所有 (x\in K_z)，相位都靠近 (0)，所以

[  
F_z(\lambda)\approx1.  
]

也就是說，對角 character 根本不會被終端列隨機性消掉。

更明確地，有估計：

[  
|F_z(\lambda)-1|  
\le  
2\pi\frac{|h|}{z}\sum_{y\in Y}\frac1y.  
]

所以若

[  
z\gg |h|\sum_{y\in Y}\frac1y,  
]

則 (|F_z(\lambda)|) 極接近 (1)。

這證明了：

[  
\boxed{  
\text{原先的稠密反聚集命題是假的。}  
}  
]

不是技術不夠，而是有真障礙。

---

這個障礙的意思很深。終端列條件

[  
\sum_{y\in Y}x_y/y\equiv0\pmod z  
]

在模 (z) 意義下平衡了，但在真實大小上，它仍然是一個很小的正數：

[  
\sum_{y\in Y}\frac{x_y}{y}.  
]

對角 character 正是在探測這個「真實大小」。因此，只靠模平衡，不能自動產生完全均勻的殘差分佈。

所以我們必須修正路線。不是放棄 Fourier，而是把稠密 character 分成兩種：

[  
\boxed{  
\text{非結構 character}  
}  
]

與

[  
\boxed{  
\text{對角／近對角 character}.  
}  
]

非結構 character 仍可望用反聚集殺掉；對角 character 必須用另一種方法處理。

---

新的核心觀察是：

若 (\lambda_y=h) 對大多數 (y) 成立，則它不會被模隨機性消掉。要讓它衰減，必須讓

[  
\frac{h}{z}\sum_{y\in Y}\frac{x_y}{y}  
]

本身有足夠變化。也就是要求

[  
\sum_{y\in Y}\frac1y  
]

相對於終端素數 (z) 夠大。

換句話說，末層 (Y) 不能只是「模上富」，還要「倒數質量大」：

[  
\boxed{  
H(Y):=\sum_{y\in Y}\frac1y  
\quad\text{必須足夠大。}  
}  
]

若所有終端素數 (z) 都滿足

[  
z\ll H(Y),  
]

那麼對角 character 的相位範圍就不再很小，有機會產生衰減。

這是一個很有埃及分數味道的條件：素數倒數和雖然發散得極慢，但畢竟發散，所以可以靠選取非常多的末層素數讓

[  
H(Y)  
]

任意大。

於是終端吸收應該改成：

1. (Y) 在模 (z) 下倒數分散，保證列核 (K_z) 混合；
    
2. (Y) 的倒數質量 (H(Y)) 足夠大，專門殺對角 character；
    
3. 對非對角稠密 character，用反聚集處理。
    

這比上一輪更準確。

---

現在可以提出新的分層：

### 第一層：稀疏 character

已經基本可控。若支撐小，列核投影近似均勻，然後用 (Z) 在模 (y) 下的倒數分散殺掉。

### 第二層：非結構稠密 character

若 (\lambda_y) 沒有在大多數 (y) 上接近同一個小整數 (h)，則相位

[  
\frac{\lambda_y z^{-1}}y+\frac{\mu y^{-1}}z  
]

不可能對許多 (y) 同時接近整數。這類仍可望用反聚集殺掉。

### 第三層：對角或近對角 character

若

[  
\lambda_y\approx h  
]

在很多 (y) 上成立，則模反聚集無效。這時要利用真實倒數和

[  
\sum_{y\in Y}\frac{x_y}{y}  
]

的變動，要求 (H(Y)) 大，並選 (z) 不太大。

所以真正的新終端策略是：

[  
\boxed{  
\text{末層 }Y\text{ 要同時有兩種富性：模富性 + 倒數質量富性。}  
}  
]

這是實質推進。它告訴我們，前面那種把所有素數都取到巨大指數級的方案是錯的；那會讓

[  
z\gg H(Y)  
]

從而完全殺不掉對角 character。

---

因此參數也要改。

不能取

[  
z\le e^{\alpha m}  
]

而讓 (Y) 是一批很大的素數。因爲即使 (|Y|=m)，若每個 (y) 都巨大，

[  
H(Y)=\sum_{y\in Y}\frac1y  
]

可能非常小。

更合理的是：末層 (Y) 應該由很多相對小的素數組成，至少要讓

[  
H(Y)\gg \max Z.  
]

這聽起來很誇張，因爲 (\sum 1/p) 增長很慢；但存在性上沒有障礙。只要取足夠多素數，素數倒數和可任意大。

這意味著整個構造可能需要一個巨大末層。這不美，但原題不要求有效界。

---

所以目前修正後的正面戰略是：

[  
\boxed{  
\begin{aligned}  
&\text{前面各層：用二部子集和傳遞殘差；}\  
&\text{末層 }Y：選成一個倒數質量巨大的素數集合；\  
&\text{終端層 }Z：選相對小於 }H(Y)\text{ 的素數；}\  
&\text{終端吸收：}\  
&\quad\text{稀疏 character 用投影混合殺；}\  
&\quad\text{非結構稠密 character 用反聚集殺；}\  
&\quad\text{對角 character 用 }H(Y)\text{ 的真實大小波動殺。}  
\end{aligned}  
}  
]

這看起來比前一版複雜，但更真實。它把一個隱藏障礙挖了出來，而不是把錯誤的稠密反聚集硬推下去。

---

下一步最該做的是專攻對角 character，因爲它是剛剛發現的真障礙。具體要證一個命題：

令

[  
\lambda_y=h  
]

對所有 (y\in Y)。從

[  
K_z=\left{x:\sum x_y y^{-1}\equiv0\pmod z\right}  
]

中隨機取 (x)。要估計

[  
\mathbb E_{x\in K_z}  
e\left(  
\frac hz\sum_{y\in Y}\frac{x_y}{y}  
\right).  
]

若 (K_z) 對真實變量

[  
\sum x_y/y  
]

沒有太強偏置，則這應接近無條件模型：

[  
\prod_{y\in Y}\frac{1+e(h/(zy))}{2}.  
]

其模大約是

[  
\exp\left(  
-2\sum_{y\in Y}  
\left|  
\frac{h}{zy}  
\right|^2  
\right).  
]

但這裏又有一個問題：

[  
\sum_{y\in Y}\frac{1}{z^2y^2}  
]

通常很小，即使 (\sum 1/y) 很大，(\sum1/y^2) 仍然收斂。這表示只靠小相位二次衰減也不夠。

真正要讓對角 character 衰減，需要相位不只是小幅波動；要有

[  
\frac h z \sum_{y\in Y}\frac{x_y}{y}  
]

在模 (1) 上跨過相當長的範圍。也就是需要

[  
\frac h z H(Y)\gg1.  
]

因此對 (h=1)，至少要

[  
H(Y)\gg z.  
]

這再次確認：倒數質量必須大到超過終端素數。

這是一個非常強、但清楚的要求。

---

目前推進後的結論是：

[  
\boxed{  
\text{終端吸收的主要敵人是對角 character；要殺它，必須讓末層素數倒數和 }H(Y)\text{ 大於終端模數。}  
}  
]

這是我們第一次看到問題中「(\sum 1/p) 發散」真正不可替代地進入證明。前面只用模分佈，現在發現不夠；還必須用素數倒數和的實數發散。

所以接下來的正確方向是：

[  
\boxed{  
\text{建立一個「大倒數質量末層」的終端吸收模型。}  
}  
]

這條路沒有之前那麼漂亮，但更貼近原題本體，也更可能避開錯誤的滿射幻想。
