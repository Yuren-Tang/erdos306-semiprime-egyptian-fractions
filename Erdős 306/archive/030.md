可以，原文已經能讀；你附的是 Irving 的 `.tex` 原稿，不只是 bibliography。這裏有一個重要更正：**我剛才用 2024 文章轉述的 Theorem 2 不是最適合我們的；你這份 Irving 原文裏的 Theorem 1 才正中要害，因爲它已經帶有 (\max_{(a,q)=1})。**

Irving 在引言中定義

[  
S_q(a;x)=\sum_{\substack{p\sim x\(p,q)=1}} e!\left(\frac{a\overline p}{q}\right),  
]

其中 (p) 跑過素數，(p\sim x) 表示 (x\le p<2x)。這正是我們要的 dyadic prime interval，不必再由 (p\le X) 差分處理。

他的 Theorem 1 是：

[  
\sum_{q\sim Q}\max_{(a,q)=1}|S_q(a;x)|  
\ll_\varepsilon  
\left(  
Q^{5/4}x^{5/8}  
+  
Qx^{9/10}  
+  
Q^{7/6}x^{13/18}  
\right)Q^\varepsilon,  
]

條件是

[  
Q^{3/2}\ge x\ge Q^{2/3}.  
]

這已經解決了我們最擔心的「是否 uniform in (a)」問題，因爲左邊直接有

[  
\max_{(a,q)=1}.  
]

取同尺度

[  
x\asymp Q.  
]

三項分別成爲：

[  
Q^{5/4}Q^{5/8}=Q^{15/8},  
]

[  
Q\cdot Q^{9/10}=Q^{19/10},  
]

[  
Q^{7/6}Q^{13/18}=Q^{17/9}.  
]

最大的是

[  
Q^{19/10}=Q^{1.9}.  
]

而平凡平均界是約

[  
Q\cdot \frac{x}{\log x}\asymp \frac{Q^2}{\log Q}.  
]

所以 Irving Theorem 1 給出 power saving。平均每個 (q) 的最大 Kloosterman 和大約是

[  
Q^{9/10+\varepsilon},  
]

遠小於

[  
Q/\log Q.  
]

因此由 Markov 不等式，除去 (o(Q/\log Q)) 個素數模數 (q\sim Q) 之外，應有

[  
\max_{(a,q)=1}|S_q(a;x)|  
\le \delta\frac{x}{\log x}  
]

對任意固定小 (\delta>0) 成立。這正是我們需要的 Lemma K 類型，而且是**所有 (a)** 同時成立。

這樣，前面最大的疑點消失了。

## 下一步應當做什麼

接下來最應該檢查的是：Irving 的平均是對所有整數 (q\sim Q)，而我們終端層需要 (q) 本身是素數。這不是大問題，因爲所有整數模數中的好 (q) 很多，未必自動保證素數好 (q) 很多；如果平均界是對所有 (q)，壞整數少，不一定排除「素數全壞」的極端可能。

因此下一個真正要核的是：

[  
\boxed{  
\text{Irving Theorem 1 的平均是否可限制到素數 }q\sim Q?  
}  
]

如果原證明對 (q) 沒有使用合數特有平均，而是對 dyadic (q) 逐項估計再求和，那麼限制到素數 (q) 應該立即成立，因爲素數 (q) 是所有 (q) 的子集：

[  
\sum_{\substack{q\sim Q\q\text{ prime}}}\max_{(a,q)=1}|S_q(a;x)|  
\le  
\sum_{q\sim Q}\max_{(a,q)=1}|S_q(a;x)|.  
]

但 Markov 後得到的「好素數 (q) 很多」還要比較基數。壞素數數量至多

# [  
\ll \frac{Q^{19/10+\varepsilon}}{\delta Q/\log Q}

\delta^{-1}Q^{9/10+\varepsilon}\log Q.  
]

而素數 (q\sim Q) 的總數是

[  
\asymp \frac Q{\log Q}.  
]

只要 (\varepsilon<1/10)，壞素數數量是

[  
o(Q/\log Q).  
]

所以確實有正比例、甚至幾乎所有素數 (q\sim Q) 是好的。

這也閉合了。

所以現在我們可以更強地說：

[  
\boxed{  
\text{Irving Theorem 1 足以推出我們需要的 prime reciprocal short-arc avoidance。}  
}  
]

下一步若繼續，就是把這個推出過程寫成一個乾淨的引理，然後回接到互逆分散塊。
