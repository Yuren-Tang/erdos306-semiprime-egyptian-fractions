很好，現在可以把 Irving 定理真正接到我們的「互逆分散塊」上。這一步要小心一個細節：**不能簡單說「好模數很多，所以從好模數裏抽 (Y,Z) 就行」**，因爲短弧避免是相對於「全部素數 (p\sim x)」而言；若再限制 (p) 也必須來自好模數集合，理論上可能破壞分佈。正確做法是：先從全部素數中隨機抽 (Y,Z)，再證明它們以高概率都落在好模數集合，並且互逆分散成立。

下面把這一步寫清。

---

## 1. Irving Theorem 1 給出好模數

Irving 定義

[  
S_q(a;x)=\sum_{\substack{p\sim x\(p,q)=1}}  
e\left(\frac{a\overline p}{q}\right),  
]

其中 (p) 跑過素數，且 (p\sim x) 表示 (x\le p<2x)。你上傳的原文第 21–23 行正是這個定義。

Theorem 1 給出：

[  
\sum_{q\sim Q}\max_{(a,q)=1}|S_q(a;x)|  
\ll_\varepsilon  
\left(  
Q^{5/4}x^{5/8}  
+  
Qx^{9/10}  
+  
Q^{7/6}x^{13/18}  
\right)Q^\varepsilon  
]

在

[  
Q^{3/2}\ge x\ge Q^{2/3}  
]

內成立；這見原文第 52–56 行。

取

[  
x=Q.  
]

右邊最大項是

[  
Qx^{9/10}=Q^{19/10}.  
]

所以

[  
\sum_{q\sim Q}\max_{(a,q)=1}|S_q(a;Q)|  
\ll_\varepsilon Q^{19/10+\varepsilon}.  
]

令

[  
\pi_Q=#{p\text{ prime}:Q\le p<2Q}\asymp \frac Q{\log Q}.  
]

若稱 (q) 為「好」當且僅當

[  
\max_{(a,q)=1}|S_q(a;Q)|  
\le \delta,\pi_Q  
]

其中 (\delta>0) 是固定小常數，則壞 (q) 的數量至多

# [  
\ll_\varepsilon  
\frac{Q^{19/10+\varepsilon}}{\delta Q/\log Q}

O_\delta(Q^{9/10+\varepsilon}\log Q).  
]

限制到素數 (q\sim Q) 仍然成立。因此壞素數的比例至多

[  
O_\delta(Q^{-1/10+\varepsilon}(\log Q)^2)=o(1)  
]

只要 (\varepsilon<1/10)。

所以：

[  
\boxed{  
\text{幾乎所有素數 }q\sim Q\text{ 都是好模數。}  
}  
]

這一步非常關鍵，且 Theorem 1 的 (\max_{(a,q)=1}) 正好解決了「所有乘子 (a)」問題。

## 2. 好模數推出短弧避免

對好模數 (q)，我們要證：

[  
\left{  
p\sim Q:  
\left|\frac{a\bar p}{q}\right|<\eta  
\right}  
]

不能佔幾乎全部素數。

用一個固定階數 (H) 的三角多項式 (T) majorize 短弧指示函數：

[  
1_{|t|<\eta}  
\le  
2\eta+c_H+\sum_{1\le |h|\le H} c_h e(ht),  
]

其中 (c_H\to0) 當 (H\to\infty)，且 (H) 是固定常數。

代入

[  
t=\frac{a\bar p}{q}.  
]

則

# [  
\sum_{p\sim Q}e\left(\frac{ha\bar p}{q}\right)

S_q(ha;Q).  
]

因爲 (q) 是素數且 (a\ne0)，只要 (1\le |h|\le H<q)，便有 (ha\not\equiv0\pmod q)。好模數性質給出

[  
|S_q(ha;Q)|\le \delta\pi_Q.  
]

因此

[  
#\left{  
p\sim Q:  
\left|\frac{a\bar p}{q}\right|<\eta  
\right}  
\le  
(2\eta+c_H+O_H(\delta))\pi_Q.  
]

先取 (\eta) 小，再取 (H) 大，最後取 (\delta) 小，可使右邊 (\le (1-\beta)\pi_Q) 某個固定 (\beta>0)。

所以對每個好素數 (q\sim Q)、每個 (a\ne0)，

[  
\boxed{  
#\left{  
p\sim Q:  
\left|\frac{a\bar p}{q}\right|<\eta  
\right}  
\le(1-\beta)\pi_Q.  
}  
]

這就是我們需要的 prime reciprocal short-arc avoidance。

## 3. 隨機抽 (Y,Z)，得到雙向互逆分散

現在從全部素數區間

[  
\mathcal P_Q={p\text{ prime}:Q\le p<2Q}  
]

中獨立隨機抽取兩組

[  
Y,Z,\qquad |Y|=|Z|=N,  
]

其中取

[  
N=Q^\alpha  
]

而

[  
0<\alpha<\frac{1}{10}  
]

足夠小。

先看好模數問題。壞素數比例是

[  
o(Q^{-0+?})  
]

更精確地是

[  
O(Q^{-1/10+\varepsilon}(\log Q)^2).  
]

故隨機抽取 (N=Q^\alpha) 個素數時，抽到壞素數的期望為

[  
O(Q^{\alpha-1/10+\varepsilon}(\log Q)^2).  
]

若

[  
\alpha<1/10-\varepsilon,  
]

這趨於 (0)。因此以高概率，(Y\cup Z) 中所有素數都是好模數。

接着固定一個好 (z\in Z) 和一個非零 (\mu\in\mathbb F_z)。由短弧避免，全部素數 (p\sim Q) 中至少 (\beta\pi_Q) 個滿足

[  
\left|\frac{\mu\bar p}{z}\right|\ge\eta.  
]

隨機抽取 (Y) 時，Chernoff 型估計給出：以概率 (1-\exp(-cN))，至少 (\beta N/2) 個 (y\in Y) 滿足

[  
\left|\frac{\mu\bar y}{z}\right|\ge\eta.  
]

所以

[  
\sum_{y\in Y}  
\left|  
\frac{\mu\bar y}{z}  
\right|^2  
\ge  
\frac{\beta}{2}\eta^2 N.  
]

需要同時對所有

[  
z\in Z,\qquad \mu\in\mathbb F_z^\times  
]

成立。可能的對象數量至多

[  
|Z|\cdot Q=Q^{1+\alpha}.  
]

而單個失敗概率是

[  
\exp(-cQ^\alpha),  
]

遠小於任何 (Q^{-A})。故 union bound 可行。

同理，交換 (Y,Z)，可得對所有 (y\in Y)、非零 (r\in\mathbb F_y)，

[  
\sum_{z\in Z}  
\left|  
\frac{r\bar z}{y}  
\right|^2  
\ge  
\frac{\beta}{2}\eta^2 N.  
]

因此存在兩組素數 (Y,Z) 滿足雙向互逆分散：

[  
\boxed{  
\begin{aligned}  
&\forall z\in Z,\ \forall \mu\ne0:\quad  
\sum_{y\in Y}  
\left|  
\frac{\mu y^{-1}}z  
\right|^2  
\gg N,\[1mm]  
&\forall y\in Y,\ \forall r\ne0:\quad  
\sum_{z\in Z}  
\left|  
\frac{r z^{-1}}y  
\right|^2  
\gg N.  
\end{aligned}  
}  
]

這一段現在基本無條件，因為 Irving Theorem 1 已經提供所需好模數。

## 4. 現在剩下的是組合—Fourier 部分

我們已經把最可疑的解析數論輸入接上了。現在路線變成：

[  
\boxed{  
\text{Irving Theorem 1}  
\Rightarrow  
\text{雙向互逆分散素數塊 }Y,Z.  
}  
]

下一步要做的是把「雙向互逆分散」正式推出「終端吸收」。

這部分我們已經有思路，但還沒有完全寫成嚴格引理。其核心是：

對每個 (z\in Z)，列核

[  
K_z=  
\left{  
x\in{0,1}^Y:  
\sum_{y\in Y}x_y y^{-1}\equiv0\pmod z  
\right}.  
]

由 (Y^{-1}\bmod z) 的分散性，(K_z) 的投影混合。

由 (Z^{-1}\bmod y) 的分散性，非對角 Fourier character 衰減。

對角 character 對應全局 CRT 相容條件，不是障礙。

所以若把這部分嚴格化，便可得到：

[  
\boxed{  
\text{雙向互逆分散素數塊}  
\Rightarrow  
\text{終端吸收}  
}  
]

然後再接：

[  
\text{終端吸收}  
\Rightarrow  
\text{局部模平衡圖}  
\Rightarrow  
\frac1b=\sum\frac1{pq}  
\Rightarrow  
\frac ab=\sum\frac1{pq}.  
]
