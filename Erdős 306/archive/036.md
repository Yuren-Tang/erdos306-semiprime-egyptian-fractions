好。現在設計「精修引理」。我們先不要試圖一次證原題，而是處理如下局部問題：

給定一個很小的目標

[  
r>0,  
]

以及一組有限模條件，能否在一個新素數塊中選若干 (pq)-邊，使

[  
\sum \frac1{pq}=r  
]

並同時滿足指定的 CRT 殘差？

這是粗調之後需要的精修步驟。

---

## 1. 精修塊的基本尺度

選一個大參數 (Q)，取兩組素數

[  
Y,Z\subset [Q,2Q],  
]

候選邊為完整二部圖

[  
Y\times Z.  
]

每條邊權約為

[  
\frac1{yz}\asymp \frac1{Q^2}.  
]

若

[  
|Y|=|Z|=N,  
]

則總可用質量約為

[  
W\asymp \frac{N^2}{Q^2}.  
]

要讓它匹配小 remainder (r)，可令

[  
\frac{N^2}{Q^2}\asymp r.  
]

也就是

[  
N\asymp Q\sqrt r.  
]

同時 Irving 給我們的隨機好塊可以允許 (N=Q^\alpha) 這種規模。因此精修塊自然適合處理

[  
r\asymp Q^{2\alpha-2}.  
]

換言之，若 remainder 很小，可以取足夠大的 (Q) 讓一個 dyadic 二部塊的總質量與它同尺度。

---

## 2. 整數化

令

[  
L=L_0\cdot \prod_{y\in Y}y\prod_{z\in Z}z,  
]

其中 (L_0) 是舊部分已用素數的乘積。

對邊 (yz)，整數權重是

[  
w_{yz}=\frac{L}{yz}.  
]

要命中目標 remainder (r)，等價於

[  
\sum_{(y,z)\in E}w_{yz}=rL.  
]

這要求 (rL) 是整數；在實際構造中，粗調階段要保證 remainder 的分母只含已用與待用素數，或更簡單地，把整個 (L) 先定好，再在模 (L) 中工作。

精修引理可先表述為有限群版本：

[  
G=\mathbb Z/L\mathbb Z.  
]

給定目標

[  
t\in G  
]

且其實數代表位於精修塊總質量範圍內，證明存在 (E\subseteq Y\times Z)，使

[  
\sum_{(y,z)\in E}w_{yz}=t.  
]

這就是二部 (pq)-權重子集和。

---

## 3. 隨機模型

令每條邊獨立選取：

[  
X_{yz}\in{0,1},\qquad \mathbb P(X_{yz}=1)=\theta.  
]

令

[  
S=\sum_{y,z}X_{yz}w_{yz}.  
]

選 (\theta) 使

[  
\mathbb E(S/L)=r.  
]

即

[  
\theta W=r.  
]

如果

[  
W\asymp r,  
]

則 (\theta) 是固定常數，例如 (1/2)。這是理想狀態：不太稀疏，Fourier 衰減也好。

---

## 4. Fourier 分析

CRT 分解後，一個 character 由以下坐標組成：

- 舊模數 (u\mid L_0) 上的 (\lambda_u)；
    
- (Y) 上的 (\lambda_y\in\mathbb F_y)；
    
- (Z) 上的 (\lambda_z\in\mathbb F_z)。
    

邊 (yz) 的相位是

# [  
\phi_{yz}(\lambda)

\sum_{u\mid L_0}\frac{\lambda_u(yz)^{-1}}u  
+  
\frac{\lambda_y z^{-1}}y  
+  
\frac{\lambda_z y^{-1}}z.  
]

第一項是舊模數殘差影響，對 (y,z) 也有倒數型依賴。這有點更複雜，但若舊模數集合固定有限，則可由普通 Dirichlet/CRT 分散輕鬆控制，或先把精修塊選在對舊模數 residue 足夠均勻的素數區間中。

主困難仍在 (Y,Z) 互相的倒數項：

[  
\frac{\lambda_y z^{-1}}y+\frac{\lambda_z y^{-1}}z.  
]

這正是 Irving 可處理的非對角部分。

---

## 5. 主弧分類

在二部塊 (Y\times Z) 中，對角 character 是：

存在整數 (h)，使

[  
\lambda_y\equiv h\pmod y,\qquad \lambda_z\equiv h\pmod z  
]

並且舊模數坐標也符合

[  
\lambda_u\equiv h\pmod u.  
]

此時每條邊相位為

[  
\phi_{yz}(\lambda)\equiv \frac{h}{yz}\pmod1.  
]

因此主弧就是實數大小方向，和以前一致。

非對角 character 由 Irving 互逆分散衰減。

所以精修塊的 Fourier 結構是乾淨的。

---

## 6. 精修引理的候選形式

現在可以提出：

**精修引理（候選版）。**  
給定有限 squarefree 整數 (L_0)、一個小正有理數 (r)，以及一個目標 residue (t\in\mathbb Z/L\mathbb Z) 與實數目標相容。可選足夠大的 (Q) 和兩組素數

[  
Y,Z\subset[Q,2Q],  
]

使：

1. (Y,Z) 對所有相關模數，包括彼此與 (L_0)，具有互逆分散；
    
2. 完整二部圖總質量
    
    [  
    W=\sum_{y\in Y,z\in Z}\frac1{yz}  
    ]
    
    滿足 (W\asymp r);
    
3. 隨機 Bernoulli 子集和在目標 (t) 上有正概率。
    

則存在邊集 (E\subseteq Y\times Z)，使

[  
\sum_{(y,z)\in E}\frac1{yz}=r  
]

並滿足所需模條件。

---

## 7. 這個引理的主弧部分如何證？

對角主弧對應

[  
X=\sum_{y,z}\frac{X_{yz}}{yz}.  
]

因為

[  
1/(yz)\asymp Q^{-2}  
]

幾乎同尺度，而且共有 (N^2) 項，這比全局多尺度模型簡單得多。

期望為 (r)，方差約為

[  
\sigma^2  
\asymp  
\theta(1-\theta)N^2 Q^{-4}.  
]

若

[  
N^2/Q^2\asymp r,  
]

則

[  
\sigma^2\asymp r/Q^2,  
\qquad  
\sigma\asymp \frac{\sqrt r}{Q}.  
]

單個權重大小是 (Q^{-2})，遠小於 (\sigma) 若

[  
Q^{-2}\ll \frac{\sqrt r}{Q}  
\quad\Longleftrightarrow\quad  
\frac1Q\ll \sqrt r.  
]

而由 (N\asymp Q\sqrt r)，這等價於

[  
N\gg1.  
]

所以只要精修塊有很多素數，局部 CLT 是合理的。

格點間距是

[  
1/L,  
]

遠小於單個權重。主弧應給出目標點正概率，量級約為

[  
\frac{1}{\sigma L}.  
]

這非常小，但只要非主弧誤差更小即可。

---

## 8. 非主弧是否能壓過主項？

這是技術核心。主項約

[  
\frac{1}{\sigma L}.  
]

而 Fourier 反演有因子 (1/L)。所以需要非零 character 總和相對於主弧小，即

[  
\sum_{\lambda\notin\text{主弧}}|\widehat\mu(\lambda)|  
\ll \frac1\sigma  
]

大致如此。

非主弧衰減來自 (N^2) 條邊。若離對角距離為 (d)，衰減可望是

[  
e^{-cN d}  
]

或更強。character 數中每個異常坐標帶來約 (Q) 種選擇，因此數量約

[  
e^{d\log Q}.  
]

所以只要

[  
N\gg \log Q,  
]

就能求和。這與前面條件一致。取

[  
N=Q^\alpha  
]

完全足夠。

因此非主弧看起來能壓住。

---

## 9. 目前精修引理的主要難點

真正需要嚴格化的有三點：

第一，建立二部塊的局部 CLT，處理權重

[  
1/(yz)  
]

的細微不等同。

第二，處理主弧範圍，不只對角整數 (h) 很小，還要控制中等 (h) 的 characteristic function。這可用標準局部 CLT 技術。

第三，舊模數 (L_0) 的 characters 要納入非主弧分類。因 (L_0) 固定或遠小於 (Q)，這應可通過選 (Y,Z) 在模 (L_0) 下均勻處理。

這三點都是真技術，但不再像一開始那樣不可名狀。

---

## 10. 粗調如何配合精修？

最後要讓 remainder (r) 落入精修塊可處理尺度。這可用普通貪心：

先使用一些合法 (pq)-項，不必追求最終模條件，把

[  
1/b  
]

減到一個小正 remainder

[  
r  
]

滿足

[  
Q^{2\alpha-2}\asymp r  
]

並且避開後續素數。因為 (pq)-單位分數非常豐富，這種粗調應該可做到；甚至可以用普通埃及分數表示的思想，選分母 (pq) 逐步逼近。

不過這裏要小心：粗調用了 (pq)-項後，它也產生模殘差。精修塊要同時修正這些殘差。這正是精修引理中「給定舊模數 (L_0) 與目標 residue」的用途。

所以完整證明會是：

1. 粗調選一個初始邊集 (E_0)，使 remainder (r) 小且正；
    
2. 記錄 (E_0) 在所有已用素數上的殘差；
    
3. 用精修引理選新二部塊，精確補上 (r) 並修正殘差。
    

---

## 11. 這條路現在的「走通感」

比上一版更好。因為精修塊把主弧 local CLT 限制在一個同尺度二部圖中，而不是整個多尺度世界。這使技術更標準：

- 非主弧：Irving + 互逆分散；
    
- 主弧：二部 Bernoulli sum local CLT；
    
- 舊模數：有限多 residue 控制；
    
- 粗調：貪心逼近。
    

所以目前最有希望的完整架構是：

[  
\boxed{  
\text{粗調}+\text{二部精修塊 Fourier}.  
}  
]

下一步如果繼續，就應該先證「二部精修塊 local Fourier 引理」：在 (Y,Z\subset[Q,2Q])、(|Y|=|Z|=N\gg\log Q)、互逆分散的條件下，Bernoulli 子集和在相容目標上有正概率。
