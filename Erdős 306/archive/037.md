好，現在直接推「二部精修塊 Fourier 引理」。但我先指出一個更簡潔的替代：與其真的證完整 local CLT，不如證一個**有限群子集和覆蓋引理**。這會更適合我們，因爲原目標是「存在一個子集」，不是要求概率漸近式。

---

## 1. 二部精修塊的有限群版本

設已有舊分母乘積

[  
L_0  
]

是 squarefree。選兩組新素數

[  
Y,Z\subset[Q,2Q],  
\qquad |Y|=|Z|=N.  
]

令

[  
L=L_0\prod_{y\in Y}y\prod_{z\in Z}z.  
]

候選邊是

[  
Y\times Z.  
]

每條邊 (yz) 給出整數權重

[  
w_{yz}=\frac{L}{yz}.  
]

我們想證：對一大批目標

[  
t\in \mathbb Z/L\mathbb Z  
]

尤其是那些實數大小與總可用質量相容的目標，存在子集 (E\subseteq Y\times Z)，使

[  
\sum_{(y,z)\in E}w_{yz}\equiv t\pmod L.  
]

如果再確保實際總和位於正確範圍，就能得到等式而非僅同餘。

所以精修塊本質是：

[  
\boxed{  
{w_{yz}:y\in Y,z\in Z}  
\text{ 的子集和在 }\mathbb Z/L\mathbb Z  
\text{ 中覆蓋一大塊目標。}  
}  
]

---

## 2. 為什麼完整覆蓋 (\mathbb Z/L\mathbb Z) 不可能？

候選邊只有

[  
N^2  
]

條，子集數最多

[  
2^{N^2}.  
]

而群大小是

[  
L=L_0\prod_{y,z} yz  
\asymp L_0 Q^{2N}.  
]

若

[  
N^2\gg N\log Q,  
]

也就是

[  
N\gg\log Q,  
]

則

[  
2^{N^2}  
]

遠大於 (L)，計數上覆蓋全群可能。

這很好。取

[  
N=Q^\alpha  
]

完全足夠。

但覆蓋全群仍可能過強，因為實際大小受限。精修塊總質量約

[  
W\asymp \frac{N^2}{Q^2}.  
]

它只能表示

[  
0\le \sum_{E}\frac1{yz}\le W.  
]

所以它不可能命中實數代表超出 ([0,W]) 的同餘類。故正確說法是：覆蓋那些代表落在 ([0,W]) 中的相容目標。

---

## 3. 隨機 Bernoulli 法

取

[  
X_{yz}\sim \mathrm{Bernoulli}(\theta)  
]

獨立，並令

[  
S=\sum_{y,z}X_{yz}w_{yz}.  
]

對目標 (t)，Fourier 反演：

# [  
\mathbb P(S=t)

\frac1L\sum_{\lambda}  
\widehat\mu(\lambda)e(-\lambda t/L).  
]

我們不一定要算精確主項；只需證它大於 (0)。

若所有非主弧 Fourier 總貢獻小，而主弧給正值，就成功。

---

## 4. 主弧估計：可用「小頻率」處理

對角 character 對應整數頻率 (h)：

[  
\lambda_r\equiv h\pmod r  
]

對所有 (r\mid L)。

這時

# [  
\widehat\mu(h)

\prod_{y,z}  
\left(1-\theta+\theta e\left(\frac{h}{yz}\right)\right).  
]

這就是實數和

[  
X=\sum_{y,z}\frac{X_{yz}}{yz}  
]

在頻率 (h) 的 characteristic function。

主弧是

[  
|h|\lesssim \frac1{\sigma},  
]

其中

# [  
\sigma^2

\mathrm{Var}(X)  
\asymp  
\theta(1-\theta)\sum_{y,z}\frac1{y^2z^2}.  
]

因為 (y,z\asymp Q)，有

[  
\sum_{y,z}\frac1{y^2z^2}  
\asymp \frac{N^2}{Q^4}.  
]

所以

[  
\sigma\asymp \frac{N}{Q^2}.  
]

若總質量

[  
W\asymp \frac{N^2}{Q^2}  
]

與 remainder (r) 同尺度，則

[  
\sigma\asymp \frac{\sqrt W}{Q}.  
]

主弧寬度約

[  
1/\sigma\asymp \frac{Q^2}{N}.  
]

這是可控的。

標準 local CLT 會說：若目標 (t/L) 距期望在 (O(\sigma)) 內，則

[  
\mathbb P(S=t)\approx \frac{1}{\sigma L}.  
]

這就是主弧正項。

---

## 5. 次弧估計：Irving 負責互逆分散

非對角 character 有某些坐標不等於同一個 (h)。設其離對角距離為

[  
D(\lambda)=  
\min_h  
#{r\in Y\cup Z:\lambda_r\not\equiv h\pmod r}.  
]

若 (D(\lambda)=d)，則有至少 (d) 個異常頂點。對每個異常 (y\in Y)，跨向 (Z) 的邊相位殘差包含

[  
\frac{(\lambda_y-h)z^{-1}}y.  
]

由 Irving Theorem 1 推出的互逆分散，對所有非零 (a) 有

[  
\sum_{z\in Z}  
\left|  
\frac{a z^{-1}}y  
\right|^2  
\gg N.  
]

所以每個異常頂點提供約 (N) 的平方相位總量。故

[  
|\widehat\mu(\lambda)|  
\le  
\exp(-c\theta Nd)  
]

至少在非主弧區域成立。

character 數方面，距離 (d) 的 character 至多

[  
\binom{2N}{d}Q^d  
]

乘上對角參數 (h) 的數量。其對數約

[  
O(d\log Q).  
]

若

[  
N\gg \log Q,  
]

則

[  
e^{-cNd}  
]

壓過

[  
e^{O(d\log Q)}.  
]

所以非對角 characters 可求和壓掉。

這是 Irving 估計真正派上用場之處。

## 6. 主弧中等頻率

還有一類 character 是對角型，但 (h) 不小。它們不是非對角，Irving 不殺它們；要靠一維 Bernoulli 和的 characteristic function 衰減。

對角 Fourier 係數：

[  
\prod_{y,z}  
\left|  
1-\theta+\theta e\left(\frac{h}{yz}\right)  
\right|.  
]

若很多 (yz) 使

[  
\left|\frac{h}{yz}\right|  
]

不太小，就有衰減。

這就是 ordinary local CLT 的 minor arcs。因 (yz\asymp Q^2)，若

[  
|h|\gg Q^2/N,  
]

相位開始震盪；若

[  
h  
]

接近某些 (yz) 的倍數，仍需平均控制。因 (yz) 種類很多，應可證沒有中等 (h) 同時對大多數 (yz) 太接近整數，除非 (h) 極小或極大接近 (L)。

這是 local CLT 的標準但技術性部分。

所以精修引理的兩個 Fourier 難點是：

1. 非對角：由 Irving；
    
2. 對角中頻：由一維權重 (1/(yz)) 的反集中。
    

第二點不依賴素數逆元分佈，只是實數權重集合的問題。

## 7. 可否避免完整 local CLT？

也許可以。取 (\theta=1/2)，考慮所有子集和

[  
\sum_{y,z}\frac{\varepsilon_{yz}}{yz}.  
]

如果能證它在區間 ([0,W]) 中形成間距至多 (1/L) 的覆蓋，則直接完成。這是 deterministic subset-sum covering。

但這通常比 local CLT 還難。隨機 Fourier 似乎是更合適的工具。

---

## 8. 精修引理目前可表述為條件定理

我們可以先把未證的主弧部分也作為一個概率引理：

**二部精修 Fourier 引理。**  
令 (Y,Z\subset[Q,2Q])，(|Y|=|Z|=N\gg\log Q)，且互逆分散成立。令 (L=L_0\prod_Y y\prod_Z z)。若目標 (t/L) 位於 Bernoulli 和

[  
\sum_{y,z}X_{yz}/(yz)  
]

的 central range，則存在 (E\subseteq Y\times Z)，使

[  
\sum_{(y,z)\in E}\frac1{yz}=t/L  
]

並滿足相應 CRT 條件。

該引理的證明分兩部分：

- 非對角 Fourier：由互逆分散；
    
- 對角 Fourier：由一維 local CLT for ({1/(yz)})。
    

因此整個原題可以歸約為：

[  
\boxed{  
\text{Irving Theorem 1}  
+  
\text{二部權重 local CLT}  
+  
\text{粗調 remainder}  
}  
]

## 9. 粗調 remainder 也要寫成引理

需要一個粗調引理：

給定

[  
1/b  
]

和任意很小尺度 (r_0)，可用互異 (pq)-分母選出一個和

[  
s<1/b  
]

使

[  
0<1/b-s<r_0.  
]

這其實容易。因為

[  
\sum_{p<q}\frac1{pq}=\infty  
]

且單項可任意小。用貪心：每次選一個尚未使用的 (pq) 分母，且

[  
\frac1{pq}<\text{當前剩餘}  
]

但不要太小。由於 (pq)-分母足夠密，應能讓剩餘趨於 (0)。嚴格證明需要一個密度引理：對任意小 (R)，存在 (pq) 使

[  
cR<\frac1{pq}<R  
]

或能用一批同尺度 (pq) 項填充固定比例。這應該不難，因為半素數很多。

粗調不要求 squarefree 分母以外的模精確，因為精修會修正。

---

## 10. 現在的完整路線

整理如下：

1. **粗調引理：** 用 (pq)-單位分數把 (1/b) 逼近到任意小 remainder (r)。
    
2. **選精修尺度：** 取 (Q,N) 使
    
    [  
    N^2/Q^2\asymp r,\qquad N\gg\log Q.  
    ]
    
3. **Irving 好塊：** 用 Irving Theorem 1 選 (Y,Z\subset[Q,2Q])，(|Y|=|Z|=N)，互逆分散。
    
4. **二部精修 Fourier：** 在 (Y\times Z) 中精確表示 remainder，同時修正粗調殘差。
    
5. **避開重複：** 每次選新素數，確保所有分母互異。
    

這是一條相當明確的攻擊線。

---

## 11. 現在最該證哪個？

我認為最該先證的是**粗調引理**，因為它比較容易，能讓整條路線少一個不確定點。然後集中火力證二部精修 Fourier 引理。

所以如果繼續，我會先證：

[  
\boxed{  
\text{用互異 }pq\text{ 單位分數可從下方逼近任意正實數。}  
}  
]

這是原題的弱化版，不含精確性，但應該可用半素數倒數和發散和單項趨零證出。
