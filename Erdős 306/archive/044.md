好。現在處理「質量設計」。這一步的任務是：給定

[  
\frac1b,  
]

選兩個大的素數集合 (C,Y)，使主塊

[  
C\times Y  
]

的總可用質量

# [  
W_{CY}=\sum_{c\in C,y\in Y}\frac1{cy}

\left(\sum_{c\in C}\frac1c\right)  
\left(\sum_{y\in Y}\frac1y\right)  
]

與 (1/b) 同階，最好略大於 (1/b)，以便用 Bernoulli 概率 (\theta\in(0,1)) 調到期望正好為 (1/b)。

---

## 1. 質量設計可以做到

因爲

[  
\sum_p\frac1p=\infty,  
]

對任意 (A>0)，都可以選一個有限素數集合 (C)，避開任意有限禁用集，使

[  
A\le \sum_{c\in C}\frac1c\le A+\varepsilon.  
]

這可用最普通的貪心／截斷：素數倒數和發散，單項趨零。

因此可以選

[  
\sum_{c\in C}\frac1c\approx A,  
\qquad  
\sum_{y\in Y}\frac1y\approx \frac{2}{bA}.  
]

那麼

[  
W_{CY}\approx \frac2b.  
]

取

[  
\theta=\frac{1/b}{W_{CY}}\approx\frac12.  
]

這很好，因爲 (\theta) 是固定遠離 (0,1) 的常數，Fourier 衰減與方差都穩定。

---

## 2. 但不能任意稀疏選素數

如果 (C,Y) 是零散挑出來的素數，Irving 的 dyadic 估計不方便用，local CLT 也不容易整理。更好的做法是讓 (C,Y) 是許多 dyadic 素數塊的併：

[  
C=\bigcup_i C_i,\qquad C_i\subset[Q_i,2Q_i],  
]

[  
Y=\bigcup_j Y_j,\qquad Y_j\subset[R_j,2R_j].  
]

每個 (C_i,Y_j) 取該 dyadic 區間中大部分 good primes。因為一個 dyadic prime interval 的倒數和約為

[  
\sum_{Q\le p<2Q}\frac1p\asymp\frac1{\log Q}.  
]

所以通過取很多 dyadic 區間，可以讓總倒數和達到任何固定大小。

這避免了「單一 dyadic 塊質量太小」的問題。

---

## 3. 是否需要所有 (C_i\times Y_j) 邊？

不一定。若 (Q_i) 和 (R_j) 尺度差太大，Irving 的 (x,Q) 範圍可能不便。最安全是只連接相近尺度的 blocks：

[  
C_i\times Y_i  
]

或

[  
C_i\times Y_j\quad\text{with}\quad Q_i\asymp R_j.  
]

於是主質量為

# [  
W_{CY}

\sum_i  
\left(\sum_{c\in C_i}\frac1c\right)  
\left(\sum_{y\in Y_i}\frac1y\right).  
]

若每個 dyadic block 的倒數和約為 (u_i)，則每對貢獻約

[  
u_i^2.  
]

我們需要

[  
\sum_i u_i^2\asymp \frac1b.  
]

而每個

[  
u_i\asymp \frac1{\log Q_i}  
]

可以很小。取足夠多 blocks 便可使平方和達到 (1/b)。

這是可行的。

---

## 4. 更靈活的配對設計

也可以令每個 block 的倒數和人為調成固定小量 (u)，例如

[  
u=\eta  
]

很小。方法是在每個 dyadic 區間中取一部分 good primes，使其倒數和約 (\eta)。只要整個區間倒數和大於 (\eta)，即可。

但高 (Q) 時整個 dyadic 區間倒數和只有約 (1/\log Q)，不能取固定 (\eta)。所以 (\eta) 需隨 (Q) 變小，或使用較寬的 interval 而非單 dyadic。

爲了保持 Irving 估計，我們可以把一個較寬 interval 切成 dyadic pieces；每 piece 分析，最後相加。

技術上可行，但 bookkeeping 會變多。

---

## 5. 最簡質量方案

選一串 (Q_i) 增長緩慢的 dyadic intervals，使

[  
\sum_i \frac1{(\log Q_i)^2}  
]

超過 (2/b)。

例如取

[  
Q_i=\exp(i)  
]

則

[  
\frac1{(\log Q_i)^2}=\frac1{i^2},  
]

平方和收斂，不夠。  
若取

[  
Q_i=\exp(\sqrt i),  
]

則

[  
\frac1{(\log Q_i)^2}=\frac1{i},  
]

和發散。這樣可積累任意質量。

但相鄰 (Q_i) 比值

[  
\exp(\sqrt{i+1}-\sqrt i)\to1,  
]

所以 dyadic intervals 會高度重疊；實際上應改成按 dyadic 不重疊區間 (2^k)。對 dyadic (Q=2^k),

[  
\frac1{(\log Q)^2}\asymp \frac1{k^2},  
]

和收斂。這說明：若每個 dyadic 區間只用自身與自身配對，總質量

[  
\sum_k \frac1{k^2}  
]

是有限的。

但有限不代表不夠；其總和約為常數。若 (1/b\le1)，可能足夠？注意

[  
\sum_k 1/k^2  
]

有限，但正數。完整所有 dyadic self-pair blocks 總質量相當於

[  
\sum_k\left(\sum_{p\in[2^k,2^{k+1}]}\frac1p\right)^2,  
]

確實收斂。它不能提供任意大質量，但我們只需 (1/b\le1)。若 (b=1)，要表示 (1)，可能總量是否足夠要看常數。所有 primes 的 pair sum

[  
\sum_{p<q}\frac1{pq}  
]

發散，然而只取同尺度 pairs 的 sum 收斂；發散主要來自跨尺度 pairs。

所以只連接同尺度 blocks 可能質量不足。

這很重要。

---

## 6. 必須使用跨尺度邊

因爲

[  
\left(\sum_p\frac1p\right)^2  
]

的發散包含大量不同尺度的乘積。若只取 (p,q) 同 dyadic 尺度，得到的是

[  
\sum_k\left(\frac1k\right)^2  
]

收斂。要得到發散，必須使用跨 dyadic block pairs：

[  
C_i\times Y_j  
\quad\text{for many }i\ne j.  
]

這是新關鍵。

於是我們不能只依賴 Irving 的同尺度估計。需要處理 (p\sim X)、(q\sim Q) 不同尺度的互逆分散。

好消息：Irving Theorem 1 的條件是

[  
Q^{3/2}\ge x\ge Q^{2/3}.  
]

也就是允許

[  
x  
]

與

[  
Q  
]

相差相當大的冪次。這正好支持跨尺度，但不是任意遠跨尺度。

所以我們可以連接尺度比在範圍內的 blocks：

[  
Q^{2/3}\le X\le Q^{3/2}.  
]

在 dyadic exponent (k=\log Q) 上，這允許相鄰尺度差一個固定乘法因子於 exponent：

[  
\frac23\log Q\le \log X\le \frac32\log Q.  
]

也就是可以連接相當寬的尺度鄰域。

若把尺度圖建成一張 graph，頂點是 dyadic levels (k)，可連接 (k,\ell) 若

[  
\frac23 k\le \ell\le \frac32 k.  
]

這張 graph 很連通，且應足以讓跨尺度 pair mass 發散。

---

## 7. 質量是否足夠：尺度圖計算

令 dyadic level (k) 的 prime reciprocal mass 約為

[  
u_k\asymp \frac1k.  
]

若允許連接 levels (k,\ell) 且

[  
\ell\asymp k,  
]

更具體 (\ell\in[2k/3,3k/2])，則每個 (k) 可連接約 (k) 個 (\ell)。總質量約為

# [  
\sum_k \sum_{\ell\asymp k}\frac1{k\ell}  
\asymp  
\sum_k k\cdot\frac1{k^2}

\sum_k\frac1k  
=\infty.  
]

很好。這說明 Irving 可用的跨尺度範圍正好足以恢復發散質量。

這是非常好的徵兆。

---

## 8. 因此主質量設計可行

選一個大尺度區間集合 (K)：

[  
k_0\le k\le K.  
]

對每個 level (k)，取素數 block

[  
P_k=[2^k,2^{k+1}]\cap\mathbb P  
]

中的 good primes。

候選主邊連接所有 pairs (P_k\times P_\ell)，其中

[  
\frac23 k\le \ell\le\frac32 k.  
]

每個 such pair 在 Irving 可控範圍內。

總可用質量約為

[  
\sum_{k=k_0}^{K}\sum_{\ell\asymp k}\frac1{k\ell}  
\asymp  
\sum_{k=k_0}^{K}\frac1k  
\asymp \log K-\log k_0.  
]

因此可調到任意固定大小，例如 (2/b)。

這是一個非常漂亮的結果：  
**Irving 的範圍不但夠處理分散，還正好允許足夠跨尺度質量發散。**

---

## 9. 新的主模型

不必再明確分 (C,Y) 兩個長集合。可以取許多 dyadic layers

[  
P_k  
]

並在尺度相近但允許冪次差的 layers 之間連邊。

圖的頂點層是 dyadic prime blocks；邊連接「Irving-compatible」block pairs。

這個模型：

- 有足夠總質量；
    
- 每個 block pair 可用 Irving 處理互逆分散；
    
- 多尺度權重有利於 local CLT；
    
- 固定邊界 (B) 可接到某些低層或專門輔助層。
    

這比四層固定模型更自然，也不是空抽象，而是由質量發散與 Irving 範圍共同逼出。

---

## 10. 目前真正的走通徵兆大幅變好

我們剛剛解決了一個很重要的擔心：單尺度質量不足。跨尺度是必須的，而 Irving 的 (Q^{2/3}\le x\le Q^{3/2}) 範圍正好允許足夠多跨尺度連邊，使總質量發散：

[  
\sum_k\frac1k=\infty.  
]

這是非常強的正面信號。

所以現在路線更凝練：

[  
\boxed{  
\text{取 dyadic prime blocks；連接 Irving 可控的尺度對；用隨機子集和 Fourier。}  
}  
]

主弧質量來自

[  
\sum_{k}\sum_{\ell\asymp k}\frac1{k\ell}\sim\sum_k\frac1k.  
]

次弧衰減來自 Irving Theorem 1。

剩下的主要技術是把這兩者合併成完整 Fourier 證明。

---

## 11. 下一步

下一步要定義「good dyadic block graph」：

- 頂點：dyadic prime blocks (P_k);
    
- 邊：((k,\ell)) 滿足 (2k/3\le \ell\le3k/2);
    
- 候選 (pq)-分母：(p\in P_k,q\in P_\ell), (p\ne q).
    

然後證兩件事：

1. 總質量可調：  
    [  
    \sum_{(k,\ell)}\sum_{p\in P_k,q\in P_\ell}\frac1{pq}  
    ]  
    可取到 (>1/b) 且有限。
    
2. 對任意非對角 character，沿這張 block graph 有足夠多 good edges 產生 Fourier 衰減。
    

這將是下一個真正的核心步驟。
