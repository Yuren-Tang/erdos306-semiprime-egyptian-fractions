是，**現在正應該檢索工具**；而且檢索結果支持這個判斷。主弧問題不是一般 CLT，而是「帶權 Bernoulli 和的格點局部極限定理」。這正是 Giuliano–Weber 那類文章的題目：他們明確研究 weighted sums of Bernoulli variables，並用 characteristic function 方法證明數論隨機模型中的 local limit theorem。這與我們的

[  
\sum_{pq\in\mathcal E}X_{pq}\frac1{pq}  
]

形式非常接近。([arXiv](https://arxiv.org/abs/1502.05939?utm_source=chatgpt.com "Local Limit Theorems in some Random models from Number Theory"))

所以現在不應繼續憑空分類 (h)，而應引入一個標準黑箱：

---

## 1. 我們需要的 local CLT 黑箱

設

[  
S=\sum_i a_i X_i,  
]

其中 (X_i) 是獨立 Bernoulli 變量，(a_i\in\mathbb Z)，且

[  
\gcd(a_i)=1.  
]

令

[  
\mu=\mathbb E S,\qquad \sigma^2=\operatorname{Var}(S).  
]

若滿足：

[  
\max_i |a_i|=o(\sigma),  
]

且對所有非小頻率 (t) 有足夠的 aperiodicity：

[  
\prod_i\left|1-\theta+\theta e(t a_i)\right|  
]

很小，則有格點局部中心極限定理：

[  
\mathbb P(S=m)  
\sim  
\frac{1}{\sqrt{2\pi}\sigma}  
\exp\left(-\frac{(m-\mu)^2}{2\sigma^2}\right)  
]

對中心區域內的整數 (m) 成立。

這種證明通常正是 characteristic function method；Giuliano–Weber 的摘要也說他們用此方法證 weighted Bernoulli variables 的 local limit theorem。([arXiv](https://arxiv.org/abs/1502.05939?utm_source=chatgpt.com "Local Limit Theorems in some Random models from Number Theory"))

---

## 2. 套到我們的權重

我們的整數權重是

[  
a_{pq}=\frac{L}{pq},  
]

其中

[  
L=\prod_{p\in P}p.  
]

隨機和是

[  
S=\sum_{pq\in\mathcal E}X_{pq}\frac{L}{pq}.  
]

歸一化後是

[  
\frac SL=\sum_{pq\in\mathcal E}X_{pq}\frac1{pq}.  
]

前面已經核對了三個條件：

[  
\gcd_{pq\in\mathcal E}\frac{L}{pq}=1,  
]

因為每個素數頂點都出現在某條邊中。

方差方面，若最小尺度為 (2^{k_0})，則

[  
\sigma_X\asymp \frac1{k_0 2^{k_0}}  
]

對實數和 (X=S/L) 而言。

最大單項權重是

[  
a_{\max}^{(X)}\asymp 2^{-2k_0}.  
]

所以

[  
\frac{a_{\max}^{(X)}}{\sigma_X}  
\asymp  
\frac{k_0}{2^{k_0}}\to0.  
]

也就是最大權重遠小於標準差。這是非常好的 local CLT 訊號。

---

## 3. 剩下唯一非平凡條件：aperiodicity

所有問題都集中到：

[  
\prod_{pq\in\mathcal E}  
\left|1-\theta+\theta e\left(\frac{h}{pq}\right)\right|  
]

在非小頻 (h) 上是否足夠小。

用估計

[  
|1-\theta+\theta e(t)|  
\le  
\exp(-c_\theta|t|^2),  
]

只需證：

[  
\sum_{pq\in\mathcal E}  
\left|\frac{h}{pq}\right|^2  
]

在非小頻 (h) 上很大。

這就是 local CLT 黑箱中的 aperiodicity condition。換言之，主弧剩下的核心已不是「證一個新 CLT」，而是：

[  
\boxed{  
\text{證我們的半素數權重陣列滿足 standard aperiodicity。}  
}  
]

---

## 4. 這比前面的分類好在哪裏？

前面我們把 (h) 分成：

- 小 (h);
    
- 大 (h);
    
- (h) 被很多素數整除;
    
- (h) 與 (L) 互素;
    
- cut case;
    
- divisor case.
    

這些分類都只是想理解何時

[  
\sum_{pq}|h/(pq)|^2  
]

會小。現在統一起來看，真正命題只有一個：

[  
\boxed{  
\mathcal E(h):=\sum_{pq\in\mathcal E}|h/(pq)|^2  
\text{ 小，只能發生在 small major arc。}  
}  
]

這就是應該提交給 local CLT 的假設。分類最多只是證明這個假設時的輔助，而不應成為最終框架。

## 5. 檢索後的工作分配

目前可把全證明拆為兩個黑箱與一個構造：

### A. Irving 黑箱：非對角衰減

Irving Theorem 1 給出 prime reciprocal Kloosterman sums 的平均界，而且帶

[  
\max_{(a,q)=1}.  
]

這足以支撐非對角方向。

### B. Weighted Bernoulli local CLT 黑箱：對角主弧

Giuliano–Weber 類工具研究 weighted Bernoulli sums，並用 characteristic functions 處理數論模型中的 local limit theorem。([arXiv](https://arxiv.org/abs/1502.05939?utm_source=chatgpt.com "Local Limit Theorems in some Random models from Number Theory"))

### C. 我們要證的特定 aperiodicity

也就是：

[  
\forall h\notin\mathfrak M,\qquad  
\sum_{pq\in\mathcal E}  
\left|\frac{h}{pq}\right|^2  
\gg \text{enough}.  
]

這是我們對現有 local CLT 工具的輸入。

## 6. 下一步應該怎樣？

不應再泛泛說「local CLT 應該成立」。應把 aperiodicity estimate 寫成一個具體命題：

**半素數權重非週期命題。**  
令 (\mathcal E) 是 dyadic Irving-compatible block graph 上的邊集合。令

[  
\sigma^2=\theta(1-\theta)\sum_{pq\in\mathcal E}\frac1{p^2q^2}.  
]

則存在 (C,c>0)，使對所有

[  
C/\sigma\le |h|\le L/2  
]

有

[  
\sum_{pq\in\mathcal E}  
\left|  
\frac{h}{pq}  
\right|^2  
\ge  
A(h),  
]

其中 (A(h)) 足以保證

[  
\sum_{C/\sigma\le |h|\le L/2}  
\exp(-cA(h))  
=o(1/\sigma).  
]

如果這個命題成立，主弧 local CLT 就基本可接上。
