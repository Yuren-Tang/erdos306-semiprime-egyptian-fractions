好，現在把 **matching–CRT aperiodicity** 寫成一個較正式的估計。

我們先抽象化。令 (P) 是有限素數集合，

[  
L=\prod_{p\in P}p.  
]

令 (G=(P,\mathcal E)) 是候選邊圖。每條邊 (e={p,q}) 對應權重

[  
w_e=\frac{L}{pq}.  
]

對角頻率 (h\in\mathbb Z/L\mathbb Z) 的 Fourier 因子是

# [  
\Phi(h)

\prod_{e={p,q}\in\mathcal E}  
\left(  
1-\theta+\theta e\left(\frac{h}{pq}\right)  
\right).  
]

其中

[  
e(x)=e^{2\pi i x}.  
]

我們要控制

[  
|\Phi(h)|.  
]

## 1. 大 Fourier 係數推出多數邊近整除

有基本估計：

[  
|1-\theta+\theta e(t)|  
\le  
\exp(-c_\theta|t|^2).  
]

因此

[  
|\Phi(h)|  
\le  
\exp\left(  
-c_\theta  
\sum_{pq\in\mathcal E}  
\left|  
\frac{h}{pq}  
\right|^2  
\right).  
]

反過來，若

[  
|\Phi(h)|\ge e^{-R},  
]

則

[  
\sum_{pq\in\mathcal E}  
\left|  
\frac{h}{pq}  
\right|^2  
\le C_\theta R.  
]

取一個小常數 (\delta>0)。令壞邊集合

# [  
\mathcal B_h(\delta)

\left{  
pq\in\mathcal E:  
\left|  
\frac{h}{pq}  
\right|>\delta  
\right}.  
]

則

[  
|\mathcal B_h(\delta)|\delta^2  
\le  
C_\theta R,  
]

所以

[  
|\mathcal B_h(\delta)|  
\le  
\frac{C_\theta R}{\delta^2}.  
]

因此若 (R) 不大，絕大多數邊滿足

[  
\left|  
\frac{h}{pq}  
\right|\le\delta.  
]

也就是說，(h) 被絕大多數候選半素數 (pq) 近整除。

---

## 2. 從好邊中抽 matching

令好邊集合

# [  
\mathcal G_h(\delta)

\mathcal E\setminus\mathcal B_h(\delta).  
]

若

[  
|\mathcal E|=m,\qquad \Delta=\max_{v\in P}\deg(v),  
]

則

[  
|\mathcal G_h(\delta)|  
\ge  
m-\frac{C_\theta R}{\delta^2}.  
]

任意圖中，含 (M) 條邊、最大度不超過 (\Delta) 的子圖有 matching 大小至少

[  
\frac{M}{2\Delta}.  
]

因此 (\mathcal G_h(\delta)) 中有 matching

[  
e_1,\dots,e_s,  
\qquad e_i={p_i,q_i},  
]

其中

[  
s  
\ge  
\frac{1}{2\Delta}  
\left(  
m-\frac{C_\theta R}{\delta^2}  
\right).  
]

這些邊兩兩不共享素數，所以

[  
n_i=p_iq_i  
]

兩兩互素。

並且對每個 (i)，

[  
\left|  
\frac{h}{n_i}  
\right|\le\delta.  
]

---

## 3. CRT 壓縮頻率數

條件

[  
\left|  
\frac{h}{n_i}  
\right|\le\delta  
]

等價於

[  
h\bmod n_i  
]

落在模 (n_i) 的一段長度約 (2\delta n_i) 的短區間中：

[  
h\bmod n_i  
\in I_i,  
\qquad |I_i|\le 2\delta n_i+2.  
]

因為 (n_i) 兩兩互素，CRT 給出：模

[  
N_*=\prod_{i=1}^s n_i  
]

下，滿足全部條件的 residue 數至多

[  
\prod_{i=1}^s (2\delta n_i+2).  
]

若所有 (n_i) 很大，且 (\delta) 固定小，則

[  
2\delta n_i+2\le 3\delta n_i.  
]

故 residue 數至多

[  
(3\delta)^s N_*.  
]

由於 (N_*\mid L)，在模 (L) 中滿足這些條件的 (h) 數量至多

[  
(3\delta)^s L.  
]

所以：

[  
\boxed{  
|{h\bmod L:\ h\text{ 滿足該 matching 的近整除條件}}|  
\le  
(3\delta)^s L.  
}  
]

這是核心壓縮。

## 4. 但 matching 本身可變，需計數

上面固定了一個 matching。可是大 Fourier 係數只保證存在某個 matching。要數所有可能 (h)，還要乘上 matching 的數量。

若圖有 (m) 條邊，大小為 (s) 的 matching 數至多

[  
\binom{m}{s}\le \left(\frac{em}{s}\right)^s.  
]

所以滿足「存在大小 (s) matching 且在其上近整除」的 (h) 數量至多

# [  
\left(\frac{em}{s}\right)^s (3\delta)^s L

\left(\frac{3e\delta m}{s}\right)^s L.  
]

這裏出現一個問題：如果

[  
m/s  
]

很大，則需要 (\delta) 極小才能使壓縮有效。

由於

[  
s\approx m/(2\Delta),  
]

有

[  
m/s\approx 2\Delta.  
]

因此壓縮因子約為

[  
( C\delta\Delta)^s.  
]

若最大度 (\Delta) 巨大，固定 (\delta) 不夠。這是重要問題。

## 5. 如何避免 matching 計數爆炸？

不應任意選 matching。應該預先固定一批 disjoint edges，也就是固定一個大 matching

[  
\mathcal M\subset\mathcal E  
]

作為測試集。

然後只用這個固定 matching 來做 aperiodicity。這樣沒有 matching 計數因子。

這是更乾淨的做法。

我們可以在候選圖中預先選一個很大的 matching

[  
\mathcal M={p_iq_i}_{i=1}^s,  
]

並確保這個 matching 的總權重與方差雖然不一定提供主質量，但足以測試高頻週期。

對 Fourier 係數有

[  
|\Phi(h)|  
\le  
\prod_{p_iq_i\in\mathcal M}  
\left|  
1-\theta+\theta e\left(\frac{h}{p_iq_i}\right)  
\right|.  
]

所以若在固定 matching 上已經有衰減，則全圖也有衰減。

這避免了「存在 matching」的計數問題。

## 6. 固定 matching 的 CRT 壓縮

對固定 matching

[  
\mathcal M={n_1,\dots,n_s},  
\qquad n_i=p_iq_i,  
]

且 (n_i) 兩兩互素。

若

# [  
|\Phi_{\mathcal M}(h)|

\prod_{i=1}^s  
\left|  
1-\theta+\theta e(h/n_i)  
\right|  
\ge e^{-R},  
]

則

[  
\sum_{i=1}^s  
\left|  
\frac{h}{n_i}  
\right|^2  
\le C R.  
]

For dyadic energy decomposition, choose threshold (\delta). Then all but at most (CR/\delta^2) of the (i)'s satisfy

[  
|h/n_i|\le\delta.  
]

Let

[  
s'=s-\frac{CR}{\delta^2}.  
]

Choose a fixed subset? Here the good indices vary. Count by choosing the exceptional set.

Number of choices for exceptional set:

[  
\sum_{j\le CR/\delta^2}\binom{s}{j}.  
]

For the remaining (s') indices, CRT compression gives

[  
(3\delta)^{s'}L.  
]

Thus number of (h) with matching-energy (\le R) is at most

[  
\sum_{j\le CR/\delta^2}  
\binom{s}{j}  
(3\delta)^{s-j}L.  
]

If (R\ll s), choose (\delta) small fixed; this is roughly

[  
\exp(O(R)) (3\delta)^s L,  
]

very small.

This is usable.

## 7. 但有一個嚴重問題：乘上 (L) 太大

我們最後需要估計

[  
\sum_h |\Phi(h)|.  
]

如果小能量 (h) 的數量上界是

[  
(3\delta)^s L,  
]

乘以每個 (|\Phi(h)|\le1)，仍然可能巨大；主項尺度需要的是約

[  
1/\sigma  
]

而不是 (L)。

因為 Fourier inversion has factor (1/L)。主弧 contribution is approximately (1/\sigma) before dividing by (L). We need

[  
\sum_{h\notin\mathfrak M}|\Phi(h)|=o(1/\sigma).  
]

So count bound must be (o(1/\sigma)), not just small fraction of (L). Since (L) is enormous, ((3\delta)^s L) may still be enormous unless

[  
s\gg \log L.  
]

Can fixed matching size (s) exceed (\log L)? Let's estimate.

(L) product of all vertices. If we use all primes up to scales k0..K, then

[  
\log L\asymp \sum_{p\in P}\log p  
\asymp \sum_k #\mathcal P_k\cdot k  
\asymp \sum_k 2^k  
\asymp 2^K.  
]

A matching can have size at most number of vertices,

[  
s\lesssim |P|\asymp \sum_k \frac{2^k}{k}\asymp \frac{2^K}{K}.  
]

So

[  
s\asymp \frac{\log L}{K}.  
]

This is smaller than (\log L) by factor (K). Fixed matching CRT compression with constant (\delta) gives exponent (-cs), not enough to beat (L=e^{\log L}).

So fixed matching alone is insufficient.

## 10. Approximate cocycle rigidity

For exact zero phase on an edge (pq):

[  
\frac{\lambda_p q^{-1}}p+  
\frac{\lambda_q p^{-1}}q  
\equiv0\pmod1.  
]

Multiplying by (pq):

[  
q\lambda_p q^{-1}+p\lambda_q p^{-1}  
\equiv0\pmod{pq}.  
]

Modulo (p):

[  
\lambda_p\equiv0\pmod p.  
]

Modulo (q):

[  
\lambda_q\equiv0\pmod q.  
]

Wait exact zero for edge alone forces both endpoint residues zero. But diagonal small h does not have zero phase; it has (h/(pq)), small but not zero. For high-frequency minor arcs, large coefficient means phases small or structured, not exactly zero.

If phases small over many edges, then for each edge:

[  
\lambda_p q^{-1}/p+\lambda_q p^{-1}/q  
]

near integer. Multiplying by pq gives:

[  
\lambda_p+\lambda_q \quad?  
]

Let's compute mod p/q:  
Exact integer condition:  
[  
q\lambda_p(q^{-1}\bmod p)+p\lambda_q(p^{-1}\bmod q)\equiv0 \mod pq.  
]  
Modulo p gives (\lambda_p\equiv0). Mod q gives (\lambda_q\equiv0). So exact edge phase integer forces both residues zero. Thus if many edge phases are near integer, likely many vertex residues near zero in modular sense, i.e. h divisible by many primes. This matches divisibility support.

Small diagonal h has residues (\lambda_p=h), not zero, and edge phase (h/(pq)) small but not integer; in integer-multiplied form it equals h, small compared to pq. Thus near integer with tolerance (\delta) means after multiplying by pq:  
[  
\text{CRT integer} \in (-\delta pq,\delta pq).  
]  
For diagonal h, this integer is h. So many edge constraints say the same global integer h is small relative to pq.

Thus high h cannot satisfy many small-phase constraints unless h divisible by endpoints causing cancellations. This suggests graph constraints are strong.

## 15. Immediate concrete lemma

**Graph phase inverse lemma.**  
Let (G=(P,E)) be the Irving-compatible dyadic graph. For a character (\lambda=(\lambda_p)), define energy

# [  
\mathcal Q(\lambda)

\sum_{pq\in E}  
\left|  
\frac{\lambda_p q^{-1}}p+  
\frac{\lambda_q p^{-1}}q  
\right|^2.  
]

Then characters with (\mathcal Q(\lambda)\le R) are contained in a union of at most (\exp(O(R))) diagonal boxes

[  
\lambda_p\equiv h\pmod p  
]

with (|h|\lesssim R)-controlled, plus negligible exceptions.

If true, Fourier sum closes:

- low-energy characters are essentially major arcs;
    
- high-energy characters have exponential decay.
    

This is the unified statement we need.

It also incorporates diagonal high h: if diagonal h has low energy, the lemma forces h small. Good.
