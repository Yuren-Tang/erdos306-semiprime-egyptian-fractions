好，我們現在攻最小核心：

[  
P\times Q  
]

一個完整二部素數塊，其中

[  
P\subset [X,2X],\qquad Q\subset [Y,2Y],  
]

且 (X,Y) 在 Irving 可控範圍內，例如

[  
Y^{2/3}\le X\le Y^{3/2}.  
]

每個 (p\in P) 有 residue

[  
a_p\in\mathbb F_p,  
]

每個 (q\in Q) 有 residue

[  
b_q\in\mathbb F_q.  
]

邊相位是

# [  
\phi_{p,q}

\frac{a_p(q^{-1}\bmod p)}p  
+  
\frac{b_q(p^{-1}\bmod q)}q.  
]

等價地，令 (H_{p,q}) 是 CRT glue：

[  
H_{p,q}\equiv a_p\pmod p,  
\qquad  
H_{p,q}\equiv b_q\pmod q,  
\qquad  
0\le H_{p,q}<pq.  
]

則

[  
\phi_{p,q}\equiv \frac{H_{p,q}}{pq}\pmod1.  
]

若很多邊相位小，即

[  
\left|\phi_{p,q}\right|\le \delta,  
]

那就是說 (H_{p,q}) 在對稱代表下滿足

[  
|H_{p,q}^{*}|\le \delta pq.  
]

我們想證：若通過的邊很多，則存在一個小整數 (h)，使

[  
a_p\equiv h\pmod p  
]

對多數 (p)，且

[  
b_q\equiv h\pmod q  
]

對多數 (q)。

---

## 1. 先看精確版本

把 (\delta=0)。若對一條邊 (pq),

[  
\phi_{p,q}=0,  
]

則

[  
H_{p,q}=0,  
]

所以

[  
a_p=0,\qquad b_q=0.  
]

若完整二部圖中很多邊精確為零，立即推出很多端點 residue 為零。這對應全局小整數

[  
h=0.  
]

精確版本很強。

但真正主弧不是相位為零，而是相位等於

[  
h/(pq)  
]

且 (h) 小。因此應該把「小相位」理解為「由某個小 (h) 解釋」。

---

## 2. 用邊的 CRT glue 反推公共 (h)

若一條邊通過，則存在一個小整數

[  
h_{p,q}  
]

使

[  
|h_{p,q}|\le \delta pq  
]

且

[  
h_{p,q}\equiv a_p\pmod p,\qquad  
h_{p,q}\equiv b_q\pmod q.  
]

如果兩條邊共享 (p)：

[  
(p,q_1),\qquad(p,q_2),  
]

則

[  
h_{p,q_1}\equiv h_{p,q_2}\pmod p.  
]

但二者大小分別至多約

[  
\delta p q_1,\qquad \delta p q_2.  
]

如果 (q_1,q_2\asymp Y)，差值大小至多

[  
O(\delta pY).  
]

所以差值可能是 (p) 的許多倍；約有 (O(\delta Y)) 種可能。單靠一個共享頂點不夠。

但是若再比較經過 (q) 的許多邊，約束會疊加。完整二部結構應迫使大多數 (h_{p,q}) 接近同一個整數。

## 3. 取一個四環

考慮四個頂點

[  
p_1,p_2\in P,\qquad q_1,q_2\in Q.  
]

若四條邊都通過，得到四個小整數：

[  
h_{11},h_{12},h_{21},h_{22}.  
]

它們滿足：

[  
h_{11}\equiv h_{12}\pmod {p_1},  
]

[  
h_{21}\equiv h_{22}\pmod {p_2},  
]

[  
h_{11}\equiv h_{21}\pmod {q_1},  
]

[  
h_{12}\equiv h_{22}\pmod {q_2}.  
]

因此

[  
h_{11}-h_{12}-h_{21}+h_{22}  
]

同時受到模 (p_1,p_2,q_1,q_2) 的限制？更仔細：

它不一定被每個整除，但有一種 cocycle 關係。若所有 (h_{ij}) 來自同一個 global (h)，則這個二階差為 (0)。

由於每個 (h_{ij}) 都小於 (\delta XY)，二階差大小至多

[  
O(\delta XY).  
]

而任何非零的 CRT obstruction 若被一個大模數乘積整除，大小至少約 (XY)。如果 (\delta) 足夠小，可能迫使 obstruction 為零。

這是好方向：**四環一致性可能把近似 agreement 變成真正 agreement。**

Maybe use three edges: (h_{11},h_{12},h_{22}). They imply (h_{11}\equiv h_{12}\mod p_1), (h_{12}\equiv h_{22}\mod q_2). No global.

四環不直接給簡單整除。

## 5. 另一種看法：edge pass as interval condition for one side

For fixed (p) and (a_p), edge pass to (q) means:

[  
H=a_p+mp  
]

for some (m), and

[  
|H|_{pq}\le \delta pq.  
]

Modulo (q), this gives

[  
b_q\equiv a_p+mp\pmod q  
]

for some (m) in an interval of length (\approx 2\delta q). Hence allowed (b_q)'s form an arithmetic progression segment:

[  
b_q\in a_p+p\cdot[-\delta q,\delta q]\pmod q.  
]

Since (p) invertible mod (q), this is an interval of density (2\delta) in (\mathbb F_q), shifted by (a_p).

For many (p)'s, (b_q) must lie in intersection of many such intervals. Unless the intervals align.

What does alignment mean? For two (p_1,p_2), intervals

[  
a_{p_1}+p_1[-\delta q,\delta q],  
]

[  
a_{p_2}+p_2[-\delta q,\delta q]  
]

in (\mathbb F_q) have large intersection only if their centers are close:

[  
a_{p_1}\approx a_{p_2}\pmod q  
]

after scaling? More exactly intervals are images under multiplication by (p_i), so slopes differ. Multiplication by (p_i) turns a standard interval into a modular arithmetic progression not an interval in usual order; it may be pseudorandom. For prime q, sets (a_p+p[-M,M]) are intervals with step p. Two such sets of size (2M) have expected intersection (\approx 4M^2/q). If (M=\delta q), intersection density about (4\delta^2 q). For many p's, intersection shrinks exponentially unless structured.

This suggests we can use additive combinatorics / large sieve on intervals with varying steps.

But perhaps Irving already gives this: distribution of (p^{-1}) modulo q controls such interval overlaps.

## 6. Maybe bipartite agreement follows from Kloosterman short-arc

Fix (q) and (b_q). Edge pass condition as function of (p) can be written in phase:

[  
\left|  
\frac{a_p q^{-1}}p+  
\frac{b_q p^{-1}}q  
\right|\le\delta.  
]

If (a_p) were arbitrary depending on p, Irving cannot apply directly. But if many p pass for a fixed q, does it force (a_p) to correlate with (p) in a special way. Since (a_p) is arbitrary, one can define (a_p) to make edge pass for a chosen q: set (a_p\equiv h_q\pmod p). But to pass for many q simultaneously, (a_p) must satisfy many such choices, forcing global h.

This is agreement-testing again.

## 7. Potential proof via two random q's

Pick two typical vertices (q_1,q_2\in Q). For many (p), both edges ((p,q_1),(p,q_2)) pass. For such p, there exist small (h_1,h_2):

[  
h_1\equiv a_p\pmod p,\quad h_1\equiv b_{q_1}\pmod{q_1},  
]

[  
h_2\equiv a_p\pmod p,\quad h_2\equiv b_{q_2}\pmod{q_2}.  
]

Thus

[  
h_1\equiv h_2\pmod p.  
]

Both (h_1,h_2) have size (O(\delta pY)). Their difference is a multiple of (p):

[  
h_1-h_2=t_p p,  
\qquad |t_p|\ll\delta Y.  
]

So for each p, the two small CRT lifts differ by (p) times a small integer (t_p).

Can this hold for many p unless (t_p) is induced by a common difference? The residues modulo (q_1,q_2) of (h_1,h_2) are fixed (b_{q_1},b_{q_2}). Therefore

[  
h_1-h_2 \equiv b_{q_1}-h_2 \mod q_1?  
]

h2 has no fixed residue mod q1. Hard.

But h1 ranges over numbers congruent to (b_{q1}\mod q1) and (a_p\mod p), small. h2 congruent to (b_{q2}\mod q2). Difference t_p p. This is a bilinear congruence. Perhaps many p imply relation between b_q1,b_q2.

This is nontrivial.

## 8. Maybe complete agreement theorem is too hard

We may not need it full. For Fourier sum, if residues are adversarial but not aligned, Irving-type exponential sum over p/q should already show edge phases spread. Instead of proving agreement theorem combinatorially, use analytic large sieve over both p and q with arbitrary coefficients (a_p,b_q). We need bound:

[  
\sum_{p\in P,q\in Q}  
e\left(  
\frac{a_p q^{-1}}p+  
\frac{b_q p^{-1}}q  
\right)  
]

unless (a_p,b_q) are diagonal.

This is a bilinear Kloosterman-type sum with arbitrary coefficient phases tied to p and q. Maybe existing bilinear forms estimates for Kloosterman sums can handle this! Irving's theorem itself likely proves average bounds using bilinear forms. Perhaps our non-diagonal inverse theorem can be replaced by a bilinear exponential sum estimate.

This is another unifying route.

## 9. Analytical bilinear minor arc estimate

For a block pair (P,Q), define

[  
S(a,b)=  
\sum_{p\in P,q\in Q}  
e\left(  
\frac{a_p \bar q}{p}  
+  
\frac{b_q \bar p}{q}  
\right).  
]

If (S(a,b)) is large, phases aligned; otherwise Fourier coefficient decays. We need upper bound for product modulus factors, but log coefficient relates to cosine energy, not just exponential sum. Still large Fourier coefficient implies many phases clustered, which implies large exponential sum for some multiple frequency. So exponential sum estimates may suffice.

Can we bound (S(a,b)) uniformly for arbitrary (a_p,b_q)? That is much stronger than Irving, because coefficients depend on the modulus p/q. It resembles sums over Kloosterman sheaves with trace functions varying both moduli. Existing results may not cover arbitrary residues.

So maybe not.

## 10. Return: agreement theorem may be necessary

The arbitrary (a_p,b_q) dependence is exactly why we need inverse theorem: if phases align despite arbitrary residues, then residues must be structured. Analytic estimates with arbitrary coefficients likely false unless one accounts for structure.

Thus agreement theorem is the right combinatorial counterpart.

## 12. A possible proof via dependent random choice

If at least (1-\epsilon) edges pass, then choose two random q's. Many p's connect pass to both. For each such p, the pair of constraints may force (a_p) into a small set depending on q1,q2. If this small set is actually one residue (h\mod p), then many p align. Then use one p to align many q.

Need prove "two q constraints force p residue to one of O(δ^2 p + 1) possibilities." Since each q constraint allows δ p? Let's compute from p-side: for fixed q,b_q, pass condition allows a_p in a subset of (\mathbb F_p) of size about (2δ p). Intersect two such subsets for q1,q2 likely size about (δ^2 p), not one. Need (O(\log p)) q's to force one possibility. Since degree huge, choose t≈C log p random q's. For many p, all t edges pass. Then allowed a_p set size about (δ^t p). If t>C log p, this is <1 unless sets align. If align, global structure emerges.

This is promising.

## 13. Entropy compression proof sketch

Choose a small test set (Q_0\subset Q) of size

[  
t=C\log X  
]

randomly. Since most edges pass, many p pass to most q in (Q_0).

For each p, the set of (a_p) compatible with all q in (Q_0) has size usually at most

[  
(2δ)^t p \ll 1,  
]

unless the constraints are highly aligned.

If for many p this set has one value, then (a_p) is determined by (Q_0)-data. Since (Q_0)-data has at most (\prod_{q\in Q_0} q \approx Y^t) possibilities, total entropy of all (a_p)'s is greatly reduced. Need convert to frequency count, not exact global h.

This may be enough for Fourier sum: low-energy characters are few. We don't need identify h.

Excellent! This avoids proving explicit global h.

## 14. Low-energy counting via random tests

For each block pair, choose a test subset (Q_0) of (t\sim C\log X) vertices. Low-energy character must pass most edges, hence for many p it passes to most q∈Q0. Thus a_p lies in a small set. Count possibilities:

- choose residues b_q for q∈Q0: at most (\prod_{q∈Q0}q).
    
- for each p, allowed a_p choices small, say O(1) for most p.
    
- exceptional p's allowed arbitrary but few.
    

This gives entropy roughly

[  
t\log Y + o(|P|\log X)  
]

instead of (|P|\log X).

If t log Y ≈ (log X)^2, while |P| log X≈ X, huge compression. Very strong.

This could close high-frequency count.

This is a strong route! It uses only density of pass edges and set-intersection behavior. Need prove intersection size bound for constraints from random q's.

## 15. Intersection bound

For fixed p, each q constraint is a subset (A_{q,b_q}\subset\mathbb F_p) of size about (2δ p). Need show for arbitrary distinct q_1,...,q_t and residues b_q, intersection

[  
\bigcap_i A_{q_i,b_i}  
]

has size at most roughly ((Cδ)^t p + O(t)), or at least becomes small for t≈log p.

Are these sets intervals under multiplication by (q^{-1})? Let's derive.

Edge pass:  
[  
\left|  
\frac{a_p q^{-1}}p+  
\frac{b_q p^{-1}}q  
\right|\le\delta.  
]

For fixed p,q,b_q, solve for a_p. Multiply phase by? The term involving a_p is  
[  
a_p q^{-1}/p.  
]  
Thus condition is  
[  
\left|  
\frac{a_p q^{-1}}p + \beta  
\right|\le\delta  
]  
where (\beta=b_q p^{-1}/q) fixed. This means  
[  
a_p q^{-1}\bmod p  
]  
lies in an interval of length (2δ p). Multiplication by q permutes (\mathbb F_p), so (a_p) lies in a modular interval with step q, i.e. (q\cdot I).

For different q_i, these are arithmetic progressions/Bohr intervals with different multipliers. Intersection of many such Bohr intervals should have size about product densities if multipliers (q_i) are sufficiently dissociated mod p. Since q_i are primes in a range and t~log p, likely.

This can be controlled by Fourier:  
indicator of interval has Fourier coefficients ~1/r. Intersection size equals  
[  
p\sum_{\xi_1+...??}  
]  
Need multipliers q_i. If multipliers q_i are distinct and generic, maybe large sieve. We can choose Q0 randomly to ensure good for all p. This again uses additive combinatorics.

But this is a known object: intersection of Bohr sets. Worst case if all q_i equal; they are distinct. Could still align if q_i congruent ±q_j mod p; but q_i<p? If q_i may be comparable to p; modulo p they vary. Random enough.

This is plausible but technical.

## 16. Where are we?

We have moved from vague case split to a concrete entropy/inverse strategy:

- low Fourier energy ⇒ many pass edges;
    
- sample test vertices;
    
- pass constraints restrict residues via intersections of modular intervals;
    
- count low-energy characters.
    

This is a serious but plausible combinatorial-analytic lemma. It may be independent of prime Kloosterman; it uses modular interval intersection and random choice.

This is enough for "審慎但有信心": no contradiction; but remaining proof is nontrivial.

## 17. Next step

To continue productively, we should isolate the modular interval intersection lemma:

**Interval intersection lemma.**  
Let (p) be prime. Let (q_1,\dots,q_t) be residues mod p chosen from a large prime set and sufficiently generic. For arbitrary shifts (\beta_i), the set of (a\in\mathbb F_p) satisfying  
[  
|a q_i^{-1}/p+\beta_i|\le\delta  
]  
for all (i) has size at most  
[  
(C\delta)^t p + p^{o(1)}  
]  
or at least (\le p^{1-\eta}) for (t=C\log p).

If this lemma holds for test sets (Q_0) simultaneously for most p, the low-energy entropy compression works.

This is our next atomic technical target.
