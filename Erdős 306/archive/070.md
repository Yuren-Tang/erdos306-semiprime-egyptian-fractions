好。現在攻最小原子：

[  
P\leftrightarrow Q  
]

一張二部測試圖。先不處理整個 dyadic graph，只看兩個素數塊之間的 CRT agreement。

---

## 1. 二部 CRT agreement 模型

令

[  
P\subset[X,2X],\qquad Q\subset[Y,2Y],  
]

其中 (X,Y) 在可控範圍內，例如

[  
Y^{2/3}\le X\le Y^{3/2}.  
]

每個

[  
p\in P  
]

有 residue

[  
a_p\in\mathbb F_p,  
]

每個

[  
q\in Q  
]

有 residue

[  
b_q\in\mathbb F_q.  
]

對邊 (pq)，定義

# [  
\phi_{pq}

\frac{a_p(q^{-1}\bmod p)}p  
+  
\frac{b_q(p^{-1}\bmod q)}q.  
]

等價地，令

[  
H_{pq}^{*}\in(-pq/2,pq/2]  
]

是滿足

[  
H_{pq}^{_}\equiv a_p\pmod p,\qquad  
H_{pq}^{_}\equiv b_q\pmod q  
]

的 CRT 對稱代表。則

[  
\phi_{pq}\equiv \frac{H_{pq}^{*}}{pq}\pmod1.  
]

邊通過若

[  
|H_{pq}^{*}|\le \delta pq.  
]

我們想證一種 local-to-global 命題：

若很多邊通過，則存在一個整數 (h)，使

[  
a_p\equiv h\pmod p  
]

對多數 (p)，且

[  
b_q\equiv h\pmod q  
]

對多數 (q)。

---

## 2. exact case 給強信號

若邊 (pq) 滿足

[  
H_{pq}^{*}=0,  
]

則

[  
a_p=0,\qquad b_q=0.  
]

所以若一張連通二部圖所有邊 exact zero，所有 residues 都是 (0)。這說明 exact ground state 剛性極強。

但我們真正需要的是 approximate case。小但非零的 (H_{pq}^{*}) 允許全局小 (h)：

[  
a_p\equiv h\pmod p,\qquad b_q\equiv h\pmod q,  
]

此時

[  
H_{pq}^{*}=h  
]

只要

[  
|h|<pq/2.  
]

所以低能態應該是一維族 (h)，而非只有 (0)。

---

## 3. 一個關鍵觀察：兩條邊共享頂點

假設兩條通過邊共享 (q)：

[  
p_1q,\quad p_2q.  
]

它們有小 CRT glues：

[  
H_{p_1q}^{_},\qquad H_{p_2q}^{_}.  
]

因為二者都同餘於

[  
b_q\pmod q,  
]

所以

[  
H_{p_1q}^{_}\equiv H_{p_2q}^{_}\pmod q.  
]

若二者的絕對值都小於 (q/2)，則必有

[  
H_{p_1q}^{_}=H_{p_2q}^{_}.  
]

這是非常強的 agreement propagation。

所以若很多通過邊不但相對小

[  
|H_{pq}^{*}|\le\delta pq,  
]

而且絕對小

[  
|H_{pq}^{*}|<\frac12\min(p,q),  
]

那麼同一頂點附近的所有好邊會共享同一個 glue 值，進而全圖連通時得到一個 global (h)。

---

## 4. 問題：相對小不等於絕對小

通過條件

[  
|H_{pq}^{*}|\le\delta pq  
]

仍可能遠大於 (q)。例如 (pq) 很大，(\delta pq\gg q)。

所以僅靠「兩個小整數同餘 mod (q) 故相等」不夠。

但對低能量而言，我們不是只有固定 (\delta) 通過，而是平方和小：

[  
\sum_{pq}\left(\frac{H_{pq}^{*}}{pq}\right)^2\le R.  
]

若 (R) 很低，很多邊的相位可能遠小於固定 (\delta)。若能達到

[  
\frac{|H_{pq}^{*}|}{pq}<\frac1{2q}  
]

即

[  
|H_{pq}^{*}|<p/2,  
]

則在共享 (p) 或 (q) 時可做 exact propagation。

這提示一個能量分層：

- **極低相位邊**：可傳播精確 glue equality；
    
- **中等相位邊**：不能直接傳播，但每條付出一定能量，可用 entropy 支付；
    
- **大相位邊**：Fourier 衰減直接殺掉。
    

這是自然的，不是任意分類。

---

## 5. 二部模型中的 entropy bound

設一個 assignment 不是由全局 (h) 控制。那麼在二部圖中會有很多「不一致」：

對某個 (q)，其鄰接邊的 glues

[  
H_{pq}^{*}  
]

不能大多等於同一個小 (h)。若它們仍要保持相位小，則對 (b_q) 或各 (a_p) 施加許多模區間約束。

從 (p)-側看，固定鄰居 residues (b_q) 後，每條通過邊要求

[  
a_p  
]

落在某個短模區間。若 (p) 有 (C\log p) 個測試鄰居，則 (a_p) 的自由度大幅下降。這就是 entropy contraction。

所以二部 agreement 可以由兩種機制共同完成：

1. 絕對小 glue 直接傳播同一 (h)；
    
2. 不能傳播時，用能量／熵支付。
    

這正是 energy–entropy inequality 的內容。

---

## 6. 嘗試寫二部熵不等式

令

[  
\mathcal A_R(P,Q)  
]

是二部 assignments ((a_p,b_q)) 使

# [  
Q_{P,Q}(a,b)

\sum_{p\in P,q\in Q}  
\left|  
\frac{a_pq^{-1}}p+\frac{b_qp^{-1}}q  
\right|^2  
\le R.  
]

我們希望有

[  
\log|\mathcal A_R(P,Q)|  
\le  
\log|\mathcal D_R(P,Q)|+C R,  
]

其中 (\mathcal D_R) 是對角族：

[  
a_p\equiv h\pmod p,\qquad b_q\equiv h\pmod q  
]

且

[  
h^2\sum_{p,q}\frac1{p^2q^2}\le R.  
]

這是二部原子版。

如果 (P,Q) 是 complete bipartite 且邊數巨大，這個不等式應該很強；若只是稀疏測試圖，需要 expansion 和足夠度。

---

## 7. 在 complete bipartite 中的直觀證明

完整二部圖有很多 4-cycles。若多數邊相位小，則對許多四邊形

[  
p_1q_1,\ p_1q_2,\ p_2q_1,\ p_2q_2  
]

四個 CRT glues 都小。這種大量四環一致性應迫使它們來自同一 (h)。這與 agreement testing 中的 local consistency tests 很像。

先前我們直接寫四環代數沒有立刻得到整除關係，但在熵／平均意義下，四環非常有力：若兩側分別有不同局部 (h)，四環會大量失敗。

所以 complete bipartite 情況很可能可用 dependent random choice 或 agreement test 的四環分析證。

---

## 8. 稀疏測試圖需要 pseudo-randomness

完整二部圖太大；實際測試圖每個頂點只取 (O(\log p)) 條測試邊。這仍可足夠，若它是 expander / sampler：

對任意低熵壞集合，測試邊能看見其不一致。

這就是 agreement testing 裏測試圖的作用。它不必完整，但要有 sampler 性質。

可設計測試圖為隨機子圖：每個頂點從 compatible block 中隨機取 (C\log p) 個鄰居。隨機圖通常有足夠 expansion，且 degree 恰好在 entropy 尺度。

之後再用 prime inverse derandomization。

---

## 9. 一個更具體的二部 sampler lemma

對固定二部 blocks (P,Q)，選隨機測試邊，使每個 (p\in P) 有

[  
d_p=C\log p  
]

個鄰居，每個 (q\in Q) 也有相近入度。

希望：

若 assignment ((a,b)) 在測試邊上通過率高，則它在完整二部圖上通過率也高，至少對能量而言：

[  
Q_{P,Q}^{\mathrm{full}}(a,b)  
\lesssim  
Q_{P,Q}^{\mathrm{test}}(a,b)+\text{small}.  
]

如果測試圖是 good sampler，則可把問題還原到 complete bipartite。這或許比直接在稀疏圖上做 agreement 更容易。

但 sampling over all assignments is hard because assignments huge. Again entropy methods needed. But random graph with degree (C\log |state|) can sample all low-complexity structures.

---

## 10. 是否回到過度抽象？

有一點風險。為了保持可控，我們定下一個最小可攻版本：

**Model complete bipartite CRT agreement.**  
先證完整 (P\times Q) 的熵界。這不涉及 graph sampling，只涉及 CRT constraints.

如果 complete bipartite 都證不出，那稀疏測試圖沒戲。若 complete bipartite 可證，再研究稀疏化。

所以現在最合理下一步是 complete bipartite 版本。

---

## 11. Complete bipartite 版本可能更容易

因為對每個 (p)，它連到所有 (q\in Q)。固定 (b_Q) 後，(a_p) 需滿足大量模區間條件。即便 shifts 任意，因 (Q) 很大且 (q^{-1}\bmod p) 分散，(a_p) 可選值很少。這是強 list-decoding。

而這裏不需要 uniform over arbitrary shifts? 在 complete bipartite entropy count中，(b_Q) 可任意，但對每個 (p)，條件數多到 (|Q|)，遠大於 (\log p)，所以即便 worst-case shifts，也可能足夠。Irving 可保證 slopes (q^{-1}) 分散。

所以 complete bipartite agreement 應更可證。

代價是完整二部圖邊太多，若用於實際 Fourier，總質量也許太大；但它可作測試圖子結構的一部分，或證明概念。
