不是整條路線又換了；主線仍是：

[

\quad+\quad

\quad+\quad

\quad+\quad

]

真正更新的是 agreement/Poincaré 的證法：  
前面我一度想用稀疏測試圖，每個頂點只抽 (C\log p) 條邊。後來發現這會留下很多「無共同鄰居」問題，agreement 傳播困難。可是我們的原始候選圖其實不是稀疏圖，而是包含許多完整 dyadic block pairs：

[  
\mathcal P_k\times \mathcal P_\ell,\qquad \frac23\ell\le k\le \frac32\ell.  
]

既然完整 block pair 已在候選圖中，Fourier 係數也確實包含這些邊，那 agreement 證明就應直接利用它們的稠密性。這是更自然的修正，不是偏離原路。

---

## 1. 稠密 block agreement lemma

現在要證一個具體命題。

令

[  
P\subset[X,2X],\qquad Q\subset[Y,2Y],  
]

其中

[  
Y\gg B,  
]

而 (B) 是主弧 cutoff，例如

[  
B\asymp C/\sigma.  
]

對每個 (p\in P)、(q\in Q)，有 residue

[  
a_p\in\mathbb F_p,\qquad b_q\in\mathbb F_q.  
]

令

[  
H_{pq}^{*}\in(-pq/2,pq/2]  
]

為 CRT glue：

[  
H_{pq}^{_}\equiv a_p\pmod p,\qquad  
H_{pq}^{_}\equiv b_q\pmod q.  
]

稱邊 (pq) 為 strong-good，若

[  
|H_{pq}^{*}|\le B.  
]

我們想證：

> 若 (P\times Q) 中 strong-good 邊佔絕大多數，則存在一個整數 (h)，(|h|\le B)，使得大多數 (p\in P) 與 (q\in Q) 的 residue 都等於 (h) 模各自素數。

這就是稠密二部 agreement。

---

## 2. 第一步：每個好高點有一個 label

固定 (q\in Q)。若 (pq) 是 strong-good，則

[  
H_{pq}^{_}\equiv b_q\pmod q,  
\qquad |H_{pq}^{_}|\le B.  
]

因為

[  
q>B  
]

所以若 (p_1q) 與 (p_2q) 都 strong-good，則

[  
H_{p_1q}^{_}\equiv H_{p_2q}^{_}\pmod q  
]

且二者都在 ([-B,B]) 中。故

[  
H_{p_1q}^{_}=H_{p_2q}^{_}.  
]

因此：

[  
\boxed{  
\text{每個 incident strong-good 邊不少的高層頂點 }q  
\text{ 有唯一 label }h_q.  
}  
]

也就是說，對所有 strong-good 鄰居 (p),

[  
H_{pq}^{*}=h_q.  
]

於是

[  
a_p\equiv h_q\pmod p,  
\qquad  
b_q\equiv h_q\pmod q.  
]

---

## 3. 第二步：兩個高點共享足夠多低鄰居，則 label 相同

取 (q_1,q_2\in Q)。若存在低層素數 (p\in P)，使

[  
pq_1,\quad pq_2  
]

都 strong-good，則

[  
a_p\equiv h_{q_1}\pmod p,  
\qquad  
a_p\equiv h_{q_2}\pmod p.  
]

所以

[  
h_{q_1}\equiv h_{q_2}\pmod p.  
]

若它們有兩個不同共同 strong-good 鄰居 (p_1,p_2)，則

[  
p_1p_2\mid h_{q_1}-h_{q_2}.  
]

而

[  
|h_{q_1}-h_{q_2}|\le 2B.  
]

若

[  
p_1p_2>2B,  
]

則必有

[  
h_{q_1}=h_{q_2}.  
]

在我們的尺度裏，低層

[  
p\asymp X  
]

而

[  
B\asymp k_0X_0.  
]

若 (X) 至少是最低尺度附近，則對大 (X),

[  
X^2\gg B.  
]

所以兩個共同 good 低鄰居就足以強迫 equality。

這是非常強的傳播機制。

---

## 4. 第三步：稠密 good edges 保證共同鄰居

假設 strong-good 邊集合 (G\subset P\times Q) 密度為

[  
1-\varepsilon.  
]

稱 (q\in Q) 為 good 高點，若它的 good degree 至少

[  
(1-\sqrt\varepsilon)|P|.  
]

由 Markov，除了至多

[  
\sqrt\varepsilon |Q|  
]

個 (q) 之外，都是 good 高點。

若 (q_1,q_2) 都是 good 高點，則它們的 good 鄰居集合在 (P) 中大小各至少

[  
(1-\sqrt\varepsilon)|P|.  
]

交集大小至少

[  
(1-2\sqrt\varepsilon)|P|.  
]

只要

[  
|P|  
]

大且 (\varepsilon) 小，交集中有許多 (p)，特別至少兩個不同素數。因此

[  
h_{q_1}=h_{q_2}.  
]

所以所有 good 高點共享同一 label：

[  
h_q=h.  
]

---

## 5. 第四步：低點也跟隨同一 (h)

對任意低點 (p\in P)，若它有一條 strong-good 邊連到 good 高點 (q)，則

[  
a_p\equiv h_q=h\pmod p.  
]

由於 good 高點佔絕大多數，且 strong-good 邊總密度高，除了少數低點之外，每個 (p) 都會連到某個 good 高點的 strong-good 邊。

更定量地，若某個 (p) 不連到任何 good 高點 strong-good 邊，則它對絕大多數 (q) 都是 bad，這會貢獻大量 bad edges。由總 bad edge 數 (\varepsilon |P||Q|) 可知這類 (p) 至多 (O(\sqrt\varepsilon)|P|) 或 (O(\varepsilon)|P|) 級別。

因此大多數低點滿足

[  
a_p\equiv h\pmod p,  
]

大多數高點滿足

[  
b_q\equiv h\pmod q.  
]

---

## 6. 稠密 block agreement lemma 成立

我們已經證出如下命題：

**Dense block CRT agreement lemma.**  
設 (P,Q) 為兩組素數，且 (q>B) 對所有 (q\in Q)，並且任意兩個不同 (p_1,p_2\in P) 滿足

[  
p_1p_2>2B.  
]

若 strong-good 邊

[  
|H_{pq}^{*}|\le B  
]

在 (P\times Q) 中密度至少

[  
1-\varepsilon,  
]

則存在整數

[  
|h|\le B  
]

使得除了 (O(\sqrt\varepsilon)) 比例外，

[  
a_p\equiv h\pmod p  
]

對 (p\in P) 成立，且

[  
b_q\equiv h\pmod q  
]

對 (q\in Q) 成立。

這個 lemma 很乾淨，而且不需要深素數分佈；只用 CRT 與稠密二部圖。

這是重大進展。

## 7. 它如何接到能量？

現在要從能量小推出 strong-good 邊密度高。

若 strong-good cutoff 是

[  
B,  
]

則 bad edge 滿足

[  
|H_{pq}^{*}|>B,  
]

所以每條 bad edge 的能量至少

[  
\left(\frac{B}{pq}\right)^2.  
]

在 block pair (P\times Q) 中

[  
pq\asymp XY.  
]

因此若 block energy

# [  
Q_{P,Q}(a)

\sum_{p\in P,q\in Q}  
\left(\frac{H_{pq}^{*}}{pq}\right)^2  
]

不超過 (R_{P,Q})，則 bad edge 數至多

[  
\ll R_{P,Q}\frac{X^2Y^2}{B^2}.  
]

除以總邊數

[  
|P||Q|  
\asymp  
\frac{XY}{\log X\log Y},  
]

bad edge 密度為

[  
\varepsilon  
\ll  
R_{P,Q}  
\frac{XY\log X\log Y}{B^2}.  
]

所以只要

[  
R_{P,Q}  
\ll  
\frac{B^2}{XY\log X\log Y},  
]

該 block pair 的 strong-good 密度高，agreement lemma 可用。

這處理了很低能的 block pair。

## 8. 若某些 block pair energy 不夠低？

那它本身已支付能量。這正是 energy–entropy 分層：

- 低能 block pair：agreement lemma 強迫同一 (h)；
    
- 高能 block pair：其能量可用於 Fourier 衰減。
    

所以不需每個 block pair 都低能；只需證如果整體 Fourier 係數大，則足夠多關鍵 block pairs 低能，從而 agreement 傳播。

若某 assignment 在許多 block pairs 上能量高，則 (Q(a)) 大，直接被 (e^{-cQ(a)}) 殺掉。

## 9. 跨 block 傳播

對每個尺度 (k)，取上層

[  
\ell=\lceil 3k/2\rceil.  
]

若 (P_k\times P_\ell) 是低能，則由 dense block agreement lemma 得到一個 label

[  
h_k  
]

解釋大多數 (P_k) 和 (P_\ell)。

相鄰尺度 pair 重疊：(P_\ell) 同時出現在另一個 agreement relation 中。若兩個 labels 在同一 block (P_\ell) 上都解釋大多數頂點，則它們必須相同。因為若

[  
h\not=h',  
]

但對很多 (p\in P_\ell),

[  
h\equiv h'\pmod p,  
]

則 (\prod p\mid h-h'), impossible once product exceeds (2B).

因此 labels 沿尺度圖傳播，最終得到全局 (h)。

這就把 dyadic graph agreement 接起來。

## 11. 剩下的新問題

現在最主要問題變成：

如何從一般低能 (Q(a)\le R) 推出足夠多 block pairs 低能到可用 dense agreement lemma？

這是能量分配問題。可能處理如下：

- 若某些核心 block pairs energy 高，則 (Q(a)) 高，Fourier 衰減已夠；
    
- 若核心 block pairs energy 低，agreement lemma 迫使 labels 一致；
    
- 能量層分解即可得到
    

[  
\log N(R)\le \log N_{\mathrm{diag}}(R)+CR.  
]

這比之前的 entropy route更具體。

下一步應把 block-pair energy threshold 與 Fourier 衰減常數精算，看能否真的得到 (CR) 熵界。
