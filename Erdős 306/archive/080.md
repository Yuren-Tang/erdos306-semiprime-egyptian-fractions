好。現在處理缺口 A：**如何從 Irving 型 prime Kloosterman 估計推出 vertex correction lemma**。

我們需要的 lemma 是：

> 若一個 block (P) 已由 label (h) 控制，而某個相鄰頂點 (q) 選了錯 residue (a_q\ne h\pmod q)，則 (q) 與 (P) 的 incident edges 產生大量 Fourier 能量。

形式上，令

[  
P=P_k\subset[X,2X],  
\qquad q\sim Q,  
]

且

[  
Q^{2/3}\le X\le Q^{3/2}.  
]

假設對大多數 (p\in P),

[  
a_p\equiv h\pmod p.  
]

若

[  
d=a_q-h\not\equiv0\pmod q,  
]

則對這些 (p),

[  
\phi_{pq}(a)  
\equiv  
\frac{h}{pq}  
+  
\frac{d,p^{-1}}q  
\pmod1.  
]

因為 (h/(pq)) 很小，所以只要

[  
\frac{d,p^{-1}}q  
]

不集中在 (0) 附近，就會得到大量能量。

## 1. 我們真正需要的分散命題

對每個 good (q)，對所有

[  
d\in\mathbb F_q^\times,  
]

希望有：

[  
#\left{  
p\in P:  
\left|  
\frac{d,p^{-1}}q  
\right|<\delta  
\right}  
\le  
(1-c)|P|  
]

某些固定 (c,\delta>0)。

等價地，至少 (c|P|) 個 (p) 滿足

[  
\left|  
\frac{d,p^{-1}}q  
\right|\ge\delta.  
]

那麼

[  
\sum_{p\in P}  
\left|  
\frac{d,p^{-1}}q  
\right|^2  
\gg |P|.  
]

這就是 vertex correction 所需的全部。

注意這比精細 equidistribution 弱很多；只要 inverse residues 不把大多數質量塞入一個短弧即可。

## 2. 如何由指數和界推出短弧不集中？

令

# [  
A_q(d;\alpha)

#\left{  
p\in P:  
\left|  
\frac{d,p^{-1}}q  
\right|<\alpha  
\right}.  
]

若 (A_q(d;\alpha)) 很大，即逆元集中於長度 (2\alpha q) 的短弧，那麼集合

[  
d p^{-1}\pmod q  
]

對某個 interval (I\subset\mathbb F_q) 有過大命中。

用 Erdős–Turán 或 Fourier 展開 interval 指示函數：

# [  
1_I(x)

\frac{|I|}{q}  
+  
\sum_{1\le |r|\le R} c_r e_q(rx)  
+  
\text{error}.  
]

於是

# [  
A_q(d;\alpha)

\frac{|I|}{q}|P|  
+  
\sum_{1\le |r|\le R} c_r  
\sum_{p\in P} e_q(r d p^{-1})  
+  
\text{error}.  
]

若所有非零 (a) 都有

[  
\left|  
\sum_{p\in P} e_q(a p^{-1})  
\right|  
\le \epsilon |P|,  
]

則

[  
A_q(d;\alpha)  
\le  
(2\alpha+O(\epsilon+\text{ET error}))|P|.  
]

選 (\alpha) 小、(\epsilon) 小，即得到固定比例分散。

所以 Irving 的 exponential sum bound 可以推出我們要的短弧不集中。

## 3. Irving 給的是平均，如何得到 good moduli？

Irving 類定理通常形式近似：

[  
\sum_{q\sim Q}  
\max_{(a,q)=1}  
\left|  
\sum_{p\le X} e_q(a p^{-1})  
\right|  
\ll  
\text{saving}\cdot  
\pi(X)\cdot Q.  
]

或者對 dyadic (p\in[X,2X]) 也可由差分處理。

若右側是

[  
\ll \epsilon |P|, |Q|  
]

級別，則由 Markov 可知，除了 (O(\sqrt\epsilon |Q|)) 或 (O(\epsilon |Q|)) 個 bad (q) 外，對所有非零 (a),

[  
\left|  
\sum_{p\in P}e_q(a p^{-1})  
\right|  
\le  
\epsilon' |P|.  
]

這些 (q) 就是 good moduli。

我們可在構造 (Q)-block 時刪去 bad moduli；只要 bad 比例小，質量和密度不受本質影響。

## 8. Good block construction from Irving

For each pair of dyadic levels (k,\ell=\gamma k), Irving gives average over (q\in P_\ell):

[  
\sum_{q\in P_\ell}  
\max_{a\ne0}  
\left|  
\sum_{p\in P_k}e_q(a p^{-1})  
\right|  
\le  
\epsilon_k |P_k||P_\ell|,  
]

with (\epsilon_k\to0) as (k\to\infty).

Then define (q) good if

[  
\max_{a\ne0}  
\left|  
\sum_{p\in P_k}e_q(a p^{-1})  
\right|  
\le  
\epsilon_k^{1/2}|P_k|.  
]

Bad (q) proportion (\le\epsilon_k^{1/2}).

Remove bad q's. The remaining high block still has density (1-o(1)), and all vertex correction estimates hold.

Do this for finitely many scale pairs (k_0\le k\le K). Since (K) finite for fixed (b), choose (k_0) large enough so all errors tiny.

Thus no serious obstacle.

## 9. Does removing bad primes harm dense agreement?

No. Dense agreement uses complete pairs (P_k\times P_{\gamma k}^{good}). If good subset has density (1-o(1)), prime counts and reciprocal mass change negligibly.

We may need also good condition in reverse direction for correcting low vertices from high labelled blocks. But if a low vertex deviates from h, we can use a labelled neighboring block on the other side where it appears as q or p. We can select good sets symmetrically for all required pairs.

Alternatively, keep vertex correction always from lower labelled block to higher vertex; downward correction follows by considering previous scale. Each block has both lower and upper neighbors except endpoints. Endpoints can be handled by extending scale range or adding boundary blocks.

---

## 10. Boundary scales

At lowest scale (k_0), there is no lower block to correct it, but it has upper block (k_1=\gamma k_0). Dense agreement on (P_{k_0}\times P_{k_1}) labels both; vertex correction from one side controls the other.

At highest scale (K), it may lack upper block. We can extend construction one extra layer beyond (K) for agreement/backbone but not necessarily use its mass for representing? Simpler: include extra buffer layers for structure. Their reciprocal mass can be included; choose K with buffer in mind. No obstacle.

## 11. Vertex correction lemma now plausible

We can state:

For every fixed (\gamma\in(1,3/2)), for all sufficiently large (k), after discarding (o(|P_{\gamma k}|)) bad primes, every (q\in P_{\gamma k}) satisfies for all (d\ne0):

[  
#\left{p\in P_k:  
\left|\frac{d p^{-1}}q\right|<\delta  
\right}  
\le  
(1-c)|P_k|.

This follows from Irving + Erdős–Turán.

That is exactly needed.
