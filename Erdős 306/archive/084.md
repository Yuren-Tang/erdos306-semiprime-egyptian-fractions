## 1. 我們要核對的不是「Irving 是否對」，而是四個適配點

我們需要的外部輸入是：

[  
\sum_{q\sim Q}  
\max_{a\ne0}  
\left|  
\sum_{p\in[X,2X]\atop p\ \mathrm{prime}}  
e_q(a p^{-1})  
\right|  
\le  
\varepsilon(X,Q),|P|,|Q|,  
]

其中

[  
\varepsilon(X,Q)\to0  
]

並且範圍覆蓋我們選的

[  
Q=X^{4/3}.  
]

所以要查四件事：

1. **是否有 (\max_a)**  
    我們需要 uniform in (d=a_q-h)。若 Irving 只有 fixed (a)，不夠；若有  
    [  
    \max_{(a,q)=1}  
    ]  
    就正好。
    
2. **是否是平均 over (q)**  
    我們不需要每個 (q) 都好；平均 over (q) 加 Markov 就可刪掉少數壞 (q)。
    
3. **尺度範圍是否覆蓋 (X=Q^{3/4})**  
    我們取 (Q=X^{4/3})，即 (X=Q^{3/4})。這落在  
    [  
    Q^{2/3}\le X\le Q^{3/2}  
    ]  
    的內部，不是邊界。
    
4. **是否可 dyadic 化**  
    若定理是對  
    [  
    p\le X  
    ]  
    的和，我們要用差分變成  
    [  
    X\le p<2X.  
    ]  
    這通常容易，但要確認定理範圍在 (X) 與 (2X) 都有效。
    

---

## 2. 一旦這四點成立，Lemma D 的推出很標準

假設 Irving 給出：

[  
\max_{a\ne0}  
\left|  
\sum_{p\in P}e_q(a p^{-1})  
\right|  
\le \varepsilon |P|  
]

對 good (q) 成立。

那麼用 Erdős–Turán 或平滑 Fourier 展開可推出：對任意短弧 (I\subset\mathbb F_q),

# [  
#{p\in P: a p^{-1}\in I}

\frac{|I|}{q}|P|  
+  
O(\varepsilon |P|)+\text{truncation error}.  
]

取 (I) 為 (0) 附近的小弧，就得到：

[  
#\left{  
p\in P:  
\left|\frac{d p^{-1}}q\right|<\delta  
\right}  
\le (1-c)|P|.  
]

因此

[  
\sum_{p\in P}  
\left|  
\frac{d p^{-1}}q  
\right|^2  
\gg |P|.  
]

這正是 vertex correction。
