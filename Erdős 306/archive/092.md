好。現在補 **Mass / lattice setup lemma**。這一塊看似初等，但它負責三件事：

[  
\text{候選邊總質量可調}  
]

[  
\text{整數化目標合法}  
]

[  
\text{格點 span 為 }1  
]

若這裏有漏洞，後面的 Fourier 主項即使成立，也可能命中錯格點。

---

# Lemma 1：Compatible dyadic graph 的質量發散

令

[  
P_k={p:\ 2^k\le p<2^{k+1},\ p\ \text{prime}}.  
]

由 Mertens / PNT in dyadic form，

[  
u_k:=\sum_{p\in P_k}\frac1p  
\asymp \frac1k.  
]

Full compatible graph 取所有 block pairs

[  
P_k\times P_\ell  
]

滿足

[  
\frac23\ell\le k\le \frac32\ell.  
]

則 reciprocal mass

# [  
W(k_0,K)

\sum_{\substack{k_0\le k,\ell\le K\  
\frac23\ell\le k\le \frac32\ell}}  
u_ku_\ell  
]

滿足

[  
W(k_0,K)\asymp  
\sum_{k=k_0}^{K}\frac1k.  
]

原因是：固定 (k)，可相連的 (\ell) 有 (\asymp k) 個，且 (\ell\asymp k)，所以

[  
\sum_{\ell\asymp k}\frac1{k\ell}  
\asymp  
\frac1k.  
]

故

[  
W(k_0,K)\asymp \log\frac K{k_0}.  
]

因此，對任意固定 (b)，取 (K) 充分大，可令

[  
W(k_0,K)> \frac1b.  
]

更好地，可以取 (K) 使

[  
\frac2b\le W(k_0,K)\le \frac3b  
]

之類，因為每新增一層的質量約 (1/K)，可在 (K) 大時細調。

然後取

[  
\theta=\frac{1/b}{W(k_0,K)}.  
]

若 (W\in[2/b,3/b])，則

[  
\theta\in[1/3,1/2].  
]

所以 Bernoulli 參數遠離 (0,1)，這對 Fourier 衰減與主弧展開很重要。

# Lemma 2：Pruning good primes 不破壞質量

Irving vertex correction 要刪去每個 relevant block 中的 (o(1)) 比例 bad primes。令保留後的 block 為

[  
P_k^\ast\subset P_k,  
\qquad  
|P_k^\ast|=(1-o(1))|P_k|.  
]

同樣有

# [  
u_k^\ast:=\sum_{p\in P_k^\ast}\frac1p

(1-o(1))u_k  
]

只要刪去的是數量比例 (o(1)) 的 primes；因同一 dyadic block 中 (1/p\asymp2^{-k})，數量比例與 reciprocal mass 比例一致。

所以 pruning 後，

# [  
W^\ast(k_0,K)

(1+o(1))W(k_0,K).  
]

因此仍可調到

[  
W^\ast\asymp 1/b  
]

並取

[  
\theta=\frac{1/b}{W^\ast}\in[c,1-c].  
]

這也說明 Irving-bad primes 不會破壞主項質量。

# Lemma 3：整數化

我們要把

[  
S=\sum_{pq\in E}X_{pq}\frac1{pq}  
]

變成整數格點問題。

令

[  
P^\ast=\bigcup_k P_k^\ast  
]

是所有使用的 primes。因 (b) squarefree，將 (b) 的所有素因子加入一個固定集合 (B_0)，並取

[  
L=\prod_{r\in P^\ast\cup B_0} r.  
]

注意：(B_0) 中 primes 不必都作為候選邊頂點；它們只需進入 (L)，使

[  
b\mid L.  
]

對每條候選邊 (pq)，因 (p,q\in P^\ast) 且 (p\ne q)，有

[  
\frac{L}{pq}\in\mathbb Z.  
]

因此

[  
S=\frac1b  
]

等價於

# [  
\sum_{pq\in E}X_{pq}\frac{L}{pq}

\frac Lb,  
]

右邊為整數。

所以目標是一個 lattice Bernoulli sum 命中整數點。

---

# Lemma 4：span 為 1

需要證

[  
\gcd\left{\frac{L}{pq}: pq\in E\right}=1.  
]

否則即使均值正好，目標也可能不在可達格點上。

對任何 prime (r\mid L)，要證存在一條邊 (pq\in E) 使

[  
r\nmid \frac{L}{pq}.  
]

如果 (r\in P^\ast)，則因圖中每個 retained vertex 都有至少一條 incident edge (rs)，取 (pq=rs)，則

[  
\frac{L}{rs}  
]

不含 factor (r)，故不被 (r) 整除。

如果 (r\in B_0\setminus P^\ast)，則 (r) 不出現在任何 denominator (pq)，所以每個 (L/(pq)) 都含 (r)。這會讓 gcd 含 (r)，造成 span 問題。

因此不能只把 (b) 的素因子放入 (L) 而完全不用。這是個重要細節。

解法：把 (b) 的每個素因子也放入 vertex set，並確保它出現在至少一條候選邊中。可是這些 primes 是固定小 primes，可能不在 large dyadic blocks，也可能破壞 estimates。

更好的做法是單獨處理 (b) 的素因子：對每個 (r\mid b)，加入一條或少數「lattice-fixing」邊 (rs_r)，其中 (s_r) 是極大新素數，且所有 (s_r) 不與其他使用衝突。這些邊的 reciprocal mass 極小，可忽略；但它們使 (r) 不再整除所有 weights。

具體：

- 將 (r\mid b) 納入 (L);
    
- 選一個 large prime (s_r\in P^\ast) 或額外 block；
    
- 將 edge (r s_r) 加入候選邊集合；
    
- 則 weight (L/(rs_r)) 不被 (r) 整除。
    

這樣每個 (r\mid b) 也被「釋放」，gcd 變成 1。

這些 finitely many lattice-fixing edges 對總質量、方差、Fourier estimates 都是 negligible；但對 span 是必要的。

另一種做法是一開始把 (b) 的 prime factors 作為一個有限底層 block 並連到大素數，效果相同。

所以 span 1 可保證。

# Lemma 5：期望精確調整

若只用 Bernoulli 參數

[  
\theta=\frac{1/b}{W}  
]

則

[  
\mathbb E S=1/b  
]

精確成立。加入 finitely many lattice-fixing edges 後，它們也有 reciprocal mass。可以把它們納入 (W)，或令它們以極小概率選取。但 Bernoulli probability 要統一嗎？

不必統一。可以允許邊依賴不同 Bernoulli 參數

[  
\theta_e\in[c,1-c]  
]

或其中少數特殊邊固定處理。

最簡單：

- lattice-fixing edges 也放入候選集；
    
- 它們總質量 (\omega) 極小；
    
- 對 full dyadic main edges 取 (\theta) 使  
    [  
    \theta W_{\mathrm{main}}+\theta_0\omega=1/b  
    ]  
    其中 (\theta_0=1/2) 固定；
    
- 因 (\omega) 可任意小，仍有 (\theta\in[c,1-c]).
    

因此期望可精確調到 (1/b).

若為了簡潔，也可先忽略 lattice-fixing edges 證明命中某個 congruence class，再用有限修正補 span；但成文時直接加入它們更乾淨。

# Lemma 6：方差與最大權重不受有限修正影響

主圖方差

[  
\sigma^2\asymp\frac1{k_0^2 2^{2k_0}}.  
]

lattice-fixing edges 可選 (s_r) 極大，使

[  
\frac1{rs_r}  
\ll \sigma  
]

甚至其平方總和

[  
\sum_{r\mid b}\frac1{r^2s_r^2}  
=o(\sigma^2).  
]

因此它們不改變主弧 Gaussian 方差，也不破壞

[  
a_{\max}=o(\sigma).  
]

若 (r) 小，edge (rs_r) 的 denominator 只有 (r s_r)，要使 weight (1/(rs_r)\ll\sigma)，取

[  
s_r\gg \frac1{r\sigma}  
\asymp \frac{k_0 2^{k_0}}r.  
]

這可做到，且仍可使 (s_r) 不與其他 primes 重合。

# 小結：地基修正

這裏發現了一個重要但可修的細節：

[  
\boxed{  
\text{若 }b\text{ 的素因子只進 }L\text{ 而不出現在任何 edge，span 會壞。}  
}  
]

修法是加入 finitely many lattice-fixing semiprime edges

[  
r s_r,\qquad r\mid b.  
]

它們使 gcd 為 1，但因 (s_r) 可取極大，對質量、方差、主弧、minor arc 幾乎無影響。

這是好事：我們及時抓到了一個真正的格點問題，且修法簡單。
