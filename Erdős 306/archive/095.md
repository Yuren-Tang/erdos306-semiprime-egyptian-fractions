# 1. 分三類邊集

我們把 full graph 的邊分成三個用途，不一定物理上完全 disjoint，但在證明中選取 disjoint subfamilies：

[  
E_{\mathrm{full}}  
\supset  
E_{\mathrm{bb}}  
\sqcup  
E_{\mathrm{corr}}  
\sqcup  
E_{\mathrm{main}}  
]

概念如下：

1. **Backbone edges (E_{\mathrm{bb}})**  
    用於判斷每條 scale link 是否 good/bad，並給 bad cuts 付款。
    
2. **Correction edges (E_{\mathrm{corr}})**  
    用於懲罰 wrong vertices。對每個 wrong vertex (q)，選一個相鄰 labelled block (P)，用 (P\times{q}) 中的一批邊來付款。
    
3. **Main/diagonal edges (E_{\mathrm{main}})**  
    用於 ordinary global label (h) 的 Gaussian energy。其實 full graph 中所有邊都可用；為避免分帳混亂，可以預留一部分 compatible block pairs 專門給主弧方差和 diagonal tail。
    

因為 full compatible graph 很稠密，每個尺度有 (O(k)) 個 compatible neighbouring scales，我們可以在尺度層面把它分成幾個 disjoint subgraphs：

- 一條 backbone chain (k_j\to k_{j+1})；
    
- 一條或數條 correction chain；
    
- 剩下的大量 block pairs 給 mass/main term。
    

這樣 bookkeeping 更安全。

# 2. Backbone cut energy

令

[  
k_j=\lfloor \gamma^j k_0\rfloor,\qquad \gamma=4/3.  
]

Backbone link

[  
\mathcal E_j=P_{k_j}^{\ast}\times P_{k_{j+1}}^{\ast}.  
]

Cutoff

[  
B_j=X_j^{5/4}.  
]

若 (\mathcal E_j) 的 bad edge density (\ge\varepsilon_0)，則

[  
j\in\mathcal B(a).  
]

每條 bad edge gives

[  
|\phi_{pq}(a)|^2  
\ge  
\left(\frac{B_j}{pq}\right)^2.  
]

在此 block 中

[  
pq\asymp X_j^{1+\gamma}=X_j^{7/3}.  
]

邊數

[  
|\mathcal E_j|  
\asymp  
\frac{X_j}{k_j}\cdot \frac{X_j^\gamma}{k_{j+1}}  
\asymp  
\frac{X_j^{1+\gamma}}{k_j^2}.  
]

所以 bad link energy 至少

# [  
\varepsilon_0  
\frac{X_j^{1+\gamma}}{k_j^2}  
\cdot  
\frac{B_j^2}{X_j^{2+2\gamma}}

c  
\frac{X_j^{2(1+\eta)-1-\gamma}}{k_j^2}.  
]

這裏 (\eta=1/4)，所以

# [  
2(1+\eta)-1-\gamma

# 2\cdot\frac54-1-\frac43

# \frac52-\frac73

\frac16.  
]

故

[  
T_j\asymp \frac{X_j^{1/6}}{k_j^2}.  
]

因此

[  
Q_{\mathrm{bb}}(a)  
\ge  
c\sum_{j\in\mathcal B(a)}T_j.  
]

這一部分乾淨。

# 3. Good components and labels

刪去 bad links 後，scale chain 分成若干 intervals：

[  
I_1,I_2,\dots,I_r.  
]

在每個 interval component (I) 中，所有 links good。Dense block agreement gives labels on each link; label merging gives a single label

[  
h_I  
]

for the whole component.

For a level (k_j\in I), most vertices in (P_{k_j}) satisfy

[  
a_p\equiv h_I\pmod p.  
]

Vertices failing this are wrong vertices, except possibly near component boundaries; boundary levels can be charged to neighbouring bad cuts.

# 4. Correction edges and wrong vertex energy

Take an interior level (k_j) in a component (I). Suppose

[  
q\in P_{k_j}^{\ast}  
]

is wrong:

[  
a_q\not\equiv h_I\pmod q.  
]

Choose a correction direction. For definiteness, use the previous level in the component:

[  
P_{k_{j-1}}^{\ast}  
]

if (j-1\in I). If (q) is at the left endpoint, use the next level, or charge to boundary cut. To keep proof simple, discard one boundary level per component and charge its entropy to adjacent bad cut.

For interior wrong (q), the block (P_{k_{j-1}}) is labelled by (h_I), and (q) is good relative to it by Irving pruning. Therefore Lemma D gives

[  
\sum_{p\in P_{k_{j-1}}^\ast}  
|\phi_{pq}(a)|^2  
\ge  
c |P_{k_{j-1}}|.  
]

Let

[  
E_q:=c |P_{k_{j-1}}|.  
]

Then

[  
E_q\gg \frac{X_{j-1}}{k_{j-1}}.  
]

Since

[  
\log q\asymp k_j\asymp k_{j-1},  
]

we have

[  
E_q\gg_A A\log q  
]

for any fixed (A), provided (k_0) is large.

---

# 5. No overlap for correction edges

For fixed level (j), different wrong vertices (q\in P_{k_j}) use edge sets

[  
P_{k_{j-1}}\times{q}.  
]

These are disjoint as (q) varies.

For different (j), these correction edge sets lie in different block pairs

[  
P_{k_{j-1}}\times P_{k_j}.  
]

Hence they are also disjoint along the chain.

Thus

[  
Q_{\mathrm{corr}}(a)  
\ge  
c\sum_{q\in W_{\mathrm{int}}(a)}E_q  
]

without overlap loss.

Boundary wrong vertices are not included in (W_{\mathrm{int}}); their entropy is charged to cuts.

This is a clean solution.

# 6. Boundary levels

Each bad cut creates at most two neighbouring boundary levels not corrected by an interior lower block. How much entropy can these boundary levels contain?

A whole level (P_k) has raw residue entropy

[  
\sum_{p\in P_k}\log p  
\asymp  
|P_k|k  
\asymp X_k.  
]

This is enormous, far larger than (T_k\asymp X_k^{1/6}/k^2). So we cannot charge an entire boundary level to a cut.

Therefore boundary treatment must be sharper: vertices at boundary cannot be entirely free. They still belong to a good link on the inside of the component, unless the component has length one.

If a level is endpoint of a component but has one good incident link, dense agreement on that link already labels most vertices, and wrong vertices can be corrected using the other side of that same good link. For example:

- If component begins at level (j), link (j\to j+1) is good.
    
- A wrong vertex in (P_j) can be corrected using labelled (P_{j+1}), but Irving lemma was stated for lower-to-higher direction.
    

Do we have Irving correction in both directions? We need it.

---

# 7. Symmetric vertex correction

If (q) is lower and labelled block is higher, phase discrepancy involves

[  
d, \bar q \pmod p  
]

as (q) varies? Let's compute.

Suppose higher block (Q) labelled by (h), and lower vertex (p) is wrong:

[  
d=a_p-h\not\equiv0\pmod p.  
]

For edge (pq),

[  
\phi_{pq}  
\equiv  
\frac{a_p\bar q}{p}  
+  
\frac{h\bar p}{q}.  
]

Diagonal h phase is (h/(pq)). Difference is

[  
\frac{d\bar q}{p}.  
]

As (q) varies in higher prime block (Q), we need distribution of

[  
\bar q\bmod p  
]

over primes (q\sim Y), modulo (p\sim X).

This is also covered by Irving, with modulus (p) and prime variable (q). Need scale condition:

If (Y=X^\gamma), and modulus is (p\sim X), variable primes (q\sim Y). Irving requires

[  
X^{2/3}\le Y\le X^{3/2}  
]

where (Q) in Irving notation is modulus size (X) and (x) is variable size (Y). Since (\gamma=4/3), yes:

[  
X^{2/3}\le X^{4/3}\le X^{3/2}.  
]

So symmetric correction holds too.

Good. Therefore any endpoint wrong vertex can be corrected using its one good incident block, in either direction.

Thus no boundary entropy leak.

# 8. Component of length zero / isolated level

If a level is isolated because both adjacent links are bad, it has no good incident link. Then its entire label/free entropy cannot be paid by one cut (T_k).

But if both adjacent links are bad, there are two cut costs:

[  
T_{j-1}+T_j.  
]

Still (T_j\asymp X_j^{1/6}) far smaller than raw level entropy (X_j). Cannot pay whole level.

So isolated levels are dangerous.

But note: if a level is isolated, it is separated from backbone labels, but full graph has many other compatible edges not in the chain. We can avoid this by choosing multiple interleaved backbone chains so no level can be isolated cheaply. Or choose backbone not as sparse chain but dense scale graph. This is a real issue.

We need fix.

# 9. Fix: do not use only one chain

The one-chain simplification fails because cutting two links can isolate an entire large block at cost only (T_k\ll X_k), giving huge entropy.

Earlier full dyadic graph has many compatible scale links. We need a scale graph with high vertex expansion so isolating a whole level requires many bad block pairs, total cost comparable to level entropy.

This is important.

So the scale-level graph must not be a simple path. It should use many compatible links per level. Good: full compatible graph already has (\asymp k) neighbouring levels per (k). If isolating (P_k) requires making many block pairs bad, total cost can be much larger.

Let's estimate.

For level (k), each compatible upward/downward link to (\ell\asymp k) has bad cost roughly

[  
T_{k,\ell}  
]

depending on cutoff. If we choose many links, total boundary cost for isolating level is about

[  
\sum_{\ell\asymp k} T_{k,\ell}.  
]

There are (\asymp k) such links. Each may be at least (X_k^{\rho}/k^2), so total

# [  
\asymp k\cdot X_k^\rho/k^2

X_k^\rho/k.  
]

Still much smaller than level entropy (X_k) if (\rho=1/6). Not enough.

But maybe choose cutoff (B) larger to increase (T). Max allowed (B<X^\gamma) and (B<X^2). For (\gamma=4/3), (B<X^{4/3}). Then exponent for (T):

[  
1-\gamma+2\eta  
]

with (\eta<\gamma-1=1/3), max exponent just below

[  
1-\frac43+\frac23=\frac13.  
]

Total over k links gives (X^{1/3}/k), still far below (X).

Thus bad cut energy cannot pay raw isolation entropy. We need vertex correction from full graph to prevent isolated levels from being arbitrary. Even if backbone links bad, full compatible graph connects level to many other labelled components; wrong residues pay incident energy unless the level forms its own component label. If it forms own label, entropy is only label entropy (O(k)), not raw (X), provided dense agreement within its links to something. If all links bad, maybe no label.

But to make all links from a level bad, full graph energy over all compatible pairs might be much larger. Let's compute with full graph bad costs; maybe enough? Number of compatible block pairs (\asymp k), each cost (X^\rho/k^2), total (X^\rho/k), still not raw.

But an arbitrary assignment on level (P_k) would cause many individual wrong vertex corrections against any neighbouring labelled block, yielding huge energy (X/k) per wrong vertex. Thus raw entropy paid by vertex correction, not cut cost, as long as there is at least one neighbouring labelled block. If all neighbours are also isolated, they form a large component of arbitrary residues? Dense agreement between them absent if all links bad, but making all internal links bad may cost, still maybe less than raw entropy? Need handle.

This indicates the component labels must be produced not only by good backbone links; if links are bad, arbitrary residues induce massive edge energy beyond threshold, not just (T). Our coarse bad cut cost (T) is too weak; for raw arbitrary entropy, actual energy is much larger and vertex correction can be used after choosing labels.

We need a more robust decomposition: every level either gets a label from at least one good incident block, or if it has no good incident blocks, then all its incident block pairs are bad, and perhaps their total bad-edge density implies many edges with (|H|>B). But cost still low under cutoff. Yet arbitrary residues likely have phases of order 1, not just (B/(pq)). The cutoff lower bound is crude.

So Section 9 remains nontrivial. This is a real gap.

# 10. Possible repair: choose local labels per level first

For each level (P_k), even without a good link, define whether it is internally coherent by comparing to many adjacent levels. If a level has no coherent label, then many vertices are wrong relative to any neighbour, and Irving correction gives large energy.

Maybe use full graph to prove: any large block of arbitrary residues either has a label or pays vertex-level energy. Dense agreement on one good link gives label; if no good link, then every adjacent link has positive bad density. But bad density under cutoff may not be enough; however if a whole level lacks label, then for most vertices there is no common residue, and edge phases against neighbours are equidistributed, giving constant energy per edge. This would pay raw entropy. Proving this is essentially a stronger agreement/energy lemma.

So our simplified bookkeeping was too optimistic.

# 11. New refined task

We must strengthen Section 9 proposition:

Not merely:

[  
\text{bad link costs }T_k,  
]

but:

[  
\text{a component without a label costs energy proportional to its raw entropy.}  
]

Dense agreement gives a label if a link has high strong-good density. If no such link, then edge phases are not mostly small; perhaps energy is actually of order number of edges, not just (T_k), after choosing a larger threshold.

We used strong-good cutoff (B=X^{5/4}). Bad edges only guarantee phase (>B/(pq)=X^{5/4}/X^{7/3}=X^{-13/12}), tiny. Hence bad density alone weak.

But Fourier factor only needs actual phase. A bad edge may still have tiny phase, just larger than cutoff. If all edges have phases around (X^{-1}), energy small but above cutoff. Could this allow huge entropy? Maybe residues constrained to labels of size (X^{4/3}), many more possibilities. Label entropy then grows with cutoff, still only (O(k)) per component, not raw. This suggests use multi-threshold labels.

---

# 12. Multi-scale cutoff / multilevel labels

Instead of one cutoff (B_k), use a hierarchy:

[  
B_k^{(r)}=2^r B_k.  
]

If edges are not good at cutoff (B), maybe they are good at larger cutoff (B'), producing a larger label. Larger label entropy increases by (\log B'), but bad edge energy threshold also increases as (B'^2).

This is natural. For any component, choose minimal cutoff at which dense agreement holds. Then:

- label entropy (\log B');
    
- if need larger (B'), energy paid by previous thresholds;
    
- if no cutoff up to (pq), phases are large and energy huge.
    

This may solve raw entropy leak.

This is a genuine refinement.

---

# 13. Interpretation

We were using a single cutoff to force small labels. But components may have legitimate larger labels (h), not major, and Gaussian energy should pay them. A hierarchy of cutoffs exactly captures this.

Then an isolated level can still have a label (h) of some size; entropy of h is (\log |h|), not raw vertex entropy. If it lacks even large label, vertex correction/energy kills.

So Section 9 should be rewritten with **adaptive labels**:

For each good block pair and each threshold (B), if most glues lie within ([-B,B]), get a label (|h|\le B). The cost of increasing (B) is reflected in diagonal energy (h^2/(pq)^2).

This becomes a multiscale energy-entropy proof, more complex but plausible.

---

# 14. Current conclusion

We found a serious but not fatal issue: a single sparse backbone chain and single cutoff cannot by itself pay for isolated levels. The remedy is either:

1. use full dense scale graph plus vertex correction more carefully; or
    
2. introduce adaptive/multi-threshold labels so every low-energy component gets some label, whose size is paid by Gaussian energy.
    

This is the next thing to develop.

This is important progress: we caught a hidden entropy leak before pretending the proof is done.
