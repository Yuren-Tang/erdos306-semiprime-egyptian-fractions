好。現在證一個 **adaptive dense block agreement lemma**。這是對前面 fixed cutoff lemma 的升級，也是修補熵漏洞的關鍵。

---

# Adaptive dense block agreement lemma

設

[  
P\subset[X,2X],\qquad Q\subset[Y,2Y],  
]

其中 (Y>X)，並給 residues

[  
a_p\in\mathbb F_p,\qquad b_q\in\mathbb F_q.  
]

對每條邊 (pq)，令

[  
H_{pq}^{*}\in(-pq/2,pq/2]  
]

為 CRT glue：

[  
H_{pq}^{_}\equiv a_p\pmod p,\qquad  
H_{pq}^{_}\equiv b_q\pmod q.  
]

定義局部能量

# [  
\mathcal E(P,Q)

\sum_{p\in P,\ q\in Q}  
\left(\frac{H_{pq}^{*}}{pq}\right)^2.  
]

令平均平方相位為

# [  
\overline e

\frac{\mathcal E(P,Q)}{|P||Q|}.  
]

我們希望從小 (\overline e) 推出存在 label。

---

## 1. Markov 產生自適應 cutoff

取固定小參數

[  
0<\varepsilon_0<10^{-3}.  
]

令

# [  
B

\varepsilon_0^{-1/2},XY,\sqrt{\overline e}.  
]

因為

[  
pq\asymp XY,  
]

若

[  
|H_{pq}^{*}|>B,  
]

則該邊能量至少約

[  
\left(\frac{B}{CXY}\right)^2  
\asymp  
\varepsilon_0^{-1}\overline e.  
]

所以由 Markov，不滿足

[  
|H_{pq}^{*}|\le B  
]

的邊比例至多 (O(\varepsilon_0))。

更精確地，調大常數可保證 bad edge density (\le\varepsilon_0)。

因此，只要 (B) 滿足 dense agreement lemma 的兩個 structural 條件：

[  
B<Y/2,  
\qquad  
X^2>2B,  
]

就可套用 dense agreement lemma，得到一個整數

[  
|h|\le B  
]

使得除 (O(\varepsilon_0)) 比例外，

[  
a_p\equiv h\pmod p,\qquad b_q\equiv h\pmod q.  
]

所以：

[  
\boxed{  
\text{若平均能量小到使 }B<\min(Y/2,X^2/2),  
\text{則 block 有 label，且 }|h|\ll XY\sqrt{\overline e}.  
}  
]

這就是 adaptive label。

---

## 2. 若 (B) 太大，能量本身已經不小

若

[  
B\ge Y/2,  
]

則

[  
XY\sqrt{\overline e}  
\gg Y,  
]

所以

[  
\sqrt{\overline e}\gg \frac1X,  
]

即

[  
\overline e\gg \frac1{X^2}.  
]

因此

[  
\mathcal E(P,Q)  
\gg  
\frac{|P||Q|}{X^2}.  
]

因為

[  
|P|\asymp \frac X{\log X},  
\qquad  
|Q|\asymp \frac Y{\log Y},  
]

有

[  
\mathcal E(P,Q)  
\gg  
\frac{Y}{X\log X\log Y}.  
]

若 (Y=X^\gamma)，這是

[  
\gg  
\frac{X^{\gamma-1}}{(\log X)^2}.  
]

對 (\gamma=4/3)，為

[  
\gg  
\frac{X^{1/3}}{(\log X)^2}.  
]

這非常大，足以支付至少許多 label entropy。

若

[  
B\ge X^2/2,  
]

則

[  
XY\sqrt{\overline e}  
\gg X^2,  
]

所以

[  
\sqrt{\overline e}\gg \frac X Y.  
]

若 (Y=X^\gamma)，得到

[  
\overline e\gg X^{2-2\gamma}.  
]

總能量

# [  
\mathcal E(P,Q)  
\gg  
|P||Q|X^{2-2\gamma}  
\asymp  
\frac{X^{1+\gamma}X^{2-2\gamma}}{(\log X)^2}

\frac{X^{3-\gamma}}{(\log X)^2}.  
]

對 (\gamma=4/3)，為

[  
\gg  
\frac{X^{5/3}}{(\log X)^2},  
]

更大。

所以 (B) 太大時，block energy 已巨大，絕不會形成低能熵漏洞。

---

## 3. Adaptive lemma 結論

因此對完整 block pair (P\times Q)，我們有二分：

### Case A：coherent case

存在 label (h)，使大多數 vertices 滿足

[  
a_p\equiv h\pmod p,\qquad b_q\equiv h\pmod q,  
]

且

[  
|h|  
\ll  
XY  
\sqrt{  
\frac{\mathcal E(P,Q)}{|P||Q|}  
}.  
]

### Case B：energetic case

局部能量至少

[  
\mathcal E(P,Q)  
\gg  
\min\left(  
\frac{|P||Q|}{X^2},  
\frac{|P||Q|X^2}{Y^2}  
\right).  
]

在 (Y=X^{4/3}) 時，這至少

[  
\gg  
\frac{X^{1/3}}{(\log X)^2}.  
]

這個 energetic case 的能量遠大於 (k=\log X)，因此能支付 label entropy，甚至大量局部 entropy。

---

## 4. 這如何修補 Section 9？

現在不再用固定 cutoff (B=X^{5/4}) 判斷一個 link good/bad。改為：

對每個 block pair，根據其實際能量 (\mathcal E(P,Q)) 產生自然 label scale

[  
B(P,Q)  
\asymp  
XY\sqrt{  
\frac{\mathcal E(P,Q)}{|P||Q|}  
}.  
]

若能量小，得到小 label。  
若不能得到 label，能量已大到足以進入 minor arc 衰減。

所以一個孤立 level 不可能完全自由：

- 若它和某相鄰 block 的能量小，adaptive lemma 產生 label；
    
- 若它和所有相鄰 blocks 的能量都大，則 total energy 很大，足以殺掉 raw entropy；
    
- 若有 label 但 label 大，則 label entropy (\log B) 很小，且 label size 對應的 Gaussian energy會支付。
    

這正是需要的 coercivity。

---

## 5. 但有一個重要細節：label 大小由邊能量控制，不直接由 label 自身能量控制

Adaptive lemma 給：

[  
|h|\ll XY\sqrt{\overline e}.  
]

等價於

[  
\overline e\gg \frac{h^2}{X^2Y^2}.  
]

這與對角 label (h) 在該 block 上的自然平均能量

[  
\left(\frac h{XY}\right)^2  
]

一致。很好。

所以若某 component label 很大，它自然在 incident block pair 中貢獻能量

[  
\asymp |P||Q|\frac{h^2}{X^2Y^2}.  
]

這正是 Gaussian energy。label entropy (\log |h|) 很容易被 (h^2)-energy 尾部支付。

---

## 6. 現在重新定義 Section 9 的 decomposition

對每條重要 block pair (P_i\times P_j)，不再標記 binary good/bad，而標記：

- 若 energetic case：直接支付 large energy；
    
- 若 coherent case：得到 label (h_{ij}) 和 scale (B_{ij}).
    

然後 dense agreement 的 label merging仍成立，只是 labels 可能有不同大小。若相鄰 labels (h_1,h_2) 都解釋同一 block 中許多 primes，則

[  
h_1\equiv h_2\pmod p  
]

對許多 (p). 若

[  
|h_1-h_2|  
]

小於這些 primes 的 product，就相等。若不小，則 (h_1,h_2) are large; their size already corresponds to energy and can pay entropy. More systematically, use product of enough shared primes; since shared block has many primes, product can exceed any label bound below (L) unless labels differ by a multiple of huge product, in which case they represent different global CRT class and full graph edges to other blocks will charge.

這裡需要一些技術，但比 fixed cutoff 更自然。

---

## 7. 更簡單的策略：用 adaptive lemma 只保證每個 level有某 label

Instead of trying to merge all adaptive labels immediately, first show:

For every low-energy assignment and every level (P_k), there exists some label (h_k) explaining most vertices of (P_k), with size controlled by local energy.

Then:

- If neighbouring levels have labels (h_k,h_\ell), edges between them force either (h_k=h_\ell) or energy proportional to their difference.
    
- This becomes an ordinary one-dimensional label field on scale graph with energy  
    [  
    \sum_{k,\ell} w_{k\ell}(h_k-h_\ell)^2  
    ]  
    plus vertex correction.
    

This is much cleaner.

So Section 9 should become two-stage:

### Stage 1: block labelling

Low energy (\Rightarrow) each dyadic block (P_k) has a dominant label (h_k).

### Stage 2: label field smoothing

Edges between blocks force labels (h_k) to agree globally or pay Gaussian energy.

This resembles graph Poincaré but now labels are ordinary integers, not arbitrary residues.

---

## 8. Can Stage 1 be proved?

For a level (P_k), choose one neighbouring block (Q=P_{\gamma k}). Apply adaptive dense block lemma. If energetic case, done: energy pays. If coherent case, (P_k) gets label (h_k).

Thus every low-energy level gets a label from at least one neighbouring pair.

If a pair energetic, we can include its large energy in budget. If too many energetic pairs, minor arc killed.

Thus Stage 1 plausible.

---

## 9. Stage 2: labels across levels

Suppose (P_k) has label (h_k) and (P_\ell) has label (h_\ell). For most edges (pq\in P_k\times P_\ell), residues are

[  
h_k\bmod p,\qquad h_\ell\bmod q.  
]

The CRT glue for such edge is the symmetric integer congruent to (h_k\mod p) and (h_\ell\mod q).

If (h_k=h_\ell=h), glue is (h), small according to label size.

If (h_k\ne h_\ell), then for many p,q the CRT glue should be large, and energy should be large. More simply: if many edges have small glue, dense agreement applied to the pair would force a common label, so (h_k,h_\ell) equal unless energy high. Thus adaptive lemma again merges labels or charges energy.

So labels become global across low-energy connected graph.

---

## 10. This is likely the right final form

The earlier fixed backbone proof was too rigid. The final proof should use:

[  
\boxed{  
\text{Adaptive dense agreement repeatedly, not fixed cut good/bad.}  
}  
]

It says: low energy itself creates labels; labels that fail to merge create energy; vertices that fail labels create Irving correction energy.

This solves the isolated block entropy issue.
