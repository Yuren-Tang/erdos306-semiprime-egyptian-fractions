# Block labelling proposition

取兩個相鄰尺度塊

[  
P=P_k,\qquad Q=P_{\ell},  
]

其中

[  
X=2^k,\qquad Y=2^\ell,\qquad Y=X^\gamma,  
]

暫取

[  
\gamma=\frac43.  
]

對 residues

[  
a_p\in\mathbb F_p,\qquad b_q\in\mathbb F_q  
]

定義二部能量

# [  
\mathcal E_{P,Q}

\sum_{p\in P,q\in Q}  
\left(  
\frac{H_{pq}^{*}}{pq}  
\right)^2.  
]

令平均能量

# [  
\overline e_{P,Q}

\frac{\mathcal E_{P,Q}}{|P||Q|}.  
]

命題要說：

要麼 (\mathcal E_{P,Q}) 已很大；要麼存在一個整數 (h)，使大多數 (P\cup Q) 的 residues 都由 (h) 解釋，且

[  
|h|  
\ll  
XY\sqrt{\overline e_{P,Q}}.  
]

這裏的 (h) 大小是能量自動決定的。

---

## 1. 證明：自適應 cutoff

取固定小 (\varepsilon_0>0)。設

[  
B=C_{\varepsilon_0}XY\sqrt{\overline e_{P,Q}},  
]

其中 (C_{\varepsilon_0}) 足夠大，使 Markov 給出：

[  
#{(p,q):|H_{pq}^{*}|>B}  
\le  
\varepsilon_0 |P||Q|.  
]

如果

[  
B<Y/2  
]

且

[  
B<X^2/2,  
]

則 dense CRT agreement lemma 直接給出 label

[  
|h|\le B  
]

解釋除了 (O(\varepsilon_0)) 比例以外的 (P\cup Q) 頂點。

所以 coherent case 成立。

---

## 2. 若 (B\ge Y/2)

由

[  
B=CXY\sqrt{\overline e}  
]

得

[  
XY\sqrt{\overline e}\gg Y,  
]

故

[  
\sqrt{\overline e}\gg \frac1X  
]

and

[  
\overline e\gg X^{-2}.  
]

因此

# [  
\mathcal E_{P,Q}

# |P||Q|\overline e  
\gg  
\frac{XY}{k\ell}\cdot X^{-2}

\frac{Y}{Xk\ell}.  
]

當

[  
Y=X^{4/3}  
]

時，

[  
\mathcal E_{P,Q}  
\gg  
\frac{X^{1/3}}{k^2}.  
]

這隨 (X\to\infty) 發散，遠大於單個 label entropy (O(k))。

---

## 3. 若 (B\ge X^2/2)

同理，

[  
XY\sqrt{\overline e}\gg X^2  
]

所以

[  
\sqrt{\overline e}\gg \frac X Y=X^{1-\gamma}.  
]

因此

[  
\overline e\gg X^{2-2\gamma}.  
]

總能量

# [  
\mathcal E_{P,Q}  
\gg  
\frac{XY}{k\ell}X^{2-2\gamma}

\frac{X^{3-\gamma}}{k\ell}.  
]

對

[  
\gamma=4/3  
]

得到

[  
\mathcal E_{P,Q}  
\gg  
\frac{X^{5/3}}{k^2}.  
]

更大。

---

## 4. 因此命題成立

所以對相鄰 block pair：

[  
P_k\times P_{\gamma k}  
]

必有二分：

1. **label case**：存在整數 (h)，大多數 vertices 服從 (h)，且
    
    [  
    |h|\ll XY\sqrt{\frac{\mathcal E_{P,Q}}{|P||Q|}};  
    ]
    
2. **energetic case**：該 pair 本身已支付至少
    
    [  
    \gg \frac{X^{1/3}}{k^2}  
    ]
    
    的能量，足以支付尺度級 label entropy。
    

這正是我們需要的 Stage 1。

---

# 5. 這如何避免整層自由熵？

假設某層 (P_k) 沒有 label。取它與一個相鄰層 (Q=P_{\gamma k}) 的完整二部 pair。若該 pair energy 不大，Block labelling proposition 立刻給 (P_k) 一個 label。若不能給 label，則該 pair energy 已經至少

[  
\gg X^{1/3}/k^2.  
]

但 raw entropy of a whole level is

[  
\asymp X.  
]

這仍然不夠支付整層完全自由。

所以還要進一步說：如果一整層真的完全自由，不只是「一個相鄰 pair energetic」，而是它對許多相鄰 blocks 都 energetic，或者每個 vertex wrong correction 會付更大能量。

Block labelling proposition 單獨還不夠支付 raw (X)，但它保證：**低能態一定會給每層至少一個 label**，除非已經有一些能量成本。若只有一個 pair energetic，不足以付 raw entropy，但我們可以選另一個相鄰 pair繼續嘗試。Full graph 有很多相鄰 blocks。

---

## 6. 使用多個 neighbours

固定 level (P_k)。Compatible neighbours (\ell) 滿足

[  
\frac23\ell\le k\le\frac32\ell,  
]

所以有 (\asymp k) 個相鄰 levels.

對每個 neighbour (P_\ell)，套 adaptive block lemma。

若對某個 neighbour 處於 label case，就得到 (P_k) 的 label。

若對所有 (\asymp k) 個 neighbours 都處於 energetic case，則總能量至少約

[  
\sum_{\ell\asymp k}  
\frac{X^{c(\ell/k)}}{k^2}.  
]

其中每項至少某個 power (X^\rho/k^2). 即使乘以 (k)，也只有

[  
X^\rho/k.  
]

這仍未達 (X)。但它可能不需要達 (X)，因為沒有 label 不代表 raw entropy (X) 全部實現於低能 states；我們要的是 counting low-energy states, not paying all possible arbitrary residues.

仍需更強控制。

---

## 7. Better approach: one neighbour label suffices, and neighbour itself likely labelled

If (P_k) has no label from (P_{\gamma k}), maybe (P_{\gamma k}) has no stable residues either. But choose a chain: low energy across the whole graph should force labels to propagate. If an entire region lacks labels, it must be energetic across many internal pairs. The energy lower bound per pair may be small but the number of pairs across region is large: full compatible graph has (\asymp k^2) block pairs in a window of length k, not just k. Summing may become larger.

Let's estimate a window of levels around k, size (\asymp k). Number of block pairs inside window is (\asymp k^2). Each energetic pair contributes at least (X^\rho/k^2). Total if all energetic:

[  
k^2\cdot X^\rho/k^2=X^\rho.  
]

Still not (X) if (\rho<1).

So raw entropy still too large.

But again arbitrary residues would cause constant energy per edge, total edges enormous:

[  
#\text{edges in block pair}\asymp X^{1+\gamma}/k^2,  
]

much larger than (X). Our energetic lower bound (X^\rho) is a worst-case for glues just above structural threshold; such states may still be highly structured by a larger label, not arbitrary. Thus raw entropy not present. The adaptive lemma should be iterated at larger cutoffs, yielding larger labels rather than no label.

---

# 8. Multi-threshold resolves this

For a block pair, if energy is modest, it always gives a label with size (B). If (B) exceeds structural threshold, dense agreement cannot compare via (Y/2) or (X^2/2). But then label size is so large that the pair energy is already polynomial. Such large label still has only (\log B) entropy, not raw. The missing step is: even when (B>Y/2), maybe dense agreement fails structurally, but we can pass to a higher comparison block where moduli are larger.

This is key.

If label size is too large for (P_k\times P_{\gamma k}), go to a higher block (P_m) with primes (Z>B). Then comparison works. Full graph compatibility only allows ratios up to (3/2), but by multi-step chain upward we can reach larger moduli. Thus large labels are compared at higher scales.

So a level with large potential label is not raw-free; it requires moving to higher scales.

This is becoming multiscale label theory.

---

# 9. Simplified final strategy

For each energy level, choose comparison scale high enough so that

[  
Y>B  
]

and low-prime product exceeds (B), but still within Irving-compatible range by taking steps. Since compatible graph connects ratios up to (3/2), a label of size (H) can be compared once scale exceeds (H^{1/(1+\eta)}) or simply primes exceed H.

Thus global proof should allow labels of arbitrary size up to (1/\sigma) and compare them using sufficiently high blocks.

This is more complex but seems structurally sound.
