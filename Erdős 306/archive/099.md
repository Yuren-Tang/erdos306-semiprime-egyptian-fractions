# Multiscale CRT coercivity theorem：目標形態

令 (P) 是一大段 dyadic prime blocks 的並集，令 (E_{\mathrm{full}}) 是 compatible graph：

[  
P_k\times P_\ell,  
\qquad  
\frac23\ell\le k\le \frac32\ell.  
]

對 character

[  
a=(a_p)_{p\in P}  
]

定義能量

[  
Q(a)=  
\sum_{pq\in E_{\mathrm{full}}}  
|\phi_{pq}(a)|^2.  
]

我們希望證：

[  
\boxed{  
\text{若 }Q(a)\text{ 小，則存在整數 }h  
\text{ 使大多數 }a_p\equiv h\pmod p.  
}  
]

而且更強：

[  
Q(a)  
\gtrsim  
\text{Gaussian energy of }h  
+  
\text{energy cost of exceptions}.  
]

其中 Gaussian energy 是

[  
h^2\sum_{pq\in E_{\mathrm{full}}}\frac1{p^2q^2}  
]

在 (h) 還未大幅繞回時的自然量。

這個 theorem 才是真正該替代「固定 cutoff bad cuts」的核心。

---

## 1. 為什麼需要 multiscale？

因為 label (h) 可能很大。若在低尺度 (p\sim X) 上，(h>X)，那麼單個低素數模 (p) 看不出 (h) 是哪個普通整數；只能看出

[  
h\bmod p.  
]

要比較兩個 labels (h_1,h_2)，需要很多模數的乘積超過

[  
|h_1-h_2|.  
]

所以大 label 必須用更高尺度、更多 primes 來比較。

這正是 multiscale 的必要性：

[  
\boxed{  
\text{小 label 可在低尺度比較；大 label 必須升到高尺度比較。}  
}  
]

---

## 2. 低能如何產生 local labels？

對任意 block pair

[  
P_k\times P_\ell  
]

若其局部能量小，adaptive dense block lemma 給出一個 local label (h_{k,\ell})，滿足

[  
|h_{k,\ell}|  
\ll  
X_kX_\ell  
\sqrt{  
\frac{Q_{k,\ell}}{|P_k||P_\ell|}  
}.  
]

這個 label 解釋大多數兩側頂點。

若能量不小，則這個 block pair 已經付費。

所以對低能 character，大多數重要 block pairs 會給出 local labels。

---

## 3. Local labels 如何合併？

若兩個 local labels 解釋同一個 block (P_m)，則對許多

[  
p\in P_m  
]

有

[  
h_1\equiv h_2\pmod p.  
]

因此

[  
\prod_{p\in S}p  
\mid h_1-h_2  
]

for a large subset (S\subset P_m), unless (h_1=h_2).

因為 (P_m) 中有約

[  
\frac{2^m}{m}  
]

個 primes，哪怕取很小一部分，乘積也巨大：

[  
\prod_{p\in S}p  
\approx \exp\left(\sum_{p\in S}\log p\right).  
]

這很快超過任何 label size that can occur without already paying huge energy.

所以在低能範圍內，local labels 必須合併成一個 global (h)。

---

## 4. 若 labels 不合併，能量如何出現？

若 (h_1-h_2\ne0)，但它被同一 block 中很多 primes 整除，那麼 (|h_1-h_2|) 必須至少是這些 primes 的乘積。這使 label size enormous。

But enormous label size implies, via adaptive bound,

[  
Q_{k,\ell}  
\gtrsim  
|P_k||P_\ell|  
\left(\frac{h}{X_kX_\ell}\right)^2.  
]

That is precisely Gaussian energy.

所以 label 不合併的代價不是 raw combinatorial entropy，而是「label 差很大」導致的 quadratic energy。

這是很好的：它把 multiscale comparison 和 Gaussian energy 結合起來。

---

## 5. Wrong vertices 仍由 Irving correction 處理

一旦 global 或 component label (h) 建立，若某頂點 (q) 不服從

[  
a_q\equiv h\pmod q,  
]

則選相鄰已標記 block (P)，用 Irving 非集中：

[  
\sum_{p\in P}  
|\phi_{pq}(a)|^2  
\gg |P|.  
]

這遠大於該頂點 residue entropy。

所以 exceptions 仍可控。

# 6. Theorem 的可能正式版本

可以先寫一個能量層版本：

**Multiscale coercivity proposition.**  
For every (A>0), if (k_0) is sufficiently large, then for every character (a) either

1. (a) is a global diagonal character:  
    [  
    a_p\equiv h\pmod p\quad\forall p,  
    ]  
    for some integer (h); or
    
2. its energy satisfies an entropy-dominating lower bound:  
    [  
    Q(a)\ge A\cdot \mathcal H(a),  
    ]  
    where (\mathcal H(a)) is the number of free residue bits not explained by a global label.
    

More concretely, one wants:

[  
#{a:Q(a)\le R}  
\le  
#{h:Q(h)\le R+O(1)},e^{\varepsilon R}  
]

with (\varepsilon) as small as needed.

This is the clean entropy formulation.

---

## 7. Can we prove it from adaptive block lemma?

A possible proof route:

### Step A：choose low-energy block pairs

For each scale (k), choose several neighbouring scales (\ell\asymp k). If (Q_{k,\ell}) is small, get labels. If too many are large, total energy is large.

### Step B：assign labels to blocks

For each block (P_k), choose one incident low-energy pair and get a label (h_k). If none exists, (P_k) is high-energy and paid.

### Step C：merge labels

For adjacent or overlapping blocks, compare (h_k,h_\ell). If they disagree, many shared vertices/edges force large label difference or large energy. Charge to energy.

### Step D：correct vertices

Use Irving to force individual vertices to obey their block label.

This produces global (h) except where energy is paid.

---

## 8. The main technical burden

Step B is subtle: must ensure every block gets an incident low-energy pair unless enough energy has already been paid. Earlier we saw a single energetic pair does not pay raw block entropy, but if all incident pairs are energetic, perhaps their total is enough only if using **actual energy**, not crude cutoff. Adaptive lemma helps because energetic means not merely (X^\rho), but energy relative to absence of label; still needs a rigorous quantification.

This is likely the deepest part.

It may require an entropy argument rather than a deterministic per-block argument:

Let (A) be uniformly random among low-energy assignments. If block (P_k) has high entropy, then across many incident edges to neighbouring blocks the conditional entropy remains high, forcing high mutual information/energy. This returns to information theory, but now with better structural insight.

So yes: the core is still entropy.

---

## 9. Is this a sign of a real obstruction?

Not necessarily. It says the proof of Section 9 is nontrivial and likely needs a serious lemma, not just paragraph bookkeeping.

The possible obstruction would be a family of many assignments with moderately small energy but no global label. We have not found such a family. The CRT rigidity and Irving correction suggest such family should not exist. But proving nonexistence is the hard part.

So current state:

[  
\boxed{  
\text{no visible counterexample, but the decisive coercivity theorem remains unproved.}  
}  
]

# 10. What should we do next?

We should try to prove a **simplified multiscale coercivity theorem** in a model setting:

- Use three levels:  
    [  
    P_0,\ P_1,\ P_2  
    ]  
    with sizes  
    [  
    X,\ X^{4/3},\ X^{(4/3)^2}.  
    ]
    
- Full complete bipartite edges between (P_0\leftrightarrow P_1) and (P_1\leftrightarrow P_2).
    
- Show low energy implies one label across all three levels, or pays entropy.
    

If the three-level lemma works, it can be iterated.

This is the most manageable next step.

So next I would focus on:

[  
\boxed{  
\text{three-level adaptive agreement lemma.}  
}  
]

It is the smallest setting where large labels can be compared at a higher scale and where isolated-level entropy can be tested.
