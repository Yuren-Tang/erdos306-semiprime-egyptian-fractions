好。現在研究最小的 **three-level adaptive agreement lemma**。

取三層素數塊：

[  
P_0\subset[X,2X],  
]

[  
P_1\subset[Y,2Y],\qquad Y=X^\gamma,  
]

[  
P_2\subset[Z,2Z],\qquad Z=Y^\gamma=X^{\gamma^2},  
]

暫取

[  
\gamma=\frac43.  
]

所以

[  
Y=X^{4/3},\qquad Z=X^{16/9}.  
]

考慮完整二部邊：

[  
P_0\times P_1,\qquad P_1\times P_2.  
]

給 residues：

[  
a_p\in\mathbb F_p\quad(p\in P_0),  
]

[  
b_q\in\mathbb F_q\quad(q\in P_1),  
]

[  
c_r\in\mathbb F_r\quad(r\in P_2).  
]

目標：若兩組二部能量都低，則三層共享一個普通整數 label (h)，至少對大多數頂點成立；若不共享，則能量足以支付新增 label 或 wrong vertices 的熵。

---

## 1. 兩個二部 pair 產生兩個 local labels

設

# [  
E_{01}

\sum_{p\in P_0,q\in P_1}  
\left(\frac{H_{pq}^{*}}{pq}\right)^2,  
]

# [  
E_{12}

\sum_{q\in P_1,r\in P_2}  
\left(\frac{H_{qr}^{*}}{qr}\right)^2.  
]

由 adaptive dense block lemma：

若 (E_{01}) 不處於 energetic case，則存在 label

[  
h_{01}  
]

使大多數 (P_0\cup P_1) 被解釋，且

[  
|h_{01}|  
\ll  
XY\sqrt{\frac{E_{01}}{|P_0||P_1|}}.  
]

同理，若 (E_{12}) 不 energetic，則存在

[  
h_{12}  
]

解釋大多數 (P_1\cup P_2)，且

[  
|h_{12}|  
\ll  
YZ\sqrt{\frac{E_{12}}{|P_1||P_2|}}.  
]

如果某一 pair energetic，則能量已經大；暫先看兩者都 coherent 的情形。

---

## 2. 中層 (P_1) 迫使 labels 合併

兩個 labels 都解釋大多數中層 (P_1)。因此對許多

[  
q\in P_1  
]

有

[  
h_{01}\equiv b_q\equiv h_{12}\pmod q.  
]

所以

[  
q\mid h_{12}-h_{01}  
]

對很多 (q\in P_1) 成立。

令

[  
D=h_{12}-h_{01}.  
]

若 (D\ne0)，則

[  
|D|\ge \prod_{q\in S}q  
]

for any subset (S) of common explaining primes.

即使只取兩個中層 primes，

[  
|D|\gg Y^2=X^{8/3}.  
]

因此，如果

[  
|h_{01}|+|h_{12}|<Y^2/2,  
]

則必有

[  
h_{01}=h_{12}.  
]

所以三層共享同一 label。

---

## 3. 若 labels 太大而無法強制相等

如果

[  
|h_{01}|+|h_{12}|\ge cY^2,  
]

則至少一個 local label 很大。由 adaptive label size bound，這迫使相應二部能量大。

例如若

[  
|h_{12}|\gg Y^2,  
]

則

[  
YZ\sqrt{\frac{E_{12}}{|P_1||P_2|}}  
\gg Y^2,  
]

故

[  
\sqrt{\frac{E_{12}}{|P_1||P_2|}}  
\gg  
\frac{Y}{Z}.  
]

因此

[  
E_{12}  
\gg  
|P_1||P_2|\frac{Y^2}{Z^2}.  
]

估算：

[  
|P_1||P_2|\asymp \frac{YZ}{(\log X)^2}.  
]

所以

# [  
E_{12}  
\gg  
\frac{YZ}{(\log X)^2}\frac{Y^2}{Z^2}

\frac{Y^3}{Z(\log X)^2}.  
]

代入

[  
Y=X^{4/3},\qquad Z=X^{16/9},  
]

得

# [  
\frac{Y^3}{Z}

# \frac{X^4}{X^{16/9}}

X^{20/9}.  
]

非常大。

這遠遠超過 label entropy

[  
\log |h_{12}|\asymp \log X.  
]

所以若 labels 大到中層 primes 不能比較，能量已經巨大，足以支付其熵。

這正是我們想要的機制。

---

## 4. 這個三層 lemma 的結論

三層情形有清晰二分：

1. 兩個二部 pair 都低能，且 labels 不太大  
    [  
    \Rightarrow h_{01}=h_{12}  
    ]  
    三層共享一個 label。
    
2. 某個二部 pair 不 coherent  
    [  
    \Rightarrow  
    ]  
    它本身 energetic，付能量。
    
3. 兩個 labels coherent 但不同  
    [  
    \Rightarrow  
    ]  
    差值被中層很多 primes 整除；若仍不同，label 必巨大；由 adaptive bound 推出能量巨大。
    

所以三層模型中，**低能態必有共同 label**。這是非常好的訊號。

---

## 5. wrong vertices in three levels

若三層已有共同 label (h)，但某些 vertices 不服從：

[  
a_p\not\equiv h\pmod p,  
]

則用相鄰 labelled block 做 Irving correction。

- (P_0) 的 wrong vertices 可用 (P_1) 修正；
    
- (P_1) 的 wrong vertices 可用 (P_0) 或 (P_2) 修正；
    
- (P_2) 的 wrong vertices 可用 (P_1) 修正。
    

因為 (\gamma=4/3) 在 Irving 範圍內，兩個方向都可用。

每個 wrong vertex 付能量 (\gg) 相鄰 block size，遠大於其 (\log p) entropy。

所以三層內 exceptions 也可控。

---

## 6. 這解決了孤立中層問題嗎？

在三層模型中，(P_1) 不可能完全自由：

- 若 (P_0P_1) 和 (P_1P_2) 都低能，它被兩邊共同 label；
    
- 若其中一邊高能，已付能量；
    
- 若它有 label 但不服從，wrong vertices 被 correction 殺掉。
    

這正是單鏈模型缺失的東西：一個層需要左右兩側同時看，而不是只由 cut 決定。

---

# 7. 推廣到多層

多層可以用 overlapping triples：

[  
P_j,\ P_{j+1},\ P_{j+2}.  
]

每個三層窗口若低能，就迫使 labels 合併。若不合併，窗口內能量大。

因此可以構造一個 local-to-global argument：

- 每個 adjacent pair (j,j+1) 產生 label (h_{j,j+1}) 或付能量；
    
- overlapping pairs share middle levels;
    
- three-level lemma forces  
    [  
    h_{j,j+1}=h_{j+1,j+2}  
    ]  
    unless energy large;
    
- hence along a low-energy chain all labels equal.
    

這避免了 isolated-level entropy leak。

---

## 8. Energy–entropy with triples

Now bad object is not a single cut but a **bad triple** or **label discontinuity**. Each bad triple costs large energy.

A level can be isolated only if adjacent triples are bad, but then it is surrounded by paid high-energy windows. Moreover if it still has arbitrary residues, wrong-vertex correction against any coherent neighbour or internal high energy kills it.

Thus bookkeeping can be based on bad triples rather than bad cuts.

Bad triple cost, from above, can be very large if labels fail to merge due to size. If pair itself energetic, also large. This is more robust.

---

## 9. Formal three-level lemma

We can state:

**Three-level adaptive agreement lemma.**  
Let

[  
P_0,P_1,P_2  
]

be as above. For every residues assignment, either:

1. there exists an integer (h) explaining all but (O(\varepsilon)) vertices in all three levels; or
    
2. the energy on  
    [  
    P_0P_1\cup P_1P_2  
    ]  
    is at least  
    [  
    \Omega(\Lambda_X),  
    ]  
    where  
    [  
    \Lambda_X\to\infty  
    ]  
    fast enough to pay all non-labelled entropy in this triple.
    

More refined version:

If local labels exist but differ, energy (\gg) quadratic size of label difference. If no local labels, energy is large. If labels equal, wrong vertices are corrected by Irving.

This is the right lemma.

---

## 10. Remaining quantitative question

What is the minimum energy (\Lambda_X) in the bad case?

We estimated:

- structural threshold failure in (P_0P_1) gives at least (X^{1/3}/(\log X)^2);
    
- label disagreement too large gives much bigger, e.g. (X^{20/9}) in one case;
    
- wrong vertices give (\gg X/\log X) per vertex.
    

The weakest is (X^{1/3}/(\log X)^2). This does not pay raw triple entropy (X^{16/9}), but bad case does not necessarily allow raw entropy; it usually still has labels. For the energy–entropy proof we must count bad triples carefully, not assign raw entropy to them.

So a fully rigorous theorem should avoid claiming (\Lambda_X) pays all raw entropy. Instead it should say: either common label exists, or the number of assignments with energy (R) is bounded by labelled choices times (e^{O(R)}). That is more subtle.

Still, three-level analysis suggests no obvious free entropy.

---

## 11. Next best step

We should formulate the three-level lemma not as a crude dichotomy but as a **counting lemma**:

[  
#{(a,b,c):E_{01}+E_{12}\le R}  
\le  
\sum_h  
\exp(O(R))  
]

up to vertex entropy paid by correction.

This would directly serve Section 9.

Can we prove this from adaptive labels? For fixed energy (R):

- possible (h_{01}) values bounded by energy;
    
- possible (h_{12}) values bounded by energy;
    
- if (h_{01}\ne h_{12}), their difference divisible by many middle primes, forcing huge size, which costs energy;
    
- wrong vertices paid.
    

Thus yes, a counting form is plausible.
