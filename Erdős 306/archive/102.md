好。現在做 **global gluing induction**。這一步的問題是：

三層 counting lemma 只說：

[  
(P_j,P_{j+1},P_{j+2})  
]

低能時有一個共同 label。  
但 full graph 有很多層，我們要證所有三層窗口的 labels 會合併成同一個全局 (h)，而不是每個窗口各自貢獻一次 label entropy。

---

# 1. 尺度鏈

取一條尺度鏈

[  
P_0,P_1,\dots,P_J  
]

其中

[  
P_j=P_{k_j}^{\ast},  
\qquad  
k_{j+1}\approx \gamma k_j,  
\qquad  
\gamma=\frac43.  
]

先只考慮鏈上相鄰 pair 的完整二部邊：

[  
P_j\times P_{j+1}.  
]

Full graph 還有更多邊，但這些已足夠做 agreement；其他邊用於 mass 和額外衰減。

對 character (a)，定義鏈能量

# [  
Q_{\mathrm{chain}}(a)

\sum_{j=0}^{J-1}  
Q_{j,j+1}(a),  
]

其中

# [  
Q_{j,j+1}

\sum_{p\in P_j,q\in P_{j+1}}  
|\phi_{pq}(a)|^2.  
]

---

# 2. 三層窗口

對每個

[  
j=0,\dots,J-2,  
]

看 triple

[  
\mathcal T_j=(P_j,P_{j+1},P_{j+2})  
]

with energy

# [  
Q(\mathcal T_j)

Q_{j,j+1}+Q_{j+1,j+2}.  
]

三層 counting lemma 告訴我們：

若

[  
Q(\mathcal T_j)\le R_j,  
]

則此 triple 的 assignments 數受控於一個 label (h_j) 加上 (e^{C R_j}) 的誤差；若沒有共同 label，則能量已進入 (e^{C R_j}) 支付項。

更結構性地說：低能狀態在 (\mathcal T_j) 上有一個 dominant label

[  
h_j  
]

解釋三層中大多數 vertices，例外由能量支付。

---

# 3. 相鄰 triples 的 labels 必合併

相鄰 triples：

[  
\mathcal T_j=(P_j,P_{j+1},P_{j+2}),  
]

[  
\mathcal T_{j+1}=(P_{j+1},P_{j+2},P_{j+3})  
]

重疊兩層：

[  
P_{j+1},P_{j+2}.  
]

若兩者都有 labels

[  
h_j,\quad h_{j+1},  
]

則二者都解釋重疊層中的大多數 vertices。比如在 (P_{j+1}) 中，對許多 primes (p),

[  
h_j\equiv h_{j+1}\pmod p.  
]

若

[  
h_j\ne h_{j+1},  
]

則差

[  
D=h_{j+1}-h_j  
]

被許多 primes 整除。

只取兩個共同 primes (p_1,p_2\in P_{j+1})，就有

[  
p_1p_2\mid D.  
]

若 (|D|<p_1p_2)，則必相等。  
若 (|D|\ge p_1p_2)，則某個 label 已很大；由三層 adaptive bound，這種大 label 已反映為 corresponding triple energy，並被 (e^{CR}) 支付。

所以在 counting lemma 層面，相鄰 triple labels 要麼合併，要麼付能量。低能主體只有一個全局 label。

---

# 4. 歸納敘述

定義

[  
N_m(R)  
]

為前 (m) 層

[  
P_0,\dots,P_m  
]

上能量 (\le R) 的 assignments 數。

我們想證歸納命題：

[  
N_m(R)  
\le  
\sum_h  
\mathbf 1_{E_m(h)\le C R}  
\exp(CR),  
]

其中

# [  
E_m(h)

\sum_{j=0}^{m-1}  
\sum_{p\in P_j,q\in P_{j+1}}  
\left(\frac{h}{pq}\right)^2  
]

是 (m) 層鏈上的 diagonal energy。

基礎 (m=2) 或 (m=3) 由 three-level counting lemma 給出。

假設對 (m) 成立，加入 (P_{m+1})。使用最後 triple

[  
(P_{m-1},P_m,P_{m+1}).  
]

此 triple 的 label 必須與前 (m) 層的 global label (h) 在重疊層 (P_{m-1},P_m) 上一致；否則由 product-of-primes 或 large-label energy 支付。故新增層不能自由引入新 label，除非付能量。

因此

[  
N_{m+1}(R)  
\le  
\sum_h  
\mathbf 1_{E_{m+1}(h)\le C R}  
\exp(CR).  
]

這就是歸納。

---

# 5. 需要避免能量重複

三層窗口 energies 重疊：

[  
Q(\mathcal T_j)=Q_{j,j+1}+Q_{j+1,j+2}.  
]

每個 adjacent pair energy (Q_{j,j+1}) 出現在兩個 triples 中。這只是 bounded overlap，乘一個常數即可。

在歸納證明中，更乾淨的方法是：每次加入新層，只把新 pair energy

[  
Q_{m,m+1}  
]

作為新增能量，而用 triple lemma 的局部結構來控制 label merging。雖然 triple lemma本身用到 (Q_{m-1,m})，但那部分已在過去能量中計入；總體 bounded overlap 常數可吸收。

所以不會產生嚴重問題。

---

# 6. wrong vertices 的全局處理

在每次 gluing 中，若新層 (P_{m+1}) 中某 vertex 不服從 global label (h)，用 (P_m) 的 labelled majority 對它做 Irving correction。

這些 correction edge sets

[  
P_m\times{q}  
]

for different (q\in P_{m+1}) are disjoint. Across (m)，不同 block pairs disjoint. So wrong vertex energy adds without overlap.

若舊層中有 wrong vertices，它們已在前一步被支付。

因此 induction naturally handles exceptions.

---

# 7. global label count

歸納最後得到：

[  
N_J(R)  
\le  
\sum_h  
\mathbf 1_{E_{\mathrm{chain}}(h)\le C R}  
e^{CR}.  
]

Now

# [  
E_{\mathrm{chain}}(h)

h^2  
\sum_{j}  
\sum_{p\in P_j,q\in P_{j+1}}  
\frac1{p^2q^2}  
]

in the no-wrap range.

這個 chain diagonal variance may be smaller than full graph variance, but enough for counting non-diagonal structures. For final Fourier sum, full graph gives stronger energy, so okay.

The number of h with (E_{\mathrm{chain}}(h)\le CR) is

[  
\ll 1+\sqrt{R}/\sigma_{\mathrm{chain}}.  
]

If (\sigma_{\mathrm{chain}}) is comparable enough to (\sigma_{\mathrm{full}}), good. But chain variance might be dominated by lowest link and still comparable to (\sigma_{\mathrm{full}}) up to constants? Let's check.

For chain links (P_jP_{j+1}),

[  
\sum\frac1{p^2q^2}  
\asymp  
\frac1{X_j k_j}\frac1{X_{j+1}k_{j+1}}.  
]

Dominated by (j=0):

# [  
\asymp  
\frac1{X_0^{1+\gamma}k_0^2}

\frac1{X_0^{7/3}k_0^2}.  
]

But full graph variance dominated by lowest same-scale or near-lowest pairs:

[  
\sigma_{\mathrm{full}}^2\asymp \frac1{X_0^2 k_0^2}.  
]

Thus

[  
\sigma_{\mathrm{chain}}  
\asymp  
X_0^{-7/6}/k_0,  
]

while

[  
\sigma_{\mathrm{full}}\asymp X_0^{-1}/k_0.  
]

So chain variance is smaller by (X_0^{-1/6}). That means chain diagonal count is larger by (X_0^{1/6}), perhaps not fatal? Major scale is (1/\sigma_{\mathrm{full}}), so an extra (X_0^{1/6}) would be too large unless full graph diagonal energy used.

We need include low-nearby full graph edges in diagonal energy. Agreement chain only proves structure; once global (h) is established, full graph energy controls (h). So final count should use full diagonal energy, not chain energy.

Thus induction conclusion should be:

[  
N_J(R)  
\le  
\sum_h  
\mathbf 1_{E_{\mathrm{full,diag}}(h)\le C R}  
e^{CR}.  
]

Because for globally labelled assignments, full graph energy is available. Good.

---

# 8. Minor sum from global counting

If

[  
#{a:Q_{\mathrm{full}}(a)\le R}  
\le  
\sum_h\mathbf 1_{Q_{\mathrm{diag}}(h)\le C R}e^{CR},  
]

then

[  
\sum_a e^{-cQ(a)}  
\le  
\sum_{R}  
e^{-cR}e^{CR}  
#{h:Q_{\mathrm{diag}}(h)\le C R}.  
]

Need (C<c). But our (C) from counting may be large. Usually one cannot ensure (C<c) unless energy payments dominate with arbitrary large constants by taking (k_0) large. For wrong vertices and cut payments yes, but for labels (e^{CR}) from adaptive counting may have fixed constant.

We need ensure counting entropy coefficient can be made smaller than Fourier decay coefficient. Is that possible? If not, use sharper partition function estimate rather than crude (e^{CR}). This is technical.

For diagonal (h), we do not pay (e^{CR}) per h; we sum exact Gaussian. Non-diagonal entropy payments can be made with arbitrarily strong constants because (k_0) large gives energy per entropy huge. Thus maybe (C) can be chosen small for non-diagonal part.

The three-level counting lemma must be formulated with (e^{\varepsilon R}) for arbitrary (\varepsilon>0), after (k_0) large. That is plausible because payments are polynomially huge versus logarithmic entropy.

So target:

[  
N(R)\le  
#{h:Q_{\mathrm{diag}}(h)\le C R}e^{\varepsilon R}  
]

with (\varepsilon<c/10).

Good.

---

# 9. This gluing induction seems plausible

We have now a route to global Section 9:

1. Prove three-level counting with (e^{\varepsilon R}).
    
2. Glue triples inductively; overlaps bounded.
    
3. Get global counting with (e^{\varepsilon R}).
    
4. Sum against (e^{-cR}).
    
5. Diagonal (h)-sum gives (O(1/\sigma)); excluding major arc gives (o(1/\sigma)).
    

This is the correct architecture.

---

# 10. Remaining hard lemma

Everything now rests on proving the three-level counting lemma with small entropy coefficient:

[  
N_3(R)\le  
\sum_h \mathbf 1_{E_{\mathrm{diag},3}(h)\le C R} e^{\varepsilon R}.  
]

This is still nontrivial, but at least it is local and concrete. It combines:

- adaptive dense agreement;
    
- product-of-primes label merging;
    
- Irving correction for exceptions.
    

This is much more manageable than the full global problem.
