好。現在把 **Three-level counting lemma** 寫成一個比較精確、可攻的版本。這一步很重要：它若能證，後面的 global gluing 就比較自然；若它證不出，整條 Fourier–agreement 路線就還有實質缺口。

---

# Three-level counting lemma：精確目標

取三層 good prime blocks：

[  
P_0\subset[X,2X],  
]

[  
P_1\subset[Y,2Y],\qquad Y=X^{4/3},  
]

[  
P_2\subset[Z,2Z],\qquad Z=Y^{4/3}=X^{16/9}.  
]

考慮完整二部邊：

[  
P_0\times P_1,\qquad P_1\times P_2.  
]

對 assignment

[  
a=(a_p)_{p\in P_0},\quad  
b=(b_q)_{q\in P_1},\quad  
c=(c_r)_{r\in P_2}  
]

定義

# [  
Q_3(a,b,c)

Q_{01}+Q_{12},  
]

其中

# [  
Q_{01}

\sum_{p\in P_0,q\in P_1}  
|\phi_{pq}|^2,  
]

# [  
Q_{12}

\sum_{q\in P_1,r\in P_2}  
|\phi_{qr}|^2.  
]

定義三層 diagonal energy：

# [  
Q_3^{\mathrm{diag}}(h)

\sum_{p,q}  
\left|\frac{h}{pq}\right|^2  
+  
\sum_{q,r}  
\left|\frac{h}{qr}\right|^2.  
]

希望證：

> 對任意 (\varepsilon>0)，若 (X) 足夠大，則對所有 (R\ge1),  
> [  
> #{(a,b,c):Q_3(a,b,c)\le R}  
> \le  
> e^{\varepsilon R}  
> #{h\bmod L_{012}: Q_3^{\mathrm{diag}}(h)\le C R}.  
> ]
> 
> 其中 (L_{012}=\prod_{p\in P_0\cup P_1\cup P_2}p)，(C) 是絕對常數。

這是三層 local energy–entropy inequality。

---

# 1. 為什麼右邊是 (h\bmod L_{012})？

因為三層全部服從同一 label (h) 時，assignment 完全由

[  
h\bmod L_{012}  
]

決定：

[  
a_p\equiv h\pmod p,\quad  
b_q\equiv h\pmod q,\quad  
c_r\equiv h\pmod r.  
]

如果 (h) 有小整數代表，這是 ordinary diagonal；如果 (h) 是大 residue，則 (Q_3^{\mathrm{diag}}(h)) 會反映它在三層邊上的繞回能量。

這樣避免了「大 (h) ordinary representative」的尷尬。

---

# 2. 證明策略

給一個低能 assignment ((a,b,c))，要把它映射到：

1. 一個 diagonal residue (h\bmod L_{012})；
    
2. 一組錯誤／例外資料；
    
3. 額外資料的總數由 (e^{\varepsilon R}) 控制。
    

也就是建立一個 encoding：

[  
(a,b,c)  
\longmapsto  
(h,\text{error data})  
]

使

[  
#{\text{error data with cost }\le R}  
\le e^{\varepsilon R}.  
]

---

# 3. Step A：由 (P_0P_1) 產生 local label

對 pair (P_0\times P_1)，用 adaptive dense agreement。

給定 (Q_{01})，取 cutoff

# [  
B_{01}

C XY\sqrt{\frac{Q_{01}}{|P_0||P_1|}}.  
]

若 structural 條件成立，得到 label (h_{01}) 解釋大多數 (P_0\cup P_1)。

若 structural 條件不成立，則 (Q_{01}) 已很大。這種情況直接歸入 error data，並由 (Q_{01}) 付款。

同理，從 (P_1P_2) 得 label (h_{12}) 或由 (Q_{12}) 付款。

所以每個低能 assignment either already costly, or yields two local labels.

---

# 4. Step B：local labels 合併

若有 (h_{01}) 和 (h_{12})，它們都解釋大多數 (P_1)。因此它們在許多 (q\in P_1) 上同餘：

[  
h_{01}\equiv h_{12}\pmod q.  
]

如果不相等 modulo (L_{012})，則差值 (D=h_{12}-h_{01}) 被許多中層 primes 整除。取足夠多這樣的 (q)，其乘積超過

[  
\max(|h_{01}|,|h_{12}|)  
]

在低能範圍內所允許的大小，除非 (Q_{01}+Q_{12}) 很大。

更形式地：

由 label size bounds，

[  
|h_{01}|  
\ll  
XY\sqrt{\frac{Q_{01}}{|P_0||P_1|}},  
]

[  
|h_{12}|  
\ll  
YZ\sqrt{\frac{Q_{12}}{|P_1||P_2|}}.  
]

若

[  
h_{01}\ne h_{12},  
]

且二者共同解釋 (m) 個中層 primes，則

[  
Y^m\ll |h_{01}-h_{12}|.  
]

這強迫

[  
Q_{01}+Q_{12}\gg Y^{2m}/(YZ)  
]

up to logs. 取 (m=2)，已是巨大 power of (X). 因此這些不合併情況的數量由 (e^{\varepsilon R}) 控制。

在非 costly 情形，必有

[  
h_{01}=h_{12}=:h.  
]

---

# 5. Step C：例外頂點

現在大多數三層頂點服從同一 (h)。剩餘錯誤頂點：

[  
v:\ a_v\not\equiv h\pmod v  
]

用 Irving correction 付款。

若 (v\in P_0)，用 (P_1) 作 correction block。  
若 (v\in P_1)，用 (P_0) 或 (P_2)。  
若 (v\in P_2)，用 (P_1)。

每個錯誤頂點 (v) 付能量

[  
E_v\gg |P_{\mathrm{adj}}|.  
]

而它的 residue choice entropy 是

[  
\log v.  
]

由於

[  
E_v/\log v\to\infty  
]

as (X\to\infty)，對任意 (\varepsilon>0)，取 (X) 大可得

[  
\prod_{v\in W} v  
\le  
\exp\left(\varepsilon \sum_{v\in W}E_v\right).  
]

因此錯誤頂點資料由 (e^{\varepsilon R}) 控制。

---

# 6. Step D：例外集合本身的選擇

除了錯誤頂點的 residues，還要選哪些 vertices 是錯誤的。這個 entropy 是

[  
\binom{|P_i|}{m_i}  
]

類型。

但如果 (m_i) 個 vertices 錯誤，每個付 (E_v\gg |P_{\mathrm{adj}}|)，總能量巨大。

選擇集合的 entropy 至多

[  
m_i\log |P_i|  
]

而

[  
E_v\gg |P_{\mathrm{adj}}|\gg \log |P_i|.  
]

所以錯誤集合的選擇也由能量支付。

因此錯誤 vertices 全部貢獻

[  
e^{\varepsilon R}.  
]

---

# 7. Step E：structural-failure cases

如果 adaptive dense agreement 的 structural conditions fail，即 cutoff (B) 太大，則 pair energy 已有 polynomial lower bound, e.g.

[  
Q_{01}\gg X^{1/3}/(\log X)^2  
]

or larger.

這種情況並不自動給 label，但它的能量非常大。要計數這類 assignments，不能讓 raw entropy漏出。

這是 lemma 內最危險的一點。

需要說明：structural failure corresponds to label size exceeding the scale visible to that pair. But then use the next pair (P_1P_2) to compare at higher scale. In three-level setting, if (P_0P_1) fails structurally, (P_1P_2) may still create a label for (P_1), and then Irving correction controls (P_0) through (P_1), or (P_0P_1) energy itself is large.

But if (P_0P_1) fails and (P_1P_2) also fails, total energy is at least a polynomial. Is that enough to count all assignments? Raw entropy is roughly (Z) at top layer, much larger. So still not enough by itself.

Therefore structural failure cannot simply be counted as arbitrary. We must in those cases use a **larger label class** rather than no label.

This means the three-level counting lemma must allow labels modulo (L) from the beginning, not only small ordinary labels.

---

# 8. Revised handling of structural failure

Instead of saying dense agreement fails when (B>Y/2), we can avoid requiring (B<Y/2) by using a different comparison method:

If (B>Y/2), then high vertex (q) no longer uniquely determines ordinary glue in ([-B,B]). But it determines glue modulo (q). We can still get a label modulo product of many (q)'s, i.e. a residue class (h\bmod M), not a small integer.

For counting, that is acceptable. Eventually overlapping levels lift this to (h\bmod L_{012}).

This is important: dense agreement should be formulated in modular rather than small-integer form at high cutoff.

Fixed small (B) gave ordinary label; large (B) gives modular label over a larger modulus. Counting diagonal (h\bmod L) naturally handles this.

So the right local label is:

[  
h \in \mathbb Z/M\mathbb Z  
]

where (M) is product of enough primes in the block, not necessarily a small integer.

Then no structural failure; there is always either a modular label or high energy.

This is more abstract but likely correct.

---

# 9. This is getting deeper

The three-level counting lemma is not a trivial consequence of previous dense agreement. It needs a **modular/adaptive agreement theorem**:

Low energy on a dense block pair implies that the residues are mostly restrictions of a common residue class (h) modulo the product of many involved primes, with energy controlling how small its representatives are on edges.

This is essentially a true agreement theorem over CRT systems.

So we are back to a serious local-to-global agreement problem, though now much better understood.

---

# 10. Honest conclusion at this point

We cannot yet claim the three-level counting lemma is proved. We can state it as the central conjectural/provable lemma needed.

The good news:

- It is local.
    
- It has clear mechanisms.
    
- It likely follows from CRT agreement plus Irving correction.
    
- It avoids previous entropy leak if established.
    

The bad news:

- It is not merely a few lines from fixed cutoff dense agreement.
    
- Large labels / modular wrap require a more sophisticated formulation.
    

---

# 11. What to do next?

There are two possible next directions:

### Option A：Develop modular agreement theorem

Try to prove the strong local theorem:

[  
\text{low energy on dense CRT block pair/triple}  
\Rightarrow  
\text{common CRT residue }h  
]

with energy controlling deviations.

This is mathematically direct but hard.

### Option B：Return to known agreement-testing machinery

Now that the needed theorem is clearly an agreement theorem over CRT restrictions, this may be exactly where abstract agreement/list-agreement tools become useful. Earlier we avoided them, but now the shape suggests they may be appropriate.

The structure is:

- local views: residues modulo primes;
    
- pair tests: CRT glue small;
    
- global object: an integer/residue (h);
    
- low energy: approximate agreement.
    

This is agreement testing in a weighted metric.

I think the right next move is to formulate this as an agreement-testing problem and see if known theorems can handle weighted approximate labels, or at least guide proof.

But if we continue self-contained, we should attack Option A in a simplified exact/modular version first.

## 下一步建議

先證 **exact modular three-level agreement**：

If for all edges in (P_0P_1\cup P_1P_2) the CRT glues are equal modulo the relevant products / have zero discrepancy, then there exists (h\bmod L_{012}) explaining all vertices.

Then extend to approximate/low-energy.

Exact version should be easy and will clarify the right modular formulation.
