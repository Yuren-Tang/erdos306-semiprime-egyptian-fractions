# Bounded list-collapse lemma

設

[  
P\subset[X,2X],\qquad Q\subset[Y,2Y],  
]

其中 (Y\ge X)。給每個頂點一個 ordinary integer label：

[  
\lambda_p\in[-B,B]\quad(p\in P),  
]

[  
\mu_q\in[-B,B]\quad(q\in Q).  
]

假設

[  
B<Y/2,\qquad X^2>2B.  
]

稱邊 (pq) 為 label-good，若

[  
\lambda_p\equiv \mu_q\pmod p  
]

且

[  
\lambda_p\equiv \mu_q\pmod q.  
]

等價地，(\lambda_p) 和 (\mu_q) 在兩端模數上都表示同一個 CRT label。

若 label-good 邊在完整二部圖 (P\times Q) 中密度至少

[  
1-\varepsilon,  
]

則存在一個整數

[  
h\in[-B,B]  
]

使得除了 (O(\varepsilon)) 比例的頂點外，

[  
\lambda_p=h,\qquad \mu_q=h.  
]

---

## 1. 先看 (Q)-側：regular 高點

稱 (q\in Q) 為 regular，若它連到至少

[  
(1-\alpha)|P|  
]

個 label-good 低點，其中 (\alpha>0) 是固定小常數，例如 (\alpha=0.01)。

因為總 bad edge 比例是 (\varepsilon)，非 regular 的 (q) 數量至多

[  
\frac{\varepsilon}{\alpha}|Q|.  
]

所以除了 (O(\varepsilon)|Q|) 個 (q)，都是 regular。

---

## 2. regular 高點的 label 實際唯一

取 regular (q)。若 (p) 是與 (q) 相連的 label-good 低點，則

[  
\lambda_p\equiv \mu_q\pmod q.  
]

但

[  
|\lambda_p|,|\mu_q|\le B<q/2.  
]

因此

[  
\lambda_p=\mu_q  
]

作為普通整數。

所以對 regular (q)，它的大多數 good 鄰居 (p) 都滿足

[  
\lambda_p=\mu_q.  
]

---

## 3. 兩個 regular 高點必有同一 label

取兩個 regular 高點

[  
q_1,q_2\in Q.  
]

它們各自有至少 ((1-\alpha)|P|) 個 good 鄰居。因此其 good 鄰居集合交集大小至少

[  
(1-2\alpha)|P|.  
]

對 (X) 足夠大，這個交集至少含有一個 (p)，甚至很多個。

若 (p) 同時是 (q_1,q_2) 的 good 鄰居，則由上一節：

[  
\lambda_p=\mu_{q_1}  
]

且

[  
\lambda_p=\mu_{q_2}.  
]

因此

[  
\mu_{q_1}=\mu_{q_2}.  
]

所以所有 regular 高點共享同一 label，記為

[  
h.  
]

這一步甚至不需要 (X^2>2B)，因為 (B<Y/2) 已讓高點 (q) 直接識別 ordinary label。

---

## 4. 大多數 (P)-側也有 label (h)

若 (p\in P) 與某個 regular (q) 有 label-good 邊，則

[  
\lambda_p=\mu_q=h.  
]

所以不滿足 (\lambda_p=h) 的 (p)，必須對所有 regular (q) 都不是 label-good。

regular (q) 佔

[  
1-O(\varepsilon)  
]

比例。因此每個這樣的 bad (p) 造成

[  
\gg |Q|  
]

條 bad edges。

總 bad edges 只有

[  
\varepsilon |P||Q|.  
]

所以這類 (p) 的數量至多

[  
O(\varepsilon)|P|.  
]

因此除了 (O(\varepsilon)|P|) 個低點外，

[  
\lambda_p=h.  
]

---

## 5. 大多數 (Q)-側也有 label (h)

regular (q) 已經全部滿足

[  
\mu_q=h.  
]

非 regular (q) 只有

[  
O(\varepsilon)|Q|  
]

個。因此除了 (O(\varepsilon)|Q|) 個高點外，

[  
\mu_q=h.  
]

Lemma 證畢。

---

# 6. 這個 lemma 與 CRT glues 的關係

剛才的形式假設每個頂點已經有 label (\lambda_p,\mu_q)。在我們的 CRT 問題中，這些 labels 來自某個能量層的 edge glues。

更實際的使用方式是：

- 若對多數邊 (pq)，CRT glue (H_{pq}^{*}) 落在某個 bounded label set ([-B,B])；
    
- 那麼對 regular 高點 (q)，所有 incident good edge glues 必相等，因為它們都同餘於 (a_q\pmod q)，且 (|H|\le B<q/2)；
    
- 因此高點獲得 label (\mu_q)；
    
- 再用上述 list-collapse 證所有 regular 高點 label 相同；
    
- 低點跟隨。
    

這正是 dense block agreement lemma 的 list-collapse 核心。

---

# 7. (X^2>2B) 何時需要？

在剛才這個 vertex-label 版本中，因為高點 (q>B)，共享一個高點已足夠使 incident edge labels 相等。

但若我們要比較兩個高點 (q_1,q_2)，只靠一個共同低點 (p) 也已經足夠，因為該低點 label 已經是 ordinary integer (\lambda_p)。如果沒有預先給低點 ordinary label，而只知道兩個高點 labels 同餘 mod (p)，則需要兩個低點乘積：

[  
p_1p_2>2B.  
]

所以 (X^2>2B) 是用於「edge-glue 直接版本」；在「vertex labels 已存在」版本裏可弱化。

這有助於後面多層 proof：只要某側能先被 ordinary label 化，collapse 就容易。

---

# 8. 對 multi-threshold 的意義

bounded list-collapse 已證明：

[  
\boxed{  
\text{在固定 label 尺度 }B\text{ 內，多個 labels 不會在稠密二部圖中共存。}  
}  
]

所以 list-agreement 的困難只剩：

1. 如何把低能 glues 分配到某個 label 尺度 (B)；
    
2. 若需要更大 (B)，如何讓其能量付費；
    
3. 如何跨尺度把不同 (B)-labels 合併。
    

第一點由 adaptive cutoff / Markov 給。  
第二點由 (h^2)-energy 給。  
第三點由 overlapping triples + product of primes 給。

這是一個合理的完整藍圖。

# 9. 下一步

接下來要把 bounded list-collapse 與 adaptive cutoff 組合，形成一個正式的 **bounded-energy dense agreement lemma**：

若

[  
\mathcal E_{P,Q}\le R,  
]

則存在 label (h) 解釋大多數頂點，且

[  
|h|\ll XY\sqrt{R/(|P||Q|)}  
]

除非 (R) 已足夠大，可以直接支付該 block 中所有未標記自由度。

我們已經證了大半；下一步要把「除非 (R) 足夠大」說準，避免 raw entropy 再漏出。
