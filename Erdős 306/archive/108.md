對，這裏確實又出現「塔」。但這次不是任意加複雜度，而是被一個很具體的不等式逼出來的。

原來我們想用固定相鄰層

[  
P_j\leftrightarrow P_{j+1}  
]

直接看出 label。這只在能量層 (R) 足夠小時可行。因為 adaptive cutoff 是

[  
B_R\asymp X_jX_{j+1}  
\sqrt{\frac{R}{|P_j||P_{j+1}|}}.  
]

若 (B_R) 小於高層素數大小 (X_{j+1})，這一對層能「看見」 ordinary label；若 (B_R) 大於 (X_{j+1})，這一層就看不清 label，必須升到更高素數尺度。

所以「塔」的功能是：

[  
\boxed{  
\text{對能量層 }R\text{，升到足夠高處，使 label 變得可見。}  
}  
]

然後一旦高處有了 label，就用 Irving correction 往下拉回所有低層。

這是比「每層都要自己 visible」更強、更省力的想法。

# Energy-layer anchoring lemma

現在正式寫這個新核心。

取尺度塔：

[  
P_0,P_1,\dots,P_J,  
]

其中

[  
P_j\subset[X_j,2X_j],  
\qquad  
X_{j+1}=X_j^\gamma,  
\qquad  
\gamma=\frac43.  
]

對任一能量層 (R)，選 (J=J(R)) 使最上方 pair

[  
P_{J-1}\times P_J  
]

對 (R) 可見。

---

## 1. 可見條件

對 pair

[  
P_{j-1}\times P_j  
]

有

[  
X_j=X_{j-1}^{4/3}.  
]

平均能量若為 (R)，adaptive cutoff 大約是

[  
B_R(j)  
\asymp  
X_{j-1}X_j  
\sqrt{  
\frac{R}{|P_{j-1}||P_j|}  
}.  
]

因為

[  
|P_{j-1}||P_j|  
\asymp  
\frac{X_{j-1}X_j}{k_j^2},  
]

所以

[  
B_R(j)  
\asymp  
k_j\sqrt{R X_{j-1}X_j}.  
]

可見條件是

[  
B_R(j)<cX_j.  
]

也就是

[  
k_j\sqrt{R X_{j-1}X_j}<cX_j.  
]

平方後：

[  
k_j^2 R X_{j-1}X_j < c^2 X_j^2.  
]

故

[  
R < c\frac{X_j}{k_j^2X_{j-1}}.  
]

因

[  
X_j=X_{j-1}^{4/3},  
]

所以

[  
\frac{X_j}{X_{j-1}}=X_{j-1}^{1/3}.  
]

因此 pair (j-1,j) 對能量層 (R) 可見，只要

[  
R\ll \frac{X_{j-1}^{1/3}}{k_j^2}.  
]

隨 (j) 上升，這個可見閾值迅速增大。

---

## 2. 選錨點層

給定 (R)，選最小 (J) 使

[  
R\le c\frac{X_{J-1}^{1/3}}{k_J^2}.  
]

那麼對最上 pair

[  
P_{J-1}\times P_J  
]

，若其局部能量不超過 (R)，adaptive dense agreement 可產生 ordinary label

[  
h  
]

解釋大多數

[  
P_{J-1}\cup P_J.  
]

而且

[  
Q_{\mathrm{diag},J-1,J}(h)\lesssim R.  
]

這就是錨點 label。

---

## 3. 若最上 pair 本身能量超過 (R)

若整體 assignment 有

[  
Q(a)\le R,  
]

那麼任何子 pair 能量都不可能超過 (R)。所以對能量層

[  
Q(a)\le R  
]

的 assignments，最上 pair 的局部能量自動 (\le R)。因此 visible agreement 一定可用。

這點很重要：我們不必擔心最上 pair energetic；因為我們按總能量層選 (J(R))。

---

## 4. 從錨點向下傳播

現在已知：

[  
P_{J-1},P_J  
]

大多數 vertices 滿足

[  
a_p\equiv h\pmod p.  
]

要控制 (P_{J-2})。對任一

[  
q\in P_{J-2}  
]

若

[  
a_q\not\equiv h\pmod q,  
]

則用 labelled block (P_{J-1}) 做 Irving correction。

相位差是

[  
\frac{(a_q-h)\bar p}{q}  
\quad\text{或}\quad  
\frac{(a_q-h)\bar p}{q}  
]

依方向略有記號差異，但本質是 inverse residues of primes in (P_{J-1}) modulo (q).

尺度檢查：

[  
P_{J-1}\sim X_{J-1}=X_{J-2}^{4/3}.  
]

對 modulus (q\sim X_{J-2})，prime variable size 是 (X_{J-1}=q^{4/3})，在 Irving 範圍

[  
q^{2/3}\le x\le q^{3/2}  
]

內。故 correction 成立。

所以每個 wrong vertex in (P_{J-2}) 付出

[  
\gg |P_{J-1}|  
]

的能量，遠大於其 residue entropy。

因此低能 assignments 中，除能量可支付的少數例外外，

[  
P_{J-2}  
]

也必須服從同一 (h).

---

## 5. 繼續下降

重複同樣論證：

若 (P_j) 已被 label (h) 控制，則 (P_{j-1}) 中任何不服從 (h) 的 vertex 由 (P_j) correction 付款。

因此 label 從錨點一路下降：

[  
P_J,P_{J-1},P_{J-2},\dots,P_0.  
]

這樣，對能量層 (R)，整座塔都被同一 (h) 控制，例外 vertices 的熵由 Irving energy 支付。

---

# 6. 這如何解決「層變塔」問題？

它把「所有層都要自己看得見」改成：

[  
\boxed{  
\text{只需要在足夠高的一層看見一次；其餘層靠 Irving correction 拉齊。}  
}  
]

這非常關鍵。

原先的孤立層 entropy leak 是因為某低層 pair 對當前 (R) 看不見 label。現在不要求它看見；只要上方錨點看見 label，低層若不跟從，就會在與上方 labelled block 的邊上付巨大能量。

---

# 7. Counting consequence

對固定能量層 (R)：

1. 選 (J=J(R)).
    
2. 最上 visible pair 給一個 label (h).
    
3. 所有下方 vertices 除能量支付的例外外由 (h) 決定.
    
4. 因此 assignments 數量至多
    

[  
#{h:Q_{\mathrm{diag}}(h)\lesssim R}  
\cdot e^{\varepsilon R}.  
]

其中

[  
e^{\varepsilon R}  
]

來自 exceptions 的 residue choices 和集合 choices；由於每個 exception 的 energy/entropy 比可隨 (k_0) 變大，(\varepsilon) 可任意小。

這正是我們想要的 global counting inequality。

---

## 8. 還要控制高於錨點的層嗎？

如果 full graph 包含比 (P_J) 更高的層，那些也要控制。方法有兩種：

### 做法一：能量層 (R) 的塔只取到頂層 (P_J)，證明只在這個子圖上進行

Fourier full graph 的 character 包括更高層 residues，但若我們構造時最高層就是 (P_J)，則無問題。不過 (J) depends on (R)，而圖不能隨 (R) 變。

### 做法二：固定一個很高的 (K)，對每個 (R) 選錨點 (J(R))，然後同時向下與向上校正

若 (P_J) 已 labelled，則向上 (P_{J+1},P_{J+2},\dots) 也可用 Irving correction：

若 (q\in P_{J+1}) 不服從 (h)，用 (P_J) labelled block correction。尺度 (P_{J+1}=P_J^{4/3})，仍在 Irving 範圍。

所以 label 可從錨點向兩側傳播，控制全 scale range。

這很好。

---

## 9. 因此固定大圖也可行

給定 energy layer (R)，選錨點 (J(R)) inside fixed scale range。只要 fixed (K) 足夠大，使所有 relevant (R) 都有 visible anchor before (K).

For very large (R) beyond top visibility, (e^{-cR}) is so small that raw entropy of remaining characters may be paid directly. Choose (K) such that top visibility threshold exceeds, say, total log number of characters in graph. Then all relevant (R) have anchor.

Since (K) can be chosen enormous while still finite, this is possible.

---

## 10. This is a strong repair

The tower is not a complication for its own sake. It gives a clean solution to intermediate energy invisibility:

[  
\text{Energy layer }R  
\Rightarrow  
\text{choose visible anchor scale}  
\Rightarrow  
\text{global label}  
\Rightarrow  
\text{exceptions paid}.  
]

This may finally close the entropy gap.

# 11. Next formal lemma

We can now state:

**Energy-layer anchoring proposition.**  
For every (R) in the relevant range, there exists a scale (J(R)) such that any character with (Q(a)\le R) has a global label (h) on all good vertices, except for a set of vertices whose entropy is at most (\varepsilon R). Moreover

[  
Q_{\mathrm{diag}}(h)\le C R.  
]

Then immediately:

[  
#{a:Q(a)\le R}  
\le  
e^{\varepsilon R}  
#{h:Q_{\mathrm{diag}}(h)\le C R}.  
]

This is the desired counting inequality.
