# 中心命題：Finite-scale weighted CRT list-agreement theorem

取一段有限 dyadic prime blocks：

[  
\mathcal P=\bigcup_{k_0\le k\le K}P_k^\ast,  
]

其中 (P_k^\ast) 是 Irving-good pruning 後的 prime block。

令 compatible graph 為

[  
E=  
\bigcup_{\frac23\ell\le k\le\frac32\ell}  
P_k^\ast\times P_\ell^\ast.  
]

對每個 character assignment

[  
a=(a_p)_{p\in\mathcal P},  
\qquad a_p\in\mathbb F_p,  
]

定義 Fourier energy

# [  
Q(a)

\sum_{pq\in E}  
|\phi_{pq}(a)|^2,  
]

其中

# [  
\phi_{pq}(a)

\frac{a_p\bar q}{p}  
+  
\frac{a_q\bar p}{q}.  
]

對 diagonal character (h) 定義

[  
a_p\equiv h\pmod p  
]

以及

# [  
Q_{\mathrm{diag}}(h)

\sum_{pq\in E}  
\left|\frac{h}{pq}\right|^2.  
]

我們真正需要的命題是：

> 對任意 (\varepsilon>0)，只要 (k_0) 足夠大，就有  
> [  
> #{a:Q(a)\le R}  
> \le  
> e^{\varepsilon R}  
> #{h\bmod L:Q_{\mathrm{diag}}(h)\le C R}  
> ]  
> 對所有 (R\ge0) 成立，其中  
> [  
> L=\prod_{p\in\mathcal P}p,  
> ]  
> (C) 是絕對常數。

這就是我們需要的 **weighted CRT list-agreement theorem**。

## 1. 它為什麼正好夠用？

因為 Fourier minor arc 要估：

[  
\sum_{a\notin\mathfrak M}e^{-cQ(a)}.  
]

若有上面的 counting bound，則按能量層分解：

[  
\sum_a e^{-cQ(a)}  
\le  
\sum_R  
e^{-cR}  
#{a:R\le Q(a)<R+1}.  
]

用中心命題：

[  
#{a:Q(a)\le R}  
\le  
e^{\varepsilon R}  
#{h:Q_{\mathrm{diag}}(h)\le C R}.  
]

只要

[  
\varepsilon<c/10,  
]

非對角的額外 entropy 就被吸收。剩下就是 diagonal (h)-sum，即一維 Gaussian / aperiodic 部分，已由 major arc 和 diagonal tail 控制。

所以中心命題一旦成立，Section 9 就閉合。

## 2. 它包含了我們所有之前的局部機制

這個中心命題不是憑空來的。它應由四個局部機制組合證明：

### A. Adaptive dense block agreement

若一個 block pair 的平均能量小到在該尺度可見，則大多數 vertices 來自一個 ordinary label (h)，且

[  
Q_{\mathrm{diag}}(h)\lesssim Q_{\mathrm{block}}.  
]

### B. Bounded list-collapse

在固定 label 尺度 (B) 內，多個 labels 不能在 complete bipartite block 中共存；否則 cross edges 造成 bad energy。這一點我們已基本證出。

### C. Multiscale visibility

若 label 太大，低尺度看不見；但在更高尺度會變可見。  
這要求利用全 finite scale interval 的多層 compatible graph，而不是一條固定 backbone。

### D. Irving vertex correction

一旦某個 block 或區域已有 label (h)，任何頂點不服從

[  
a_p\equiv h\pmod p  
]

都會在 incident edges 上付出大量 energy，足以支付其 residue entropy。

這四者合起來正是 list-agreement theorem 的內容。

## 3. 目前哪些部分真的已經穩？

可以較有信心的：

1. **Irving correction**：已核對 Irving theorem 形式有 (\sum_q\max_a)，且尺度 (x=Q^{3/4}) 有 power saving。這一塊可寫成外部引理。
    
2. **Fixed (B) dense agreement / list-collapse**：完整二部圖中 bounded labels 塌縮為單一 label，這是初等的。
    
3. **Major arc Gaussian**：最大權重 (o(\sigma))，三次項可控，主項正。
    
4. **Mass and span**：質量發散、格點 span 可用 lattice-fixing edges 修正。
    

仍未證的核心：

[  
\boxed{  
\text{Adaptive / multiscale list-agreement counting theorem。}  
}  
]

這是整條路線的真正剩餘難點。

## 5. 接下來最合理的研究路線

現在應該停止擴大主證明，集中攻中心命題。具體分三步：

### 第一步：證一個簡化版

限制 (h) 在 ordinary visible range：

[  
|h|<\min_{p\in\mathcal P}p^{1+\delta}  
]

或乾脆限制所有 local glues 都小於某固定 (B)。  
這就是我們已經基本證出的 bounded agreement。

### 第二步：加入 dyadic label levels

把 possible label size 分層：

[  
2^r\le |h|<2^{r+1}.  
]

對每個 (r)，選足夠高的 blocks 使該 label level visible。證明該層 labels 會 collapse。

### 第三步：能量求和

大 label level 的 diagonal energy 是

[  
\asymp h^2\sum\frac1{p^2q^2}.  
]

證明 label-level entropy

[  
\log 2^r  
]

被 quadratic energy 支付；多 labels / exceptions 由 cross-edge energy 和 Irving correction 支付。

這樣應能得到

[  
e^{\varepsilon R}  
]

型 counting bound。

# 7. 下一步我會做什麼？

我會先寫 **dyadic label-level collapse lemma**，也就是第二步的核心：

固定一個 label size range

[  
H\le |h|<2H.  
]

選所有 blocks with primes (\gg H^\alpha) so that labels of size (H) are visible. Then prove:

If many vertices locally look like labels in this range and many edges are low-energy, then all such local labels collapse to one global (h), except energy-paid exceptions.
