對，這一段看起來確實比剛才的全體 non-diagonal agreement 更可攻。原因是 diagonal 情形中所有 residues 本來就來自同一個 (h\bmod L)，所以少了一層「不同頂點任意亂選」的熵。現在把它整理成正式命題。

---

# Diagonal aperiodicity proposition

令

[  
E_{\mathrm{full}}  
]

為 compatible semiprime edge graph，並令

[  
L=\prod_{p\in\mathcal P}p.  
]

對

[  
h\in\mathbb Z/L\mathbb Z  
]

定義 diagonal energy

# [  
E(h)

\sum_{pq\in E_{\mathrm{full}}}  
\left|  
\frac{h}{pq}  
\right|^2,  
]

其中

[  
\left|\frac{h}{pq}\right|  
]

理解為 (h\bmod pq) 的對稱代表除以 (pq)。

主弧為

# [  
\mathfrak M

{h:\ |h|_{\mathrm{sym}}\le C/\sigma}.  
]

要證：

# [  
\sum_{h\notin\mathfrak M}  
\exp(-cE(h))

o(1/\sigma).  
]

---

## 1. 能量層分解

按

[  
R\le E(h)<R+1  
]

分層。若能證 counting bound

[  
#{h:E(h)\le R,\ h\notin\mathfrak M}  
\le  
e^{\varepsilon R},  
#{m\in\mathbb Z:\ m^2\sigma^2\le C R,\ |m|>C/\sigma}  
+e^{-\Omega(R)}\text{ 型小項},  
]

則求和後由 Gaussian tail 得

[  
o(1/\sigma).  
]

但對 diagonal 情形，可更直觀地證：若 (E(h)\le R)，則 (h) 要麼等於一個 ordinary 小整數 (m) modulo almost all primes，要麼例外 primes 付能量。最後 (m) 的數量由

[  
m^2\sigma^2\lesssim R  
]

控制。

---

## 2. 從低能到 visible small-glue

固定一個 block pair

[  
P\subset[X,2X],\qquad Q\subset[Y,2Y].  
]

其局部 diagonal energy 是

# [  
E_{P,Q}(h)

\sum_{p\in P,q\in Q}  
\left|  
\frac{h}{pq}  
\right|^2.  
]

若

[  
E(h)\le R,  
]

則

[  
E_{P,Q}(h)\le R.  
]

取 adaptive cutoff

[  
B_{P,Q}(R)  
\asymp  
XY\sqrt{\frac{R}{|P||Q|}}.  
]

Markov 給出：除了小比例邊外，

[  
|h\bmod pq|_{\mathrm{sym}}\le B_{P,Q}(R).  
]

若

[  
B_{P,Q}(R)<Y/2,\qquad B_{P,Q}(R)<X^2/2,  
]

則 diagonal small-glue collapse lemma 給出 ordinary label

[  
m_{P,Q},\qquad |m_{P,Q}|\le B_{P,Q}(R),  
]

使

[  
h\equiv m_{P,Q}\pmod p  
]

對大多數

[  
p\in P\cup Q  
]

成立。

---

## 3. 可見 block pair 的 labels 合併

若兩個可見 block pairs 重疊，例如

[  
P_iP_j,\qquad P_jP_k,  
]

它們分別給出

[  
m_{ij},\quad m_{jk}.  
]

二者都解釋大多數 (P_j)，所以

[  
m_{ij}\equiv m_{jk}\pmod p  
]

對許多 (p\in P_j) 成立。

若

[  
m_{ij}\ne m_{jk},  
]

則差值被很多 primes 整除。只要取足夠多共同 primes 的乘積超過

[  
|m_{ij}-m_{jk}|,  
]

就得

[  
m_{ij}=m_{jk}.  
]

而 (|m_{ij}|)、(|m_{jk}|) 都由 (R) 和相應尺度控制。若差值大到不能被這些 primes 比較，則 label 本身巨大，其 diagonal energy 也大，已被 (R) 計入。

所以在能量 (\le R) 的範圍內，所有 visible local labels 合併成一個 ordinary integer

[  
m.  
]

---

## 4. 例外 primes 的處理

若某個 prime vertex (q) 不滿足

[  
h\equiv m\pmod q,  
]

但它有一個相鄰 block (P) 其中大多數 (p) 滿足

[  
h\equiv m\pmod p,  
]

則對邊 (pq)，

[  
\frac{h}{pq}  
\equiv  
\frac{m}{pq}  
+  
\frac{(h-m)\bar p}{q}  
\pmod1.  
]

令

[  
d=h-m\not\equiv0\pmod q.  
]

Irving correction 給出：

[  
\sum_{p\in P}  
\left|  
\frac{d\bar p}{q}  
\right|^2  
\gg |P|.  
]

而

[  
m/(pq)  
]

在可見範圍內為小擾動，因此

[  
\sum_{p\in P}  
\left|  
\frac{h}{pq}  
\right|^2  
\gg |P|.  
]

所以每個例外 prime (q) 付能量

[  
\gg |P|.  
]

其選擇熵只有

[  
\log q,  
]

可由能量支付。

因此，低能 (h) 只能在極少數、能量可支付的 primes 上不等於 (m)。

---

## 5. 若沒有例外，則 (h) 就是 (m)

若

[  
h\equiv m\pmod p  
\quad\forall p\in\mathcal P,  
]

則

[  
h\equiv m\pmod L.  
]

所以作為 character，

[  
h=m.  
]

如果

[  
|m|\le C/\sigma,  
]

就是 major arc；如果

[  
|m|>C/\sigma,  
]

則 ordinary Gaussian energy

[  
E(m)\asymp m^2\sigma^2  
]

使其在 diagonal tail 中衰減。

這說明：diagonal high-frequency 若不等於小 major (m)，必須通過例外 primes 或 visible collapse failure 支付能量。

---

# 6. 仍須精確處理的點

這個 proposition 的證明還有三個技術點：

### (i) 對每個 (R)，visible block pairs 是否足夠覆蓋整個圖？

若某些最高 blocks 對該 (R) 沒有 visible higher pair，則用已 labelled lower block + Irving correction 處理。這在 diagonal 情形應可行。

### (ii) labels 的合併是否會因 (m) 太大失效？

若 (m) 太大，則它的 diagonal energy (m^2\sigma^2) 也大。能量層 (R) 會限制

[  
|m|\lesssim \sqrt R/\sigma.  
]

因此只需用足夠多中間 primes，其乘積超過 (\sqrt R/\sigma)。在大尺度 blocks 中這很容易，除非 (R) 極大；極大 (R) 由 Fourier 衰減處理。

### (iii) 例外 primes 的 correction 邊是否重疊？

可為每個例外 prime 選一個固定相鄰 labelled block。不同例外 vertex 的 edge stars disjoint；跨尺度 overlap 有界。故可處理。

---

# 7. 對角命題的可信度

相比全 non-diagonal CRT Poincaré，diagonal aperiodicity 可信度明顯更高，因為：

- local label 本來就是 (h) 的小代表；
    
- 不存在多個 vertex 自由 residue 的巨大熵；
    
- 例外 prime 用 Irving correction 直接殺掉；
    
- local labels 合併由 CRT product rigidity 控制。
    

所以這部分大概可以被完整寫成一個嚴格證明。

---

# 8. 對整體主證明的意義

若 diagonal aperiodicity 成立，則剩下的唯一大難點是 non-diagonal weighted CRT list-agreement。

但這個 diagonal 子結果也會反過來幫 non-diagonal：它說一旦 non-diagonal assignment 被壓到 diagonal family，所有高頻對角都會自動消失，不需要另用黑箱 local CLT。
