# Diagonal energy-layer counting lemma

令

[  
E(h)=\sum_{pq\in E_{\mathrm{full}}}  
\left|\frac{h}{pq}\right|^2.  
]

我們想證：對任意 (R)，滿足

[  
E(h)\le R  
]

的 diagonal characters (h\bmod L)，基本上都由一個 ordinary integer (m) 控制；例外 primes 的選擇由能量支付。

形式上希望得到：

[  
#{h\bmod L:E(h)\le R}  
\le  
e^{\varepsilon R}  
#{m\in\mathbb Z:E_{\mathrm{diag}}(m)\le CR}.  
]

然後主弧外求和即可由 Gaussian tail 控制。

---

## 1. 固定 (R)，選可見尺度

對一個 block pair

[  
P\sim X,\qquad Q\sim Y=X^{4/3},  
]

若

[  
E(h)\le R,  
]

則局部能量也 (\le R)。取 adaptive cutoff：

# [  
B_R(X)

CXY\sqrt{\frac{R}{|P||Q|}}  
\asymp  
Ck\sqrt{RXY}.  
]

該 pair 可見若

[  
B_R(X)<cY.  
]

等價於

# [  
R\ll \frac{Y}{Xk^2}

\frac{X^{1/3}}{k^2}.  
]

所以對固定 (R)，只要選

[  
X\gg (Rk^2)^3,  
]

該 pair 就可見。

因為我們的圖有許多尺度，只要存在這樣的 pair，就可在該 pair 上套 diagonal small-glue collapse。

---

## 2. 在可見 pair 上得到小代表 (m)

對可見 pair (P\times Q)，由 Markov：

除了 (O(\delta)) 比例的邊外，

[  
|h\bmod pq|_{\mathrm{sym}}\le B_R(X)  
]

其中 (\delta) 固定小。

由 diagonal small-glue collapse，存在

[  
m,\qquad |m|\le B_R(X),  
]

使得除了 (O(\delta)) 比例外，

[  
h\equiv m\pmod p  
]

對

[  
p\in P\cup Q  
]

成立。

所以 (m) 是 (h) 在該可見 pair 上的 ordinary representative。

---

## 3. (m) 的能量受 (R) 控制

因為

[  
|m|\le B_R(X),  
]

而

[  
B_R(X)  
\asymp  
XY\sqrt{\frac{R}{|P||Q|}},  
]

所以

[  
|P||Q|  
\left(\frac{m}{XY}\right)^2  
\lesssim R.  
]

也就是

[  
E_{P,Q}^{\mathrm{diag}}(m)\lesssim R.  
]

但我們最終需要 full diagonal energy：

# [  
E_{\mathrm{full}}^{\mathrm{diag}}(m)

\sum_{uv\in E_{\mathrm{full}}}  
\left|\frac{m}{uv}\right|^2  
]

受 (R) 控制。

這需要利用下一步：若 (h\equiv m) 在所有 primes 上，則 (E(h)=E(m)\le R)。若有例外，例外由能量支付。扣除例外後，full diagonal energy of (m) 應該 (\lesssim R)。

---

## 4. 從可見 pair 向全圖傳播 (m)

設

[  
S_m={p\in\mathcal P: h\equiv m\pmod p}.  
]

我們已知 (S_m) 在某個可見 pair 的兩個 blocks 中佔大多數。

現在若某個相鄰 block (P') 中有 vertex

[  
q\notin S_m,  
]

即

[  
d=h-m\not\equiv0\pmod q,  
]

而 (q) 與一個已知大多數在 (S_m) 中的 labelled block (P) 相連，則對 (p\in P\cap S_m)：

[  
\frac{h}{pq}  
\equiv  
\frac{m}{pq}  
+  
\frac{d\bar p}{q}  
\pmod1.  
]

由 Irving correction，

[  
\sum_{p\in P\cap S_m}  
\left|  
\frac{d\bar p}{q}  
\right|^2  
\gg |P|.  
]

若 (m/(pq)) 足夠小，這直接給

[  
\sum_{p\in P}  
\left|\frac{h}{pq}\right|^2  
\gg |P|.  
]

但若 (m/(pq)) 不小，則 diagonal energy of (m) itself is already large on these edges. Either way,

[  
\sum_{p\in P}  
\left|\frac{h}{pq}\right|^2  
+  
\sum_{p\in P}  
\left|\frac{m}{pq}\right|^2  
\gg |P|.  
]

這是更穩的 correction inequality：wrong vertex forces either (h)-energy or (m)-energy.

因為 (E(h)\le R)，太多 wrong vertices 不可能；否則 energy 超過 (R). 同時若 (m)-energy很大，則 (m) 不會被計入 (E_{\mathrm{diag}}(m)\le CR) 的 label set。

所以傳播可行。

---

## 5. 例外 primes 的計數

對每個例外 prime (q\notin S_m)，correction energy 至少約

[  
E_q\gg |P_{\mathrm{adj}}|  
]

除非 (m) 的 diagonal energy支付。

該 (q) 對 (h\bmod L) 的自由度是：

[  
h\equiv \text{anything not }m\pmod q,  
]

約 (q) 種，entropy

[  
\log q.  
]

因

[  
E_q\gg |P_{\mathrm{adj}}|\gg \varepsilon^{-1}\log q  
]

for large (k_0)，所以例外 primes 的 residue choices 和集合 choices 被

[  
e^{\varepsilon R}  
]

控制。

因此：

[  
h  
]

由

1. ordinary representative (m);
    
2. 一個能量可支付的例外 prime set;
    
3. 例外 residues;
    

決定。

所以

[  
#{h:E(h)\le R}  
\le  
e^{\varepsilon R}  
#{m:E_{\mathrm{diag}}(m)\le CR}.  
]

這就是 diagonal counting lemma。
