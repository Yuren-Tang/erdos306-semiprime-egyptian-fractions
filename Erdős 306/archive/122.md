好。現在做 Peierls 框架的下一步：**contour entropy bound**。

前一步我們得到局部邊界估計：

若兩個相鄰 blocks (P,Q) 分屬不同 phase (h,h')，且這個差異在 (Q) 或 (P) 上 genuinely visible，即

[  
h'\not\equiv h\pmod q  
]

對正比例 (q\in Q) 成立，則跨邊能量至少

[  
\gg |P||Q|.  
]

這比我們之前 cutoff 型成本大得多。現在要看它能不能支付 contour 的所有熵。

---

# 1. Contour 是什麼？

在尺度—頂點混合圖中，先想一個簡化版本：

- 大多數 vertices 處於 exterior phase (h)；
    
- 某個 connected cluster (C) 處於另一 phase (h')；
    
- (C) 與外部之間的 compatible edges 構成 boundary。
    

在 block 層面，(C) 可能包含若干 prime blocks 的部分或全部 vertices。最簡潔地先把 contour 粗化到 block level：

[  
\mathcal C={P_{k_1},\dots,P_{k_s}}  
]

是一組 blocks，內部 label 為 (h')，外部 label 為 (h)。

真正情形中，一個 block 內也可能只有部分 vertices 屬於 (h')。那反而更容易，因為部分 defects 可用 single-site Irving correction 付費。最危險的是整個 block 或大比例 block 都變成另一 phase，因為這樣單點 correction 不夠，要靠 boundary energy。

---

# 2. Contour 選擇熵

若尺度範圍是

[  
k_0\le k\le K,  
]

block 數約

[  
K-k_0.  
]

選一個 block-level contour 的熵至多

[  
\exp(O(|\partial \mathcal C|\log K))  
]

或更粗

[  
2^{K-k_0}.  
]

但 boundary block pair 的能量成本是：

[  
\sum_{(P,Q)\in\partial\mathcal C}  
c|P||Q|.  
]

對最低尺度邊界 pair，也有

[  
|P||Q|\asymp \frac{XY}{k^2}.  
]

這遠大於 (\log K)，只要 (k_0) 大。因此 block contour 的選擇熵可忽略。

也就是：

[  
\text{block contour entropy}  
\ll  
\text{boundary energy}.  
]

---

# 3. Phase label 熵

內部 phase (h') 的選擇數可能很多。要估計它的熵。

若 (h') 是一個 ordinary label of size roughly (H)，那麼 label entropy 是

[  
\log H.  
]

Boundary energy 若 genuine，是

[  
\gg |P||Q|  
]

per boundary block pair，通常遠大於 (\log H)，除非 (H) 接近整個 modulus (L) 級別。

但如果 (h') 是 global diagonal high phase，則它不是 contour entropy，而應進入 diagonal phase sum：

[  
\sum_{h'}e^{-cQ_{\mathrm{diag}}(h')}.  
]

Peierls expansion 通常是「相對於每個 pure phase (h) 展開 defects」。因此不應用 boundary energy 支付所有 (h') 的全局數量；應該：

- 外部 phase (h) 在最外層 diagonal sum 中計數；
    
- 內部 droplet phase (h') 的選擇若不同於 (h)，boundary energy支付；
    
- 若 (h') 巨大但 droplet 很小，single-site/cross energy更大；
    
- 若整個系統都是 (h')，那不是 droplet，而是另一個 pure phase，交給 diagonal sum。
    

所以 label entropy 沒有致命問題。

---

# 4. Vertex-level contour entropy

若一個 block (P) 中只有一部分 vertices 屬於 droplet，選擇這些 vertices 的 entropy 是

[  
\log \binom{|P|}{m}  
\le  
m\log\frac{e|P|}{m}.  
]

每個 such vertex 若和外部 labelled block 相連且不服從 exterior label，Irving correction 給成本

[  
\gg |Q|  
]

或

[  
\gg |P_{\mathrm{adj}}|.  
]

對大尺度，

[  
|Q|\gg \log |P|+\log p.  
]

所以 vertex-level subset entropy 由 single-site defect cost 支付。

因此最危險的 contour 可粗化為「整塊 block 改相」，而這時 block-level boundary energy 很大。

---

# 5. Internal disorder entropy

Droplet 內部也許不是 clean phase (h')，而是亂的。那麼有兩種情況：

1. 內部大多數可由某個 (h') 解釋；
    
2. 不能由單一 (h') 解釋。
    

第二種情況在內部相鄰 blocks 上會產生同樣的 list-agreement / boundary energy。換言之，內部 disorder 可以遞歸分解成更小 contours 或 single-site defects。

這正是 cluster expansion 的遞歸結構。只要每個 genuine interface 都有 Peierls cost，總 entropy 可控。

---

# 6. Contour entropy bound：形式

可期待證：

對任意 (T)，所有 contour families (\Gamma) with total Peierls cost

[  
\mathcal C(\Gamma)=T  
]

的數量滿足

[  
#{\Gamma:\mathcal C(\Gamma)=T}  
\le  
e^{\varepsilon T}  
]

for any fixed (\varepsilon>0), if (k_0) is sufficiently large.

原因：

- 每個 contour boundary unit has cost at least (\gg |P||Q|);
    
- 描述該 boundary unit 只需 (O(\log K+\log H)) bits;
    
- vertex exceptions have cost (\gg) their residue/choice entropy;
    
- large labels either are pure phases or are paid by boundary cost.
    

這是 Peierls entropy estimate。

---

# 7. 結合 Peierls 能量

若對每個 contour family 有

[  
Q(a)  
\ge  
Q_{\mathrm{diag}}(h)  
+  
c\mathcal C(\Gamma),  
]

且

[  
#{\Gamma:\mathcal C(\Gamma)=T}  
\le e^{\varepsilon T},  
]

則

[  
\sum_{a\text{ near phase }h}e^{-c_0Q(a)}  
\le  
e^{-c_1Q_{\mathrm{diag}}(h)}  
\sum_T e^{-c_2T}e^{\varepsilon T}.  
]

取 (\varepsilon<c_2/2)，得到

[  
\sum_{a\text{ near phase }h}e^{-c_0Q(a)}  
\le  
C e^{-c_1Q_{\mathrm{diag}}(h)}.  
]

再對 (h) 求和：

[  
\sum_a e^{-c_0Q(a)}  
\le  
C\sum_h e^{-c_1Q_{\mathrm{diag}}(h)}.  
]

這正是我們想要的 partition function domination。

然後去掉主弧：

[  
\sum_{a\notin\mathfrak M}e^{-c_0Q(a)}  
\le  
C\sum_{h\notin\mathfrak M}e^{-c_1Q_{\mathrm{diag}}(h)}  
+  
\text{small contour errors}.  
]

若 diagonal tail 已控，就完成 minor arc。

---

# 8. 但注意：這裏仍依賴 diagonal tail

Peierls 把 non-diagonal partition function 壓到 diagonal partition function。  
但 diagonal high-frequency still needs control.

不過現在 diagonal tail 可以被視為 pure phase sum，而不再混在 contour entropy裏。這大幅簡化。

也就是整體分工變成：

1. **Peierls theorem**：non-diagonal excitations (\le) diagonal phase sum。
    
2. **Diagonal aperiodicity theorem**：diagonal phase sum outside major arc is (o(1/\sigma)).
    

這是清晰且合理的結構。

---

# 9. Contour entropy bound 的可信度

這部分現在看起來比較可信，因為 two-phase boundary energy 是常數相位級：

[  
\gg |P||Q|,  
]

而不是弱 cutoff 級。

這足以壓住：

- block contour 選擇；
    
- vertex subset 選擇；
    
- residue choices；
    
- local label choices.
    

因此 non-diagonal Peierls theorem 有希望。

真正剩下的硬點反而是 diagonal aperiodicity。  
不過 diagonal aperiodicity可以作為一維問題單獨研究。

---

# 10. 下一步

現在應該寫 **Peierls domination proposition**：

[  
\sum_{a}e^{-cQ(a)}  
\le  
C\sum_h e^{-c'Q_{\mathrm{diag}}(h)}.  
]

先不證 diagonal tail，只證 non-diagonal dominated by diagonal.

這是一個乾淨命題，且由：

- single-site Irving correction；
    
- two-phase boundary energy；
    
- contour entropy bound；
    

組成。
