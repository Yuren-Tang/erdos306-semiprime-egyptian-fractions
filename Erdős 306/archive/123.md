好。現在寫 **Peierls domination proposition**。它的作用是把整個 non-diagonal minor arc 壓到 diagonal family 上。

# Peierls domination proposition

令

[  
Q(a)=\sum_{pq\in E_{\mathrm{full}}}|\phi_{pq}(a)|^2  
]

為 general character energy。令 diagonal family 為

[  
\mathcal D={a^{(h)}:\ a_p\equiv h\pmod p\ \forall p}.  
]

其中

[  
h\in \mathbb Z/L\mathbb Z.  
]

對角能量為

# [  
Q_{\mathrm{diag}}(h)=Q(a^{(h)})

\sum_{pq\in E_{\mathrm{full}}}  
\left|\frac{h}{pq}\right|^2.  
]

希望證明：

[  
\boxed{  
\sum_a e^{-cQ(a)}  
\le  
C  
\sum_{h\bmod L}e^{-c'Q_{\mathrm{diag}}(h)}.  
}  
]

更強地，對每個主相 (h)：

[  
\sum_{a\text{ in basin of }h}  
e^{-cQ(a)}  
\le  
C e^{-c'Q_{\mathrm{diag}}(h)}.  
]

這就是「非對角激發被 Peierls 缺陷成本壓住」。

---

## 1. Basin of a phase (h)

要使命題可證，需要給每個 assignment (a) 分配一個或少數可能的 phase (h)。

理想方式：

選使

[  
Q(a,a^{(h)})  
]

或某種 mismatch cost 最小的 (h)。但直接定義可能麻煩。

更實用：在 proof 中不必唯一分配 basin；可以用 union bound over (h)。對每個 (h)，考慮 deviations from (h)：

[  
D_h(a)={p:\ a_p\not\equiv h\pmod p}.  
]

若 (D_h(a)) 小或有清楚 contours，就估計其貢獻。對所有 (h) 求和，多重計數只給常數或由 energy 支付。

所以 Peierls 估計可寫成：

[  
\sum_a e^{-cQ(a)}  
\le  
\sum_h e^{-cQ_{\mathrm{diag}}(h)}  
\sum_{\Gamma}  
e^{-c\mathrm{Cost}_h(\Gamma)}  
\cdot \mathrm{Entropy}(\Gamma).  
]

若

[  
\sum_{\Gamma}e^{-c\mathrm{Cost}_h(\Gamma)}  
\mathrm{Entropy}(\Gamma)  
\le C  
]

uniformly in (h)，則命題成立。

---

## 2. Single-site defect polymer

對固定 (h)，一個 vertex (q) 是 defect 若

[  
a_q\ne h\pmod q.  
]

若 (q) 附近有一個 block (P) 多數非 defect，則 Irving correction 給：

[  
\mathrm{Cost}(q)\gg |P|.  
]

其 entropy：

- 選擇 (q)：(\log |P_{\mathrm{level}(q)}|)；
    
- 選擇 residue (a_q)：(\log q).
    

總共 (O(\log q+\log |P_j|)\asymp O(k_j))。

而

[  
|P|\asymp \frac{X}{k}  
\gg k.  
]

所以 single-site polymers are summable.

This is solid.

---

## 3. Large defect droplet

若 defects form a large connected droplet (C), they may internally follow another label (h'). Then single-site correction against (h) only applies at boundary, not inside.

So define a contour boundary:

[  
\partial C={pq:\ p\in C,\ q\notin C,\ pq\in E}.  
]

If (C) has phase (h'\ne h), then for a boundary block pair (P\subset C,\ Q\subset C^c), two-phase boundary lemma gives:

[  
\mathrm{Cost}_{P,Q}(h,h')\gg |P||Q|  
]

unless (h\equiv h') modulo almost all boundary primes, in which case the phases are actually not different there and the contour can be merged.

Thus genuine droplet boundaries have enormous cost.

---

## 4. Boundary entropy

A block-level boundary component is described by:

- set of involved block indices;
    
- approximate vertex subset if partial;
    
- internal phase label (h');
    
- possible smaller internal defects.
    

For a whole-block droplet, block index entropy is at most exponential in number of boundary blocks:

[  
\exp(O(|\partial_{\mathrm{block}} C|\log K)).  
]

Boundary cost per block pair is at least

[  
\gg |P||Q|.  
]

Even at the lowest relevant scale,

[  
|P||Q|\gg \log K.  
]

Thus block-shape entropy is negligible.

For partial vertex boundaries, choosing vertices costs

[  
\le \exp(O(m\log |P_j|)),  
]

but each boundary/defect vertex costs (\gg |P_{\mathrm{adj}}|), so this is also negligible.

For internal phase (h'), if (C\ne) whole system, boundary energy pays the choice of (h') unless (h') is indistinguishable from (h) on boundary, in which case it is not a genuine contour.

This is the usual Peierls dichotomy.

---

## 5. Recursive contours

Inside a droplet phase (h'), there may be smaller droplets of phase (h''), etc. The contour family (\Gamma) is a collection of nested contours.

Peierls expansion estimates:

[  
\sum_{\Gamma}  
\prod_{\gamma\in\Gamma} w(\gamma)  
]

where

[  
w(\gamma)  
\le  
\exp(-c\mathrm{Cost}(\gamma)+\mathrm{Entropy}(\gamma)).  
]

If

[  
\mathrm{Entropy}(\gamma)\le \varepsilon\mathrm{Cost}(\gamma)  
]

with (\varepsilon<c), then

[  
w(\gamma)\le e^{-c'\mathrm{Cost}(\gamma)}.  
]

Since minimal contour cost tends to infinity with (k_0), the polymer expansion converges absolutely.

We do not need full statistical mechanics machinery if we accept a crude tree bound: number of contour families of total cost (T) is (\le e^{\varepsilon T}), so

[  
\sum_T e^{-cT}e^{\varepsilon T}<\infty.  
]

---

## 6. Main Peierls inequality

Thus, for fixed exterior phase (h),

[  
\sum_{a\text{ with exterior phase }h}  
e^{-cQ(a)}  
\le  
e^{-c'Q_{\mathrm{diag}}(h)}  
\sum_T e^{-c''T}e^{\varepsilon T}  
\le  
C e^{-c'Q_{\mathrm{diag}}(h)}.  
]

Summing over (h):

[  
\sum_a e^{-cQ(a)}  
\le  
C\sum_h e^{-c'Q_{\mathrm{diag}}(h)}.  
]

This is the Peierls domination proposition.

---

# 7. What is rigorous and what remains sketch?

### Rigorous or nearly rigorous:

1. **Single-site defect cost** via Irving correction.
    
2. **Two-phase boundary cost** via Irving applied to (D=h'-h).
    
3. **Bounded list-collapse** in complete bipartite blocks.
    
4. **Entropy per vertex/block** is much smaller than the corresponding Irving boundary cost.
    

### Still sketch-level:

1. **Canonical contour decomposition**: how exactly to assign each (a) to phases and contours without overcounting.
    
2. **Handling indistinguishable labels**: if (h-h') divisible by many boundary primes, decide whether to merge regions or charge large label energy.
    
3. **Nested contour bookkeeping**: can be done, but not yet written.
    
4. **Diagonal high-frequency tail**: still separate.
    

So this proposition is plausible and now much better founded, but not yet a complete proof.

---

# 8. Interaction with diagonal tail

Once Peierls domination is granted, minor arc reduces to:

[  
\sum_{h\notin\mathfrak M}  
e^{-c'Q_{\mathrm{diag}}(h)}  
=o(1/\sigma).  
]

This is still needed.

But Peierls domination means we no longer have to control arbitrary non-diagonal characters separately. That is a huge reduction.

---

# 9. Next target

We now have two remaining central lemmas:

1. **Peierls domination lemma** — plausible, needs contour formalization.
    
2. **Diagonal aperiodicity lemma** — still hard, one-dimensional but number-theoretic.
    

Which should we attack next?

I think next we should push diagonal aperiodicity again, but now with a better viewpoint: since diagonal tail is pure phase sum, we can try to prove it using a divisor/CRT expansion or known weighted Bernoulli LCLT criteria.

However, before that, it may be worth writing a precise statement of the final reduction:

[  
\text{Main theorem}  
\Leftarrow  
\text{Major arc}  
+  
\text{Peierls domination}  
+  
\text{Diagonal aperiodicity}  
+  
\text{Irving}.  
]

This will clarify exactly what remains open.
