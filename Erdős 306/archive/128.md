是，這個問題確實是核心；不是枝節。更精確地說，我們反覆撞到的是同一個母題：

[  
\boxed{  
\text{大 label / 高頻 / 無序 residue 的巨大熵，是否必被 CRT 邊能量壓住？}  
}  
]

每次它換一件衣服出現：

一開始叫 **minor arc 熵—能量 bookkeeping**；  
後來叫 **isolated block entropy leak**；  
再後來叫 **adaptive cutoff / large label visibility**；  
再後來叫 **top entropy problem**；  
再後來叫 **diagonal high-frequency aperiodicity**；  
現在又以 **small-phase Peierls 不能處理大相** 的形式出現。

所以你說得對：這不是某個證明技巧的小障礙，而是全問題的核心難點之一。

但這不等於我們只是在徒勞繞圈。比較準確的判斷是：

[  
\boxed{  
\text{我們不是繞開核心，而是在逐步判定哪些框架會錯誤地低估它。}  
}  
]

例如：

- 單一 cutoff 方案失敗，因為它把大 label 當成「bad」但成本太低；
    
- 單鏈 backbone 方案失敗，因為 isolated block 的 raw entropy 太大；
    
- top anchor 方案失敗，因為加高尺度也帶來新的高層 entropy；
    
- small-phase-only Peierls 方案失敗，因為大 diagonal phase 不能被粗暴當 defect。
    

這些失敗都在告訴我們一件事：**大 label 不能當作普通缺陷處理；它必須先被允許作為一個 diagonal phase，然後再單獨證 diagonal phases 的高頻尾部小。**

所以現在比較穩的分解應該是：

[  
\boxed{  
\text{非對角問題：用 Peierls domination 壓到所有 diagonal phases。}  
}  
]

[  
\boxed{  
\text{對角高頻問題：另證 diagonal aperiodicity。}  
}  
]

這不是逃避，而是把兩種不同性質的熵分開。非對角熵是「頂點 residues 任意亂選」的熵；對角熵是「一個全局 (h\bmod L)」的一維高頻熵。它們不應用同一把刀砍。

## 1. 哪個纔是真正核心？

現在可以說：真正核心有兩層。

第一層是 **Peierls domination**：

[  
\sum_a e^{-cQ(a)}  
\le  
C\sum_{h\bmod L} e^{-c'Q_{\mathrm{diag}}(h)}.  
]

這句話說：所有非對角自由度都不如 diagonal phase 重要；非對角激發的熵會被 Irving correction 與 boundary energy 壓住。

這一層看起來相當有希望，因為我們有強局部估計：

[  
\text{單點 defect cost}\gg |P|,  
]

[  
\text{兩相 boundary cost}\gg |P||Q|.  
]

第二層是 **diagonal aperiodicity**：

[  
\sum_{h\notin\mathfrak M} e^{-cQ_{\mathrm{diag}}(h)}  
=o(1/\sigma).  
]

這句話說：即使允許所有全局 (h\bmod L)，除了小整數 (h) 之外，高頻 (h) 的總貢獻仍小。

這一層更接近一維 weighted Bernoulli local CLT 的高頻問題。它是最硬的、最獨立的剩餘問題。

---

## 2. 繞開是對的嗎？

對 **非對角部分**，繞開 small-phase-only 框架是對的。因為大 label 本來就是 legitimate diagonal phase；把它當 defect 會強行製造不必要的困難。

對 **diagonal high-frequency**，不能再繞。這是無法消去的核心。除非換一條完全不同的主路線，否則 Fourier/local CLT 方法一定要處理它。

所以現在的判斷是：

[  
\boxed{  
\text{Peierls 可繞開非對角熵；但 diagonal aperiodicity 不能繞，必須正面證。}  
}  
]

---

# 3. 接下來應繼續哪一邊？

我建議現在繼續 Peierls domination over all diagonal phases，而不是立刻又回 diagonal tail。原因是 Peierls domination 已經接近閉合；它一旦成立，就把整個 non-diagonal 世界清掉，剩下一個乾淨的一維問題。這樣整體路線會清楚得多。

接下來要把 Peierls domination 寫成更正式的「相分解」：

- 每個頂點 residue (a_p) 總可以被任意 diagonal phase (h) 解釋為 (h\bmod p)；但一個 assignment 未必接近同一個 (h)。
    
- 若不同區域接近不同 (h,h')，邊界由 two-phase boundary lemma 付費。
    
- 若某區域不接近任何單一 diagonal phase，則它內部必有很多局部不一致，dense agreement/list-collapse 付費。
    
- 最後所有 configuration 被分解為一個 global phase (h) 加 contours。
    
- contours 的總權重可求和。
    

這裏大 label 不再是問題，因為 (h,h') 可以是任意 (L)-diagonal phase。若 (h) 本身高頻，這不由 Peierls 處理，而留給 diagonal sum。

---

# 4. Peierls domination 的下一個精確版本

我們現在應證如下命題：

**Proposition P.**  
存在常數 (c,c'>0)，使

[  
\sum_{a} e^{-cQ(a)}  
\le  
C  
\sum_{h\bmod L} e^{-c'Q_{\mathrm{diag}}(h)}.  
]

更局部地，對每個 (h),

[  
\sum_{a}  
e^{-cQ(a)}  
\mathbf 1{a\text{ is assigned to phase }h}  
\le  
C e^{-c'Q_{\mathrm{diag}}(h)}.  
]

這裏「assigned to phase (h)」可不必唯一；只要每個 (a) 被覆蓋有限次或覆蓋次數由能量支付即可。

---

## 5. 如何定義 phase assignment？

可用一個 coarse majority rule：

對每個 block (P_k)，若存在 (h) 使

[  
a_p\equiv h\pmod p  
]

對大多數 (p\in P_k) 成立，稱該 block 有 phase (h)；否則稱為 disordered block。

但是任意 residues 對整個 block 總可由某個 (h\bmod \prod_{p\in P_k}p) 解釋，所以這裏的 (h) 不能是 arbitrary modulo full local product，否則 phase entropy太大。應該讓 local phase 由相鄰 block-pair 的低能 CRT glues決定，而不是由單個 block 的 CRT 任意拼接決定。

更自然：

- 在 block pair (P_kP_\ell) 上，若能量低，adaptive agreement 產生 local edge phase (h_{k\ell})。
    
- 相鄰 local phases 若不同，two-phase boundary pays.
    
- 沒有低能 local phase的 pair，是 disorder edge and pays energy.
    

這是 agreement-testing/Peierls 的混合版本。

---

# 6. 這裏我們不再要求 (h_{k\ell}) 小

這是關鍵改動。以前 bounded dense agreement需要 (|h|\le B) 可見；現在允許 arbitrary (h\bmod L). 那麼如何從低能 edge glues產生 arbitrary local phase？

如果 (H_{pq}^{*}) 小，產生的是小 ordinary glue；如果 phase (h) 很大但 (h\bmod pq) 小，仍可產生小 representative on that edge. Across many edges，這些 small representatives會合併成某個 local residue class (h) modulo product of involved primes。它不必是小 globally。

也就是 local phase 應該是：

[  
h_{U}\in \mathbb Z/M_U\mathbb Z  
]

而非 ordinary integer。這正是 CRT agreement 的自然語言。

如此一來，大相被正確表示為模某個大 product 的 phase，而不是被錯判為無序。

---

# 7. 這是否又變難？

是，但這是正確困難。  
Peierls domination over all phases 需要 **modular agreement theorem**：

若 (P\times Q) 上多數 edges 的 CRT glues are mutually consistent，則存在一個 residue class

[  
h\bmod M_{P,Q}  
]

解釋大多數頂點；local phases在重疊處合併。

Exact 版由 CRT 基本成立；approximate 版由 dense agreement/list-collapse支撐。

這就是我們反覆靠近的核心工具。不同的是，現在它不需控制 label 大小，只需控制 local phase 熵；全局 phase 熵留給右側的 (\sum_h)。

這更合理。

---

# 8. 目前可前進的一步

先證一個 **modular exact/list collapse**：

若兩個 block-pair local phases (h_1\bmod M_1), (h_2\bmod M_2) 在重疊 prime set 上同意多數 residues，則它們合併成一個 phase modulo lcm/product；若不同，cross boundary pays energy.

這避免 ordinary size問題。

然後 Peierls contours are disagreements between modular phases, not necessarily small integer labels.

This should close non-diagonal domination.

你問「是否核心而我們一直繞開」。答案是：

是，**高頻／大相熵**是核心；  
但把它分成「非對角大相」和「對角高頻」是對的。

現在正確的策略是：

[  
\boxed{  
\text{非對角：modular Peierls domination 到所有 diagonal phases。}

[  
\boxed{  
\text{對角：單獨 aperiodicity。}  
}  
]

我們接下來繼續推第一個，因為它現在不再被 small-label visibility 卡住。
