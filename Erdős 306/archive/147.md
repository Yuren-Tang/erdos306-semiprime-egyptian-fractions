# Single-block counting theorem

令

[  
P\subset[X,2X],\qquad N=|P|\asymp \frac{X}{\log X}.  
]

對 residues

[  
a=(a_p)_{p\in P},\qquad a_p\in\mathbb F_p,  
]

定義同尺度 complete graph energy：

# [  
Q_P(a)

\sum_{\substack{p,q\in P\p<q}}  
|\phi_{pq}(a)|^2.  
]

其中

# [  
\phi_{pq}(a)

\frac{a_p\bar q}{p}  
+  
\frac{a_q\bar p}{q}  
\pmod1.  
]

令

# [  
\sigma_P^2

\sum_{p<q}\frac1{p^2q^2}  
\asymp  
\frac1{X^2(\log X)^2}.  
]

目標是證 counting form：

[  
#{a:Q_P(a)\le R}  
\le  
e^{\varepsilon R}  
#{m\in\mathbb Z:m^2\sigma_P^2\le C R}.  
]

這會推出

[  
Z_P=\sum_a e^{-cQ_P(a)}  
\ll \frac1{\sigma_P}  
\asymp X\log X.  
]

## 1. Energy layer cutoff

若

[  
Q_P(a)\le R,  
]

取 cutoff

# [  
B_R

# A X^2\sqrt{\frac{R}{N^2}}

A\frac{X^2}{N}\sqrt R  
\asymp  
A X\log X\sqrt R.  
]

由 Markov，若 (A) 夠大，則 bad edges

[  
|H_{pq}^{*}|>B_R  
]

的比例至多

[  
O(A^{-2}).  
]

取 (A) 大，使 good edge density 至少

[  
1-\eta.  
]

對 good edge 有

[  
|H_{pq}^{*}|\le B_R.  
]

## 2. 選 regular base vertex

在 good graph 中，除了 (O(\sqrt\eta N)) 個 vertices 外，每個 vertex 的 good degree 至少

[  
(1-\sqrt\eta)N.  
]

取一個 regular base vertex

[  
p_0.  
]

定義 base list：

# [  
\mathcal L

{n\in[-B_R,B_R]\cap\mathbb Z:  
n\equiv a_{p_0}\pmod{p_0}}.  
]

其大小為

[  
|\mathcal L|  
\le  
1+\frac{2B_R}{p_0}  
\ll  
1+\log X\sqrt R.  
]

若 (q) 與 (p_0) 是 good edge，則

[  
H_{p_0q}^{*}\in\mathcal L  
]

且

[  
H_{p_0q}^{*}\equiv a_q\pmod q.  
]

所以大多數 vertices (q) 被 (\mathcal L) 覆蓋：存在

[  
m\in\mathcal L  
]

使

[  
a_q\equiv m\pmod q.  
]

這是 list decoding。  
我們已把每個 vertex 的 (q) 種 residue 自由度壓到一個短 list (\mathcal L)。

## 3. 將 vertices 分入 label classes

## 4. Cross-label energy

取兩個不同 labels

[  
m\ne m'\in\mathcal L.  
]

令

[  
D=m'-m.  
]

對

[  
p\in C_m,\qquad q\in C_{m'},  
]

有

[  
a_p\equiv m\pmod p,\qquad a_q\equiv m'\pmod q.  
]

邊相位為

[  
\phi_{pq}  
\equiv  
\frac{m}{pq}  
+  
\frac{D\bar p}{q}  
\pmod1.  
]

若

[  
D\not\equiv0\pmod q,  
]

則由 Irving-good 性質，對固定 (q),

[  
\sum_{p\in C_m}  
\left|  
\frac{D\bar p}{q}  
\right|^2  
]

應該有下界。但 Irving 原本給的是對整個 (P) 求和，不是任意 subset (C_m)。

這裏要小心：若 (C_m) 是任意稀疏 subset，不能直接用 Irving。  
但若 (C_m) 是大類，比如

[  
|C_m|\ge \delta N,  
]

我們需要一個 subset 版 equidistribution，或改用整個 regular majority class。

這是第一個技術點。
