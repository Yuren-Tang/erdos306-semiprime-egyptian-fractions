好。現在正式寫 **cross-label divisor-energy lemma**，並給出證明骨架。

# Cross-label divisor-energy lemma

令

[  
P\subset[X,2X]  
]

是一個 prime block。取兩個 subsets

[  
A,B\subset P,  
]

以及兩個不同 ordinary labels

[  
m\ne m',  
\qquad |m|,|m'|\le B_0.  
]

令

[  
D=m'-m\ne0.  
]

對

[  
p\in A,\qquad q\in B  
]

定義 cross CRT representative

[  
H_{pq}^{*}\in(-pq/2,pq/2]  
]

滿足

[  
H_{pq}^{_}\equiv m\pmod p,  
\qquad  
H_{pq}^{_}\equiv m'\pmod q.  
]

cross energy 為

# [  
E(A,B;m,m')

\sum_{p\in A,q\in B}  
\left(\frac{H_{pq}^{*}}{pq}\right)^2.  
]

我們希望證：

[  
\boxed{  
E(A,B;m,m')  
\gg  
\frac{|A||B|}{(\log X)^C}  
}  
]

除非 (A) 或 (B) 很小，或者 (D) 被異常多的 (p,q) 整除；後者在 (|D|\le 2B_0) 下也可由 divisor bound 控制。

這個估計的意義是：兩個不同 label classes 之間的跨邊，平均相位不能普遍很小。即使有 polylog 損失，也足以支付 entropy。

---

# 1. 小 representative 的計數

對 (T\le X^2/2)，令

# [  
N(T)

#{(p,q)\in A\times B:\ |H_{pq}^{*}|\le T}.  
]

若 (|H_{pq}^{*}|\le T)，則存在整數 (n) with

[  
|n|\le T,  
]

且

[  
n\equiv m\pmod p,  
\qquad  
n\equiv m'\pmod q.  
]

因此

[  
p\mid n-m,  
\qquad  
q\mid n-m'.  
]

所以

[  
N(T)  
\le  
\sum_{|n|\le T}  
\omega_A(n-m),\omega_B(n-m'),  
]

其中

[  
\omega_A(t)=#{p\in A:p\mid t}.  
]

---

# 2. 粗 divisor bound

對任意非零整數 (t)，在 ([X,2X]) 中的 prime divisors 個數至多

[  
\omega_X(t)  
\le  
\frac{\log |t|}{\log X}+O(1).  
]

因此

[  
\omega_A(t),\omega_B(t)  
\ll  
1+\frac{\log(|t|+2)}{\log X}.  
]

若 (|n|\le T), (|m|,|m'|\le B_0)，則

[  
|n-m|,\ |n-m'|  
\le T+B_0.  
]

所以

[  
N(T)  
\ll  
T\left(1+\frac{\log(T+B_0)}{\log X}\right)^2  
+  
\text{special terms}.  
]

Special terms occur when

[  
n=m  
]

or

[  
n=m',  
]

because then (n-m=0) or (n-m'=0) is divisible by all primes. But if (n=m), then we also need

[  
q\mid m-m'=D.  
]

Thus special contribution is at most

[  
|A|\cdot \omega_B(D)  
+  
|B|\cdot \omega_A(D).  
]

And

[  
\omega_A(D),\omega_B(D)  
\ll  
1+\frac{\log |D|}{\log X}.  
]

Since

[  
|D|\le2B_0,  
]

this is small unless (B_0) is astronomically large; in our application (B_0) is at most an energy-layer cutoff, so manageable.

Thus

[  
\boxed{  
N(T)  
\ll  
T L_X^2  
+  
(|A|+|B|)L_X  
}  
]

where

[  
L_X=1+\frac{\log(B_0+T+2)}{\log X}.  
]

This is the basic incidence bound.

---

# 3. 從 small-count 到 energy lower bound

令

[  
M=|A||B|.  
]

把

[  
r_{pq}=|H_{pq}^{*}|  
]

視為 (M) 個數，位於 ([0,X^2/2])。

能量

[  
E(A,B;m,m')  
\asymp  
\frac1{X^4}  
\sum_{p,q}r_{pq}^2,  
]

因為

[  
pq\asymp X^2.  
]

如果很多 (r_{pq}) 很小，則 (N(T)) 大。反過來，用 (N(T)) 上界可推出許多 (r_{pq}) 不小。

取一個 threshold (T_*) 使

[  
N(T_*)\le \frac12 M.  
]

則至少 (M/2) 個 pairs 有

[  
r_{pq}>T_*,  
]

故

[  
E(A,B;m,m')  
\gg  
M\frac{T_*^2}{X^4}.  
]

由 incidence bound，取

[  
T_*  
\asymp  
\frac{M}{L_X^2}  
]

只要 special terms 不超過 (M/4)。同時 (T__\le X^2) 必須成立；若 (T__>X^2)，則可取 (T_*\asymp X^2)，得到 constant energy per edge.

所以

[  
E(A,B;m,m')  
\gg  
M\cdot  
\min\left(  
1,\frac{M^2}{X^4L_X^4}  
\right).  
]

這個下界在 (M) 大時強，在 (M) 小時弱。

---

# 4. 是否足夠？

若 (A,B) 都是正比例大小：

[  
|A|,|B|\gg N\asymp X/\log X,  
]

則

[  
M\asymp \frac{X^2}{\log^2 X}.  
]

因此

[  
\frac{M^2}{X^4L_X^4}  
\asymp  
\frac1{\log^4X,L_X^4}.  
]

所以

[  
E(A,B;m,m')  
\gg  
\frac{X^2}{\log^{6+O(1)}X}.  
]

這極大，遠超過 block entropy

[  
\asymp X.  
]

所以對大 classes，足夠。

若 classes 較小，總 entropy也小；可用 energy-entropy分層處理。

因此即使 divisor bound 有 polylog 損失，也足夠壓制多個 substantial labels 共存。

這很好。

---

# 5. 用於 list-collapse

在 single-block list reduction 中，base list大小

[  
s=|\mathcal L|\ll 1+B_R/X.  
]

若沒有 dominant label，則 classes (C_m) 分散。  
按大小分 dyadic：

[  
\mathcal C_j={m:2^j\le |C_m|<2^{j+1}}.  
]

若總覆蓋 vertices 是 (\gg N)，則存在某個 dyadic size class 產生大量 cross pairs。

對這些 cross pairs，套 cross-label divisor-energy lemma。  
若 cross energy太大，則 (Q_P(a)>R)，矛盾。  
因此低能下，只能有一個 large class；其餘 labels 的 total vertices 必須小。  
小 classes 作 exceptions，由 incident/cross energy支付其 entropy。

這給出 dominant label。

---

# 6. Single-block counting 完成形態

因此，對 (Q_P(a)\le R)：

1. Markov 得 high-density good graph at (B_R).
    
2. Base list reduction 得短 list (\mathcal L).
    
3. Cross-label divisor-energy lemma迫使 (\mathcal L) 中一個 label (m) dominate.
    
4. Exceptions由 cross/Irving energy支付.
    
5. Label choices數量 (\ll B_R\asymp \sqrt R/\sigma_P).
    

得到：

[  
#{a:Q_P(a)\le R}  
\le  
e^{\varepsilon R}  
\frac{\sqrt R}{\sigma_P}.  
]

這就是 single-block counting theorem。

---

# 7. 這一步的技術要求

需要把「exceptions由 energy支付」精寫。大致可以用：

- 若 exception set (D) 大，cross energy between (D) and majority class (\gg |D||G|/\log^C X)，支付 (|D|\log X).
    
- 若 (D) 小，直接以  
    [  
    \binom N{|D|}X^{|D|}  
    ]  
    計數，其 entropy (\ll |D|\log X)，仍由 same cross energy支付。
    

因

[  
|G|\asymp N\asymp X/\log X,  
]

cross energy per exception至少

[  
\gg \frac{N}{\log^C X}  
\gg \log X  
]

對 (X) 大成立。  
所以 exceptions沒問題。
