好。現在處理最後這個一維問題：

[  
\boxed{  
#{m\bmod L:Q_{\mathrm{diag}}(m)\le R}  
\ll  
\frac{\sqrt R}{\sigma}.  
}  
]

這是 **ordinary diagonal counting lemma**。它比原來的 arbitrary CRT high-frequency 小得多，因為現在只有一個整數 (m)，不是每個 prime 各自自由選 residue。

---

# Ordinary diagonal counting lemma

令

[  
L=\prod_{p\in\mathcal P}p.  
]

對

[  
m\in\mathbb Z/L\mathbb Z  
]

定義 diagonal energy：

# [  
Q_{\mathrm{diag}}(m)

\sum_{pq\in E}  
\left|  
\frac{m}{pq}  
\right|^2.  
]

要證：

[  
#{m\bmod L:Q_{\mathrm{diag}}(m)\le R}  
\ll  
e^{\varepsilon R}\frac{\sqrt R}{\sigma}.  
]

或至少在 partition sense：

[  
\sum_{m\bmod L}e^{-cQ_{\mathrm{diag}}(m)}  
\ll \frac1\sigma.  
]

---

## 1. 小 ordinary representative 情形

若 (m) 的對稱代表滿足

[  
|m|\le c\min_{pq\in E} pq,  
]

則沒有 wrap-around：

# [  
\left|\frac{m}{pq}\right|

\frac{|m|}{pq}.  
]

因此

# [  
Q_{\mathrm{diag}}(m)

# m^2  
\sum_{pq\in E}\frac1{p^2q^2}

m^2\sigma^2.  
]

所以

[  
Q_{\mathrm{diag}}(m)\le R  
\quad\Rightarrow\quad  
|m|\le \sqrt R/\sigma.  
]

這給出預期數量：

[  
\ll \sqrt R/\sigma.  
]

問題只在於 large (m\bmod L) 可能在很多 (pq) 上 wrap，使 (m\bmod pq) 小。

---

## 2. 同尺度 block 會抓住 wrap

考慮一個 internal complete block

[  
P\subset[X,2X].  
]

若

# [  
Q_P^{\mathrm{diag}}(m)

\sum_{p<q\in P}  
\left|  
\frac{m}{pq}  
\right|^2  
\le R_P,  
]

則由 single-block counting theorem applied to diagonal assignment

[  
a_p\equiv m\pmod p  
]

可知：存在 ordinary representative

[  
m_P,\qquad |m_P|\lesssim \frac{\sqrt{R_P}}{\sigma_P}  
\asymp X\log X\sqrt{R_P},  
]

使

[  
m\equiv m_P\pmod p  
]

對大多數 (p\in P)。

但因 (m) 本身就是同一個 residue class，這表示：

[  
p\mid m-m_P  
]

對大多數 (p\in P)。

若這些 primes 的乘積超過

[  
|m-m_P|  
]

則

[  
m=m_P  
]

作為 ordinary integer representative。  
如果 (m) 很大，未必能直接比較；但跨多個 blocks 可以把 product 放大。

---

## 3. 用多 blocks 的 product rigidity

若對許多 blocks (P_k) 都有 local representative (m_k)，且跨 block label collapse 已證

[  
m_k=m_{k'}=m_0  
]

對多數 blocks 成立，則

[  
m\equiv m_0\pmod p  
]

對大量 primes (p\in\mathcal P) 成立。

令

[  
S={p:m\equiv m_0\pmod p}.  
]

則

[  
\prod_{p\in S}p \mid m-m_0.  
]

若

[  
\prod_{p\in S}p>L/2  
]

或者至少超過 (|m-m_0|) 的代表範圍，則

[  
m\equiv m_0\pmod L  
]

up to exceptions.

若有例外 primes (T), those exceptions correspond to blocks/vertices where diagonal assignment does not match (m_0), and internal/cross energy already pays them in the full counting theorem.

For pure diagonal (m), if (m\ne m_0\bmod L), then there must be many primes not in S; their incident edges create energy.

Thus low energy forces (m) to be congruent to one ordinary (m_0) on almost all primes, and exceptions paid.

---

# 4. 對一維 counting 的更簡單觀點

其實 ordinary diagonal counting 可以直接從前面的 full global counting theorem得到，因為 diagonal assignments 是所有 assignments 的子集：

[  
#{m:Q_{\mathrm{diag}}(m)\le R}  
\le  
#{a:Q(a)\le R}.  
]

但在證明結構中，full global counting theorem本身用到了 diagonal label sum。  
為避免循環，我們只需對 pure labels (m) 做單獨估計。

可以這樣做：

1. 用 internal block single-block theorem：對每個 block，(m) 必接近某個 local ordinary (m_k)。
    
2. 用 cross-block label energy：不同 (m_k) 合併。
    
3. 得到 global (m_0)。
    
4. 若 (m\not\equiv m_0\bmod L)，則例外 primes付能量。
    
5. 因此 (m) 由 (m_0) 加能量可支付的例外 primes決定。
    
6. 但 (m) 是一個 single CRT class；例外 primes的 choices同樣被 incident energy支付。
    
7. 所以低能 (m) 數量 (\le e^{\varepsilon R}#m_0).
    

這與 general assignment counting 相同，但更簡單。

---

# 5. 最終 estimate

得到：

[  
#{m\bmod L:Q_{\mathrm{diag}}(m)\le R}  
\le  
e^{\varepsilon R}  
#{m_0\in\mathbb Z:m_0^2\sigma^2\le CR}.  
]

而

[  
#{m_0:m_0^2\sigma^2\le CR}  
\ll  
\frac{\sqrt R}{\sigma}.  
]

因此：

[  
#{m\bmod L:Q_{\mathrm{diag}}(m)\le R}  
\le  
e^{\varepsilon R}  
\frac{\sqrt R}{\sigma}.  
]

This is the desired ordinary diagonal counting.

---

# 6. 這裏是否仍有循環？

要小心：這個 ordinary diagonal proof uses the same internal block collapse and cross-block collapse, but **not** the full arbitrary-assignment global theorem.  
它只用：

- single-block counting/list-Helly for one block;
    
- cross-block label energy;
    
- exception charging.
    

這些是已作為獨立局部引理處理的。因此不循環。

---

# 7. 至此小弧主體閉合

現在我們有以下鏈條：

1. **Single-block counting**：每個 block residue assignment (\leadsto) ordinary block label.
    
2. **Cross-block label energy**：block labels (\leadsto) single global ordinary label.
    
3. **Ordinary diagonal counting**：global ordinary labels have Gaussian counting.
    
4. Therefore:  
    [  
    #{a:Q(a)\le R}  
    \le  
    e^{\varepsilon R}\frac{\sqrt R}{\sigma}.  
    ]
    
5. Therefore:  
    [  
    \sum_a e^{-cQ(a)}  
    \ll  
    \frac1\sigma.  
    ]
    
6. 主弧外：  
    [  
    o(1/\sigma).  
    ]
    

這是我們一直想要的小弧控制。

# 8. 最後還需檢查 mass/variance

加入 internal edges 後：

# [  
\sigma^2

\sum_{\text{internal}}\frac1{p^2q^2}  
+  
\sum_{\text{cross}}\frac1{p^2q^2}.  
]

Internal contribution over block (k):

[  
\sigma_k^2\asymp \frac1{X_k^2k^2}.  
]

由最低 block主導。  
這不壞，反而增加 variance slightly. Major arc scale仍為

[  
1/\sigma\asymp X_{k_0}k_0  
]

up to constants.

Mass contribution internal:

[  
\sum_k\sum_{p<q\in P_k}\frac1{pq}  
\asymp  
\sum_k\frac1{k^2}  
]

收斂；可視為固定小補充。  
Cross edges仍負責讓總 mass達到需要的 (1/b)。可調 (\theta) 或選邊密度處理。

所以結構上沒有破壞。
