# 1. 隨機模型

我們已選好一組 semiprime edges

[

[

[  
\theta\in[c,1-c].

定義

[  
X=\sum_{e=pq\in E}\frac{\xi_e}{pq}.

我們希望

[  
\mathbb P\left(X=\frac1b\right)>0.

# 2. 期望調整

總 reciprocal mass

[  
W=\sum_{pq\in E}\frac1{pq}  
]

可藉由選擇跨尺度 edge range 調整，使

[  
\theta W=\frac1b.  
]

同尺度 internal edges 的 mass 為

[  
\sum_k\sum_{p<q\in P_k}\frac1{pq}  
\asymp  
\sum_k \frac1{k^2},  
]

是收斂的，作為固定補充項，不會妨礙跨尺度部分調節 (W)。  
必要時可把 (\theta) 或某一小批 edge probabilities 作微調，使期望精確等於

[  
\frac1b.  
]

因此

[  
\mathbb E X=\frac1b.  
]

# 5. 主弧

主弧 characters 是 ordinary small labels：

[  
h=m,\qquad |m|\le C/\sigma,  
]

其中

# [  
\sigma^2

\theta(1-\theta)  
\sum_{pq\in E}\frac1{p^2q^2}.  
]

對這些 (m)，展開：

# [  
\log\widehat\mu(m)

\sum_{pq}  
\log\left(  
1-\theta+\theta e(m/pq)  
\right).  
]

因

[  
|m|/(pq)  
]

很小，得：

# [  
\log\widehat\mu(m)

## 2\pi i m\theta\sum_{pq}\frac1{pq}

2\pi^2\theta(1-\theta)m^2  
\sum_{pq}\frac1{p^2q^2}  
+  
o(1).  
]

由

[  
\theta\sum_{pq}\frac1{pq}=\frac1b,  
]

linear phase 正好被

[  
e(-m/b)  
]

抵消。

因此主弧項為：

# [  
\widehat\mu(m)e(-m/b)

\exp(-2\pi^2\sigma^2m^2+o(1)).  
]

所以

# [  
\sum_{|m|\le C/\sigma}  
\widehat\mu(m)e(-m/b)

(1+o(1))  
\sum_{|m|\le C/\sigma}  
e^{-2\pi^2\sigma^2m^2}.  
]

取 (C) 大，這是正的，且

[  
\asymp \frac1\sigma.  
]

# 6. 小弧 bound

對任意 character assignment (a=(a_p))，有標準估計：

[  
|\widehat\mu(a)|  
\le  
\exp(-cQ(a)),  
]

其中

[  
Q(a)=  
\sum_{pq\in E}  
|\phi_{pq}(a)|^2.  
]

前面建立的 partition theorem 給：

[  
\sum_{a\notin\mathfrak M}  
e^{-cQ(a)}  
=o(1/\sigma),  
]

其中

# [  
\mathfrak M

{a_p\equiv m\pmod p,\ |m|\le C/\sigma}.  
]

因此

[  
\sum_{a\notin\mathfrak M}  
|\widehat\mu(a)|  
\le  
o(1/\sigma).  
]

也就是小弧總貢獻比主弧小。

# 8. 從 (1/b) 到 (a/b)

若要表示

[  
\frac ab,  
]

取 (a) 批 pairwise disjoint prime blocks：

[  
\mathcal P^{(1)},\dots,\mathcal P^{(a)}  
]

彼此不相交，且每批都按同樣方法構造一組 semiprime edges (E^{(j)})。

對每一批，得到 disjoint subset

[  
S_j\subset E^{(j)}  
]

使

# [  
\sum_{pq\in S_j}\frac1{pq}

\frac1b.  
]

因各批 primes disjoint，所有 semiprime denominators 互異。相加得到：

# [  
\sum_{j=1}^a\sum_{pq\in S_j}\frac1{pq}

\frac ab.  
]

# 9. 整體證明目前的依賴表

整個方案現在依賴如下：

### 外部數論輸入

Irving-type inverse prime equidistribution：

[  
\sum_{p\in P}  
\left|  
\frac{d\bar p}{q}  
\right|^2  
\gg |P|

for most / pruned good (q), uniformly in (d\ne0).

### 自證局部引理

1. **Single-block counting theorem**  
    同尺度 complete graph 將 block residues 壓縮成 ordinary block label。
    
2. **Cross-label divisor-energy lemma**  
    不同 labels 的 cross glues 不能大量小；用 divisor incidence 證。
    
3. **Cross-block label energy lemma**  
    不同 block labels 在跨尺度邊上付 Irving energy。
    
4. **Block-label Peierls collapse**  
    block labels 合併成單一 global (m)。
    

### Fourier 分析

6. 小弧 partition theorem；
    
7. lattice span fixing；
