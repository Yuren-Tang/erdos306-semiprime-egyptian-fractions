對，這份 Irving 文稿正好補上我們最擔心的一環。看了你傳的 `.tex`，它的主定理之一正是我們需要的平均型 Kloosterman-prime bound。

文中定義

[  
S_q(a;x)=\sum_{\substack{p\sim x\(p,q)=1}} e!\left(\frac{a\overline p}{q}\right),  
]

其中 (p) 跑過 primes。其 Theorem 1 給出

[  
\sum_{q\sim Q}\max_{(a,q)=1}|S_q(a;x)|  
\ll_\varepsilon  
\left(  
Q^{5/4}x^{5/8}  
+Qx^{9/10}  
+Q^{7/6}x^{13/18}  
\right)Q^\varepsilon  
]

在

[  
Q^{3/2}\ge x\ge Q^{2/3}  
]

範圍內成立。

這對我們很有用。特別是在我們最常用的同尺度或近尺度情形

[  
x\asymp Q  
]

時，右側主量約為

[  
Q^{19/10+\varepsilon}.  
]

而一個 prime block 中 primes 的個數是

[  
N\asymp \frac{Q}{\log Q}.  
]

所以平均每個 (q) 的最大 Kloosterman sum 約為

[  
Q^{9/10+\varepsilon},  
]

比

[  
N\asymp Q/\log Q  
]

小一個冪次。這正好足以做 pruning。

---

# 1. 從 Irving 平均估計到 good (q)

固定兩個同尺度或近尺度 blocks：

[  
P\subset[x,2x],\qquad Q\subset[Q,2Q],  
]

先看 (x\asymp Q)。

Irving 給

[  
\sum_{q\sim Q}\max_{(a,q)=1}  
\left|  
\sum_{p\sim x} e_q(a\overline p)  
\right|  
\ll Q^{19/10+\varepsilon}.  
]

設

[  
N_P\asymp x/\log x.  
]

定義 bad (q) 為

[  
\max_{(a,q)=1}  
\left|  
\sum_{p\in P} e_q(a\overline p)  
\right|

\delta N_P.  
]

則 bad (q) 的個數至多

# [  
\ll  
\frac{Q^{19/10+\varepsilon}}{\delta N_P}  
\asymp  
\frac{Q^{19/10+\varepsilon}}{\delta Q/\log Q}

\delta^{-1}Q^{9/10+\varepsilon}\log Q.  
]

而 (Q)-block 中 primes 數約為

[  
N_Q\asymp Q/\log Q.  
]

所以 bad primes 比例是

[  
\ll  
\delta^{-1} Q^{-1/10+\varepsilon}(\log Q)^2=o(1)  
]

只要 (\varepsilon<1/10) 且 (Q\to\infty)。

因此刪去 (o(N_Q)) 個 bad (q) 後，對所有保留的 (q) 都有

[  
\max_{(a,q)=1}  
\left|  
\sum_{p\in P}e_q(a\overline p)  
\right|  
\le \delta N_P.  
]

這正是 pruning 所需的第一步。

---

# 2. 從 exponential sum 小到 dispersion lower bound

現在固定 good (q)。對任意

[  
d\in\mathbb F_q^\times  
]

考慮點集

[  
\left{\frac{d\overline p}{q}:p\in P\right}\subset\mathbb R/\mathbb Z.  
]

因為

[  
d\ne0\pmod q,  
]

乘以 (d) 只是在 Fourier 頻率中換參數。對任意非零 (h)，

# [  
\sum_{p\in P} e!\left(h\frac{d\overline p}{q}\right)

\sum_{p\in P} e_q(hd\overline p).  
]

只要 (hd\not\equiv0\pmod q)，這被 good (q) 條件控制。  
對 (1\le h\le H<q)，皆有

[  
hd\not\equiv0\pmod q.  
]

因此

[  
\left|  
\sum_{p\in P} e!\left(h\frac{d\overline p}{q}\right)  
\right|  
\le \delta N_P.  
]

由 Erdős–Turán discrepancy inequality，可得這組點在 (\mathbb R/\mathbb Z) 上分布接近均勻。取 (H) 固定但足夠大，再取 (\delta) 足夠小，得到：

[  
\sum_{p\in P}  
\left|  
\frac{d\overline p}{q}  
\right|^2  
\ge cN_P  
]

其中 (c>0) 為絕對常數。

直觀上，均勻分布時

[  
\int_0^1 |t|^2,dt=\frac1{12},  
]

所以只要 discrepancy 足夠小，平方距離總和必為 (\gg N_P)。

於是我們正得到所需形式：

[  
\sum_{p\in P}  
\left|  
\frac{d\overline p}{q}  
\right|^2  
\gg |P|.  
]

而且它是 **uniform in (d\ne0)** 的，因為 Irving 定理中有

[  
\max_{(a,q)=1}  
]

這一點非常關鍵。

---

# 3. 同尺度 internal block

同尺度 (P=Q) 時，對

[  
q\in P  
]

我們需要

[  
\sum_{\substack{p\in P\p\ne q}}  
\left|  
\frac{d\overline p}{q}  
\right|^2  
\gg |P|.  
]

上面的估計對全 block (p\in P) 給出 (\gg |P|)。刪掉 (p=q) 一項最多損失 (1/4)，無關緊要。因此 internal block 所需 Irving-good 性質成立。

---

# 4. 跨尺度範圍

文中 Theorem 1 的範圍是

[  
Q^{3/2}\ge x\ge Q^{2/3}.  
]

所以只要我們的 compatible block pairs 滿足

[  
Q^{2/3}\le x\le Q^{3/2},  
]

就能用此定理。

這正與我們之前一直使用的「compatible scale」吻合。特別是：

- 同尺度 (x\asymp Q)：可以；
    
- 相鄰 dyadic scale：可以；
    
- (4/3)-jump：也在範圍內。
    

因此 skeleton block graph 可選在這個範圍內，Irving input 可用。

---

# 5. Pruning across many block pairs

對每個 control block pair ((P_k,P_\ell))，上述 argument 刪去該 pair 中 (o(|P_\ell|)) 的 bad (q)。

若 control skeleton degree bounded，例如每個 block只連接 (O(1)) 或 (O(\log K)) 個 compatible blocks，則對所有相鄰 pairs 取 union bound 後，每個 block只刪去

[  
o(|P_k|)  
]

個 primes。

所以我們得到 pruned blocks

[  
P_k^\ast  
]

滿足：對每個 control pair ((k,\ell))，每個

[  
q\in P_\ell^\ast,  
]

每個

[  
d\in\mathbb F_q^\times,  
]

都有

[  
\sum_{p\in P_k^\ast}  
\left|  
\frac{d\overline p}{q}  
\right|^2  
\gg |P_k^\ast|.  
]

這正是我們所需的 Irving-good pruning lemma。

---

# 6. 結論：強版本可保留

因此，你傳的 Irving 文稿確實足以支持我們原來使用的強型 pruning，至少在 (x\asymp Q) 或 compatible range

[  
Q^{2/3}\le x\le Q^{3/2}  
]

中。

所以不用退回較弱的 dense-majority incidence 替代版。那個替代思路可以保留作備援，但主證明中可以直接引用 Irving theorem 來得到：

[  
\boxed{  
\forall q\in Q^\ast,\ \forall d\ne0\pmod q,\quad  
\sum_{p\in P^\ast}  
\left|  
\frac{d\overline p}{q}  
\right|^2  
\gg |P^\ast|.  
}  
]

這是很好的消息。它讓 single-block exceptions、cross-block label mismatch、block-label Peierls 的關鍵能量下界都站住了。
