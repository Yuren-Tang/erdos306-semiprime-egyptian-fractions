# Irving-good pruning lemma

**Lemma.**  
令

[  
P\subset[x,2x],\qquad Q\subset[Q,2Q]  
]

為 prime blocks，且

[  
Q^{2/3+\delta}\le x\le Q^{3/2}  
]

處於 Irving 定理的有效範圍內。則刪去 (Q)-block 中 (o(|Q|)) 個 primes 後，對每個保留的

[  
q  
]

以及每個

[  
d\in\mathbb F_q^\times  
]

有

[  
\sum_{p\in P}  
\left|  
\frac{d\overline p}{q}  
\right|^2  
\gg |P|.  
]

同理可對每個有限 bounded-degree block skeleton 同時 pruning，使所有 control block pairs 均滿足此性質。

# 證明

由 Irving Theorem 1：

[  
\sum_{q\sim Q}  
\max_{(a,q)=1}  
\left|  
\sum_{p\sim x}e_q(a\overline p)  
\right|  
\ll_\varepsilon  
\left(  
Q^{5/4}x^{5/8}  
+Qx^{9/10}  
+Q^{7/6}x^{13/18}  
\right)Q^\varepsilon.  
]

在同尺度 (x\asymp Q) 下，右側是

[  
\ll Q^{19/10+\varepsilon}.  
]

而

[  
|P|\asymp \frac{x}{\log x}\asymp\frac Q{\log Q}.  
]

給定小常數 (\delta_0>0)，稱 (q) 為 bad 若

[  
\max_{(a,q)=1}  
\left|  
\sum_{p\in P}e_q(a\overline p)  
\right|

\delta_0 |P|.  
]

則 bad (q) 數至多

# [  
\ll  
\frac{Q^{19/10+\varepsilon}}  
{\delta_0 Q/\log Q}

O_{\delta_0}!\left(Q^{9/10+\varepsilon}\log Q\right).  
]

而 (Q)-block 中 primes 數量約為

[  
|Q|\asymp Q/\log Q.  
]

所以 bad 比例為

[  
O_{\delta_0}!\left(Q^{-1/10+\varepsilon}(\log Q)^2\right)=o(1)  
]

取 (\varepsilon<1/10)。刪去 bad primes 後，對每個保留 (q) 有

[  
\max_{(a,q)=1}  
\left|  
\sum_{p\in P}e_q(a\overline p)  
\right|  
\le  
\delta_0 |P|.  
]

接下來固定 (q) 與 (d\ne0\pmod q)。對 (1\le h\le H<q)，

[  
hd\not\equiv0\pmod q,  
]

故

# [  
\left|  
\sum_{p\in P}  
e!\left(h\frac{d\overline p}{q}\right)  
\right|

\left|  
\sum_{p\in P}e_q(hd\overline p)  
\right|  
\le  
\delta_0|P|.  
]

由 Erdős–Turán 或 Weyl discrepancy 估計，取固定大 (H)，再取 (\delta_0) 足夠小，可知點列

[  
\left{\frac{d\overline p}{q}:p\in P\right}  
]

在 (\mathbb R/\mathbb Z) 上不可能集中於 (0) 附近。具體地，

[  
\sum_{p\in P}  
\left|  
\frac{d\overline p}{q}  
\right|^2  
\ge c|P|  
]

其中 (c>0) 為絕對常數。因為均勻分布時

[  
\int_0^1|t|^2,dt=\frac1{12},  
]

而低階 Fourier coefficients 足夠小即可保證此二次平均有正比例下界。

這證出 desired dispersion lower bound。

# 同尺度與多 block pruning

同尺度 internal block 時 (P=Q)。刪去 bad primes 後，對每個

[  
q\in P^\ast  
]

和 (d\ne0) 有

[  
\sum_{\substack{p\in P^\ast\p\ne q}}  
\left|  
\frac{d\overline p}{q}  
\right|^2  
\gg |P^\ast|.  
]

去掉 (p=q) 一項只損失 (O(1))，無礙。

對 bounded-degree skeleton 的多個 block pairs，逐 pair 刪除 bad primes；因每個 block只參與 (O(1)) 或 (O(\log K)) 個 control pairs，union bound 後每個 block刪除比例仍是 (o(1))。得到 pruned blocks (P_k^\ast)，同時滿足所有 control pair 的 Irving-good 性質。
