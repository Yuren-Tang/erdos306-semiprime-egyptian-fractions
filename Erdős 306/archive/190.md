# 1. Clique CRT Helly lemma 的候選命題

給 distinct primes

[  
r_1,\dots,r_t\sim X  
]

以及 lists

[  
L_i={n\in[-B,B]:n\equiv a_i\pmod {r_i}}.  
]

假設任意兩個 lists 都相交：

[  
L_i\cap L_j\ne\varnothing  
\qquad\forall i\ne j.  
]

問：是否推出

[  
\bigcap_i L_i\ne\varnothing?  
]

若這為真，single-block 的 agreement 問題會簡化很多。

---

# 2. 無限 AP 情形為真

若不限制在 ([-B,B])，只看同餘類：

[  
n\equiv a_i\pmod {r_i},  
]

因 (r_i) 兩兩互素，任意 residues 都有全局 CRT 解。  
所以無限同餘類其實總是全體相交，不需要 pairwise 假設。

困難全在 bounded window：

[  
[-B,B].  
]

pairwise intersection 說：每一對同餘條件的 CRT 解有一個小代表。  
但全體 CRT 解的代表可能很大。

---

# 3. 三個模數時可能有反例

對三個 primes (r_1,r_2,r_3)，設 pairwise representatives 為

[  
h_{12},h_{13},h_{23}\in[-B,B].  
]

它們必須滿足 congruence compatibility：

[  
h_{12}\equiv h_{13}\pmod {r_1},  
]

[  
h_{12}\equiv h_{23}\pmod {r_2},  
]

[  
h_{13}\equiv h_{23}\pmod {r_3}.  
]

因此可寫：

[  
h_{13}=h_{12}+\alpha r_1,  
]

[  
h_{23}=h_{12}+\beta r_2,  
]

並要求

[  
\alpha r_1-\beta r_2\equiv0\pmod {r_3}.  
]

若

[  
|\alpha|,|\beta|\ll B/X.  
]

所以只要存在小 (\alpha,\beta) 使

[  
\alpha r_1\equiv\beta r_2\pmod {r_3},  
]

就可能構造三角形 pairwise small 但 global representative未必小。

這種小係數 congruence 並非不可能。  
因此三個模數的 bounded Helly 不應無條件成立。

---

# 4. 但大 clique 會更受限

若有很多 (r_i)，pairwise representatives 必須滿足大量類似條件。  
可以固定 (r_1)，令

[  
h_{1i}=h_{12}+\alpha_i r_1.  
]

對 (i,j)，pairwise representative (h_{ij}) must satisfy

[  
h_{ij}\equiv h_{1i}\pmod {r_i},  
\qquad  
h_{ij}\equiv h_{1j}\pmod {r_j}.  
]

要求 (h_{ij}\in[-B,B])。  
這相當於很多小係數 CRT constraints：

# [  
h_{1j}-h_{1i}

\alpha_j r_1-\alpha_i r_1  
]

必須在模 (r_i,r_j) 下有小解。

直覺上，大 clique without common point 很難；但「難」不等於已證。

---

# 5. 更直接的反例形狀

可以從全局 CRT representative (M) 出發。任意 residues (a_i=M\bmod r_i)。  
若 (M) 本身很大，則全體 common representative 在 ([-B,B]) 中可能不存在。  
但 pairwise representative modulo (r_ir_j) 是

[  
M\bmod r_ir_j  
]

的對稱代表。若對所有 pairs (i,j)，這個代表都小於 (B)，則得到 pairwise bounded clique without global bounded point.

所以問題變成：

是否存在一個大 (M) such that for many primes (r_i\sim X),

[  
|M\bmod r_ir_j|\le B  
\quad\forall i,j,  
]

but

[  
|M\bmod \prod_i r_i|>B?  
]

這是更清楚的形式。

如果 (M) 是某個小整數加上大量 primes 的乘積倍數，可能 pairwise residues小而 global大。  
例如取

[  
M=M_0+\prod_i r_i.  
]

則 modulo每個 pair，(M\equiv M_0)，所以 pairwise representative 小；但 modulo全體 product，(M\equiv M_0) 也小。不是反例。

要全體無小 representative，需 (M\bmod R) 在大模數上遠離小區間，但每個 pair projection落在小區間。這類「所有二維投影小而全局大」是可能的嗎？  
對 CRT product lattice，pair projections小可能不控制 full representative。應該可能。

例如選

[  
M=\sum_i c_i \frac{R}{r_i}  
]

可使 pair projections depend on two terms? Pair projection modulo (r_ir_j) retains terms for k not i,j maybe vanish? Need compute. This may produce high-dimensional cocycle.

所以 clique Helly 也不是顯然成立。

這是一個重要警訊。若 bounded CRT has high-dimensional cocycle obstructions, then complete graph pairwise tests may not force a common label.
