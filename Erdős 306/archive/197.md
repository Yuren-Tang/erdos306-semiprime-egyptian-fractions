好。現在寫一版乾淨的 **conditional proof**。它的意思是：

[  
\boxed{  
\text{若 single-block CRT Poincaré theorem 成立，則主定理成立。}  
}  
]

這會把已完成的全局部分固定下來，並把唯一真正剩下的問題單獨標出。

# Conditional Theorem

## Main local hypothesis: single-block CRT Poincaré

令

[  
P\subset[X,2X]  
]

為經 pruning 後的 prime block，

[  
N=|P|\asymp \frac X{\log X}.  
]

對 residue assignment

[  
a=(a_p)_{p\in P}\in\prod_{p\in P}\mathbb F_p,  
]

令 (H_{pq}(a)\in(-pq/2,pq/2]) 滿足

[  
H_{pq}(a)\equiv a_p\pmod p,\qquad  
H_{pq}(a)\equiv a_q\pmod q.  
]

定義 internal energy

[  
Q_P(a)=  
\sum_{p<q\in P}  
\left(\frac{H_{pq}(a)}{pq}\right)^2.  
]

令

[  
\sigma_P^2=  
\sum_{p<q\in P}\frac1{p^2q^2}  
\asymp  
\frac1{X^2(\log X)^2}.  
]

假設以下局部定理成立：

[  
\boxed{  
\sum_{a\in\prod_{p\in P}\mathbb F_p}  
e^{-cQ_P(a)}  
\ll  
\sum_{m\in\mathbb Z}  
e^{-c'm^2\sigma_P^2}  
\asymp  
\frac1{\sigma_P}.  
}  
]

更強的 counting 版本為：

[  
\boxed{  
#{a:Q_P(a)\le R}  
\le  
e^{\varepsilon R}  
\left(1+\frac{\sqrt R}{\sigma_P}\right).  
}  
]

這就是目前唯一作為假設保留的核心局部命題。

# Conditional Main Theorem

**Theorem.**  
Assume the single-block CRT Poincaré theorem. Then for every squarefree positive integer (b) and every positive integer (a), there exist finitely many distinct squarefree semiprimes (n_i=p_iq_i) such that

[  
\frac ab=\sum_i\frac1{n_i}.  
]

---

# Proof

## 1. Squarefree denominator is necessary

若所有 (n_i) 都 squarefree，則

[  
L=\operatorname{lcm}(n_i)  
]

也 squarefree。有限和

[  
\sum_i\frac1{n_i}  
]

可寫為

[  
\frac{A}{L},  
]

化簡後分母整除 (L)，故仍 squarefree。  
因此 squarefree denominator 是必要條件。

---

## 2. Reduction to (1/b)

先證

[  
\frac1b  
]

可表示。  
若能表示 (1/b)，則取 (a) 批互不相交的 prime blocks，分別構造 (1/b)，相加得到

[  
\frac ab.  
]

各批 primes disjoint，故所有 semiprime denominators互異。

---

## 3. Prime blocks and edge architecture

取大尺度 dyadic prime blocks

[  
P_k\subset[2^k,2^{k+1}]  
]

經 Irving-good pruning。  
邊集合分為：

1. internal edges:  
    [  
    E_{\mathrm{int}}=\bigcup_k\binom{P_k}{2};  
    ]
    
2. bounded-degree cross skeleton:  
    [  
    E_{\mathrm{skel}},  
    ]  
    用於合併 block labels；
    
3. high-scale mass edges:  
    [  
    E_{\mathrm{mass}},  
    ]  
    用於調整期望；
    
4. lattice gadget edges:  
    [  
    E_{\mathrm{gad}},  
    ]  
    對每個 (r\mid b) 加入 (rs_r)，其中 (s_r) 是大 prime。
    

令

[  
E=E_{\mathrm{int}}\cup E_{\mathrm{skel}}\cup E_{\mathrm{mass}}\cup E_{\mathrm{gad}}.  
]

控制邊為

[  
E_{\mathrm{ctrl}}=E_{\mathrm{int}}\cup E_{\mathrm{skel}}.  
]

---

## 4. Irving-good pruning

引用 Irving 的 Kloosterman-prime average bound：

[  
\sum_{q\sim Q}\max_{(a,q)=1}  
\left|  
\sum_{p\sim x}e_q(a\overline p)  
\right|  
\ll_\varepsilon  
\left(  
Q^{5/4}x^{5/8}  
+  
Qx^{9/10}  
+  
Q^{7/6}x^{13/18}  
\right)Q^\varepsilon.  
]

在 compatible range

[  
Q^{2/3+\delta}\le x\le Q^{3/2}  
]

中，它給非平凡平均估計。  
Markov + pruning + Erdős–Turán 推出：

對每個 control block pair (P,Q)，刪去 (o(|Q|)) bad primes 後，對每個保留 (q) 和每個 (d\ne0\pmod q)，

[  
\sum_{p\in P}  
\left|  
\frac{d\overline p}{q}  
\right|^2  
\gg |P|.  
]

bounded-degree skeleton 允許 union bound，故所有 control pairs 同時滿足此性質。

---

## 5. Random model and mass tuning

令

[  
\xi_{pq}\in{0,1}  
]

獨立，參數

[  
\theta_{pq}\in[c,1-c].  
]

定義

[  
X_E=\sum_{pq\in E}\xi_{pq}\frac1{pq}.  
]

選擇 high-scale mass edges 與 (\theta_{pq})，使

# [  
\mathbb E X_E

# \sum_{pq\in E}\frac{\theta_{pq}}{pq}

\frac1b.  
]

Internal mass

[  
\sum_k\sum_{p<q\in P_k}\frac1{pq}  
\asymp  
\sum_k\frac1{k^2}  
]

收斂，可從大 (k_0) 開始使其可控。  
High-scale cross edges 可提供任意所需 mass，同時其 variance 很小。

令

[  
\sigma_E^2=  
\sum_{pq\in E}  
\theta_{pq}(1-\theta_{pq})  
\frac1{p^2q^2}.  
]

安排 mass edges 在足夠高尺度，使

[  
\sigma_E\asymp\sigma_{\mathrm{ctrl}}.  
]

---

## 6. Lattice span

因 (b) squarefree，把所有

[  
r\mid b  
]

納入 prime set (\mathcal P)。令

[  
L=\prod_{p\in\mathcal P}p.  
]

則

[  
b\mid L.  
]

整數化：

# [  
Y=LX_E

\sum_{pq\in E}\xi_{pq}\frac{L}{pq}.  
]

目標為

[  
Y=L/b.  
]

因每個 (p\in\mathcal P) incident to some edge (pq\in E)，有

[  
\gcd_{pq\in E}\frac{L}{pq}=1.  
]

故無 lattice periodic obstruction。

---

## 7. Fourier inversion

在

[  
\mathbb Z/L\mathbb Z  
]

上：

# [  
\mathbb P(Y=L/b)

\frac1L  
\sum_{h\bmod L}  
\widehat\mu(h)e(-h/b).  
]

將 character (h) 寫成 residues

[  
a_p=h\bmod p.  
]

對 edge (pq)，phase 為

# [  
\phi_{pq}(a)

\frac{H_{pq}(a)}{pq}  
\pmod1.  
]

Fourier coefficient：

# [  
\widehat\mu(a)

\prod_{pq\in E}  
\left(  
1-\theta_{pq}  
+\theta_{pq}e(\phi_{pq}(a))  
\right).  
]

標準估計：

[  
|\widehat\mu(a)|  
\le  
\exp(-cQ_E(a)),  
]

其中

[  
Q_E(a)=  
\sum_{pq\in E}|\phi_{pq}(a)|^2.  
]

---

## 8. Major arc

Major arc characters are ordinary labels:

[  
a_p\equiv m\pmod p,\qquad |m|\le C/\sigma_E.  
]

Then

[  
\phi_{pq}=\frac{m}{pq}.  
]

Taylor expansion gives:

# [  
\widehat\mu(m)e(-m/b)

\exp(-2\pi^2\sigma_E^2m^2+o(1)).  
]

Hence

# [  
\sum_{|m|\le C/\sigma_E}  
\widehat\mu(m)e(-m/b)

(1+o(1))  
\sum_{|m|\le C/\sigma_E}  
e^{-2\pi^2\sigma_E^2m^2}  
\asymp  
\frac1{\sigma_E}.  
]

Taking (C) large, this contribution is positive.

---

## 9. Minor arc via conditional block Poincaré

Since

[  
Q_E(a)\ge Q_{\mathrm{ctrl}}(a),  
]

and

[  
\sigma_E\asymp\sigma_{\mathrm{ctrl}},  
]

it suffices to prove

# [  
\sum_{a\notin\mathfrak M}  
e^{-cQ_{\mathrm{ctrl}}(a)}

o(1/\sigma_{\mathrm{ctrl}}).  
]

Now use the assumed single-block CRT Poincaré theorem on every internal block.

For each block (P_k), the internal energy compresses the residues in (P_k) to an ordinary block label (m_k), with partition cost comparable to

[  
\sum_{m_k}e^{-c m_k^2\sigma_k^2}.  
]

The cross skeleton, using Irving-good pruning, gives label mismatch cost:

if

[  
m_k\ne m_\ell,  
]

then

[  
Q_{k\ell}(m_k,m_\ell)  
+  
Q_{k\ell}^{\mathrm{diag}}(m_k)  
\gg |P_k||P_\ell|  
]

except for divisor-negligible cases.  
Thus Peierls collapse forces all block labels to be equal to a single global (m). Therefore

[  
\sum_a e^{-cQ_{\mathrm{ctrl}}(a)}  
\ll  
\sum_m e^{-c'Q_{\mathrm{diag}}(m)}  
\ll  
\frac1{\sigma_{\mathrm{ctrl}}}.  
]

Moreover, outside

[  
|m|\le C/\sigma_{\mathrm{ctrl}},  
]

the Gaussian tail is

[  
o(1/\sigma_{\mathrm{ctrl}}).  
]

Hence

[  
\sum_{a\notin\mathfrak M}  
|\widehat\mu(a)|  
\le  
o(1/\sigma_E).  
]

---

## 10. Positivity

Fourier sum:

# [  
\sum_h\widehat\mu(h)e(-h/b)

\text{major arc}  
+  
\text{minor arc}.  
]

Major arc is

[  
\asymp 1/\sigma_E>0.  
]

Minor arc is

[  
o(1/\sigma_E).  
]

Therefore for sufficiently large scales,

[  
\mathbb P(Y=L/b)>0.  
]

Thus there exists a subset

[  
S\subset E  
]

such that

# [  
\sum_{pq\in S}\frac1{pq}

\frac1b.  
]

Repeating with (a) disjoint prime systems gives (a/b).  
The conditional theorem is proved.

---

# Remaining core problem

The only unproved ingredient in this conditional proof is the single-block CRT Poincaré theorem:

[  
\sum_{a\in\prod_{p\in P}\mathbb F_p}  
e^{-cQ_P(a)}  
\ll  
1/\sigma_P.  
]

Equivalently, the CRT code

[  
m\mapsto (m\bmod p)_{p\in P}  
]

should be locally testable by pairwise squarefree CRT energy.

The earlier Helly/list-agreement route is false in its naive form; small examples show bounded pairwise intersections need not have a common bounded representative.  
However numerical experiments suggest non-diagonal cocycles have a genuine quadratic energy gap, so this local theorem remains plausible but nontrivial.
