是的，若「它」指的是最鋒利的命題

[  
\forall K>2,\quad  
\left|\pi-\frac pq\right|<q^{-K}  
\text{ 只有有限多解},  
]

那麼我們搞不定。更準確地說，不是「我們這裏一時沒想到」，而是它等價於或至少正面撞上

[  
\mu(\pi)=2  
]

這個目前數論中尚未證明的問題。若要證這個，就不是補一個技巧、換一個分層、算幾頁不等式可以解決的事。

但這不等於全線失敗。現在我們已經把問題從「迷霧中追 π」推到一個清楚的位置：

[  
\text{原問題的強版本}  
\quad\Longrightarrow\quad  
\mu(\pi)=2\text{ 型未解問題}.  
]

這本身就是有價值的歸約。數學上常見的進展不是「直接證出原命題」，而是先判明：這個命題到底是初等困難、技術困難，還是其實要求解決一個著名硬核問題。現在我們得到的是第三種。

但我們還可以繼續做兩件可靠的事。

第一，證出完整的「歸約定理」：

[  
\mu(\pi)=2  
\Longleftrightarrow  
\forall K>2,\ P(K),  
]

其中

[  
P(K):\quad  
\left|\pi-\frac pq\right|<q^{-K}  
\text{ 只有有限多解}.  
]

第二，證出無條件弱版本：

[  
K>7.103205334137\ldots  
\Longrightarrow  
P(K).  
]

這樣原問題就有一個嚴格結論：

> 若原命題需要 (K>7.103205334137\ldots)，它已可由現有定理推出；若它需要所有 (K>2)，它目前不可證，除非解決 (\mu(\pi)=2)；若它要求 (C/q^2) 統一下界，則它甚至強到 badly approximable，方向大概率不對。

現在我把第一個歸約定理證乾淨。

定義

# [  
\mu(\alpha)

\sup\left{  
\lambda>0:  
\left|\alpha-\frac pq\right|<\frac1{q^\lambda}  
\text{ 有無窮多有理數 }p/q  
\right}.  
]

對無理數 (\alpha)，Dirichlet 定理保證對無窮多 (p/q) 有

[  
\left|\alpha-\frac pq\right|<\frac1{q^2},  
]

所以

[  
\mu(\alpha)\geq2.  
]

現在令 (\alpha=\pi)。

若

[  
\mu(\pi)=2,  
]

取任意 (K>2)。若 (P(K)) 不成立，則

[  
\left|\pi-\frac pq\right|<q^{-K}  
]

有無窮多解。按定義便有

[  
\mu(\pi)\geq K>2,  
]

矛盾。因此 (P(K)) 成立。

反過來，若對所有 (K>2)，(P(K)) 都成立。假設

[  
\mu(\pi)>2.  
]

則可取某個 (K) 滿足

[  
2<K<\mu(\pi).  
]

按 (\mu(\pi)) 的定義，逼近指數 (K) 應有無窮多次達到，即

[  
\left|\pi-\frac pq\right|<q^{-K}  
]

有無窮多解。這與 (P(K)) 成立矛盾。故

[  
\mu(\pi)\leq2.  
]

又因 (\mu(\pi)\geq2)，所以

[  
\mu(\pi)=2.  
]

於是得到：

[  
\boxed{  
\mu(\pi)=2  
\Longleftrightarrow  
\forall K>2,\  
\left|\pi-\frac pq\right|<q^{-K}  
\text{ 只有有限多解}.  
}  
]

這就正式說明：想證所有 (K>2) 的版本，就是想證 (\mu(\pi)=2)。

現在再補第二個歸約：爲甚麼 (K>2) 的異常逼近必須落在連分數收斂子上。

Legendre 定理說：若 (\alpha) 是無理數，且有理數 (p/q) 滿足

[  
\left|\alpha-\frac pq\right|<\frac1{2q^2},  
]

則 (p/q) 必為 (\alpha) 的連分數收斂子。

若 (K>2)，則當 (q) 充分大時，

[  
q^{-K}<\frac1{2q^2}.  
]

所以任何充分大的解

[  
\left|\pi-\frac pq\right|<q^{-K}  
]

都必然是 π 的某個連分數收斂子。這說明 (P(K)) 的無窮性問題可以縮到收斂子序列上。有限多個小分母不影響「是否無窮多」。

於是：

[  
P(K)  
]

等價於說，除有限多個 (n) 外，

[  
\left|\pi-\frac{p_n}{q_n}\right|\geq q_n^{-K}.  
]

又因為

[  
\left|\pi-\frac{p_n}{q_n}\right|  
\asymp  
\frac{1}{a_{n+1}q_n^2},  
]

所以它本質上等價於說：

[  
a_{n+1}\not\gtrsim q_n^{K-2}  
]

無窮多次發生。

這裏的意思是：若 (K=2+\delta)，那麼問題就是 π 的連分數中是否無窮多次出現

[  
a_{n+1}\gtrsim q_n^\delta.  
]

而我們目前不知道如何排除這種事情。

所以結論很硬：

我們搞不定 (\delta>0) 任意小的情形；現有文獻只能排除非常大的 (\delta)，約莫相當於

[  
K>7.103205334137\ldots,  
]

也就是

[  
\delta>5.103205334137\ldots.  
]

而自然猜想需要排除所有

[  
\delta>0.  
]

差距很大。

但這個失敗不是無內容的失敗。現在我們知道：若某人聲稱用一個初等分層、格點估計、均勻分布直覺，就能證 π 的 (q^{-2-\varepsilon}) 級排除，那高度可疑；因為他實際上很可能是在暗中證明 (\mu(\pi)=2)。除非他真的提供了全新的 transcendence／Padé approximation／linear forms 方法，否則大概率有漏洞。

所以我們下一步最合理的「繼續」不是再往 π 的連分數坑裏鑽，而是把整個成果改寫成一個可保存的結論：

[  
\boxed{  
\text{原命題的強版不可由現有常規方法推出；它本質上屬於 }\mu(\pi)=2\text{ 問題。}  
}  
]

而若仍要產出一個真正定理，就採用：

[  
\boxed{  
K>7.103205334137\ldots  
\Rightarrow  
\left|\pi-\frac pq\right|<q^{-K}  
\text{ 只有有限多解。}  
}  
]

這就是目前可負責任地推到的邊界。
