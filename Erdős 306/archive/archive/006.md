好，往前推一步。真正正面的方法，我現在覺得應該是這個：**把等式問題改成一個模** L **的圖邊界值問題**。這不是小例子，而是原問題的骨架。

  

設 b squarefree，先只處理核心情形

  

\frac1b.

  

若能證明對任意 squarefree b，甚至避開任意有限禁用素數集合 F，都有

  

\frac1b=\sum_i\frac1{p_iq_i},

  

那麼一般

  

\frac ab

  

就可以用 a 份互不重疊的 1/b 表示湊出來。所以原問題可壓縮為：

  

\boxed{ \text{對任意 squarefree }b,\ \frac1b\text{ 可用互異 }pq\text{ 分母表示，且可避開有限禁用集。} }

  

這是第一個正面歸約。

  

現在取一個有限素數集合 P，包含所有 b 的素因子。令

  

L=\prod_{p\in P}p.

  

若我們在 P 上選一張簡單圖 E，把每條邊 \{p,q\} 對應到分母 pq，則

  

\sum_{\{p,q\}\in E}\frac1{pq} = \frac1L\sum_{\{p,q\}\in E}\frac{L}{pq}.

  

所以要等於 1/b，等價於

  

\sum_{\{p,q\}\in E}\frac{L}{pq} = \frac Lb.

  

這本來是精確整數等式。但關鍵觀察是：**如果我們只先做到模** L **同餘，再控制大小小於** L**，同餘就會強迫等式成立。**

  

也就是說，如果能構造 E，使

  

\sum_{\{p,q\}\in E}\frac{L}{pq} \equiv \frac Lb \pmod L,

  

並且

  

0< \sum_{\{p,q\}\in E}\frac{L}{pq} <L,

  

那麼左邊與 \frac Lb 都在 [0,L) 內，且同餘模 L，所以必然相等。

  

換回分數，第二個條件只是

  

0<\sum_{\{p,q\}\in E}\frac1{pq}<1.

  

這個可以靠選很大的輔助素數來保證。因此真正的難處變成：

  

\boxed{ \text{構造素數圖 }E\text{，使其滿足一組局部模條件。} }

  

具體地，對每個頂點 r\in P 看模 r。一條邊 \{p,q\} 的權

  

\frac{L}{pq}

  

模 r 只有在 r=p 或 r=q 時才可能非零；否則它含因子 r，模 r 為零。因此整個同餘條件分解為每個頂點的局部條件：

  

若 r\mid b，則

  

\sum_{s:\{r,s\}\in E}\frac1s \equiv \frac1{b/r} \pmod r.

  

若 r\nmid b，則

  

\sum_{s:\{r,s\}\in E}\frac1s \equiv 0 \pmod r.

  

這很重要。原來的埃及分數問題變成了：

  

給定邊界素數 r\mid b 上的指定值 \frac1{b/r}\pmod r，能否添加足夠多輔助素數，並選邊，使每個輔助頂點局部平衡為 0，每個邊界頂點達到指定值？

  

這就是一個**素數圖的邊界值問題**。

  

因此可以提出一個真正核心的引理：

  

\boxed{ \textbf{邊界值引理：} }

  

給定有限素數集合 B、每個 p\in B 上的指定剩餘類 c_p\in\mathbb F_p，以及有限禁用素數集合 F，存在一個有限素數集合 C，與 B\cup F 不交，並存在 B\cup C 上的簡單圖 E，使得：

  

對 p\in B,

  

\sum_{s\sim p}\frac1s\equiv c_p\pmod p;

  

對 r\in C,

  

\sum_{s\sim r}\frac1s\equiv0\pmod r;

  

並且

  

\sum_{\{p,q\}\in E}\frac1{pq}<1.

  

若這個引理成立，原問題幾乎立即成立。證法如下。

  

取

  

B=\{p:p\mid b\}, \qquad c_p=\frac1{b/p}\pmod p.

  

由邊界值引理得到圖 E。令

  

P=B\cup C,\qquad L=\prod_{r\in P}r.

  

局部條件對每個 r\in P 都成立，所以

  

\sum_{\{p,q\}\in E}\frac{L}{pq} \equiv \frac Lb \pmod r

  

對所有 r\in P 成立。因為 L squarefree，所以合併得

  

\sum_{\{p,q\}\in E}\frac{L}{pq} \equiv \frac Lb \pmod L.

  

又由大小條件，

  

0< \sum_{\{p,q\}\in E}\frac{L}{pq} <L.

  

而

  

0<\frac Lb<L.

  

所以兩者相等：

  

\sum_{\{p,q\}\in E}\frac{L}{pq} = \frac Lb.

  

除以 L，便得

  

\frac1b=\sum_{\{p,q\}\in E}\frac1{pq}.

  

這就完成 1/b。再用避開有限集合的版本反覆構造 a 份互不重疊的表示，即得 a/b。

  

所以我們現在把原問題推到一個更清楚的核心：

  

\boxed{ \text{原 Erdős 問題可由「素數圖邊界值引理」推出。} }

  

這比前面的 1/210 小例子強多了，因為它不再依賴某個特殊分母，而是把所有 b 統一處理。

  

現在問題變成：這個邊界值引理有沒有希望？

  

我覺得有，而且原因很具體。若加入 N 個輔助素數，未知邊大約有

  

\binom{N}{2}+N|B|

  

條，但局部同餘方程只有

  

N+|B|

  

條。變量數二次增長，約束數一次增長。自由度巨大。直覺上，應該能解。

  

但不能太樂觀。因為變量是 0/1，不是任意整數；而且每個模數都不同。這不是普通線性代數，而是「帶素數倒數權重的 0/1 同餘圖構造」。所以核心技術不是算例，而是要造一種**局部平衡 gadget**。

  

最小的 gadget 目標是：

  

給定一個邊界素數 p 與目標

  

c\in\mathbb F_p,

  

能否造一團輔助素數，使它們內部全部平衡為 0，而在 p 上留下恰好 c？

  

也就是造一個「帶一個端口的電路」：

  

\sum_{s\sim p}\frac1s\equiv c\pmod p,

  

但所有內部頂點 r 滿足

  

\sum_{s\sim r}\frac1s\equiv0\pmod r.

  

若這種單端口 gadget 存在，且可以避開任意有限素數集合，那麼邊界值引理就很可能成立：對每個 p\mid b 分別接上一個 gadget 即可。

  

這一步纔是真正的攻關點。

  

所以，現在我們不是在問「能不能找一個奇怪分數表示」，而是在問：

  

\boxed{ \text{能否用素數圖造出任意指定邊界剩餘類，同時讓所有內部頂點模自身平衡？} }

  

這是原問題的一個相當正面的核心形態。

  

我目前的進展可以濃縮成三句：

  

第一，原問題可降為「可避障地表示 1/b」。  
第二，表示 1/b 可降為一個模 L 的素數圖邊界值問題。  
第三，若能證明單端口局部平衡 gadget 存在，原 Erdős 問題就基本打通。

  

這裏真正下一步不應再搜 1/210，而應專攻：

  

\boxed{ \text{構造一個單端口 gadget：在指定素數 }p\text{ 上產生 }c\pmod p,\text{ 內部全平衡。} }

  

這已經是相當清楚的正面戰場。
