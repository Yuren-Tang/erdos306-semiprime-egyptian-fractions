好。先推稀疏層。為避免空泛，我把它寫成一個可檢查的命題。

我們有輔助素數

[  
C={c_1,\dots,c_N},  
]

以及映射

[  
\Phi(G)_j=\sum_{i\sim j}\frac1{c_i}\pmod{c_j}.  
]

Fourier 係數為

# [  
\widehat\mu(\lambda)

\prod_{i<j}  
\frac{1+e^{2\pi i\theta_{ij}(\lambda)}}2,  
]

其中

# [  
\theta_{ij}(\lambda)

\frac{\lambda_i c_j^{-1}}{c_i}  
+  
\frac{\lambda_j c_i^{-1}}{c_j}.  
]

若 (\lambda_j=0)，則對邊 ((i,j)) 有

# [  
\theta_{ij}(\lambda)

\frac{\lambda_i c_j^{-1}}{c_i}.  
]

所以稀疏層真正需要的不是整個複雜相位，而是下面這個較簡單的「倒數分散」條件。

---

## 稀疏分散條件

要求存在常數 (\delta>0)，使對每個 (i)，每個非零 (\lambda_i\in\mathbb F_{c_i})，以及任意足夠大的集合 (J\subseteq{1,\dots,N}\setminus{i})，只要

[  
|J|\ge \frac N3,  
]

便有

[  
\sum_{j\in J}  
\left|  
\frac{\lambda_i c_j^{-1}}{c_i}  
\right|^2  
\ge  
\delta |J|.  
]

直觀地說：對模 (c_i) 而言，其他素數 (c_j) 的倒數不能大量擠在 (0) 附近；乘上任意非零 (\lambda_i) 之後仍要分散。

這條件若成立，稀疏 Fourier 層就可以正式關閉。

---

## 稀疏層命題

**命題。**  
假設輔助素數集 (C) 滿足上述稀疏分散條件，且所有 (c_i) 不太大，例如

[  
\log c_i\le K\log N  
]

某個固定 (K)。則所有支撐大小

[  
1\le t=|\operatorname{supp}\lambda|\le \frac N2  
]

的非零 (\lambda) 對 Fourier 和的總貢獻趨於 (0)：

[  
\sum_{1\le |\operatorname{supp}\lambda|\le N/2}  
|\widehat\mu(\lambda)|=o(1).  
]

這就是稀疏層的正式結論。

---

## 證明

設

[  
S=\operatorname{supp}\lambda,\qquad |S|=t\le N/2.  
]

考慮所有從 (S) 到補集 (S^c) 的邊。對 (i\in S,\ j\notin S)，因為 (\lambda_j=0)，所以

# [  
\theta_{ij}(\lambda)

\frac{\lambda_i c_j^{-1}}{c_i}.  
]

由稀疏分散條件，對每個 (i\in S)，

[  
\sum_{j\notin S}  
\left|  
\frac{\lambda_i c_j^{-1}}{c_i}  
\right|^2  
\ge  
\delta |S^c|  
\ge  
\frac{\delta N}{2}.  
]

因此

[  
\sum_{i<j}|\theta_{ij}(\lambda)|^2  
\ge  
\sum_{i\in S,\ j\notin S}  
|\theta_{ij}(\lambda)|^2  
\ge  
\frac{\delta tN}{2}.  
]

又有標準估計

# [  
\left|\frac{1+e^{2\pi ix}}2\right|

|\cos \pi x|  
\le  
\exp(-2|x|^2),  
]

故

[  
|\widehat\mu(\lambda)|  
\le  
\exp\left(  
-2\sum_{i<j}|\theta_{ij}(\lambda)|^2  
\right)  
\le  
\exp(-\delta tN).  
]

現在數有多少個支撐大小為 (t) 的 (\lambda)。選支撐有

[  
\binom Nt  
]

種；每個支撐上，(\lambda_i) 有 (c_i-1) 種非零值。若 (\log c_i\le K\log N)，則

[  
\prod_{i\in S}(c_i-1)\le N^{Kt}.  
]

所以支撐大小為 (t) 的 (\lambda) 數量至多

[  
\binom Nt N^{Kt}  
\le  
N^{(K+1)t}.  
]

其總貢獻至多

# [  
N^{(K+1)t}e^{-\delta tN}

\exp\bigl((K+1)t\log N-\delta tN\bigr).  
]

對 (1\le t\le N/2) 求和：

[  
\sum_{t=1}^{N/2}  
\exp\bigl((K+1)t\log N-\delta tN\bigr)  
=o(1).  
]

因為 (N\gg\log N)。命題得證。

---

## 這一步的意義

這不是假進展。它說明：

[  
\boxed{  
\text{稀疏 character 不再是主要困難。}  
}  
]

只要輔助素數集 (C) 滿足一個相當自然的倒數分散條件，所有稀疏 Fourier 項都會被指數衰減殺掉。

所以問題被推進為：

[  
\boxed{  
\text{能否選取素數 }c_1,\dots,c_N\text{，使倒數分散條件成立？}  
}  
]

這是一個比原問題小得多的解析數論問題。

---

## 這個倒數分散條件有多可信？

對固定 (c_i=p)，我們看集合

[  
{c_j^{-1}\bmod p:j\ne i}.  
]

若這些倒數像隨機非零剩餘類，那麼

[  
\left|  
\frac{\lambda c_j^{-1}}p  
\right|^2  
]

的平均值大約是

[  
\int_{-1/2}^{1/2}x^2,dx=\frac1{12}.  
]

所以取

[  
\delta<\frac1{20}  
]

之類應該很安全。

因此從隨機模型看，稀疏分散條件非常合理。

但要嚴格證明，不能只說「像隨機」。需要一個明確的數論輸入：

[  
\boxed{  
\text{素數集合在模 }p\text{ 下取倒數後，不會大量集中於短弧。}  
}  
]

換言之，要控制

[  
#\left{  
j:  
\left|  
\frac{\lambda c_j^{-1}}{c_i}  
\right|<\varepsilon  
\right}.  
]

若對所有 (\lambda\ne0) 和所有小 (\varepsilon) 都有近似

[  
#{\cdots}\ll \varepsilon N+o(N),  
]

那麼平方和下界就跟着來。

---

## 這裏有一個要小心的點

不能隨便選 (c_j) 極大。因為 Fourier 求和裏 character 的總數是

[  
\prod_i c_i.  
]

如果 (c_i) 增長太快，比如由 CRT 構造導致指數塔，則 character 數暴增，前面的

[  
e^{-\delta tN}  
]

壓不住。  
所以輔助素數既要分散，又不能太大。比較理想的是

[  
c_i\le N^K  
]

某個固定 (K)。

這正是稀疏層目前的真技術要求：

[  
\boxed{  
\text{在多項式大小範圍內選 }N\text{ 個素數，使各模數下的互逆分佈足夠分散。}  
}  
]

這看起來不是荒唐要求，但也不是初等一行可證。

---

## 下一步應當攻的引理

我們可以把下一個靶子定為：

**互逆分散引理。**  
存在固定 (K) 與 (\delta>0)，使得對任意大 (N)，可取互異素數

[  
c_1,\dots,c_N\le N^K  
]

滿足：對所有 (i)、所有非零 (\lambda\in\mathbb F_{c_i})、所有 (J\subseteq{1,\dots,N}\setminus{i}) 且 (|J|\ge N/3)，有

[  
\sum_{j\in J}  
\left|  
\frac{\lambda c_j^{-1}}{c_i}  
\right|^2  
\ge  
\delta |J|.  
]

若這個引理成立，稀疏層完全解決。

---

## 這條路目前的狀態

我們已經把稀疏層從原問題中剝離出來，變成一個乾淨的、可獨立研究的命題：

[  
\boxed{  
\text{多項式大小素數集的互逆分散。}  
}  
]

這比「證 Erdős 問題」小得多，也更可能被現有解析數論工具碰到。可能的工具是：

素數在算術級數中的分佈；  
模反元素的短區間估計；  
大篩；  
Kloosterman 和估計；  
或隨機選素數加概率法。

但到這一步我不能誠實地說它已證。能確定的是：**稀疏 Fourier 層已被歸約到一個明確分散引理，而這個引理看起來合理，且不是原題那樣巨大。**

接下來若繼續推，就是要嘗試證這個互逆分散引理。我的判斷是：這一步值得攻；它是真正有內容的一塊磚，不是「日取其半」。
