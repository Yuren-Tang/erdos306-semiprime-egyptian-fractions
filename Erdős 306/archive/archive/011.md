好，繼續。但這一步我先修正上一輪的「互逆分散引理」。它的方向對，形式卻有點過強，尤其是這一條：

[  
\forall J,\ |J|\ge N/3,\quad  
\sum_{j\in J}  
\left|  
\frac{\lambda c_j^{-1}}{c_i}  
\right|^2  
\ge \delta |J|.  
]

要求「任意大子集 (J)」都分散，等於要求不能有 (N/3) 個點集中在某個很短弧附近。這未必錯，但比 Fourier 稀疏層真正需要的稍強。稀疏層只需要對

[  
J=S^c  
]

這種補集成立，而 (S) 是 character 的支撐。因為 (S) 會隨 (\lambda) 變動，要求任意大 (J) 是方便但可能太奢侈。

更合理的版本是：不用管所有 (J)，而證一個**排序下界**。

對固定 (i) 和非零 (\lambda)，令

[  
x_j=  
\left|  
\frac{\lambda c_j^{-1}}{c_i}  
\right|^2  
\qquad (j\ne i).  
]

若任意刪去最多 (N/2) 個 (x_j)，剩下的和仍 (\gg N)，等價於說：把 (x_j) 從小到大排列，後半部分之和 (\gg N)。這比所有大子集條件更精準。

所以稀疏層真正需要的是：

[  
\boxed{  
\sum_{\text{largest }N/2\text{ terms? actually remaining after deleting }N/2}  
x_j\gg N.  
}  
]

更準確地說，因為最壞情況是刪掉最大的那些，保留下來的是最小的 (N/2) 項；因此需要

[  
\boxed{  
\sum_{\text{smallest }N/2\text{ of the }x_j} x_j\ge \delta N.  
}  
]

這個條件等價於：不能有一半以上的 (c_j^{-1}) 在模 (c_i) 下太靠近 (0)。

---

## 1. 稀疏層真正要證的條件

對所有 (i) 與所有非零 (\lambda\in\mathbb F_{c_i})，要求

[  
\sum_{j\in T}  
\left|  
\frac{\lambda c_j^{-1}}{c_i}  
\right|^2  
\ge \delta N  
]

對每個大小至少 (N/2) 的集合 (T\subseteq{1,\dots,N}\setminus{i}) 成立。

這仍然強，但比「所有 (N/3) 大子集」更貼近需要。

如果 (c_j^{-1}\bmod c_i) 像均勻分佈，那麼這個條件很自然。因為在 ([-1/2,1/2]) 上，最靠近 (0) 的一半點落在 ([-1/4,1/4])，其平方平均為

# [  
\frac{1}{1/2}\int_{-1/4}^{1/4}x^2,dx

\frac1{48}.  
]

所以最小的一半平方和期望大約是

[  
\frac{N}{96}  
]

量級。取很小的 (\delta)，例如 (1/200)，從隨機模型看完全合理。

---

## 2. 但「選素數」這一步不簡單

上一輪我說可以希望取

[  
c_i\le N^K.  
]

這其實太樂觀。因為我們需要每個 (c_i) 看其他所有 (c_j) 的倒數都分散。若所有 (c_i) 都在同一個短素數區間裏，比如 (c_i\asymp N^K)，那麼對固定模數 (c_i)，其他素數 (c_j) 的 residue distribution 未必容易控制。尤其當模數和素數大小同階時，素數在模 (c_i) 下的分佈不再是初等可控的。

所以更誠實的說法是：

[  
\boxed{  
\text{稀疏層不只是組合問題，還需要可控大小下的素數互逆分佈。}  
}  
]

這是一個真正的解析數論輸入。

---

## 3. 可以放寬大小要求嗎？

Fourier 稀疏估計中，支撐大小為 (t) 的 character 數大約受

[  
\prod_{i\in S}c_i  
]

控制。若

[  
\log c_i\le \eta N  
]

且 (\eta>0) 足夠小，那麼支撐大小 (t) 的 character 數至多約

[  
\exp(\eta tN+O(t\log N)).  
]

而 Fourier 衰減是

[  
\exp(-\delta tN).  
]

所以只要

[  
\eta<\delta  
]

的一個固定小倍數，仍能壓住。

因此我們不必堅持

[  
c_i\le N^K.  
]

可以允許

[  
\boxed{  
c_i\le e^{\eta N}  
}  
]

其中 (\eta) 是很小的常數。

這很重要。它給選素數留下更多空間。若只要求輔助素數指數級小，而非多項式小，CRT 或概率選擇的餘地就大得多。

但也不能讓 (c_i) 變成 (e^{N\log N}) 或更大；那樣 character 數會壓倒 Fourier 衰減。

---

## 4. 改良後的稀疏層引理

現在比較合理的版本是：

**稀疏互逆分散引理。**  
存在常數 (\eta,\delta>0)，使得對任意大 (N)，可取互異素數

[  
c_1,\dots,c_N\le e^{\eta N},  
]

滿足：對所有 (i)、所有非零 (\lambda\in\mathbb F_{c_i})，以及所有 (T\subseteq{1,\dots,N}\setminus{i}) 且

[  
|T|\ge N/2,  
]

有

[  
\sum_{j\in T}  
\left|  
\frac{\lambda c_j^{-1}}{c_i}  
\right|^2  
\ge  
\delta N.  
]

若這個引理成立，稀疏 Fourier 層成立。

證明同前，但 character 數估計改為：

[  
\prod_{i\in S}c_i  
\le  
e^{\eta tN}.  
]

於是支撐大小 (t) 的總貢獻至多

[  
\exp(O(t\log N)+\eta tN-\delta' tN).  
]

只要 (\eta<\delta')，求和仍為 (o(1))。

這個版本比多項式大小版更現實。

---

## 5. 能否證這個互逆分散引理？

這裏有兩條路。

### 路線 A：隨機選素數

從某個區間

[  
[X,2X],\qquad X=e^{\eta N}  
]

中隨機選 (N) 個素數。希望證明對每個選中的 (c_i)，其他選中素數的倒數模 (c_i) 具有足夠分散性。

困難是：模數 (c_i\sim X)，樣本也在 ([X,2X])。對固定 (c_i)，另一個素數 (q\in[X,2X]) 的 residue (q\bmod c_i) 大約只跑過長度 (\asymp c_i) 的區間，形式上覆蓋一圈左右。這不是「很多圈」平均，證明均勻性不會很便宜。

不過倒數映射

[  
q\mapsto q^{-1}\pmod {c_i}  
]

會把區間扭曲得很厲害；這裏可能可用 Kloosterman 型估計。但這已經不是初等工具。

### 路線 B：先設計 residue，再用 Dirichlet/CRT 找素數

對每個 (c_i)，我們希望其他 (c_j) 在模 (c_i) 下落在某些指定 residue classes。理想做法是先選一個「模矩陣」

[  
r_{ij}\in\mathbb F_{c_i}^{\times},  
]

讓每行分散；再要求

[  
c_j\equiv r_{ij}^{-1}\pmod{c_i}  
]

對所有 (i\ne j)。

對固定 (j)，這是一組模不同素數 (c_i) 的同餘條件。CRT 能給出一個 residue class modulo

[  
\prod_{i\ne j}c_i.  
]

Dirichlet 可保證該類中有無窮多素數。問題是，最小這樣的素數可能極大。模數大約是

[  
\prod_{i\ne j}c_i,  
]

其對數約為

[  
(N-1)\eta N\asymp \eta N^2.  
]

這會使 (c_j) 可能大到 (e^{O(N^2)})，超出 Fourier 稀疏層能承受的 (e^{\eta N})。

所以全矩陣 CRT 不可行。這條路若要走，必須用局部設計或分塊設計，不能對每對 ((i,j)) 都指定 residue。

這是一個重要結論：**不能靠粗暴 CRT 設計所有互逆 residue。**

---

## 6. 因此稀疏層的正確選擇

我會選：

[  
\boxed{\text{隨機素數 + 指數和估計，而不是全矩陣 CRT。}}  
]

也就是取

[  
c_i  
]

在一段可控大小的素數區間中隨機分佈，然後證明所需互逆分散條件以高概率成立。

這條路需要一個分析估計，形式上類似：

對素數模 (p)、任意非零 (\lambda)，以及區間中的素數 (q)，估計

[  
\sum_{q\in[X,2X]\atop q\text{ prime}}  
e\left(\frac{\lambda q^{-1}}p\right).  
]

若這個和比素數數量小很多，則倒數分佈分散。

這正是 Kloosterman-sum-with-primes 的類型。若 (X) 比 (p) 稍大或同階，這很深；若能安排 (X) 明顯大於所有 (p)，則容易一些，但所有 (p) 又也是從同一區間選的，互相同階。

可以改成分層選素數：

[  
C=C_1\cup C_2\cup\cdots  
]

讓大多數用來分散的 (c_j) 對每個較小模數 (c_i) 來說來自長區間，便較容易證。但映射 (\Phi) 對所有頂點對稱；若大小差距太大，character 數與稠密層會受影響。這可能是下一個調整方向。

---

## 7. 這裏出現一個新策略：分塊而非支撐分層

與其把 character 分成稀疏/稠密，不如把輔助素數分成幾個大小層級：

[  
C^{(1)},C^{(2)},\dots,C^{(m)}.  
]

讓每一層的素數都遠大於前一層。當要平衡某一層時，主要用更高層的素數作鄰居。這樣對模 (p) 而言，鄰居素數 (q) 跑過很多完整 residue cycles，倒數分散性更容易用解析數論保證。

這會把原本對稱的 (\Phi) 改成一個**上三角封口系統**。

也就是說，不再要求所有輔助頂點同時由同一批內部邊平衡，而是分層處理：

第一層端口的殘差，用第二層素數補；  
第二層由第三層補；  
……  
最後一層用某種特殊封口 gadget 關閉。

這樣就避免了同階互逆分佈的難題。

但代價是：每往上一層，素數大小會增長；不能無限層，否則又變成「日取其半」。所以需要有限層，最好兩層或三層。

---

## 8. 我現在認為最應調整的方向

我們仍保留 Fourier / 模圖主路，但改為：

[  
\boxed{  
\text{分塊上三角封口，而不是完全對稱滿射。}  
}  
]

具體設想：

給定一批待平衡端口 (B)。選一批較大素數 (C)，使 (C) 對每個 (p\in B) 的倒數模 (p) 分散，從而可用 (B-C) 邊滿足 (B) 的殘差。

這會在 (C) 上造成新殘差。再選更大一批 (D)，用 (C-D) 邊滿足 (C) 的殘差。如此下去。

如果每次用更大的素數層來平衡上一層，則每一步是「二部圖映射」：

[  
\Psi_{A,B}:{0,1}^{A\times B}\to\prod_{a\in A}\mathbb F_a,  
]

[  
\Psi_{A,B}(E)_a=  
\sum_{b\in B,, ab\in E}\frac1b\pmod a.  
]

這個映射比原來的 (\Phi) 簡單很多，因為每條邊只需要滿足上一層的坐標；下一層的殘差留待以後處理。

對固定上一層 (A)，若下一層 (B) 的素數在每個模 (a) 下倒數分散，則 (\Psi_{A,B}) 很可能滿射。這是一個標準子集和問題，而非對稱圖系統。

這比完全 Fourier 滿射更可證。

---

## 9. 但最後一層怎麼封口？

這是此策略的核心風險。若每一層都把殘差推到下一層，最後必須有一個終止裝置；否則又是無窮後退。

有兩種可能：

第一，用最後一層內部的對稱 Fourier 滿射封口。  
但最後一層可以取得很大、很分散，並且不再需要所有層同時控制。這比原來整體封口容易。

第二，設計一個特殊的零殘差層，使最後一層自動平衡。  
例如選最後一層 (Z) 並在其中取完整圖或某種規則圖，使對每個 (z\in Z)：

[  
\sum_{w\in Z,\ w\ne z}\frac1w\equiv -\beta_z\pmod z.  
]

這仍是封口問題，但規模集中在最後一層。

所以分塊法不是完全解決封口，而是把困難推到一個更可控的終端。

---

## 10. 目前最合理的前進方向

我現在會把主路修正為：

[  
\boxed{  
\text{用多層二部子集和先處理邊界殘差，最後用一個對稱 Fourier 封口處理終端層。}  
}  
]

這比原先「一大坨輔助素數上完全滿射」更合理，因為：

第一，前面各層只需處理二部子集和，容易得多；  
第二，素數大小可分層，倒數分散更容易保證；  
第三，最後的對稱封口只處理最後一層，不必同時承擔所有邊界條件；  
第四，若失敗，失敗點會明確落在終端封口，而不是整個系統混在一起。

---

## 這一步的結論

我們推進後得到一個重要調整：

[  
\boxed{  
\text{完全對稱 Fourier 滿射太硬；稀疏層可控，但選素數分散不易。}  
}  
]

因此應改成：

[  
\boxed{  
\text{分層上三角封口：用大素數層逐層消除前一層殘差，最後集中封口。}  
}  
]

這不是放棄 Fourier 路線，而是把它改成更可操作的形態。下一步應證的不是原來的互逆分散引理，而是更簡單的二部滿射引理：

[  
\boxed{  
\Psi_{A,B}:{0,1}^{A\times B}\to\prod_{a\in A}\mathbb F_a,\qquad  
(\Psi E)_a=\sum_{b\in B_a}\frac1b\pmod a  
}  
]

在 (B) 足夠大且對每個 (a\in A) 倒數分散時，是否滿射。

這個二部引理比對稱 (\Phi) 易得多，也更可能成為真正可證的第一個核心工具。
