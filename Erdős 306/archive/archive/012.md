好，繼續。現在我們專攻二部滿射引理。這是目前最值得先打下的一塊磚。

設 (A) 是一層已經帶有殘差、需要被平衡的素數集合：

[  
A={a_1,\dots,a_m}.  
]

我們選下一層較大的輔助素數集合

[  
B={b_1,\dots,b_n}.  
]

只允許 (A) 與 (B) 之間連邊。若選邊集 (E\subseteq A\times B)，則對每個 (a\in A)，它收到的補償是

[  
\sum_{b:,(a,b)\in E}\frac1b\pmod a.  
]

所以得到映射

[  
\Psi_{A,B}:{0,1}^{A\times B}\to\prod_{a\in A}\mathbb F_a,  
]

# [  
\Psi_{A,B}(E)_a

\sum_{b:,(a,b)\in E}\frac1b\pmod a.  
]

注意這個映射比前面的對稱圖映射簡單得多：每條邊 ((a,b)) 只影響一個舊頂點 (a) 的方程。它不會同時影響另一個舊方程。當然，它會在 (b\in B) 上造成新殘差，但那留給下一層處理。

因此這一步的目標只是：

[  
\boxed{  
\text{給定任意目標殘差 }(\gamma_a)_{a\in A},  
\text{能否選 }A\text{-}B\text{ 邊，使 }  
\sum_{b\in E(a)}b^{-1}\equiv\gamma_a\pmod a？  
}  
]

這其實分解成每個 (a) 的獨立問題。因為不同 (a) 使用的邊 ((a,b)) 不衝突；同一個 (b) 可以同時連到多個不同 (a)，分母分別是 (ab)，仍然互異。

所以二部滿射引理可以降為一個一維子集和命題：

---

## 一維子集和引理

給定素數 (a)，若集合

[  
R_a(B)={b^{-1}\bmod a:b\in B}  
\subseteq\mathbb F_a  
]

的子集和覆蓋整個 (\mathbb F_a)，那麼對任何殘差 (\gamma_a\in\mathbb F_a)，都可以選一部分 (B_a\subseteq B)，使

[  
\sum_{b\in B_a}\frac1b\equiv\gamma_a\pmod a.  
]

因此，只要對每個 (a\in A)，(R_a(B)) 的子集和覆蓋 (\mathbb F_a)，二部映射 (\Psi_{A,B}) 就滿射。

這非常好：我們不再需要高維 Fourier，只需要證一維有限域中的子集和覆蓋。

---

## 一個足夠條件：有足夠多不同 residue

在 (\mathbb F_a) 中有一個基本事實：

若 (S\subseteq\mathbb F_a) 且

[  
|S|>\sqrt{2a},  
]

那麼所有子集和

[  
\Sigma(S)=\left{\sum_{x\in T}x:T\subseteq S\right}  
]

很可能已經覆蓋 (\mathbb F_a)。更精確地，有 Olson 類定理：在循環群 (\mathbb Z/p\mathbb Z) 中，若 (S) 不含 (0)，且元素互異，當 (|S|) 約大於 (\sqrt{2p}) 時，子集和可覆蓋整個群；相關結果常以 Olson constant 或 additive basis 形式出現。

即使不用最尖銳常數，我們也可以使用較粗條件：若 (R_a(B)) 在 (\mathbb F_a) 中包含足夠長、足夠分散的一批非零元素，那麼其子集和覆蓋 (\mathbb F_a)。

所以二部引理可由下面較具體的條件推出：

[  
\boxed{  
\forall a\in A,\quad  
|R_a(B)|\ge C\sqrt a  
\quad\Longrightarrow\quad  
\Psi_{A,B}\text{ 滿射。}  
}  
]

這已經比前面的 Fourier 滿射容易很多。

---

## 如何讓 (R_a(B)) 足夠大？

因為

[  
b\mapsto b^{-1}\pmod a  
]

是 (\mathbb F_a^\times) 上的雙射，所以

[  
|R_a(B)|=|{b\bmod a:b\in B}|.  
]

也就是說，我們只要 (B) 中的素數在模 (a) 下佔據足夠多個不同非零剩餘類即可。

這比要求倒數分佈均勻弱得多。只需很多不同 residue，不需精確均勻。

因此可以提出：

**二部滿射引理，實用版。**  
設 (A) 是有限素數集合。若可選素數集合 (B)，使對每個 (a\in A)，

[  
|{b\bmod a:b\in B}|\ge C\sqrt a,  
]

則對任意殘差 ((\gamma_a)_{a\in A})，存在邊集 (E\subseteq A\times B)，使

[  
\sum_{b:,(a,b)\in E}\frac1b\equiv\gamma_a\pmod a  
]

對所有 (a\in A) 成立。

這一步是真進展：它把一層平衡問題降成了「下一層素數在上一層每個模數下佔據足夠多 residue」。

---

## 這件事能做到嗎？

若 (B) 取得夠大，當然可以。比如對每個 (a\in A)，我們需要 (C\sqrt a) 個不同 residue。若 (B) 是一段足夠長的素數區間，素數在模 (a) 的非零 residue classes 中應該會出現很多類。

更強的是，若我們不太在乎 (B) 大小，甚至可用 Dirichlet 定理直接為每個 (a) 和每個想要的 residue class 找素數。但若同時要控制所有 (a\in A)，就要避免 CRT 導致素數暴漲。

不過現在情況比前面好得多，因為我們不需要為每對 ((a,b)) 指定 residue；只需要 (B) 整體在每個模 (a) 下有足夠多不同 residue。這是素數自然分佈就很可能滿足的條件。

所以二部步驟應該是相當穩的。

---

## 但這一步會產生什麼新殘差？

若我們用邊 ((a,b)) 平衡了 (a\in A)，那麼在新頂點 (b\in B) 上會產生殘差

# [  
\rho_b

\sum_{a:,(a,b)\in E}\frac1a\pmod b.  
]

也就是說，(A) 被清零了，但 (B) 變成了新的待平衡層。

這正是上三角封口：

[  
A_0\to A_1\to A_2\to\cdots.  
]

每一步用 (A_{r+1}) 平衡 (A_r)，然後殘差轉移到 (A_{r+1})。

二部滿射引理保證：只要下一層 (A_{r+1}) 在上一層每個模數下有足夠 residue，我們總能清掉上一層。

---

## 問題變成：如何終止？

目前我們能做到：

[  
\text{殘差從 }A_0\text{ 推到 }A_1,  
\quad  
A_1\text{ 推到 }A_2,  
\quad  
A_2\text{ 推到 }A_3,\dots  
]

但若沒有終止機制，這就只是把灰塵掃到下一個房間。

所以主問題現在縮成：

[  
\boxed{  
\text{能否用有限多層，最後把末層殘差封閉？}  
}  
]

這比原問題小多了。因為前面各層都可以用一維子集和處理；全部困難集中到「末層封口」。

末層封口可以有兩種路線。

---

## 終止方案一：最後用對稱 Fourier 封口

選最後一層 (Z) 足夠大，在 (Z) 內部選邊，使每個 (z\in Z) 的殘差被消掉：

[  
\sum_{w\sim z}\frac1w\equiv -\rho_z\pmod z.  
]

這就是前面的對稱映射 (\Phi)，但現在它只作用於最後一層，不需要同時處理原始邊界。技術上仍難，但乾淨許多。

然而，這仍要證一個對稱滿射或足夠強的命中定理。它可能還是最硬的一步。

---

## 終止方案二：設計最後一層殘差為零

更巧的是：在倒數第二步，不只是任意選邊來平衡上一層，而是額外要求在最後一層產生的殘差恰好為零。

也就是說，對二部圖 (Y-Z)，我們同時要求：

對 (y\in Y)：

[  
\sum_{z:,(y,z)\in E}\frac1z\equiv\gamma_y\pmod y;  
]

對 (z\in Z)：

[  
\sum_{y:,(y,z)\in E}\frac1y\equiv0\pmod z.  
]

這又是二部雙邊問題。不過它比一般對稱圖更簡單，因為所有邊都跨層，沒有層內邊。

這可稱為：

[  
\boxed{\text{二部封口問題。}}  
]

如果二部封口問題可解，那就完全不需要對稱 Fourier 封口。

---

## 二部封口問題是否更容易？

設 (Y={y_1,\dots,y_m})，(Z={z_1,\dots,z_n})。邊變量是

[  
x_{ij}\in{0,1}.  
]

方程是：

對每個 (y_i)：

[  
\sum_j x_{ij}z_j^{-1}\equiv\gamma_i\pmod {y_i};  
]

對每個 (z_j)：

[  
\sum_i x_{ij}y_i^{-1}\equiv0\pmod {z_j}.  
]

這仍是 (0/1) 線性系統，但比一般圖好，因為是矩陣形式：行約束與列約束分別在不同模數中。

如果 (n) 遠大於 (m)，行約束容易滿足；列約束較難，因為每列只有 (m) 個可選項。若 (z_j) 很大，而 (2^m<z_j)，則列中所有可能子集和不到 (z_j) 種，不可能任意清零。但我們只要求清零，不要求任意值；仍可能可安排。

一個簡單辦法是讓每個 (z_j) 的列要麼全不選，要麼選一個預先設計的子集 (I_j\subseteq Y)，使

[  
\sum_{i\in I_j}\frac1{y_i}\equiv0\pmod {z_j}.  
]

也就是選 (z_j) 為某個分子因子的素因子：

[  
z_j\mid \sum_{i\in I_j}\frac{Y_0}{y_i},  
\qquad Y_0=\prod_i y_i.  
]

這回到了結構化設計：用 (Y) 的子集和產生可用的末層素數 (z_j)。

這很有意思。它把終止問題變成：

[  
\boxed{  
\text{從 }Y\text{ 的倒數子集和中挑出很多素因子作 }Z。  
}  
]

這比完全隨機 Fourier 更具體，但也需要素因子分佈知識。

---

## 目前最清楚的進展鏈

我們現在有一個更強的結構框架：

1. 原問題歸約為可避障表示 (1/b)。
    
2. 表示 (1/b) 歸約為素數圖局部模平衡。
    
3. 局部模平衡可用分層方式處理。
    
4. 每個非終端層的平衡由二部滿射引理處理。
    
5. 二部滿射引理可由一維子集和覆蓋推出。
    
6. 真正剩下的是終端封口問題。
    

也就是說，原問題的難點已被集中到：

[  
\boxed{  
\text{如何有限終止殘差傳遞？}  
}  
]

這比「原題整體很難」更具體。

---

## 下一步我會選哪個終止方案？

我認為最應繼續攻的是**二部封口**，不是回到對稱 Fourier。

理由：

第一，二部封口和前面的分層框架一致。  
第二，它保留矩陣結構，可能比一般圖易分析。  
第三，它可能與一維子集和、Olson 型結果結合。  
第四，如果二部封口可解，整條證明會非常乾淨：全程只用跨層邊，最後沒有亂圖。

所以新的核心命題是：

[  
\boxed{  
\textbf{二部封口引理：}  
}  
]

給定一層素數 (Y) 及其待補殘差 ((\gamma_y))，能否選一層新素數 (Z) 與邊集 (E\subseteq Y\times Z)，使：

[  
\sum_{z:,(y,z)\in E}\frac1z\equiv\gamma_y\pmod y  
\quad(y\in Y),  
]

同時

[  
\sum_{y:,(y,z)\in E}\frac1y\equiv0\pmod z  
\quad(z\in Z)?  
]

如果此引理成立，原問題就基本打通。

下一步就應該正面研究這個二部封口引理。它現在看起來是整條路線的真正核心。
