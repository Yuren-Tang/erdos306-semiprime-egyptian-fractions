我們繼續，這一步我覺得有一個真正的轉折：前面把終端封口說成「找 (z\mid N_I) 的素因子」，其實是把 (z) 當作由 (I) 事後產生。這樣會陷入不可控的素因子分佈。更好的做法是反過來：

[  
\boxed{\text{先選很多終端素數 }z,\text{ 再為每個 }z\text{ 選一個能使它平衡的鄰接子集 }I_z\subseteq Y。}  
]

這樣終端封口變成一個乾淨的「帶列約束的二部矩陣問題」，比研究 (N_I) 的素因子要可控得多。

---

## 1. 終端封口的正確形式

末層是

[  
Y={y_1,\dots,y_m}.  
]

我們再選一批終端素數

[  
Z={z_1,\dots,z_n},  
]

只使用 (Y-Z) 之間的邊。對每個 (z\in Z)，為了讓 (z) 自己平衡，需要

[  
\sum_{y:,yz\in E}\frac1y\equiv0\pmod z.  
]

也就是說，對每個 (z)，它連到的子集 (I_z\subseteq Y) 必須滿足

[  
\boxed{  
\sum_{y\in I_z}\frac1y\equiv0\pmod z.  
}  
]

在這個條件下，(z) 不會留下新殘差。

而它對每個 (y\in Y) 的貢獻是：

[  
\frac1z\pmod y  
]

如果 (y\in I_z)，否則貢獻 (0)。

所以終端封口問題變成：

> 能否選擇很多素數 (z)，並為每個 (z) 選一個滿足  
> [  
> \sum_{y\in I_z}1/y\equiv0\pmod z  
> ]  
> 的子集 (I_z)，使得在每個 (y\in Y) 上得到指定殘差？

形式化地，給定目標

[  
\gamma=(\gamma_y)_{y\in Y}\in\prod_{y\in Y}\mathbb F_y,  
]

要找 (I_z\subseteq Y)，使

[  
\sum_{z:,y\in I_z}\frac1z\equiv\gamma_y\pmod y  
]

對所有 (y\in Y) 成立，且每個 (z) 同時滿足列約束

[  
\sum_{y\in I_z}\frac1y\equiv0\pmod z.  
]

這比「(z\mid N_I)」好，因為 (z) 是我們先選的；約束只是 (I_z) 落在一個核裏。

---

## 2. 矩陣表述

令變量

[  
x_{y,z}\in{0,1},  
]

表示是否選邊 (yz)。

列約束是，對每個 (z\in Z)：

[  
\sum_{y\in Y}x_{y,z},y^{-1}\equiv0\pmod z.  
]

行目標是，對每個 (y\in Y)：

[  
\sum_{z\in Z}x_{y,z},z^{-1}\equiv\gamma_y\pmod y.  
]

這就是一個帶列約束的 (0/1) 矩陣問題：

[  
\boxed{  
\begin{cases}  
\sum_y x_{y,z}y^{-1}\equiv0\pmod z,& z\in Z,\[2mm]  
\sum_z x_{y,z}z^{-1}\equiv\gamma_y\pmod y,& y\in Y.  
\end{cases}  
}  
]

這個形式很漂亮。它把「終端吸收」從不可控的素因子問題，變成了受約束的隨機矩陣問題。

---

## 3. 為什麼這比前面的終端封口更好？

對固定 (z)，可用的列向量是

[  
K_z=\left{x\in{0,1}^Y:  
\sum_{y\in Y}x_y y^{-1}\equiv0\pmod z  
\right}.  
]

若 (m=|Y|) 大，而 (z) 不太大，比如

[  
z\ll 2^m,  
]

那麼 (K_z) 通常應該很大，約有

[  
|K_z|\approx\frac{2^m}{z}  
]

個元素。也就是說，每個終端素數 (z) 有大量可選平衡子集，不是稀少偶然事件。

這是核心改善。以前用 (z\mid N_I) 的角度，好像必須碰運氣找素因子；現在反過來看，對一個不太大的 (z)，在 (2^m) 個子集中，落入核的應該很多。

---

## 4. 終端吸收分佈

現在可以定義一個隨機過程：

對每個 (z\in Z)，從

[  
K_z  
]

中均勻隨機選一列

[  
x^{(z)}=(x_{y,z})_{y\in Y}.  
]

這些列彼此獨立。

它們在 (Y) 上產生殘差

[  
R_y=\sum_{z\in Z}x_{y,z}z^{-1}\pmod y.  
]

得到一個分佈

[  
R\in G_Y:=\prod_{y\in Y}\mathbb F_y.  
]

如果這個分佈接近均勻，那麼對任意目標 (\gamma)，都有正概率命中：

[  
R=\gamma.  
]

於是終端封口完成。

所以終端封口的核心變成：

[  
\boxed{  
\text{受列約束的隨機矩陣，其行殘差分佈是否接近均勻？}  
}  
]

這比原來的問題清楚得多。

---

## 5. Fourier 形式

對 character

[  
\lambda=(\lambda_y)_{y\in Y}\in G_Y,  
]

考慮 Fourier 係數

# [  
\widehat\nu(\lambda)

\mathbb E\exp\left(  
2\pi i  
\sum_{y\in Y}  
\frac{\lambda_y R_y}{y}  
\right).  
]

因為不同 (z) 的列獨立，

# [  
\widehat\nu(\lambda)

\prod_{z\in Z}F_z(\lambda),  
]

其中

# [  
F_z(\lambda)

\mathbb E_{x\in K_z}  
\exp\left(  
2\pi i  
\sum_{y\in Y}  
x_y\frac{\lambda_y z^{-1}}{y}  
\right).  
]

若對每個非零 (\lambda)，有許多 (z) 使

[  
|F_z(\lambda)|\le 1-\epsilon,  
]

則

[  
|\widehat\nu(\lambda)|  
]

會被指數壓小；最後可用 Fourier inversion 證明 (R) 近似均勻。

所以終端封口的 Fourier 核心是：

[  
\boxed{  
\text{對非零 }\lambda,\text{ 隨機列 }x\in K_z  
\text{ 不能讓相位幾乎固定。}  
}  
]

---

## 6. (F_z(\lambda)) 可以怎麼估？

用列約束的指示函數展開：

# [  
1_{K_z}(x)

\frac1z\sum_{\mu\in\mathbb F_z}  
\exp\left(  
2\pi i\frac{\mu}{z}  
\sum_{y\in Y}x_y y^{-1}  
\right).  
]

因此

# [  
F_z(\lambda)

\frac{  
\sum_{\mu\in\mathbb F_z}  
\prod_{y\in Y}  
\left(  
1+  
e^{2\pi i\left(  
\frac{\lambda_y z^{-1}}{y}  
+  
\frac{\mu y^{-1}}{z}  
\right)}  
\right)  
}{  
\sum_{\mu\in\mathbb F_z}  
\prod_{y\in Y}  
\left(  
1+  
e^{2\pi i\frac{\mu y^{-1}}{z}}  
\right)  
}.  
]

這個式子看起來複雜，但它有一個直觀含義：

如果 (K_z) 在 ({0,1}^Y) 中分佈得足夠均勻，那麼從 (K_z) 中隨機選 (x) 幾乎像獨立 Bernoulli 變量，只是加了一個線性條件。於是

[  
F_z(\lambda)  
]

應該接近無約束情況：

[  
\prod_{y\in Y}  
\frac{1+  
e^{2\pi i\lambda_y z^{-1}/y}  
}{2}.  
]

而無約束情況只要 (\lambda\ne0)，通常就會有一些坐標造成衰減。

這裏比原先對稱 Fourier 好很多：每個 (z) 是一列，列與列獨立；沒有邊同時影響兩個終端方程的糾纏。

---

## 7. 一個可行的充分條件

為了避免處理上面完整公式，可以提出一個足夠條件：

對每個 (z\in Z)，列核 (K_z) 對小坐標投影近似均勻。也就是說，對任意不太大的 (T\subseteq Y)，投影

[  
K_z\to{0,1}^T  
]

幾乎均勻。

這在什麼情況下成立？如果剩餘坐標 (Y\setminus T) 的子集和

[  
\sum_{y\in Y\setminus T}x_y y^{-1}\pmod z  
]

覆蓋 (\mathbb F_z) 很均勻，那麼給定 (T) 上的任意取值，都有差不多一樣多的方式在補集上補成列約束。

因此需要：

[  
\boxed{  
{y^{-1}\bmod z:y\in Y\setminus T}  
\text{ 的子集和在 }\mathbb F_z\text{ 中分佈近均勻。}  
}  
]

這又是一維子集和分佈問題。若

[  
|Y|\gg \log z  
]

且 residues (y^{-1}\bmod z) 不集中，這通常應該成立。

所以終端吸收可被推到一個非常標準的命題：

[  
\boxed{  
\text{許多非零 residue 的隨機 }0/1\text{ 子集和在 }\mathbb F_z\text{ 中近均勻。}  
}  
]

這比原始 Erdős 問題小得多。

---

## 8. 參數選擇

設

[  
m=|Y|.  
]

若選終端素數

[  
z\le 2^{\alpha m}  
]

且 (\alpha<1)，那麼每個核

[  
K_z  
]

的平均大小約為

[  
2^m/z\ge 2^{(1-\alpha)m},  
]

很大。

同時，行殘差空間大小是

[  
|G_Y|=\prod_{y\in Y}y.  
]

若 (Y) 中的素數大小不超過 (e^{\eta m})，則

[  
\log |G_Y|\le \eta m^2.  
]

若終端素數 (Z) 有 (n) 個，每列可帶來大量隨機性。只要 (n) 與 (m) 成比例或稍大，總自由度約 (nm)，有機會覆蓋 (G_Y)。

所以合理參數是：

[  
|Y|=m,\qquad |Z|=Cm,\qquad z\le e^{\alpha m},  
\qquad y\le e^{\eta m},  
]

其中 (\alpha,\eta) 小常數。

這保持所有素數大小在指數級，不至於讓 Fourier character 數失控。

---

## 9. 得到的條件性終端吸收定理

我們可以正式提出：

**條件性終端吸收定理。**  
設 (Y,Z) 為兩組互異素數。若：

1. 對每個 (z\in Z)，核  
    [  
    K_z={x\in{0,1}^Y:\sum_y x_y/y\equiv0\pmod z}  
    ]  
    很大且對坐標投影足夠混合；
    
2. 對每個非零 (\lambda\in G_Y)，有足夠多 (z\in Z) 使  
    [  
    |F_z(\lambda)|\le1-\epsilon;  
    ]
    

則由平衡列生成的吸收集等於整個

[  
G_Y=\prod_{y\in Y}\mathbb F_y.  
]

也就是：任意末層殘差都可被終端 (Z) 封掉。

這雖是條件定理，但比之前更接近可證，因為條件都是一維子集和混合性，而不是神秘的素因子分佈。

---

## 10. 這一步的意義

我們把終端封口從：

[  
z\mid N_I\text{ 的素因子問題}  
]

改寫成：

[  
\boxed{  
\text{對每個預選終端素數 }z，  
\text{從核 }K_z\text{ 中選一列。}  
}  
]

這是實質進展。因為：

第一，避免了不可控的 (N_I) 素因子分佈；  
第二，終端封口變成受約束隨機矩陣問題；  
第三，核心估計變成有限域中的子集和混合；  
第四，它與前面分層二部傳遞完全銜接。

所以現在整條路線變成：

[  
\boxed{  
\begin{aligned}  
&\text{原始殘差}\  
&\xrightarrow{\text{二部滿射}}  
\text{推到末層 }Y\  
&\xrightarrow{\text{受約束隨機矩陣吸收}}  
\text{用終端層 }Z\text{ 封掉。}  
\end{aligned}  
}  
]

下一步最該攻的是一個具體有限域命題：

[  
\boxed{  
\text{若 }Y\text{ 在模 }z\text{ 下倒數分散且 }|Y|\gg\log z，  
\text{則 }K_z\text{ 的投影近似均勻。}  
}  
]

這個命題若成立，終端吸收就有真正希望。
