繼續。這一步我必須先把上一輪的錯誤判斷徹底修正，因爲這裏露出的不是小麻煩，而是整條路線的真正形狀。

前面我說：若末層 (Y) 的倒數質量

[  
H(Y)=\sum_{y\in Y}\frac1y  
]

足夠大，或許可以殺掉對角 character。這個說法不準確，甚至方向上有誤。因爲 Fourier 係數的模長不由平均值控制，而由波動控制。對隨機子集

[  
S(x)=\sum_{y\in Y}\frac{x_y}{y},  
]

雖然

[  
\sum_{y\in Y}\frac1y  
]

可以發散，但方差由

[  
\sum_{y\in Y}\frac1{y^2}  
]

控制，而

[  
\sum_{p\ \mathrm{prime}}\frac1{p^2}  
]

是收斂的。也就是說，加入越來越多大素數，總平均會慢慢長大，但隨機波動不會無限長大。這使得「靠大倒數質量殺對角 Fourier」這個想法不能直接成立。

這不是壞消息，而是非常重要的結構發現。

---

## 一、對角 character 其實不是敵人，而是不變量

末層 (Y)，終端素數 (z)，列核為

[  
K_z=\left{x\in{0,1}^Y:  
\sum_{y\in Y}x_y y^{-1}\equiv0\pmod z  
\right}.  
]

終端列對 (Y) 上殘差的貢獻是

[  
R_y(x)=x_y z^{-1}\pmod y.  
]

若取對角 character

[  
\lambda_y=h  
]

對所有 (y\in Y)，則相位為

[  
\sum_{y\in Y}x_y\frac{h(z^{-1}\bmod y)}{y}.  
]

利用 CRT 恆等式：

[  
\frac{z^{-1}\bmod y}{y}  
+  
\frac{y^{-1}\bmod z}{z}  
\equiv  
\frac1{yz}\pmod1,  
]

所以

## [  
\sum_{y}x_y\frac{h(z^{-1}\bmod y)}y  
\equiv  
h\sum_y\frac{x_y}{yz}

\frac hz\sum_y x_y(y^{-1}\bmod z)  
\pmod1.  
]

而 (x\in K_z) 使第二項為整數，因此

[  
\boxed{  
\sum_{y}x_y\frac{h(z^{-1}\bmod y)}y  
\equiv  
h\sum_y\frac{x_y}{yz}  
\pmod1.  
}  
]

右邊正是這一列實際貢獻的真實大小乘以 (h)。

所以對角 character 不是普通噪聲；它探測的是：

[  
\boxed{\text{終端列的實際有理數大小。}}  
]

這意味著：若我們只看有限群殘差

[  
\prod_{y\in Y}\mathbb F_y,  
]

就一定會遇到假障礙。因爲終端殘差不是任意均勻分佈；它還帶着一個「實數大小」座標。

---

## 二、因此終端吸收不能要求「殘差均勻」

原先想證：

[  
R\in\prod_{y\in Y}\mathbb F_y  
]

接近均勻。這是錯的，因爲對角 character 會阻止它均勻。

正確目標應該是：終端吸收在「殘差 + 實際大小」的聯合空間中均勻，或者更準確地說，均勻分佈在一個由 CRT 決定的圖像上。

定義終端邊集的實際大小：

[  
M=\sum_{z\in Z}\sum_{y\in Y}\frac{x_{y,z}}{yz}.  
]

同時定義殘差向量：

[  
R_y=\sum_{z\in Z}x_{y,z}z^{-1}\pmod y.  
]

則由剛才的 CRT 恆等式可得：

[  
\boxed{  
\sum_{y\in Y}\frac{R_y}{y}  
\equiv  
M  
\pmod1.  
}  
]

這是一條必然關係。  
所以 (R) 不可能在整個有限群中均勻；它必須與 (M) 綁在一起。

這就是對角 character 不衰減的真正原因：它不是反例，而是在提醒我們少了一個座標。

---

## 三、修正後的正確模型：殘差—質量聯合分佈

終端吸收應研究隨機變量

[  
(R,M),  
]

其中

[  
R\in G_Y:=\prod_{y\in Y}\mathbb F_y,  
\qquad  
M\in\mathbb R/\mathbb Z.  
]

對 character 應考慮

[  
(\lambda,\tau),  
]

其中

[  
\lambda\in G_Y,\qquad \tau\in\mathbb Z.  
]

Fourier 相位為

[  
\sum_{y\in Y}\frac{\lambda_y R_y}{y}  
+  
\tau M.  
]

對一列 (z)，相位變成

[  
\sum_{y\in Y}  
x_y\left(  
\frac{\lambda_y(z^{-1}\bmod y)}y  
+  
\frac{\tau}{yz}  
\right).  
]

若 (\lambda_y=h) 對所有 (y)，並取

[  
\tau=-h,  
]

那麼由 CRT 恆等式和列約束，這個相位會完全抵消。也就是說，((\lambda,\tau)) 落在一條「對角不變量」上。

這不是要殺掉的 Fourier 項，而是描述支撐所在子群的主項。

因此新的目標不是：

[  
R\text{ 在 }G_Y\text{ 中均勻},  
]

而是：

[  
\boxed{  
(R,M)\text{ 在滿足 }  
M\equiv\sum_y R_y/y\pmod1  
\text{ 的圖像上均勻。}  
}  
]

這是本路線真正的形狀。它已經有點像 circle method：非主弧 Fourier 要衰減；主弧對角項保留，給出正確的局部條件。

## 四、這如何回到原題？

原題最終要證明：

[  
\sum \frac1{pq}=\frac1b.  
]

前面用模條件的方法說：若所有局部同餘對，且總和在 (0) 到 (1) 之間，則等式成立。現在我們看到，局部同餘和實際大小不是兩件獨立的事；對角 character 正在把它們連起來。

因此修正後的終端任務不是「給任意殘差 (\gamma)，封掉它」，而是：

給定前面層留下的殘差 (\gamma) 以及剩餘應補的真實大小 (m)，若它們滿足必要相容條件

[  
m\equiv \sum_{y\in Y}\frac{\gamma_y}{y}\pmod1,  
]

則終端吸收應能找到邊集，使

[  
R=\gamma,\qquad M=m.  
]

因爲 (m) 是由前面已選邊後剩餘的大小決定的；而相容條件正是全局 CRT 同餘。

這比「任意殘差皆可封」更正確，也更難。

## 五、現在真正的 Fourier 分層

修正後，Fourier character 應分為：

### 1. 對角主項

即

[  
\lambda_y=h,\qquad \tau=-h.  
]

這類不應衰減；它對應必要相容條件。

### 2. 近對角項

[  
\lambda_y  
]

在多數 (y) 上接近同一個整數 (h)，而 (\tau\approx-h)。  
這類是 major arcs，需要精細估計。

### 3. 非對角項

[  
\lambda_y  
]

沒有此共同結構。這類應由反聚集殺掉。

所以不是「稠密 character 全部殺掉」，而是：

[  
\boxed{  
\text{殺掉所有非對角 character；保留對角 character 作為局部相容主項。}  
}  
]

這是重大修正。

## 六、這比前一版更有希望

前一版錯在企圖證明不可能的均勻性。  
新版只要求在正確的支撐上均勻。這與許多加法數論問題的典型結構一致：你不能消滅所有 Fourier 項，因為有局部障礙；你要證明除了局部障礙以外沒有別的障礙。

這正好呼應原 Erdős 問題的本質：

[  
b\text{ squarefree}  
]

就是已知局部障礙。  
我們現在發現終端吸收中還有一個表面上的「對角障礙」，但它其實不是新障礙，而是全局等式本身的 CRT 影子。

所以路線並未死亡；它變得更正確了。

## 七、下一步應做什麼？

現在最該證的是一個「非對角衰減引理」：

令 (Y,Z) 為適當選取的素數層。對任意 character pair

[  
(\lambda,\tau)  
]

若它不接近對角形式

[  
\lambda_y=h\quad\text{對大多數 }y,\qquad \tau=-h,  
]

則對許多 (z\in Z)，有

[  
\left|  
\mathbb E_{x\in K_z}  
e\left(  
\sum_y x_y  
\left[  
\frac{\lambda_y(z^{-1}\bmod y)}y  
+  
\frac{\tau}{yz}  
\right]  
\right)  
\right|  
\le 1-\epsilon.  
]

若能證這個，終端吸收就會變成標準的「主項 + 小誤差」形式。

換言之，下一個核心命題是：

[  
\boxed{  
\textbf{非對角反聚集引理：}  
\text{除 CRT 對角不變量外，終端列沒有其他大 Fourier 係數。}  
}  
]

這纔是正確靶子。

## 八、到目前為止真正推進到哪裏？

我們已經從原題得到一條相當清楚的研究路線：

[  
\begin{aligned}  
&\text{Erdős 原問題}\  
\Rightarrow;&\text{可避障表示 }1/b\  
\Rightarrow;&\text{素數圖局部模平衡}\  
\Rightarrow;&\text{分層二部殘差傳遞}\  
\Rightarrow;&\text{終端殘差—質量吸收}\  
\Rightarrow;&\text{非對角 Fourier 反聚集。}  
\end{aligned}  
]

其中我們剛剛修正了最關鍵的一點：**終端吸收不能只看有限殘差，必須同時看實際有理數大小。**

這個修正避免了假均勻性，也把對角 character 從「障礙」變成了「主項」。

所以接下來真正該攻的是：

[  
\boxed{  
\text{證明非對角 character 衰減；對角 character 給出唯一主項。}  
}  
]

這一步若能推通，整條 Fourier–分層路線就不再只是腳手架，而會真正接近原問題的核心。
