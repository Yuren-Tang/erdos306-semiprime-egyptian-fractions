好。現在把上一輪的分析正式合併成一個條件性定理。這一步的目標是：**把終端封口問題從模糊的構造困難，壓縮為兩個明確分散假設。**

---

設末層素數集合為

[  
Y={y_1,\dots,y_m},  
]

終端素數集合為

[  
Z={z_1,\dots,z_n},  
]

且 (Y\cap Z=\varnothing)。我們只使用跨邊 (Y-Z)。對每個 (z\in Z)，選一個子集

[  
I_z\subseteq Y  
]

作為 (z) 的鄰居。為了讓 (z) 自身平衡，必須有

[  
\sum_{y\in I_z}y^{-1}\equiv0\pmod z.  
]

換成 (0/1) 變量 (x_{y,z})，列核為

[  
K_z=\left{x\in{0,1}^Y:  
\sum_{y\in Y}x_y y^{-1}\equiv0\pmod z  
\right}.  
]

若從每個 (K_z) 中選一列 (x^{(z)})，則它在 (Y) 上造成殘差

[  
R_y=\sum_{z\in Z}x^{(z)}_y z^{-1}\pmod y,  
]

並造成實際質量

[  
M=\sum_{z\in Z}\sum_{y\in Y}\frac{x^{(z)}_y}{yz}.  
]

終端吸收要做到的是：給定某個殘差—質量目標 ((\gamma,m))，找到列選擇，使

[  
R_y=\gamma_y\quad(y\in Y),  
]

且

[  
M=m.  
]

嚴格說 (M) 是一個有理數，不只是模 (1)；但在前面整個 CRT 框架中，先控制模 (1) 的相容性，再配合大小區間，可以把它轉回精確等式。因此終端 Fourier 先研究

[  
M\pmod1.  
]

---

## 1. 必要相容條件

由 CRT 恆等式可知，對每條邊 (yz)，

[  
\frac{z^{-1}\bmod y}{y}  
+  
\frac{y^{-1}\bmod z}{z}  
\equiv  
\frac1{yz}\pmod1.  
]

而每列 (z) 滿足

[  
\sum_y x_{y,z}(y^{-1}\bmod z)\equiv0\pmod z.  
]

因此總體上有

[  
\boxed{  
M\equiv \sum_{y\in Y}\frac{R_y}{y}\pmod1.  
}  
]

所以任何可吸收目標 ((\gamma,m)) 必須滿足

[  
\boxed{  
m\equiv\sum_{y\in Y}\frac{\gamma_y}{y}\pmod1.  
}  
]

這就是終端吸收的唯一顯然局部條件。對角 Fourier 主項正是這個條件的反映。

---

## 2. 終端吸收的條件性定理

我們現在提出一個條件性版本：

**條件性終端吸收定理。**  
假設 (Y,Z) 滿足以下兩個分散條件。

### 條件 A：列核混合

對每個 (z\in Z)，對任意足夠大的 (U\subseteq Y)，以及任意非零 (\mu\in\mathbb F_z)，有

[  
\sum_{y\in U}  
\left|  
\frac{\mu(y^{-1}\bmod z)}{z}  
\right|^2  
\ge  
\kappa |U|  
]

某個固定 (\kappa>0)。

這保證隨機子集和

[  
\sum_{y\in U}\varepsilon_y y^{-1}\pmod z  
]

在 (\mathbb F_z) 中混合，從而 (K_z) 在小坐標投影上近似均勻。

### 條件 B：反向互逆分散

對每個 (y\in Y)，對任意非零 (r\in\mathbb F_y)，有

[  
\sum_{z\in Z}  
\left|  
\frac{r(z^{-1}\bmod y)}{y}  
\right|^2  
\ge  
\rho |Z|  
]

某個固定 (\rho>0)。

這保證非對角 character 在跨越所有終端列 (z) 後指數衰減。

若再有合適的參數條件，使

[  
|Z|  
]

足以壓過 (Y) 上 character 的總數，則終端列選擇可實現所有滿足相容條件的目標 ((\gamma,m))。

更直白地說：

[  
\boxed{  
\text{在條件 A、B 下，除了必要 CRT 相容條件外，終端吸收沒有其他障礙。}  
}  
]

---

## 3. Fourier 證明骨架

考慮隨機過程：對每個 (z\in Z)，獨立均勻地從 (K_z) 選一列。得到 ((R,M))。

取 character pair

[  
(\lambda,\tau),  
]

其中

[  
\lambda=(\lambda_y)_{y\in Y},\qquad \lambda_y\in\mathbb F_y,  
]

而 (\tau\in\mathbb Z) 是質量 (M\pmod1) 的 Fourier 頻率。

對單列 (z)，Fourier 因子為

# [  
F_z(\lambda,\tau)

\mathbb E_{x\in K_z}  
e\left(  
\sum_y x_y  
\left[  
\frac{\lambda_y(z^{-1}\bmod y)}y  
+  
\frac{\tau}{yz}  
\right]  
\right).  
]

整體 Fourier 係數為

# [  
\widehat\nu(\lambda,\tau)

\prod_{z\in Z}F_z(\lambda,\tau).  
]

---

## 4. 對角主項

若存在整數 (h)，使

[  
\lambda_y\equiv h\pmod y  
\quad\forall y\in Y,  
]

且

[  
\tau=-h,  
]

則對每個 (z) 和每個 (x\in K_z)，相位恰為整數。故

[  
F_z(\lambda,\tau)=1,  
]

所以

[  
\widehat\nu(\lambda,\tau)=1.  
]

這些 character 不能也不應衰減；它們構成對角族

[  
\mathcal D=  
{(\lambda,\tau):\lambda_y\equiv h\pmod y\ \forall y,\ \tau=-h}.  
]

Fourier inversion 中，這些對角項正好給出相容條件

[  
m\equiv\sum_y\gamma_y/y\pmod1.  
]

---

## 5. 非對角距離

對一般 ((\lambda,\tau))，定義離對角距離：

# [  
D(\lambda,\tau)

#{y\in Y:\lambda_y+\tau\not\equiv0\pmod y}.  
]

若 (D=0)，就是對角族。若 (D>0)，則每個異常坐標 (y) 在最危險的列約束 multiplier 下留下殘餘相位：

[  
\frac{(\lambda_y+\tau)(z^{-1}\bmod y)}y.  
]

由條件 B，

[  
\sum_{z\in Z}  
\left|  
\frac{(\lambda_y+\tau)(z^{-1}\bmod y)}y  
\right|^2  
\ge  
\rho |Z|.  
]

對所有異常 (y) 加總：

[  
\sum_{z\in Z}\sum_{y:\lambda_y+\tau\ne0}  
\left|  
\frac{(\lambda_y+\tau)(z^{-1}\bmod y)}y  
\right|^2  
\ge  
\rho |Z|D(\lambda,\tau).  
]

由列核混合和 Fourier 展開，可推出非對角衰減：

[  
\boxed{  
|\widehat\nu(\lambda,\tau)|  
\le  
\exp(-c|Z|D(\lambda,\tau))  
}  
]

其中 (c>0) 依賴於分散常數。

這是終端分析的核心估計。

---

## 6. 為什麼這足以求和？

距離為 (D=d) 的 character 數大約是：

選 (d) 個異常坐標：

[  
\binom md;  
]

每個異常坐標可有約 (y) 種值。因此總數至多類似

[  
\binom md \prod_{y\in E}y  
]

乘上一個對角參數 (h) 的數量。

若末層素數大小滿足

[  
\log y \le \eta |Z|  
]

且 (\eta>0) 足夠小，則距離 (d) 的 character 數至多

[  
\exp(O(d\log m)+\eta d|Z|),  
]

而 Fourier 衰減為

[  
\exp(-c d|Z|).  
]

只要

[  
\eta<c,  
]

求和即收斂：

[  
\sum_{(\lambda,\tau)\notin\mathcal D}  
|\widehat\nu(\lambda,\tau)|  
=o(1).  
]

於是 Fourier inversion 給出：

# [  
\mathbb P((R,M)=\text{target})

\text{對角主項}  
+  
o(\text{主項}).  
]

若 target 滿足相容條件，對角主項為正，故存在所需終端列選擇。

---

## 7. 這定理還有一個未處理的技術點

上面我把 (M) 當成 (\mathbb R/\mathbb Z) 變量，但實際 (M) 是來自有限分母

[  
\prod_{y\in Y}y\prod_{z\in Z}z  
]

的有理數，所以它其實落在有限循環群中。Fourier 頻率 (\tau) 不是任意整數，而可模這個分母取值。

這會帶來大量高頻 (\tau)。上面的估計對所有 (\tau) 都要成立，這可能過強。

解法有兩種：

第一，把終端吸收只用來控制模 (1) 相容，不單獨指定精確 (M)，而由整個構造的總和範圍保證等式。這回到早先「同餘 + 大小」框架，較簡潔。

第二，完整處理有限循環群中的 (M)，但要對高頻 (\tau) 加入額外估計。這會變得很技術。

我認為第一種更合適：終端封口負責局部模條件；精確大小由整個圖的總和控制在 (0,1) 區間內後自動得到。因此不必讓終端獨立命中指定實數 (m)。我們只需避免對角假障礙，知道它正是全局 CRT 相容即可。

## 8. 回到原問題的路線

現在整體證明藍圖可以寫成：

### 第一步：原始邊界

給定 squarefree (b)，邊界素數為

[  
B={p:p\mid b}.  
]

希望構造素數圖 (E)，使對 (p\in B)：

[  
\sum_{s\sim p}\frac1s\equiv\frac1{b/p}\pmod p,  
]

對所有輔助頂點 (r)：

[  
\sum_{s\sim r}\frac1s\equiv0\pmod r.  
]

### 第二步：分層傳遞

用二部滿射引理，把 (B) 上的殘差推到某個大末層 (Y)。每一步只清掉上一層。

### 第三步：終端吸收

用終端層 (Z)，並從每個列核 (K_z) 中選列，清掉 (Y) 的殘差，同時不留下 (Z) 殘差。

### 第四步：全局同餘

所有頂點局部條件成立，所以

[  
\sum_{e={p,q}}\frac{L}{pq}  
\equiv  
\frac Lb  
\pmod L,  
]

其中

[  
L=\prod_{\text{所有用到的素數}}p.  
]

### 第五步：大小控制

選輔助素數足夠大，使

[  
0<\sum_{e}\frac1{pq}<1.  
]

則

[  
0<\sum_e\frac{L}{pq}<L,  
\qquad  
0<\frac Lb<L.  
]

兩者模 (L) 同餘，故相等：

[  
\sum_e\frac1{pq}=\frac1b.  
]

最後反覆避開已用素數，得到 (a) 份互不重疊的 (1/b) 表示，從而得到 (a/b)。

---

## 9. 現在剩下的真正外部難點

整個框架已經把原題壓縮為兩類素數分散存在性：

### A. 二部傳遞所需的一維子集和覆蓋

對上一層 (A) 和下一層 (B)，需要 (B) 在每個模 (a\in A) 下有足夠多不同 residues，使

[  
{b^{-1}\bmod a:b\in B}  
]

的子集和覆蓋 (\mathbb F_a)。

這相對較容易。

### B. 終端吸收所需的雙向分散

需要 (Y,Z) 同時滿足：

[  
Y^{-1}\pmod z  
]

在每個 (z\in Z) 中混合；

以及

[  
Z^{-1}\pmod y  
]

在每個 (y\in Y) 中分散。

這是最核心的解析數論輸入。

換言之，現在真正要證的是：

[  
\boxed{  
\text{存在兩組素數 }Y,Z\text{，大小受控，互相在對方模數下倒數分散。}  
}  
]

若這個存在性可證，則整條路線有望完成。
