好。現在我們回到更正後的正路：**全局子集和 Fourier 模型**。

原問題可改寫如下。取一個有限素數集合 (P)，包含所有 (b) 的素因子。令

[  
L=\prod_{p\in P}p.  
]

對每條邊 ({p,q}\subset P)，定義整數權重

[  
w_{pq}=\frac{L}{pq}.  
]

若選出一組邊 (E)，則

# [  
\sum_{{p,q}\in E}\frac1{pq}

\frac1L\sum_{{p,q}\in E}w_{pq}.  
]

所以要表示

[  
\frac1b  
]

等價於要找邊集 (E)，使

# [  
\sum_{{p,q}\in E}w_{pq}

\frac Lb.  
]

這是 (\mathbb Z/L\mathbb Z) 上的子集和問題；若同時控制總和大小在 ((0,L)) 內，同餘即推出等式。

---

## 1. 隨機模型

給每條邊 ({p,q}) 一個獨立 Bernoulli 變量

[  
X_{pq}\in{0,1}.  
]

先取同一概率 (\theta)：

[  
\mathbb P(X_{pq}=1)=\theta.  
]

令

[  
S=\sum_{p<q}X_{pq}w_{pq}.  
]

等價地，在實數中

[  
\frac SL=\sum_{p<q}\frac{X_{pq}}{pq}.  
]

我們希望

[  
S=\frac Lb.  
]

期望為

# [  
\mathbb E\frac SL

\theta\sum_{p<q}\frac1{pq}.  
]

所以若選 (P) 使

[  
\sum_{p<q}\frac1{pq}>\frac1b,  
]

便可取

# [  
\theta

\frac{1/b}{\sum_{p<q}1/(pq)}  
]

使期望正好是 (1/b)。

這一步利用

[  
\sum_{p<q}\frac1{pq}=\infty.  
]

---

## 2. Fourier 表達

在群

[  
G=\mathbb Z/L\mathbb Z  
]

上，目標是

[  
t=\frac Lb.  
]

隨機變量 (S) 的分佈在 (G) 上有 Fourier 反演：

# [  
\mathbb P(S=t)

\frac1L  
\sum_{\chi\in\widehat G}  
\widehat\mu(\chi)\chi(-t).  
]

由 CRT，

[  
G\simeq\prod_{p\in P}\mathbb F_p.  
]

一個 character 可寫作

[  
\lambda=(\lambda_p)_{p\in P},\qquad \lambda_p\in\mathbb F_p.  
]

邊 ({p,q}) 的相位是

# [  
\phi_{pq}(\lambda)

\frac{\lambda_p(q^{-1}\bmod p)}p  
+  
\frac{\lambda_q(p^{-1}\bmod q)}q.  
]

所以

# [  
\widehat\mu(\lambda)

\prod_{p<q}  
\left(  
1-\theta+\theta e(\phi_{pq}(\lambda))  
\right).  
]

我們要證：

[  
\mathbb P(S=t)>0.  
]

主項 (\lambda=0) 是

[  
\frac1L.  
]

如果能證

[  
\sum_{\lambda\ne0}|\widehat\mu(\lambda)|<1,  
]

就得到正概率。但這太強，因為有主弧 character 不會小。正確做法是分主弧與次弧。

---

## 3. 對角主弧

先看對角 character。若存在整數 (h)，使

[  
\lambda_p\equiv h\pmod p  
]

對所有 (p\in P)，則對邊 (pq)，

# [  
\phi_{pq}(\lambda)

h\left(\frac{q^{-1}\bmod p}{p}  
+  
\frac{p^{-1}\bmod q}{q}\right)  
\equiv  
\frac{h}{pq}\pmod1.  
]

因此

# [  
\widehat\mu(\lambda)

\prod_{p<q}  
\left(  
1-\theta+\theta e\left(\frac{h}{pq}\right)  
\right).  
]

這正是實數隨機和

[  
X=\sum_{p<q}\frac{X_{pq}}{pq}  
]

的 characteristic function 在頻率 (h)。

所以對角 character 不是錯誤項，而是控制真實大小

[  
X\approx \frac1b  
]

的主弧。

換言之，原問題的「大小」完全藏在對角主弧中。這是我們剛才反覆摸到的事。

---

## 4. 次弧：非對角 character

若 (\lambda) 不是接近同一個整數 (h) 的對角型，則希望很多邊 ({p,q}) 使

[  
\phi_{pq}(\lambda)  
]

遠離整數，從而

[  
|1-\theta+\theta e(\phi_{pq})|  
\le  
\exp(-c\theta|\phi_{pq}|^2)  
]

給出衰減。

這裏 Irving 的定理進場。它告訴我們，在大素數塊中，對固定模 (q)，倒數 (p^{-1}\bmod q) 對 prime (p) 不集中。這正是要證：

[  
\sum_{p}  
\left|  
\frac{a(p^{-1}\bmod q)}q  
\right|^2  
\gg #p  
]

的工具。

因此對非對角 (\lambda)，只要有許多坐標 (q) 使

[  
\lambda_q-h\not\equiv0\pmod q,  
]

跨塊邊會提供大量衰減。

這給出一個自然的策略：把 (P) 分成若干 dyadic 素數塊

[  
P_1,P_2,\dots,P_J,  
]

只使用不同塊之間的邊，尤其使用許多「好塊對」 (P_i,P_j)。Irving 定理保證每個好塊對在互逆模分佈上有分散。這比在全部 (P) 上一次性處理更可控。

---

## 5. 主弧應該如何給出正主項？

對角主弧對應隨機實數和

[  
X=\sum_{e}\frac{X_e}{n_e},\qquad n_e=pq.  
]

其期望已調成

[  
\mathbb E X=\frac1b.  
]

若方差足夠小，則 (X) 集中在 (1/b) 附近；但我們需要的是**精確命中**離散點

[  
\frac1b.  
]

在整數化後，(S) 的步長是 (1/L)，所以主弧需要給出一個局部中心極限定理：

[  
\mathbb P(S=L/b)  
\approx  
\frac{1}{\sigma L}  
]

某種量級，而非僅僅集中。

這是全局隨機模型的又一個技術要求：不但要有非對角衰減，還要主弧分析足夠精確。

不過這比之前的「質量覆蓋」更標準。因為 (S) 是很多獨立 Bernoulli 小權重的和。若權重足夠細、方差不太小，局部 CLT 應該可用。

---

## 6. 需要選什麼樣的邊集？

若全部素數都很大，權重

[  
1/(pq)  
]

都極小，方差可能太小或太大可調。  
我們可以選許多 dyadic 塊，讓邊權尺度逐漸變小。這有助於局部覆蓋，類似用不同面額的硬幣表示數字。

更具體地，選素數塊 (P_j) 使

[  
\sum_{p\in P_j}\frac1p  
]

約為某個小常數 (u_j)。則完整跨塊 (P_i-P_j) 的總質量約為

[  
u_i u_j.  
]

通過很多塊，可以調整總可用質量與方差。

最簡單是選一個大素數集合 (P) 使

[  
W:=\sum_{p<q}\frac1{pq}  
]

遠大於 (1/b)，然後取小 (\theta=(1/b)/W)。  
但若 (\theta) 太小，隨機邊太稀疏，Fourier 衰減可能不夠。較好是讓

[  
W\asymp \frac1b  
]

且 (\theta) 為固定常數，例如 (1/2)。這可通過選 (P) 使總質量

[  
W\approx \frac{2}{b}  
]

來實現。

因為 (\sum_p1/p) 連續緩慢發散，應可選 (P) 使 (W) 落在合適範圍。

---

## 7. 但 (b) 的素因子必須包含在 (P)

邊界分母 (b) 的素因子必須在 (L) 中，所以

[  
B={p:p\mid b}\subset P.  
]

這些小素數可能破壞 dyadic 分佈。但有限多個固定素數可以單獨處理：把它們視為特殊邊界坐標。主要隨機性由大量大輔助素數提供。Fourier character 若只支撐在小邊界上，也要被連到大塊的邊殺掉；這可由大塊在模小素數下的 residue 覆蓋輕鬆處理，不需要 Irving。

所以 (b) 不構成大障礙。

---

## 8. 全局證明要走的形式

我們可以提出一個「全局主弧—次弧定理」：

取一個有限素數集合 (P\supset B)，以及邊集合 (\mathcal E\subset\binom P2)，使：

1. 總質量
    
    [  
    W=\sum_{e\in\mathcal E}\frac1{n_e}  
    ]
    
    約為 (2/b)；
    
2. 邊權足夠細，Bernoulli 和有局部 CLT；
    
3. 對所有非對角 character (\lambda)，有 Fourier 衰減
    
    [  
    |\widehat\mu(\lambda)|\le \exp(-cD(\lambda))  
    ]
    
    且總和可控；
    
4. 對角 character 主弧給出正主項於 (L/b)。
    

則存在子集 (E\subset\mathcal E)，使

[  
\sum_{e\in E}\frac1{n_e}=\frac1b.  
]

Irving 定理負責第 3 點。  
概率局部 CLT / 子集和光滑性負責第 2、4 點。

---

## 9. 現在最大的剩餘難點是主弧局部 CLT

我們本以為最大難點是模分散；Irving 幫我們處理了相當多。現在看來，真正還需要開發的是：

[  
\boxed{  
\text{Bernoulli 和 }\sum X_{pq}/(pq)\text{ 在精細格點上的局部中心極限定理。}  
}  
]

這不是不可行，但要小心。因為權重不是等距、不是同尺度，而是有大量相關分母。  
不過這是純概率／加法組合問題，比原題清楚很多。

如果局部 CLT 太難，也可改用 deterministic greedy + random smoothing：先用貪心把差距縮到很小，再用一個小尺度隨機塊精確補齊。這常比全局 local CLT 更容易。

---

## 10. 下一步應選哪條？

我認為下一步應研究「二階方案」：

1. **粗調質量。** 用貪心或大塊選擇，把總和調到距離 (1/b) 很近；
    
2. **精修模與質量。** 用一個 Irving-good 隨機塊，在很小尺度上做 Fourier，精確命中剩餘差額。
    

這可能比全局 local CLT 更可控。  
因為粗調不需要模精確；精修塊的目標較小，權重尺度也可選得匹配。

但精修塊又必須能處理指定殘差與指定小質量；這回到終端吸收，但現在差額小且尺度匹配，不再要求小塊吸收任意大殘差。

所以最終可能的證明形態是：

[  
\boxed{  
\text{粗調質量}+\text{隨機 Fourier 精修}  
}  
]

這是目前我覺得最有希望的完整證明架構。

下一步若繼續，我會嘗試設計這個「精修塊」：給定一個很小的 rational remainder，選一個 dyadic good block，使其總可用質量與 remainder 同尺度，然後用 Fourier 證明它能精確命中該 remainder。
