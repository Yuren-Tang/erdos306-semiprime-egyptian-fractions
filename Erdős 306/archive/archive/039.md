我們繼續攻「二部精修引理」。這裏我先採取一個保守而更可能走通的版本：**不要要求精修塊命中任意小 remainder，而是在粗調時把 remainder 安排成精修塊容易命中的形式。**

這樣可以避免立刻證一個很重的全局 local CLT。

---

## 1. 問題重設：讓粗調配合精修

粗調引理已證明：對任意小 (\varepsilon>0)，可以選一批 (pq)-項，使

[  
0<r=\frac1b-s<\varepsilon.  
]

但它沒有控制 (r) 的形狀。精修若要表示 (r)，最好 (r) 本身落在精修塊的「高概率區域」。

所以與其先任意粗調出 (r)，不如反過來：

1. 先選一個精修塊 (Y\times Z)；
    
2. 選一個該塊容易表示的值 (r)；
    
3. 再用粗調逼近
    

[  
\frac1b-r  
]

而不是逼近 (\frac1b)。

也就是說，我們只需：

[  
\frac1b = s+r,  
]

其中 (s) 由粗調給出，(r) 由精修塊給出。

這避免要求精修塊表示任意小 (r)。

---

## 2. 精修塊容易表示什麼？

取

[  
Y,Z\subset[Q,2Q],\qquad |Y|=|Z|=N.  
]

完整二部圖總質量

[  
W=\sum_{y\in Y,z\in Z}\frac1{yz}  
\asymp \frac{N^2}{Q^2}.  
]

如果用 Bernoulli (\theta=1/2)，則隨機和

[  
X=\sum_{y,z}\frac{X_{yz}}{yz}  
]

期望約為

[  
W/2.  
]

所以精修塊最容易表示的值在

[  
W/2  
]

附近，波動尺度

[  
\sigma\asymp \frac{N}{Q^2}.  
]

因此我們可以先選一個精修目標

[  
r\approx W/2,  
]

再讓粗調表示

[  
\frac1b-r.  
]

只要

[  
0<r<\frac1b,  
]

粗調引理就可逼近 (\frac1b-r)。而如果需要精確等式，粗調不能只是逼近；它要留下正好 (r)。但粗調引理只能逼近，不能保證精確留下指定 (r)。

所以還不夠。

---

## 3. 更好的方案：粗調只控制到精修窗口

令精修塊可表示的不只是單一 (r)，而是一個小窗口

[  
I=[W/2-C\sigma,\ W/2+C\sigma].  
]

如果精修塊能表示窗口中「所有相容格點」，那麼粗調只需把 remainder 落入這個窗口：

[  
r=\frac1b-s\in I.  
]

這就可由粗調引理做到，因為粗調可從下方把剩餘量逼近到任意小；我們可以在它首次穿入窗口時停止。

更具體地：

若粗調使用的每一項都小於窗口長度

[  
|I|\asymp \sigma,  
]

那麼貪心過程的剩餘量不會跳過整個窗口；它會落入 (I)。

所以我們需要在粗調後期只使用小於 (\sigma) 的 (pq)-項。這可做到，因為可禁用所有小分母，使用足夠大的半素數。

這樣粗調不需要精確，精修負責窗口內精確命中。

這是重要改進。

---

## 4. 精修引理的合理形式

現在精修引理可改成：

**二部精修窗口引理。**  
設 (Y,Z\subset[Q,2Q])，(|Y|=|Z|=N\gg\log Q)，且互逆分散成立。令

[  
L=L_0\prod_{y\in Y}y\prod_{z\in Z}z.  
]

考慮所有子集和

[  
\sum_{(y,z)\in E}\frac1{yz}.  
]

則在中心窗口

[  
I=\left[W/2-C\sigma,\ W/2+C\sigma\right],  
]

其中

[  
W=\sum_{y,z}\frac1{yz},\qquad  
\sigma^2=\frac14\sum_{y,z}\frac1{y^2z^2},  
]

精修塊能命中所有與 CRT 條件相容的格點，並同時修正任意給定舊模數殘差。

這比「命中任意 (r)」合理得多。

---

## 5. 這個窗口有多大？

有

[  
W\asymp \frac{N^2}{Q^2},  
]

[  
\sigma\asymp \frac{N}{Q^2}.  
]

窗口長度約

[  
\sigma=\frac{N}{Q^2}.  
]

單個邊權約

[  
Q^{-2}.  
]

所以窗口長度約等於

[  
N  
]

個單邊權。若 (N\to\infty)，窗口裏有很多可能值。這正是 local CLT 的自然尺度。

---

## 6. 粗調如何落入窗口？

給定窗口 (I\subset(0,1/b))，我們要選粗調和 (s) 使

[  
\frac1b-s\in I.  
]

也就是

[  
s\in \frac1b-I.  
]

這是一個長度約 (\sigma) 的窗口。

粗調引理可加強為「可命中任意小區間」：

給定區間 ((A,A+\Delta)\subset(0,T))，若允許使用足夠小的 (pq)-項，則可選互異項使和落入該區間。

證法與貪心類似：逼近 (A) 後，因為下一項可小於 (\Delta)，可以落入窗口。或者更直接，用從下方逼近 (A) 到誤差 (<\Delta)。

所以粗調可以把 remainder 安排到精修中心窗口。

這解決了精修目標任意性的問題。

---

## 7. 舊模數殘差怎麼辦？

粗調用了某些素數，形成舊模數乘積

[  
L_0.  
]

精修塊 (Y,Z) 要同時修正 modulo (L_0) 的殘差。這會在 Fourier 中加入舊坐標。

如果 (L_0) 很大且包含很多粗調素數，這可能麻煩。解法：粗調時不要把所有粗調素數都納入需要修正的邊界？但最終所有頂點都必須平衡，不能忽略。

這裏出現新問題：粗調任意選 (pq)-項，會產生大量舊頂點，每個都需要精修塊連邊去平衡。精修塊只用 (Y\times Z) 邊，並不連到舊頂點，無法修正它們。

所以粗調不能是任意獨立項。它必須自身也形成一個局部平衡結構，或者至少留下一個可由精修連接的有限邊界。

這是重要硬點。

---

## 8. 修正：粗調也要用可封閉塊

不能先隨便粗調，再指望精修只在新塊內解決。因為每個粗調邊的兩端素數會留下殘差。

因此粗調應使用「已封閉」或「帶小邊界」的塊。

有兩種方案：

### 方案 A：粗調用完整合法表示塊

每個粗調塊本身已經滿足內部模條件，代表某個有理質量。  
但小正零模塊不存在，除非該塊代表一個整數。這不適合小質量。

### 方案 B：粗調產生殘差，但保持邊界小

粗調只使用邊界集合 (B) 與大量新素數之間的邊，讓所有新素數殘差可傳遞到下一層，最後由精修塊一起吸收。

也就是我們又需要「分層傳遞」，但現在它傳遞的是粗調殘差，且同時安排質量窗口。

這說明粗調和模傳遞仍不能完全分開。

---

## 9. 更正後的完整方案

現在比較可行的是：

1. 從原邊界 (B={p:p\mid b}) 出發；
    
2. 用若干分層二部塊逐步累積質量，同時把所有未平衡殘差推到最後一個邊界層 (A)；
    
3. 控制累積質量，使剩餘量落入最終精修窗口；
    
4. 用二部精修塊 (A) 與 (Y,Z) 共同完成：修正 (A) 上殘差並精確補足質量。
    

也就是：

[  
\boxed{  
\text{粗調不能是無約束貪心；它必須是「帶邊界的質量傳遞」。}  
}  
]

這比單純粗調難，但仍比原題更結構化。

---

## 10. 可簡化：把原邊界直接接到精修塊

或許不需要大量粗調層。可以讓原邊界 (B) 直接連到許多大素數塊，這些邊既提供質量，又把殘差傳進精修塊。由於 (B) 是固定有限集合，控制它比較容易。

例如選大素數集合 (Y)，使用 (B\times Y) 邊。每條邊質量約

[  
\frac1{p y}  
]

其中 (p\mid b) 固定。總質量可由

[  
\sum_{y\in Y}\frac1y  
]

調節；此和可發散。這比 (Y\times Z) 邊質量大得多，也更適合粗調。

然後 (Y) 上留下殘差，再用 (Y\times Z) 精修／封口。

這看起來更好：

- (B\times Y) 提供大部分質量；
    
- (Y\times Z) 提供隨機精修和封口；
    
- 邊界只是一層 (Y)，沒有大量散亂舊素數。
    

這回到了分層傳遞，但非常簡潔：只需兩層。

---

## 11. 兩層模型

令 (B) 是固定邊界。選大素數層 (Y)，再選終端層 (Z)。

使用兩類邊：

[  
B\times Y  
]

和

[  
Y\times Z.  
]

不使用 (B\times Z) 或內部邊。

目標：

1. (B) 上達到指定殘差；
    
2. (Y) 上由 (B\times Y) 造成的殘差，被 (Y\times Z) 修正；
    
3. (Z) 自身平衡，或在全局同餘中被處理；
    
4. 總質量等於 (1/b).
    

這非常像之前的終端吸收，但現在 (B\times Y) 提供大質量水庫，(Y\times Z) 提供精修與封口。

這可能是目前最好的簡化模型。

---

## 12. 下一步

應該研究這個兩層模型的 Fourier 分佈：

隨機選 (B\times Y) 邊和 (Y\times Z) 邊，考慮總和

[  
S=  
\sum_{p\in B,y\in Y}X_{p,y}\frac{L}{py}  
+  
\sum_{y\in Y,z\in Z}X_{y,z}\frac{L}{yz}.  
]

這個模型有足夠質量，因為

# [  
\sum_{p\in B,y\in Y}\frac1{py}

\left(\sum_{p\in B}\frac1p\right)  
\left(\sum_{y\in Y}\frac1y\right),  
]

而 (\sum_y1/y) 可任意大。

同時 (Y\times Z) 提供大量隨機性與互逆分散。

這比「任意粗調 + 精修」更 coherent。

所以接下來應該從兩層模型開始，而不是繼續抽象精修塊。這可能是最有希望的完整證明形態。
