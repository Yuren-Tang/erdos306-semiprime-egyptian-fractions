好。現在研究兩層模型：

[  
B \longleftrightarrow Y \longleftrightarrow Z,  
]

其中

[  
B={p:p\mid b}  
]

是固定邊界素數集合，(Y,Z) 是大素數層。只允許兩類邊：

[  
B\times Y,\qquad Y\times Z.  
]

不使用 (B\times Z)，也不使用層內邊。

目標是選邊，使

[  
\sum\frac1{pq}=\frac1b.  
]

---

## 1. 局部模條件

令邊變量為

[  
X_{p,y}\in{0,1}\qquad(p\in B,\ y\in Y),  
]

[  
X_{y,z}\in{0,1}\qquad(y\in Y,\ z\in Z).  
]

對 (p\in B)，需要

[  
\sum_{y\in Y}X_{p,y}\frac1y  
\equiv  
\frac1{b/p}  
\pmod p.  
]

對 (y\in Y)，需要

[  
\sum_{p\in B}X_{p,y}\frac1p  
+  
\sum_{z\in Z}X_{y,z}\frac1z  
\equiv0\pmod y.  
]

對 (z\in Z)，需要

[  
\sum_{y\in Y}X_{y,z}\frac1y  
\equiv0\pmod z.  
]

這正是「邊界—中層—終端」傳遞結構。

---

## 2. 實際質量

總質量是

[  
M=  
\sum_{p\in B,y\in Y}\frac{X_{p,y}}{py}  
+  
\sum_{y\in Y,z\in Z}\frac{X_{y,z}}{yz}.  
]

第一部分

[  
M_{BY}=  
\sum_{p\in B,y\in Y}\frac{X_{p,y}}{py}  
]

可以提供主要質量。因為

# [  
\sum_{p\in B,y\in Y}\frac1{py}

\left(\sum_{p\in B}\frac1p\right)  
\left(\sum_{y\in Y}\frac1y\right).  
]

只要讓

[  
\sum_{y\in Y}\frac1y  
]

足夠大，就能提供 (1/b) 量級的質量。這可以通過選很多 dyadic blocks 中的 (Y) 來做到。若 (Y) 只在一個 ([Q,2Q]) 裏，則 (\sum_{y\in Y}1/y) 很小；所以 (Y) 應該是多塊集合，而非單一 dyadic 層。

第二部分

[  
M_{YZ}  
]

主要用來封掉 (Y,Z) 的局部殘差，也可作精修質量。

---

## 3. 這個模型的好處

它避免了「粗調產生大量散亂舊素數」的問題。所有粗調邊都只接在固定邊界 (B) 和中層 (Y) 上。粗調後未平衡的頂點只有 (Y)，再由 (Y\times Z) 統一封掉。

所以邊界集合始終只有三層：

[  
B,\quad Y,\quad Z.  
]

這比任意貪心粗調乾淨多了。

---

## 4. 隨機模型

令

[  
X_{p,y}\sim \mathrm{Bernoulli}(\alpha_p)  
]

其中 (\alpha_p) 可依 (p\in B) 而定；令

[  
X_{y,z}\sim \mathrm{Bernoulli}(\theta)  
]

或讓 (\theta) 依 block 而定。

我們希望調參數使

[  
\mathbb E M=\frac1b.  
]

更細地，還要使 (B) 上的期望殘差接近目標：

[  
\sum_{y\in Y}\alpha_p\frac1y  
]

在模 (p) 上不是直接有意義，但隨機分佈要能覆蓋指定殘差。因為 (B) 是固定有限集合，這很容易：只要 (Y) 在每個模 (p) 下有足夠多 residue，(B\times Y) 邊的子集和即可覆蓋 (\mathbb F_p)。

真正複雜的是 (Y) 與 (Z) 層的聯動。

---

## 5. 先固定 (B\times Y)，再封 (Y) 是否可行？

假設先選了 (B\times Y) 邊，使 (B) 上條件正確，且主要質量接近需要值。它在每個 (y\in Y) 上造成殘差：

[  
\rho_y=  
\sum_{p\in B}X_{p,y}\frac1p\pmod y.  
]

接着要用 (Y\times Z) 選邊，使

[  
\sum_{z\in Z}X_{y,z}\frac1z  
\equiv -\rho_y\pmod y  
]

對所有 (y\in Y)，並且對每個 (z\in Z)：

[  
\sum_{y\in Y}X_{y,z}\frac1y  
\equiv0\pmod z.  
]

這正是之前的終端吸收問題，但現在 (Y) 的殘差 (\rho_y) 不是任意外來的；它由 (B\times Y) 的選擇產生。這很重要，因為我們可以選 (B\times Y) 時讓 (\rho) 落在 (Y\times Z) 可吸收集合中。

所以不應該要求 (Y\times Z) 能吸收任意 (\rho)。只需要：

[  
\boxed{  
\text{可選 }B\times Y\text{ 邊，使產生的 }\rho  
\text{ 被 }Y\times Z\text{ 吸收，且總質量正確。}  
}  
]

這降低了終端吸收負擔。

---

## 6. 聯合 Fourier 更自然

把 (B\times Y) 和 (Y\times Z) 一起隨機選，直接看總子集和。Fourier character 由三層坐標組成：

[  
\lambda_p\in\mathbb F_p\quad(p\in B),  
]

[  
\lambda_y\in\mathbb F_y\quad(y\in Y),  
]

[  
\lambda_z\in\mathbb F_z\quad(z\in Z).  
]

邊 (p y) 的相位：

# [  
\phi_{p,y}

\frac{\lambda_p y^{-1}}p  
+  
\frac{\lambda_y p^{-1}}y.  
]

邊 (y z) 的相位：

# [  
\phi_{y,z}

\frac{\lambda_y z^{-1}}y  
+  
\frac{\lambda_z y^{-1}}z.  
]

對角 character 是

[  
\lambda_r\equiv h\pmod r  
]

對所有 (r\in B\cup Y\cup Z)。此時每條邊相位為

[  
\frac h{pq}.  
]

這是主弧質量。

非對角 character 若在 (Y) 或 (Z) 上有大量偏離，可由 (Y\times Z) 邊殺掉。若只在固定的 (B) 上偏離，可由 (B\times Y) 邊殺掉，因為 (Y) 可選得在模 (p) 下分散。

這非常漂亮：兩層模型的非對角衰減機制正好覆蓋所有可能偏離。

---

## 7. 質量主弧是否夠？

主弧對應實數隨機和

[  
M=  
\sum_{p,y}\frac{X_{p,y}}{py}  
+  
\sum_{y,z}\frac{X_{y,z}}{yz}.  
]

其中 (B\times Y) 邊權約為

[  
1/(p y),  
]

比 (Y\times Z) 的

[  
1/(yz)  
]

大很多，因為 (p\in B) 固定而 (y,z) 大。

所以主弧 local CLT 主要由 (B\times Y) 部分控制。這是好事：它提供大質量與較大的方差。(Y\times Z) 提供細修。

可把 (Y) 分為多個 dyadic blocks，讓 (1/(py)) 面額從較大到較小分佈，形成很好的局部覆蓋。這類隨機和比單一精修塊更有機會滿足 local CLT。

---

## 8. 參數設想

令 (Y) 是許多 dyadic 好塊的並：

[  
Y=\bigcup_{j=1}^J Y_j,\qquad Y_j\subset[Q_j,2Q_j].  
]

使

[  
\sum_{y\in Y}\frac1y  
\asymp 1.  
]

這樣

[  
\sum_{p\in B,y\in Y}\frac1{py}  
]

有 (1/b) 量級。

對每個 (Y_j)，選對應的 (Z_j\subset[Q_j,2Q_j])，用 (Y_j\times Z_j) 封掉該塊殘差。或者用一個大 (Z) 與全部 (Y) 相連，但分塊更利於 Irving。

邊集合為：

[  
B\times Y_j,\qquad Y_j\times Z_j  
]

對所有 (j)。

這樣每個 dyadic 層都有自己的終端吸收與互逆分散，互不干擾。

---

## 9. 兩層分塊模型的完整輪廓

候選邊：

[  
\mathcal E=  
\bigcup_j (B\times Y_j)  
\cup  
\bigcup_j (Y_j\times Z_j).  
]

隨機選邊。調整各層選邊概率 (\alpha_{p,j})、(\theta_j)，使期望總質量為

[  
1/b.  
]

Fourier 分析：

- 對角主弧：一維隨機和 local CLT；
    
- 非對角在 (B)：由 (B\times Y_j) 的 residue 分散殺；
    
- 非對角在 (Y_j,Z_j)：由 Irving 的互逆分散殺；
    
- block 之間獨立，總衰減相乘。
    

若 local CLT 成功，即得正概率命中。

---

## 10. 這比前一路線更有走通感

這個模型解決了三個問題：

第一，質量不夠：由 (B\times Y) 提供主要質量，(\sum 1/y) 發散。  
第二，粗調舊殘差散亂：舊邊界固定為 (B)，中間殘差集中在 (Y)。  
第三，終端任意吸收過強：(Y\times Z) 與 (B\times Y) 聯合選擇，不要求獨立吸收任意殘差。

所以目前最有希望的證明架構應該是：

[  
\boxed{  
\text{固定邊界 }B  
\quad\to\quad  
\text{多 dyadic 中層 }Y_j  
\quad\to\quad  
\text{對應終端層 }Z_j.  
}  
]

這是比「粗調 + 單精修塊」更自然的結構。

---

## 11. 下一步

下一步應該證一個「分塊三層隨機模型定理」：

給定 squarefree (b)，存在有限多個 dyadic blocks (Y_j,Z_j) 和選邊概率，使隨機和

[  
S=\sum_{e\in\mathcal E}X_e\frac{L}{n_e}  
]

滿足

[  
\mathbb P(S=L/b)>0.  
]

其證明分為：

1. 主弧 local CLT for real mass;
    
2. 非主弧 Fourier decay using Irving;
    
3. boundary (B) residue control.
    

這就是現在最清楚、最可能走通的完整證明目標。
