繼續。現在檢查三層模型

[  
B\longleftrightarrow Y\longleftrightarrow Z  
]

有沒有隱藏硬傷。這一步很重要，因爲 (B={p:p\mid b}) 是固定有限集合，可能很小。若 (b) 是素數，甚至

[  
B={p}  
]

只有一個點。這會使 (B\times Y) 層的自由度不如我們剛才想得那麼大。

---

## 1. (B\times Y) 的自由度問題

對每個 (y\in Y)，(B\times Y) 中可選的邊是

[  
py\qquad(p\in B).  
]

它對實際質量的貢獻是

# [  
\sum_{p\in S_y}\frac1{py}

\frac1y\sum_{p\in S_y}\frac1p,  
]

其中

[  
S_y\subseteq B.  
]

所以每個 (y) 只提供

[  
2^{|B|}  
]

種可能係數。若 (|B|) 很小，這個選擇相當有限。

最極端地，若

[  
B={p},  
]

則對每個 (y)，只有兩種選擇：

[  
0,\qquad \frac1{py}.  
]

這仍然能提供質量，因爲

# [  
\sum_{y\in Y}\frac1{py}

\frac1p\sum_{y\in Y}\frac1y  
]

可以任意大；但它在模 (y) 上造成的殘差也只有兩種：

[  
0,\qquad \frac1p\pmod y.  
]

然後 (Y\times Z) 要吸收這些殘差。這未必致命，但說明 (B\times Y) 不是一個很平滑的隨機層。

---

## 2. 是否需要「邊界放大器」？

爲增加自由度，可以在 (B) 和 (Y) 之間加一層 (C)：

[  
B\longleftrightarrow C\longleftrightarrow Y\longleftrightarrow Z.  
]

其中 (C) 是一組輔助素數，大小可自選。這樣主要質量可由

[  
C\times Y  
]

提供，而 (B) 只負責設定邊界條件。

優點是：(C) 可以取很多素數，使每個 (y) 有很多可選鄰居，從而每個 (y) 上的殘差與質量變得平滑。

但缺點是：又多了一層未平衡頂點 (C)，需要處理。

不過這其實不是壞事。分層傳遞本來就是：

[  
B\to C\to Y\to Z.  
]

其中 (C\times Y) 可作主要質量水庫，(Y\times Z) 作終端封口。這可能比直接 (B\times Y) 穩健。

---

## 3. 改良模型：四層而非三層

考慮：

[  
B\quad-\quad C\quad-\quad Y\quad-\quad Z.  
]

候選邊：

[  
B\times C,\qquad C\times Y,\qquad Y\times Z.  
]

用途：

- (B\times C)：滿足原邊界 (B) 的殘差，把未平衡量傳到 (C)；
    
- (C\times Y)：提供主要質量，並把未平衡量傳到 (Y)；
    
- (Y\times Z)：封掉 (Y) 和 (Z) 的殘差，並作精修。
    

這樣 (C) 可以選得很大，因此主質量層

[  
C\times Y  
]

有充足自由度：

# [  
\sum_{c\in C,y\in Y}\frac1{cy}

\left(\sum_{c\in C}\frac1c\right)  
\left(\sum_{y\in Y}\frac1y\right).  
]

由於兩個素數倒數和都可控制，質量水庫更靈活。

---

## 4. 這是否又變得太複雜？

表面上多一層，但 Fourier 結構仍然同一類。對任意邊 (uv)，相位總是

[  
\frac{\lambda_u v^{-1}}u+\frac{\lambda_v u^{-1}}v.  
]

只要每一對相鄰層都是 Irving-good 或普通 residue-good，就可以殺非對角 character。

對角 character 仍是

[  
\lambda_r\equiv h\pmod r  
]

對所有頂點 (r)，並對應實際質量

[  
\sum X_{uv}/(uv).  
]

所以層數增加不改變原理，只增加 bookkeeping。

這其實是一個好信號：我們可以自由加層來獲得質量與自由度，而 Fourier 分析形式不變。

---

## 5. 可以抽象成「鏈狀素數圖」

更一般地，取若干層

[  
P_0,P_1,\dots,P_k,  
]

其中

[  
P_0=B,  
]

只允許相鄰層之間連邊：

[  
P_i\times P_{i+1}.  
]

最終層 (P_k) 用於封口；或者整條鏈作全局隨機模型處理。

邊集是

[  
\mathcal E=\bigcup_{i=0}^{k-1}P_i\times P_{i+1}.  
]

這樣：

- 質量由所有相鄰層乘積提供；
    
- 模殘差沿鏈傳遞；
    
- 非對角 Fourier 由相鄰層互逆分散殺掉；
    
- 對角主弧處理實際大小。
    

這是比三層、四層更自然的抽象。

---

## 6. 質量如何控制？

對相鄰層 (P_i,P_{i+1})，總可用質量是

[  
W_i=  
\left(\sum_{p\in P_i}\frac1p\right)  
\left(\sum_{q\in P_{i+1}}\frac1q\right).  
]

總可用質量

[  
W=\sum_i W_i.  
]

因爲我們可選 (P_i) 的大小與位置，便可使

[  
W\asymp \frac1b  
]

或略大於 (1/b)。

尤其只要有兩個可自由選的大層 (C,Y)，就能用

[  
C\times Y  
]

提供任意所需量級。

所以質量不再困難。

---

## 7. 主弧 local CLT 在鏈模型中更可行

隨機實數和是

[  
X=\sum_{i=0}^{k-1}\sum_{p\in P_i,q\in P_{i+1}}\frac{X_{pq}}{pq}.  
]

如果各層有多個 dyadic blocks，權重 (1/(pq)) 有多尺度分佈；這通常有利於局部覆蓋。若想簡化，可讓主質量集中在某一對大層 (C\times Y)，而其他層提供模修正。

主弧 local CLT 只需對這個主要二部塊證明。其他層可視爲平滑噪音或小擾動。

---

## 8. 非對角 Fourier 在鏈模型中的判準

對 character

[  
\lambda=(\lambda_r)_{r\in\cup P_i},  
]

若不是對角型，則存在某一層或某些頂點偏離共同整數 (h)。

如果偏離發生在內部層 (P_i)，它與左右兩層都有邊，至少一側應提供衰減。

如果偏離發生在 (P_0=B)，則由 (B\times P_1) 殺掉；因 (B) 固定小，只要 (P_1) 在模 (p) 下分散即可，這是初等的。

如果偏離發生在末層，則由前一層殺掉。

所以只要每對相鄰大層互逆分散，非對角衰減應可全覆蓋。

這比單一終端吸收更穩健。

---

## 9. 現在可提出「鏈模型定理」

**鏈模型定理（目標版）。**  
給定 squarefree (b) 和有限禁用素數集 (F)，存在有限多層素數集合

[  
P_0=B,\ P_1,\dots,P_k,  
]

其中 (P_i\cap F=\varnothing) 對 (i\ge1)，以及相鄰層邊集合

[  
\mathcal E=\bigcup_i P_i\times P_{i+1},  
]

使得存在 (E\subseteq\mathcal E) 滿足

[  
\sum_{{p,q}\in E}\frac1{pq}=\frac1b.  
]

證明策略：

1. 用 Irving 定理選相鄰大層互逆分散；
    
2. 選層大小使總質量 (W) 合適；
    
3. 在 (\mathcal E) 上取 Bernoulli 隨機邊；
    
4. Fourier 分析：
    
    - 對角主弧：local CLT 給正主項；
        
    - 非對角：相鄰層互逆分散給衰減；
        
5. 得到正概率命中 (L/b)。
    

---

## 10. 現在剩下最硬的是主弧 local CLT

鏈模型已經很好地處理了自由度、殘差傳遞和質量水庫。非對角側有 Irving 支持。真正需要獨立證明的是：

[  
\boxed{  
\text{鏈狀二部權重 }1/(pq)\text{ 的 Bernoulli 子集和有局部中心極限定理。}  
}  
]

這可先在單個二部塊 (C\times Y) 中證，然後把其他層作擾動。

如果這個 local CLT 成功，整條路非常可能走通。

---

## 11. 這裏是否有現成工具？

這類問題接近：

- lattice local central limit theorem for triangular arrays;
    
- Bernoulli sums with small unequal weights;
    
- Littlewood–Offord anti-concentration;
    
- subset sum smoothing.
    

條件通常包括：

1. 最大單項權重 (\max a_i) 遠小於標準差 (\sigma)；
    
2. 格點跨度 gcd 為 (1)；
    
3. 沒有大量權重集中在同一週期結構。
    

對二部 (1/(pq)) 權重，前兩點可控；第三點需要檢查。因權重整數化後是

[  
L/(pq),  
]

有強乘法結構，但 gcd 為 (1)，且素數層多樣，應該不會有嚴重週期障礙。非對角 Fourier 分析本身其實就是在證無週期障礙。

所以主弧 local CLT 與非對角衰減不是完全分離：完整 Fourier 證明會同時處理。

## 12. 目前的判斷

現在路線比最初清楚很多：

- 小例子與星形公式：只是啓發；
    
- 模平衡圖：正確語言；
    
- Irving 定理：提供非對角分散；
    
- 終端吸收：單獨做會因質量問題失敗；
    
- 全局／鏈狀隨機模型：目前最穩健；
    
- 剩餘核心：主弧 local CLT / Bernoulli 子集和精確命中。
    

所以我目前會把整個研究計畫凝練為：

[  
\boxed{  
\text{用 Irving 的 prime Kloosterman 平均估計處理次弧；用格點 local CLT 處理主弧；在鏈狀素數二部圖中證明 }L/b\text{ 可命中。}  
}  
]

這是目前最像真正證明的路線。
