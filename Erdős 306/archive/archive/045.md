好。現在正式建立 **dyadic prime block graph**。

令

[  
\mathcal P_k={p\text{ prime}:2^k\le p<2^{k+1}}.  
]

記

[  
u_k=\sum_{p\in \mathcal P_k}\frac1p.  
]

由素數定理與 Mertens 型估計，

[  
u_k\asymp \frac1k.  
]

我們選一段尺度

[  
k_0\le k\le K,  
]

並把每個 (\mathcal P_k) 視作一層素數。候選邊不是所有跨層邊，而是只取 Irving 定理可控範圍內的層對。

Irving 的條件是，若模數尺度為 (Q)，素數變量尺度為 (x)，則需要

[  
Q^{3/2}\ge x\ge Q^{2/3}.  
]

若

[  
Q\asymp 2^\ell,\qquad x\asymp 2^k,  
]

則這等價於

[  
\frac23\ell\le k\le\frac32\ell.  
]

所以定義尺度圖：

[  
k\sim \ell  
\quad\Longleftrightarrow\quad  
\frac23\ell\le k\le\frac32\ell.  
]

候選邊集為

# [  
\mathcal E

\bigcup_{k\sim\ell}  
\mathcal P_k\times\mathcal P_\ell,  
]

其中若 (k=\ell)，取不同素數 (p\ne q)。

---

## 1. 總質量發散

候選邊的總質量為

# [  
W(k_0,K)

\sum_{k_0\le k,\ell\le K\atop k\sim\ell}  
\sum_{p\in\mathcal P_k,\ q\in\mathcal P_\ell}  
\frac1{pq}.  
]

忽略對角 (p=q) 的極小影響，有

[  
W(k_0,K)  
\asymp  
\sum_{k_0\le k,\ell\le K\atop k\sim\ell}  
u_k u_\ell.  
]

又

[  
u_k\asymp \frac1k.  
]

對固定 (k)，與它相連的 (\ell) 約在

[  
\frac23 k\le \ell\le\frac32 k,  
]

數量 (\asymp k)，且每個 (\ell\asymp k)。因此

# [  
\sum_{\ell\sim k}u_k u_\ell  
\asymp  
\sum_{\ell\asymp k}\frac1{k\ell}  
\asymp  
k\cdot\frac1{k^2}

\frac1k.  
]

所以

[  
W(k_0,K)  
\asymp  
\sum_{k=k_0}^{K}\frac1k  
\asymp  
\log\frac K{k_0}.  
]

這是關鍵結論：

[  
\boxed{  
\text{Irving 可控的尺度圖上，候選 }pq\text{ 邊總質量仍然發散。}  
}  
]

這比只取同尺度邊強得多；同尺度只給

[  
\sum_k\frac1{k^2}<\infty,  
]

不夠。而 Irving 的 (2/3) 到 (3/2) 尺度窗口剛好給每個 (k) 約 (k) 個可連尺度，恢復

[  
\sum_k\frac1k  
]

發散。

這是非常好的徵兆。

---

## 2. 可調到 (1/b) 量級

因為

[  
W(k_0,K)\asymp\log(K/k_0),  
]

給定 (b)，我們可以選 (K) 使

[  
W(k_0,K)> \frac{2}{b},  
]

甚至調到

[  
W(k_0,K)\asymp\frac{1}{b}  
]

的固定倍數。

然後在全部候選邊上以概率 (\theta) 選取，令

[  
\theta W(k_0,K)=\frac1b.  
]

只要 (W(k_0,K)) 選為例如 (2/b)，便有

[  
\theta=\frac12.  
]

這很好，因為 (\theta) 遠離 (0,1)，Fourier 衰減最穩。

---

## 3. 權重最大值與方差

隨機實數和

[  
X=\sum_{{p,q}\in\mathcal E}\frac{X_{pq}}{pq}  
]

的方差為

# [  
\sigma^2

\theta(1-\theta)  
\sum_{{p,q}\in\mathcal E}\frac1{p^2q^2}.  
]

估計 dyadic 層：

# [  
\sum_{p\in\mathcal P_k}\frac1{p^2}  
\asymp  
\frac{#\mathcal P_k}{2^{2k}}  
\asymp  
\frac{2^k/k}{2^{2k}}

\frac{1}{k2^k}.  
]

所以

[  
\sum_{k\sim\ell}  
\sum_{p\in\mathcal P_k,q\in\mathcal P_\ell}  
\frac1{p^2q^2}  
\asymp  
\sum_{k\sim\ell}  
\frac{1}{k2^k}\frac{1}{\ell2^\ell}.  
]

這個和主要由最小尺度 (k_0) 附近主導。粗略地，

[  
\sigma^2\asymp \frac{1}{k_0^2 2^{2k_0}}  
]

乘上一個尺度窗口常數級或緩慢因子。最大單項權重約為

[  
a_{\max}\asymp 2^{-2k_0}.  
]

所以

# [  
\frac{a_{\max}}{\sigma}  
\asymp  
\frac{2^{-2k_0}}{1/(k_0 2^{k_0})}

\frac{k_0}{2^{k_0}}\to0.  
]

這是 local CLT 的好條件：最大單項遠小於標準差。

## 4. 格點 span

令

[  
P=\bigcup_{k_0\le k\le K}\mathcal P_k  
\cup B  
]

並

[  
L=\prod_{p\in P}p.  
]

每條邊 (pq) 的整數權重是

[  
w_{pq}=\frac{L}{pq}.  
]

只要每個頂點 (r\in P) 至少出現在某條候選邊中，則對該 (r)，存在邊 (rs)，使

[  
w_{rs}=L/(rs)  
]

不含因子 (r)。所以 (r) 不整除所有權重。對所有 (r\mid L) 如此，得

[  
\gcd_{{p,q}\in\mathcal E} w_{pq}=1.  
]

因此整數格 span 是 (1)。這消除了 local CLT 的基本格點障礙。

---

## 5. 邊界 (B) 如何加入？

上面的 dyadic block graph只涉及大素數；但原目標 (1/b) 需要 (B={p:p\mid b}) 在 (L) 中，且在這些坐標上要有指定殘差。

最簡單做法：把 (B) 也連到一批大素數層，例如最低可用層 (\mathcal P_{k_0})。加邊

[  
B\times \mathcal P_{k_0}  
]

或

[  
B\times \bigcup_{k_0\le k\le k_0+C}\mathcal P_k.  
]

這部分質量很小，可忽略或納入總質量；但它提供 (B) 坐標的 Fourier 衰減與殘差自由度。

因 (B) 固定有限，對每個 (p\in B)，只需大素數在模 (p) 下分散。這可由 Dirichlet/素數在剩餘類中的基本分佈處理，甚至可用 CRT 選一小批輔助素數補強。

所以 (B) 不是主要障礙。

## 6. 非對角 Fourier 衰減：尺度圖版本

對 character

[  
\lambda=(\lambda_r)_{r\in P},  
]

定義離對角距離

# [  
D(\lambda)

\min_h  
#{r\in P:\lambda_r\not\equiv h\pmod r}.  
]

若 (D(\lambda)>0)，我們希望有很多候選邊跨過異常頂點，產生相位分散。

在尺度圖中，每個 dyadic block (\mathcal P_k) 連到約 (k) 個相近 blocks，總鄰接質量大約

[  
\sum_{\ell\sim k}u_\ell\asymp 1.  
]

所以每個異常頂點所在 block 有很多相鄰素數可用。Irving 定理保證對大多數模數 (q) 在相鄰可控尺度中，對所有非零乘子 (a),

[  
\sum_{p\in \text{neighbor blocks}}  
\left|  
\frac{a p^{-1}}q  
\right|^2  
\gg #{\text{neighbor primes}}  
]

或至少在質量加權意義下給出固定比例衰減。

因此可望證：

[  
|\widehat\mu(\lambda)|  
\le  
\exp(-c,M(\lambda)),  
]

其中 (M(\lambda)) 是異常頂點與相鄰邊的某種加權度量。若每個異常坐標帶來至少 (\gg 1) 的總衰減，則足以求和。

這裏要比之前的 (e^{-cND}) 更細，因為 blocks 大小不同，且全圖多尺度。正確距離應是加權的：

# [  
D_w(\lambda)

\sum_{\text{異常 }r} \text{local mass around }r.  
]

但因每個 dyadic block 的 local mass 可控制，應能求和。

## 7. 主弧 local CLT：多尺度版本

對角 character (h) 的 Fourier 係數是

[  
\prod_{pq\in\mathcal E}  
\left(1-\theta+\theta e\left(\frac{h}{pq}\right)\right).  
]

這是 weighted Bernoulli sum 的 characteristic function。

由於最大權重 (\ll\sigma)，小頻率部分應給 Gaussian：

[  
\exp\left(2\pi i h\mu -2\pi^2h^2\sigma^2\right).  
]

中頻與大頻需證衰減。多尺度權重有利，因為不同 (pq) 尺度阻止 (h/(pq)) 同時接近整數。

這正是 weighted Bernoulli local CLT 的適用場景。

因此主弧可望由現有 local CLT 工具或直接 characteristic-function proof 完成。

## 8. 現在可提出較精確的總定理

**Dyadic block graph theorem（目標）.**  
給定 squarefree (b) 與有限禁用素數集合 (F)，可選足夠大的 (k_0<K)，取所有不在 (F) 中的好素數 blocks (\mathcal P_k) for (k_0\le k\le K)，並連接所有 Irving-compatible block pairs

[  
\frac23\ell\le k\le\frac32\ell.  
]

適當加入 (B)-to-block 邊。則存在子邊集 (E) 使

[  
\sum_{pq\in E}\frac1{pq}=\frac1b.  
]

證明通過 Bernoulli 選邊與 Fourier：

- 取 (\theta) 使期望為 (1/b);
    
- 主弧 local CLT 給出正概率主項；
    
- 非對角 character 由 Irving-good blocks 衰減；
    
- span (1) 保證格點無障礙。
    

這就是現在最清楚的總路線。

## 9. 剩餘風險

仍有兩個真風險：

### 風險一：非對角加權衰減不夠求和

因為全圖頂點數巨大，character 數是

[  
\prod_{p\in P}p  
]

非常大。必須確保每個異常坐標帶來的衰減足以壓過約 (\log p) 的 choices。單純「固定比例短弧避免」可能要轉成足夠強的 exponential decay。Irving 給的分散是否足夠，要精算。

### 風險二：主弧 local CLT 在極細格 (1/L) 上

(L) 極大，目標格點非常細。Local CLT 需達到 span (1) 的精度，而不只是連續密度近似。這需要強 aperiodicity；可能要用非對角分析共同保證。

所以路線有希望，但仍重。

---

## 10. 下一步應精算風險一

我建議下一步先估算非對角衰減與 character 數是否平衡。

對一個異常頂點 (q\sim 2^k)，它有約

[  
\sum_{\ell\sim k}#\mathcal P_\ell  
\asymp  
\sum_{\ell\asymp k}\frac{2^\ell}{\ell}  
\asymp  
\frac{2^{3k/2}}{k}  
]

可鄰接素數？但我們也許不需要連到全部，且不同尺度權重/相位都可用。衰減可能極強，遠超 (\log q\asymp k)。

如果每個異常頂點真的能得到 (\gg 2^{ck}) 級別的相位衰減，那 character 數完全不是問題。

所以接下來要量化：Irving-good block pairs 能給每個異常坐標多少 Fourier 衰減。這會決定次弧是否真正閉合。
