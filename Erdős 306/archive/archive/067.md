好，現在先把最乾淨的基準版完整證出。這不是原問題本身，但它給出一個「若測試集合真像隨機集合，list-decoding 立刻成立」的標尺。然後我們再看 random dilated intervals 多出的困難。

---

## 1. Random subset 基準版

令 (p) 為大素數，令

[  
S_1,\dots,S_t\subset \mathbb F_p  
]

為獨立隨機集合，每個 (a\in\mathbb F_p) 以概率

[  
\alpha  
]

獨立收入每個 (S_i)。取參數

[  
0<\alpha<\tau<1.  
]

定義

[  
L=\left{  
a\in\mathbb F_p:  
#{i:a\in S_i}\ge \tau t  
\right}.  
]

我們要證：若

[  
t=C\log p  
]

且 (C) 夠大，則高概率下

[  
|L|\le p^\varepsilon.  
]

這裏 (\tau) 對應前面的 (1-\rho)，而 (\alpha) 對應短弧密度 (2\delta)。條件 (\tau>\alpha) 表示「要求通過的比例高於隨機期望」。

---

## 2. 對單點的 Chernoff

固定 (a\in\mathbb F_p)。令

[  
X_a=#{i:a\in S_i}.  
]

則

[  
X_a\sim \operatorname{Binomial}(t,\alpha).  
]

因為

[  
\tau>\alpha,  
]

Chernoff 給出某個常數 (c=c(\alpha,\tau)>0)，使

[  
\Pr(X_a\ge \tau t)\le e^{-ct}.  
]

所以

# [  
\mathbb E|L|

\sum_{a\in\mathbb F_p}\Pr(a\in L)  
\le  
p e^{-ct}.  
]

若取

[  
t>\frac{1-\varepsilon}{c}\log p,  
]

則

[  
\mathbb E|L|\le p^\varepsilon.  
]

但這只給期望，還不夠高概率。

---

## 3. 用 Chernoff 控制 (|L|)

在 random subset 模型中，不同 (a) 的事件其實獨立，因為各元素是否屬於各 (S_i) 是完全獨立的。因此

[  
1_{a\in L}  
]

對不同 (a) 獨立同分佈，且

[  
\Pr(a\in L)\le e^{-ct}.  
]

所以

[  
|L|  
\preceq \operatorname{Binomial}(p,e^{-ct}).  
]

令

[  
\mu=p e^{-ct}.  
]

取 (t) 使

[  
\mu\le \frac12 p^\varepsilon.  
]

再用 Chernoff：

[  
\Pr(|L|>p^\varepsilon)  
\le  
\exp(-c' p^\varepsilon).  
]

這是極強的尾界。

若還要對最多

[  
p^t=\exp(O((\log p)^2))  
]

種 shift patterns 做 union bound，仍然可以，因為

[  
\exp(-c'p^\varepsilon)  
]

遠小於

[  
\exp(O((\log p)^2)).  
]

所以 random subset 版完全成立。

---

## 4. 這個基準版告訴我們什麼？

如果每條測試邊對 (a_p) 施加的條件真像獨立隨機密度 (\alpha=2\delta) 的 subset，那麼只需

[  
t=C\log p  
]

條測試邊，就能把 (a_p) 的候選值壓到

[  
p^\varepsilon  
]

甚至更少。

這正好符合 entropy 尺度：每個頂點原本有 (\log p) 的自由度，用 (C\log p) 條獨立測試邊消耗掉它。

---

## 5. 但我們的集合不是 random subset

我們真正的測試集合是

[  
S_i=r_i^{-1}I_i,  
]

其中 (I_i) 是 (\mathbb F_p) 中一段長度約 (2\delta p) 的 interval，而

[  
r_i=q_i^{-1}\pmod p.  
]

這叫 random dilated intervals，而不是 random subsets。

對固定 (a)，仍有

[  
\Pr(a\in S_i)\approx 2\delta.  
]

但對不同 (a,a')，事件相關。若 (A\subset\mathbb F_p) 很結構化，某些 dilated intervals 可能同時覆蓋 (A) 的大部分。這是唯一多出的難點。

所以現在我們的任務變成：

[  
\boxed{  
\text{random dilated intervals 是否足夠像 random subsets？}  
}  
]

不需要完全像；只需要在「高通過比例」這個問題上足夠像。

---

## 6. Random dilated intervals 的可攻弱版

我們可以先證一個弱版：

對任意集合 (A\subset\mathbb F_p)，(|A|=m\ge p^\varepsilon)，隨機 (r\in\mathbb F_p^\times) 時，

[  
|A\cap r^{-1}I|  
]

很少會大於

[  
\tau m  
]

其中

[  
\tau>2\delta.  
]

如果能證

[  
\Pr_r\left(|A\cap r^{-1}I|\ge \tau m\right)  
\le e^{-c m}  
]

或至少非常小，則對 (t=C\log p) 個獨立 (r_i) 可得到類似 random subset 的高矩估計。

這個命題的意思是：一個固定大集合 (A) 不可能經常被一個隨機乘法膨脹的短 interval 覆蓋大比例。

---

## 7. 反例壓力測試

若

[  
A=s^{-1}I  
]

本身就是某個 dilated interval，則當

[  
r=s  
]

時，

[  
rA=I,  
]

命中比例為 (1)。但這只發生在一個 (r) 上，概率約

[  
1/p.  
]

如果 (m=|A|\approx 2\delta p)，那麼

[  
e^{-cm}  
]

比 (1/p) 小得多，所以不可能有那麼強的

[  
e^{-cm}  
]

界。這是一個重要修正。

因此對單次 (r)，最強尾界不可能是 (e^{-cm})。因為結構化 (A) 可有少數 exceptional dilates 完全命中。

但對 (t) 次獨立 dilates，要每次都命中高比例，就需要每個 (r_i) 落入一個小 exceptional set。若 exceptional set 大小不太大，仍可。

所以正確單次命題應是：

[  
#\left{  
r:  
|A\cap r^{-1}I|\ge\tau m  
\right}  
\le K(A,\tau,\delta),  
]

其中 (K) 應遠小於 (p)。

若能證

[  
K\le p^{1-\kappa}  
]

對所有 (A) with (m\ge p^\varepsilon)，則每次高命中概率 (\le p^{-\kappa})。取 (t=C\log p)，概率變成

# [  
p^{-\kappa t}

\exp(-\kappa C(\log p)^2),  
]

足以壓過許多 union bounds，也可能足夠。

---

## 8. 是否有這樣的 (K) 界？

這變成一個 incidence 問題：

給定 (A,I\subset\mathbb F_p)，估計

# [  
R_{\tau}

\left{  
r\in\mathbb F_p^\times:  
|rA\cap I|\ge\tau |A|  
\right}.  
]

若 (r\in R_\tau)，則許多 (a\in A) 使

[  
ra\in I.  
]

也就是有很多 incidences

[  
(r,a)\in R_\tau\times A,\quad ra\in I.  
]

總 incidence 至少

[  
\tau |A||R_\tau|.  
]

但平均 incidence 是

[  
\frac{|A||I||R_\tau|}{p}  
\approx 2\delta |A||R_\tau|.  
]

因為 (\tau>2\delta)，這是超過平均的。要控制它，需要乘法 incidence estimate。

這可用有限域 incidence / sum-product 工具。若 (A) 和 (R_\tau) 都大，乘積集 (R_\tau A) 應該很大，不可能大量落在短 interval (I)。所以 (R_\tau) 不能太大。

這正是 sum-product 型命題。

---

## 9. 新的原子工具

因此 random dilated interval 版歸結為：

**Multiplicative interval incidence lemma.**  
給定 (A\subset\mathbb F_p)、interval (I) of length (\alpha p)，若

[  
|rA\cap I|\ge \tau |A|  
]

with (\tau>\alpha)，則這樣的 (r)'s 不多。更定量地，

[  
|R_\tau|\le p^{1-\kappa}  
]

在合適範圍內成立，除非 (A) 極小。

這看起來是 finite-field sum-product / incidence 的標準味道。這比「所有 shifts uniform」更具體可查。

---

## 10. 對我們是否足夠？

若每個測試條件的 high-hit (r) 集合大小最多 (p^{1-\kappa})，那麼隨機 (r_i) 讓一個固定 large (A) 被高比例命中的概率約

[  
p^{-\kappa t}.  
]

取

[  
t=C\log p  
]

可得

[  
\exp(-\kappa C(\log p)^2).  
]

對所有 (A) union bound 仍困難，因 (A) 有 (2^p) 個。但在高矩/entropy proof 中不一定 union over all (A)；可以用 incidence lemma 在過程中控制。

若非要 union over all (A) of size (m=p^\varepsilon)，數量是

[  
\exp(p^\varepsilon\log p),  
]

遠大於 (\exp((\log p)^2))。所以還不夠。

這說明：單純 (p^{-\kappa}) 概率不夠 union over all (A)。需要 either 更強尾界，or 避免 union over all (A)，用 expectation/high-moment 直接數 assignments。

Again，平均/entropy method is better.

---

## 11. 回到平均 counting

對 Fourier 低能量 assignments，我們不需要所有 (A)。(A) 是候選 values induced by assignments and tests. 可以直接用 entropy chain rule：

[  
H(a_P)\le H(a_{\text{neighbors}})+\sum_p H(a_p\mid a_{T(p)}).  
]

如果 average conditional entropy

[  
H(a_p\mid a_{T(p)})  
]

小，則總低能量 assignment 少。這避開 union over arbitrary (A)。

This is exactly information-theoretic agreement proof.

---

## 12. 當前最穩的下一步

不要再試圖證所有 (A) uniform 的強命題。改用信息論語言：

對低能量 assignment 集合 (\mathcal A_R)，令 (A) 隨機均勻取自 (\mathcal A_R)。研究隨機變量

[  
A_p\in\mathbb F_p.  
]

測試邊通過率高意味

[  
A_p  
]

與鄰居 residues (A_{q_i}) 有很強約束。用 Fano-type / entropy inequality 證

[  
H(A_p\mid A_{q_1},\dots,A_{q_t})  
\le \varepsilon\log p.  
]

然後把所有頂點加總，用 Shearer / entropy subadditivity / agreement expansion 證

[  
H(A)\ll \text{major arc entropy}+O(R).  
]

這正是 agreement testing 的現代證明語言。

---

## 13. 方向更新

這一步有點抽象，但比 union bound over all sets 更正確。現在的 CRT agreement theorem 應該用 entropy 證，而不是逐集合 discrepancy 證。

簡化：

[  
\boxed{  
\text{低能量 assignments 的 entropy 很小。}  
}  
]

不是：

[  
\text{每個可能 list 都小。}  
]

這與 Fourier 求和目標完全對齊，因為我們本來就在數 assignments。

---

## 14. 接下來

下一步應寫 entropy version：

令 (\mathcal A_R) 是低能量 assignments 集合，令 (A) uniform on (\mathcal A_R). 若測試邊多數通過，證每個頂點的 conditional entropy (H(A_p\mid A_{T(p)})) 小。再用圖 entropy inequality 將 local conditional entropy 轉成 global entropy bound。

這會把 agreement testing 工具真正接上。
