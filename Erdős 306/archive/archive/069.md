## 1. Fourier 分配函數

令

[  
P  
]

是選定的素數頂點集合，

[  
L=\prod_{p\in P}p.  
]

候選邊集合為

[  
E.  
]

對每條邊 (e=pq)，權重為

[  
w_{pq}=\frac{L}{pq}.  
]

Bernoulli 隨機選邊後，Fourier 係數為

# [  
\widehat\mu(a)

\prod_{pq\in E}  
\left(  
1-\theta+\theta e(\phi_{pq}(a))  
\right),  
]

其中 character

[  
a=(a_p)_{p\in P},\qquad a_p\in\mathbb F_p,  
]

而

# [  
\phi_{pq}(a)

\frac{a_pq^{-1}}p+  
\frac{a_qp^{-1}}q.  
]

有基本估計

[  
|\widehat\mu(a)|  
\le  
\exp(-cQ(a)),  
]

其中

[  
Q(a)=  
\sum_{pq\in E}  
|\phi_{pq}(a)|^2.  
]

所以錯誤項本質上受控於

# [  
Z_{\mathrm{minor}}

\sum_{a\notin\mathfrak M}  
e^{-cQ(a)}.  
]

這就是「分配函數」：

[  
\boxed{  
Z=\sum_{\text{states}}e^{-c\cdot \text{energy}}.  
}  
]

我們要證：

[  
Z_{\mathrm{minor}}=o(Z_{\mathrm{major}}),  
]

或者至少

[  
Z_{\mathrm{minor}}\ll \frac{1}{\sigma}  
]

在 Fourier 反演未除以 (L) 前的尺度上。

## 2. 熵界正好是什麼？

設

[  
N(R)=  
#{a:Q(a)\le R}.  
]

若能證

[  
N(R)  
\le  
N_{\mathrm{maj}}(R),e^{\kappa R}  
]

其中 (\kappa<c)，那麼

[  
\sum_a e^{-cQ(a)}  
]

按能量層分解：

[  
\sum_{j\ge0}  
#{a:j\le Q(a)<j+1}e^{-cj}  
]

可由

[  
\sum_j e^{\kappa j}e^{-cj}  
]

控制，收斂。

所以我們真正需要的是：

[  
\boxed{  
\log N(R)  
\le  
\log N_{\mathrm{major}}(R)+\kappa R.  
}  
]

這就是低能態熵界。

換言之：

[  
\text{能量 }R\text{ 可購買的自由度至多 }O(R).  
]

這是非常「熱力學」的敘述。

## 3. 主弧族的大小

主弧族是對角 characters：

[  
a_p\equiv h\pmod p  
]

其中 (h) 是小整數。

若主弧頻率範圍約為

[  
|h|\le H,  
]

則

[  
N_{\mathrm{major}}\asymp H.  
]

通常

[  
H\asymp \frac1\sigma.  
]

所以主弧熵約為

[  
\log H\asymp \log(1/\sigma),  
]

非常小，相比

[  
\log L=\sum_p\log p  
]

幾乎可以忽略。

因此熵界要說：

[  
\boxed{  
\text{低能 assignments 的數量不接近 }L，  
\text{而只接近 }1/\sigma\text{ 乘以 }e^{O(R)}.  
}  
]

這是強命題，但形狀正確。

## 4. 為什麼 exact zero energy 支持它？

若

[  
Q(a)=0,  
]

則每條測試邊都有

[  
\phi_{pq}(a)=0.  
]

對一條邊而言，這等價於 CRT glue

[  
H_{pq}=0,  
]

所以

[  
a_p=0,\qquad a_q=0.  
]

若測試圖連通，則

[  
a_p=0  
]

對所有 (p)。

所以零能態只有一個：

[  
h=0.  
]

這完全符合熵界：

[  
N(0)=1.  
]

沒有零能玻璃態。這是很強的好兆頭。

## 5. 小能態為何應接近對角？

若

[  
a_p\equiv h\pmod p  
]

且 (h) 小，則每條邊

[  
pq  
]

的 CRT glue 是

[  
H_{pq}=h  
]

只要 (|h|<pq/2)。所以

[  
\phi_{pq}(a)=\frac{h}{pq}.  
]

能量約

# [  
Q(h)

h^2\sum_{pq\in E}\frac1{p^2q^2}.  
]

這就是 Gaussian 主弧。

所以低能基態族就是

[  
h=0,\pm1,\pm2,\dots  
]

的小整數族。

Agreement/Poincaré 命題應說：除此以外，任何 assignment 若想偏離對角，就必須在許多邊上付能量。

## 6. Poincaré 型命題

可以把核心 lemma 寫成：

**CRT Poincaré inequality，目標版。**  
對任何 assignment (a=(a_p))，存在一個整數 (h)，使

[  
Q(a)  
\ge  
c,D(a,h),  
]

其中 (D(a,h)) 衡量

[  
a_p\ne h\pmod p  
]

的加權程度。

更熵化地說：

[  
#{a:Q(a)\le R}  
\le  
\sum_h  
\exp(CR)  
]

其中 (h) 的範圍由能量控制。

這就是從 analytic Fourier 變成 graph/CRT Poincaré inequality。

## 7. 能量如何支付錯誤頂點？

假設某些頂點不服從同一 (h)。令

[  
S={p:a_p\not\equiv h\pmod p}.  
]

若測試圖有 expansion，則 (S) 有很多邊跨到 (S^c)。對跨邊 (pq)，一端 residue 來自 (h)，另一端不是 (h)。通常 CRT glue 不會小，除非該錯誤 residue 被精心選擇。

而每個錯誤頂點有 (C\log p) 條測試邊。要讓它們都小，需要滿足多個模區間條件，這壓低其可選值。若不滿足，就支付能量。

這就是：

[  
\text{entropy of wrong choices}  
\le  
\text{energy paid}.  
]

這句話正是我們要證的熵界。

## 8. 這避免了「素數鐵壁」嗎？

目前看，基本避免了要求極強素數隨機性。需要的素數輸入被限制在：

1. 可以選到足夠多測試鄰居；
    
2. 這些鄰居的逆元斜率不高度重合；
    
3. Irving-compatible block graph 有足夠 expansion 與質量。
    

第 1 點靠素數數量；  
第 2 點靠 prime residues / Irving / large sieve；  
第 3 點靠 Irving 的尺度範圍與 Mertens。

這都比「素數在所有短區間完全隨機」弱很多。

真正高難度在熵／agreement，而不是素數分佈鐵壁。這是較好的狀態。

## 9. 現在可以定義一個研究里程碑

我們可以把後續研究切成三個可完成小目標：

### 里程碑 1：模型 CRT Poincaré

把素數拿掉，令每個模數 (p) 是一般大 prime，但測試斜率 (r_{pq}) 隨機。證：

[  
\log N(R)\le \log H + CR.  
]

若模型版成立，說明熵機制本身可靠。

### 里程碑 2：prime derandomization

把隨機斜率替換為

[  
q^{-1}\bmod p  
]

並用 Irving/large sieve 選 good test graph。

### 里程碑 3：接回 Fourier

把 Poincaré 熵界接到 weighted Bernoulli local CLT，完成隨機子集和命中。

這樣研究不會散。

## 10. 下一步怎麼推？

最自然的下一步是證 **模型 CRT Poincaré** 的弱版。

即假設測試圖是一張隨機正則圖，邊上的斜率像獨立隨機，並證：

低能 assignments 只能是對角小 (h) 附近。

這個模型不涉及素數深分布。若它失敗，整條熵路線有問題；若它成功，再處理素數化。

我建議接下來做模型版，因為它能最清楚測試「信息論／熱力學」主意本身是否對。
