好。現在把剛才的 **dense block CRT agreement lemma** 接到能量分層上。目標是從：

[  
Q(a)=\sum_{pq\in E}|\phi_{pq}(a)|^2  
]

推出：

[  
\text{低能 }a \Longrightarrow a\text{ 接近某個全局對角 }h.  
]

---

## 1. 對一個 block pair 定義能量

取兩個 dyadic prime blocks：

[  
P_k={p:2^k\le p<2^{k+1}},  
]

[  
P_\ell={q:2^\ell\le q<2^{\ell+1}}.  
]

設

[  
X=2^k,\qquad Y=2^\ell.  
]

對這個完整 block pair，定義局部能量

# [  
Q_{k,\ell}(a)

\sum_{p\in P_k,\ q\in P_\ell}  
\left|  
\phi_{pq}(a)  
\right|^2.  
]

其中

[  
\phi_{pq}(a)=  
\frac{a_pq^{-1}}p+  
\frac{a_qp^{-1}}q.  
]

等價地，若 (H_{pq}^{*}) 是 CRT glue 的對稱代表，則

# [  
|\phi_{pq}(a)|

\left|\frac{H_{pq}^{*}}{pq}\right|.  
]

---

## 2. 設定 strong-good cutoff

取一個主弧尺度

[  
B\asymp C/\sigma.  
]

稱邊 (pq) 為 strong-good 若

[  
|H_{pq}^{*}|\le B.  
]

若不是 strong-good，則

[  
|H_{pq}^{*}|>B.  
]

於是該邊能量至少

[  
\left(\frac{B}{pq}\right)^2.  
]

在 (P_k\times P_\ell) 裏，

[  
pq\asymp XY.  
]

所以每條 bad edge 能量至少

[  
\gg \frac{B^2}{X^2Y^2}.  
]

---

## 3. bad edge 密度由能量控制

設 bad edge 比例為

[  
\varepsilon_{k,\ell}.  
]

總邊數約為

# [  
|P_k||P_\ell|  
\asymp  
\frac{X}{k}\cdot\frac{Y}{\ell}

\frac{XY}{k\ell}.  
]

因此 bad edges 對能量的貢獻至少

# [  
\varepsilon_{k,\ell}  
\frac{XY}{k\ell}  
\cdot  
\frac{B^2}{X^2Y^2}

\varepsilon_{k,\ell}  
\frac{B^2}{XY,k\ell}.  
]

所以

[  
Q_{k,\ell}(a)  
\gg  
\varepsilon_{k,\ell}  
\frac{B^2}{XY,k\ell}.  
]

換言之：

[  
\boxed{  
\varepsilon_{k,\ell}  
\ll  
Q_{k,\ell}(a)\frac{XY,k\ell}{B^2}.  
}  
]

若局部能量很低，bad edge 密度很低，dense block agreement lemma 就可用。

---

## 4. agreement threshold

Dense block lemma 需要 bad density 小，比如

[  
\varepsilon_{k,\ell}<\varepsilon_0  
]

其中 (\varepsilon_0) 是固定小常數。這等價於要求

[  
Q_{k,\ell}(a)  
\ll  
\frac{B^2}{XY,k\ell}.  
]

定義 threshold：

# [  
T_{k,\ell}

c_0\frac{B^2}{XY,k\ell}.  
]

若

[  
Q_{k,\ell}(a)\le T_{k,\ell},  
]

則 block pair (P_k\times P_\ell) 是 agreement-good，產生一個 label

[  
h_{k,\ell}  
]

解釋該 pair 中大多數頂點。

若

[  
Q_{k,\ell}(a)>T_{k,\ell},  
]

則這個 block pair 付出至少

[  
T_{k,\ell}  
]

能量。

---

## 5. 這個 threshold 大不大？

這是關鍵。

取典型相連尺度

[  
\ell\asymp k,  
]

尤其上層比較器情形

[  
Y=X^{3/2}.  
]

先估 (B)。最低尺度 (X_0=2^{k_0})，主弧寬度

[  
B\asymp 1/\sigma.  
]

前面估過

[  
\sigma\asymp \frac{1}{k_0X_0}  
]

大致由最低尺度主導，所以

[  
B\asymp k_0X_0.  
]

若現在 (X\ge X_0)，則

[  
T_{k,\ell}  
\asymp  
\frac{k_0^2X_0^2}{XY,k\ell}.  
]

對最低層向上比較：

[  
X=X_0,\qquad Y=X_0^{3/2},  
]

有

[  
T  
\asymp  
\frac{k_0^2X_0^2}  
{X_0^{5/2}k_0\ell}  
\asymp  
\frac{k_0}{\ell X_0^{1/2}}  
\asymp  
\frac{1}{X_0^{1/2}}.  
]

這很小。

也就是：一個 block pair 要變成 agreement-bad，只需付出很小能量。這看起來有風險。

但注意：Fourier 能量尺度本來也可能很小；主弧 Gaussian 衰減在 (h\sim B) 時是常數。若每個 bad block 只付 (X_0^{-1/2})，那可能不足以壓 entropy。

這提示：strong-good cutoff (B=1/\sigma) 太大，導致 bad edge 能量門檻太低。

---

## 6. 調整 cutoff：多層 cutoff

不應只用單一

[  
B=1/\sigma.  
]

對 agreement 傳播而言，在 block pair (P_k\times P_\ell) 上，可以用更小 cutoff：

[  
B_{k,\ell}  
]

例如取

[  
B_{k,\ell}\asymp \min(2^k,2^\ell)^{1-\eta}  
]

或與該 block 的自然 Gaussian scale相關。

主弧 (h) 若大於 (B_{k,\ell})，則該 block pair 不是 strong-good，但它的相位

[  
h/(pq)  
]

仍小，並貢獻 Gaussian 能量。

也就是說：

- 小 (h) 由 agreement strong-good 傳播；
    
- 較大 (h) 不靠 agreement，而由主弧 Gaussian 控制。
    

因此在 Fourier 分析中，agreement theorem 只需處理非對角 characters；對角 (h) 可以允許在某些 low block 上不是 strong-good。

這讓 cutoff 可選得更小。

---

## 7. 改成相對於「候選 (h)」的 agreement

更自然的做法：不是先固定 (B)，而是對每個可能 (h) 比較：

邊 (pq) 對 (h) 是 good 若

[  
H_{pq}^{*}=h  
]

或

[  
|H_{pq}^{*}-h|  
\le B_{local}.  
]

但 (H_{pq}^{*}) 是整數；若 (h) 是整數，且局部比較模數 product 足夠大，可以直接要求 equality modulo many primes。

這會使 agreement theorem 變成：

若 assignment 不接近任何 diagonal (h)，則在每個 (h) 周圍都會有很多 bad edges，支付能量。

此時 (h) 的大小由主弧 local CLT 處理，不必在 agreement 裏用一個固定巨大 (B)。

這比單一 cutoff 更準確。

---

## 8. 兩層處理：先找局部 labels，再讓 labels 進入主弧

Dense block lemma 用 cutoff (B_{\text{block}}) 產生局部 label (h)。這個 cutoff 可以取為較小值，例如

[  
B_{\text{block}}<\min(Q\text{-side prime})/2  
]

但不必等於全局主弧寬度。

若某對角 (h) 大於 (B_{\text{block}})，它不會被這個 agreement lemma 視為 strong-good；但這樣的 (h) 在該 block 上有相位

[  
h/(pq)  
]

，貢獻能量。這能量正是主弧 Gaussian tail。

所以把 diagonal (h) 分給主弧，不由 agreement 控制。

對非對角 characters，如果它想避免能量，必須在許多 block 上有某些 local labels (h_{block})。稠密 agreement 會迫使這些 local labels 一致。最終得到 diagonal (h)。若這個 h 大，則由對角 high-frequency 衰減處理。

這是更好的分工：

[  
\boxed{  
\text{agreement 只證「大係數必近似某個對角 }h\text{」；}  
}  
]

[  
\boxed{  
\text{該 }h\text{ 大小由一維 local CLT / Gaussian 控制。}  
}  
]

---

## 9. 因此 cutoff 可取 block-natural

對 block pair (P_k\times P_\ell)，取 cutoff

[  
B_{k,\ell}  
]

滿足：

1. (B_{k,\ell}<\frac12\max(X,Y))，保證高層比較器有效；
    
2. 若兩個低 primes (p_1,p_2) 共同比較，要求
    

[  
p_1p_2>2B_{k,\ell}.  
]

可取

[  
B_{k,\ell}=X^{1+\eta}  
]

若低層為 (X)，高層為 (Y=X^{3/2}). Then:

[  
B_{k,\ell}<Y/2  
]

且

[  
X^2>2B_{k,\ell}  
]

若 (\eta<1)。

這給 agreement 傳播。

Bad edge energy threshold:

# [  
T_{k,\ell}

\frac{B_{k,\ell}^2}{XY,k\ell}.  
]

若

[  
B=X^{1+\eta},\quad Y=X^{3/2},  
]

則

# [  
T  
\asymp  
\frac{X^{2+2\eta}}{X^{5/2}k\ell}

\frac{X^{-1/2+2\eta}}{k\ell}.  
]

若取

[  
\eta>1/4,  
]

則 (T) grows as a power of (X) divided by logs. Great.

Also need (B<X^2), (\eta<1); and (B<Y=X^{3/2}), (\eta<1/2). So choose

[  
1/4<\eta<1/2.  
]

Excellent! Then an agreement-bad block pair pays large energy.

This is a major improvement.

---

## 10. What about diagonal h with (h>B_{k,\ell})?

It will make many edges bad under this cutoff, paying energy (h^2\sum1/(p^2q^2)). That is okay; if h is that large, the diagonal Fourier coefficient should decay. Such h belongs not to small major arc for that block.

Global main arc can be defined by all blocks' Gaussian energy; large h is handled by energy, not agreement.

Thus choosing smaller block cutoffs is valid.

---

## 11. Updated dense agreement lemma with block cutoff

For each upward block pair (P_k\times P_{3k/2}):

- Low side (X=2^k);
    
- High side (Y=X^{3/2});
    
- Choose
    

[  
B_k=X^{1+\eta},\qquad 1/4<\eta<1/2.  
]

Then:

[  
B_k<Y/2,  
]

and for any two distinct low primes (p_1,p_2),

[  
p_1p_2>2B_k.  
]

So dense block agreement works.

If strong-good density is (1-\varepsilon), get a label (h_k) with

[  
|h_k|\le B_k  
]

explaining most vertices in both blocks.

If no such agreement, block energy at least

# [  
T_k\asymp \frac{B_k^2}{XY,k^2}

\frac{X^{-1/2+2\eta}}{k^2}.  
]

Since (-1/2+2\eta>0), (T_k\to\infty). Thus bad block pair is extremely costly in Fourier.

This is excellent.

---

## 12. Consequence

For large scales, any character with non-negligible Fourier coefficient must be agreement-good on all upward comparison block pairs. Therefore it yields labels (h_k) across scales. Overlaps force these labels to agree where cutoffs allow.

Eventually character is diagonal with some h, at least on most vertices. Then diagonal analysis takes over.

This dramatically strengthens the route.
