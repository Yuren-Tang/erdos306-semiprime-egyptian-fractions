好。現在把剛才形成的核心整理成一個可用命題，稱為 **backbone agreement proposition**。

---

## 1. Backbone 的選擇

我們不把所有候選邊都拿來做 agreement。候選圖仍然是完整的 compatible dyadic graph：

# [  
E_{\mathrm{full}}

\bigcup_{\frac23\ell\le k\le\frac32\ell}  
P_k\times P_\ell.  
]

它負責兩件事：

1. 提供總質量
    

[  
\sum_{pq\in E_{\mathrm{full}}}\frac1{pq}  
\asymp  
\sum_k\frac1k  
]

的發散；

2. 提供 Bernoulli 子集和的主弧 local CLT。
    

而 agreement 只需要一條骨架：

# [  
E_{\mathrm{bb}}

\bigcup_k P_k\times P_{\lceil 3k/2\rceil}.  
]

這條 backbone 不必提供發散質量；它只負責控制 Fourier characters。

對每個 (k)，令

[  
X_k=2^k,  
\qquad  
Y_k=X_k^{3/2}.  
]

選 cutoff

[  
B_k=X_k^{1+\eta},  
\qquad  
\frac14<\eta<\frac12.  
]

這個範圍很關鍵，因為它同時保證：

[  
B_k<Y_k/2,  
]

以及

[  
X_k^2>2B_k  
]

對大 (k) 成立。

---

## 2. Backbone block 的 good/bad

對一條 backbone 邊

[  
p\in P_k,\quad q\in P_{\lceil3k/2\rceil},  
]

令

[  
H_{pq}^{*}\in(-pq/2,pq/2]  
]

為 CRT glue：

[  
H_{pq}^{_}\equiv a_p\pmod p,  
\qquad  
H_{pq}^{_}\equiv a_q\pmod q.  
]

稱此邊為 (k)-strong-good，若

[  
|H_{pq}^{*}|\le B_k.  
]

否則為 (k)-bad。

若 (k)-bad，則其 Fourier 能量至少

[  
\left(\frac{B_k}{pq}\right)^2.  
]

因為

[  
pq\asymp X_kY_k=X_k^{5/2},  
]

故單條 bad 邊貢獻至少

# [  
\asymp  
\frac{B_k^2}{X_k^5}

# X_k^{2+2\eta-5}

X_k^{-3+2\eta}.  
]

但 block 中邊數約

# [  
|P_k||P_{3k/2}|  
\asymp  
\frac{X_k}{k}\cdot\frac{X_k^{3/2}}{k}

\frac{X_k^{5/2}}{k^2}.  
]

若 bad 邊密度為 (\varepsilon_k)，則該 block 的 bad energy 至少

# [  
\varepsilon_k\frac{X_k^{5/2}}{k^2}  
\cdot X_k^{-3+2\eta}

\varepsilon_k\frac{X_k^{-1/2+2\eta}}{k^2}.  
]

設

[  
T_k=  
c\frac{X_k^{-1/2+2\eta}}{k^2}.  
]

因為

[  
\eta>\frac14,  
]

所以

[  
T_k\to\infty.  
]

這是極好的：在大尺度上，一個 backbone block 若有固定比例 bad edges，就付出巨大能量。

---

## 3. Dense agreement on one backbone block

若 (P_k\times P_{3k/2}) 中 (k)-bad 邊密度很小，則 dense block agreement lemma 給出一個 label

[  
h_k,\qquad |h_k|\le B_k,  
]

使得除少數例外外，

[  
a_p\equiv h_k\pmod p  
\quad(p\in P_k),  
]

且

[  
a_q\equiv h_k\pmod q  
\quad(q\in P_{3k/2}).  
]

證明核心是：

- 高層 (q) 因為 (q>B_k)，所有 incident strong-good glues 必相等；
    
- 兩個高層點若共享兩個 strong-good 低鄰居，則 labels 相等，因為兩個低素數乘積 (>2B_k);
    
- 稠密 good 邊保證共享鄰居大量存在。
    

這一步基本已經清楚。

---

## 4. Adjacent labels merge

若兩個相鄰 backbone blocks 都 good：

[  
P_k\leftrightarrow P_{3k/2},  
]

[  
P_{3k/2}\leftrightarrow P_{(3/2)^2k},  
]

則它們分別給 labels

[  
h_k,\qquad h_{3k/2}.  
]

二者都解釋中間 block (P_{3k/2}) 的大多數頂點。故對許多

[  
q\in P_{3k/2}  
]

有

[  
h_k\equiv h_{3k/2}\pmod q.  
]

若 (h_k\ne h_{3k/2})，則

[  
D=h_{3k/2}-h_k  
]

被許多 (q\asymp X_k^{3/2}) 整除。但

[  
|D|  
\le  
B_{3k/2}+B_k  
\ll  
X_k^{\frac32(1+\eta)}.  
]

兩個這樣的 (q) 的乘積為

[  
\asymp X_k^3,  
]

而

[  
X_k^3

X_k^{\frac32(1+\eta)}  
]

只要

[  
3>\frac32(1+\eta),  
]

即

[  
\eta<1.  
]

當然成立。所以兩個共同中間素數已迫使

[  
h_k=h_{3k/2}.  
]

因此 good backbone blocks 的 labels 沿尺度圖合併。

## 5. Backbone agreement proposition

現在可以正式寫：

**Backbone agreement proposition（目標版）.**  
固定

[  
\frac14<\eta<\frac12.  
]

取足夠大的起始尺度 (k_0)。令 backbone 為

# [  
E_{\mathrm{bb}}

\bigcup_{k_0\le k\le K}  
P_k\times P_{\lceil3k/2\rceil}.  
]

對任意 character assignment (a=(a_p))，定義 backbone energy

# [  
Q_{\mathrm{bb}}(a)

\sum_{pq\in E_{\mathrm{bb}}}  
|\phi_{pq}(a)|^2.  
]

則存在一組 bad backbone levels (\mathcal B(a))，使得：

1. 每個 (k\in\mathcal B(a)) 支付能量至少
    

[  
T_k\asymp \frac{X_k^{2\eta-1/2}}{k^2};  
]

2. 在去掉這些 bad levels 後，每個連通尺度分量有一個整數 label (h_C)，使得該分量中大多數頂點滿足
    

[  
a_p\equiv h_C\pmod p;  
]

3. 不同連通分量的 labels 是唯一的；若分量重新連接，labels 會合併。
    

因此，若

[  
Q_{\mathrm{bb}}(a)\le R,  
]

則能出現的 bad levels 只能滿足

[  
\sum_{k\in\mathcal B(a)}T_k\ll R.  
]

換言之：

[  
\boxed{  
\text{每切斷一次全局 agreement，要付出 }T_k\text{ 能量。}  
}  
]

## 6. 熵如何計算？

若所有 levels 都 good，則只有一個 label

[  
h.  
]

若有 (m) 個 bad cuts，尺度 backbone 被分成 (m+1) 個 components，每個 component 有自己的 label

[  
h_j.  
]

第 (j) 個 component 的 label 大小受該 component 最低 cutoff 控制，大致

[  
|h_j|\le B_{k_j}.  
]

所以每個 component label 的 entropy 大約

[  
\log B_{k_j}\ll k_j.  
]

而切斷該 component 的 bad level (k_j) 支付

[  
T_{k_j}  
\asymp  
\frac{X_{k_j}^{2\eta-1/2}}{k_j^2}.  
]

對足夠大的 (k_0)，有

[  
T_{k_j}\gg k_j.  
]

因此一個 bad cut 的能量足以支付新增 label 的 entropy。

所以可得：

[  
\log N_{\mathrm{bb}}(R)  
\le  
\log N_{\mathrm{diag}}(R)+C R.  
]

這正是我們要的 energy–entropy bound。

## 7. Exception vertices 的 entropy

Dense block agreement lemma 每個 good block pair 只保證「大多數」頂點被 label 解釋，仍有少數例外。

這些例外來自 bad edges。若一個頂點例外很多，它必 incident to many bad edges, hence paid energy. Thus their residues' entropy is also paid by energy.

更具體地，若在 block (P_k) 中有 (\delta |P_k|) 個例外頂點，每個例外頂點在 adjacent good block pair 中少了許多 strong-good 邊，造成 bad edge 密度 (\gg\delta), hence energy (\gg\delta T_k). The entropy of those vertices is at most

[  
\delta |P_k|\log X_k,  
]

and because (T_k) arose from all block edges, one must check constants. But dense block edge count is huge, so bad edge energy should dominate exception entropy for (k_0) large.

這裏需要一個精算，但方向正確。

## 8. 接到 full graph

Backbone 只控制 characters。Full graph Fourier coefficient 滿足

[  
|\widehat\mu_{\mathrm{full}}(a)|  
\le  
|\widehat\mu_{\mathrm{bb}}(a)|  
]

因為 full graph product includes backbone factors and all factors have modulus (\le1).

所以若 backbone 已使非對角 characters 衰減，full graph更強。

主弧 local CLT 則用 full graph 權重，因為 full graph提供足夠質量與方差。

因此分工完美：

[  
\boxed{  
\text{backbone controls structure;}  
}  
]

[  
\boxed{  
\text{full graph supplies mass and Gaussian local CLT.}  
}  
]

---

## 9. 這是目前最好的路線

相比前面的 sparse test / entropy / list agreement，現在版本簡潔得多：

- 不需要隨機測試圖；
    
- 不需要 random dilated interval discrepancy；
    
- 不需要深 list agreement theorem；
    
- 只用稠密二部 CRT agreement + 多尺度 cutoff；
    
- Irving 仍用於 full graph 的 non-diagonal reciprocal dispersion / local CLT compatibility，而 backbone agreement 主要是初等 CRT。
    

這大幅降低了難度。
