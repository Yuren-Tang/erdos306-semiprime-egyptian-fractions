好。現在證或至少精確化 **vertex correction lemma**。這是剛才發現的關鍵：例外頂點不是自由的；只要它連到一個已經有 label 的 block，它若不採用同一 label，就會在大量 incident edges 上產生相位分散，從而付出巨大 Fourier 能量。

---

## 1. 設定

令 (P) 是一個已經被 label (h) 控制的素數 block：

[  
P\subset[X,2X],  
]

並假設對大多數

[  
p\in P  
]

有

[  
a_p\equiv h\pmod p.  
]

取另一個頂點

[  
q\sim Q  
]

與 (P) 相連，且尺度在 Irving-compatible 範圍內：

[  
Q^{2/3}\le X\le Q^{3/2}.  
]

現在看 (q) 的 residue

[  
a_q\in\mathbb F_q.  
]

若

[  
a_q\equiv h\pmod q,  
]

則邊相位接近對角：

# [  
\phi_{pq}

\frac{h}{pq}.  
]

若

[  
a_q\not\equiv h\pmod q,  
]

令

[  
d=a_q-h\not\equiv0\pmod q.  
]

我們要證：這個錯誤 (d) 會在 (p\in P) 的 incident edges 上造成大量能量。

---

## 2. 相位差

對一條邊 (pq)，若 (a_p\equiv h\pmod p)，則

# [  
\phi_{pq}

\frac{h(q^{-1}\bmod p)}p  
+  
\frac{a_q(p^{-1}\bmod q)}q.  
]

而對角 (h) 的相位是

# [  
\phi_{pq}^{\mathrm{diag}}

\frac{h}{pq}.  
]

兩者相差

[  
\phi_{pq}-\phi_{pq}^{\mathrm{diag}}  
\equiv  
\frac{(a_q-h)(p^{-1}\bmod q)}q  
\pmod1.  
]

也就是

[  
\phi_{pq}  
\equiv  
\frac{h}{pq}  
+  
\frac{d,p^{-1}}q  
\pmod1.  
]

其中

[  
d\not\equiv0\pmod q.  
]

因為

[  
\frac{h}{pq}  
]

很小，主要分散來自

[  
\frac{d,p^{-1}}q.  
]

所以只要

[  
p^{-1}\pmod q  
]

在 (p\in P) 中分散，則錯誤 residue (d) 會造成大量相位遠離整數。

---

## 3. 需要的分散命題

我們需要如下形式：

對大多數模數 (q\sim Q)，對所有非零

[  
d\in\mathbb F_q^\times,  
]

有

[  
\sum_{p\in P}  
\left|  
\frac{d,p^{-1}}q  
\right|^2  
\gg |P|.  
]

甚至弱一些也夠：

存在固定 (c,\delta>0)，使至少 (c|P|) 個 (p\in P) 滿足

[  
\left|  
\frac{d,p^{-1}}q  
\right|\ge \delta.  
]

這正是 prime inverse residues 不集中於短 interval 的命題。

Irving 的 Kloosterman-over-primes estimates 正應該給這種結果，至少對平均上的 good (q)。如果某些 (q) 是 bad moduli，我們可以避開它們或把它們放入例外集合；Irving 是平均定理，足以選 good blocks。

---

## 4. 對 (h/(pq)) 的小擾動

需要確認：

[  
\frac{h}{pq}  
]

不會抵消

[  
\frac{d p^{-1}}q.  
]

若

[  
\left|\frac{d p^{-1}}q\right|\ge 2\delta  
]

且

[  
\left|\frac{h}{pq}\right|\le \delta,  
]

則

[  
|\phi_{pq}|\ge \delta.  
]

對主弧或 labels (h) 的大小，我們通常有

[  
|h|\le B_{\text{component}}.  
]

在 relevant block 中，只要

[  
B_{\text{component}}\ll pq,  
]

則

[  
|h/(pq)|  
]

很小。這當然成立，因為 (pq) 巨大；即使 (h) 是某個 cutoff (B_k=X^{1+\eta})，而 (pq\sim X^{5/2})，也有

[  
h/(pq)\sim X^{-3/2+\eta}\to0.  
]

所以小擾動無問題。

---

## 5. Vertex correction energy

若 (q) 選錯 residue，則對至少 (c|P|) 條 incident edges，

[  
|\phi_{pq}|\ge \delta.  
]

因此 incident energy

[  
\sum_{p\in P}|\phi_{pq}|^2  
\gg |P|.  
]

而

[  
|P|\asymp \frac{X}{\log X}.  
]

這遠大於該頂點的 entropy

[  
\log q\asymp \log Q.  
]

所以錯誤 residue 付出的 energy 遠遠足以支付其自由度。

這非常強：

[  
\boxed{  
\text{一個錯誤頂點的能量成本 } \gg X/\log X  
\gg \log q.  
}  
]

因此例外頂點不會造成 entropy 問題。

---

## 6. 若 labelled block 只有大多數正確？

前面假設 (P) 中所有 (p) 都滿足

[  
a_p\equiv h\pmod p.  
]

實際上只有大多數。令好集合

[  
P_h={p\in P:a_p\equiv h\pmod p}.  
]

若

[  
|P_h|\ge (1-\epsilon)|P|,  
]

則對 (p\in P_h) 使用上述分散。只要 (\epsilon<c/2)，仍有

[  
\gg |P|  
]

條邊提供能量。

所以少數例外不影響 vertex correction。

---

## 7. 若 (q) 本身是 Irving-bad modulus？

Irving 平均定理可能只保證大多數 (q) 是 good。對 bad (q)，上述分散不一定成立。

解法：

- 在選 vertex sets 時，刪去 Irving-bad (q)；
    
- 或把 bad (q) 當成例外頂點，由其他鄰接 blocks 控制；
    
- Irving 平均估計若足夠強，bad (q) 的比例很小，其 entropy/edges 可被吸收。
    

最好做法：在構造 dyadic blocks 時，只保留 good primes，令

[  
Q^{\mathrm{good}}\subset Q  
]

仍有正比例或幾乎全部質量。因候選質量只需常數倍，刪去小比例 primes 無問題。

所以我們可以把 vertex correction lemma 限定在 good vertices 上。

---

## 8. Vertex correction lemma 正式版

**Vertex correction lemma.**  
令 (P\subset[X,2X]) 是一個 good prime block，(q\sim Q) 是 good modulus，且 (X,Q) 在 Irving-compatible 範圍內。假設

[  
a_p\equiv h\pmod p  
]

對至少 ((1-\epsilon)|P|) 個 (p\in P) 成立。若

[  
a_q\not\equiv h\pmod q,  
]

則

[  
\sum_{p\in P}  
|\phi_{pq}(a)|^2  
\gg |P|.  
]

其中常數在 (\epsilon) 小時正。

因此，若總 Fourier energy 不大，則每個 good (q) connected to a labelled block 必須滿足

[  
a_q\equiv h\pmod q.  
]

---

## 9. 這關閉了 exception entropy

若一個 block component 有 label (h)，任何相鄰 good block 的 vertex 若不採用 (h)，都會付出 incident energy (\gg |P|). 因此：

- 錯誤 vertices 的數量 (m) 至少造成能量
    

[  
\gg m\cdot |P|  
]

（需避免重複計數，但可選 disjoint incident blocks 或除以常數）；

- 它們的 entropy 只有
    

[  
\sum \log q.  
]

因為

[  
|P|\gg \log q,  
]

energy 足夠支付 entropy。

所以 dense block agreement 中留下的 (O(\varepsilon)) exception vertices 不再是自由問題；它們若真錯，就被 Irving correction 殺掉。

---

## 10. 這也處理 bad scale components 的邊界頂點

若尺度 graph 被 bad cuts 切開，component label 可能不同。對位於 component 邊界附近的 vertices，若它們連到相鄰 component with different label，則 incident energy large unless the connecting block pair is marked bad and energy-paid.

因此 labels 只能在 paid bad cuts 之間變化。這與前面「bad cuts cost energy」一致。

---

## 11. 現在的 agreement proof 幾乎閉合

我們有三個強部件：

1. **Dense block agreement:**  
    Low bad-edge density in (P_k\times P_{1.5k}) gives block label.
    
2. **Label merging:**  
    Adjacent good block labels agree by product of common primes.
    
3. **Vertex correction via Irving:**  
    Any vertex deviating from established label pays huge incident energy.
    

這三者合起來給：

[  
\boxed{  
\text{low-energy character } \Rightarrow \text{piecewise diagonal, cuts energy-paid}.  
}  
]

再加上 cut energy entropy accounting，得到

[  
\log N(R)\le \log N_{\mathrm{diag}}(R)+CR.  
]

這就是我們需要的次弧結構控制。
