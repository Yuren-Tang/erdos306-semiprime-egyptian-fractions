好。現在把 **minor-arc Fourier summation** 的框架寫清楚。這一步的目標是證：

[  
\sum_{a\notin\mathfrak M}|\widehat\mu(a)|  
=o(1/\sigma),  
]

其中 (\mathfrak M) 是小對角主弧。

Fourier 反演中最後會除以 (L)，主項大小約

[  
\frac1{\sigma L},  
]

所以在除以 (L) 前，minor arcs 總量要小於

[  
\frac1\sigma.  
]

---

## 1. Fourier 能量

對 character

[  
a=(a_p)_{p\in P},  
]

Fourier 係數滿足

[  
|\widehat\mu(a)|  
\le  
\exp(-cQ_{\mathrm{full}}(a)),  
]

其中

# [  
Q_{\mathrm{full}}(a)

\sum_{pq\in E_{\mathrm{full}}}  
|\phi_{pq}(a)|^2.  
]

我們用 backbone 和 Irving correction 控制 (Q_{\mathrm{full}})。把 character 分成三種偏離：

1. **bad scale cuts**：某些 backbone block pair 不 agreement-good；
    
2. **wrong vertices**：在一個 labelled component 中，某些頂點 residue 不等於 component label；
    
3. **diagonal high frequency**：所有頂點都等於同一 (h)，但 (h) 不在小主弧。
    

這三種都要付能量。

---

## 2. Bad scale cuts 的熵與能量

Backbone levels 取

[  
k_0,\gamma k_0,\gamma^2k_0,\dots  
]

或者在 dyadic integer levels 中取近似鏈，其中

[  
1<\gamma<3/2,  
]

例如

[  
\gamma=\frac43.  
]

對 level (k)，cutoff

[  
B_k=X_k^{1+\eta},  
\qquad  
X_k=2^k,  
]

其中

[  
\frac{\gamma-1}{2}<\eta<\gamma-1.  
]

例如 (\gamma=4/3,\eta=1/4)。

若 backbone pair (k\to\gamma k) 不是 agreement-good，則它付出能量至少

[  
T_k\asymp  
\frac{X_k^{1-\gamma+2\eta}}{k^2}.  
]

我們選參數使

[  
1-\gamma+2\eta>0,  
]

所以

[  
T_k\to\infty.  
]

每個 bad cut 可把尺度 graph 切開，讓左右 components 有不同 label。這新增一個 label，其 entropy 約

[  
\log B_{\gamma k}  
\ll k.  
]

而

[  
T_k\gg k  
]

只要 (k_0) 取足夠大。因此

[  
\text{新增 label entropy}  
\ll  
\text{bad cut energy}.  
]

所以 bad cuts 的總貢獻可被

[  
e^{-c\sum T_k}  
]

壓住。

---

## 3. Wrong vertices 的熵與能量

在某個 labelled component 中，label 為

[  
h_C.  
]

若一個 vertex (q) 選了錯 residue：

[  
a_q\not\equiv h_C\pmod q,  
]

則取一個相鄰 labelled block (P)，其中大多數 (p\in P) 滿足

[  
a_p\equiv h_C\pmod p.  
]

令

[  
d=a_q-h_C\not\equiv0\pmod q.  
]

對 (p\in P)，

[  
\phi_{pq}  
\equiv  
\frac{h_C}{pq}  
+  
\frac{d,p^{-1}}q.  
]

Irving-goodness 給出：對固定比例的 (p\in P),

[  
\left|  
\frac{d,p^{-1}}q  
\right|  
\ge \delta.  
]

因

[  
h_C/(pq)  
]

很小，得

[  
|\phi_{pq}|\ge \delta/2  
]

對 (\gg |P|) 條 incident edges 成立。因此該錯誤頂點支付能量

[  
\gg |P|.  
]

而該頂點 residue 的 entropy 只有

[  
\log q.  
]

因為

[  
|P|\asymp \frac{X}{\log X}\gg \log q  
]

對大尺度成立，故 wrong vertices 的自由度被 incident energy 遠遠壓住。

所以 wrong vertices 對 partition function 的總貢獻也可控。

---

## 4. Piecewise diagonal characters

移除 bad cuts 和 wrong vertices 後，character 變成 piecewise diagonal：

在每個 scale component (C) 上存在 label

[  
h_C  
]

使

[  
a_p\equiv h_C\pmod p  
]

對 component 內所有頂點成立。

如果沒有 bad cuts，只有一個 label (h)，即真正對角 character。

如果有 bad cuts，labels 可不同，但每個分裂都已由 bad cut energy 支付。故 piecewise diagonal labels 的 entropy 也由 cuts 支付。

因此，非對角 characters 的總量可壓到與對角族同一量級再乘

[  
e^{-c\cdot\text{paid energy}}.  
]

---

## 5. 對角高頻

現在只剩真正對角：

[  
a_p\equiv h\pmod p  
]

對所有 (p)。

Fourier 係數變成

# [  
\Phi(h)

\prod_{pq\in E_{\mathrm{full}}}  
\left(  
1-\theta+\theta e(h/pq)  
\right).  
]

對小 (h),

[  
|h|\lesssim 1/\sigma,  
]

這是主弧。

對大 (h),

[  
|h|\gg1/\sigma,  
]

若仍在小相位區，即

[  
|h|a_{\max}\ll1,  
]

則二次展開給出

[  
|\Phi(h)|  
\le  
\exp(-c h^2\sigma^2).  
]

所以

[  
\sum_{|h|>C/\sigma,\ |h|a_{\max}\ll1}  
|\Phi(h)|  
\ll  
\sum_{|h|>C/\sigma}e^{-c h^2\sigma^2}  
=o(1/\sigma)  
]

取 (C) 大即可。

若

[  
|h|a_{\max}\gtrsim1,  
]

則相位開始繞圈。這是 lattice high-frequency。這裏有兩種處理方式：

1. 直接用 weighted Bernoulli local CLT 的 aperiodicity condition；
    
2. 用我們的 agreement/backbone energy 觀點：大 (h) 超過許多 block cutoffs (B_k)，因此在那些 backbone blocks 上不是 strong-good，支付 (T_k) 級能量。
    

第二種很有用。若 (h>B_k), 則對 block (k\to\gamma k) 中所有邊，

[  
H_{pq}^{*}=h \quad \text{mod }pq  
]

只要 (|h|<pq/2)，但 strong-good cutoff (B_k) 失敗。該 block 被視為 bad，支付 energy threshold。若 (h) 更大超過 (pq/2)，實際相位也不會長期保持小；full graph 多尺度應更強地衰減。

所以 diagonal high-frequency 也可由 energy in many scales 控制。

---

## 6. 對角高頻的 scale counting

若

[  
|h|>B_k=X_k^{1+\eta},  
]

則 backbone level (k) 不是 strong-good under cutoff (B_k), unless reduction modulo (pq) somehow brings (h) back small. For diagonal h, (H_{pq}^{_}) is the symmetric representative of (h\bmod pq). If (|h|<pq/2), then indeed (H_{pq}^{_}=h).

For scales with

[  
pq\asymp X_k^{1+\gamma}>2|h|,  
]

badness is clear. These are scales satisfying

[  
X_k^{1+\gamma}>2|h|.  
]

Also need

[  
B_k<|h|.  
]

Thus scales with

[  
X_k^{1+\eta}<|h|<\tfrac12 X_k^{1+\gamma}  
]

charge (h).

Because

[  
1+\eta<1+\gamma,  
]

there is a logarithmic range of such (k). Each charged scale pays

[  
T_k\to\infty.  
]

So large (h) gets strong decay unless it sits outside all ranges; multiscale coverage should handle all high (h) up to (L/2), possibly using modulo symmetry for very high h.

This needs technical partition, but looks plausible.

Alternatively, rely on the local CLT theorem for this one-dimensional part. That may be cleaner.

## 7. Summation skeleton

We can now express the total minor sum schematically:

[  
\sum_{a\notin\mathfrak M}|\widehat\mu(a)|  
\le  
\sum_{\text{bad cuts }C}  
\sum_{\text{labels }h_j}  
\sum_{\text{wrong vertices }W}  
\exp\left(  
-c\left[  
E_{\mathrm{cuts}}(C)  
+  
E_{\mathrm{wrong}}(W)  
+  
E_{\mathrm{diag}}(h_j)  
\right]  
\right).  
]

Entropy factors:

- choices of cuts: at most exponential in number of levels, small;
    
- choices of component labels: (\exp(O(\sum \log B_{k_j}))), paid by (E_{\mathrm{cuts}});
    
- choices of wrong residues: (\exp(\sum_{q\in W}\log q)), paid by (E_{\mathrm{wrong}});
    
- diagonal labels (h): summed by Gaussian/local CLT.
    

For (k_0) sufficiently large, all payment constants can be made favourable.

Thus

[  
\sum_{a\notin\mathfrak M}|\widehat\mu(a)|  
=o(1/\sigma).  
]

This is not a finished proof, but the bookkeeping now has no obvious entropy leak.

## 8. Boundary and finite issues

We need handle:

1. bottom and top scale boundaries;
    
2. deleted Irving-bad primes;
    
3. prime factors of (b);
    
4. finite forbidden set from previous (a/b) repetitions.
    

All can be handled by starting at very large (k_0), adding buffer levels, and incorporating fixed primes into (L) without using them heavily as vertices. Since the construction uses arbitrarily large new primes, finite disturbances are negligible.

## 9. Main term positivity

Once minor arcs are controlled, major arc gives

# [  
\mathbb P(S=1/b)

\frac1L  
\left[  
\sum_{|h|\lesssim1/\sigma}  
\Phi(h)e(-h/b)  
+  
o(1/\sigma)  
\right].  
]

Since

[  
\mathbb E S=1/b,  
]

the phase linear term cancels at the target, and the Gaussian sum is positive:

[  
\sum_h e^{-2\pi^2h^2\sigma^2}  
\asymp  
\frac1\sigma.  
]

Thus

[  
\mathbb P(S=1/b)  
\asymp  
\frac1{\sigma L}>0.  
]

So there exists a subset.

## 10. Status after this stage

The route now has a coherent near-proof architecture:

1. Full compatible dyadic graph gives enough reciprocal mass.
    
2. Bernoulli selection centres expectation at (1/b).
    
3. Major arc local CLT gives positive main term.
    
4. Dense backbone agreement reduces large Fourier characters to piecewise diagonal.
    
5. Irving correction kills wrong vertices.
    
6. Bad cuts pay for multiple labels.
    
7. Diagonal high frequencies are controlled by Gaussian/local CLT or multiscale energy.
    

The biggest remaining technical work is rigorous implementation, not conceptual discovery.

This is a very good sign.
