## 1. 從短弧集中推出大指數和

設

# [  
R_q(d;\delta)

\left{  
p\in P:  
\left|  
\frac{d,p^{-1}}q  
\right|<\delta  
\right}.  
]

若逆元集中，譬如

[  
|R_q(d;\delta)|>(1-c)|P|,  
]

那麼 (d p^{-1}\bmod q) 有絕大部分落在以 (0) 為中心、長度約 (2\delta q) 的短弧。

這種集中會造成某個低頻 Fourier 係數很大。具體地，取一個平滑 bump 函數 (f:\mathbb R/\mathbb Z\to[0,1])，滿足：

- (f(x)\ge 1) on (|x|\le\delta)；
    
- (f) 支持在 (|x|\le 2\delta)；
    
- Fourier 展開  
    [  
    f(x)=\sum_{|r|\le R} \widehat f(r)e(rx)+\text{小誤差}.  
    ]
    

若很多 (p) 落入短弧，則

[  
\sum_{p\in P} f\left(\frac{d p^{-1}}q\right)  
]

遠大於隨機期望

[  
\widehat f(0)|P|\asymp \delta |P|.  
]

因此必有某個

[  
1\le |r|\le R  
]

使

[  
\left|  
\sum_{p\in P} e_q(r d,p^{-1})  
\right|  
\gg |P|.  
]

所以短弧集中會違反 uniform Kloosterman-over-primes 指數和界。

因此，若對某個 (q) 有

[  
\max_{a\ne0}  
\left|  
\sum_{p\in P}e_q(a p^{-1})  
\right|  
\le \varepsilon |P|,  
]

其中 (\varepsilon) 足夠小，則對所有 (d\ne0),

[  
|R_q(d;\delta)|\le (1-c)|P|.  
]

這就是我們要的短弧不集中。

## 3. Irving 平均界如何給 good (q)

我們需要的 good condition 是：

[  
\max_{a\ne0}  
\left|  
\sum_{p\in P}e_q(a p^{-1})  
\right|  
\le \varepsilon |P|.  
]

Irving 類定理給的是平均型 bound：

[  
\sum_{q\sim Q}  
\max_{(a,q)=1}  
\left|  
\sum_{\substack{p\in P\ p\text{ prime}}}  
e_q(a p^{-1})  
\right|  
\le  
\varepsilon(X,Q), |P|,|Q|,  
]

其中

[  
\varepsilon(X,Q)\to0  
]

在範圍

[  
Q^{2/3}\le X\le Q^{3/2}

內成立，或至少有 power saving。

那麼由 Markov，不滿足 good condition 的 (q) 數量至多

[  
\varepsilon(X,Q)^{1/2}|Q|

若 good threshold 取

[  
\varepsilon(X,Q)^{1/2}|P|.

因此對 (1-o(1)) 比例的 (q)，good condition 成立。

這足以。因為我們可以把 bad (q) 從 high block 中刪去，剩下的 good primes 仍佔 (1-o(1))，不影響 reciprocal mass、dense agreement 或 Fourier 主項。

## 6. Vertex correction lemma 正式化

我們可以寫成：

**Lemma D（Vertex correction）.**  
Fix (\gamma=4/3). For every sufficiently large (k), after removing (o(|P_{\gamma k}|)) primes from (P_{\gamma k}), the following holds.

Let (P=P_k) and (q\in P_{\gamma k}^{\mathrm{good}}). For every (d\in\mathbb F_q^\times),

[  
\sum_{p\in P}  
\left|  
\frac{d p^{-1}}q  
\right|^2  
\ge c|P|.  
]

Consequently, if (a_p\equiv h\pmod p) for all but (o(|P|)) primes (p\in P), and (a_q\not\equiv h\pmod q), then

[  
\sum_{p\in P}  
|\phi_{pq}(a)|^2  
\ge c'|P|.  
]

Proof sketch:

1. Irving average bound gives exponential sum small for all nonzero (a) for good (q).
    
2. Erdős–Turán gives inverse residues not concentrated in any fixed short arc.
    
3. Hence square-distance average bounded below.
    
4. Remove the small set of (p)'s not labelled by (h); the bound survives.
    
5. The perturbation (h/(pq)) is negligible.
    

This is exactly what we need.

## 7. Bad primes and graph construction

Since bad primes depend on a lower block (P_k), a prime (q) might be good relative to one neighbor block and bad relative to another.

This is manageable. Define a vertex to be retained in block (P_j) if it is good relative to all finitely many lower/upper blocks it must interact with in our backbone and correction scheme. Each block has only (O(1)) such neighboring scale pairs. Since each bad set is (o(|P_j|)), their union is still (o(|P_j|)).

Thus after pruning, each block retains

[  
(1-o(1))|P_j|  
]

good primes.

Dense agreement still works because densities remain (1-o(1)). Full graph mass changes by (1-o(1)), harmless.

## 8. Does pruning affect span?

We must ensure every retained prime appears in at least one edge and the gcd of weights remains (1).

If every retained block has many primes and the graph connects them, each prime appears in many edges. Then for every retained prime (r), there is an edge (rs), so weight (L/(rs)) is not divisible by (r), while all other primes still in (L). Hence no prime divides all weights. Span remains (1).

For primes in (b) or forbidden fixed set, handle separately: include (b\mid L) but do not need to use those primes as variable vertices. The lattice target remains integral by multiplying (L) by (b); if (L) includes extra fixed factors not in weights, one must set up the group carefully, but this is technical.

Simpler: include the prime factors of (b) among retained vertices at some harmless scale or multiply all denominators by their product in the integerization. Since (b) is squarefree and finite, this causes no asymptotic issue.

## 10. If Irving lacks uniform max?

If Irving only gives for fixed (a), not max over (a), then vertex correction becomes harder because (d=a_q-h) depends on the character.

Could use large sieve over (a), but there are (q) possible (a)'s; union bound may lose too much. Therefore max over (a) is essential or must be replaced by a strong enough second moment over (a).

So this is a critical checkpoint.
