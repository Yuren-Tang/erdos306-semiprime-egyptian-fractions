是，這個看起來正是我們要找的 Irving 文章，而且比「像」更強：你傳的 `.tex` 裏定理形式正好包含我們最關心的幾個元素。

文章題名是：

[  
\textit{Average Bounds for Kloosterman Sums Over Primes}  
]

作者 A. J. Irving。文中定義

[  
S_q(a;x)=\sum_{\substack{p\sim x\(p,q)=1}} e!\left(\frac{a\overline p}{q}\right),  
]

其中 (p) 跑過素數，(p\sim x) 即

[  
x\le p<2x.  
]

這已經是 dyadic prime interval，不需要再由 (\le x) 差分。

關鍵是 Theorem 1：

[  
\sum_{q\sim Q}\max_{(a,q)=1}|S_q(a;x)|  
\ll_\epsilon  
\left(  
Q^{5/4}x^{5/8}  
+  
Qx^{9/10}  
+  
Q^{7/6}x^{13/18}  
\right)Q^\epsilon  
]

有效範圍是

[  
Q^{3/2}\ge x\ge Q^{2/3}.  
]

這正有我們需要的：

1. **平均 over (q)**；
    
2. **(\max_{(a,q)=1})**；
    
3. **素數 (p\sim x)**；
    
4. **範圍覆蓋我們選的 (x=Q^{3/4})**。

## 對我們的尺度是否給 power saving？

我們選

[  
Q=x^{4/3},  
]

也就是

[  
x=Q^{3/4}.  
]

代入三項：

# [  
Q^{5/4}x^{5/8}

# Q^{5/4}Q^{15/32}

Q^{55/32},  
]

# [  
Qx^{9/10}

# Q\cdot Q^{27/40}

Q^{67/40},  
]

# [  
Q^{7/6}x^{13/18}

# Q^{7/6}Q^{13/24}

Q^{41/24}.  
]

而 trivial average size 是約

[  
Qx=Q^{7/4}.  
]

比較指數：

[  
\frac{7}{4}=\frac{56}{32},  
]

第一項是

[  
\frac{55}{32},  
]

已經有

[  
Q^{-1/32}  
]

的 saving；第三項也有 saving，第二項更強。取 (\epsilon>0) 足夠小，仍有 power saving。

所以它確實給出

[  
\sum_{q\sim Q}\max_{(a,q)=1}|S_q(a;x)|  
=o(Qx).  
]

更精確地，因為我們最後只在 prime (q) 上使用，(#{q\sim Q:q\text{ prime}}\asymp Q/\log Q)，power saving 足以壓過 (\log Q)。因此可推出：除了 (o(Q/\log Q)) 個 prime (q\sim Q) 外，

[  
\max_{(a,q)=1}|S_q(a;x)|\le \varepsilon_x |P_x|,  
\qquad \varepsilon_x\to0,  
]

其中

[  
|P_x|\asymp \frac{x}{\log x}.  
]

這正是我們的 **good modulus** 條件。

## 對 Lemma D 是否夠用？

夠用。流程是：

由 Theorem 1 得到對大多數 prime (q\sim Q)：

[  
\max_{a\ne0}  
\left|  
\sum_{p\sim x}e_q(a\overline p)  
\right|  
=o(|P_x|).  
]

再由 Erdős–Turán / interval Fourier 展開推出：對任意非零 (d\bmod q)，集合

[  
\left{  
p\sim x:  
\left|  
\frac{d\overline p}{q}  
\right|<\delta  
\right}  
]

不可能佔幾乎全部 (p)。換言之，存在固定 (c,\delta>0)，使

[  
#\left{  
p\sim x:  
\left|  
\frac{d\overline p}{q}  
\right|\ge\delta  
\right}  
\ge c|P_x|.  
]

因此

[  
\sum_{p\sim x}  
\left|  
\frac{d\overline p}{q}  
\right|^2  
\gg |P_x|.  
]

這就是 vertex correction lemma 的核心。

## 小結

你傳的就是我們需要的那篇，而且 Theorem 1 的形式非常合適。尤其重要的是它真的有

[  
\sum_{q\sim Q}\max_{(a,q)=1}  
]

這個最大值形式；如果沒有這個 (\max_a)，我們的 Lemma D 會困難很多。

所以現在可以把 Lemma D 標為：

[  
\boxed{  
\text{由 Irving Theorem 1 + Erdős--Turán 推出，尺度取 }x=Q^{3/4}\text{ 可用。}  
}  
]

下一步就應該把這個推論正式寫成一個 lemma。
