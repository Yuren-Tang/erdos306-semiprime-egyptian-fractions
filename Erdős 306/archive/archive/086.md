好。現在把 **Irving 推出 vertex correction** 寫成可放入 proof skeleton 的正式推論。

## Lemma D：Prime inverse non-concentration / vertex correction

取

[  
x=2^k,\qquad Q=x^{4/3}.  
]

令

[  
P_x={p:\ x\le p<2x,\ p\ \text{prime}}.  
]

則對充分大的 (x)，存在一個 prime subset

[  
\mathcal Q^{\mathrm{good}}\subset {q:\ Q\le q<2Q,\ q\ \text{prime}}  
]

使

# [  
|\mathcal Q^{\mathrm{good}}|

(1-o(1))|{q:\ Q\le q<2Q,\ q\ \text{prime}}|,  
]

並且對所有

[  
q\in\mathcal Q^{\mathrm{good}},  
\qquad d\in\mathbb F_q^\times,  
]

有

[  
\sum_{p\in P_x}  
\left|  
\frac{d\bar p}{q}  
\right|^2  
\gg |P_x|.  
]

其中 (\bar p) 表示 (p\bmod q) 的乘法逆元。

---

## 證明

Irving 定義

[  
S_q(a;x)=  
\sum_{\substack{p\sim x\(p,q)=1}}  
e\left(\frac{a\bar p}{q}\right),  
]

其中 (p\sim x) 就是

[  
x\le p<2x.  
]

Theorem 1 給出

[  
\sum_{q\sim Q}  
\max_{(a,q)=1}|S_q(a;x)|  
\ll_\epsilon  
\left(  
Q^{5/4}x^{5/8}  
+  
Qx^{9/10}  
+  
Q^{7/6}x^{13/18}  
\right)Q^\epsilon  
]

在

[  
Q^{2/3}\le x\le Q^{3/2}  
]

內成立。

現在取

[  
x=Q^{3/4}.  
]

這在範圍內。比較 trivial size：

[  
Q|P_x|\asymp Q\frac{x}{\log x}.  
]

代入 (x=Q^{3/4})，三項分別為

[  
Q^{55/32},\qquad Q^{67/40},\qquad Q^{41/24}.  
]

而

[  
Qx=Q^{7/4}=Q^{56/32}.  
]

所以 Irving 給出相對於 (Qx) 的 power saving；扣除 (\log) 因子仍有

[  
\sum_{q\sim Q}  
\max_{(a,q)=1}|S_q(a;x)|  
=o(Q|P_x|).  
]

因此由 Markov，除去 (o(Q/\log Q)) 個 prime (q\sim Q)，其餘 prime (q) 滿足

[  
\max_{a\ne0}|S_q(a;x)|  
=o(|P_x|).  
]

這些 (q) 稱為 good。

接著由 Erdős–Turán inequality，或者用平滑短弧指示函數的 Fourier 展開可得：若

[  
\max_{a\ne0}  
\left|  
\sum_{p\in P_x}e_q(a\bar p)  
\right|  
=o(|P_x|),  
]

則對任意固定小弧 (I\subset\mathbb F_q),

# [  
#{p\in P_x:\ d\bar p\in I}

\frac{|I|}{q}|P_x|+o(|P_x|)  
]

uniformly in

[  
d\in\mathbb F_q^\times.  
]

取

[  
I={r\bmod q:\ |r/q|<\delta},  
]

其中 (\delta>0) 固定且很小，便有

[  
#\left{  
p\in P_x:  
\left|  
\frac{d\bar p}{q}  
\right|<\delta  
\right}  
\le  
(2\delta+o(1))|P_x|.  
]

選 (\delta) 使 (2\delta<1/2)，例如 (\delta=1/10)，則對充分大 (x),

[  
#\left{  
p\in P_x:  
\left|  
\frac{d\bar p}{q}  
\right|\ge\delta  
\right}  
\ge c|P_x|  
]

for some fixed (c>0). 因而

[  
\sum_{p\in P_x}  
\left|  
\frac{d\bar p}{q}  
\right|^2  
\ge  
c|P_x|\delta^2  
\gg |P_x|.  
]

Lemma 證畢。

## Vertex correction corollary

現在把它翻譯回 character 能量。

假設一個 block (P_x) 已由 label (h) 控制，即對至少

[  
(1-o(1))|P_x|  
]

個 (p\in P_x) 有

[  
a_p\equiv h\pmod p.  
]

取 good prime

[  
q\sim Q=x^{4/3},  
]

若

[  
a_q\not\equiv h\pmod q,  
]

令

[  
d=a_q-h\not\equiv0\pmod q.  
]

對 labelled 的 (p)，

[  
\phi_{pq}(a)  
\equiv  
\frac{h}{pq}  
+  
\frac{d\bar p}{q}  
\pmod1.  
]

因為我們的 labels 滿足某個 cutoff

[  
|h|\le B_x=x^{1+\eta}  
]

且

[  
pq\asymp x\cdot x^{4/3}=x^{7/3},  
]

所以

[  
\left|\frac{h}{pq}\right|  
\ll x^{1+\eta-7/3}.  
]

取例如 (\eta=1/4)，則

[  
1+\eta-\frac73=\frac54-\frac73=-\frac{13}{12}<0,  
]

故此項趨於 (0)。因此它不能抵消

[  
d\bar p/q  
]

的短弧分散。

由 Lemma D 得

[  
\sum_{p\in P_x}  
|\phi_{pq}(a)|^2  
\gg |P_x|.  
]

也就是說：

[  
\boxed{  
a_q\not\equiv h\pmod q  
\quad\Longrightarrow\quad  
q\text{ 付出 incident energy }\gg |P_x|.  
}  
]

而

[  
|P_x|\asymp \frac{x}{\log x}  
\gg \log q.  
]

所以一個錯誤 vertex 的能量成本遠遠大於它的 residue entropy。

## 這一步的意義

這正式關閉了之前最大的漏洞之一：dense agreement lemma 只控制「大多數頂點」，但剩下頂點不能自由亂選；只要它連到一個 labelled good block，選錯 residue 就會被 Irving 分散立刻懲罰。

所以現在 minor arc 結構變得很強：

[  
\text{大 Fourier 係數}  
\Rightarrow  
\text{good backbone blocks}  
\Rightarrow  
\text{共同 labels}  
\Rightarrow  
\text{所有 good vertices 必服從 label}  
\Rightarrow  
\text{除非支付巨大能量}.  
]

下一步就可以把 A–D 四個 lemma 合成一個正式的 **Minor Arc Proposition**。
