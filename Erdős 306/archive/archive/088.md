好。現在處理最後大塊：**對角高頻 tail**。

我們要估計

[  
\Phi(h)=  
\prod_{pq\in E_{\mathrm{full}}}  
\left(1-\theta+\theta e\left(\frac{h}{pq}\right)\right)  
]

在

[  
h\notin[-C/\sigma,C/\sigma]  
]

時的總和：

[  
\sum_{h\notin\mathfrak M_{\mathrm{diag}}}|\Phi(h)|  
=o(1/\sigma).  
]

這是一維 weighted Bernoulli local CLT 的 aperiodicity 部分。

---

## 1. 基本能量估計

對固定

[  
0<\theta<1  
]

有

[  
|1-\theta+\theta e(t)|  
\le  
\exp(-c_\theta |t|^2).  
]

所以

[  
|\Phi(h)|  
\le  
\exp\left(  
-c\sum_{pq\in E_{\mathrm{full}}}  
\left|\frac{h}{pq}\right|^2  
\right).  
]

令

[  
E(h)=  
\sum_{pq\in E_{\mathrm{full}}}  
\left|\frac{h}{pq}\right|^2.  
]

我們需要證：

[  
\sum_{h\notin\mathfrak M_{\mathrm{diag}}}e^{-cE(h)}  
=o(1/\sigma).  
]

---

## 2. 中頻：(h) 尚未繞圈

若

[  
|h|a_{\max}\ll 1,  
]

也就是

[  
|h|\ll 2^{2k_0},  
]

則對所有邊

[  
pq\ge 2^{2k_0}  
]

有

# [  
\left|\frac{h}{pq}\right|

\frac{|h|}{pq}.  
]

此時

[  
E(h)=h^2\sum_{pq}\frac1{p^2q^2}  
\asymp h^2\sigma^2.  
]

因此

[  
|\Phi(h)|\le e^{-c h^2\sigma^2}.  
]

所以

[  
\sum_{|h|>C/\sigma,\ |h|\ll 2^{2k_0}}  
|\Phi(h)|  
\le  
\sum_{|h|>C/\sigma}  
e^{-c h^2\sigma^2}  
=o(1/\sigma)  
]

只要 (C) 大。

這一段完全標準。

---

## 3. 真正高頻：(h\gtrsim 2^{2k_0})

困難在於

[  
h/(pq)  
]

開始取模，可能對許多 (pq) 接近整數。這正是 aperiodicity 問題。

但現在我們有一個新工具：**backbone cutoff 能量**。

對角 character

[  
a_p\equiv h\pmod p  
]

在 backbone block

[  
P_k\times P_{\gamma k}  
]

上的 CRT glue 是

[  
H_{pq}^{*}=h\bmod pq  
]

的對稱代表。

若存在一段尺度使

[  
B_k<|h|<\frac12 pq  
\asymp \frac12 X_k^{1+\gamma},  
]

則在該 block 中對每條邊

[  
H_{pq}^{*}=h,  
]

所以所有邊都不是 (k)-strong-good。該 block 完全 bad，支付至少

[  
T_k\asymp \frac{X_k^{1-\gamma+2\eta}}{k^2}.  
]

因此對角高頻 (h) 若落入某些尺度窗口，就受到巨大衰減。

---

## 4. 尺度窗口覆蓋

取

[  
\gamma=\frac43,\qquad \eta=\frac14.  
]

則

[  
B_k=X_k^{5/4},  
]

而 backbone edge product scale 是

[  
pq\asymp X_k^{1+\gamma}=X_k^{7/3}.  
]

所以 level (k) 會懲罰 (h) 若

[  
X_k^{5/4}<|h|<cX_k^{7/3}.  
]

對數尺度上，窗口為

[  
\frac54 k<\log_2 |h|<\frac73 k.  
]

這些窗口隨 (k) 移動。因為

[  
\frac73>\frac54,  
]

窗口很寬，而且相鄰 (k) 的窗口大量重疊。從最低高頻開始，到非常高的 (h)，幾乎都能被某個 (k) 捕捉，只要 (h) 不接近模 (pq) 的倍數。

但注意：條件

[  
|h|<pq/2  
]

只處理 (h) 小於該 block modulus product 的情形。若 (h) 更大，(h\bmod pq) 可能又小，不能用這個簡單窗口。

所以這能處理從

[  
h\sim X_{k_0}^{5/4}  
]

到某些較高尺度 product 之前的範圍，但不是全部 (h\bmod L)。

---

## 5. 高頻模回繞問題

因為 (h) 是 modulo (L) 的頻率，最高可到

[  
L/2,  
]

非常巨大。對某個 block product (pq)，即使 (h) 很大，也可能

[  
h\equiv 0\pmod{pq}  
]

或接近，造成該 block 不衰減。

但若 (h) 對許多 (pq) 都接近 (0\pmod{pq})，那意味 (h) 被大量 semiprimes 近整除。這應該非常罕見，且與我們早先的 matching/CRT counting 問題相似。

可是現在對角高頻只是一維 (h)，也許可用更初等的 divisibility argument。

## 6. Divisibility support argument

令

[  
S(h)={p\in P: p\mid h}.  
]

如果 (pq\mid h)，則 edge (pq) gives zero phase. If either p or q does not divide h, phase may not be zero. But (|h/(pq)|) can be small also if h is close to multiple of pq, not necessarily divisible.

先看 exact divisibility。若 (h) 被很多 semiprimes (pq) 整除，則它必含有很多 vertex primes。若 (h) 含有 (r) 個 prime factors from (P)，它 is divisible by at most (\binom r2) candidate semiprimes. To kill a positive fraction of complete block pairs, (r) must be positive fraction of vertices, then (h) divisible by product of many primes, forcing (|h|) enormous. But (h<L/2) can still be divisible by many but not all primes. Count such h via entropy vs divisibility.

This resembles previous energy-entropy but one-dimensional and easier.

However, approximate multiples complicate.

## 7. 可以用 full graph span/aperiodicity theorem

At this point, rather than proving diagonal high-frequency from scratch, it is legitimate to invoke a weighted lattice Bernoulli local CLT if its hypotheses include:

[  
\sup_{|t|\in[\epsilon/\sigma,\pi]}  
\prod_i |1-\theta+\theta e(t w_i)|  
=o(1/\sigma)  
]

or similar.

But such theorem usually requires aperiodicity condition, not merely span 1 and max weight (o(\sigma)). Span 1 alone is not enough in triangular arrays: weights could all be even except one tiny variance weight.

Our weights are rich enough, but one must prove it. The diagonal high-frequency is exactly that proof.

Maybe our backbone/full graph agreement can serve as aperiodicity proof for diagonal too, but we need make it rigorous.

## 8. Use vertex correction for diagonal high h?

For diagonal (h), every vertex has residue

[  
a_p=h\bmod p.  
]

If (h) is not small as an ordinary integer, can it be piecewise diagonal with small labels? On each scale component, the label is (h\bmod) product? Dense agreement with cutoff (B_k) says: if block has many strong-good edges, there is a small integer label (h_k) with residues (h_k\bmod p). For diagonal (h), this means

[  
h\equiv h_k\pmod p  
]

for many p in block. Thus

[  
h-h_k  
]

is divisible by product of many primes. If (|h|) is not astronomically large relative to that product, this forces

[  
h=h_k  
]

as integer.

Therefore for many ranges of (h), if dense agreement thinks block is good, then (h) must equal a small integer. If not, block is bad and pays energy.

This is promising for all h: even if (h) wraps modulo (pq), being strong-good on a dense block implies existence of small label (h_k) agreeing with (h) mod many primes, so (h) is constrained to a residue class modulo a huge product. There are few such h.

Thus diagonal high-frequency can be controlled by same dense agreement plus counting, not by direct windows only.

## 9. Diagonal entropy-energy version

For diagonal (h), suppose many backbone blocks are agreement-good. Then each good component has a small label (h_C). Since the character is diagonal, for all p in component,

[  
h\equiv h_C\pmod p.  
]

Therefore

[  
h\equiv h_C\pmod{M_C},  
]

where

[  
M_C=\prod_{p\in C'}p  
]

over many primes in the component. Thus possible h modulo L are confined to

[  
h=h_C \mod M_C.  
]

If there is one large component containing most primes, then (M_C) is enormous, and within ([0,L)) there are (L/M_C) possibilities. But if component contains most primes, (L/M_C) is product of few exceptional primes, whose entropy is paid by cuts/wrong vertices.

So diagonal high h is also included in the general piecewise-diagonal counting. The major arc corresponds to the case (h=h_C) as small integer, not just residue.

However, for Fourier sum over h modulo L, congruence classes modulo huge (M_C) may still contain many h differing by multiples of (M_C). Do those produce same character on component but arbitrary on complement; complement entropy paid. Their Fourier energy on full graph to complement/cuts should pay. Thus consistent.

This suggests our Minor Arc Proposition already covers diagonal high h if we formulate major arc as the only case with no paid cuts and small integer label. If global component includes all primes and label (h_0) small, then diagonal character h modulo all p equals (h_0), so h≡h0 mod L. Since h modulo L, this is exactly h0. Thus no high h remains. Wait: if all vertices labelled by h0, then diagonal h must satisfy h≡h0 mod every p in P, hence h≡h0 mod L_P. In frequency group modulo L_P, h equals h0. So high diagonal h cannot be globally good with small label. It must have bad cuts or wrong vertices. Great.

Thus diagonal high-frequency is actually killed by the same agreement proposition, except ordinary small h major arc.

This is a key unification.

## 10. Reinterpretation

We should not separately sum diagonal high h. The full minor arc proposition says:

If a character is not one of the small diagonal labels (h) with (|h|\le C/\sigma), then either:

- some backbone blocks bad;
    
- some vertices wrong relative to labels;
    
- or global label (h_0) exists but (|h_0|>C/\sigma).
    

The last case is diagonal large ordinary integer h0. It has energy (h_0^2\sigma^2) and Gaussian decay. So only ordinary large h remains, not modular wrap high h. Modular high h would not be globally small-labelled; hence paid by agreement.

This resolves high-frequency wrap issue.

## 11. Major arc set should be labels, not raw h modulo L

Define major arc characters as:

[  
a_p\equiv h\pmod p\quad\forall p,  
]

for an integer representative

[  
|h|\le C/\sigma.  
]

Any other character, including diagonal raw frequency h modulo L not represented by small integer, is minor and controlled by agreement/cuts/wrong vertices.

Indeed if h modulo L is large but congruent to small h0 mod all p, then h=h0 mod L, same character. So no ambiguity.

## 12. Minor arc proposition absorbs aperiodicity

Therefore the separate one-dimensional high-frequency local CLT may not be needed. The agreement proposition gives aperiodicity for all non-major characters, including diagonal high residues.

For ordinary diagonal labels with (|h|>C/\sigma) but still represented as integer h, they are minor but globally labelled. They are not cut/wrong. Their Fourier decay is Gaussian:

[  
|\Phi(h)|\le e^{-c h^2\sigma^2}  
]

at least while (|h|a_{\max}\ll1). If (|h|) exceeds that, then it is not small relative to low block cutoffs and becomes bad under agreement. Thus covered.

So diagonal tail splits:

- (C/\sigma<|h|\ll a_{\max}^{-1}): Gaussian.
    
- (|h|\gtrsim a_{\max}^{-1}): agreement bad-block energy.
    

Good.

Thus minor arcs can be controlled by combined scheme:

2. Characters not globally small-labelled: agreement/correction energy.
    

This should give

Then diagonal high-frequency is no longer a separate scary external local CLT aperiodicity problem. We only need ordinary Gaussian analysis in the no-wrap range; beyond that agreement catches it.
