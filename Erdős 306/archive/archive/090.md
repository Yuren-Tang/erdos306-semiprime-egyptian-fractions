# Energy–Entropy Bookkeeping Lemma

我們有一組尺度 levels

[  
\mathcal K={k_0,k_1,k_2,\dots,k_m},  
\qquad  
k_{j+1}\approx \gamma k_j,  
]

例如

[  
\gamma=\frac43.  
]

每個 level (k) 有 block

[  
P_k.  
]

對每條 backbone link

[  
k\to \gamma k  
]

有一個 cut cost

[  
T_k\asymp \frac{X_k^{1-\gamma+2\eta}}{k^2}.  
]

取

[  
\gamma=\frac43,\qquad \eta=\frac14,  
]

則

[  
T_k\asymp \frac{X_k^{1/6}}{k^2}.  
]

因此

[  
T_k\gg k  
]

for (k\ge k_0) if (k_0) sufficiently large.

每個 vertex (q\in P_j) 若違背其 component label，則有 wrong-vertex cost

[  
E_q\gg |P_i|  
]

for some adjacent lower block (P_i). In particular,

[  
E_q\gg \log q.  
]

現在要證：若每個新增自由度都由這些 costs 支付，則所有 minor structures 的 weighted count 可控。

---

## 1. Bad cuts 的求和

令

[  
\mathcal B\subset\mathcal K  
]

為 bad cuts 集合。它們的能量成本至少

# [  
E_{\mathrm{cut}}(\mathcal B)

\sum_{k\in\mathcal B}T_k.  
]

選擇 bad cuts 本身的 combinatorial entropy 至多

[  
2^{|\mathcal K|},  
]

很小。更精細地，對每個 bad cut (k)，它最多新增一個 component label，而該 label 的可能數量至多

[  
O(B_{\gamma k})  
]

或粗略

[  
\exp(O(k)).  
]

所以 bad cut (k) 帶來的 entropy factor 至多

[  
\exp(C_1 k).  
]

因此 bad cuts 的總貢獻由乘積估計：

## [  
\sum_{\mathcal B}  
\exp\left(  
C_1\sum_{k\in\mathcal B}k

# c_1\sum_{k\in\mathcal B}T_k  
\right)

\prod_{k\in\mathcal K}  
\left(  
1+  
\exp(C_1k-c_1T_k)  
\right).  
]

因為

[  
T_k\gg k,  
]

取 (k_0) 大，使

[  
c_1T_k\ge 2C_1k+10k  
]

則每項

[  
\exp(C_1k-c_1T_k)  
\le e^{-10k}.  
]

所以乘積收斂且趨於

[  
1+o(1)  
]

as (k_0\to\infty).

結論：bad cuts 的 entropy 被 energy 完全支付。

---

## 2. Component labels 的求和

若沒有 bad cuts，只有一個 component label (h)。  
若有 bad cuts，components 數量

[  
r\le |\mathcal B|+1.  
]

每個 component label (h_C) 由其最低 adjacent cutoff 控制，粗略有

[  
|h_C|\le \exp(Ck_C).  
]

所以 label entropy 已在上一節按 bad cuts 支付。剩下一個 global component label，即無 bad cuts 時的 label (h)，不能由 cut energy 支付；它正是 diagonal family，需要單獨處理：

- 若 (|h|\le C/\sigma)，它是 major arc；
    
- 若 (|h|>C/\sigma)，Gaussian diagonal energy
    
    [  
    \gg h^2\sigma^2  
    ]
    
    支付它。
    

對 global label 的 tail：

[  
\sum_{|h|>C/\sigma}e^{-c h^2\sigma^2}  
=o(1/\sigma)  
]

取 (C) 大即可。

所以 component labels 也可控。

---

## 3. Wrong vertices 的求和

令 (W) 為 wrong vertices 集合。每個

[  
q\in W  
]

有 residue choices 至多

[  
q  
]

個，entropy factor

[  
\le e^{\log q}.  
]

但 vertex correction gives energy cost

[  
E_q\ge A\log q  
]

where (A\to\infty) as (k_0\to\infty), because actually

[  
E_q\gg \frac{x}{\log x}  
]

while

[  
\log q\asymp \log x.  
]

因此 wrong vertices contribution is bounded by

## [  
\sum_{W}  
\exp\left(  
\sum_{q\in W}\log q

# c_2\sum_{q\in W}E_q  
\right)

\prod_{q}  
\left(  
1+\exp(\log q-c_2E_q)  
\right).  
]

For (k_0) large,

[  
c_2E_q\ge 3\log q + 10\log q  
]

say, so

[  
\exp(\log q-c_2E_q)\le q^{-10}.  
]

Then

[  
\prod_q(1+q^{-10})=1+o(1)  
]

over the large-prime range.

Thus wrong vertices are harmless.

---

## 4. Overlap issue

We used costs additively:

[  
\sum_{q\in W}E_q.  
]

But incident energy neighborhoods for different wrong vertices may overlap. Need avoid double-counting.

There are two ways:

### Method A: assign disjoint correction blocks

For each scale (j), wrong vertices in (P_j) are corrected using one fixed adjacent labelled lower block (P_{j/\gamma}). The incident edge sets for different (q\in P_j) are disjoint because they use different high vertex (q) but edges ((p,q)) are distinct. So no overlap within same scale.

Across scales, choose correction directions consistently; each edge belongs to at most (O(1)) correction families because scale graph degree is bounded. Thus total double-counting is bounded by an absolute constant.

### Method B: divide energy by overlap constant

If each full graph edge appears in at most (D) assigned correction sums, then

[  
Q_{\mathrm{full}}(a)  
\ge  
\frac1D\sum_{q\in W}E_q.  
]

Since (D=O(1)), constants absorb.

Thus overlap does not kill the argument.

---

## 5. Bad cut and wrong vertex interaction

A wrong vertex near a bad cut might lack a labelled neighbor for correction. Then it belongs to a component boundary. But if the cut is bad, its entropy has already been paid by cut energy, and the component may get its own label. Within each component, vertices are corrected using internal good links. Boundary issues affect only (O(1)) adjacent levels per bad cut, and their entropy can be charged to the same (T_k), which is huge.

So the bookkeeping can define components after removing bad cuts, then perform vertex correction only inside components; boundary leftovers are absorbed by cut costs.

---

## 6. Final bookkeeping inequality

Combining:

[  
\sum_{a\notin\mathfrak M}  
|\widehat\mu(a)|  
\le  
\left[  
\sum_{\text{global labels }h\notin[-C/\sigma,C/\sigma]}  
e^{-c h^2\sigma^2}  
\right]  
\cdot(1+o(1))  
+  
o(1/\sigma).  
]

The bracket is

[  
o(1/\sigma)  
]

for (C) large.

Thus

[  
\sum_{a\notin\mathfrak M}|\widehat\mu(a)|  
=o(1/\sigma).  
]

This is the desired minor arc bound.

---

## 7. What remains in a fully written proof?

This bookkeeping lemma is now conceptually closed, but a formal proof must specify:

1. exact scale set (\mathcal K) and buffer levels;
    
2. exact definition of bad cut;
    
3. constants in dense agreement lemma;
    
4. assignment of correction edges with bounded overlap;
    
5. treatment of pruned Irving-bad primes;
    
6. final inequality with (k_0) chosen after all constants.
    

None of these looks like a new mathematical obstruction. They are technical.
