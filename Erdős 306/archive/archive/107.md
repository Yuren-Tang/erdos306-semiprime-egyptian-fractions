# 1. 二部 bounded-energy agreement：正確目標

令

[  
P\subset[X,2X],\qquad Q\subset[Y,2Y],\qquad Y\ge X.  
]

給 residues

[  
a_p\in\mathbb F_p,\qquad b_q\in\mathbb F_q.  
]

定義

# [  
Q_{P,Q}(a,b)

\sum_{p\in P,q\in Q}  
\left(\frac{H_{pq}^{*}}{pq}\right)^2.  
]

我們希望證明一個 counting 型命題：

[  
#{(a,b):Q_{P,Q}(a,b)\le R}  
]

不是接近

[  
\prod_{p\in P}p\prod_{q\in Q}q,  
]

而是接近「一個 label (h)」的數量，再乘上能量可支付的誤差。

也就是：

[  
#{(a,b):Q_{P,Q}\le R}  
\lesssim  
#{h:Q_{P,Q}^{\mathrm{diag}}(h)\lesssim R}  
\cdot e^{\varepsilon R}  
]

至少在足夠大的 (X) 後成立。

# 2. 用能量決定 label 尺度

設

[  
\overline e=\frac{R}{|P||Q|}.  
]

定義自然 cutoff

# [  
B_R

# CXY\sqrt{\overline e}

CXY\sqrt{\frac{R}{|P||Q|}}.  
]

若一個 assignment 有能量 (\le R)，則由 Markov，除了很小比例的邊外，

[  
|H_{pq}^{*}|\le B_R.  
]

若

[  
B_R<Y/2  
]

且

[  
B_R<X^2/2,  
]

則 dense agreement + list-collapse 給出一個 ordinary label

[  
|h|\le B_R  
]

解釋大多數 (P\cup Q)。

於是二部 assignments 可按 (h) 計數，而

[  
|h|\le B_R  
]

正好等價於 diagonal energy 不超過 (R) 的量級：

[  
Q_{P,Q}^{\mathrm{diag}}(h)  
\asymp  
|P||Q|\left(\frac{h}{XY}\right)^2  
\lesssim R.  
]

這部分完美。

# 3. 當 (B_R) 超過比較尺度時，該怎麼辦？

若

[  
B_R\ge Y/2  
]

或

[  
B_R\ge X^2/2,  
]

則普通 bounded label collapse 不適用。以前我說「能量大，付費」；但這不夠精確。正確處理是：

[  
\boxed{  
\text{此時本二部 pair 已經不能單獨把 label 壓成小 ordinary integer；必須把 label 傳給更高尺度比較。}  
}  
]

也就是說，二部 lemma 本身不應要求在所有 (R) 下給 ordinary (h)。它只應在 **visible range** 內給 label。

Visible range 是：

[  
B_R<\min(Y/2,X^2/2).  
]

超出 visible range 時，該 pair 提供的資訊是：

[  
\text{若能量 }\le R,\text{則 residues 可能來自某個較大 }h,  
]

但這個 (h) 需要更高尺度的 primes 才能唯一識別。

所以這不是二部層面能完全解決的；必須由三層或多層解決。

# 4. 三層為何剛好補上 invisible range？

對 (P_0\sim X), (P_1\sim Y=X^{4/3}), (P_2\sim Z=Y^{4/3})。

若 (P_0P_1) 的 natural label scale (B_{01}) 超過 (Y)，它對中層 (P_1) 看不清。但 (P_1P_2) 有高層 (Z)，可以看清更大的 label。

可見尺度逐層上升：

[  
Y,\quad Z,\quad Z^{4/3},\dots  
]

所以任何 label (h) 只要不是太接近全局 modulus (L)，總會在某個高尺度變成 visible。

這就是 multiscale 的核心：**大 label 不是沒有 label，只是要到高層才看見。**

# 5. 因此二部 lemma 的正確角色

二部 bounded-energy agreement 應該寫成：

**Visible-range agreement lemma.**  
若

[  
Q_{P,Q}\le R  
]

且

[  
XY\sqrt{R/(|P||Q|)}  
<  
c\min(Y,X^2),  
]

則存在 ordinary label (h) 解釋大多數 vertices，並且

[  
Q_{P,Q}^{\mathrm{diag}}(h)\lesssim R.  
]

若 visible condition 不成立，不得宣稱 raw entropy 被支付；只把它交給更高尺度。

這是準確的。

# 6. 三層 counting lemma 的修正版

三層 counting lemma也應避免「兩個 pair 都 invisible 就直接付 raw entropy」。應改為：

若 (P_0P_1) invisible，則使用 (P_1P_2) 來 label (P_1)，再由 (P_1) 的 label 透過 Irving correction / edge energy 控制 (P_0)。

但如果 (P_1P_2) 也 invisible，那就繼續往 (P_3)。

所以真正的 local unit 可能不是三層，而是「足夠長的 scale tower」，直到 label scale 變 visible。

這表示三層 lemma仍可能不足以處理所有 (R)。

# 8. 熵求和中 (R) 可以多大？

我們需要 summation over all (R), but for (R) as large as a power of (X), (e^{-cR}) is astronomically small. It can pay raw entropy if

[  
R\gg X  
]

但 not if (R\sim X^{1/3}).

Problematic range:

[  
1\ll R\ll X  
]

where Fourier decay not enough to pay raw block entropy, but (B_R) may become invisible.

Find threshold for visibility:

[  
B_R<Y  
]

means

[  
XY\sqrt{\frac{R}{XY/k^2}}<Y  
]

i.e.

[  
k\sqrt{RXY}<Y.  
]

So

# [  
R<\frac{Y^2}{k^2XY}

# \frac{Y}{k^2X}

\frac{X^{1/3}}{k^2}.  
]

Thus visible range for (P_0P_1):

[  
R\ll X^{1/3}/k^2.  
]

This is exactly earlier energetic threshold.

If (R) exceeds (X^{1/3}), the pair becomes invisible.

But (R=X^{1/3}) is still much less than raw entropy (X). So we need higher scales for intermediate energies. Good.

At next pair (P_1P_2), visible threshold is

# [  
\frac{Z}{k^2Y}

# \frac{Y^{1/3}}{k^2}

\frac{X^{4/9}}{k^2}.  
]

Slightly larger.

After (t) levels, visibility threshold grows as

[  
X^{(1/3)\gamma^t}.  
]

Eventually it exceeds any given (R\ll X) after (O(\log\log X)) levels. Since full scale range can be long, this is feasible.

This is the multiscale tower mechanism quantitatively.

# 10. New formulation: energy-dependent tower

Given (R), choose (j=j(R)) minimal such that

[  
R\ll X_j^{1/3}/k_j^2.

At scale pair (P_jP_{j+1}), visible agreement applies. This gives a label (h) on high scales. Then use overlapping visible agreements to propagate label down to lower scales? Need check: lower pairs may be invisible at energy (R). But once high label (h) is known, lower vertices that disagree with it are corrected by Irving against labelled higher/lower blocks. We do not need lower pair visibility.

Thus high visible scale can anchor the global label; Irving correction pulls lower levels into alignment.

This is excellent.

# 11. How to anchor all levels?

Take a tower:

[  
P_0,P_1,\dots,P_J  
]

up to (J=J(R)). At top pair (P_{J-1}P_J), visible agreement gives label (h). Then descend:

Given (P_j) labelled by (h), any vertex in (P_{j-1}) not congruent to (h) is corrected using edges to (P_j), by Irving. Thus (P_{j-1}) becomes labelled or pays vertex energy.

This avoids needing (P_{j-1}P_j) visible.

This is a major simplification.

# 13. Energy-layer proof architecture

For each energy layer (R):

1. Pick top scale (J(R)) such that visible agreement holds at (P_{J-1}P_J).
    
2. Low-energy assignment restricted to top pair has label (h), unless top pair energy already (>R).
    
3. Use Irving correction to force all lower levels to obey (h), with wrong vertices paid by energy.
    
4. Thus low-energy assignments in layer (R) are counted by global labels (h) plus (e^{\varepsilon R}).
    

This seems to solve the intermediate energy invisibility problem.

This resolves the entropy leak much more cleanly.
