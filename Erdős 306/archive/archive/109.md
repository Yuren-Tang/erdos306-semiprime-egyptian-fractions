# Energy-layer anchoring proposition

取尺度塔

[  
P_0,P_1,\dots,P_K,  
]

其中

[  
P_j\subset[X_j,2X_j],  
\qquad  
X_{j+1}=X_j^\gamma,  
\qquad  
\gamma=\frac43.  
]

假設每一層已做 Irving-good pruning。令 full graph 至少包含相鄰層完整二部邊

[  
P_j\times P_{j+1}.  
]

對 character (a)，設能量

# [  
Q(a)

\sum_{pq\in E_{\mathrm{full}}}  
|\phi_{pq}(a)|^2.  
]

命題目標：

> 對任意能量層 (R)，若 (Q(a)\le R)，則存在一個整數 label (h)，使所有層的大多數 vertices 滿足  
> [  
> a_p\equiv h\pmod p,  
> ]  
> 例外 vertices 的資料可由 (e^{\varepsilon R}) 支付，且  
> [  
> Q_{\mathrm{diag}}(h)\lesssim R.  
> ]

這樣即可推出

[  
#{a:Q(a)\le R}  
\le  
e^{\varepsilon R}  
#{h:Q_{\mathrm{diag}}(h)\le C R}.  
]

---

## 1. 可見閾值

對相鄰 pair

[  
P_{j-1}\times P_j  
]

若總能量 (\le R)，則此 pair 的局部能量也 (\le R)。Adaptive cutoff 為

[  
B_R(j)  
\asymp  
X_{j-1}X_j  
\sqrt{  
\frac{R}{|P_{j-1}||P_j|}  
}.  
]

由

[  
|P_{j-1}||P_j|  
\asymp  
\frac{X_{j-1}X_j}{k_j^2},  
]

得

[  
B_R(j)  
\asymp  
k_j\sqrt{R X_{j-1}X_j}.  
]

要讓 ordinary label 在高層 (P_j) 可見，需要

[  
B_R(j)<cX_j.  
]

等價於

[  
R< c\frac{X_j}{k_j^2X_{j-1}}.  
]

因為

[  
X_j=X_{j-1}^{4/3},  
]

所以

[  
\frac{X_j}{X_{j-1}}=X_{j-1}^{1/3}.  
]

可見閾值為

[  
V_j  
\asymp  
\frac{X_{j-1}^{1/3}}{k_j^2}.  
]

隨 (j) 快速增大。

---

## 2. 對給定 (R) 選錨點

給定 (R)，選最小 (J=J(R)) 使

# [  
R\le c V_J

c\frac{X_{J-1}^{1/3}}{k_J^2}.  
]

只要 (K) 足夠大，對所有 relevant (R) 都能找到這樣的 (J\le K)。

對任何滿足 (Q(a)\le R) 的 assignment，pair

[  
P_{J-1}\times P_J  
]

的局部能量 (\le R)，且 (R\le cV_J)，所以 adaptive dense agreement 可見，得到 ordinary label

[  
h  
]

解釋大多數 (P_{J-1}\cup P_J)。

而且

[  
|h|  
\ll  
B_R(J)  
\asymp  
X_{J-1}X_J  
\sqrt{\frac{R}{|P_{J-1}||P_J|}}.  
]

---

## 3. 錨點 label 的 diagonal energy

在 anchor pair 上，diagonal energy of (h) 是

[  
Q^{\mathrm{diag}}_{J-1,J}(h)  
\asymp  
|P_{J-1}||P_J|  
\left(\frac{h}{X_{J-1}X_J}\right)^2.  
]

由上式對 (h) 的 bound，立刻有

[  
Q^{\mathrm{diag}}_{J-1,J}(h)  
\lesssim R.  
]

所以 label (h) 的大小已被 (R) 控制。

對 full graph 而言，

[  
Q_{\mathrm{diag}}(h)  
]

可能比 anchor pair 上更大，因為 full graph 有更多邊，尤其低層邊權重更大。這是個關鍵點。

若 (h) 在 anchor scale 上 energy (\le R)，它在低層可能 energy 很大。  
但如果低層 vertices 也被 forced to follow (h)，那 full graph energy of the actual character includes that larger diagonal energy; since (Q(a)\le R), automatically

[  
Q_{\mathrm{diag,full}}(h)\le R+(\text{exception costs})  
]

should follow. Need justify.

Reason: once all vertices follow (h), edges in full graph have phases (h/(pq)), so their contribution equals diagonal energy. If full diagonal energy were (>R), the assignment could not have total energy (\le R). Thus after correction, we get

[  
Q_{\mathrm{diag,full}}(h)\lesssim R.  
]

Exceptions are paid separately.

So label count uses full diagonal energy, as desired.

---

## 4. 從錨點向下校正

已知 (P_J) 和 (P_{J-1}) 大多數服從 (h)。

現在考慮 (P_{J-2})。若

[  
q\in P_{J-2}  
]

不服從 (h)，則用 labelled block (P_{J-1}) 做 Irving correction：

[  
\sum_{p\in P_{J-1}}  
|\phi_{pq}(a)|^2  
\gg |P_{J-1}|.  
]

因為這些邊在 full graph 中存在，錯誤 vertex 付出巨大能量。

因此在能量層 (R) 中，錯誤 (P_{J-2})-vertices 的總 entropy 可由其能量支付。其餘 (P_{J-2}) 服從 (h).

再用 (P_{J-2}) 或 (P_{J-1}) 作 labelled block，繼續往下。

實際上往下第 (m) 步，用 (P_{j+1}) 校正 (P_j). 因為相鄰尺度比例 (4/3)，Irving 條件始終成立。

---

## 5. 從錨點向上校正

同樣，對 (P_{J+1})，若

[  
q\in P_{J+1}  
]

不服從 (h)，用 labelled block (P_J) 做 correction。尺度 (P_{J+1}=P_J^{4/3})，仍在 Irving 範圍內。

於是 label 從錨點向上傳播到 (P_K).

所以整座塔中，所有 good vertices either 服從 (h)，or pay correction energy.

## 6. 例外資料計數

每個 wrong vertex (v) 的 residue choices 至多 (v)，集合選擇 entropy 也至多 (O(\log v)) per vertex. 但 correction energy

[  
E_v\gg |P_{\mathrm{adj}}|  
]

且

[  
|P_{\mathrm{adj}}|\gg_\varepsilon \varepsilon^{-1}\log v  
]

for (k_0) sufficiently large.

因此對任意預先給定的 (\varepsilon>0),

[  
#{\text{wrong vertex data with total correction energy}\le R}  
\le e^{\varepsilon R}.  
]

這是關鍵：例外不會漏熵。

## 7. 錨點 label 的選擇數

對於每個 (a) with (Q(a)\le R)，我們得到一個 label (h) satisfying

[  
Q_{\mathrm{diag,full}}(h)\le C R  
]

after absorbing exceptions.

因此 possible labels 數至多

[  
#{h:Q_{\mathrm{diag,full}}(h)\le C R}.  
]

所以

[  
#{a:Q(a)\le R}  
\le  
e^{\varepsilon R}  
#{h:Q_{\mathrm{diag,full}}(h)\le C R}.  
]

這就是我們要的 counting inequality。

# 8. 剩下的兩個技術條件

### 條件 A：錨點必須存在

需要固定 (K) 使對所有 relevant (R) 有

[  
R\le V_K.  
]

Relevant (R) 可以取到總 character entropy 級別：

# [  
\log |\widehat G|

\sum_{p\in P}\log p  
\asymp  
\sum_{j\le K} X_j.  
]

這非常大，可能比 (V_K) 大很多，因為

[  
V_K\asymp X_{K-1}^{1/3}.  
]

所以不可能讓 (V_K) 超過全體 raw entropy。

但我們不需要處理如此大的 (R) 精細：當 (R) 超過某個 (R_{\mathrm{large}})，(e^{-cR}) 可直接壓過全部 characters.

需要選 (R_{\mathrm{large}}) 使

[  
e^{-cR_{\mathrm{large}}}|\widehat G|\ll 1/\sigma.  
]

這要求

[  
R_{\mathrm{large}}\gg \log|\widehat G|.  
]

若 (\log|\widehat G|\asymp X_K), 那仍很大，錨點不可見。

所以直接 raw bound 太差。

But we can choose not to include character variables up to extremely high K? Yet K needed for mass (W\asymp\log(K/k_0)). K may be exponentially large in (1/b), but finite. (\log|\widehat G|) dominated by (X_K), enormous. Raw bound impossible.

Thus for high (R), we still need structure; but high (R) could be much less than (X_K). Visibility at top (V_K\sim X_K^{1/(3\gamma)}) maybe still far below (X_K). There is an uncovered range (V_K\ll R\ll X_K).

This is a problem.

### 條件 B：top visibility threshold grows only as (X^{1/3})

To cover energies up to raw entropy of top block, one would need an even higher anchor beyond top. This suggests adding a buffer tower above all active blocks. For every active top, include much higher blocks solely for anchoring.

If include buffer up to (X_{K'}), raw entropy grows again. But active representation mass may only use lower blocks; buffer blocks can be used for Fourier control with very small reciprocal mass? They still add character entropy if included in L.

Maybe do not include buffer primes as variables? But correction edges require their residues/characters. If included in modulus, yes entropy.

This becomes recursive.

Maybe we should not use a tower with rapidly increasing (X_j) up to huge (K). Original dyadic full graph used all levels (k_0\le k\le K) linearly, not tower (X_{j+1}=X_j^{4/3}). # 10. Where does this leave us?

Energy-layer anchoring solves low and intermediate R up to visibility threshold of some available higher scale. But because every added scale brings new entropy faster than visibility threshold, it cannot by itself cover all R using a finite top.

Thus still need a local high-energy counting argument for top/buffer levels, or an infinite limiting construction with compactness? For finite proof, need close top boundary.

Possible fix: choose top levels not included in target group, use them only as auxiliary random denominators? But any denominator using top prime includes it in L and character group.

Alternative: construct representation in an infinite probabilistic model and use limiting? Hard.

More likely: the original global energy–entropy theorem must be local at every scale and does not rely on a top anchor. Agreement/list-agreement tools handle this without top boundary issue.

So anchoring is useful but not complete.

# 11. Honest assessment

We have made progress but found another real boundary problem. The top entropy issue shows that a simple anchor-tower proof is insufficient.

The correct final proof probably requires a scale-local coercivity/list-agreement theorem that works uniformly at every scale, including top, without needing a higher anchor. Or one must add boundary conditions/lattice-fixing labels at top.

Could fix by imposing top block residues fixed? But then denominators involving top primes lose randomness/mass? Maybe top primes could be used only as second factors with fixed residues? Not straightforward.

# 12. Best next step

We should not keep patching with towers. The robust path is to formulate a **finite-scale weighted CRT list-agreement theorem with boundary-free conclusion**:

For a dense compatible graph on a finite scale interval, low energy assignments are close to a global diagonal (h\bmod L), and entropy is controlled, with no top anchor.

This is exactly an agreement theorem. The top issue is precisely why local-to-global agreement is nontrivial.
