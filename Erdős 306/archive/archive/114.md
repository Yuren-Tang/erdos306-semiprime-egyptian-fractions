是，這裏確實是硬骨頭，而且是整條路線目前最硬的一塊。

不是因爲前面錯了；相反，前面的分析把問題越壓越準，最後壓到這個核心：

[  
\boxed{  
\text{若一個 character 在大量 }pq\text{ 邊上相位很小，是否必然接近小對角 }h？  
}  
]

這就是 **CRT Poincaré / aperiodicity**。它同時包含兩種困難：

一是非對角 character：不同 (a_p) 之間是否能局部相容卻不來自同一 (h)？

二是對角高頻：(a_p\equiv h\pmod p)，但 (h) 是大頻率時，是否能讓很多 (|h/(pq)|) 同時很小？

這兩者其實是同一個問題：大量 CRT glue 小，是否強迫一個小全局整數 label。

---

# 1. 先降成 small-glue density theorem

我們先不要直接證完整能量版。先證一個密度版：

**Small-glue density theorem, desired form.**  
令 (E) 是 compatible dyadic graph。給 residues (a_p\in\mathbb F_p)。若對某個尺度 (B_{pq}) 或統一 (B)，有正比例甚至絕大多數邊 (pq) 滿足

[  
|H_{pq}^{*}|\le B,  
]

其中 (H_{pq}^{*}) 是 CRT glue 的對稱代表，則存在整數 (h)，(|h|\le B')，使大多數頂點滿足

[  
a_p\equiv h\pmod p.  
]

這是 bounded dense agreement 的全圖版本。

在一個完整二部 block pair 中，我們已經能證。現在要做的是把它跨多個 blocks 推廣。

---

# 2. 一個 block pair 已經成立

對

[  
P\subset[X,2X],\qquad Q\subset[Y,2Y],  
]

若

[  
B<Y/2,\qquad X^2>2B,  
]

且

[  
|H_{pq}^{*}|\le B  
]

在 (P\times Q) 中佔 (1-\varepsilon)，則存在

[  
|h|\le B  
]

使大多數 (P,Q) 頂點服從 (h)。

這一點沒問題。

---

# 3. 全圖問題：如何選一個看得見 (B) 的 block pair？

若 (B) 固定，而圖中有足夠高的 blocks，使

[  
\text{高層 prime }Y>B  
]

且

[  
X^2>B,  
]

那麼在這些高 blocks 中，只要 small-glue 邊密度高，就得到 label。

所以困難不是單個 pair，而是：

[  
\boxed{  
\text{如何從全局能量小推出某個高 pair 的 small-glue 密度高？}  
}  
]

這其實可用平均：

若總能量

[  
Q(a)\le R,  
]

在某個高 block pair (P_kP_\ell) 上，平均能量不高，則 Markov 給 small-glue density。只要該 block pair 的自然 cutoff (B) 可見，即可產生 label。

這又回到能量層 (R) 與可見尺度的匹配。

---

# 4. Top 問題怎麼避免？

剛纔 tower anchor 遇到 top entropy 問題，是因爲我們想覆蓋所有 (R)。但對 minor arc 求和，不一定要對所有 (R) 用同一種方法。

可以分三段：

## 低—中能 (R)

對每個 (R)，選一個可見高尺度 block pair 作 anchor，取得 (h)，再用 Irving correction 傳播。這部分可行。

## 很高能 (R)

若

[  
R  
]

已大到足以壓過剩餘 character entropy，就直接用

[  
e^{-cR}  
]

殺掉。這部分也可行。

## 中間到高能的缺口

問題是是否存在一段：

[  
V_{\mathrm{top}}\ll R\ll \log|\widehat G|  
]

其中 anchor 看不見，而 raw bound 又殺不掉。這是剛纔指出的 top gap。

要關掉此 gap，有兩種方法：

1. 增加更高的 control blocks，但不讓它們帶來同量級 character entropy；
    
2. 證明高頻 diagonal / non-diagonal 在 top 附近已有局部 aperiodicity，不需要更高 anchor。
    

第一種需要設計「control primes」；第二種需要 aperiodicity theorem。

---

# 5. 新想法：control blocks 不參與目標質量？

我們可以加入一些非常高的 auxiliary prime blocks，只用於 Fourier 控制， reciprocal mass 極小，不影響期望和主弧太多。可是只要它們出現在 denominators (pq)，其 primes 就進入 (L)，character group 也增加 entropy。

但如果 auxiliary blocks 只以很低 Bernoulli probability 使用？Fourier factor 變弱；character entropy仍在。若概率太低，控制力不足。

如果 auxiliary primes 不進入 denominators，不能控制 residues。  
所以這條不簡單。

---

# 6. 也許原 full graph 線性 dyadic range 已足夠

之前 top entropy 用 raw (\log|\widehat G|\sim X_K) 太粗。其實 top block vertices雖多，但每個 top vertex有大量 edges to nearby blocks；若它亂，Irving correction / local energy 會立即殺掉。也就是說，top entropy 不應用 raw bound，而應用本地 correction，即使沒有更高 anchor，也可以用低一層 labelled block。

但如果 top block本身沒有 label呢？它與低一層 pair 若對當前 R invisible，仍可能由一個大 label控制；而這個大 label的 entropy只有 (\log H)，不是 raw (X_K). 所以 top gap也許可由「label not raw」處理，不一定需要上方 anchor。

這表示我們又回到 weighted list-agreement theorem；但現在更清楚：top block若不是 labelled，才有問題；若 labelled by large (h)，其 entropy小，diagonal energy處理。

---

# 7. 先證一個「one-sided labelled neighbour」命題

我們已有 vertex correction：若鄰居 block有 label (h)，則本 block所有 wrong vertices付能量。

所以只要每個 connected component有至少一個 labelled block，就能傳播 label。  
若某 component完全沒有 labelled block，那它所有 incident pair都不可見或高能。這種 component應該只能由 large labels構成，否則能量大。

這提示可用 **seed labels**：

- 在每個 energy level (H)，找所有 blocks where (H) visible；
    
- 在那些 blocks獲得 seed label；
    
- labels propagate to adjacent lower/higher blocks;
    
- components without seed correspond to labels above all visible scales, hence high-frequency diagonal tail.
    

仍舊是 multiscale.

---

# 8. 我們可以先把困難孤立爲「seedless component lemma」

**Seedless component lemma.**  
Fix a label scale (H). Consider a connected set of blocks all below visibility for (H). If residues are not explained by a single label of size (>H), then energy is large enough to pay entropy.

This is hard, but more local.

Maybe exact list-agreement tools can prove this.

---

# 9. 目前最誠實的推進方式

現在不宜再聲稱「下一步即可閉合」。應該承認核心 theorem 未證，然後做兩件事：

1. 記錄已證或可信的 lemmas；
    
2. 把未證的 CRT Poincaré theorem 作為明確 conjectural lemma，並研究簡化案例。
    

簡化案例可以是：

- only diagonal (h): prove aperiodicity for (\sum|h/(pq)|^2);
    
- two-label mixture: prove cross energy lower bound;
    
- one top block + lower labelled block: vertex correction already done.
    

其中 diagonal aperiodicity 是最值得先攻的，因爲它是 central theorem 的子問題。如果 diagonal high tail都證不了，整體更難。

# 10. 下一步：diagonal aperiodicity 子問題

我們下一步專攻：

[  
E(h)=\sum_{pq\in E}\left|\frac{h}{pq}\right|^2.  
]

要證：

[  
\sum_{h\notin[-C/\sigma,C/\sigma]}e^{-cE(h)}=o(1/\sigma).  
]

這完全是一維問題，沒有 arbitrary residues。若能解，至少 high-frequency/top issue的一半解決。

而且它也可能有標準 analytic number theory / local CLT 工具可用。

我建議接下來就把 diagonal aperiodicity theorem 獨立出來推。這是目前最可控的硬骨頭。
