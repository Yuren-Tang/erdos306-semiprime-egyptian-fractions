查了。結論很清楚：**可以借鑑 weighted Bernoulli local limit theorem 的文獻，但它不會自動替我們解掉 diagonal aperiodicity。** 也就是說，這裏仍需我們證一個專門的數論／CRT aperiodicity lemma。

最相關的一篇是 Giuliano–Weber, _Local Limit Theorems in some Random models from Number Theory_。它明確研究「weighted sums of Bernoulli variables」，並說這類問題在一般 local limit theorem 理論中重要但「not well explored」，且它們用 characteristic function method 證明若干新 local limit theorem，並應用到數論中的表示問題。這和我們的 diagonal 子問題方向非常接近。([arXiv](https://arxiv.org/abs/1502.05939?utm_source=chatgpt.com "Local Limit Theorems in some Random models from Number Theory"))

但問題是：一般 lattice LCLT 給的多是框架，真正難點仍是 characteristic function 在高頻區的衰減，也就是我們的

[  
\sum_{h\notin\mathfrak M}  
\prod_{pq}  
\left|1-\theta+\theta e\left(\frac{h}{pq}\right)\right|  
]

要小。這正等價於證

[  
E(h)=\sum_{pq}\left|\frac{h}{pq}\right|^2  
]

在非主弧上足夠大。一般「span (=1)」不是充分條件；triangular array 中還需要強 aperiodicity。這點不能偷懶。

## 所以接下來怎麼辦？

我們應把 diagonal 子問題獨立成一個定理：

**Diagonal aperiodicity theorem.**  
對我們的 semiprime weight set

[  
\mathcal W=\left{\frac1{pq}: pq\in E_{\mathrm{full}}\right},  
]

證明

# [  
\sum_{\substack{h\bmod L\ h\notin[-C/\sigma,C/\sigma]}}  
\exp\left(  
-c\sum_{pq\in E_{\mathrm{full}}}  
\left|\frac{h}{pq}\right|^2  
\right)

o(1/\sigma).  
]

若此定理成立，diagonal high-frequency 就關掉。  
若不成立，主 Fourier proof 會失敗，因爲即使 non-diagonal agreement 做完，高頻對角也可能污染主項。

---

# 1. 先分解 diagonal high-frequency

令

[  
L=\prod_{p\in\mathcal P}p.  
]

每個 (h\bmod L) 對應 residues

[  
h_p=h\bmod p.  
]

有三類：

### A. 小整數代表

[  
|h|\le a_{\max}^{-1}\asymp X_0^2.  
]

這裏沒有繞回：

[  
\left|\frac{h}{pq}\right|=\frac{|h|}{pq}  
]

對最低邊也成立。因此

[  
E(h)\asymp h^2\sigma^2.  
]

主弧外由 Gaussian tail 處理。這段已穩。

### B. 中等大，但可在某高尺度可見

也就是存在某個 block pair (P_kP_\ell)，使

[  
|h\bmod pq|_{\mathrm{sym}}  
]

若對很多邊小，dense agreement 迫使 (h) 等於一個小 label modulo 很多 primes；若不小，能量大。這是 multiscale dense agreement 可處理的範圍。

### C. 真正高頻 / above-top

(h) 在所有可用尺度都可能繞回。這一段不能靠小相位展開，需要 aperiodicity。這是難點。

---

# 2. 可嘗試的直接證明路線

對 diagonal (h)，如果很多邊滿足

[  
\left|\frac{h}{pq}\right|\le \delta_{pq},  
]

則這等價於 CRT glue (h\bmod pq) 的對稱代表小。對任一完整二部 block pair，若這種小邊密度高，bounded dense agreement 會說：

存在小整數 (h_{k,\ell}) 使

[  
h\equiv h_{k,\ell}\pmod p  
]

對大多數 (p) in the two blocks.

也就是

[  
h-h_{k,\ell}  
]

被許多 primes 整除。若在多個 blocks 上都如此，(h) 被迫等於某個小 (h_0) modulo 一個巨大 prime product。若這個 product 接近 (L)，那麼 (h) 在 character group 中就是小 (h_0)，即 major arc；否則未覆蓋的 primes 對應 edges 應提供能量。

這是 diagonal 版的 CRT agreement，應比 general non-diagonal 更容易，因為所有頂點 residues 已由同一 (h\bmod L) 給定。

---

# 3. 這給出一個更可攻的命題

我們先證：

**Diagonal small-glue collapse lemma.**  
若對每個主要 block pair，至少 (1-\varepsilon) 的邊滿足

[  
|h\bmod pq|_{\mathrm{sym}}\le B_{k,\ell},  
]

且 (B_{k,\ell}) 在該 pair 可見，則 (h\bmod L) 等於一個小整數 (h_0) on all but (O(\varepsilon)) primes.

然後未覆蓋 primes 的 entropy／數量可由 incident edge energy支付。

這其實是 central agreement theorem 的 diagonal special case，但少了一層 arbitrary residue 的困難。

# 5. 下一步具體推進

我建議現在證 diagonal small-glue collapse 的第一個版本：

固定一個 visible block pair (P\times Q)。若

[  
|h\bmod pq|_{\mathrm{sym}}\le B  
]

在 (1-\varepsilon) 的邊上成立，且

[  
B<Y/2,\quad X^2>2B,  
]

則存在 (|h_0|\le B)，使

[  
h\equiv h_0\pmod p  
]

對除 (O(\varepsilon)) 外的 (p\in P\cup Q) 成立。

這其實直接由 dense agreement lemma 得出。  
然後我們再把它跨 blocks 黏合，處理 diagonal high-frequency。
