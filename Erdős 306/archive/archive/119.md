# 1. Diagonal character 也是一組 vertex residues

對 diagonal (h)，每個 prime vertex (p) 有 residue

[  
h_p\equiv h\pmod p.  
]

邊相位可以寫成 CRT 形式：

# [  
\phi_{pq}(h)

\frac{h_p\bar q}{p}  
+  
\frac{h_q\bar p}{q}  
\pmod1.  
]

若存在一個 ordinary integer (m)，使大多數 vertices 滿足

[  
h_p\equiv m\pmod p,  
]

那麼 diagonal (h) 在大多數頂點上就是 ordinary label (m)。

若某頂點 (q) 不服從 (m)，即

[  
d=h_q-m\not\equiv0\pmod q,  
]

而相鄰 block (P) 中大多數 (p) 服從 (m)，則邊相位為

[  
\phi_{pq}(h)  
\equiv  
\frac{m}{pq}  
+  
\frac{d\bar p}{q}  
\pmod1.  
]

這正是 vertex correction。Irving 給出：

[  
\sum_{p\in P}  
\left|  
\frac{d\bar p}{q}  
\right|^2  
\gg |P|.  
]

所以 diagonal high-frequency 中，任何不服從某個 established (m) 的 prime，也會被懲罰。

這說明：diagonal case 其實可以完全放回「label + wrong vertices」框架。

---

# 2. 核心問題變成：是否存在 dominant ordinary (m)

如果對某個 block (P_k)，residues

[  
h_p=h\bmod p  
]

在大多數 (p\in P_k) 上不等於任何同一 ordinary (m)，那麼這個 block 沒有 dominant label。

但如果它和相鄰 block (Q) 的邊能量很低，adaptive dense agreement 會產生一個 dominant label。  
因此：

[  
\boxed{  
\text{低邊能量 } \Rightarrow \text{dominant }m\text{ on adjacent blocks。}  
}  
]

若沒有 dominant (m)，則它與相鄰 blocks 的邊能量必大。

所以 diagonal high-frequency 的 aperiodicity 可以按如下分解：

1. 找一個低能 block pair，得到 (m)；
    
2. 用 Irving correction 把 (m) 傳播到所有 blocks；
    
3. 若某處不能傳播，付能量；
    
4. 若傳播到全部 primes，則 (h\equiv m\pmod L)，即 (h=m) as character。
    

這就是完整閉環。

# 3. 為何這也處理 above-top？

above-top 的問題是：(h) 很大，單看最高尺度未必可見。

但 residue-distribution 角度不要求 (h) 作為整數小；只要求存在某個 ordinary (m) 解釋一個低能 pair。

若 (E(h)) 很小，則在許多 pair 中 average phase 小。對那些 pair，只要 adaptive cutoff visible，就能得到 (m)。如果當前 pair 不 visible，換更高 pair。若沒有更高 pair，則該 high block 若不服從已知 lower label (m)，Irving correction會付能量。

所以 above-top 不是單獨反例；它只是缺少初始 (m) 的情況。只要某處能找到 seed (m)，top 也可被拉齊。

仍有問題：若完全找不到 seed (m)，即所有可見 pair 都能量不低或不可見，那怎麼辦？  
對 diagonal (h)，若完全找不到 seed，說明在所有可見 pair 上 small-glue density 不高，因而每個可見 pair 提供固定能量。若可見 pairs 足夠多，總能量大。這應能處理。

# 4. Seed-or-energy lemma

我們可以提出：

**Diagonal seed-or-energy lemma.**  
給定 diagonal (h) and energy layer (R). Either:

1. 存在某個 visible block pair (P_kP_\ell) 產生 ordinary seed label (m)，即  
    [  
    h\equiv m\pmod p  
    ]  
    for most vertices in (P_k\cup P_\ell); or
    
2. (E(h)\ge R_0) with (R_0) large enough that its Fourier contribution is negligible.
    

這個 lemma 比前面 top anchor 更合理，因為它不要求指定最高 pair，而是在所有 visible pairs 中尋找一個 seed。

若沒有 seed，所有 visible pairs 都失敗，每個失敗給能量。可見 pairs 的數量很多，因此總能量可能足夠大。

# 5. 估算 no-seed energy

取一族 disjoint visible block pairs，避免重複計能量。例如取許多相隔的 pairs

[  
P_j\times P_{j+1}  
]

where visibility threshold for chosen cutoff holds.

若每個 pair 沒有 seed，則每個 pair energy 至少

[  
\gg V_j  
\asymp X_j^{1/3}/k_j^2  
]

at the maximal visible cutoff.

Sum over many (j):

[  
\sum_j V_j.  
]

因 (X_j) grows geometrically/exponentially in j, sum dominated by top visible j. Thus no-seed energy at least top visible threshold.

Still finite. But for high R, top visible threshold may be below R. However if (E(h)\le R), no-seed energy (\le R) possible. It does not force contradiction; but for counting, no-seed h may still be few? Need count.

No-seed h are diagonal residues not close to small m on any visible pair. There may be many, but each has energy at least (V_{\max}). Their contribution to Fourier sum could be bounded by (L e^{-cV_{\max}}), too weak.

So seed-or-energy alone not enough unless (V_{\max}\gg\log L), false.

# 7. Maybe use average over h exactly

For diagonal h, we can compute average of (e^{-cE(h)}) over h modulo L maybe factors by CRT?

# 8. Where we stand

Even diagonal high-frequency is not trivial. It seems to require a partition function estimate akin to the general CRT Poincaré theorem, though simpler.

We can still proceed by building Peierls-style proof:

- ground states: ordinary small h;
    
- defect vertex: residue differs from h, costs incident energy via Irving;
    
- contours: regions with different h, cost cross energy via dense agreement;
    
- sum defects by energy.
    

This is essentially what we were building. So diagonal case confirms the statistical-mechanics interpretation.

# 9. Useful reframing: low-temperature ferromagnetic CRT model

The Fourier minor arc partition function

[  
\sum_a e^{-cQ(a)}  
]

is a Gibbs partition function.

Ground states are diagonal labels (h).  
Major arc are low-energy ground states near (h=0).  
Minor arcs are excitations.

We need a Peierls bound: excitations cost more energy than entropy.

This is conceptually right and may be the cleanest final language.

The hard theorem is then a Peierls condition for CRT interactions, with Irving giving single-site defect cost and dense agreement giving contour cost.

# 10. Continue: formulate Peierls condition

Define for each assignment (a) its best global label (h). Defect set:

[  
D_h(a)={p:a_p\ne h\bmod p}.  
]

Peierls condition:

[  
Q(a)-Q(h)  
\ge  
\kappa \sum_{p\in D_h(a)}\log p  
]

for some (\kappa\to\infty) or at least (>) entropy constant.

If true, partition function around each h is controlled.

Then sum over h using diagonal energy.

This would solve everything.

Can we prove Peierls condition? If h is best label and p defect, Irving correction with nondefect neighbouring block gives cost. Need ensure enough nondefect neighbours. If defects percolate forming a large cluster, contour boundary to nondefect region pays. If all vertices defect relative to h, choose different h. This is classic Peierls.

So final hard lemma becomes Peierls contour theorem.
