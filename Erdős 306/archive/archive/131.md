# Semiprime reciprocal diagonal partition theorem

設 (G=(\mathcal P,E)) 是一個 semiprime graph，邊為

[  
pq\in E.  
]

令

[  
L_G=\prod_{p\in\mathcal P}p.  
]

對

[  
h\in\mathbb Z/L_G\mathbb Z  
]

定義

# [  
E_G(h)

\sum_{pq\in E}  
\left|  
\frac{h}{pq}  
\right|^2.  
]

這裏

[  
\left|\frac{h}{pq}\right|  
]

就是 (h\bmod pq) 的對稱代表除以 (pq)。

要證的是：

[  
Z_G  
:=  
\sum_{h\bmod L_G}  
e^{-cE_G(h)}  
\ll  
\frac1{\sigma_G},  
]

其中

# [  
\sigma_G^2

\sum_{pq\in E}\frac1{p^2q^2}.  
]

而且主弧外：

[  
\sum_{\substack{h\bmod L_G\ |h|_{\mathrm{sym}}>C/\sigma_G}}  
e^{-cE_G(h)}  
=o(1/\sigma_G).  
]

這就是真正的 diagonal aperiodicity。

---

## 1. 為什麼這個定理合理？

如果 (h) 是小整數，且無繞回，則

# [  
E_G(h)

h^2\sigma_G^2.  
]

因此

[  
\sum_{h\in\mathbb Z}e^{-c h^2\sigma_G^2}  
\asymp  
\frac1{\sigma_G}.  
]

所以 theorem 說的是：

[  
\boxed{  
\text{除了 ordinary small }h\text{ 之外，不存在大量其他低能 }h\bmod L_G。  
}  
]

也就是一維 aperiodicity。

---

## 2. 先看極端反例風險

有沒有可能存在很多 (h) 使

[  
h\bmod pq  
]

對所有邊 (pq) 都很小？

若對所有邊都等於 (0)，則

[  
pq\mid h  
\quad\forall pq\in E.  
]

如果 graph 連通且每個 prime 有 incident edges，這迫使

[  
p\mid h  
\quad\forall p\in\mathcal P,  
]

所以

[  
h\equiv0\pmod{L_G}.  
]

零能只有 (h=0)。這很好。

若對所有邊都等於同一個小 (m)，則

[  
h\equiv m\pmod p  
\quad\forall p,  
]

所以

[  
h\equiv m\pmod{L_G}.  
]

這正是小 diagonal。

所以 exact low-energy states 沒有奇怪 obstruction。

---

## 3. Approximate low-energy 的圖像

若

[  
E_G(h)  
]

小，則對多數邊

[  
h\bmod pq  
]

的對稱代表很小。

在一個 dense block pair 中，small-glue collapse 告訴我們：

存在 ordinary (m) 使

[  
h\equiv m\pmod p  
]

對大多數頂點成立。

如果這種結論在足夠多 blocks 中成立並且 local (m)'s 合併，則 (h) 接近 ordinary small (m)。  
不服從 (m) 的 primes 由 Irving correction 付能量。

所以 theorem 的證明可走「低能 implies small global representative」路線。

---

# 4. 但高頻 counting 要更強

我們不能只證「每個低能 (h) 接近某個 (m)」，還要計數：

[  
#{h:E_G(h)\le R}  
]

不能太大。

理想 counting bound：

[  
#{h:E_G(h)\le R}  
\le  
e^{\varepsilon R}  
#{m\in\mathbb Z:m^2\sigma_G^2\le C R}.  
]

若這成立，則

[  
Z_G  
\ll  
\sum_R e^{-cR}e^{\varepsilon R}  
\frac{\sqrt R}{\sigma_G}  
\ll  
\frac1{\sigma_G}.  
]

所以我們要證的其實是 diagonal 版的 energy–entropy inequality。

# 5. 對角 counting 的自然編碼

給 (h)，若它接近 ordinary (m)，定義例外 prime set：

[  
T={p\in\mathcal P:\ h\not\equiv m\pmod p}.  
]

若 (T=\varnothing)，則

[  
h\equiv m\pmod{L_G}.  
]

若 (T\ne\varnothing)，則 (h) 由以下資料決定：

1. (m);
    
2. 例外集合 (T);
    
3. 對每個 (p\in T)，residue (h_p\ne m\bmod p).
    

這些例外資料的 entropy 是

[  
\sum_{p\in T}\log p  
]

加上集合選擇熵。

若每個例外 prime (p) 付 incident energy

[  
\gg |\text{neighbour block}|,  
]

則 entropy 被支付。

這就是 diagonal counting 的核心。

## 6. 關鍵是找到 (m)

如果 (E_G(h)\le R)，怎麼找到 (m)？

選一個 block pair (P\times Q) 使局部能量不太大且 cutoff 可見。  
若存在，就用 small-glue collapse 得到 (m)。

若不存在，說明每個可見 block pair either energy high or small-glue density低。累積起來應使 (E_G(h)) 大。這裏要有一個 graph averaging lemma：

**Seed lemma.**  
For every (h) with (E_G(h)\le R), there exists a block pair (P\times Q) such that:

- (E_{P,Q}(h)\le C) times its fair share;
    
- adaptive cutoff is visible;
    
- hence small-glue collapse produces (m).
    

這個 lemma 對 (R) 很大時可能失敗，但如果 (R) 很大，tail weight (e^{-cR}) 小；計數仍需處理。  
所以更好的是 energy-layer seed:

For every (h), either:

- seed (m) exists and exceptions pay; or
    
- (E_G(h)) is already so large that number of such h is controllable.
    

後者仍需計數，不能 raw bound。

# 8. Fourier/Parseval 方法？

對 diagonal partition function

[  
Z_G=\sum_h e^{-cE_G(h)}  
]

也許能用乘積結構估計，而不是 level set agreement。

注意：

[  
e^{-c|x|^2}  
]

有 Fourier series on (\mathbb T):

# [  
e^{-c|x|^2}

\sum_{n\in\mathbb Z}\alpha_n e(nx),  
]

其中 (\alpha_n) 衰減很快且正？若係數非負或可控，則

# [  
Z_G

\sum_{h\bmod L}  
\prod_{pq\in E}  
e^{-c|h/(pq)|^2}  
]

可展開為

[  
\sum_{{n_{pq}}}  
\left(\prod \alpha_{n_{pq}}\right)  
\sum_{h\bmod L}  
e\left(  
h\sum_{pq}\frac{n_{pq}}{pq}  
\right).  
]

內和非零 iff

[  
\sum_{pq} n_{pq}\frac{L}{pq}  
\equiv0\pmod L,  
]

即

[  
\sum_{pq}\frac{n_{pq}}{pq}\in\mathbb Z.  
]

這是一個 additive relation among reciprocal semiprimes.

如果只有 trivial small relations，則 partition function controlled.

這很有趣：diagonal aperiodicity 等價於 weighted count of integer relations among (1/(pq)).

## 9. 關係條件的 CRT 解讀

條件

[  
\sum_{pq} n_{pq}\frac{L}{pq}\equiv0\pmod L  
]

等價於對每個 prime (r\in\mathcal P)：

[  
\sum_{q:rq\in E} n_{rq}\frac{L}{rq}  
\equiv0\pmod r?  
]

Modulo (r)，只有 terms with denominator containing (r) survive? More precisely (L/(pq)) is divisible by (r) unless (r=p) or (r=q). Thus mod (r),

[  
\sum_{q:rq\in E} n_{rq}\frac{L}{rq}\equiv0\pmod r.  
]

Since

[  
\frac{L}{rq}\not\equiv0\pmod r  
]

and depends on (q), this is a linear condition on incident coefficients.

This resembles a flow/cycle condition on the graph. Nontrivial relations correspond to integer flows satisfying local congruences. Their count may be controllable because coefficients (n_{pq}) are small due to Fourier coefficient decay.

This could be a new route.
