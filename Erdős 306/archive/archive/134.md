對，又是「層變塔」；但這次應該把它看清楚。

以前的「塔」是我們企圖用更高尺度去看見更大的 label。這個思路本身沒錯，但如果只靠「加一層更高的塔」來解決問題，就會一直遞歸下去：高層本身又帶來新的 CRT 熵。這就是你說的反覆遇到同一問題。

所以正確結論不是：

[  
\text{再加一層塔。}  
]

而是：

[  
\boxed{  
\text{需要一個真正的多尺度定理，一次性控制所有尺度。}  
}  
]

也就是 **Multiscale CRT Poincaré theorem**。

# 1. 為什麼單層、二層、三層都不夠？

單層沒有邊，完全沒有控制。

星圖失敗，因為每條邊可以單獨調整，留下大量自由度。

二層完整二部圖有 agreement 現象，但只在某個可見 cutoff 內有效。若 label 太大，二層看不見。

三層能讓兩個 local labels 在中間層合併，比二層強；但它仍只能處理某些能量層。更大的 label 要更多層。

所以，真正需要的是：

[  
\text{所有尺度一起工作。}  
]

這就是多尺度 Poincaré，不是簡單 tower patch。

---

# 2. 正確核心命題

現在最清楚的核心命題是：

[  
\boxed{  
Z:=\sum_{a\in\prod_{p\in\mathcal P}\mathbb F_p}  
\exp(-cQ(a))  
\ll  
\frac1\sigma.  
}  
]

其中

[  
Q(a)=\sum_{pq\in E}|\phi_{pq}(a)|^2.  
]

而且主弧外要有：

[  
\boxed{  
\sum_{a\notin\mathfrak M}e^{-cQ(a)}  
=o(1/\sigma).  
}  
]

主弧 (\mathfrak M) 不是任意 CRT class，而是 ordinary small labels：

[  
a_p\equiv m\pmod p,\qquad |m|\le C/\sigma.  
]

這個命題一旦成立，minor arc 就閉合。

它同時處理：

- arbitrary CRT high frequency；
    
- non-diagonal residue disorder；
    
- large ordinary labels；
    
- local labels 不合併；
    
- isolated blocks；
    
- top entropy。
    

所以它是最終硬核。

# 3. 是否需要新工具？

我裁決：**需要。**

不是說現有東西一定能直接套，而是現在這個核心命題已經超出「手工 Fourier bookkeeping」的舒適範圍。它像以下幾類工具的交界：

1. **large-alphabet agreement testing**  
    局部 pair tests 若多數通過，推出接近一個 global object。
    
2. **list agreement / direct product testing**  
    因為在中間能量層，local label 可能不是唯一，需要先得到短 list，再 collapse。
    
3. **Peierls / contour expansion**  
    多個 local phases 共存時，boundary energy 壓 entropy。
    
4. **lattice local CLT / aperiodicity**  
    最終要控制整個 characteristic function 的 (L^1) 或 partition function。
    

我們前面已經查到 agreement testing 和 list agreement 的文獻方向是 relevant；但現在目標更準了：不是找普通 agreement theorem，而是找 **large alphabet + partition/soundness bound**，最好能處理 weighted energy 而非 Hamming error。

# 4. 但現在還可以繼續推一個自證方向

即使要查工具，也先把自證方向寫成可比較的形式。

我建議把 Multiscale CRT Poincaré theorem 拆成三個 lemma：

## Lemma A：local inverse theorem

在一個 visible block pair (P\times Q) 中，若局部能量低，則大多數頂點由一個 ordinary label (m) 解釋；若能量不低，該 pair 付出相應成本。

這部分我們基本已有。

## Lemma B：multiscale propagation theorem

若在某尺度得到 label (m)，則沿 compatible graph 向上下尺度傳播；任何不服從 (m) 的 vertex 由 Irving correction 付能量。

這部分也很有力。

## Lemma C：no-seed entropy theorem

若一個 configuration 在任何尺度都不能產生 ordinary label seed，則其總能量足夠大，並且該類 configuration 的數量被

[  
e^{\varepsilon Q}  
]

控制。

這是最硬的部分。它就是 large-alphabet agreement / Poincaré 的核心。

# 5. 為什麼 no-seed 是核心？

因為一旦有 seed (m)，Irving correction 很強，問題好辦。  
麻煩的是那些看似每個局部 pair 都「不夠好」但又不付足夠能量的 configurations。

它們可能有：

- 中等大的 local labels；
    
- 多個 labels；
    
- CRT wrap-around；
    
- high-frequency residues；
    
- block clusters。
    

這就是反覆出現的那個問題。

所以我們不應再把它叫作 top issue、diagonal issue、large label issue；統一叫：

[  
\boxed{\text{no-seed entropy problem}.}  
]

# 6. 下一步：先寫 no-seed theorem 的可測版本

**No-seed theorem, desired form.**  
Fix (R). Let

[  
\mathcal A_R={a:Q(a)\le R}.  
]

Then every (a\in\mathcal A_R) either:

1. has an ordinary seed label (m) on some visible block pair, with  
    [  
    m^2\sigma^2\lesssim R;  
    ]  
    after propagation, all exceptions cost energy; or
    
2. belongs to a no-seed class whose cardinality satisfies  
    [  
    #\mathcal N_R\le e^{\varepsilon R}.  
    ]
    

If this holds, then

[  
#\mathcal A_R  
\le  
e^{\varepsilon R}  
#{m:m^2\sigma^2\lesssim R}.  
]

Then

[  
Z\ll1/\sigma.  
]

這就是我們要證的 counting form。

# 7. 怎樣證 no-seed theorem？

可能需要 agreement testing 的 reverse union bound。

直觀上，如果沒有 seed，則每個 block pair 的 low-glue edges 不可能形成大稠密子圖。於是大量 pair tests fail。若 test failure probability bounded below across many independent-ish tests，總能量大。若 failure 很小，agreement theorem 反而產生 seed。

所以這是一個典型二分：

[  
\text{many tests pass}  
\Rightarrow  
\text{agreement / seed};  
]

[  
\text{not many tests pass}  
\Rightarrow  
\text{energy large}.  
]

我們缺的是把「many tests pass」從 edge-level 變成 global seed 的 theorem。

這就是 agreement testing。

# 9. 現在繼續推：ordinary integer global object 如何從 agreement theorem 出現？

一般 agreement theorem 的 global object 會是任意 assignment (p\mapsto a_p)，而不是 ordinary (m)。這不夠。

我們需要額外 CRT-smallness：edge glue small means the two residues are not merely consistent with some arbitrary CRT class, but with a **small ordinary integer** on that edge.

若 many such pairwise small ordinary glues agree, then the global object is an ordinary integer (m), not arbitrary residues.

所以我們的 test is stronger than ordinary agreement:

For edge (pq), test passes if the CRT combination of (a_p,a_q) lies near (0) as an ordinary integer.

This smallness is what selects ordinary (m).

Thus existing agreement theorem can provide global consistency; CRT smallness then forces the global object to be ordinary small (m).

這很有希望。

# 10. 繼續方向

接下來我會把 no-seed theorem 轉寫成 agreement-test form：

- vertices: primes;
    
- local views: residues (a_p);
    
- tests: edge (pq) passes at threshold (B) if (|H_{pq}^{*}|\le B);
    
- if many tests pass in a multiscale family, dense agreement yields ordinary seed (m);
    
- if not, energy lower bound by failed tests across thresholds;
    
- partition bound follows by energy-layer counting.
    

這是目前最正確的技術路線。
