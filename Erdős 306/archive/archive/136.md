# 下一步：把 no-seed theorem 寫成 agreement-testing lemma

現在先回到內部推導。核心 lemma 應該是：

**CRT reverse-union agreement lemma.**  
固定能量層 (R)。若一個 assignment (a) 在足夠多 compatible block pairs 上有高 pass density，即多數邊滿足

[  
|H_{pq}^{*}|\le B_{pq}(R),  
]

則存在 ordinary label (m)，且

[  
m^2\sigma^2\lesssim R,  
]

使大多數 vertices 服從 (m)。若不存在這樣的 (m)，則 failed tests 的能量總和至少足以支付該類 assignments 的熵。

這就是我們要從 agreement testing 借鑑的精確形態。

注意：這裏 global object 是 ordinary (m)，不是 arbitrary CRT class。這個區別非常關鍵；否則命題就變成空話。

# 接下來推 block-level good-component lemma

定義一個 block pair (P_iP_j) 為 (R)-good，若：

1. 它的局部能量不超過合理份額；
    
2. adaptive cutoff (B_{ij}(R)) 在該 pair 可見；
    
3. pass density 至少 (1-\eta)。
    

則 local agreement 給出一個 label

[  
m_{ij}.  
]

如果兩個 (R)-good pairs 重疊一個 block，則它們的 labels 必須一致，否則在重疊 block 上產生矛盾，或者付出額外能量。這是三層 agreement。

因此：

- 若 (R)-good pairs 在 block graph 中形成大連通分量，得到一個 global seed (m)；
    
- 分量外 vertices 若不服從 (m)，由 Irving correction 或 bad-pair energy 支付；
    
- 若沒有大 good component，則 bad pairs 很多，總能量大，形成 no-seed cost。
    

這就是下一個要正式寫的 lemma。
