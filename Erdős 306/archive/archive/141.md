# Labelled-boundary transfer lemma

取相鄰或 compatible 的兩個 prime blocks：

[  
P\subset[X,2X],\qquad Q\subset[Y,2Y],  
]

且尺度在 Irving 範圍內。設 (P)-側已完全服從 ordinary label (m)：

[  
a_p\equiv m\pmod p  
\qquad(p\in P).  
]

對 (Q)-側 residues

[  
b=(b_q)_{q\in Q}  
]

定義二部能量

# [  
Q_{P,Q}(m,b)

\sum_{p\in P,q\in Q}  
|\phi_{pq}(m,b)|^2.  
]

其中

# [  
\phi_{pq}(m,b)

\frac{m\bar q}{p}  
+  
\frac{b_q\bar p}{q}  
\pmod1.  
]

若 (b_q\equiv m\pmod q)，則

[  
\phi_{pq}(m,b)=\frac{m}{pq}\pmod1.  
]

所以 diagonal contribution 是

# [  
Q_{P,Q}^{\mathrm{diag}}(m)

\sum_{p,q}  
\left|  
\frac{m}{pq}  
\right|^2.  
]

---

## Lemma

存在常數 (c,c'>0)，使得

[  
\sum_{b\in\prod_{q\in Q}\mathbb F_q}  
\exp\bigl(-cQ_{P,Q}(m,b)\bigr)  
\le  
C  
\exp\bigl(-c'Q_{P,Q}^{\mathrm{diag}}(m)\bigr),  
]

其中 (C) 對 (P,Q,m) 一致有界；只要 (P,Q) 已經做過 Irving-good pruning，且 (X) 充分大。

---

# 證明

因為 (P)-側固定為 (m)，能量對 (Q)-側各個 (q) 分解：

# [  
Q_{P,Q}(m,b)

\sum_{q\in Q}  
E_q(b_q),  
]

其中

# [  
E_q(r)

\sum_{p\in P}  
\left|  
\frac{m}{pq}  
+  
\frac{(r-m)\bar p}{q}  
\right|^2.  
]

因此

# [  
\sum_b e^{-cQ_{P,Q}(m,b)}

\prod_{q\in Q}  
\sum_{r\in\mathbb F_q}  
e^{-cE_q(r)}.  
]

現在固定 (q)。

若

[  
r=m\pmod q,  
]

則

# [  
E_q(r)

E_q^{\mathrm{diag}}(m)  
:=  
\sum_{p\in P}  
\left|  
\frac{m}{pq}  
\right|^2.  
]

若

[  
r\ne m\pmod q,  
]

令

[  
d=r-m\ne0\pmod q.  
]

則

# [  
E_q(r)

\sum_{p\in P}  
\left|  
\frac{m}{pq}  
+  
\frac{d\bar p}{q}  
\right|^2.  
]

用不等式

[  
|x+y|^2+|x|^2\gg |y|^2  
]

得

[  
E_q(r)+E_q^{\mathrm{diag}}(m)  
\gg  
\sum_{p\in P}  
\left|  
\frac{d\bar p}{q}  
\right|^2.  
]

由 Irving-good 性質，

[  
\sum_{p\in P}  
\left|  
\frac{d\bar p}{q}  
\right|^2  
\gg |P|.  
]

故

[  
E_q(r)  
\ge  
c_1|P|-C_1E_q^{\mathrm{diag}}(m).  
]

更穩妥地，可把 diagonal 項吸收到右側，得到

[  
e^{-cE_q(r)}  
\le  
e^{-c_2|P|}  
e^{C_2cE_q^{\mathrm{diag}}(m)}.  
]

但為了與 (r=m) 項比較，可選小一點的 (c)，得到

[  
\sum_{r\ne m}  
e^{-cE_q(r)}  
\le  
q,e^{-c_3|P|}  
e^{-c_4E_q^{\mathrm{diag}}(m)}  
]

或至少

[  
\sum_{r\in\mathbb F_q}  
e^{-cE_q(r)}  
\le  
e^{-c' E_q^{\mathrm{diag}}(m)}  
\left(1+q e^{-c''|P|}\right).  
]

因為

[  
|P|\asymp \frac{X}{\log X}  
]

而

[  
q\asymp Y  
]

只呈指數前的多項式大小，故

[  
q e^{-c''|P|}=o(|Q|^{-1})  
]

在 (X) 充分大時成立。於是

[  
\prod_{q\in Q}  
\left(1+q e^{-c''|P|}\right)  
\le C.  
]

因此

[  
\sum_b e^{-cQ_{P,Q}(m,b)}  
\le  
C  
\exp\left(  
-c'\sum_{q\in Q}E_q^{\mathrm{diag}}(m)  
\right),  
]

即

[  
\sum_b e^{-cQ_{P,Q}(m,b)}  
\le  
C  
e^{-c'Q_{P,Q}^{\mathrm{diag}}(m)}.  
]

證畢。

---

# 這個 lemma 的意義

它非常強。它說：

[  
\boxed{  
\text{一旦一側 block 有 label }m,\text{ 下一側 block 幾乎沒有自由度。}  
}  
]

所有不等於 (m) 的 residues 都被

[  
e^{-c|P|}  
]

殺掉，而 (q) 種 residue 選擇完全不足以抵消。

所以「有 seed label」之後，整條鏈都容易控制。

---

# 推論：有邊界 label 的整條鏈

若最左 block (P_{k_0}) 被固定為 label (m)，則沿最近鄰 skeleton chain 逐步套 labelled-boundary transfer lemma，得到

[  
\sum_{\text{all later blocks}}  
e^{-cQ_{\mathrm{ch}}}  
\le  
C^{K-k_0}  
\exp\left(  
-c'\sum_{k}Q_{k,k+1}^{\mathrm{diag}}(m)  
\right).  
]

若 (C^{K-k_0}) 是問題，可把 pruning / (k_0) 調大，或把每步常數做到 (1+o(1/K))。本質上，非 label residues 被逐層殺掉。

所以有 boundary condition 時，partition function 完全由 ordinary (m) 控制。

---

# 剩餘問題：free boundary

原問題沒有固定左端 label。自由鏈 partition function 是

# [  
Z_{\mathrm{ch}}

\sum_{u_{k_0},\dots,u_K}  
\prod_k K_k(u_k,u_{k+1}).  
]

Labelled-boundary lemma 控制的是：

[  
Z_{\mathrm{ch}}(m\text{ fixed at one boundary}).  
]

但自由左端

[  
u_{k_0}  
]

有巨大 residue entropy。這正是 no-seed / spontaneous alignment 問題：

[  
\boxed{  
\text{為何自由邊界不產生巨大 partition function？}  
}  
]

直觀上，因為若左端 block 不接近某個 ordinary label，則它很難同時與右邊所有後續 blocks 保持低能；但要證這點，需要一個 free-boundary Poincaré / agreement theorem。

---

# 下一個精確命題

我們現在可把剩餘問題寫成：

**Free-boundary chain Poincaré lemma.**  
For the nearest-neighbour block chain,

# [  
Z_{\mathrm{ch}}

\sum_{u_{k_0},\dots,u_K}  
e^{-c\sum_kQ_{k,k+1}(u_k,u_{k+1})}  
\ll  
\sum_{m\in\mathbb Z}  
e^{-c'\sum_kQ_{k,k+1}^{\mathrm{diag}}(m)}.  
]

這個 lemma 若成立，主 minor arc 就基本閉合。  
Labelled-boundary transfer lemma 是它的半邊證明；剩下是「自由邊界自發選擇 ordinary label」的部分。
