好。現在把核心局部工具正式命名為 **CRT list-Helly lemma**，然後說明它如何推出 single-block counting theorem。

這一步很重要，因為它把原本龐大的 Fourier minor arc 問題，壓縮到一個具體的組合／數論交界命題。

---

# 1. CRT list-Helly lemma

令

[  
P\subset[X,2X],\qquad N=|P|.  
]

每個頂點 (p\in P) 有 residue

[  
a_p\in\mathbb F_p.  
]

固定一個整數範圍

[  
[-B,B].  
]

對每個 (p)，定義 list

# [  
L_p

{n\in[-B,B]\cap\mathbb Z:\ n\equiv a_p\pmod p}.  
]

所以

[  
|L_p|\le 1+\frac{2B}{p}  
\ll 1+\frac{B}{X}.  
]

對一對 primes (p,q)，邊 (pq) 的 CRT glue 有小代表 (|H_{pq}^{*}|\le B)，等價於

[  
L_p\cap L_q\ne\varnothing.  
]

因為若存在 (n\in L_p\cap L_q)，則

[  
n\equiv a_p\pmod p,\qquad n\equiv a_q\pmod q,  
]

故 (n) 是該 CRT class 的一個 bounded representative。

因此，single-block good-edge graph 正是 list intersection graph：

[  
pq\text{ good}  
\quad\Longleftrightarrow\quad  
L_p\cap L_q\ne\varnothing.  
]

---

## CRT list-Helly lemma：目標表述

若

[  
#{pq:L_p\cap L_q\ne\varnothing}  
\ge (1-\eta)\binom N2,  
]

則存在一個整數

[  
m\in[-B,B]  
]

使

[  
m\in L_p  
]

對除 (O(\eta^cN)) 外的大多數 (p\in P) 成立。

換回 residues，就是：

[  
a_p\equiv m\pmod p  
]

對大多數 (p) 成立。

這正是我們需要的：高密度 pairwise CRT-small agreement 推出 one ordinary label majority。

---

# 2. 這個 lemma 為什麼可信？

它是一個特殊的 list-agreement / Helly 型命題。一般任意集合的 pairwise intersections 高密度，未必有大 common intersection；但我們的 lists 很特殊：

[  
L_p  
]

是同一整數區間中的一條等差數列，模數是不同的大 primes (p\sim X)。

這種結構非常剛性。

若選 regular base vertex (p_0)，則大多數 (q) 有

[  
L_q\cap L_{p_0}\ne\varnothing.  
]

因此大多數 vertices 被短 list

[  
L_{p_0}  
]

覆蓋。

而

[  
|L_{p_0}|\ll 1+\frac{B}{X}.  
]

若多個候選 (m\in L_{p_0}) 都覆蓋大群 vertices，則不同候選之間會形成不同 classes。  
兩個大 classes 之間若仍有很多 good edges，就意味著大量 arithmetic progressions 有交點，應迫使兩個候選其實相同，或產生可計量的矛盾。  
若 cross good edges 少，則在能量問題中這些 cross edges 就是 bad edges，提供能量。

所以即使 CRT list-Helly 的純組合版本需要細證，在 energy counting 裏我們甚至只需要較弱版本：

[  
\text{要麼有 dominant }m,\quad  
\text{要麼 bad/cross edges 足夠多。}  
]

這更容易成立。

---

# 3. Energy version of CRT list-Helly

我們真正需要的是：

**Energy list-Helly lemma.**  
若 good edge density 很高，則存在 dominant label (m)。  
若不存在 dominant label，則 bad edge density 有正下界，或者 cross-label energy 有正下界，足以支付 entropy。

形式上：

[  
#{a:Q_P(a)\le R}  
]

可以由以下資料編碼：

1. dominant (m);
    
2. exceptional vertices；
    
3. exceptional residues；
    

其中 exceptions 的總 entropy 由 (Q_P(a)) 支付。

這比純粹證所有 pairwise-intersecting lists 有 common element更貼近我們需求。

---

# 4. 從 CRT list-Helly 到 single-block counting

現在假設 list-Helly lemma 成立。證 single-block counting 很直接。

給定

[  
Q_P(a)\le R.  
]

取

# [  
B_R

A\frac{X^2}{N}\sqrt R  
\asymp A X\log X\sqrt R.  
]

由 Markov，good edge density 至少

[  
1-\eta.  
]

對每個 (p)，建 list

[  
L_p={n\in[-B_R,B_R]:n\equiv a_p\pmod p}.  
]

List-Helly 給 dominant label

[  
m\in[-B_R,B_R]  
]

使大多數 vertices obey

[  
a_p\equiv m\pmod p.  
]

---

# 5. 例外 vertices 的能量付款

令

[  
D={p:a_p\not\equiv m\pmod p}.  
]

對每個例外 vertex (q\in D)，若 majority set

[  
G={p:a_p\equiv m\pmod p}  
]

很大，則對 (p\in G)：

# [  
\phi_{pq}(a)

\frac{m}{pq}  
+  
\frac{(a_q-m)\bar p}{q}  
\pmod1.  
]

由 Irving-good 性質，若 (a_q-m\not\equiv0\pmod q)，則

[  
\sum_{p\in G}  
|\phi_{pq}(a)|^2  
+  
\sum_{p\in G}  
\left|\frac{m}{pq}\right|^2  
\gg |G|.  
]

因為 (|G|\asymp N)，每個例外 vertex 成本

[  
\gg N.  
]

而該 vertex 的 residue entropy 是

[  
\log q\asymp \log X.  
]

且

[  
N\asymp \frac{X}{\log X}\gg\log X.  
]

所以例外 residues 與例外集合選擇都由能量支付。

因此低能 assignments 數量至多：

[  
e^{\varepsilon R}  
\cdot  
#{m:|m|\le B_R}.  
]

---

# 6. label 數量與 Gaussian 尺度

因

[  
B_R\asymp X\log X\sqrt R  
]

而

[  
1/\sigma_P\asymp X\log X,  
]

所以

[  
#{m:|m|\le B_R}  
\ll  
\frac{\sqrt R}{\sigma_P}.  
]

也就是

[  
#{m:Q_P^{\mathrm{diag}}(m)\lesssim R}.  
]

故

[  
#{a:Q_P(a)\le R}  
\le  
e^{\varepsilon R}  
\frac{\sqrt R}{\sigma_P}.  
]

分層求和：

# [  
Z_P

\sum_a e^{-cQ_P(a)}  
\ll  
\sum_R  
e^{-cR}e^{\varepsilon R}  
\frac{\sqrt R}{\sigma_P}  
\ll  
\frac1{\sigma_P}.  
]

這就證出 single-block partition bound。

---

# 7. 這一步解決了什麼？

它解決了此前最大的 raw entropy 問題：單個 block 的 residue 空間大小原為

[  
\prod_{p\in P}p  
\approx e^{X},  
]

但同尺度 complete graph energy 把它壓成

[  
X\log X  
]

級別，即 ordinary label 的 Gaussian 規模。

這是極大的降維。

一旦每個 block 先局部塌縮成 ordinary label (m_k)，全局問題就從巨大 CRT alphabet 變成普通整數 labels 的多尺度 chain/graph 問題。

---

# 8. 接下來的全局步驟

加入所有同尺度 block edges後：

[  
Q_{\mathrm{full}}  
\ge  
\sum_k Q_{P_k}^{\mathrm{internal}}.  
]

所以每個 block 先被壓成一個 block label

[  
m_k.  
]

再使用跨 block edges (P_kP_\ell) 控制 labels (m_k,m_\ell)。若

[  
m_k\ne m_\ell,  
]

則跨邊 two-phase Irving energy 很大，除非兩 labels 在相關 primes 上不可區分；但 ordinary labels 若在 enough primes 上不可區分，就相等。

因此 block labels 合併成 single global ordinary (m)，例外 block labels由 boundary energy支付。

最後得到：

[  
Z_{\mathrm{full}}  
\ll  
\sum_m e^{-c m^2\sigma^2}  
\asymp  
\frac1\sigma.  
]

這正是所需。

---

# 9. 目前剩餘核心

現在最窄的未證命題是：

[  
\boxed{\text{CRT list-Helly lemma。}}  
]

它是局部、單尺度、純組合數論命題，比原問題小很多。

下一步應該集中攻這個 lemma：  
高密度 pairwise intersections of congruence-class lists inside ([-B,B]) 是否推出大 common intersection。

這是目前最值得投入的點。
