好。現在攻 **CRT list-Helly lemma**。我先不硬證最強的純 Helly 版，而證一個更適合我們的 **energy list-Helly dichotomy**：

[  
\boxed{  
\text{高密度 pairwise CRT-small agreement}  
\Rightarrow  
\text{要麼有 dominant ordinary label，要麼多 label 之間產生可收費的 cross energy。}  
}  
]

這個版本比「必有共同交點」更弱，但正好夠用。

---

# 1. 設定

取

[  
P\subset[X,2X],\qquad N=|P|.  
]

每個 (p\in P) 有 residue

[  
a_p\in\mathbb F_p.  
]

固定

[  
B.  
]

定義

[  
L_p={n\in[-B,B]\cap\mathbb Z:n\equiv a_p\pmod p}.  
]

若

[  
L_p\cap L_q\ne\varnothing,  
]

則邊 (pq) 是 good，等價於 CRT glue 有 bounded representative：

[  
|H_{pq}^{*}|\le B.  
]

假設 good graph 密度至少

[  
1-\eta.  
]

---

# 2. Base list reduction

選一個 regular base vertex (p_0)，使其 good degree 至少

[  
(1-\sqrt\eta)N.  
]

令

[  
\mathcal L=L_{p_0}.  
]

則

[  
|\mathcal L|\le 1+\frac{2B}{p_0}\ll 1+\frac BX.  
]

對每個與 (p_0) good-connected 的 vertex (q)，有

[  
L_q\cap\mathcal L\ne\varnothing.  
]

因此除了 (O(\sqrt\eta N)) 個 vertices 外，所有 vertices 都被短 list (\mathcal L) 覆蓋。

這一步已經把每個 vertex 的 (p) 種 residue 壓成

[  
|\mathcal L|\ll B/X  
]

個候選 ordinary labels。

---

# 3. 若某個 label 覆蓋絕大多數，則完成

若存在

[  
m\in\mathcal L  
]

使

[  
m\in L_q  
]

對至少

[  
(1-\rho)N  
]

個 (q) 成立，則

[  
a_q\equiv m\pmod q  
]

對大多數 (q) 成立。  
這就是 dominant ordinary label。

例外 vertices 由 Irving correction 付款，這部分前面已經處理過。

---

# 4. 若沒有 dominant label

若沒有 dominant label，則覆蓋 vertices 分散到多個 labels：

[  
C_m={q:m\in L_q}.  
]

注意一個 vertex 可能屬於多個 (C_m)，因為 (L_q) 可能與 (\mathcal L) 有多個交點。但

[  
|L_q\cap\mathcal L|  
]

通常很小。為了簡化，可任選一個 label 指派給 (q)，得到 disjoint classes

[  
P=\bigsqcup_{m\in\mathcal L}C_m  
]

加上少量 uncovered vertices。

若沒有 dominant class，則至少有相當多 cross pairs

[  
(p,q)\in C_m\times C_{m'},\qquad m\ne m'.  
]

我們要說：這些 cross pairs 若能量低／good density高，會迫使 labels collapse；否則 cross energy 足以付費。

---

# 5. Cross labels 的算術限制

取

[  
m\ne m'.  
]

令

[  
D=m'-m.  
]

因

[  
m,m'\in[-B,B],  
]

所以

[  
0<|D|\le 2B.  
]

對

[  
p\in C_m,\qquad q\in C_{m'},  
]

有

[  
a_p\equiv m\pmod p,\qquad a_q\equiv m'\pmod q.  
]

若邊 (pq) 也 good，則存在

[  
n\in[-B,B]  
]

使

[  
n\equiv m\pmod p,  
\qquad  
n\equiv m'\pmod q.  
]

等價於

[  
n=m+tp  
]

且

[  
m+tp\equiv m'\pmod q,  
]

即

[  
tp\equiv D\pmod q.  
]

對固定 (p,q,D)，在 ([-B,B]) 中這樣的 (n) 至多

[  
1+O\left(\frac{B}{pq}\right)  
]

個。當

[  
B\ll X^2  
]

時，至多 (O(1)) 個。

但我們需要的是：對大量 (p,q)，這種 good cross edge 不能太密，除非 (D) 有特殊整除結構。直觀上，(tp\equiv D\pmod q) 隨 (p) 分散，good cross edges 只佔約

[  
\frac{B}{X^2}  
]

比例。

這比直接 Irving subset 更像一個 incidence estimate。

# 6. 需要的 incidence bound

我們需要如下局部估計：

**Cross-incidence bound.**  
對任意兩個 disjoint subsets

[  
A,B\subset P  
]

和任意

[  
0<|D|\le2B,  
]

令 edge (pq\in A\times B) cross-good 若 CRT of (m\pmod p), (m+D\pmod q) has representative (\le B). Then

[  
#{(p,q)\in A\times B:\text{cross-good}}  
\le  
C\left(  
\frac{B}{X^2}|A||B|  
+  
\text{divisor-error}  
\right).  
]

若 (B\ll X^2), cross-good density is small unless (A,B) tiny.

這種估計應可用初等 divisor/incidence 方法證。因為 cross-good 意味著存在 (n) with

[  
|n|\le B,\quad n\equiv m\pmod p,\quad n\equiv m+D\pmod q.  
]

則

[  
p\mid n-m,\qquad q\mid n-m-D.  
]

所以每個 (n) contributes primes

[  
p\mid n-m,\qquad q\mid n-m-D.  
]

因此 cross-good pairs are counted by

[  
\sum_{|n|\le B}  
\omega_P(n-m),\omega_B(n-m-D),  
]

where (\omega_A(t)) counts prime divisors of (t) lying in (A).

This can be bounded by divisor estimates. Since each integer (t) has at most

[  
O(\log |t|/\log X)  
]

prime factors in ([X,2X]), the sum is at most

[  
\sum_{|n|\le B}  
O\left(\frac{\log B}{\log X}\right)^2  
\ll  
B\left(\frac{\log B}{\log X}\right)^2.  
]

This is independent of (|A||B|).

For large (A,B), this is much smaller than (|A||B|) provided

[  
B(\log B/\log X)^2\ll |A||B|.  
]

If (A,B) have sizes at least (N/s) and (s\sim B/X), then

[  
|A||B|\sim \frac{N^2X^2}{B^2}  
\sim \frac{X^4}{B^2\log^2 X}.  
]

Condition becomes roughly

[  
B\log^2B \ll X^4/B^2,  
]

i.e.

[  
B^3\ll X^4  
]

up to logs. For (B\sim X\log X\sqrt R), this allows

[  
R\ll X^{2/3}  
]

roughly. That covers quite high energy but not all. Maybe enough locally? Partition tail beyond huge (R) can be handled by raw because block entropy (\sim X) and (R\sim X^{2/3}) not enough to kill (e^X). Not enough.

Need sharper incidence using |A|,|B| maybe. Or use Irving subset? Hmm.

# 12. State needed cross-energy lemma

**Cross-label incidence-energy lemma.**  
For (P\subset[X,2X]), labels (m\ne m') with (|m|,|m'|\le B\le X^A), and subsets (A,B\subset P), unless one subset is tiny,

[  
\sum_{p\in A,q\in B}  
|\phi_{pq}(m,m')|^2  
\gg_A  
\frac{|A||B|}{(\log X)^C}.  
]

This is enough to collapse multiple classes.

This lemma is plausible via divisor counting of small residues, not requiring subset Irving.
