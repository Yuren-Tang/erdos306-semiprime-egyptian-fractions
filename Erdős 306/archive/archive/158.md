# Single-block counting theorem：嚴格版草案

令

[  
P\subset[X,2X],\qquad N=|P|\asymp \frac X{\log X}.  
]

對

[  
a=(a_p)_{p\in P},\qquad a_p\in\mathbb F_p  
]

定義

[  
Q_P(a)=  
\sum_{{p,q}\subset P}  
\left|  
\frac{H_{pq}(a)}{pq}  
\right|^2,  
]

其中 (H_{pq}(a)) 是滿足

[  
H_{pq}(a)\equiv a_p\pmod p,\qquad  
H_{pq}(a)\equiv a_q\pmod q  
]

的對稱代表。

令

[  
\sigma_P^2=  
\sum_{{p,q}\subset P}\frac1{p^2q^2}  
\asymp  
\frac1{X^2(\log X)^2}.  
]

目標命題：

[  
\boxed{  
#{a:Q_P(a)\le R}  
\le  
\exp(\varepsilon R+o(R))  
\left(1+\frac{\sqrt R}{\sigma_P}\right).  
}  
]

等價地，粗略為

[  
#{a:Q_P(a)\le R}  
\le  
e^{\varepsilon R}  
\cdot  
O(X\log X\sqrt R).  
]

---

## 1. Low-energy gives a dense good graph

令

[  
B=A X\log X\sqrt R  
]

其中 (A) 是足夠大的常數。

若

[  
Q_P(a)\le R,  
]

則 bad edges

[  
\mathcal B={{p,q}: |H_{pq}(a)|>B}  
]

滿足

[  
|\mathcal B| B^2/X^4  
\ll R.  
]

因

[  
B^2\asymp A^2X^2(\log X)^2R  
]

而

[  
N^2\asymp X^2/(\log X)^2,  
]

所以

[  
|\mathcal B|  
\ll  
\frac{R X^4}{B^2}  
\asymp  
\frac{X^2}{A^2(\log X)^2}  
\asymp  
\frac{N^2}{A^2}.  
]

取 (A) 大，good graph

[  
\mathcal G={{p,q}: |H_{pq}(a)|\le B}  
]

密度至少

[  
1-\eta.  
]

---

## 2. Base-list reduction

選一個 regular vertex (p_0)，使其 good degree 至少

[  
(1-\sqrt\eta)N.  
]

定義 base list：

# [  
\mathcal L

{n\in[-B,B]\cap\mathbb Z:\ n\equiv a_{p_0}\pmod {p_0}}.  
]

則

[  
|\mathcal L|  
\le  
1+\frac{2B}{p_0}  
\ll  
1+\log X\sqrt R.  
]

若 (q) 與 (p_0) 是 good edge，則

[  
H_{p_0q}(a)\in\mathcal L  
]

且

[  
H_{p_0q}(a)\equiv a_q\pmod q.  
]

所以除 (O(\sqrt\eta N)) 個 vertices 外，每個 (q) 至少被某個

[  
m\in\mathcal L  
]

覆蓋：

[  
a_q\equiv m\pmod q.  
]

這一步完成了第一個降熵：

[  
q\text{ 種 residue}  
\quad\leadsto\quad  
|\mathcal L|\ll \log X\sqrt R\text{ 種 label候選}.  
]

---

## 3. 分類與 dominant label

把被覆蓋 vertices 指派到某個候選 label：

[  
C_m={q:a_q\equiv m\pmod q},\qquad m\in\mathcal L.  
]

若存在 (m_*) 使

[  
|C_{m_*}|\ge (1-\rho)N,  
]

則 (m_*) 是 dominant label。  
此時例外集合

[  
D=P\setminus C_{m_*}  
]

會由 cross energy 付款。這是好情形。

若沒有 dominant label，則至少有 substantial cross mass：

[  
\sum_{m\ne m'}|C_m||C_{m'}|  
\gg_\rho N^2.  
]

我們要證這會強迫

[  
Q_P(a)  
]

很大，或者至少足以支付這種多 label 配置的 entropy。

# Cross-label divisor-energy lemma：嚴格命題

取不同 labels

[  
m\ne m',\qquad |m|,|m'|\le B.  
]

取 subsets

[  
A,B'\subset P.  
]

對

[  
p\in A,\quad q\in B'  
]

令 (H_{pq}^{m,m'}) 是滿足

[  
H_{pq}^{m,m'}\equiv m\pmod p,\qquad  
H_{pq}^{m,m'}\equiv m'\pmod q  
]

的對稱代表。

則對所有 (T\le X^2/2)，

[  
#{(p,q)\in A\times B': |H_{pq}^{m,m'}|\le T}  
\ll  
T L^2+(|A|+|B'|)L,  
]

其中

[  
L=1+\frac{\log(B+T+2)}{\log X}.  
]

由此可推出：

[  
\sum_{p\in A,q\in B'}  
\left(\frac{H_{pq}^{m,m'}}{pq}\right)^2  
\gg  
|A||B'|\min\left(  
1,\frac{|A|^2|B'|^2}{X^4L^4}  
\right)  
]

除非 (A) 或 (B') 很小；小的情形在 entropy bookkeeping 中作例外處理。

證明只用：

[  
p\mid n-m,\qquad q\mid n-m'  
]

和 dyadic prime divisor bound。

---

# 4. 多 label 情形的能量付款

若沒有 dominant class，將各 (C_m) 按大小 dyadic 分組。存在一組 classes，其總量不小，且任意兩個不同 label classes 的 cross-pair 總數不小。

對這些 cross pairs 套 cross-label divisor-energy lemma。得到：

[  
Q_P(a)  
\gg  
\text{multi-label entropy cost}.  
]

更具體地，若例外／非 dominant 部分大小為 (t)，則與 majority 或其他 substantial classes 的 cross energy 至少

[  
\gg  
t\cdot \frac{N}{(\log X)^C}  
]

在足夠大的範圍內成立。

而描述這 (t) 個 vertices 的熵至多

[  
O\bigl(t\log X+t\log |\mathcal L|+\log\binom Nt\bigr).  
]

因

# [  
\frac{N}{(\log X)^C}

\frac{X}{(\log X)^{C+1}}  
\gg  
\log X+\log|\mathcal L|,  
]

所以只要 (X) 足夠大，能量支付熵。

因此所有非 dominant 情形總數可吸收到

[  
e^{\varepsilon R}.  
]

---

# 5. dominant label 的數量

dominant label (m) 屬於某個 interval：

[  
|m|\le B=A X\log X\sqrt R.  
]

因此可能的 dominant labels 數量至多

# [  
O(B)

O(X\log X\sqrt R).  
]

而

[  
1/\sigma_P\asymp X\log X.  
]

所以

[  
#{m}  
\ll  
\frac{\sqrt R}{\sigma_P}.  
]

---

# 6. Single-block counting theorem 完成

綜合：

- 選 dominant label (m)：(\ll \sqrt R/\sigma_P) 種；
    
- 選例外 vertices 與 residues：由能量支付，總 factor (\le e^{\varepsilon R})；
    
- 無 dominant label：cross-label energy 過大，或同樣由 (e^{\varepsilon R}) 吸收。
    

得到

[  
#{a:Q_P(a)\le R}  
\le  
e^{\varepsilon R}  
\frac{\sqrt R}{\sigma_P}.  
]

這就是 Lemma 1。
