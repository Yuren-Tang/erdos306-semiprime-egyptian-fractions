好。現在把 **single-block counting theorem** 寫成一個完整證明草案。這一版的目標不是把每個常數最佳化，而是讓邏輯閉合、沒有概念性缺口。

# Single-block counting theorem

令

[  
P\subset[X,2X],\qquad N=|P|\asymp \frac X{\log X}.  
]

給定 residues

[  
a=(a_p)_{p\in P},\qquad a_p\in\mathbb F_p.  
]

對每條邊 (pq) 定義 CRT 對稱代表

[  
H_{pq}(a)\in(-pq/2,pq/2]  
]

滿足

[  
H_{pq}(a)\equiv a_p\pmod p,\qquad  
H_{pq}(a)\equiv a_q\pmod q.  
]

內部能量：

[  
Q_P(a)=  
\sum_{{p,q}\subset P}  
\left(\frac{H_{pq}(a)}{pq}\right)^2.  
]

又

[  
\sigma_P^2=  
\sum_{{p,q}\subset P}\frac1{p^2q^2}  
\asymp  
\frac1{X^2(\log X)^2}.  
]

欲證：

[  
\boxed{  
#{a:Q_P(a)\le R}  
\le  
e^{\varepsilon R}  
\left(1+\frac{\sqrt R}{\sigma_P}\right).  
}  
]

---

## 1. 低能推出稠密 good graph

設

[  
Q_P(a)\le R.  
]

取

[  
B=A X\log X\sqrt R  
]

其中 (A) 是足夠大的常數。

稱邊 (pq) 為 good 若

[  
|H_{pq}(a)|\le B.  
]

若 bad 邊數為 (M_{\mathrm{bad}})，則每條 bad 邊至少貢獻

[  
\gg \frac{B^2}{X^4}  
]

能量。故

[  
M_{\mathrm{bad}}\frac{B^2}{X^4}\ll R.  
]

代入

[  
B^2=A^2X^2(\log X)^2R  
]

得

[  
M_{\mathrm{bad}}\ll \frac{X^2}{A^2(\log X)^2}\asymp \frac{N^2}{A^2}.  
]

取 (A) 大，使 good graph 密度至少

[  
1-\eta.  
]

---

## 2. Base-list reduction

取一個 regular vertex (p_0)，使其 good degree 至少

[  
(1-\sqrt\eta)N.  
]

定義

[  
\mathcal L=  
{n\in[-B,B]\cap\mathbb Z:n\equiv a_{p_0}\pmod{p_0}}.  
]

則

[  
|\mathcal L|\le 1+\frac{2B}{p_0}  
\ll 1+\log X\sqrt R.  
]

若 (q) 與 (p_0) 相連為 good edge，則

[  
H_{p_0q}(a)\in\mathcal L  
]

且

[  
a_q\equiv H_{p_0q}(a)\pmod q.  
]

所以除了 (O(\sqrt\eta N)) 個 vertices 外，每個 (q) 都至少服從某個

[  
m\in\mathcal L.  
]

將這些 covered vertices 任意指派給一個可行 label，得 classes

[  
C_m={q\in P:a_q\equiv m\pmod q},\qquad m\in\mathcal L.  
]

uncovered vertices 暫列入 exception set (U)，大小

[  
|U|\ll\sqrt\eta N.  
]

---

## 3. 若有 dominant label

若存在 (m_*\in\mathcal L) 使

[  
|C_{m_*}|\ge (1-\rho)N,  
]

則 (m_*) 是 dominant label。  
例外集合為

[  
D=P\setminus C_{m_*}.  
]

對每個 (q\in D)，若

[  
a_q\not\equiv m_*\pmod q,  
]

則與 majority set

[  
G=C_{m_*}  
]

之間的邊給能量。因 (G) 是 (1-\rho) dense subset of (P)，Irving-good estimate 對刪去少量點仍保留：

[  
\sum_{p\in G}  
\left|  
\frac{(a_q-m_*)\bar p}{q}  
\right|^2  
\gg |G|\asymp N.  
]

因此每個真正例外 vertex 成本

[  
\gg N  
]

而其 residue 熵只有

[  
\ll \log X.  
]

因

[  
N\asymp X/\log X\gg \log X,  
]

例外集合與例外 residues 的總熵可由能量吸收：

[  
#{\text{exceptions of total cost }\le R}  
\le e^{\varepsilon R}.  
]

dominant label 的選擇數至多

[  
|\mathcal L|  
]

for fixed (p_0)，但若不固定 (p_0)，整體 label (m_*) 必滿足

[  
|m_*|\le B.  
]

故總 label 數

[  
\ll B\asymp X\log X\sqrt R  
\asymp \frac{\sqrt R}{\sigma_P}.  
]

所以 dominant 情形貢獻

[  
\le e^{\varepsilon R}\frac{\sqrt R}{\sigma_P}.  
]

---

## 4. 若無 dominant label

假設沒有 label 覆蓋 (1-\rho) 比例 vertices。

令 covered part 為

[  
V_{\mathrm{cov}}=\bigcup_{m\in\mathcal L}C_m.  
]

因 (p_0) regular，

[  
|V_{\mathrm{cov}}|\ge (1-O(\sqrt\eta))N.  
]

無 dominant 意味著

[  
\max_m |C_m|<(1-\rho)N.  
]

故 cross mass

# [  
\sum_{m\ne m'}|C_m||C_{m'}|

|V_{\mathrm{cov}}|^2-\sum_m|C_m|^2  
\gg_\rho N^2.  
]

我們要證這導致 (Q_P(a)) 過大，除非這些 non-dominant vertices 本身可作 energy-paid exceptions。

---

## 5. Cross-label divisor-energy lower bound

對兩個不同 labels

[  
m\ne m'  
]

與 classes

[  
A=C_m,\qquad B'=C_{m'},  
]

令 (H_{pq}^{m,m'}) 為滿足

[  
H_{pq}^{m,m'}\equiv m\pmod p,\qquad  
H_{pq}^{m,m'}\equiv m'\pmod q  
]

的對稱代表。

前面證明的 divisor incidence 給：若

[  
M=|A||B'|  
]

且 (A,B') 不太小，則

# [  
E(A,B';m,m')

\sum_{p\in A,q\in B'}  
\left(\frac{H_{pq}^{m,m'}}{pq}\right)^2  
\gg  
M\min\left(  
1,\frac{M^2}{X^4L^4}  
\right),  
]

其中

[  
L=1+\frac{\log(B+X^2+2)}{\log X}  
]

是 polylog 級量。

對 substantial classes，此下界非常大。

---

## 6. Dyadic decomposition：排除多個 substantial classes

將 classes 按大小 dyadic 分組：

[  
2^j\le |C_m|<2^{j+1}.  
]

若有兩組 substantial classes，其總頂點數分別為正比例級，則 cross pairs 數至少 (\gg N^2)，並且其中相當多 pair 的 class product (M) 不小。Cross-label divisor-energy lemma 給出

[  
Q_P(a)\gg \frac{N^2}{(\log X)^C}.  
]

而

[  
\frac{N^2}{(\log X)^C}  
\asymp  
\frac{X^2}{(\log X)^{C+2}}  
]

遠大於任何需要保留的 entropy scale；特別是對 (Q_P(a)\le R)，若 (R) 小於此量，矛盾。若 (R) 大到這個量級，則 (e^{-cR}) 在 partition sum 中已足以壓過 block 的全部 residue entropy (e^{O(X)})，因為

[  
\frac{X^2}{(\log X)^{C+2}}\gg X.  
]

所以 substantial multi-label 情形可全部吸收到 (e^{\varepsilon R}) 的 energy-paid factor 中。

若沒有 substantial competing classes，則除某個最大 class外，其餘 vertices總數 (t) 較小。這些 vertices當作 exceptions。與最大 class 的 cross / Irving energy給每個 exception 至少

[  
\gg N/(\log X)^C  
]

成本，仍遠大於 (\log X+\log|\mathcal L|)。因此其 entropy 也由能量支付。

所以無 dominant label 的所有情形，總計亦可吸收到

[  
e^{\varepsilon R}  
]

因子中，不額外產生 label-scale 以上的自由度。

---

## 7. Label 數量

最後，dominant label (m) 的絕對值滿足

[  
|m|\le B=A X\log X\sqrt R.  
]

故可選 label 數

[  
\ll B+1.  
]

由

[  
\sigma_P^{-1}\asymp X\log X  
]

得

[  
B+1\ll 1+\frac{\sqrt R}{\sigma_P}.  
]

---

# 8. 結論

所以

[  
#{a:Q_P(a)\le R}  
\le  
e^{\varepsilon R}  
\left(  
1+\frac{\sqrt R}{\sigma_P}  
\right).  
]

這就是 single-block counting theorem。

# 9. 評估

這個證明草案的主要估計都已明確：

1. good graph by Markov；
    
2. base-list reduction；
    
3. cross-label divisor-energy；
    
4. dyadic class decomposition；
    
5. exceptions paid by majority Irving；
    
6. dominant label count.
