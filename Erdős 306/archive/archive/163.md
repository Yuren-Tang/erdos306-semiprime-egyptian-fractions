不能說「已經證完」。更準確地說：

[  
\boxed{  
\text{我們已經找到了看起來能解核心困難的主機制，並把原問題歸約到幾個明確局部引理。}  
}  
]

這比「只是有想法」強很多；但還不到「只剩排版」的程度。原因是有幾個引理雖然輪廓可信，仍需嚴格逐行證明，尤其是：

1. **single-block counting theorem**  
    這是核心。它依賴 base-list reduction、cross-label divisor-energy、無 dominant label 時的 dyadic bookkeeping。這些都很有希望，但必須正式寫完。
    
2. **cross-label divisor-energy lemma**  
    小 representative 計數  
    [  
    N(T)\ll T L^2+(|A|+|B|)L  
    ]  
    已很清楚；但從它到「能量支付所有多 label 熵」仍要小心處理小類、special terms、dyadic 分層。
    
3. **ordinary diagonal counting / wrap-around**  
    現在已不是原來那種巨大 CRT 熵問題，但仍需正式證明  
    [  
    #{m\bmod L:Q_{\mathrm{diag}}(m)\le R}  
    \ll e^{\varepsilon R}\frac{\sqrt R}{\sigma}.  
    ]
    
4. **Irving-good pruning 的 uniform 化**  
    需要明確說明如何從 Irving 的平均估計刪去 bad primes，使所有需要的 block pair、所有非零 (d) 都有足夠好的下界。
    

所以判斷應是：

[  
\boxed{  
\text{核心障礙似乎已突破；但完整證明尚未完成。下面要做的是把幾個局部引理嚴格寫實。}  
}  
]

這其實是很好的狀態。若接下來寫嚴格證明時沒有新反例冒出，整條路線就可成為完整 proof attempt。

我們下一步應從最關鍵處開始，不再泛談全局，而是正式寫第一個核心引理：

[  
\boxed{\text{Cross-label divisor-energy lemma}}  
]

因為它是 single-block counting theorem 的心臟。先把它做成可引用的乾淨命題。

---

# Cross-label divisor-energy lemma：正式命題

令

[  
P\subset[X,2X]  
]

為一組 primes。令

[  
A,B\subset P.  
]

取不同整數

[  
m\ne m',\qquad |m|,|m'|\le B_0.  
]

對 (p\in A,q\in B)，令 (H_{pq}) 是對稱代表：

[  
H_{pq}\equiv m\pmod p,\qquad  
H_{pq}\equiv m'\pmod q,\qquad |H_{pq}|\le pq/2.  
]

設

[  
M=|A||B|.  
]

令

[  
L_X=1+\frac{\log(B_0+X^2+2)}{\log X}.  
]

若

[  
\min(|A|,|B|)\ge C L_X,  
]

則

[  
\sum_{p\in A,q\in B}  
\left(\frac{H_{pq}}{pq}\right)^2  
\gg  
M\min\left(  
1,\frac{M^2}{X^4L_X^4}  
\right).  
]

若 (\min(|A|,|B|)<CL_X)，則稱該側為 tiny class；在 single-block counting 中作例外處理。

---

# 證明

對 (T\le X^2/2)，定義

[  
N(T)=#{(p,q)\in A\times B:|H_{pq}|\le T}.  
]

若 (|H_{pq}|\le T)，則存在 (n) with

[  
|n|\le T,\qquad n\equiv m\pmod p,\qquad n\equiv m'\pmod q.  
]

等價於

[  
p\mid n-m,\qquad q\mid n-m'.  
]

因此

[  
N(T)  
\le  
\sum_{|n|\le T}\omega_A(n-m)\omega_B(n-m'),  
]

其中

[  
\omega_A(t)=#{p\in A:p\mid t}.  
]

若 (t\ne0)，則

[  
\omega_A(t)\le 1+\frac{\log |t|}{\log X}.  
]

當 (|n|\le T\le X^2/2) 且 (|m|,|m'|\le B_0) 時，

[  
|n-m|,\ |n-m'|\le B_0+X^2.  
]

故除 (n=m,m') 外，

[  
\omega_A(n-m),\omega_B(n-m')\ll L_X.  
]

所以 ordinary terms 給

[  
\ll T L_X^2.  
]

special terms：

若 (n=m)，則 (p\mid0) 自動，但需

[  
q\mid m-m'.  
]

因此 contribution 至多

[  
|A|\omega_B(m-m').  
]

同理 (n=m') contribution 至多

[  
|B|\omega_A(m-m').  
]

而

[  
0<|m-m'|\le2B_0,  
]

所以

[  
\omega_A(m-m'),\omega_B(m-m')\ll L_X.  
]

故

[  
N(T)\ll T L_X^2+(|A|+|B|)L_X.  
]

若

[  
\min(|A|,|B|)\ge C L_X,  
]

則

[  
(|A|+|B|)L_X\le \frac{M}{10}  
]

取 (C) 足夠大即可。

現在取

[  
T_*=  
c\min\left(\frac{M}{L_X^2},X^2\right)  
]

其中 (c>0) 足夠小。則

[  
N(T_*)\le \frac M2.  
]

所以至少 (M/2) 對 ((p,q)) 滿足

[  
|H_{pq}|>T_*.  
]

因此

[  
\sum_{p,q}H_{pq}^2  
\ge  
\frac M2T_*^2.  
]

又 (pq\asymp X^2)，故

[  
\sum_{p,q}  
\left(\frac{H_{pq}}{pq}\right)^2  
\gg  
\frac{M T_*^2}{X^4}.  
]

代入 (T_*)，得

[  
\sum_{p,q}  
\left(\frac{H_{pq}}{pq}\right)^2  
\gg  
M\min\left(  
1,\frac{M^2}{X^4L_X^4}  
\right).  
]

證畢。
