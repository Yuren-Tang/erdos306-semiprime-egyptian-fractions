# 1. AP-list agreement lemma：精確問題

令

[  
P\subset[X,2X]  
]

為 primes。對每個

[  
p\in P  
]

給 residue

[  
a_p\in\mathbb F_p.  
]

固定窗口

[  
[-B,B].  
]

定義 arithmetic-progression list：

[  
L_p={n\in[-B,B]\cap\mathbb Z:n\equiv a_p\pmod p}.  
]

邊 (pq) good 當且僅當

[  
L_p\cap L_q\ne\varnothing.  
]

也等價於 CRT representative

[  
H_{pq}(a)  
]

滿足

[  
|H_{pq}(a)|\le B.  
]

假設 good graph 很稠密：

[  
#{p<q:L_p\cap L_q\ne\varnothing}  
\ge (1-\eta)\binom N2.  
]

我們想證存在一個整數

[  
m\in[-B,B]  
]

使

[  
m\in L_p  
]

對大多數 (p\in P) 成立。

即：

[  
a_p\equiv m\pmod p  
]

對大多數 primes 成立。

---

# 2. 這個命題若對任意 (B) 可能太強

如果 (B) 非常大，比如接近

[  
\prod_{p\in P}p,  
]

那麼每個 (L_p) 很長，任意兩個 lists 幾乎都相交，但不一定有共同交點。  
所以必須利用我們的 (B) 範圍。

在 single-block 中：

[  
B\asymp X\log X\sqrt R.  
]

對主要能量範圍，(B) 遠小於 full CRT modulus product。通常也小於 (X^2) 或至多 (X^{2-\epsilon}) 在中低能層。高能層需另用粗 bound 或 energy tail處理。

所以合理版本應是分層的：

[  
B\le X^{2-\kappa}  
]

時，AP-list agreement 很強；  
(B) 更大時，可能需要更高階 common-neighbour 或多 base vertices。

---

# 3. (B<X^2/10) 時有簡化

若

[  
B<X^2/10,  
]

則對任意兩個 primes (p,q\sim X)，兩個 lists

[  
L_p,L_q  
]

的交集至多一個點。因為若 (n,n'\in L_p\cap L_q)，則

[  
pq\mid n-n',  
]

但

[  
|n-n'|\le2B<pq.  
]

所以

[  
n=n'.  
]

因此 good edge (pq) 有唯一 label

[  
h_{pq}\in[-B,B].  
]

這大大簡化：good graph 是一個 edge-labelled graph，且在 vertex (p) 上所有 incident labels都屬於同一 residue class mod (p)。

---

# 4. Dense good graph + edge labels

若 good graph 稠密，選 regular base vertex (p_0)。  
對大多數 (p)，邊 (p_0p) good，令

[  
m_p=h_{p_0p}.  
]

則

[  
m_p\in L_{p_0}\cap L_p.  
]

若對大多數 (p) 有同一個 (m_p=m)，完成。

假設不是。則 base-neighbours 被分到許多不同 (m\in L_{p_0})。

現在取兩個 vertices (p,q) 使

[  
m_p\ne m_q.  
]

若 (pq) good，則有唯一

[  
h_{pq}\in L_p\cap L_q.  
]

三角形 (p_0,p,q) 有三條 edge labels：

[  
m_p=h_{p_0p},\quad m_q=h_{p_0q},\quad h_{pq}.  
]

它們滿足：

[  
m_p\equiv m_q\pmod {p_0},  
]

因為都在 (L_{p_0})；

[  
m_p\equiv h_{pq}\pmod p;  
]

[  
m_q\equiv h_{pq}\pmod q.  
]

若 (m_p\ne m_q), 能否大量存在這種 triangle？

Let

[  
m_q-m_p = t p_0.  
]

Also

[  
h_{pq}=m_p+\alpha p  
]

and

[  
h_{pq}=m_q+\beta q.  
]

Thus

[  
\alpha p-\beta q=t p_0.  
]

All quantities bounded by (B), so

[  
|\alpha|,|\beta|,|t|\ll B/X.  
]

This is a linear equation in primes:

[  
\alpha p-\beta q=t p_0.  
]

For fixed (m_p,m_q), equivalently fixed (t), how many pairs (p,q) can satisfy with small (\alpha,\beta)?  
Since

[  
p,q,p_0\sim X,  
]

and

[  
|\alpha|,|\beta|,|t|\ll B/X.  
]

If (B<X^{2-\kappa}), then coefficients are (\ll X^{1-\kappa}). There may be many solutions, but maybe not dense.

This becomes an incidence equation. But unlike before, it uses base congruence and may be tractable.

---

# 5. Count cross good edges between two base labels

Fix two distinct base labels

[  
m,m'\in L_{p_0}.  
]

Let

[  
A_m={p:h_{p_0p}=m},  
\quad  
A_{m'}={q:h_{p_0q}=m'}.  
]

For (p\in A_m,q\in A_{m'}), edge (pq) good iff there exists (h\in[-B,B]) with

[  
h\equiv m\pmod p,\qquad h\equiv m'\pmod q.  
]

This is exactly cross-label incidence with labels (m,m').  
But now (A_m,A_{m'}) are defined by base-edge labels. They might still be arbitrary subsets, but if both large, cross-label divisor-energy lemma says many cross edges cannot all have small h. More importantly, good graph density requires most cross pairs to be good. If cross-label incidence bound says number of good cross pairs is small, then two large base classes cannot coexist.

So instead of energy, use **good edge count** at threshold (B).

Our divisor incidence bound gives, for (A,B'):

[  
N(B)  
\ll  
B L_X^2+(|A|+|B'|)L_X.  
]

This counts good cross edges between labels (m,m').

If good graph density says cross edges between (A_m,A_{m'}) are mostly good, then

[  
|A_m||A_{m'}|  
\lesssim  
B L_X^2+(|A_m|+|A_{m'}|)L_X.  
]

Thus unless one class is small,

[  
|A_m||A_{m'}|\ll B L_X^2.  
]

This is much stronger than energy lower bound.

Now if two base classes both have size (\gg \sqrt B,L_X), they cannot have dense good edges between them.

Since whole good graph is dense, if no dominant base class, many cross edges between sizable classes would be missing, contradiction.

This is promising.

# 6. Agreement via missing edges

Let base classes (A_m) partition most vertices adjacent to (p_0).  
Suppose no dominant class. Then cross pairs between different (A_m)'s are numerous:

[  
\sum_{m\ne m'}|A_m||A_{m'}|\gg N^2.  
]

But total number of good cross edges between different labels is bounded by summing incidence bounds:

[  
\sum_{m\ne m'}N_{m,m'}(B).  
]

For each pair:

[  
N_{m,m'}(B)  
\ll  
B L_X^2+(|A_m|+|A_{m'}|)L_X.  
]

Naively summing over all label pairs gives (s^2B), too large. But number of labels (s\sim B/X), so (s^2B\sim B^3/X^2). For (B\sim X\log X\sqrt R), this is (X(\log X)^3R^{3/2}). Could be less than (N^2\sim X^2/\log^2X) for (R\ll X^{2/3}), but not all.

Need refine summation by class sizes: many labels may have empty/small classes. The term (B L_X^2) per pair is wasteful for tiny classes.

Use bound:

[  
N_{m,m'}(B)  
\le |A_m||A_{m'}|  
]

also. Thus

[  
N_{m,m'}(B)  
\ll  
\min\left(|A_m||A_{m'}|,  
B L_X^2+(|A_m|+|A_{m'}|)L_X\right).  
]

Can we sum this and show if cross mass large, missing edges large unless one class dominant? Need inequality depending on (s,B,N).

Worst case: classes of size about (\sqrt B). Then per pair bound comparable to product, so cross edges could all be good under this estimate. Number of such classes (N/\sqrt B); cross mass (N^2). Estimate doesn't force missing. Need higher-order common-neighbour constraints.

Thus pairwise incidence still not enough in regime class size (\le\sqrt B). This corresponds many medium classes.

# 7. Use triangles/common neighbours

Dense good graph gives not only cross edges, but many common neighbours. If many base classes, take p in A_m, q in A_m'. They have many common neighbours r. Each common neighbour imposes additional congruence. Multiple common neighbours should force (m=m') when product of their primes exceeds B. This was our early multi-overlap idea.

Now we can formalize: if p and q are in different base labels, and they have r common neighbours with edges pr, qr good, then for each such r there are labels h_{pr},h_{qr}. Since both share (r), h_{pr}\equiv h_{qr}\pmod r. These constraints may imply something about m,m'.

Need connect h_{pr} to base label m via p:  
[  
h_{pr}\equiv m\pmod p  
]  
and h_{qr}\equiv m'\pmod q.  
Not same difference.

But maybe with many r, cross edge pq good plus triangles force m,m' linked. Complex.

This is genuine agreement theorem territory.
