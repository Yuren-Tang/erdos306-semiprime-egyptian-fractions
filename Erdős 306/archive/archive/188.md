# 1. 重新設計 single-block 的 local decoder

原來 single-block 中，我們只選一個 base vertex (p_0)，得到 base list：

[  
L_{p_0}.  
]

問題是每個 (p) 可能與 (L_{p_0}) 有多個交點，且不同 vertices 可通過第三 labels 相交，導致 agreement bookkeeping 複雜。

現在改成選一個小 anchor set：

[  
A={r_1,\dots,r_t}\subset P.  
]

我們希望 (A) 本身形成一個 good clique，並且大多數 vertices (p\in P) 與所有 anchors 都 good-connected。

對每個 vertex (p)，若它與所有 anchors 都 good，則對每個 (i) 有

[  
L_p\cap L_{r_i}\ne\varnothing.  
]

我們想從這些 intersections 中解出唯一 ordinary label。

---

# 2. 直接「共同交點」太強

自然想定義：

[  
L_A=\bigcap_{i=1}^t L_{r_i}.  
]

若

[  
m\in L_A\cap L_p,  
]

則 (p) 被 anchor label (m) 解釋。

但是注意：即使 (p) 與每個 (r_i) 分別 good，也不代表

[  
L_p\cap \bigcap_i L_{r_i}\ne\varnothing.  
]

也就是 pairwise intersections 不推出 common intersection。  
這正是 Helly 問題。

所以 anchor method 不能只靠「與 anchors 分別 good」；還要處理 pairwise intersections 是否 glue 成共同 label。

---

# 3. 但 anchors 可以提供一個 consistency test

假設 anchors 本身有共同 bounded label

[  
m_A\in[-B,B]  
]

滿足

[  
m_A\equiv a_{r_i}\pmod {r_i}  
\qquad\forall i.  
]

如果一個 vertex (p) 不服從 (m_A)，即

[  
a_p\not\equiv m_A\pmod p,  
]

那麼它與 anchor (r_i) 的 good edge gives a bounded representative

[  
h_i\in L_p\cap L_{r_i}.  
]

此時

[  
h_i\equiv a_p\pmod p,  
\qquad  
h_i\equiv m_A\pmod{r_i}.  
]

與 (m_A) 比較：

[  
h_i-m_A\equiv0\pmod{r_i},  
]

[  
h_i-m_A\equiv a_p-m_A\pmod p.  
]

令

[  
d=a_p-m_A\not\equiv0\pmod p.  
]

則對每個 anchor (r_i)，存在

[  
h_i=m_A+\lambda_i r_i  
]

且

[  
\lambda_i r_i\equiv d\pmod p,  
]

with

[  
|\lambda_i|\ll B/X.  
]

所以

[  
\lambda_i r_i\equiv d\pmod p  
]

for many (r_i).

若有兩個 anchors (r_i,r_j)，則

[  
\lambda_i r_i\equiv \lambda_j r_j\pmod p.  
]

因此

[  
p\mid \lambda_i r_i-\lambda_j r_j.  
]

這是強約束。若 anchors數 (t) 固定，對一個 wrong vertex p 仍可能存在；但若 (p) 與很多 anchors good，則有多個 divisibility constraints。

---

# 4. 用多 anchors 讓 wrong vertex 罕見

設 anchor set (A) 大小不是固定 2 或 3，而是

[  
t=t_X  
]

比如 (C\log X) 或小冪？  
若 (p) 不服從 (m_A)，則對每個 (r\in A) 需存在

[  
\lambda_r,\quad |\lambda_r|\le K:=B/X,  
]

使

[  
\lambda_r r\equiv d\pmod p.  
]

也就是

[  
\lambda_r\equiv d, r^{-1}\pmod p.  
]

Since (|\lambda_r|\le K). If (K\ll p), for each (r) there is at most one such small (\lambda_r). The probability that (d r^{-1}\mod p) lands in ([-K,K]) is about (K/p).

For many random anchors, probability is ((K/p)^t). Thus wrong vertices become impossible if anchors random and (t) large.

This suggests a probabilistic anchor set can decode robustly.

But our proof needs deterministic existence. We can select anchors randomly from the good block; by averaging, with high probability it will have the desired property for all p,d. This is like sampling test.

# 5. Anchor sampling lemma

We need a lemma:

For (P\subset[X,2X]), choose a random anchor set (A) of size (t). With high probability, for every (p\in P) and every (d\ne0\pmod p), not too many (r\in A) satisfy

[  
\left|\frac{d\bar r}{p}\right|\le K/p  
]

or equivalently there exists (|\lambda|\le K) with

[  
\lambda r\equiv d\pmod p.  
]

But this follows from Irving-good dispersion or even simple counting plus random sampling if for the full block

[  
#{r\in P: \exists |\lambda|\le K,\ \lambda r\equiv d\pmod p}  
]

is small.

For fixed p,d, and (\lambda), r is determined mod p:

[  
r\equiv d\lambda^{-1}\pmod p.  
]

Since (r\sim X) and modulus (p\sim X), for each (\lambda) there are (O(1)) candidates r. Therefore total candidates (\ll K).  
Full block size (N\sim X/\log X). Fraction (\ll K/N).

Here

[  
K=B/X=\log X\sqrt R.  
]

At raw entropy threshold (R\sim X), (K\sim \sqrt X\log X), while

[  
N\sim X/\log X,  
]

so

[  
K/N\sim \frac{\log^2X}{\sqrt X}\to0.  
]

Great. Thus a random anchor set of size (t=C\log N) will, for every p,d, have very few bad anchors.

Union over p,d: about (N\cdot X\sim X^2/\log X) pairs; with t~C logX and bad fraction (o(1)), concentration should work.

This is promising.

# 6. Revised local decoder with many anchors

Instead of relying on dense good graph alone, we choose anchors (A) such that:

1. Anchors are mutually good and share a common label (m_A).
    
2. Most vertices are good-connected to most anchors.
    
3. For any wrong residue (d\ne0), only a small fraction of anchors can have a bounded CRT glue with that wrong residue.
    

Then if a vertex (p) is good-connected to many anchors, it must satisfy

[  
a_p\equiv m_A\pmod p.  
]

Because if (a_p\ne m_A), only few anchors could be good to p, contradicting many good anchor connections.

This bypasses AP-list agreement entirely.

Key: Need anchors share common label (m_A). How to get such anchors?

We can pick anchors from a small good clique generated from dense good graph, but common label not automatic. However choose anchors greedily starting from an edge label (m_A): Let (p_0,p_1) good with label (m_A). Then choose anchors (r) that are good to both and satisfy the same (m_A) in their list. Need many such anchors; that's exactly dominant label problem again.

Alternative: choose a single base label (m_A) as one of (L_{p_0}); do we know many anchors satisfy it? Not yet.

So anchor set with common label is not available before solving agreement. Hm.

But maybe anchors need not share common label initially. We can use anchor residues as a codeword; for each candidate global m compatible with anchors. If no common m, then anchor set itself has high internal energy. But anchors are selected from low-energy block, so their internal edges good; still pairwise good not common.

Back to list agreement among anchors, but anchors size t=O(logX). We can brute force: if anchor set has all pairwise good but no common label, maybe probability small / energy? We need find an anchor clique with common label. Dense graph only gives good clique, not common intersection.

Could choose anchor set of size 2: their common label m_A exists by good edge. Then use many anchors? Not common.

Maybe use base pair (p0,p1) with m_A. Use random test set A of additional anchors not necessarily sharing m_A. To test a vertex p wrong relative to m_A, we only need edge p-r? But r not labelled by m_A, so good edge p-r can occur for other reasons. Need r itself also obey m_A.

Thus need a pool of anchors obeying m_A, equivalent to dominant label. circular.
