# 1. 新核心命題：single-block CRT spectral gap

我們應把 single-block theorem 改寫成：

**Single-block CRT Poincaré theorem.**  
令

[  
P\subset[X,2X],  
]

對 residues (a_p\in\mathbb F_p)，定義

[  
Q_P(a)=  
\sum_{p<q}  
\left(\frac{H_{pq}(a)}{pq}\right)^2.  
]

則 partition function 滿足

[  
\sum_{a\in\prod_p\mathbb F_p}e^{-cQ_P(a)}  
\ll  
\sum_{m\in\mathbb Z} e^{-c' m^2\sigma_P^2}  
\asymp \frac1{\sigma_P}.  
]

這比 counting theorem 更直接，也更符合 Fourier minor arc。

它說：所有非 diagonal states 雖然可能 pairwise good，但其 quadratic energy 有足夠 gap；所以 partition function 仍由 diagonal ordinary labels 控制。

# 2. 如何證 spectral gap？

從小例子看，edge labels (H_{pq}) 是關鍵。  
它們滿足 triangle congruence：

[  
H_{pq}\equiv H_{pr}\pmod p,  
]

[  
H_{pq}\equiv H_{qr}\pmod q,  
]

[  
H_{pr}\equiv H_{qr}\pmod r.  
]

非 diagonal 態就是這個 edge-label system 的非平凡 cocycle。  
平方能量是

[  
\sum_{p<q}H_{pq}^2/X^4.  
]

所以我們需要一個「CRT edge-label Hodge/Poincaré inequality」：

[  
\sum_{p<q}H_{pq}^2  
]

的低能模態只有常數 edge labels

[  
H_{pq}=m.  
]

非 constant cocycles有 gap。

# 3. 一個可能的證明路線：最小邊標籤法

取一個 assignment (a)，選一條 edge ((p_0,q_0))，使

[  
|H_{p_0q_0}|  
]

接近 edge labels 的 median 或最小能量中心，稱其為 (m)。

對任意 (r)，比較三角形

[  
p_0,q_0,r.  
]

已知：

[  
H_{p_0r}\equiv H_{p_0q_0}\pmod {p_0},  
]

[  
H_{q_0r}\equiv H_{p_0q_0}\pmod {q_0},  
]

[  
H_{p_0r}\equiv H_{q_0r}\pmod r.  
]

若

[  
H_{p_0r}\ne m  
]

則

[  
H_{p_0r}-m  
]

是非零 (p_0)-倍數，因此大小至少 (\asymp X)。  
同理對 (q_0r)。

所以每個不服從 (m) 的 vertex (r) 會在 incident edges 上造成至少 (X^2) 級別的 unnormalised energy。

更重要的是，若很多 vertices 有不同 deviations，則它們之間的 edges再造成二次放大。

這可望導出：

[  
Q_P(a)\ge Q_P^{\mathrm{diag}}(m)+c\cdot \frac{tN}{X^2}  
]

或類似式子，其中 (t) 是不服從 (m) 的 vertices 數。  
這正是 exceptions energy payment 的 spectral 版本。

# 4. 為什麼比 list agreement 更好？

小反例中三個 lists pairwise intersect but no common label；可是 edge labels 是：

[  
10,\quad0,\quad-11.  
]

它們不是一個常數，而是相差 (\asymp X)。  
在大 block 中，如果這種 defect 延伸到很多 edges，平方能量會累積。

所以我們不需要禁止 defect；只需證 defect 的數量與幅度不能在低能 partition 裏產生太多 entropy。

# 5. 下一步應寫的 lemma

我建議下一步攻下面這個更具體的命題：

**Base-edge spectral decoding lemma.**  
選一條 regular edge (p_0q_0)，令

[  
m=H_{p_0q_0}.  
]

對任意 vertex (r)，若

[  
a_r\not\equiv m\pmod r,  
]

則三角形 (p_0,q_0,r) 迫使

[  
|H_{p_0r}-m|\ge X  
]

或

[  
|H_{q_0r}-m|\ge X.  
]

因此 (r) 在兩條 anchor edges 上造成 unnormalised energy至少

[  
\gg X^2.  
]

如果 (r) 多，總 energy至少

[  
\gg tX^2.  
]

這本身 normalized 為

[  
\gg t/X^2.  
]

還不夠支付 (t\log X)，但它只是 anchor-edge cost。  
接着利用 (r) 與所有 good decoded vertices 的 edges，Irving dispersion 可把每個 wrong vertex成本放大到

[  
\gg N  
]

normalized scale中足以支付 entropy。

也就是：

1. base edge gives candidate (m);
    
2. anchor triangle detects wrong vertices;
    
3. majority decoded set + Irving gives full cost per wrong vertex.
    

這可能把之前的 agreement problem改成 spectral decoding + Irving correction。
