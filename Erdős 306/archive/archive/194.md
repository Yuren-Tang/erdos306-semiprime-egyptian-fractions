好。現在正式推導 **base-edge spectral decoding lemma**，並檢查它是否真能接上 Irving amplification。

---

# 1. 設定

令

[  
P\subset[X,2X],\qquad N=|P|\asymp X/\log X.  
]

對每個

[  
p\in P  
]

有 residue

[  
a_p\in\mathbb F_p.  
]

對每條邊 (pq)，令

[  
H_{pq}\in(-pq/2,pq/2]  
]

滿足

[  
H_{pq}\equiv a_p\pmod p,\qquad  
H_{pq}\equiv a_q\pmod q.  
]

single-block energy 為

[  
Q_P(a)=\sum_{p<q}\left(\frac{H_{pq}}{pq}\right)^2.  
]

為簡化量級，因 (pq\asymp X^2)，先看未歸一化能量：

[  
\mathcal E_P(a)=\sum_{p<q}H_{pq}^2.  
]

則

[  
Q_P(a)\asymp \frac{\mathcal E_P(a)}{X^4}.  
]

---

# 2. 選 base edge

取一條邊

[  
p_0q_0  
]

令

[  
m=H_{p_0q_0}.  
]

這個 (m) 是候選 ordinary label。

若一個 vertex (r) 滿足

[  
a_r\equiv m\pmod r,  
]

則稱 (r) 服從 (m)。

若

[  
a_r\not\equiv m\pmod r,  
]

令

[  
d_r=a_r-m\in\mathbb F_r^\times.  
]

我們希望證：這個 (r) 會在與大量 vertices 的 edges 上付出能量。

---

# 3. 僅靠 base triangle 的能量太小

看三角形

[  
p_0,q_0,r.  
]

有

[  
H_{p_0r}\equiv m\pmod {p_0},  
]

因為

[  
H_{p_0r}\equiv a_{p_0}\pmod {p_0},  
\quad  
m=H_{p_0q_0}\equiv a_{p_0}\pmod {p_0}.  
]

所以

[  
H_{p_0r}=m+\alpha p_0.  
]

同理

[  
H_{q_0r}=m+\beta q_0.  
]

又兩者都 congruent to (a_r\pmod r)，所以

[  
m+\alpha p_0\equiv m+\beta q_0\equiv a_r\pmod r.  
]

若 (a_r\not\equiv m\pmod r)，則

[  
\alpha p_0\not\equiv0\pmod r,  
]

所以 (\alpha\ne0)。因此

[  
|H_{p_0r}-m|\ge p_0\asymp X.  
]

這說明 wrong vertex 會讓至少一條 anchor edge label 距 (m) 很遠。

但這只給 unnormalised cost (X^2)，歸一化後是

[  
X^2/X^4=1/X^2,  
]

太小，不能支付一個 vertex 的 residue entropy (\log X)。

所以 base triangle 只能 detect，不能 pay。

---

# 4. 需要 majority set amplification

如果已經有一大批 vertices (G) 服從 (m)，即

[  
a_p\equiv m\pmod p,\qquad p\in G,  
]

那麼 wrong vertex (r) 與 (G) 的所有 edges 有相位：

# [  
\phi_{pr}

\frac{m}{pr}  
+  
\frac{(a_r-m)\bar p}{r}  
\pmod1.  
]

由 Irving-good dispersion，

[  
\sum_{p\in G}  
\left|  
\frac{(a_r-m)\bar p}{r}  
\right|^2  
\gg |G|,  
]

provided (G) dense enough in a good block, or at least after pruning if (G) is majority.

再用

[  
|x+y|^2+|x|^2\gg|y|^2  
]

得

[  
\sum_{p\in G}|\phi_{pr}|^2  
+  
\sum_{p\in G}\left|\frac{m}{pr}\right|^2  
\gg |G|.  
]

所以每個 wrong vertex 成本

[  
\gg |G|.  
]

這足以支付 entropy。

但關鍵問題是：我們如何先得到大 (G)？

這又是 agreement 問題。

---

# 5. Base edge alone 不足以產生 (G)

選 base edge (p_0q_0) 後，定義

[  
G_m={r:a_r\equiv m\pmod r}.  
]

我們要證 (G_m) 大。  
但 base triangle 只說：若 (r\notin G_m)，則 anchor edges differ from (m) by multiples of (X)。這本身不貴。許多 vertices 可以各自 wrong，而總 anchor-edge cost仍小：

[  
t\cdot \frac1{X^2}  
]

歸一化太小。

所以 base-edge spectral decoding **不能單獨建立 majority**。  
它必須與完整 graph energy結合。

---

# 6. 完整 graph 中 wrong vertices 彼此交互

設 wrong vertices 集合

[  
D=P\setminus G_m.  
]

如果 (D) 很大，則 wrong vertices 之間的 edges (rs) 也有 constraints。  
可能兩個 wrong vertices (r,s) share their own labels and form a coherent cocycle with low energy.  
小例子說明這種 cocycle可以局部存在；但 complete graph 大規模是否昂貴，正是問題。

所以我們需要證：

[  
D\text{ large}  
\quad\Rightarrow\quad  
\sum_{{r,s}\subset D\cup G_m}H_{rs}^2  
\text{ large}.  
]

這是 full spectral gap again.

# 7. 將問題線性代數化

也許可以將 residues (a_p) 映入一個共同實向量空間。

對每個 (p)，選代表

[  
x_p\in(-p/2,p/2]  
]

of (a_p).  
對 (p,q)，CRT representative (H_{pq}) roughly solves

[  
H_{pq}=x_p+p k_p=x_q+q k_q.  
]

若 (H_{pq}) 小，則 (x_p) 和 (x_q) are congruent through small integer.

Could define normalized variable

[  
y_p=\frac{a_p}{p}\in\mathbb R/\mathbb Z.  
]

Then edge phase for semiprime is roughly

[  
\phi_{pq}= a_p \frac{\bar q}{p}+a_q\frac{\bar p}{q}  
]

not simply (y_p-y_q).  
But maybe after multiplying by (pq), (H_{pq}) is a bilinear linear form in residues.

There is a linear map

[  
T:\prod_p\mathbb F_p\to \prod_{p<q}\mathbb Z/(pq)  
]

taking (a\mapsto H_{pq}).  
Energy is squared distance of (T(a)) to 0 in symmetric reps.  
Diagonal labels (m) are low modes.

We need a discrete Gaussian sum over this code. This resembles theta function of a lattice/code. Maybe use geometry of numbers / minimum distance of a CRT code.

# 8. Code viewpoint

The set of all edge labels ((H_{pq})) arising from residues is a code (C) inside

[  
\prod_{p<q}\mathbb Z/(pq).  
]

Diagonal labels form a one-dimensional subcode

[  
m\mapsto (m)_{p<q}.  
]

We need bound theta series:

[  
\sum_{c\in C}e^{-c|c|^2/X^4}  
]

by diagonal theta.

This is a coding theory / lattice theta problem.  
The spectral gap numerics suggest the code has large minimum distance away from diagonal.

Can we prove minimum distance? Need: if edge labels all small in (\ell^2), then codeword near diagonal.

This may follow from local testability of the direct product code. Indeed! The map from global integer (m) to residues (m\bmod p) is a Chinese remainder code; pair tests check consistency on pairs. Non-diagonal assignments are words far from the code; pair test rejection probability/energy lower bounded. This is exactly locally testable code / agreement theorem, but with weighted distance and bounded representatives.

So agreement theorem literature may be relevant after all, but as code testing, not simple Helly.

The code is a CRT code; pairwise tests are 2-query consistency tests. Need robust soundness. Could perhaps cite/derive known direct product test? But weights due bounded reps.
