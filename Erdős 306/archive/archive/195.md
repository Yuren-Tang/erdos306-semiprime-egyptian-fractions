[  
\mathbb Z/L\mathbb Z\simeq \prod_{p\in P}\mathbb F_p

的表示，無限同餘類有 Helly 性質，但 bounded window 破壞了直接 Helly。這些都支持我們的重新定位，但不能替我們證 single-block CRT Poincaré。([維基百科](https://en.wikipedia.org/wiki/Locally_testable_code?utm_source=chatgpt.com "Locally testable code"))

然後把這個 theorem 作為核心局部問題單獨研究。這不是放棄，而是把原來混在長證明中的真正難點抽出來。

## 1. 新核心命題

令

[  
P\subset[X,2X],\qquad N=|P|\asymp X/\log X.  
]

對

[  
a=(a_p)_{p\in P}\in\prod_{p\in P}\mathbb F_p  
]

定義

[  
H_{pq}(a)\in(-pq/2,pq/2]  
]

滿足

[  
H_{pq}(a)\equiv a_p\pmod p,\qquad  
H_{pq}(a)\equiv a_q\pmod q.  
]

定義 single-block energy：

[  
Q_P(a)=\sum_{p<q}  
\left(\frac{H_{pq}(a)}{pq}\right)^2.  
]

我們真正需要的是：

[  
\boxed{  
\sum_{a\in\prod_{p\in P}\mathbb F_p}  
e^{-cQ_P(a)}  
\ll  
\sum_{m\in\mathbb Z}e^{-c'm^2\sigma_P^2}  
\asymp \frac1{\sigma_P},  
}  
]

其中

[  
\sigma_P^2=\sum_{p<q}\frac1{p^2q^2}  
\asymp \frac1{X^2(\log X)^2}.  
]

這就是 **single-block CRT Poincaré theorem**。

它不再聲稱「pairwise good implies global label」，因為 bounded-window Helly 反例已經否定了那條路。它只聲稱：所有非 diagonal cocycle 的 quadratic energy 足以壓住其熵。

---

## 2. 這個命題為何合理？

小規模枚舉顯示最低能態確實是 ordinary diagonal labels：

[  
a_p\equiv m\pmod p,  
]

此時所有 edge labels

[  
H_{pq}=m.  
]

非 diagonal cocycle 雖可存在，但能量跳高。這表示真正機制不是 Helly agreement，而是某種「CRT edge-label code」的譜間隙。

從編碼觀點看，ordinary labels 是 CRT code 的碼字：

[  
m\longmapsto (m\bmod p)_{p\in P}.  
]

pair energy 測試的是兩個坐標是否有小 CRT glue。  
所以我們需要的是 CRT code 的一個 weighted local-testability / theta bound。一般 locally testable code 理論正是研究「局部測試拒絕概率是否控制到碼字距離」，但目前未見可直接套用到這個 weighted CRT setting 的定理。([維基百科](https://en.wikipedia.org/wiki/Locally_testable_code?utm_source=chatgpt.com "Locally testable code"))

## 3. 下一步有兩條路

### 路線 A：把全文整理成「conditional proof」

即：

> Assuming the single-block CRT Poincaré theorem, the squarefree denominator theorem follows.

這條路的好處是立刻能把全局 Fourier、Irving、mass tuning、lattice span 都整理成一份清晰的 proof manuscript。它會非常有價值，因為它明確說明：整個大問題只剩一個局部 CRT Poincaré theorem。

### 路線 B：繼續攻 single-block CRT Poincaré

這是更難但更本質的路。可能工具包括：

1. **edge-label Hodge / cocycle analysis**  
    研究三角形約束  
    [  
    H_{pq}\equiv H_{pr}\pmod p,  
    \quad  
    H_{pq}\equiv H_{qr}\pmod q,  
    \quad  
    H_{pr}\equiv H_{qr}\pmod r.  
    ]
    
2. **theta series / lattice-code viewpoint**  
    把所有 edge labels 看成 CRT code 的局部測試像，證非 diagonal codewords 的 Gaussian weight總和小。
    
3. **numerical spectral exploration**  
    先對小 prime blocks 計算低能態結構，猜測正確的 spectral inequality。
    

我建議先走 **路線 A**，把 conditional proof 整理出來；同時把 single-block CRT Poincaré theorem 作為「Main local theorem / remaining core lemma」獨立標出。這樣我們不會在局部泥潭裏反覆繞，而是保留已完成的大量全局成果。
