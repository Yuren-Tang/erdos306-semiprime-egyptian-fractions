好。現在我們從 conditional proof 回到唯一剩餘核心：

[  
\boxed{  
\text{single-block CRT Poincaré theorem}  
}  
]

它目前是整個方案的真正瓶頸。接下來要做的不是再談全局，而是把這個局部命題拆成更小的、可檢驗的層次。

---

# 1. 局部命題的三層版本

我們需要的最強形式是 theta / partition bound：

[  
\sum_{a\in\prod_{p\in P}\mathbb F_p}  
e^{-cQ_P(a)}  
\ll  
\frac1{\sigma_P}.  
]

但這個可以分成三層來攻。

## 第一層：minimum distance / spectral gap

證：

若 (a) 不是 ordinary diagonal label

[  
a_p\equiv m\pmod p,  
]

則

[  
Q_P(a)  
]

至少比最近的 diagonal 態大一個明確量。

這只是「非對角低能態有 gap」。

## 第二層：energy controls distance to diagonal

證存在 (m)，使 deviations from

[  
a_p\equiv m\pmod p  
]

的 vertices 由能量支付。形式上類似：

[  
Q_P(a)\ge Q_P^{\mathrm{diag}}(m)+c,\mathcal D(a,m),  
]

其中 (\mathcal D) 是某種到 diagonal code 的距離。

## 第三層：theta bound

由第二層推出：

[  
\sum_a e^{-cQ_P(a)}  
\ll  
\sum_m e^{-c'm^2\sigma_P^2}.  
]

這是 Fourier 小弧真正需要的形式。

目前小規模數值支持第一層；第二、三層才是完整證明所需。

# 2. 先攻第一層：minimum non-diagonal energy

我們可以把問題縮放一下。因 (p,q\sim X)，

[  
Q_P(a)\asymp \frac1{X^4}\sum_{p<q}H_{pq}^2.  
]

令

[  
\mathcal E(a)=\sum_{p<q}H_{pq}^2.  
]

Diagonal label (m) 給

[  
H_{pq}=m  
]

所以

[  
\mathcal E_{\mathrm{diag}}(m)=\binom N2 m^2.  
]

最低 diagonal 是 (m=0)，能量 (0)。  
要看非 diagonal minimum，就是：

# [  
\lambda_1(P)

\min_{a\not\equiv 0\ \mathrm{diagonal}}  
\sum_{p<q}H_{pq}^2.  
]

數值例子顯示 (\lambda_1(P)>0)，而且不小。

若能證

[  
\lambda_1(P)\gg N X^2  
]

或甚至

[  
\gg N^2  
]

之類，就能開始支付非 diagonal entropy。

---

# 3. 可能的最小非對角構造

要理解 gap，先構造低能非對角態。

最簡單：只改一個 prime (r) 的 residue。

令

[  
a_p=0\quad(p\ne r),  
\qquad  
a_r=1.  
]

那麼對不含 (r) 的邊：

[  
H_{pq}=0.  
]

對邊 (rp)，(H_{rp}) 是滿足

[  
H\equiv1\pmod r,  
\qquad  
H\equiv0\pmod p.  
]

寫

[  
H=kp.  
]

需要

[  
kp\equiv1\pmod r,  
]

所以 (k) 是 (p^{-1}\pmod r) 的小代表，大小通常 (\asymp r)。  
因此

[  
|H|\asymp X^2  
]

通常，而歸一化每條邊貢獻 (O(1))。  
有 (N) 條 incident edges，所以 energy (\gg N)。這很大，足以支付一個 residue 的 entropy。

但最小非 diagonal 可能不是 single-vertex defect；可能有 coordinated cocycles，讓每條 edge label只有 (O(X)) 而非 (O(X^2))。小例子中的 defect labels正是 (O(X))。這種態能量約：

# [  
\mathcal E\sim N^2X^2,  
\qquad  
Q\sim \frac{N^2X^2}{X^4}

\frac{N^2}{X^2}  
\asymp \frac1{\log^2 X}.  
]

這很小，不足以直接壓 entropy。  
但是否能在大 complete block 上維持 all edges (O(X)) without diagonal? 這是關鍵。

數值暗示不能大規模維持，或者其自由度很小。

# 4. Edge-label cocycle equations

對三角形 (p,q,r)，edge labels satisfy：

[  
H_{pq}\equiv H_{pr}\pmod p,  
]

[  
H_{pq}\equiv H_{qr}\pmod q,  
]

[  
H_{pr}\equiv H_{qr}\pmod r.  
]

若所有 (H_{pq}=O(X))，則差值如

[  
H_{pq}-H_{pr}  
]

是 (p)-倍數且大小 (O(X))，所以只能是

[  
0,\ \pm p,\ \pm2p,\ldots  
]

with very small coefficient.

Thus edge labels define small integer coefficients on oriented wedges:

[  
H_{pr}-H_{pq}=c_{p;qr}p,  
\qquad |c_{p;qr}|=O(1).  
]

For a large complete graph, these coefficients must satisfy many compatibility equations.  
A plausible rigidity theorem is:

If all edge labels are (O(X)), then either all equal one common (m), or the number of vertices is bounded by a constant depending on the coefficient bound.

This would explain why small 3-prime examples exist, but cannot extend to large (N).

This is an excellent target.

---

# 5. Bounded cocycle rigidity lemma

Formulate:

**Lemma.**  
Let (P\subset[X,2X]) and suppose edge labels (H_{pq}) satisfy the CRT triangle congruences. If

[  
|H_{pq}|\le C X  
]

for all (p<q), then for (N) sufficiently large depending on (C), all (H_{pq}) are equal.

More generally, if many edges satisfy (|H_{pq}|\le C X), then on a large subset the labels are constant.

This would be powerful. It says nontrivial bounded-window cocycles cannot live on large cliques.

The small example with moduli (5,7,11) is allowed because (N=3). The lemma would say such defects cannot scale to large blocks.

---

# 6. How to prove bounded cocycle rigidity?

Fix two vertices (p_0,q_0), set

[  
m=H_{p_0q_0}.  
]

For any (r),

[  
H_{p_0r}=m+\alpha_r p_0,  
]

[  
H_{q_0r}=m+\beta_r q_0.  
]

If (|H|\le CX), then

[  
|\alpha_r|,|\beta_r|=O_C(1).  
]

Also

[  
H_{p_0r}\equiv H_{q_0r}\pmod r,  
]

so

[  
\alpha_r p_0-\beta_r q_0\equiv0\pmod r.  
]

Since the left side has size (O_C X) and (r\sim X), this means

[  
\alpha_r p_0-\beta_r q_0=k_r r  
]

with

[  
|k_r|=O_C(1).  
]

For fixed integers ((\alpha,\beta,k)), the equation

[  
\alpha p_0-\beta q_0=k r  
]

determines (r).  
There are only (O_C(1)) triples ((\alpha,\beta,k)). Therefore only (O_C(1)) possible (r) with nonzero ((\alpha,\beta)).

Thus for all but (O_C(1)) vertices (r), we must have

[  
\alpha_r=\beta_r=0,  
]

so

[  
H_{p_0r}=H_{q_0r}=m.  
]

Then for two such vertices (r,s), using edge (rs) and congruences:

[  
H_{rs}\equiv m\pmod r,\qquad H_{rs}\equiv m\pmod s.  
]

If (|H_{rs}|\le CX<X^2/2), then

[  
H_{rs}=m.  
]

So all but (O_C(1)) vertices are diagonal with label (m).

This is excellent. This is the missing rigidity in bounded edge-label regime.

---

# 7. From bounded labels to energy partition

Now use energy layering.

If many edge labels are small, say (|H_{pq}|\le C X), bounded cocycle rigidity gives a large diagonal core.

Edges outside this bounded regime have (|H_{pq}|>CX), each contributes at least

[  
(CX)^2/X^4=C^2/X^2  
]

to normalized energy.

If many such edges exist, energy pays.

Thus one can prove:

Either a large subset is diagonal with label (m), or there are many edges with (|H_{pq}|>CX), giving energy.

Once a large diagonal core exists, wrong vertices are paid by Irving-majority correction as before.

This may finally close single-block theorem.

---

# 8. This is promising

This avoids false Helly because it does not say pairwise intersections imply common label; it says **uniformly small edge labels on a large clique** imply common label.  
The counterexample has labels (10,0,-11), all (O(X)), but clique size 3, allowed by (O_C(1)) exceptional vertices. It cannot proliferate to (N\to\infty).

This is exactly what we needed.
