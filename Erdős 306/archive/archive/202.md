# Partial single-block theorem

令

[  
P\subset[X,2X],\qquad N=|P|\asymp X/\log X.  
]

對

[  
a=(a_p)_{p\in P}  
]

定義

[  
H_{pq}(a)\in(-pq/2,pq/2]  
]

滿足

[  
H_{pq}\equiv a_p\pmod p,\qquad H_{pq}\equiv a_q\pmod q.  
]

令

[  
Q_P(a)=\sum_{p<q}\left(\frac{H_{pq}}{pq}\right)^2.  
]

我們目前可以證出如下低能版本：

[  
\boxed{  
Q_P(a)\le R,\quad R\ll X/\log^3X  
\quad\Longrightarrow\quad  
a \text{ 在大多數 primes 上接近某個 diagonal label }m.  
}  
]

更具體地，存在整數 (m)，使

[  
a_p\equiv m\pmod p  
]

對 (1-o(1)) 比例的 (p\in P) 成立；其餘例外 vertices 可由 Irving dispersion 支付。

---

# 1. 證明骨架

設

[  
T=KX.  
]

稱 edge (pq) 為 (T)-small 若

[  
|H_{pq}|\le T.  
]

若

[  
Q_P(a)\le R,  
]

則大多數邊在 cutoff

[  
T\asymp X\log X\sqrt R  
]

下是 (T)-small。此時

[  
K\asymp \log X\sqrt R.  
]

在 (T)-small graph 中選一條 regular base edge

[  
p_0q_0,  
]

令

[  
m=H_{p_0q_0}.  
]

對 common (T)-small neighbour (r)，有

[  
H_{p_0r}=m+\alpha_r p_0,  
]

[  
H_{q_0r}=m+\beta_r q_0,  
]

且

[  
|\alpha_r|,\ |\beta_r|\ll K.  
]

又二者同餘於 (a_r\pmod r)，所以

[  
r\mid \alpha_rp_0-\beta_rq_0.  
]

若

[  
(\alpha_r,\beta_r)=(0,0),  
]

則

[  
H_{p_0r}=H_{q_0r}=m,  
]

從而

[  
a_r\equiv m\pmod r.  
]

若 ((\alpha_r,\beta_r)\ne(0,0))，則 (r) 是某個非零整數

[  
\alpha p_0-\beta q_0,\qquad |\alpha|,|\beta|\ll K  
]

的 prime divisor。粗略計數給出 bad (r) 數至多

[  
O(K^2).  
]

因此 common small neighbourhood 中除

[  
O(K^2)  
]

個 vertices 外，皆服從同一 label (m)。

若

[  
K^2=o(N),  
]

也就是

[  
R\log^2X=o(X/\log X),  
]

即

[  
R\ll X/\log^3X,  
]

則得到一個密度 (1-o(1)) 的 diagonal core。

---

# 2. 例外 vertices 的支付

一旦有 dense core

[  
G={p:a_p\equiv m\pmod p},  
\qquad |G|=(1-o(1))N,  
]

對任意 wrong vertex (r\notin G)，令

[  
d=a_r-m\ne0\pmod r.  
]

由 Irving-good pruning，

[  
\sum_{p\in G}  
\left|  
\frac{d\bar p}{r}  
\right|^2  
\gg |G|\asymp N.  
]

因此 wrong vertex 與 core 的 edges 給出

[  
\gg N  
]

的能量成本。  
而描述一個 wrong vertex 的 entropy 只有

[  
O(\log X).  
]

所以所有例外可被能量支付。

---

# 3. 此 partial theorem 的意義

這已經解決了 single-block 的低能主區域。  
它也避開了 false Helly route，因為我們不再說「pairwise good implies common label」；我們說的是：

[  
\boxed{  
\text{大量 uniformly small edge labels around a base edge force a diagonal core.}  
}  
]

這是正確的，且由 bounded cocycle rigidity 支持。

---

# 4. 剩餘缺口

完整 single-block theta bound 還需處理

[  
X/\log^3X\lesssim R\lesssim X.  
]

這不是層變塔，而是一個明確的 **logarithmic middle-energy gap**。

它來自 bad-neighbour 粗界

[  
O(K^2).  
]

如果能將其改進為

[  
O(K^2/\log^A X)  
]

並取得足夠的 (A)，或者用另一種方法在中能區建立 diagonal core / entropy saving，就能封口。

---

# 5. 下一步最該攻什麼？

現在有三個合理方向，但我建議優先第一個：

## 方向一：改進 bad-neighbour 計數

研究

# [  
\mathcal B(p_0,q_0;K)

{r\sim X:\exists |\alpha|,|\beta|\le K,\ r\mid \alpha p_0-\beta q_0}.  
]

粗界：

[  
|\mathcal B|\ll K^2.  
]

希望證某種平均或典型 bound：

[  
|\mathcal B|  
\ll \frac{K^2}{\log^A X}+O(KX^\varepsilon).  
]

這是最直接的 log gap 問題。

## 方向二：多 base edge intersection

若能找到幾條 base edges 同一 label (m)，wrong vertex 必須同時落入多個 bad sets。  
這可能用 bounded rigidity先得小 core，再自舉。但它需要小心避免循環。

## 方向三：接受 conditional proof，單獨列出 local theorem

如果目標是整理研究稿，則現在可以把主結果寫成 conditional theorem，並把 single-block CRT Poincaré作為主要未解局部問題。

---

我建議下一步先攻方向一：**bad-neighbour divisor incidence 的 log saving**。這是具體、可計算、最不像塔的一步。
