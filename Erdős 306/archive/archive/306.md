我們現在應當做的事，不是硬闖 (\mu(\pi)=2)，而是把問題改造成一個「可推進而不自欺」的研究框架。換言之：承認主峰未登，但不要停在山腳；先把能證的山脊、等價命題、條件命題、反例邊界全部整理清楚。

我會選擇下面這條路：

**主線：研究 (P(K)) 的結構，而不再直接攻擊 π。**

[  
P(K):\quad  
\left|\pi-\frac pq\right|<q^{-K}  
\text{ 只有有限多解。}  
]

我們已知：

[  
K>7.103205334137\ldots \Rightarrow P(K).  
]

我們預期：

[  
K>2 \Rightarrow P(K).  
]

我們不能證：

[  
2<K\leq 7.103205334137\ldots  
\Rightarrow P(K).  
]

所以「我們要」做的第一件事，是把原命題分解成三種版本：

一、**無條件弱定理**：

[  
K>7.103205334137\ldots  
]

時成立。

二、**條件強定理**：

若假設 (\mu(\pi)=2)，則所有 (K>2) 成立。

三、**錯誤或過強版本**：

若要求存在 (C>0)，使

[  
\left|\pi-\frac pq\right|>\frac{C}{q^2}  
]

對所有 (p/q) 成立，那就不是 (\mu(\pi)=2)，而是要求 π 是 badly approximable；這比自然猜想強得多，不應作為目標。

這樣做的好處是：原問題不再是「我們能否證明 π 的正常性」，而變成一個清楚的判定問題：

> 你的原命題到底需要哪一級強度？

若只需要 (K>7.1032)，則可證。  
若需要任意 (K>2)，則等價於未解猜想。  
若需要 (C/q^2) 的統一下界，則方向很可能錯了。

接下來我會推進第二步：把 (P(K)) 與 (\mu(\pi)) 的關係證成完全等價，而不是只作口頭說明。

設

[  
\mu(\pi)=\sup\left{\mu:\left|\pi-\frac pq\right|<\frac{1}{q^\mu}  
\text{ 有無窮多有理數 }p/q\right}.  
]

那麼：

[  
P(K)\text{ 成立}  
]

意思是

[  
\left|\pi-\frac pq\right|<q^{-K}  
]

只有有限多解。也就是說，(K) 不是能被無窮多次達到的逼近指數。因此所有可無窮多次達到的指數都 (\leq K)，或至少不超過 (K) 的邊界。嚴格地說：

[  
\forall K>\mu(\pi),\quad P(K)\text{ 成立}.  
]

反過來，若

[  
P(K)\text{ 對所有 }K>2\text{ 成立},  
]

則不存在任何 (\mu>2) 能無窮多次滿足

[  
\left|\pi-\frac pq\right|<q^{-\mu}.  
]

所以

[  
\mu(\pi)\leq2.  
]

但任何無理數都有

[  
\mu(\pi)\geq2.  
]

故

[  
\mu(\pi)=2.  
]

因此：

[  
\boxed{  
\mu(\pi)=2  
\Longleftrightarrow  
\forall K>2,\ P(K).  
}  
]

這是我們的第一個正式歸約。

然後，我們再把這件事翻成連分數語言，但只翻譯，不深入。若

[  
\frac{p_n}{q_n}  
]

是 π 的連分數收斂子，則

[  
\left|\pi-\frac{p_n}{q_n}\right|  
\asymp  
\frac{1}{a_{n+1}q_n^2}.  
]

所以

[  
\left|\pi-\frac{p_n}{q_n}\right|<q_n^{-K}  
]

大致等價於

[  
a_{n+1}>q_n^{K-2}.  
]

因此 (P(K)) 大致等價於：

[  
a_{n+1}>q_n^{K-2}  
]

只發生有限多次。

令

[  
\delta=K-2.  
]

則自然猜想 (\mu(\pi)=2) 等價於：

[  
\forall \delta>0,\quad  
a_{n+1}>q_n^\delta  
]

只發生有限多次。

換句話說，π 的連分數 partial quotients 可以無界，可以偶爾很大，但不能相對於當前分母 (q_n) 以任何固定冪次級爆發。

這個說法很乾淨，而且避免了「π 的連分數是否正常」這個更大坑。因爲我們不需要知道 (a_n) 是否服從 Gauss–Kuzmin 分布；我們只需要知道它是否有冪次級大爆發。可惜，即使這個較弱問題，目前也未解。

到此，路線圖應該改成這樣：

[  
\begin{array}{c|c|c}  
\text{層級} & \text{命題} & \text{狀態}\  
\hline  
\text{普通 Dirichlet 層} &  
\left|\pi-\frac pq\right|<q^{-2}  
\text{ 無窮多次} &  
\text{已知，且對所有無理數成立}\  
\text{自然正常層} &  
\forall K>2,\ P(K) &  
\text{預期真，等價於 }\mu(\pi)=2\  
\text{已知定理層} &  
\forall K>7.103205334137\ldots,\ P(K) &  
\text{已證}\  
\text{過強層} &  
\exists C>0,\ \left|\pi-\frac pq\right|>Cq^{-2} &  
\text{等價於 partial quotients 有界，不宜期待}  
\end{array}  
]

所以我們下一步不應再問「能不能證 (\mu(\pi)=2)」。那大概率會死。應問一個更可控的問題：

> 原命題若不能無條件證明，能不能改成「在 (\mu(\pi)=2) 下成立」；同時給出一個「在已知 measure 上界下成立」的弱版本？

這是最正當的數學寫法。

因此我建議我們把成果整理成一個標準結論：

---

**命題 A，無條件版本。**  
對任意

[  
K>7.103205334137\ldots,  
]

不等式

[  
\left|\pi-\frac pq\right|<q^{-K}  
]

只有有限多個有理數解 (p/q)。

**命題 B，條件版本。**  
若假設

[  
\mu(\pi)=2,  
]

則對任意

[  
K>2,  
]

不等式

[  
\left|\pi-\frac pq\right|<q^{-K}  
]

只有有限多個有理數解。

**命題 C，非目標版本。**  
命題

[  
\exists C>0,\quad  
\left|\pi-\frac pq\right|>\frac{C}{q^2}  
]

等價於 π 的連分數 partial quotients 有界；這不是目前自然可證的目標，也不是一般預期中 π 應有的性質。

---

這三個命題就是我們目前最應該保存的「骨架」。

下一步若繼續，我會做一件更細的事：把「任意有理逼近」和「連分數收斂子」之間的橋完全證清楚，尤其用 Legendre 定理說明爲甚麼 (K>2) 時只需檢查收斂子。這會讓整個論證不再停留在直覺，而成爲一個嚴格的歸約。
