# Development notes (archive)

This folder collects the working notes, plans, dependency maps, logs, and
outsourcing task files produced during the development of the Erdős 306
formalization. They are kept for reference and provenance only.

**These documents are not authoritative.** Many are outdated or superseded by
later work; some record exploratory routes (e.g. the SBEE / rational-collision
program) that are not on the final spine. The authoritative, current description
of the result lives in the top-level `README.md` and in the module header of
`RequestProject/Erdos306Unconditional.lean`.

Rough orientation among the numbered notes:

* `00 README.md` — the previous project overview (superseded by the top-level
  `README.md`).
* `01`–`28` — the earlier "rational-collision / Fourier positivity" exploratory
  route.
* `29`–`33` — the SBEE single-block counting program.
* `34`–`45` — the block-to-global chain, circle method spec, and Phase-G (G5/G6/G7)
  formalization.
* `46`–`57` — the `R2` construction endgame (reduction of `exists_arcConstruction`
  to concrete supply estimates).
* `ARISTOTLE_*`, `CODEX_*`, `TASK.md`, `ACTIVE_PROMPT.md`, `REGENERATE.sh` —
  outsourcing task packets and helper scripts.
