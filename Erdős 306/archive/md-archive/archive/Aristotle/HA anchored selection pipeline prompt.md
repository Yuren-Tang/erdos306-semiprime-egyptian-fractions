## Mathematical meaning

The paper-side hard theorem is that anchored primitive short hits are sparse
unless the seed primes are structured. This Lean task should formalize the
finite part around that theorem:

$$
\text{anchored hit}
\Longrightarrow
\text{membership in an anchored reciprocal cluster}
\Longrightarrow
\text{good tuple or low-codegree cover or high-incidence ledger}.
$$

The final arithmetic estimate should remain a parameter/hypothesis.

## Goal 1: shortness predicate

Use an abstract predicate for legal short variables, so no asymptotics are
needed.

Suggested:

```lean
structure ShortPredicate where
  Short : Int -> Prop
```

or simply use a parameter `Short : Int -> Prop` in definitions. If a finite
enumeration is needed, use a finite set `ShortSet : Finset Int`.

Avoid proving any facts about intervals unless they are trivial. The paper will
later instantiate `Short x` as `0 < |x| ∧ |x| ≤ M`.

## Goal 2: anchored cluster as a finite set

For a finite seed universe `F : Finset Int`, define the anchored cluster
generated by parameters `(p q0 x0)`:

```lean
def anchoredCluster
    (Short : Int -> Prop) (F : Finset Int)
    (p q0 x0 : Int) : Finset Int :=
  F.filter (fun q =>
    ∃ x y : Int, Short x ∧ Short y ∧ q * x - q0 * x0 = p * y)
```

If decidability is annoying, add the required `[DecidablePred Short]` and local
decidability assumptions.

Also define a finite family of clusters from finite parameter sets:

```lean
def anchoredClusterFamily
    (Short : Int -> Prop) (F P X0 : Finset Int) (q0 : Int) :
    Finset (Finset Int) :=
  ...
```

Here `P` is the finite residual-prime candidate set and `X0` is the finite
reference-short-coordinate set.

## Goal 3: anchored witness gives cluster membership

Prove that an `AnchoredClusterWitness` places each non-reference seed in the
corresponding anchored cluster.

Suggested theorem shape:

```lean
theorem mem_anchoredCluster_of_witness_i
    (Short : Int -> Prop) (F : Finset Int)
    (W : AnchoredClusterWitness)
    (hqiF : W.qi ∈ F)
    (hx0 : Short W.x0)
    (hxi : Short W.xi)
    (hyi : Short W.yi) :
    W.qi ∈ anchoredCluster Short F W.p W.q0 W.x0
```

Do this for `i = 1,2,3,4`, or define a small index API if convenient. Four
separate theorems are fine and probably easier.

## Goal 4: bad tuple covered by one anchored cluster

has an anchored witness.

For example, for a finite set `S : Finset Int`, define:

```lean
def coveredByAnchoredHit
    (Short : Int -> Prop) (F P X0 : Finset Int) (q0 : Int)
    (S : Finset Int) : Prop :=
  ∃ p ∈ P, ∃ x0 ∈ X0,
    S ⊆ anchoredCluster Short F p q0 x0
```

Then prove:

```lean
theorem coveredBySomeCluster_of_coveredByAnchoredHit
    ...
```

where the target is `coveredBySomeCluster` for the finite family
`anchoredClusterFamily Short F P X0 q0`.

If you prefer, define `anchoredClusterFamily` first and make
`coveredByAnchoredHit` definitionally equal to `coveredBySomeCluster`.

## Goal 6: low-codegree cover theorem in anchored language

Use `allCovered_lowCodegree_gives_card_bound` to prove:

If:

- every `k`-subset of `F` is covered by an anchored cluster;
- `T ⊆ F`;
- `T.card + 1 = k`;
- the anchored cluster codegree of `T` is at most `R`;
- every anchored cluster containing `T` has size at most `L`;

then

$$
|F|\le |T|+R L.
$$

Use the existing generic theorem rather than reproving the finite cover lemma.

## Goal 8: optional conditional endpoint theorem

If feasible, package the finite pipeline as one theorem:

