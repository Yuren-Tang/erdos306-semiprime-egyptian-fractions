好。那我們不再繞小例子，而是把「單端口 gadget」向前推成一個真正可研究的核心命題，並檢查它是不是過強。

先固定一個端口素數 (p)。一個單端口 gadget 是一張有限素數圖，頂點包含 (p) 與若干內部素數 (r)。邊 ({u,v}) 對應分母 (uv)。要求：

對每個內部頂點 (r)，

[  
\sum_{s\sim r}\frac1s\equiv 0\pmod r.  
]

而在端口 (p) 上留下指定邊界值：

[  
\sum_{s\sim p}\frac1s\equiv c\pmod p.  
]

這不是形式遊戲。設所有頂點乘積為

[  
L=p\prod_{r\in C}r.  
]

若內部全部平衡，則

[  
\sum_{{u,v}\in E}\frac{L}{uv}  
]

在所有內部素數模 (r) 下都是 (0)。因此總和約分後的分母只能含 (p)。也就是說，整個 gadget 的實際值必然是

[  
\sum_{{u,v}\in E}\frac1{uv}=\frac{d}{p}  
]

某個整數 (d)。而端口條件中的 (c) 正是

[  
d\equiv c\pmod p.  
]

若再把 gadget 做得足夠小，使總和 (<1)，則 (d\in{1,\dots,p-1})，於是端口值 (c) 幾乎就是實際值 (d/p)。

所以問題變成一個更硬但更清楚的命題：

[  
\boxed{  
\text{能否對任意素數 }p\text{ 和 }1\le d<p,\text{ 構造一個 }pq\text{-邊圖，使其和為 }d/p？  
}  
]

若這個命題成立，而且可以避開任意有限禁用素數集合，原 Erdős 問題基本就通了。

不過這個命題可能比原題還強。因為原題只要求 (a/b) 可表示；它不要求每個邊界值都能由單端口 gadget 單獨實現。也就是說，單端口路線若成功，固然漂亮；若失敗，原題仍可能為真，只是需要多端口 gadget。

現在來真正推一步：嘗試構造單端口 gadget。

最簡單的內部平衡方式是讓每個內部頂點度數為 (2)。假設有一個環：

[  
p-r_1-r_2-\cdots-r_k-p.  
]

對內部頂點 (r_i)，平衡條件是

[  
\frac1{r_{i-1}}+\frac1{r_{i+1}}\equiv0\pmod {r_i},  
]

等價於

[  
r_{i-1}+r_{i+1}\equiv0\pmod {r_i}.  
]

也就是

[  
r_i\mid r_{i-1}+r_{i+1}.  
]

端口值則是

[  
\frac1{r_1}+\frac1{r_k}\equiv c\pmod p.  
]

這給出一個很具體的環形構造問題：

[  
\boxed{  
\text{找素數鏈 }p,r_1,\dots,r_k\text{，使每個內部素數整除相鄰兩項之和。}  
}  
]

這已經相當有數學味道。它類似一種素數版 continuant / Euclidean chain：

[  
r_{i+1}=m_i r_i-r_{i-1}.  
]

若能選 (m_i) 使每個 (r_{i+1}) 為素數，並最後閉合回 (p)，就得到 gadget。

但這條路立刻遇到硬牆：它要求一串由線性遞推產生的數同時為素數。這接近素數 k-tuples / Dirichlet 型問題的高階版本，現有工具未必能保證。也就是說，**純環形度數二 gadget 太剛性**。

所以要放寬。內部頂點不應只允許度數 (2)。若內部頂點 (r) 的鄰居是 (s_1,\dots,s_m)，平衡條件為

[  
\sum_{j=1}^m\frac1{s_j}\equiv0\pmod r.  
]

這比度數二靈活得多。於是核心變成：

給定素數 (r) 和若干已連接的鄰居所造成的殘差

[  
\alpha\in\mathbb F_r,  
]

能否再找一些新素數 (s_1,\dots,s_m)，使

[  
\frac1{s_1}+\cdots+\frac1{s_m}\equiv -\alpha\pmod r?  
]

這一步其實容易，因為按 Dirichlet 定理，對任意非零剩餘類 (u\pmod r)，都有無窮多素數 (s\equiv u^{-1}\pmod r)，所以 (1/s\equiv u\pmod r)。因此**局部補償本身不難**。

真正難的是：你找來的新素數 (s_j) 自己也要平衡。這就是遞歸問題。你補一個洞，產生幾個新洞。

所以我們需要一種「封口裝置」：能同時讓一批新頂點全部平衡，而不再產生未處理邊界。這引出一個更合理的核心引理。

---

## 關鍵引理應該不是單端口，而是多端口封口

設 (R={r_1,\dots,r_m}) 是一批尚未平衡的素數，每個 (r_i) 有一個需要補上的殘差 (\alpha_i\in\mathbb F_{r_i})。我們需要加入一批新素數 (C)，並在 (R\cup C) 上加邊，使：

對舊端口 (r_i)，

[  
\sum_{s\sim r_i,\ s\in C}\frac1s\equiv \alpha_i\pmod {r_i};  
]

對所有新頂點 (c\in C)，

[  
\sum_{s\sim c}\frac1s\equiv0\pmod c.  
]

這叫「多端口封口引理」。若它成立，原問題就很有希望。

它比單端口更自然，因為原題本來就有多個邊界素數 (p\mid b)。強行拆成一個個單端口，可能反而把問題弄硬。

於是我們得到新的正面目標：

[  
\boxed{  
\textbf{多端口封口引理：}  
\text{任意有限端口殘差系統，都可用一批新素數封閉平衡。}  
}  
]

這是真正值得攻的版本。

---

## 為什麼多端口封口引理有希望？

因為它有大量自由度。

若新加入 (N) 個素數，可能加的邊有約

[  
mN+\binom N2  
]

條，而需要滿足的同餘條件只有

[  
m+N  
]

條。變量數二次增長，約束數線性增長。這種自由度差距非常大。

更重要的是，素數可以按多個模數同時指定剩餘類。由中國剩餘定理與 Dirichlet 定理，若我們要求一個新素數 (s) 同時滿足

[  
s\equiv a_i\pmod {r_i}  
]

對有限多個舊素數 (r_i)，只要整體剩餘類與模數互素，就有無窮多這樣的素數。這意味著我們可以讓新素數在所有舊端口上的倒數貢獻幾乎任意指定。

所以「舊端口被補到指定殘差」可以做到。卡點只剩新素數之間的自我平衡。

---

## 一個可能的封口策略

我們可以分兩層建造。

第一層：選新素數 (c_1,\dots,c_N)，使它們對舊端口提供所需殘差。這只用 CRT + Dirichlet，大致可行。

第二層：在 (c_1,\dots,c_N) 之間加邊，使每個 (c_j) 自己平衡。

對每個 (c_j)，它已經因為連到舊端口而有一個殘差 (\beta_j\in\mathbb F_{c_j})。現在要在新頂點之間選邊 (x_{ij}\in{0,1})，使

[  
\sum_{i\ne j} x_{ij}\frac1{c_i}\equiv -\beta_j\pmod {c_j}.  
]

這是一個在不同有限域上的對稱 (0/1) 線性系統。每條邊 (ij) 同時影響兩個方程：

在模 (c_i) 方程中貢獻 (1/c_j)，  
在模 (c_j) 方程中貢獻 (1/c_i)。

這是整個問題的代數核心。

若邊變量不是 (0/1)，而是任意整數，這個系統幾乎肯定可解，因為變量數遠多於方程數。  
但邊只能取 (0) 或 (1)，這就是難點。

所以我們可以再提出一個純有限代數命題：

[  
\boxed{  
\textbf{素數圖線性系統命題：}  
}  
]

給定足夠大的互異素數 (c_1,\dots,c_N)，是否對任意殘差 (\beta_j\in\mathbb F_{c_j})，存在簡單圖 (G) 使

[  
\sum_{i\sim j}\frac1{c_i}\equiv \beta_j\pmod {c_j}  
]

對所有 (j) 成立？

若這命題成立，封口引理成立；若封口引理成立，原 Erdős 問題基本成立。

---

## 這裏有一個嚴重問題

上述「任意殘差」命題可能過強。因為每個頂點 (c_j) 最多有 (N-1) 個鄰居，所以可形成的和只有 (2^{N-1}) 種；而模 (c_j) 有 (c_j) 種殘差。若 (c_j) 遠大於 (2^{N-1})，不可能覆蓋所有殘差。

但這不是致命障礙，因為我們可以選 (N) 很大，並選 (c_j) 不要太大，使

[  
2^{N-1}\gg c_j.  
]

例如選前 (N) 個大素數時，最大素數約 (N\log N)，而 (2^{N-1}) 遠大於它。從計數上看，覆蓋所有殘差完全有可能。

這給出一個更精準的策略：不要選稀疏巨大素數，而要選一大塊相對緊湊的素數區間。如此每個模數 (c_j) 都遠小於可能子集數量。

## 目前真正的瓶頸

我們現在已經把原題推到如下純粹組合數論核心：

[  
\boxed{  
\text{對一大塊素數 }c_1,\dots,c_N，  
\text{由鄰居倒數和產生的圖映射是否近似滿射？}  
}  
]

定義映射

[  
\Phi:{0,1}^{\binom N2}\to \prod_{j=1}^N\mathbb F_{c_j},  
]

其中

[  
\Phi(G)_j=\sum_{i\sim j}\frac1{c_i}\pmod {c_j}.  
]

若 (\Phi) 是滿射，或者至少能達到我們需要的殘差向量，那麼封口完成。

這是一個很清楚的數學問題。它不再提埃及分數，不再提 (1/210)，而是問一個素數圖的離散線性映射是否有強覆蓋性。

我看不出立即證明，但這是可以真正前進的形態。它可能可用以下工具：

1. 隨機圖方法：令每條邊獨立取 (0/1)，研究 (\Phi(G)) 在乘積有限環中的分佈是否接近均勻。
    
2. Fourier 分析：估計非平凡 character 的期望。
    
3. 子集和覆蓋定理：證明每個坐標的倒數集合有充分加法展開。
    
4. 局部修正法：先滿足大多數方程，再用小塊修正剩餘坐標。
    

其中 Fourier 路線最正面。對任意非零 character (\chi)，需要估計

[  
\mathbb E_G \chi(\Phi(G)).  
]

因為邊獨立，這個期望分解為所有邊的乘積。每條邊 (ij) 對兩個坐標有貢獻，所以因子類似

[  
\frac12\left(1+  
e\left(  
\frac{\lambda_i}{c_i c_j}  
+  
\frac{\lambda_j}{c_j c_i}  
\right)  
\right),  
]

更準確地說是有限域加性 character 的乘積。若大多數因子都離 (1) 足夠遠，整個乘積就很小，於是 (\Phi(G)) 接近均勻，從而滿射或高概率命中目標殘差。

這是目前最像「正面攻擊」的方向。

## 推進後的結論

我們現在可以把整個 Erdős 問題壓縮成一條清晰的攻擊鏈：

[  
\text{Fourier 隨機圖覆蓋}  
\Rightarrow  
\text{素數圖線性系統可解}  
\Rightarrow  
\text{多端口封口引理}  
\Rightarrow  
\text{可避障 }1/b\text{ 表示}  
\Rightarrow  
\text{原 }a/b\text{ 表示}.  
]

這條鏈不是證明，但已經是正面研究計畫。它告訴我們真正需要證的是：

[  
\boxed{  
\Phi:{0,1}^{\binom N2}\to\prod_j\mathbb F_{c_j}  
\text{ 是否對大素數塊具有強覆蓋性。}  
}  
]

如果這一步能證出來，原題就很可能被打開。

所以，前進到這裏，我會暫時放棄尋找漂亮公式，轉而攻這個有限映射 (\Phi)。這纔是可望把問題從「Erdős 式靈感」推向「可證定理」的地方。
