好。現在我們專攻：

[  
\boxed{\text{何時單列 Fourier 因子 }F_z(\lambda,\tau)\text{ 會很大？}}  
]

若能回答這個問題，就能判斷「對角主項」之外是否還有別的巨大 Fourier 項。

---

## 1. 單列 Fourier 因子

末層素數集為

[  
Y.  
]

終端素數為

[  
z.  
]

列核是

[  
K_z=\left{x\in{0,1}^Y:  
\sum_{y\in Y}x_y y^{-1}\equiv0\pmod z  
\right}.  
]

現在同時記錄殘差 (R) 與實際質量 (M)。對 character pair

[  
(\lambda,\tau),  
\qquad  
\lambda=(\lambda_y)_{y\in Y},\quad \lambda_y\in\mathbb F_y,\quad \tau\in\mathbb Z,  
]

單列相位是

[  
\sum_{y\in Y}x_y  
\left(  
\frac{\lambda_y(z^{-1}\bmod y)}{y}  
+  
\frac{\tau}{yz}  
\right).  
]

所以

# [  
F_z(\lambda,\tau)

\mathbb E_{x\in K_z}  
e\left(  
\sum_{y\in Y}x_y  
\left[  
\frac{\lambda_y(z^{-1}\bmod y)}{y}  
+  
\frac{\tau}{yz}  
\right]  
\right).  
]

我們要問：

[  
|F_z(\lambda,\tau)|\approx 1  
]

何時可能發生？

---

## 2. 用列約束化簡相位

關鍵仍是 CRT 恆等式。對互異素數 (y,z)，有

[  
\frac{z^{-1}\bmod y}{y}  
+  
\frac{y^{-1}\bmod z}{z}  
\equiv  
\frac1{yz}  
\pmod1.  
]

因此

# [  
\frac{\lambda_y(z^{-1}\bmod y)}{y}

## \frac{\lambda_y}{yz}

\frac{\lambda_y(y^{-1}\bmod z)}{z}  
\pmod1.  
]

代入相位，得到

## [  
\sum_y x_y  
\left(  
\frac{\lambda_y+\tau}{yz}

\frac{\lambda_y(y^{-1}\bmod z)}{z}  
\right).  
]

也就是

## [  
\frac1z  
\sum_y x_y  
\left(  
\frac{\lambda_y+\tau}{y}

\lambda_y(y^{-1}\bmod z)  
\right).  
]

因為 (x\in K_z) 滿足

[  
\sum_y x_y(y^{-1}\bmod z)\equiv0\pmod z.  
]

若 (\lambda_y) 是常數 (h)，則

# [  
\sum_y x_y\lambda_y(y^{-1}\bmod z)

h\sum_y x_y(y^{-1}\bmod z)  
\equiv0\pmod z,  
]

所以這一項在 (e(\cdot)) 中消失。若再取

[  
\tau=-h,  
]

則前一項也消失。這就是對角主項：

[  
\boxed{  
\lambda_y=h\ \forall y,\qquad \tau=-h.  
}  
]

此時

[  
F_z(\lambda,\tau)=1  
]

精確成立，不是近似。

---

## 3. 若 (F_z) 很大，必然近似對角

現在反過來。若

[  
|F_z(\lambda,\tau)|  
]

很大，直覺上相位在 (K_z) 上幾乎不變。由於 (K_z) 很大，這表示係數

# [  
A_y

\frac{\lambda_y(z^{-1}\bmod y)}{y}  
+  
\frac{\tau}{yz}  
\pmod1  
]

在列核的許多方向上必須幾乎線性相依於列約束

[  
\sum_y x_y y^{-1}\equiv0\pmod z.  
]

也就是說，存在某個 (\mu\in\mathbb F_z)，使對很多 (y)：

[  
A_y\approx \frac{\mu(y^{-1}\bmod z)}{z}\pmod1.  
]

把 (A_y) 展開：

[  
\frac{\lambda_y(z^{-1}\bmod y)}{y}  
+  
\frac{\tau}{yz}  
\approx  
\frac{\mu(y^{-1}\bmod z)}{z}.  
]

用 CRT 恆等式替換第一項：

## [  
\frac{\lambda_y}{yz}

\frac{\lambda_y(y^{-1}\bmod z)}{z}  
+  
\frac{\tau}{yz}  
\approx  
\frac{\mu(y^{-1}\bmod z)}{z}.  
]

即

[  
\frac{\lambda_y+\tau}{yz}  
\approx  
\frac{(\lambda_y+\mu)(y^{-1}\bmod z)}{z}.  
]

乘以 (z)：

[  
\frac{\lambda_y+\tau}{y}  
\approx  
(\lambda_y+\mu)(y^{-1}\bmod z)  
\pmod 1.  
]

右邊是整數模 (1) 的量；更精確地，它的分母是 (z)，而左邊大小約為

[  
\frac{|\lambda_y+\tau|}{y}.  
]

若 (y) 相對較大，左邊很小。要讓這種近似對許多 (y) 同時成立，最自然的方式是：

[  
\lambda_y+\mu\equiv0\pmod z  
]

但 (\lambda_y) 是模 (y) 的 residue，不是模 (z) 的元素。要比較它們，需取整數代表。若我們取

[  
0\le \lambda_y<y,  
]

而 (z) 比 (y) 大，則 (\lambda_y+\mu\equiv0\pmod z) 意味着

[  
\lambda_y+\mu=0  
\quad\text{或}\quad  
\lambda_y+\mu=z.  
]

這迫使 (\lambda_y) 幾乎為同一個常數。這正是對角結構。

因此可以提出一個非嚴格但清晰的判準：

[  
\boxed{  
F_z(\lambda,\tau)\text{ 大}  
\quad\Rightarrow\quad  
\lambda_y\approx h\text{ 對大多數 }y,\quad \tau\approx -h.  
}  
]

這就是非對角反聚集的核心。

---

## 4. 為什麼要控制素數大小關係？

上面的推理用到了「(z) 比 (y) 大」或至少兩者大小可比較。若 (z) 太小或太大，近似關係會變形。

因此終端層參數最好安排為：

[  
z> \max Y  
]

但不要大得離譜。這樣 (\lambda_y\in[0,y)) 真的可以作為小於 (z) 的整數看待。

若 (z>\max Y)，則「(\lambda_y+\mu\equiv0\pmod z)」幾乎等於「(\lambda_y) 是固定常數」。這使對角分類可行。

所以終端設計應選：

[  
\boxed{  
\max Y< z < C\max Y  
}  
]

或至少 (Z) 位於 (Y) 之後的一個不太遠的素數區間。這與前面「(z) 要遠大於 (Y) 或遠小於倒數質量」的想法不同；前者會破壞對角分類，後者也不真正控制 Fourier 模長。

現在看來，終端 (Z) 應該是一個與 (Y) 同尺度或略大的素數層。

---

## 5. 但同尺度帶來另一個問題

若 (Y) 和 (Z) 同尺度，則對每個 (z)，列核

[  
K_z  
]

的大小約為

[  
2^{|Y|}/z.  
]

要它大，需要

[  
2^{|Y|}\gg z.  
]

這要求

[  
|Y|\gg \log z.  
]

如果 (Y) 是一段同尺度素數區間，這可以做到：取很多個 (Y)-素數，而 (z) 的大小只是指數以下即可。

例如令

[  
|Y|=m,\qquad z\le e^{\alpha m},  
]

且 (\alpha<\log2)。那麼

[  
2^m/z  
]

仍指數大。

同時要求

[  
z>\max Y.  
]

所以可以選

[  
Y\subset [P,2P],\qquad Z\subset [Q,2Q],  
]

其中

[  
2P<Q<e^{\alpha m}.  
]

只要 (m) 比 (\log Q) 大，這可行。

這給出一個比較合理的終端參數區間。

---

## 6. 非對角反聚集的正式候選命題

現在可以把應證命題寫清楚：

**非對角單列引理。**  
設 (Y) 是 (m) 個素數，(z) 是一個素數，且

[  
\max Y<z<e^{\alpha m}  
]

某個小 (\alpha)。設

[  
K_z=\left{x\in{0,1}^Y:  
\sum_{y\in Y}x_y y^{-1}\equiv0\pmod z  
\right}.  
]

對任意 character pair ((\lambda,\tau))，若不存在整數 (h) 使得

[  
\lambda_y=h\pmod y  
]

對除了少數 (y) 外都成立，且

[  
\tau=-h,  
]

則

[  
|F_z(\lambda,\tau)|\le 1-\epsilon  
]

對某個 (\epsilon>0)，至少對許多 (z\in Z) 成立。

這就是終端吸收所需的核心分類命題。

---

## 7. 這個命題還是太強，要再精細化

原因是如果 (\lambda_y=h) 對 (99%) 的 (y) 成立，但少數例外不同，那麼 (F_z) 可能不是 (1)，但也不一定有固定 (\epsilon) 衰減；衰減量可能只與例外數成比例。

所以更合理的形式是：

令

# [  
d(\lambda,h)

#{y\in Y:\lambda_y\ne h\pmod y}.  
]

定義

[  
d(\lambda)=\min_h d(\lambda,h).  
]

則希望證：

[  
\boxed{  
|F_z(\lambda,\tau)|  
\le  
\exp(-c,d(\lambda))  
}  
]

除非 (\tau) 也接近 (-h)。若 (\tau\ne -h)，則即使 (\lambda) 對角，也應該有衰減，衰減量取決於 (|\tau+h|)。

這樣就形成了合理的「距離對角」估計。

對多個終端素數 (z\in Z) 相乘後：

[  
|\widehat\nu(\lambda,\tau)|  
\le  
\exp(-c,|Z|,d(\lambda)).  
]

而距離對角為 (d) 的 character 數大約是

[  
\binom md \prod_{\text{例外 }y}y  
]

乘以對角參數 (h) 的數量。只要素數大小與 (|Z|) 參數配得好，就能求和。

這是可行形態。

---

## 8. 對角項如何處理？

若

[  
\lambda_y=h\quad\forall y,  
\qquad  
\tau=-h,  
]

則單列 (F_z=1)。這些是主項，不能殺。

在 Fourier inversion 中，這些主項會產生一個條件：目標 ((R,M)) 必須滿足

[  
M\equiv\sum_y\frac{R_y}{y}\pmod1.  
]

若目標滿足此相容條件，對角主項會全部貢獻正量；若不滿足，主項相消。這正是我們要的。

所以最終證明形式會是：

# [  
\mathbb P((R,M)=\text{target})

\text{對角主項}  
+  
\text{非對角誤差}.  
]

對角主項由相容條件給出，非對角誤差由距離對角估計壓下。

這非常像 circle method 的主弧／次弧分解。

---

## 9. 這一步的實質推進

我們現在有了真正準確的終端 Fourier 圖像：

[  
\boxed{  
\text{唯一不可衰減的 Fourier 模式是 CRT 對角模式。}  
}  
]

需要證明的是：

[  
\boxed{  
\text{所有離對角有距離的模式，都按距離指數衰減。}  
}  
]

具體估計是：

[  
|F_z(\lambda,\tau)|  
\le  
\exp(-c,d((\lambda,\tau),\mathcal D)),  
]

其中 (\mathcal D) 是對角族

[  
\mathcal D={(\lambda,\tau):\lambda_y=h\ \forall y,\ \tau=-h}.  
]

如果這個估計成立，終端吸收就不再是神秘問題，而是一個帶明確主項的 Fourier 定理。

---

## 10. 下一步最該做什麼？

下一步應該嘗試證明單列距離估計。也就是：

給定 ((\lambda,\tau))，考察相位係數

# [  
A_y

\frac{\lambda_y(z^{-1}\bmod y)}{y}  
+  
\frac{\tau}{yz}.  
]

在列核

[  
\sum_y x_y y^{-1}\equiv0\pmod z  
]

上平均

[  
e\left(\sum_y x_y A_y\right).  
]

若 (A_y) 不等於某個倍數的列約束係數

[  
\frac{\mu(y^{-1}\bmod z)}z  
]

加上整數，則平均應衰減。  
而 (A_y) 等於這種形式，正好逼出對角結構。

所以單列估計的核心代數判準是：

[  
\boxed{  
A_y\approx \frac{\mu(y^{-1}\bmod z)}z  
\text{ 對多數 }y  
\quad\Longleftrightarrow\quad  
(\lambda,\tau)\text{ 近對角。}  
}  
]

這就是下一步要攻的精確命題。
