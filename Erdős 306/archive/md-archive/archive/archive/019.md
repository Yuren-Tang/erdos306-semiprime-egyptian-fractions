好。現在要證的核心不是一開始就碰素數，而是先處理一個純粹的有限群事實：

> 在一個 (0/1) 立方體的線性核上平均某個 character；若平均值很大，則這個 character 必須接近該線性約束的倍數。

這正是我們需要的「單列距離估計」的抽象骨架。

---

設

[  
Y={y_1,\dots,y_m},\qquad z\text{ 為素數}.  
]

令

[  
a_y=y^{-1}\pmod z\in\mathbb F_z.  
]

列核為

[  
K_z=\left{x\in{0,1}^Y:\sum_{y\in Y}a_yx_y=0\pmod z\right}.  
]

給定實數相位係數

[  
A_y\in\mathbb R/\mathbb Z,  
]

考慮

# [  
F(A)

\mathbb E_{x\in K_z}  
e\left(\sum_{y\in Y}A_yx_y\right).  
]

我們想知道：什麼時候 (|F(A)|) 很大？

由列核指示函數展開：

# [  
1_{K_z}(x)

\frac1z\sum_{\mu\in\mathbb F_z}  
e\left(\frac{\mu}{z}\sum_y a_yx_y\right).  
]

因此

# [  
F(A)

\frac{  
\sum_{\mu\in\mathbb F_z}  
\prod_{y\in Y}  
\left(1+e\left(A_y+\frac{\mu a_y}{z}\right)\right)  
}{  
\sum_{\mu\in\mathbb F_z}  
\prod_{y\in Y}  
\left(1+e\left(\frac{\mu a_y}{z}\right)\right)  
}.  
]

分母約等於 (2^m/z)，前提是子集和 (\sum a_yx_y) 在 (\mathbb F_z) 上混合良好。這是我們前面已有的「列核混合」條件。

於是若 (|F(A)|) 很大，分子中必有某個 (\mu) 使

[  
\left|  
\prod_y  
\frac{1+e\left(A_y+\frac{\mu a_y}{z}\right)}2  
\right|  
]

不太小。也就是

[  
\prod_y  
\left|  
\cos \pi\left(A_y+\frac{\mu a_y}{z}\right)  
\right|  
]

不太小。

用估計

[  
|\cos\pi t|\le \exp(-2|t|^2),  
]

得到：若乘積不小，則存在 (\mu\in\mathbb F_z)，使

[  
\sum_{y\in Y}  
\left|  
A_y+\frac{\mu a_y}{z}  
\right|^2  
]

不大。

換句話說：

[  
\boxed{  
F(A)\text{ 大}  
\quad\Longrightarrow\quad  
A\text{ 接近某個列約束 character }-\mu a/z.  
}  
]

這是純 Fourier 結構，不涉及 Erdős 問題。

---

現在把實際的 (A_y) 代入。對終端吸收，我們有

# [  
A_y

\frac{\lambda_y(z^{-1}\bmod y)}{y}  
+  
\frac{\tau}{yz}.  
]

記

[  
b_y=z^{-1}\pmod y,\qquad a_y=y^{-1}\pmod z.  
]

則

[  
A_y=\frac{\lambda_y b_y}{y}+\frac{\tau}{yz}.  
]

若 (F_z(\lambda,\tau)) 大，則存在 (\mu\in\mathbb F_z)，使對多數 (y)，

[  
\frac{\lambda_y b_y}{y}+\frac{\tau}{yz}  
+  
\frac{\mu a_y}{z}  
\approx0\pmod1.  
]

用 CRT 恆等式

[  
\frac{b_y}{y}+\frac{a_y}{z}  
\equiv  
\frac1{yz}\pmod1,  
]

可將

# [  
\frac{\lambda_y b_y}{y}

## \frac{\lambda_y}{yz}

\frac{\lambda_y a_y}{z}  
\pmod1.  
]

代入得

[  
\frac{\lambda_y+\tau}{yz}  
+  
\frac{(\mu-\lambda_y)a_y}{z}  
\approx0\pmod1.  
]

乘以 (z)：

[  
\frac{\lambda_y+\tau}{y}  
+  
(\mu-\lambda_y)a_y  
\approx0\pmod1.  
]

但 ((\mu-\lambda_y)a_y) 是模 (z) 的 residue 除以 (1) 的整數型項；更精確地，

[  
\frac{(\mu-\lambda_y)a_y}{z}  
]

才是小數。上面乘以 (z) 後不能直接視作 (\pmod1) 的同一結構；要稍微小心。較安全的方式是乘以 (yz) 看同餘。

原式

[  
\frac{\lambda_y b_y}{y}+\frac{\tau}{yz}+\frac{\mu a_y}{z}  
\approx0\pmod1  
]

乘以 (yz)，得到：

[  
z\lambda_y b_y+\tau+y\mu a_y  
\approx0\pmod{yz}.  
]

因為

[  
zb_y\equiv1\pmod y,\qquad ya_y\equiv1\pmod z.  
]

模 (y) 看：

[  
\tau+\mu y a_y\equiv0\pmod y.  
]

模 (z) 看：

[  
\lambda_y z b_y+\tau\equiv0\pmod z.  
]

但這還沒有把 (\lambda_y) 和 (\mu,\tau) 直接連起來。用 CRT 恆等式更直觀：若相位精確為零，即

[  
\frac{\lambda_y b_y}{y}+\frac{\tau}{yz}+\frac{\mu a_y}{z}\in\mathbb Z,  
]

則乘 (yz)：

[  
z\lambda_y b_y+\tau+y\mu a_y\equiv0\pmod{yz}.  
]

模 (y)：

[  
\tau+y\mu a_y\equiv0\pmod y.  
]

由於 (y\mu a_y\equiv0\pmod y)，得

[  
\tau\equiv0\pmod y.  
]

這不對，因為對角例子 (\lambda_y=h,\tau=-h,\mu=h) 應該成立，而 (\tau\not\equiv0\pmod y)。問題在於我把 (\mu a_y/z) 的符號取反方向不一致。回到前式。

我們需要 (A_y+\mu a_y/z) 接近整數，其中

[  
A_y=\frac{\lambda_y b_y}{y}+\frac{\tau}{yz}.  
]

對角例子 (\lambda_y=h,\tau=-h)。若取 (\mu=h)，則

# [  
A_y+\frac{\mu a_y}{z}

\frac{h b_y}{y}-\frac{h}{yz}+\frac{h a_y}{z}.  
]

由

[  
\frac{b_y}{y}+\frac{a_y}{z}  
\equiv \frac1{yz}\pmod1  
]

可知它為整數。好。

所以精確零條件是：

[  
\frac{\lambda_y b_y}{y}+\frac{\tau}{yz}+\frac{\mu a_y}{z}\in\mathbb Z.  
]

乘以 (yz)：

[  
z\lambda_y b_y+\tau+y\mu a_y\equiv0\pmod{yz}.  
]

模 (y)：

[  
z\lambda_y b_y+\tau\equiv0\pmod y.  
]

因為

[  
zb_y\equiv1\pmod y,  
]

得

[  
\lambda_y+\tau\equiv0\pmod y.  
]

模 (z)：

[  
\tau+y\mu a_y\equiv0\pmod z.  
]

因為

[  
ya_y\equiv1\pmod z,  
]

得

[  
\mu+\tau\equiv0\pmod z.  
]

所以精確相位消失等價於

[  
\boxed{  
\lambda_y\equiv-\tau\pmod y,\qquad \mu\equiv-\tau\pmod z.  
}  
]

這正是對角結構：存在同一個整數

[  
h=-\tau  
]

使

[  
\lambda_y=h\pmod y  
]

對所有 (y)，且 (\mu=h\pmod z)。

這個推導非常關鍵。它說：

[  
\boxed{  
\text{單列相位能在某坐標 }y\text{ 上完全消失}  
\iff  
\lambda_y\equiv-\tau\pmod y.  
}  
]

因此，若 (\lambda_y\ne-\tau\pmod y)，那個坐標必然造成相位偏離。這正是我們需要的距離對角估計。

---

## 單列距離估計的雛形

令

# [  
D(\lambda,\tau)

{y\in Y:\lambda_y+\tau\not\equiv0\pmod y}.  
]

這是「離對角的坐標集合」。

對每個 (y\in D(\lambda,\tau))，無論選什麼 (\mu\in\mathbb F_z)，相位

[  
A_y+\frac{\mu a_y}{z}  
]

不可能等於整數。它與整數的距離至少約為

[  
\frac{1}{yz}  
]

級別，因為乘以 (yz) 後是一個非零整數模 (yz)。

更精確地，

[  
\left|  
A_y+\frac{\mu a_y}{z}  
\right|  
\ge  
\frac{1}{yz}.  
]

這給出

[  
\sum_{y\in D}  
\left|  
A_y+\frac{\mu a_y}{z}  
\right|^2  
\ge  
\sum_{y\in D}\frac1{y^2z^2}.  
]

但這太弱。若 (y,z) 很大，這個平方和極小，無法給出固定指數衰減。這又暴露了一個嚴重問題：**僅靠最小間距下界不夠。**

我們需要更強：對大多數 (y\in D)，相位不只是離整數 (1/(yz))，而是平均有常數級分散。這取決於 residues 的分佈，而不是純代數。

所以目前得到的是：

1. 精確零分類完全是對角；
    
2. 但要得到有效衰減，還需非對角坐標的相位在圓周上分散，而不是只非零。
    

---

## 這裏應該如何前進？

我們要看相位偏差的精確形狀。設

[  
h=-\tau.  
]

則

[  
\lambda_y+\tau=\lambda_y-h.  
]

令

[  
r_y=\lambda_y-h\pmod y,  
]

其中 (r_y\ne0) 對 (y\in D)。

對取 (\mu=h) 的最危險方向，因為對角部分抵消，剩下相位應為

[  
\frac{r_y b_y}{y}  
]

加上相應修正。實際上若 (\mu=h)，則

# [  
A_y+\frac{h a_y}{z}

\frac{\lambda_y b_y}{y}+\frac{\tau}{yz}+\frac{h a_y}{z}.  
]

由 (\tau=-h)：

# [

\frac{\lambda_y b_y}{y}-\frac{h}{yz}+\frac{h a_y}{z}.  
]

又

[  
\frac{h b_y}{y}+\frac{h a_y}{z}  
\equiv \frac{h}{yz}.  
]

所以

# [  
A_y+\frac{h a_y}{z}  
\equiv  
\frac{(\lambda_y-h)b_y}{y}

\frac{r_y b_y}{y}  
\pmod1.  
]

因此非對角衰減取決於

[  
\left|\frac{r_y(z^{-1}\bmod y)}{y}\right|.  
]

這非常好。它回到我們熟悉的倒數分散形式。

若 (r_y\ne0)，乘以 (r_y) 只是 (\mathbb F_y^\times) 的置換；所以對一個固定的 (y)，當 (z) 在終端素數集合 (Z) 中變動時，

[  
r_y z^{-1}\pmod y  
]

應該分散。因此對許多 (z)，

[  
\left|  
\frac{r_y z^{-1}}y  
\right|^2  
]

平均有常數下界。

這表示：即使單個 (z) 的最小下界很弱，乘過許多 (z\in Z) 後，非對角坐標會被殺掉。

---

## 因此正確估計不是「單列固定衰減」，而是「多列平均衰減」

這是重要修正。

不要要求每個 (z) 都有

[  
|F_z|\le e^{-c d}.  
]

只需對整個 (Z) 有：

[  
\prod_{z\in Z}|F_z|  
\le  
\exp(-c|Z|d)  
]

或至少

[  
\sum_{z\in Z}  
\sum_{y\in D}  
\left|  
\frac{r_y z^{-1}}y  
\right|^2  
\gg |Z|,|D|.  
]

這正是可以由 (Z) 在每個模 (y) 下的倒數分散推出的條件。

所以非對角衰減的正確命題是：

[  
\boxed{  
\text{若 }D={y:\lambda_y+\tau\ne0\pmod y},  
\text{則整個終端層 }Z\text{ 給出衰減 }  
e^{-c|Z||D|}.  
}  
]

這不要求單個 (z) 強，只要求 (Z) 集體分散。

---

## 條件性非對角衰減命題

假設 (Z) 對 (Y) 滿足互逆分散：存在 (\rho>0)，使對所有 (y\in Y) 與所有非零 (r\in\mathbb F_y)，

[  
\sum_{z\in Z}  
\left|  
\frac{r(z^{-1}\bmod y)}{y}  
\right|^2  
\ge  
\rho |Z|.  
]

則對任意 ((\lambda,\tau))，令

[  
D={y:\lambda_y+\tau\not\equiv0\pmod y}.  
]

最危險的列約束 multiplier (\mu) 會隨 (z) 取

[  
\mu\equiv-\tau\pmod z.  
]

在這個方向下，每個 (y\in D) 的殘餘相位是

[  
\frac{(\lambda_y+\tau)(z^{-1}\bmod y)}y.  
]

由互逆分散，

[  
\sum_{z\in Z}\sum_{y\in D}  
\left|  
\frac{(\lambda_y+\tau)(z^{-1}\bmod y)}y  
\right|^2  
\ge  
\rho |Z||D|.  
]

因此非對角 Fourier 係數應有

[  
|\widehat\nu(\lambda,\tau)|  
\le  
\exp(-c\rho |Z||D|)  
]

至列核混合誤差。

這正是我們要的距離對角估計。

---

## 還有一個細節：(\mu) 不一定等於 (-\tau)

在單列 Fourier 展開中，每個 (z) 都會對 (\mu\in\mathbb F_z) 求和。除了

[  
\mu\equiv-\tau\pmod z  
]

之外的 (\mu) 怎麼辦？

若 (\mu\ne-\tau\pmod z)，則即使對角坐標也不抵消。對所有 (y)，相位中會有一個共同的非零偏差。這反而更容易衰減，因為許多 (y) 都貢獻。

所以最危險項確實是

[  
\mu=-\tau.  
]

其他 (\mu) 可由 (Y) 在模 (z) 下的倒數分散殺掉。這正是列核混合條件已經需要的東西。

因此整體上，非對角衰減的主困難可歸結為 (Z) 對 (Y) 的互逆分散；這和前面稀疏層用的是同一類條件。

---

## 目前的結論

我們現在得到一個相當清楚的條件性核心定理：

若終端兩層 (Y,Z) 滿足：

1. **列核混合：** 對每個 (z\in Z)，(Y) 的倒數在 (\mathbb F_z) 中有子集和混合；
    
2. **互逆分散：** 對每個 (y\in Y)，(Z) 的倒數在 (\mathbb F_y) 中有平方分散；
    

則終端吸收的 Fourier 係數滿足：

[  
|\widehat\nu(\lambda,\tau)|  
\le  
\exp(-c|Z|D(\lambda,\tau))  
]

除對角族

[  
\lambda_y+\tau\equiv0\pmod y\quad\forall y  
]

外。

這就是我們一直要找的「非對角反聚集」。

---

## 這一步的實質推進

我們不再只是說「非對角應該衰減」。我們已經識別出：

[  
\boxed{  
D(\lambda,\tau)=#{y:\lambda_y+\tau\ne0\pmod y}  
}  
]

是正確的離對角距離。

並且得到衰減來源：

[  
\boxed{  
\sum_{z\in Z}  
\left|  
\frac{(\lambda_y+\tau)z^{-1}}y  
\right|^2  
}  
]

這正好可由 (Z) 在模 (y) 下的倒數分散控制。

所以終端 Fourier 分析被壓縮為兩個可理解的分散假設。這已經是一條相當乾淨的路。

下一步應該把它合併成一個「條件性終端吸收定理」：

> 若 (Y,Z) 滿足列核混合與互逆分散，且參數足以壓過 character 數，則任意相容的殘差—質量目標都能被終端列吸收。

證出這個條件性定理後，剩下的外部問題就是：能否選出滿足兩個分散假設的素數層 (Y,Z)。這就是真正的解析數論輸入。
