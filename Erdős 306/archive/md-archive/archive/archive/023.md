好。現在把整條路線收束成一個**條件性定理**。我們不再繼續發散，而是確認：若有一個明確的解析數論引理，是否真的可以推出原 Erdős 問題。

原問題：

[  
\frac ab\in \mathbb Q_{>0},\quad b\ \text{squarefree}.  
]

問是否存在互異的

[  
n_i=p_iq_i,\qquad p_i\ne q_i\text{ primes},  
]

使

[  
\frac ab=\sum_i\frac1{n_i}.  
]

我現在要證的條件性命題是：

[  
\boxed{  
\text{若素數互逆短弧避免引理成立，則 Erdős 問題成立。}  
}  
]

---

## 一、所需外部引理

我們需要如下解析數論輸入。

**素數互逆短弧避免引理。**  
存在常數 (\eta,\beta>0)，使得對任意充分大的 (P)，在素數集合

[  
\mathcal P=[P,2P]\cap{\text{primes}}  
]

中有一個正密度子集 (\mathcal G\subseteq\mathcal P)，滿足：對每個 (q\in\mathcal G)、每個非零 (a\in\mathbb F_q^\times),

[  
#\left{  
p\in\mathcal P:  
\left|\frac{a p^{-1}}q\right|<\eta  
\right}  
\le (1-\beta)|\mathcal P|.  
]

意思是：當 (p\sim P) 在素數中變動時，(a p^{-1}\bmod q) 不會有幾乎全部落在 (0) 附近的一小段弧中。

這個引理可由某種弱形式的素數 Kloosterman 和估計推出，例如控制

[  
\sum_{p\sim P,\ p\ \mathrm{prime}} e\left(\frac{a p^{-1}}q\right).  
]

我們先把它作為外部引理。

---

## 二、由外部引理得到互逆素數塊

取

[  
N=P^\varepsilon  
]

其中 (\varepsilon>0) 很小。從 (\mathcal G) 中隨機選兩個互異集合

[  
Y,Z\subseteq\mathcal G,\qquad |Y|=|Z|=N.  
]

由短弧避免和 Chernoff 集中不等式，對固定 (z\in Z)、固定非零 (\mu\in\mathbb F_z)，集合

[  
\left{  
y\in Y:  
\left|\frac{\mu y^{-1}}z\right|<\eta  
\right}  
]

至多佔 (1-\beta/2) 的比例，概率極高。對所有 (z,\mu) 做 union bound，因失敗概率約為 (\exp(-cN))，而需要控制的對象數量只有多項式級，故同時成立。

因此得到：

[  
\sum_{y\in Y}  
\left|  
\frac{\mu y^{-1}}z  
\right|^2  
\ge  
\eta^2\cdot\frac{\beta}{2}N  
]

對所有 (z\in Z)、非零 (\mu) 成立。

同理反向也成立：

[  
\sum_{z\in Z}  
\left|  
\frac{r z^{-1}}y  
\right|^2  
\ge  
\eta^2\cdot\frac{\beta}{2}N  
]

對所有 (y\in Y)、非零 (r\in\mathbb F_y) 成立。

所以外部引理推出：

[  
\boxed{  
\text{存在雙向互逆分散的素數塊 }Y,Z。  
}  
]

這正是終端吸收需要的材料。

---

## 三、二部傳遞：把殘差推到末層

現在回到原問題。先只處理

[  
\frac1b.  
]

令

[  
B={p:p\mid b}.  
]

我們要構造一張素數圖 (E)，使對每個 (p\in B),

[  
\sum_{s\sim p}\frac1s\equiv \frac1{b/p}\pmod p,  
]

而所有輔助頂點 (r\notin B) 滿足

[  
\sum_{s\sim r}\frac1s\equiv0\pmod r.  
]

這樣，若

[  
L=\prod_{\text{用到的素數}}r,  
]

則可推出

[  
\sum_{{u,v}\in E}\frac{L}{uv}  
\equiv  
\frac Lb  
\pmod L.  
]

這是全局 CRT 同餘。

如何構造局部條件？用分層二部傳遞。

設第一層是 (A_0=B)。給定 (A_i) 上的殘差 (\gamma_a)，選下一層素數 (A_{i+1}) 足夠大，且在每個模 (a\in A_i) 下有足夠多不同 residue。由一維子集和覆蓋，可以選邊 (A_i-A_{i+1})，使每個 (a\in A_i) 被清零：

[  
\sum_{s\in A_{i+1},\ as\in E}\frac1s\equiv\gamma_a\pmod a.  
]

這會在 (A_{i+1}) 上造成新殘差。如此推進，直到末層 (Y)。

這一步所需的數論很弱：只需下一層素數在每個上一層模數下佔據足夠多的剩餘類；可用 Dirichlet 或更粗的素數分佈安排。因為這裏不要求終止，只是把殘差搬到下一層。

於是我們可把所有未平衡殘差集中到末層 (Y)。

---

## 四、終端吸收：用 (Z) 封掉末層

現在末層 (Y) 上有殘差

[  
\gamma_y\in\mathbb F_y.  
]

取由外部引理構造出的互逆分散終端層 (Z)。對每個 (z\in Z)，我們選一個子集

[  
I_z\subseteq Y  
]

並連邊 (zy) 對 (y\in I_z)。為使 (z) 自身平衡，要求

[  
\sum_{y\in I_z}\frac1y\equiv0\pmod z.  
]

也就是

[  
I_z\in K_z,  
]

其中

[  
K_z=\left{  
x\in{0,1}^Y:  
\sum_{y\in Y}x_y y^{-1}\equiv0\pmod z  
\right}.  
]

終端吸收要做到：

[  
\sum_{z:\ y\in I_z}\frac1z\equiv -\gamma_y\pmod y  
]

對所有 (y\in Y)。

這是一個受列約束的隨機矩陣問題。

由 (Y^{-1}\bmod z) 的分散性，列核 (K_z) 很大且投影混合。  
由 (Z^{-1}\bmod y) 的分散性，非對角 Fourier character 被指數衰減。  
唯一不衰減的是對角族

[  
\lambda_y=h\pmod y\quad\forall y,  
]

它對應的不是新障礙，而是 CRT 相容條件本身。

前面我們已經識別：對角族反映的是

[  
M\equiv\sum_y\frac{R_y}{y}\pmod1,  
]

即實際質量與殘差之間的必然關係。若整個分層構造的殘差來自同一個全局目標，那麼這個相容條件自動成立。因此對角主項保留，非對角誤差被壓下，終端吸收成立。

所以得到：

[  
\boxed{  
\text{末層 }Y\text{ 的任意相容殘差可由終端層 }Z\text{ 封掉。}  
}  
]

---

## 五、得到 (1/b) 的表示

此時所有頂點都滿足局部條件：

對 (p\mid b),

[  
\sum_{s\sim p}\frac1s\equiv\frac1{b/p}\pmod p;  
]

對所有其他頂點 (r),

[  
\sum_{s\sim r}\frac1s\equiv0\pmod r.  
]

令

[  
L=\prod_{\text{所有頂點 }r}r.  
]

對每個頂點 (r) 取模，可得

[  
\sum_{{u,v}\in E}\frac{L}{uv}  
\equiv  
\frac Lb  
\pmod r.  
]

因為 (L) squarefree，所以合併為

[  
\sum_{{u,v}\in E}\frac{L}{uv}  
\equiv  
\frac Lb  
\pmod L.  
]

接着控制大小。所有輔助素數都可選得足夠大，使

[  
0<\sum_{{u,v}\in E}\frac1{uv}<1.  
]

因此

[  
0<  
\sum_{{u,v}\in E}\frac{L}{uv}  
<L.  
]

同時

[  
0<\frac Lb<L.  
]

兩個位於 ((0,L)) 的整數模 (L) 同餘，所以相等：

# [  
\sum_{{u,v}\in E}\frac{L}{uv}

\frac Lb.  
]

除以 (L)，得到

[  
\boxed{  
\frac1b=\sum_{{u,v}\in E}\frac1{uv}.  
}  
]

每條邊 ({u,v}) 對應分母 (uv)，且 (u,v) 是不同素數，所以分母都是 (pq) 型。

---

## 六、從 (1/b) 到 (a/b)

現在要表示

[  
\frac ab.  
]

若我們已經能對任意有限禁用素數集合 (F) 構造

[  
\frac1b=\sum\frac1{pq}  
]

且所有使用素數都避開 (F)，則可以反覆構造 (a) 份互不相交的表示：

[  
\frac1b=\sum_{i\in E_1}\frac1{p_iq_i},  
]

[  
\frac1b=\sum_{i\in E_2}\frac1{p_iq_i},  
]

……

[  
\frac1b=\sum_{i\in E_a}\frac1{p_iq_i},  
]

每次把已用素數加入禁用集。相加得

[  
\frac ab=\sum_{j=1}^a\sum_{i\in E_j}\frac1{p_iq_i}.  
]

因為每次用的素數不重複，所有分母 (p_iq_i) 也互異。因此原題成立。

整條鏈現在是：

}  
]

[  
\Downarrow

[

}

}

[

\boxed{

所以我們已經把 Erdős 原問題歸約到一個很具體的解析數論引理：

[  
\boxed{  
\sum_{p\sim P,\ p\text{ prime}} e(a p^{-1}/q)  
\text{ 不應有大偏差。}  
}  
]

這是目前最清楚、最有價值的「前進點」。

## 八、此路線的價值與風險

價值在於：它不是小例子、不是猜公式，而是給出一條全局證明方案。若外部解析引理可證，原題會整體解決。

風險在於：外部引理可能本身很深，甚至現有工具未必足夠。特別是 (p\sim q\sim P) 的素數互逆分佈，是非平凡的 Kloosterman 型問題。

但這已經比最初的問題清楚得多。現在真正要問的不是「這個奇怪 Erdős 問題怎麼辦」，而是：

[  
\boxed{  
\text{現有解析數論能否給出足夠弱的 prime reciprocal distribution estimate？}  
}  
]

這是下一個自然研究任務。
