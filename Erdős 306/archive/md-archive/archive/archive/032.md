好，繼續。但這一步必須先修正我們剛才幾乎要犯的一個大錯：**終端吸收若只看有限域殘差，是假的；必須同時控制實際大小。**

這不是細節，而是整條證明能否成立的核心。

## 1. 為什麼「任意殘差都可終端吸收」是假的？

末層 (Y)、終端層 (Z)，邊為 (yz)。每條邊的實際貢獻是

[  
\frac1{yz}.  
]

若 (Y,Z\subset[Q,2Q])，且

[  
|Y|=|Z|=Q^\alpha,\qquad \alpha<1,  
]

則終端區塊全部邊的總質量至多

# [  
\sum_{y\in Y,z\in Z}\frac1{yz}  
\asymp  
\frac{|Y||Z|}{Q^2}

Q^{2\alpha-2},  
]

這趨於 (0)。

可是若要封掉任意殘差 (\gamma_y\in\mathbb F_y)，它對應的 CRT 實際相容量

[  
\sum_{y\in Y}\frac{\gamma_y}{y}\pmod 1  
]

通常不是很小。終端邊總質量太小，根本不可能實現這種一般殘差。

所以先前說「雙向互逆分散 (\Rightarrow) 任意終端殘差可吸收」太強，甚至不對。

正確說法應該是：

[  
\boxed{  
\text{終端區塊只能吸收與其實際可提供質量相容的殘差。}  
}  
]

這正是對角 character 給我們的警告。

---

## 2. 這會不會毀掉整條路線？

不必然。但它要求我們把證明改成「殘差—質量聯合傳遞」，而不是只傳遞殘差。

原先我們追蹤的是局部模條件：

[  
\sum_{s\sim r}\frac1s\pmod r.  
]

現在還必須追蹤實際質量：

[  
\sum_{{u,v}\in E}\frac1{uv}.  
]

因為最終要的是精確等式

[  
\sum_{{u,v}\in E}\frac1{uv}=\frac1b.  
]

先前的「同餘 + (0<\text{sum}<1)」確實可以推出精確等式，但前提是我們真的能同時滿足那些同餘。若候選邊總質量太小，這些同餘其實不可能全被滿足；對角 Fourier 項正是在檢測這個不可能性。

因此正確框架應改成：

[  
\boxed{  
\text{在每一層同時傳遞殘差與質量。}  
}  
]

## 3. 單個 dyadic 終端塊質量太小，需要「質量水庫」

一個 (Y,Z\subset[Q,2Q]) 的終端塊提供的總質量至多約

[  
\left(\sum_{y\in Y}\frac1y\right)  
\left(\sum_{z\in Z}\frac1z\right).  
]

若 (Y,Z) 只取 (Q^\alpha) 個素數，這極小。即使取整個 dyadic 素數區間，質量也只有大約

[  
\left(\sum_{Q\le p<2Q}\frac1p\right)^2  
\asymp \frac1{(\log Q)^2}.  
]

仍然趨於 (0)。

所以若要提供固定大小，例如 (1/b)，不能只用一個 dyadic 塊。必須使用很多 dyadic 塊，形成一個「質量水庫」。

而這正好用到素數倒數和發散：

[  
\sum_p\frac1p=\infty.  
]

雖然每個 dyadic 區間貢獻小，但累積很多區間可以提供任意大的總質量。

因此應改為：

[  
\boxed{  
\text{取許多互相獨立的好塊 }(Y_j,Z_j)，  
\text{其總可用質量超過 }1.  
}  
]

每個塊由 Irving 定理保證可選成互逆分散；許多塊合起來提供足夠實際質量。

## 4. 修正後的終端吸收命題

原先錯誤版本是：

> 給定任意末層殘差 (\gamma)，一個終端塊 (Y,Z) 能封掉它。

正確版本應該是：

> 給定殘差 (\gamma) 和目標質量 (m)，若二者滿足 CRT 相容條件，且質量水庫總容量足夠，則許多終端塊的聯合可以實現它們。

形式上，設有許多好塊

[  
(Y_j,Z_j),\qquad j=1,\dots,J.  
]

在每個 (Z_j) 的每個 (z) 上選一列

[  
x^{(z)}\in K_z.  
]

總殘差為

[  
R_y=\sum_{j,z\in Z_j}x^{(z)}_y z^{-1}\pmod y,  
]

總質量為

[  
M=\sum_{j,z,y}\frac{x^{(z)}_y}{yz}.  
]

修正後要證：

[  
\boxed{  
\text{在許多好塊的聯合上，}\ (R,M)  
\text{ 對所有相容目標有足夠覆蓋。}  
}  
]

對角 Fourier 項不再是麻煩；它正是給出相容條件：

[  
M\equiv\sum_y\frac{R_y}{y}\pmod1.  
]

非對角項仍由 Irving 推出的互逆分散殺掉。  
對角主項則負責「質量分佈」。

## 5. 現在問題變成兩部分

### A. 非對角均勻性

這部分基本沿用前面：

Irving Theorem 1  
[  
\Rightarrow  
]  
好模數集合  
[  
\Rightarrow  
]  
互逆分散塊  
[  
\Rightarrow  
]  
非對角 Fourier 項衰減。

這條路仍然有效。

### B. 對角／質量方向的覆蓋

這是新出現、也是此前沒有處理好的部分。

我們需要證明：在許多終端塊中，實際質量

[  
M=\sum\frac{x_{y,z}}{yz}  
]

可以在需要的範圍內足夠細地調整，使它達到與殘差相容的目標值。

這不再是有限域問題，而是埃及分數問題本體。好消息是：我們現在不是從零開始，因為每個列 (x^{(z)}\in K_z) 有大量選擇；不同選擇給出不同質量。許多塊合起來，應該形成很密的可達質量集合。

但這一步需要新的引理：

[  
\boxed{  
\textbf{質量覆蓋引理：}  
\text{許多好終端塊的列核選擇，其實際質量可覆蓋所需區間。}  
}  
]

這才是現在真正剩下的核心。

## 6. 這是否又回到原問題？

有點接近，但不是完全回到原點。因為現在我們已經把模障礙處理掉了，剩下的是在大量受約束但高自由度的選擇中調整一個實數質量。這比原始「同時處理模與質量」要分離得多。

換句話說：

原問題是：

[  
\text{同時找 }pq\text{ 分母，使模與大小都對。}  
]

現在變成：

[  
\text{模方向由 Kloosterman/Fourier 控制；大小方向由質量水庫控制。}  
]

這是有進展的分解。

---

## 7. 下一步應當攻哪個引理？

現在最應該攻的是：

[  
\boxed{  
\text{質量覆蓋引理。}  
}  
]

一個可能的版本是：

設有許多互不相交的 dyadic 素數塊 (B_j)，每塊可選若干平衡列，且每塊的總可用質量約為

[  
w_j.  
]

若

[  
\sum_j w_j>1  
]

且每個 (w_j) 很小，則可通過選擇各塊中的列，使總質量落入任何指定相容小區間。

這有點像 subset-sum covering / coin problem：很多小權重，總和足夠大，能否覆蓋一段區間。

但它還有列核約束，所以不是普通子集和。不過每個塊內可選列很多，應該提供近似連續的質量調節。

## 8. 當前狀態重新評估

我們現在比上一輪更清楚，也更謹慎：

已經比較可靠的是：

[  
\boxed{  
\text{Irving Theorem 1 足以給出互逆分散塊，從而處理非對角模均勻性。}  
}  
]

尚未處理的是：

[  
\boxed{  
\text{對角／質量方向的覆蓋。}  
}  
]

所以不能再說「剩下只是終端吸收引理，應該容易」。終端吸收其實包含兩半：

1. 非對角 Fourier 衰減——Irving 幫助很大；
    
2. 對角質量覆蓋——仍需新的組合—埃及分數引理。
    

這是目前最真實的進度。

下一步若繼續，我建議直接研究「質量覆蓋引理」：看它能否由大量小 (pq)-權重的子集和密度推出，或者是否需要回到已知 squarefree Egyptian fraction 定理。
