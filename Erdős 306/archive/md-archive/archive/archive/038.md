好，先把「粗調引理」真正證掉。這一步其實很乾淨。

## 粗調引理

**命題。** 給定任意 (T>0)、任意 (\varepsilon>0)、任意有限禁用素數集合 (F)，存在互異的

[  
n_i=p_iq_i,\qquad p_i\ne q_i,\quad p_i,q_i\notin F,  
]

使

[  
0<T-\sum_i\frac1{n_i}<\varepsilon.  
]

換言之：只用兩個不同素數乘積作分母，且避開任意有限素數集，可以從下方逼近任意正實數。

---

## 證明

令

[  
\mathcal A_F={pq:p,q\text{ 為不同素數，且 }p,q\notin F}.  
]

按大小遞增排列：

[  
m_1<m_2<m_3<\cdots .  
]

我們要用級數

[  
\sum_{j=1}^{\infty}\frac1{m_j}.  
]

首先它發散。因為若取所有不在 (F) 中的素數集合 (P_F)，則

# [  
\sum_{\substack{p<q\p,q\in P_F}}\frac1{pq}

## \frac12\left[  
\left(\sum_{p\in P_F}\frac1p\right)^2

\sum_{p\in P_F}\frac1{p^2}  
\right].  
]

而

[  
\sum_p\frac1p=\infty,  
]

去掉有限多個素數仍發散；同時

[  
\sum_p\frac1{p^2}<\infty.  
]

所以

[  
\sum_{\substack{p<q\p,q\in P_F}}\frac1{pq}=\infty.  
]

又因為

[  
m_j\to\infty,  
]

所以

[  
\frac1{m_j}\to0.  
]

現在使用貪心選擇。令初始剩餘量

[  
R_0=T.  
]

依次檢查 (1/m_j)。若

[  
\frac1{m_j}<R_{j-1},  
]

就選取它，令

[  
R_j=R_{j-1}-\frac1{m_j}.  
]

若

[  
\frac1{m_j}\ge R_{j-1},  
]

就不選，令

[  
R_j=R_{j-1}.  
]

這樣 (R_j) 始終為正，且所選項互異。

若 (R_j) 不趨於 (0)，則存在 (\delta>0)，使最終

[  
R_j>\delta.  
]

但因為 (1/m_j\to0)，充分大的 (j) 都有

[  
\frac1{m_j}<\delta<R_j,  
]

所以從某處以後每一項都會被選取。可是尾級數

[  
\sum_{j\ge J}\frac1{m_j}  
]

發散，這會迫使 (R_j) 最終變成負數，矛盾。故

[  
R_j\to0.  
]

因此可取有限階段 (N)，使

[  
0<R_N<\varepsilon.  
]

於是

[  
0<T-\sum_{j\le N,;1/m_j\text{ 被選}}\frac1{m_j}<\varepsilon.  
]

證畢。

---

這個引理給我們一件重要的事：

[  
\boxed{  
\text{大小方向可以粗調到任意小剩餘，而且可避開任意有限素數集。}  
}  
]

也就是說，對原問題中的

[  
\frac1b  
]

我們可以先選一批合法 (pq)-分母，使

[  
0<r=\frac1b-\sum_{e\in E_0}\frac1{pq}<\varepsilon.  
]

這裏 (r) 可以小到我們想要的尺度，並且後續精修所需的素數可以全部避開前面已用過的素數。

但粗調本身不保證模條件正確；它只是把「大小」壓到一個很小的剩餘。接下來精修塊要做兩件事：

[  
\boxed{  
\text{精確補上 }r,\quad \text{並同時修正粗調留下的 CRT 殘差。}  
}  
]

所以現在路線變成：

1. 粗調引理：已證。
    
2. 選精修尺度 (Q,N)，使二部塊總質量與 (r) 同階。
    
3. 用 Irving 定理選互逆分散的 (Y,Z\subset[Q,2Q])。
    
4. 證二部精修 Fourier 引理。
    

下一個真正關鍵就是第 4 步。
