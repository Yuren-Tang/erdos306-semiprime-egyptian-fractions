是的，這一步**很應該檢索現有工具**。而且檢索結果給出一個有用判斷：我們需要的主弧工具不一定要自己從零造，已有兩類相近技術可借：

1. **weighted Bernoulli sums 的 local limit theorem**。Giuliano–Weber 有一篇明確題為 _Local Limit Theorems in some Random models from Number Theory_，研究「weighted sums of Bernoulli variables」，且正是用 characteristic function 方法處理數論隨機模型中的局部極限定理。這與我們的  
    [  
    \sum X_{pq}/(pq)  
    ]  
    非常接近。([arXiv](https://arxiv.org/abs/1502.05939?utm_source=chatgpt.com "Local Limit Theorems in some Random models from Number Theory"))
    
2. **Littlewood–Offord / subset-sum anti-concentration**。這一傳統本來就是研究「子集和落入小區間或小集合的數量」；它對我們控制主弧之外的集中現象有參考價值。不過經典 Littlewood–Offord 多給上界／反集中，不直接給「精確命中」的 local CLT。([維基百科](https://en.wikipedia.org/wiki/Littlewood%E2%80%93Offord_problem?utm_source=chatgpt.com "Littlewood–Offord problem"))
    

此外，Weber 2023 年那篇使用 Bernoulli part 來處理 lattice local limit / almost sure local limit 的文章，也說明「格點上的獨立變量和」已有一套較成熟框架；我們的變量就是 Bernoulli 型格點變量，只是權重非常特殊。([arXiv](https://arxiv.org/abs/2305.19372?utm_source=chatgpt.com "A general correlation inequality for level sets of sums of independent random variables using the Bernoulli part with applications to the almost sure local limit theorem"))

所以回答你的括號問題：**是，需要檢索，而且已有工具看起來相當相關。** 這意味著我們不必把「主弧 local CLT」當成全新難題；更合理的工作是把我們的權重陣列核對到這些定理的假設中。

現在繼續推。假設我們有一個二部主質量塊

[  
C\times Y,  
]

其中

[  
C\subset [R,2R],\qquad Y\subset [Q,2Q],  
]

邊權為

[  
a_{c,y}=\frac1{cy}.  
]

取 Bernoulli 變量

[  
X_{c,y}\sim\operatorname{Bernoulli}(\theta).  
]

主弧要研究

[  
X=\sum_{c\in C,y\in Y}\frac{X_{c,y}}{cy}.  
]

這正是一個 weighted Bernoulli sum。其均值與方差為

[  
\mu=\theta\sum_{c,y}\frac1{cy},  
]

[  
\sigma^2=\theta(1-\theta)\sum_{c,y}\frac1{c^2y^2}.  
]

若

[  
|C|=M,\quad |Y|=N,  
]

且 (c\asymp R,\ y\asymp Q)，則

[  
\mu\asymp \theta\frac{MN}{RQ},  
]

[  
\sigma^2\asymp \theta(1-\theta)\frac{MN}{R^2Q^2},  
]

所以

[  
\sigma\asymp \frac{\sqrt{MN}}{RQ}.  
]

最大單項權重是

[  
a_{\max}\asymp \frac1{RQ}.  
]

因此

[  
\frac{a_{\max}}{\sigma}  
\asymp  
\frac{1}{\sqrt{MN}}.  
]

只要

[  
MN\to\infty,  
]

最大單項遠小於標準差。這正是三角陣列 CLT / local CLT 通常需要的 Lindeberg 型條件。

這是一個好徵兆。

## 1. 格點問題

整數化後，令

[  
L=L_0\prod_{c\in C}c\prod_{y\in Y}y.  
]

則

[  
\frac1{cy}  
]

對應整數權重

[  
A_{c,y}=\frac{L}{cy}.  
]

我們要在格點上命中某個整數目標。local CLT 需要知道 span，也就是所有可取值所在格點的最大公因數。

如果只看 (C\times Y) 這一塊，則

[  
\gcd{L/(cy):c\in C,y\in Y}  
]

可能仍有一些固定因子，尤其是 (L_0) 和不在某條邊中的素數。但一旦把其他層一起加入，例如 (B\times C) 和 (Y\times Z)，整體權重 gcd 應為 (1)，因為每個素數都出現在某些邊中，且不是所有權重都含它。

所以 local CLT 應該作用在整體鏈圖上，而非孤立主塊。主塊提供方差；附層幫助打破格點週期與修正模條件。

這點很重要：

[  
\boxed{\text{主塊給 Gaussian mass；全鏈給 span }1\text{ 與 CRT 覆蓋。}}  
]

---

## 2. 已有 local CLT 工具怎樣接？

Giuliano–Weber 那篇明確研究 weighted Bernoulli variables，且說他們用 characteristic function method 證 local limit theorem，並把它用於數論隨機模型。這與我們準備做的 Fourier 主弧分析方法一致。([arXiv](https://arxiv.org/abs/1502.05939?utm_source=chatgpt.com "Local Limit Theorems in some Random models from Number Theory"))

一般 local CLT 的證明大致分三段：

小頻率：

[  
\prod_i \mathbb E e(t a_i X_i)  
]

近似

[  
e^{it\mu-\frac12t^2\sigma^2}.  
]

中頻：

證 characteristic function 衰減。

大頻：

利用格點 span / aperiodicity 證沒有週期性峰值。

我們的非對角 Irving 分析主要處理的是多維 CRT character 的「大頻非對角」；主弧 local CLT 處理對角 character 的小頻與中頻。

所以現有工具的角色是：

[  
\boxed{  
\text{local CLT 工具處理對角 character；Irving 處理非對角 character。}  
}  
]

這個分工現在很清楚。

## 3. 還不能直接套用的一點

標準 weighted Bernoulli local CLT 通常處理的是一維和

[  
\sum a_i X_i  
]

落在其自然格點上。但我們的完整 Fourier 空間是

[  
\mathbb Z/L\mathbb Z\simeq \prod_{p\mid L}\mathbb F_p.  
]

所以我們不能只證一維 local CLT；還要證非對角 CRT directions 沒有額外質量。這就是 Irving 部分。

換言之，現有 local CLT 不會單獨解決整題，但它可以解決主弧半邊。

## 4. 現在可把證明分成三個可查模塊

### 模塊 A：Irving 互逆分散

已由你上傳的 Irving Theorem 1 強烈支持。它給

[  
\sum_{q\sim Q}\max_{(a,q)=1}|S_q(a;x)|  
]

的 power-saving 平均界；在 (x\asymp Q) 時足以推出好模數與短弧避免。

### 模塊 B：weighted Bernoulli local CLT

用 Giuliano–Weber 類工具或直接 characteristic function 證：

[  
\sum X_{uv}/(uv)  
]

在主弧中有局部高斯近似。([arXiv](https://arxiv.org/abs/1502.05939?utm_source=chatgpt.com "Local Limit Theorems in some Random models from Number Theory"))

### 模塊 C：格點 aperiodicity / Littlewood–Offord

用 span (1)、多層鏈圖、以及反集中工具控制中高頻週期峰值。Littlewood–Offord 傳統提供反集中背景，但具體仍要針對我們權重寫。([維基百科](https://en.wikipedia.org/wiki/Littlewood%E2%80%93Offord_problem?utm_source=chatgpt.com "Littlewood–Offord problem"))

這三塊合起來比「再加一層抽象」更像真正可執行的證明計畫。

## 5. 下一步的具體工作

我現在會選擇不再增加層數，而是固定一個足夠一般的鏈模型：

[  
P_0=B,\quad P_1,\quad P_2,\quad P_3,  
]

只用相鄰層邊：

[  
P_0P_1,\quad P_1P_2,\quad P_2P_3.  
]

其中 (P_1P_2) 是主質量塊，(P_0P_1) 處理原邊界，(P_2P_3) 提供終端封口與 span smoothing。

然後證一個定理：

> 若 (P_1,P_2,P_3) 是足夠大的 Irving-good 素數塊，且 (P_1P_2) 的總質量調到 (1/b) 量級，則隨機邊子集以正概率命中 (L/b)。

這比「任意鏈」更具體，也避免抽象失控。

## 6. 目前走通徵兆更新

我會這樣評價：

- **非對角：強好徵兆。** Irving 定理幾乎正中所需。
    
- **主弧：中等好徵兆。** weighted Bernoulli local CLT 有現有工具，但需核假設。
    
- **整合：仍硬。** 因為要把一維 local CLT 和多維 CRT Fourier 併在一個證明裏。
    
- **反例障礙：目前未見。** squarefree 條件與 CRT 結構吻合，gcd/span 也可望處理。
    

所以這條路不是「越走越玄」，而是逐步落到兩套既有工具上：Kloosterman-over-primes 與 weighted Bernoulli local CLT。真正剩下的是高難度整合。
