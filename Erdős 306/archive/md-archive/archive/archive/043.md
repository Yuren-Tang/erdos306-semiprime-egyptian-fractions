好，現在固定模型，不再繼續抽象化。取四層：

[  
P_0=B,\qquad P_1=C,\qquad P_2=Y,\qquad P_3=Z,  
]

只允許相鄰層邊：

[  
B\times C,\qquad C\times Y,\qquad Y\times Z.  
]

其中

[  
B={p:p\mid b}  
]

固定，(C,Y,Z) 是可選的大素數集合，互不相交且避開一切已用素數。

目標是證明存在邊集 (E) 使

[  
\sum_{{u,v}\in E}\frac1{uv}=\frac1b.  
]

---

## 1. 為什麼四層夠用？

三層 (B-Y-Z) 的問題是 (B) 太小，(B\times Y) 的自由度有限。加一層 (C) 後，主質量可由

[  
C\times Y  
]

提供，而 (B) 只負責原始邊界條件。這樣：

- (B\times C)：把 (B) 的指定殘差送入 (C)；
    
- (C\times Y)：提供主要質量與大量自由度；
    
- (Y\times Z)：封掉終端殘差、提供非對角平滑。
    

這個模型比任意鏈簡潔，但自由度已足夠。

---

## 2. 質量可調

令

[  
A_C=\sum_{c\in C}\frac1c,\qquad A_Y=\sum_{y\in Y}\frac1y.  
]

主塊 (C\times Y) 的總可用質量是

[  
W_{CY}=A_C A_Y.  
]

由於

[  
\sum_p\frac1p=\infty,  
]

我們可以選 (C,Y) 使

[  
W_{CY}  
]

為 (1/b) 的同階量，例如

[  
W_{CY}\asymp \frac{2}{b}.  
]

然後在 (C\times Y) 上用 Bernoulli 概率 (\theta) 選邊，使

[  
\mathbb E\sum_{c,y}\frac{X_{c,y}}{cy}  
]

接近

[  
1/b.  
]

(B\times C) 與 (Y\times Z) 的質量可安排為較小修正，或一起納入總期望。

所以質量來源沒有問題。

---

## 3. Span / gcd 沒有顯然障礙

令

[  
L=\prod_{r\in B\cup C\cup Y\cup Z}r.  
]

邊 (uv) 的整數權重是

[  
w_{uv}=\frac{L}{uv}.  
]

若整個圖連通，且每個頂點至少接一條邊，則這些 (w_{uv}) 的 gcd 通常為 (1)。在四層鏈中，只要各層非空且邊類足夠多，對任一素數 (r\mid L)，存在含 (r) 的邊 (rv)，使

[  
L/(rv)  
]

不含因子 (r)。因此 (r) 不整除所有權重。對所有 (r) 如此，故

[  
\gcd{w_{uv}:uv\in\mathcal E}=1.  
]

所以格點 span 可為 (1)，沒有顯然同餘障礙。

這點非常重要：local CLT 至少不會被一個固定 gcd 阻止。

---

## 4. Fourier 相位

一個 character 為

[  
\lambda=(\lambda_r)_{r\in B\cup C\cup Y\cup Z},  
\qquad \lambda_r\in\mathbb F_r.  
]

邊 (uv) 的相位是

# [  
\phi_{uv}(\lambda)

\frac{\lambda_u(v^{-1}\bmod u)}u  
+  
\frac{\lambda_v(u^{-1}\bmod v)}v.  
]

若存在整數 (h) 使

[  
\lambda_r\equiv h\pmod r  
]

對所有頂點 (r) 成立，則

[  
\phi_{uv}(\lambda)\equiv \frac h{uv}\pmod1.  
]

這是對角主弧，對應實數質量。

若不是這樣，則是非對角 character，應由相鄰層分散殺掉。

---

## 5. 非對角衰減機制

定義離對角距離：

[  
D(\lambda)=\min_h  
#{r:\lambda_r\not\equiv h\pmod r}.  
]

若 (D(\lambda)>0)，存在一些異常頂點。

若異常在 (C) 或 (Y)，主塊 (C\times Y) 就提供大量邊來衰減。  
若異常在 (Z)，由 (Y\times Z) 衰減。  
若異常在 (B)，由 (B\times C) 衰減；因 (B) 固定小，只需 (C) 在模 (p) 下足夠分散，這可初等安排。

對大層間的分散，由 Irving Theorem 1 提供好素數塊。更精確地，可選 (C,Y,Z) 使：

對每個 (y\in Y)、非零 (a\in\mathbb F_y),

[  
\sum_{c\in C}\left|\frac{a c^{-1}}y\right|^2\gg |C|;  
]

對每個 (c\in C)、非零 (a\in\mathbb F_c),

[  
\sum_{y\in Y}\left|\frac{a y^{-1}}c\right|^2\gg |Y|;  
]

同理 (Y,Z) 互相分散。

然後可望得到

[  
|\widehat\mu(\lambda)|  
\le  
\exp(-\kappa N D(\lambda)),  
]

其中 (N) 是大層大小的下界。

距離為 (d) 的 character 數大約是

[  
\exp(O(d\log Q)).  
]

只要

[  
N\gg\log Q,  
]

非對角總和可被壓住。

這部分目前最有把握。

---

## 6. 主弧：weighted Bernoulli local CLT

對角 character 對應一維和

[  
X=\sum_{uv\in\mathcal E}\frac{X_{uv}}{uv}.  
]

主塊 (C\times Y) 可設為主要方差來源。若

[  
C,Y\subset[Q,2Q],\qquad |C|=|Y|=N,  
]

則

[  
\sum_{c,y}\frac1{cy}\asymp \frac{N^2}{Q^2},  
]

[  
\sum_{c,y}\frac1{c^2y^2}\asymp \frac{N^2}{Q^4}.  
]

若 (X_{c,y}\sim \operatorname{Bernoulli}(\theta))，則

[  
\sigma^2\asymp \theta(1-\theta)\frac{N^2}{Q^4},  
\qquad  
\sigma\asymp \frac{N}{Q^2}.  
]

最大單項權重約為

[  
Q^{-2}.  
]

所以

[  
\frac{\max a_i}{\sigma}\asymp \frac1N.  
]

若 (N\to\infty)，這是良好的 Lindeberg 型條件。這正適合 weighted Bernoulli local CLT。

所以主弧也有合理希望。

---

## 7. 但是：單一 dyadic 主塊的可調總質量小

若 (C,Y\subset[Q,2Q]) 且 (|C|=|Y|=N=Q^\alpha)，則

[  
W_{CY}\asymp Q^{2\alpha-2}.  
]

若 (\alpha<1)，它趨於 (0)。要達到 (1/b) 固定量，不能只用一個 dyadic 主塊，除非取 (N) 接近 (Q)，即幾乎整個素數區間。

可行方案有兩個：

### 方案 A：取 (N\asymp Q/\sqrt b)

若 (b) 固定，且 (Q) 大，取 (C,Y) 為正比例大小的素數子集，則

[  
N^2/Q^2\asymp 1/b.  
]

但 (N) 不能超過區間內素數數量

[  
\asymp Q/\log Q.  
]

所以最大 (W_{CY}) 約

[  
\frac{1}{(\log Q)^2},  
]

仍趨於 (0)。

因此單一 dyadic 區間不可能提供固定質量。因為全部素數 (p\in[Q,2Q]) 的倒數和只有約 (1/\log Q)。

### 方案 B：使用很多 dyadic blocks

這才是正確方案。取一串區間

[  
[Q_j,2Q_j],  
]

在每個區間選 (C_j,Y_j)，並使用 (C_j\times Y_j)。每個塊貢獻約

[  
(\sum_{c\in C_j}1/c)(\sum_{y\in Y_j}1/y).  
]

若取整個或大比例素數區間，每塊約

[  
\frac{1}{(\log Q_j)^2}.  
]

許多塊相加可達 (1/b)，因為

[  
\sum_j \frac1{(\log Q_j)^2}  
]

是否發散取決於 (Q_j) 選法。若取 (Q_j) 增長不太快，例如 (Q_j\sim j) 的 dyadic 不合適；但可用連續素數區間累積，本質上

[  
\sum_p\frac1p  
]

發散，故可以使層倒數和累積到任意大。

更簡單：不必把 (C,Y) 放在同一 dyadic 區間。選 (C) 與 (Y) 為長範圍素數集合，使

[  
\sum_{c\in C}\frac1c\asymp 1,  
\qquad  
\sum_{y\in Y}\frac1y\asymp 1/b.  
]

但 Irving 估計在 dyadic 同尺度上使用方便。於是可把長範圍拆成多個 dyadic 子塊，逐塊處理非對角。

---

## 8. 這是否又使模型複雜？

是，但這是必要複雜，不是空抽象。原因簡單：任何單一高 dyadic 區間中的全部素數倒數和都太小，無法提供固定質量。因此必須累積多個尺度。

所以主質量層應是：

[  
\bigcup_j C_j \times Y_j  
]

而不是單一 (C\times Y)。

非對角分析逐塊使用 Irving；主弧 local CLT 用整個權重陣列。

這與現有 weighted Bernoulli local CLT 更吻合：它本來就處理三角陣列、多尺度權重。

---

## 9. 簡化主質量設計

可以取 (C=Y=P)，其中 (P) 是一大段素數，不限於 dyadic，例如

[  
P={p: R\le p\le S}  
]

並使用兩份拷貝 (C,Y) 中的不同素數以避免同一頂點問題。其倒數和

[  
\sum_{p\in P}\frac1p  
\sim \log\log S-\log\log R.  
]

可調到

[  
A:=\sum_{p\in P}\frac1p\asymp \sqrt{1/b}.  
]

則

[  
W_{CY}\asymp A^2\asymp 1/b.  
]

這樣質量足夠。

然後把 (P) 分解成 dyadic blocks，用 Irving 在每個 block pair 上處理非對角。若兩端素數尺度不同，Irving 的範圍仍可能覆蓋一定比例；若尺度差太大，需分層連接相近尺度的 blocks。

所以主質量層可用「長集合 + dyadic analysis」方式處理。

---

## 10. 目前最穩健的形態

我會把最終模型設計成：

- (B)：固定邊界；
    
- (C)：長素數集合，倒數和約 (A);
    
- (Y)：另一個長素數集合，倒數和約 (1/(bA));
    
- (Z)：終端集合，用來封 (Y) 方向與提供 span/分散。
    

候選邊：

[  
B\times C,\qquad C\times Y,\qquad Y\times Z.  
]

主質量由 (C\times Y) 提供：

[  
\left(\sum_{c\in C}\frac1c\right)  
\left(\sum_{y\in Y}\frac1y\right)\asymp\frac1b.  
]

把 (C,Y,Z) 各自拆成 dyadic blocks；在相近尺度 block pairs 上用 Irving；在尺度不相近處可少用或不用邊。

這樣既有足夠質量，又可使用 dyadic Kloosterman estimates。

---

## 11. 現在的下一個具體任務

要證：

[  
\boxed{  
\text{長素數集合 }C,Y\text{ 可分解為 Irving-good dyadic blocks，且 }C\times Y\text{ 的 Bernoulli 和滿足 local CLT。}  
}  
]

這又分兩步：

1. **質量設計：** 選 (C,Y) 使倒數和合適。
    
2. **分析設計：** 只在 good block pairs 之間放邊，仍保留足夠質量與方差。
    

如果刪掉壞 block pairs 之後，質量仍足夠，則模型可行。Irving 給「幾乎所有模數好」，所以壞部分應可忽略。

---

## 12. 小結

你剛才的遲疑是對的：模型確實又多一層抽象。但現在我們明白為什麼：**固定質量 (1/b) 不能由單一大 dyadic 塊提供**。必須使用長素數集合或多尺度 blocks。

不過這不是任意加層，而是由

[  
\sum_{Q\le p<2Q}\frac1p\asymp\frac1{\log Q}  
]

這個事實強迫的。

目前路線仍有走通感，但主弧部分的技術工作比一開始預期更大。下一步應證「質量設計」：給定 (b)，如何選長素數集合 (C,Y)，使 (C\times Y) 提供正確量級，同時每個 dyadic block 足夠大以使用 Irving 與 local CLT。
