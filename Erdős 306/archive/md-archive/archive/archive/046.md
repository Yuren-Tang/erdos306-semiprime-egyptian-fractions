好。現在精算次弧衰減與 character 數的平衡。

取一個 dyadic block

[  
\mathcal P_k={q\text{ prime}:2^k\le q<2^{k+1}}.  
]

設 (q\in\mathcal P_k) 是一個「異常頂點」，意思是相對於某個最佳對角整數 (h)，有

[  
\lambda_q\not\equiv h\pmod q.  
]

令

[  
a_q=\lambda_q-h\not\equiv0\pmod q.  
]

若 (q) 與另一個 block (\mathcal P_\ell) 相連，則對邊 (pq) 的相位殘餘中有項

[  
\frac{a_q(p^{-1}\bmod q)}{q}.  
]

若 (\mathcal P_\ell) 在模 (q) 下是 Irving-good，則對固定小 (\eta>0)，至少固定比例的 (p\in\mathcal P_\ell) 滿足

[  
\left|  
\frac{a_q p^{-1}}q  
\right|\ge \eta.  
]

對這些邊，每個 Bernoulli Fourier 因子有模長

[  
\left|1-\theta+\theta e(\phi_{pq})\right|  
\le 1-c_\theta  
]

某個固定 (c_\theta>0)，前提是 (\theta) 遠離 (0,1)。所以若一個異常 (q) 連到 (M) 個好鄰居素數，便可得到衰減

[  
\exp(-cM).  
]

這是非常強的。

---

## 1. 一個異常頂點有多少好鄰居？

對 (q\in\mathcal P_k)，Irving 可控的鄰接 levels (\ell) 滿足

[  
\frac23 k\le \ell\le \frac32 k.  
]

在 level (\ell) 中，素數數量約為

[  
#\mathcal P_\ell\asymp \frac{2^\ell}{\ell}.  
]

若我們連接全部可控鄰接 levels，則鄰居數量約為

[  
M_k  
\asymp  
\sum_{\ell=2k/3}^{3k/2}\frac{2^\ell}{\ell}.  
]

這個和由最大 (\ell\approx 3k/2) 主導：

[  
M_k\asymp \frac{2^{3k/2}}{k}.  
]

這巨大無比。

而 (q) 的 character 選擇數只有

[  
q\asymp 2^k,  
]

即對數成本

[  
\log q\asymp k.  
]

所以單個異常頂點帶來的衰減

[  
\exp\left(-c\frac{2^{3k/2}}{k}\right)  
]

遠遠壓過 character 數

[  
2^k.  
]

這表明：**若使用全部可控鄰接 levels，次弧衰減綽綽有餘。**

---

## 2. 其實不用那麼多邊

使用所有可控邊會讓總質量可能太大，但我們可以用 Bernoulli 概率 (\theta) 調整期望。不過 (\theta) 若太小，單邊衰減變弱，約為

[  
\theta |\phi|^2.  
]

如果總質量 (W) 被調到 (1/b) 量級，而候選邊總質量很大，則

[  
\theta=\frac{1/b}{W}  
]

可能很小。這會削弱衰減。

所以更好是選候選邊集本身使

[  
W\asymp 1/b,  
]

再取固定 (\theta), 例如 (\theta=1/2)。

這意味着不能使用過多候選邊；要稀疏化 block graph，但仍保留每個頂點足夠多好鄰居。

---

## 3. 可以稀疏化到什麼程度？

對異常 (q\sim2^k)，為壓過 character 選擇數 (q), 其實只需

[  
M_k\gg k  
]

個好鄰居即可，因爲衰減

[  
e^{-cM_k}  
]

要壓過

[  
e^{O(k)}.  
]

取

[  
M_k=Ck  
]

已夠。

每個鄰接素數 (p\sim 2^\ell) 對總質量貢獻約

[  
\frac1{pq}\asymp 2^{-k-\ell}.  
]

若選 (\ell\asymp k)，每個邊質量約

[  
2^{-2k}.  
]

每個頂點選 (Ck) 條邊，該頂點附近總質量約

[  
Ck\cdot 2^{-2k}.  
]

所有 level (k) 中有約

[  
2^k/k  
]

個頂點，所以 level (k) 總質量約

# [  
\frac{2^k}{k}\cdot Ck\cdot 2^{-2k}

C2^{-k}.  
]

對 (k) 求和收斂，且很小。這不足以提供固定 (1/b) 質量。

所以若只保留每頂點 (O(k)) 邊，次弧夠，但主質量不夠。  
質量需要更多邊。

---

## 4. 質量需求下的平均度

要在 level (k) 附近得到質量約 (1/k)，如前面總質量發散計算，需要該 level 有總邊質量

[  
\asymp \frac1k.  
]

每條同尺度邊質量約

[  
2^{-2k}.  
]

所以需要約

[  
\frac{2^{2k}}{k}  
]

條邊。

level (k) 有約

[  
2^k/k  
]

個頂點，因此平均每個頂點 degree 約

# [  
\frac{2^{2k}/k}{2^k/k}

2^k.  
]

也就是平均度接近 level 中頂點總數，幾乎是稠密連接。這符合原來的完整 block-pair 圖。

所以質量需求本身強迫很多邊；而很多邊自然給巨大次弧衰減。這是好消息。

---

## 5. 若跨尺度連接，質量與度如何？

若 level (k) 連到約 (k) 個鄰近 levels (\ell\asymp k)，且使用完整 block pairs，則 level (k) 的鄰接質量約

# [  
u_k\sum_{\ell\asymp k}u_\ell  
\asymp  
\frac1k\cdot 1

\frac1k.  
]

這就是需要的發散量。

同時每個頂點 (q\in\mathcal P_k) 連到約

[  
\sum_{\ell\asymp k}\frac{2^\ell}{\ell}  
\asymp \frac{2^{3k/2}}k  
]

個鄰居，如果允許到 (3k/2)。衰減極強。

因此採用完整 Irving-compatible block graph 不僅質量足，且次弧大幅過剩。

問題只剩：候選總質量可能太大，但可以選 (K) 使總質量剛好達到 (2/b)，再取 (\theta=1/2)。因為總質量隨 (K) 像 (\log K) 增長，可停在合適位置。

---

## 6. 次弧求和的粗略閉合

設所有使用素數的最大尺度為 (K)，最小尺度為 (k_0)。總 character 數約

[  
\exp\left(\sum_{p\in P}\log p\right).  
]

但不能全體粗暴求和；要按異常集合分解。

如果 character 在某個頂點 (q\sim2^k) 異常，且該頂點有

[  
M_k\gg 2^{ck}  
]

有效好邊，則 Fourier 衰減含

[  
\exp(-cM_k).  
]

該頂點的非對角選擇數最多 (q\asymp2^k)，其他對角部分由 (h) 控制。因此單頂點異常的總貢獻約

[  
2^k\exp(-cM_k),  
]

極小。

若有多個異常頂點，衰減乘積更強；即使邊的貢獻重疊，也可用選取匹配或半數獨立鄰域避免重複計數。因 graph 很稠密，可抽取足夠多邊，使每個異常頂點得到 (M_k'\gg 2^{ck}) 個不重複有效邊。

所以非對角總和應閉合。

這是強正面徵兆。

---

## 7. 但要注意「近對角」而非「異常頂點」問題

如果 (\lambda_q\equiv h\pmod q) 對大多數頂點成立，但少數異常，剛才處理可行。

若 (h) 不唯一？因各 (q) 不同，(\lambda_q) 是模 (q) residue。對角型的 (h) 可在區間

[  
0\le h<L  
]

內有很多。但若 (\lambda_q\equiv h\pmod q) 對所有 (q)，CRT 決定一個 class mod (L)，這就是對角 family。主弧只應包括小 (|h|) 的頻率；大 (h) 由一維 local CLT 衰減。

對非對角分類，我們可以對每個 (h) 定義異常集合，再取最小異常數。這會引入許多 (h) 的選擇，但主弧/對角分析本來要對所有 (h) 處理。

所以非對角求和需要避免多算：

[  
\sum_{\lambda\notin\mathcal D}  
]

可按最佳 (h) 分解。(h) 的數量約 (L)，很大。但 Fourier inversion 本身也有 (L) characters；主弧 local CLT 已處理對角 (h)。非對角若對每個 (h) 都求一遍，可能太大。

更好的方式是不用顯式枚舉 (h)，而用「若 (\lambda) 非對角，則存在一條邊上殘餘相位大」的直接能量估計：

[  
\sum_{uv\in\mathcal E}|\phi_{uv}(\lambda)|^2  
]

大，除非 (\lambda) 屬於對角族。這避免最佳 (h) 計數。

這是更正確的 Fourier 次弧表述。

---

## 8. 核心次弧命題

應證：

**次弧能量命題。**  
對 dyadic block graph 的候選邊 (\mathcal E)，存在一個主弧集合 (\mathfrak M) 對應對角低頻，使得若 (\lambda\notin\mathfrak M)，則

[  
\sum_{uv\in\mathcal E}  
\left|  
\frac{\lambda_u v^{-1}}u+  
\frac{\lambda_v u^{-1}}v  
\right|^2  
\gg  
\Omega(\lambda),  
]

其中 (\Omega(\lambda)) 足夠大，使

[  
\prod_{uv\in\mathcal E}  
|1-\theta+\theta e(\phi_{uv})|  
]

可求和。

Irving 定理提供對每個 fixed (u) 和非零殘差的平均分散。把這些局部估計拼起來，應能證能量命題。

這是完整次弧的正式目標。

---

## 9. 次弧目前看來不是瓶頸

因爲 graph 稠密得多，衰減富餘巨大。真正瓶頸仍是主弧 local CLT 與主弧/次弧邊界的精確切分。

主弧應包括對角 (h) 滿足

[  
|h|\lesssim 1/\sigma.  
]

對角中高頻由一維 weighted Bernoulli local CLT 控制。  
非對角由能量命題控制。

這是標準 circle method 形態：

[  
\text{small diagonal frequencies}=\text{major arcs},  
]

[  
\text{large diagonal + non-diagonal}=\text{minor arcs}.  
]

---

## 10. 下一步最應做什麼？

應該轉向主弧 local CLT 的核對。因為次弧衰減量看起來非常富餘；若主弧 local CLT 可用，整體會很有希望。

具體要證：

令

[  
X=\sum_{uv\in\mathcal E}\frac{X_{uv}}{uv},  
\quad X_{uv}\sim\mathrm{Bernoulli}(\theta).  
]

在 dyadic block graph 上，若

[  
\mathbb E X=1/b,  
]

則在格點 (L^{-1}\mathbb Z) 上，

[  
\mathbb P(X=1/b)  
]

有正主項，約為

[  
\frac{1}{\sigma L}.  
]

這可望由 weighted Bernoulli local CLT 給出。下一步就是把其條件逐項核對：

1. 方差 (\sigma^2\to0) 但 (L\sigma\to\infty);
    
2. 最大權重 (o(\sigma));
    
3. span (1);
    
4. characteristic function 在中頻無大峰。
    

其中第 4 點可能與次弧能量命題部分重疊。
