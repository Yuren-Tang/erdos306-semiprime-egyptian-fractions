好。現在轉向主弧 local CLT 的條件核對。

我們的隨機和是

[  
X=\sum_{e={p,q}\in\mathcal E}\frac{X_e}{pq},  
\qquad X_e\sim \operatorname{Bernoulli}(\theta),  
]

其中 (\mathcal E) 是 dyadic block graph 上的候選邊。整數化後，

[  
S=LX=\sum_{e\in\mathcal E}X_e\frac{L}{pq},  
]

目標是

[  
S=\frac Lb.  
]

主弧 local CLT 大致需要四件事：

[  
\boxed{  
\text{方差足夠大、最大單項足夠小、span 為 }1,\text{ 中頻無大峰。}  
}  
]

逐項看。

---

## 1. 方差

# [  
\sigma^2=\operatorname{Var}(X)

\theta(1-\theta)\sum_{e={p,q}\in\mathcal E}\frac1{p^2q^2}.  
]

這個和由最小尺度 (k_0) 附近主導。若候選圖包含從 (k_0) 到 (K) 的 Irving-compatible dyadic block pairs，則粗略有

[  
\sigma^2  
\asymp  
\sum_{k\ge k_0}\sum_{\ell\asymp k}  
\frac1{k2^k}\frac1{\ell 2^\ell}.  
]

最大貢獻來自 (k,\ell\asymp k_0)，所以

[  
\sigma^2\asymp \frac{1}{k_0^2,2^{2k_0}},  
]

忽略常數與緩慢因子。故

[  
\sigma\asymp \frac{1}{k_0,2^{k_0}}.  
]

這趨於 (0)，但沒問題；我們只需要格點數 (L\sigma) 很大。因 (L) 是許多大素數的乘積，極巨大，所以

[  
L\sigma\to\infty  
]

毫無問題。

---

## 2. 最大單項

最大邊權來自最小尺度：

# [  
a_{\max}

\max_{pq\in\mathcal E}\frac1{pq}  
\asymp  
2^{-2k_0}.  
]

因此

# [  
\frac{a_{\max}}{\sigma}  
\asymp  
\frac{2^{-2k_0}}{1/(k_0 2^{k_0})}

k_0 2^{-k_0}\to0.  
]

這正是 Lindeberg 型條件：

[  
\boxed{a_{\max}=o(\sigma).}  
]

所以小頻率 Gaussian 近似應該成立。

---

## 3. Span

整數權重是

[  
w_{pq}=\frac{L}{pq}.  
]

若候選圖 (\mathcal E) 覆蓋每個頂點，即每個素數 (r\mid L) 至少出現在某條候選邊中，則對每個 (r) 可取一條含 (r) 的邊 (rs)，此時

[  
w_{rs}=\frac{L}{rs}  
]

不含因子 (r)。因此 (r) 不整除所有權重。對所有 (r\mid L) 如此，得

[  
\gcd_{pq\in\mathcal E}w_{pq}=1.  
]

所以格點 span 是 (1)。

這也很好。

---

## 4. 中頻特徵函數：真正的主弧硬點

對角 Fourier 係數是

# [  
\Phi(h)

\prod_{pq\in\mathcal E}  
\left(  
1-\theta+\theta e\left(\frac{h}{pq}\right)  
\right).  
]

local CLT 要控制

[  
\sum_h \Phi(h)e(-h/b)  
]

中除小頻率

[  
|h|\lesssim 1/\sigma  
]

以外的部分。

用基本估計：

[  
|1-\theta+\theta e(t)|  
\le  
\exp\bigl(-c_\theta |t|^2\bigr).  
]

所以

[  
|\Phi(h)|  
\le  
\exp\left(  
-c_\theta  
\sum_{pq\in\mathcal E}  
\left|\frac{h}{pq}\right|^2  
\right).  
]

因此需要證明：

[  
\boxed{  
\sum_{pq\in\mathcal E}  
\left|\frac{h}{pq}\right|^2  
\text{ 很大}  
}  
]

對所有中頻 (h) 成立。

這是純一維權重反集中問題。它不需要 Irving；只看 (pq) 作為整數的分佈。

---

## 5. 中頻能否證？

先看一個尺度 (pq\asymp M)。若 (h) 滿足

[  
1/\sigma\ll |h|\ll M,  
]

則

[  
\left|\frac{h}{pq}\right|\asymp \frac{|h|}{M}  
]

對這個尺度的所有邊差不多，平方和約為

[  
#E_M\frac{h^2}{M^2}.  
]

這正是 Gaussian 區域延伸。

若

[  
h\gg M,  
]

則 (h/(pq)) 會在模 (1) 中振盪。需要證明不會對大多數 (pq) 都接近整數。

這類問題與乘法表、divisor distribution 有關：有沒有很多 (pq) 近似整除 (h)？對固定 (h)，能整除 (h) 的 (pq) 很少，因為 (p,q) 是大素數。接近整數也應很少，除非 (h) 很特殊。但要 uniform 控制所有 (h) 直到 (L)，不小心會很難。

這裏可以借助多尺度：即使某個 (h) 對某一尺度的 (pq) 有週期峰值，也不可能對許多不同尺度同時有峰值。這是多尺度設計的另一個好處。

---

## 6. 可用一個簡單分段策略

對任意非零 (h)，選一個 dyadic 邊尺度 (M) 使

[  
M\asymp |h|  
]

或

[  
M\asymp |h|/T  
]

讓 (h/(pq)) 落在非小頻區域。因候選圖包含從 (2^{2k_0}) 到很大尺度的一系列 (pq)，總能找到某個尺度使 (h/(pq)) 不全接近整數。

具體地，若邊尺度為

[  
pq\asymp 2^m,  
]

可用尺度 (m) 在一個長區間內變化。對每個 (h)，選 (m) 使

[  
2^m\in [4|h|,8|h|]  
]

則

[  
\frac{h}{pq}\in [1/8,1/4]  
]

量級，遠離整數，對該尺度所有邊都有固定相位距離。若這個尺度存在於候選圖中，就得到大量衰減。

若 (|h|) 超出最大邊尺度，則改用模 (L) 對稱，考慮 (L-h)。在 Fourier on (\mathbb Z/L\mathbb Z) 中，頻率可取

[  
|h|\le L/2.  
]

但 (h) 可能遠大於所有 (pq)。這時 (h/(pq)) 的小數部分不受上述簡單尺度控制。

然而若 (h) 很大，要求

[  
h/(pq)  
]

接近整數等價於 (h) 接近 (pq) 的倍數。對很多不同 (pq) 同時成立非常不可能。需要一個 divisor/large sieve 型引理。

---

## 7. 中高頻一維引理

我們可以提出：

**乘積權重一維反聚集引理。**  
對 dyadic block graph 的邊集合 (\mathcal E)，存在 (c>0)，使對所有整數

[  
1/\sigma\ll |h|\le L/2,  
]

有

[  
\sum_{pq\in\mathcal E}  
\left|  
\frac{h}{pq}  
\right|^2  
\ge c,R(h),  
]

其中 (R(h)\to\infty) 足以讓

[  
\sum_h \exp(-cR(h))=o(1/\sigma)  
]

或足以壓過主弧。

這是主弧 local CLT 的核心一維難題。

---

## 8. 是否已有工具？

這就是為何需要 weighted Bernoulli local CLT。現有 local CLT 定理通常把這種中高頻反聚集作為假設，例如 characteristic function decay 或 aperiodicity condition。若 Giuliano–Weber 類結果能直接處理形如

[  
a_i=1/(pq)  
]

的權重，那最好；否則我們要自己證這個一維反聚集引理。

這裏應當回頭查 local CLT 工具的假設，而不是盲證。

但從直覺看，因 (pq) 是大量不同半素數，多尺度分佈，這個一維反聚集應該成立。

---

## 9. 非對角與中頻可能可合併

注意對角中高頻 (h) 其實是所有坐標 (\lambda_p=h\bmod p)。若 (h) 很大，它在每個模 (p) 看起來隨機，不像小對角。

也就是說，從 CRT 坐標看，大 (h) 的對角 character 可能可以用 Irving 同樣殺掉，因為 (\lambda_p=h\bmod p) 不是一個「小共同整數」；但對每個 (p) 它仍然是同一整數 (h) mod (p)，邊相位精確等於

[  
h/(pq),  
]

所以 Irving 的互逆分散退化為一維半素數權重問題。

因此大對角需要單獨一維處理，不能靠 Irving。

---

## 10. 主弧條件目前的狀態

前三個條件：

[  
\sigma,\quad a_{\max}=o(\sigma),\quad \mathrm{span}=1  
]

都很好。

真正剩下：

[  
\boxed{  
\text{對角中高頻反聚集。}  
}  
]

這是主弧 local CLT 的核心。它是現有 weighted Bernoulli local CLT 工具可能覆蓋的地方；若不能直接覆蓋，就需要自己證一個半素數倒數權重的 characteristic function bound。

所以當前最精確的瓶頸是：

[  
\boxed{  
\sum_{pq\in\mathcal E}\left|h/(pq)\right|^2  
\text{ 的 uniform 下界。}  
}  
]

這比「local CLT」四字更具體，也更可攻。

---

## 11. 下一步

我建議下一步直接研究這個一維反聚集引理。先從簡單情形開始：

取所有半素數 (pq) 於一個尺度窗口

[  
M\le pq<2M  
]

或多尺度窗口，估計

[  
\sum_{pq}  
\left|\frac{h}{pq}\right|^2.  
]

看能否對所有 (h) 給出足夠下界。若這一步能做，主弧 local CLT 就有實質希望。
