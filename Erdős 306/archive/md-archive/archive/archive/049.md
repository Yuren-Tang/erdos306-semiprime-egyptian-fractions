你說得對。剛才的分類有一部分是探索性拆解：小頻率、大對角、可整除支持、互素高頻……這有助於發現障礙，但如果最後還停在一堆 case 裏，證明會很難看，也很可能不穩。現在應把它們統一。

統一的對象應該不是 (h) 的大小，也不是 (h) 被哪些素數整除，而是下面這個量：

# [  
\mathcal E(h)

\sum_{pq\in\mathcal E}  
\left|  
\frac{h}{pq}  
\right|^2.  
]

對角 Fourier 係數滿足

[  
|\Phi(h)|  
\le  
\exp(-c_\theta \mathcal E(h)).  
]

所以主弧問題的本質只有一句話：

[  
\boxed{  
\mathcal E(h)\text{ 何時小？}  
}  
]

如果能證明

[  
\mathcal E(h)\text{ 小}  
\quad\Longrightarrow\quad  
h\text{ 是真正的小頻率，}  
]

那所有分類都被統一了。

---

## 1. 正確的統一命題

我們希望證一個形如：

**對角頻率反聚集命題。**  
存在常數 (C,c>0)，使對所有整數 (h) with (1\le |h|\le L/2)，若

[  
|h|>C/\sigma,  
]

則

# [  
\mathcal E(h)

\sum_{pq\in\mathcal E}  
\left|  
\frac{h}{pq}  
\right|^2  
\ge  
A(h),  
]

其中 (A(h)) 足以使

[  
\sum_{|h|>C/\sigma}\exp(-cA(h))  
=o(1/\sigma).  
]

若這成立，對角中高頻全部被處理；小頻率

[  
|h|\le C/\sigma  
]

由 Gaussian 主弧處理。

這比前面所有分類都乾淨。

---

## 2. 怎樣證 (\mathcal E(h)) 小會逼出小頻率？

若

[  
\mathcal E(h)  
]

很小，則對大多數候選半素數 (n=pq)，

[  
h/n  
]

非常接近整數。也就是說，存在整數 (m_n)，使

[  
|h-m_n n|\ll n\epsilon.  
]

這表示大量半素數 (n) 都是 (h) 的「近因子」。

所以 (\mathcal E(h)) 小等價於：

[  
\boxed{  
h\text{ 被大量候選半素數近似整除。}  
}  
]

這可以用「近因子密度」來處理，而不必按 (h) 是否含某些素因子分類。

具體地，對一個尺度區間

[  
n\asymp N,  
]

若

[  
\left|\frac{h}{n}\right|<\delta,  
]

則 (h) 落在某個倍數 (mn) 的區間

[  
|h-mn|<\delta n.  
]

等價於 (n) 落在短區間

[  
\frac{h}{m+\delta}<n<\frac{h}{m-\delta}.  
]

對固定 (m)，這個 (n)-區間長度約

[  
\frac{\delta h}{m^2}.  
]

而 (n\asymp h/m)，所以長度約

[  
\delta n.  
]

問題變成：候選半素數在許多短相對區間中有多少個。

若候選半素數分佈不過度集中，就可證只有約 (O(\delta)) 比例的 (n) 滿足。那麼平方距離平均有常數下界。

這是更統一的路。

---

## 3. 我們的候選 (n=pq) 分佈是否足夠平滑？

候選 (n) 不是所有半素數，而是來自 Irving-compatible block graph：

[  
p\in\mathcal P_k,\quad q\in\mathcal P_\ell,\quad \ell\asymp k.  
]

在尺度

[  
n\asymp 2^{k+\ell},  
]

這些 (pq) 的數量很多。半素數在乘法尺度上應該相當平滑。對短相對區間

[  
[N,(1+\delta)N],  
]

候選 (pq) 的數量應約為 (\delta) 倍總數，至少不應超過 (C\delta) 倍加小誤差。

這類估計比 prime Kloosterman 簡單得多；它是半素數計數問題，可由素數定理、Brun–Titchmarsh、甚至較粗的篩法處理。

所以對角高頻反聚集可能不需要深新工具，只需候選半素數的相對區間分佈上界。

---

## 4. 形式化：半素數短區間上界

需要一個引理：

**半素數短區間上界。**  
令 (\mathcal N_M) 是候選半素數 (n=pq) 落在某個 dyadic 尺度 (M\le n<2M) 的集合。則對任意區間 (I\subset[M,2M]) 長度 (|I|=\delta M)，有

[  
#(\mathcal N_M\cap I)  
\le  
C\delta,#\mathcal N_M  
+  
\text{acceptable error}.  
]

若這成立，則對固定 (h)，集合

[  
\left{n\in\mathcal N_M:  
\left|\frac hn\right|<\eta  
\right}  
]

可按倍數 (m) 分解成若干相對長度 (O(\eta)) 的區間，其總覆蓋比例 (O(\eta))。於是至少固定比例的 (n) 有

[  
\left|\frac hn\right|\ge\eta.  
]

這給

[  
\sum_{n\in\mathcal N_M}  
\left|\frac hn\right|^2  
\gg  
#\mathcal N_M  
]

對某些合適尺度 (M)。

這就是強衰減。

---

## 5. 何時選得到合適尺度 (M)？

如果 (h) 是中頻，存在候選邊尺度 (M) 與 (h) 可比，則容易。

如果 (h) 遠大於所有候選 (n)，則 (m) 很大，仍可以做上述分解。對 (n\in[M,2M])，(m\approx h/M)。相鄰倍數區間的中心

[  
h/m  
]

間距約

[  
h/m^2\asymp M^2/h.  
]

而每個壞區間長度約

[  
\eta M.  
]

若

[  
\eta M \gg M^2/h,  
]

壞區間可能高度重疊，甚至覆蓋全部。這發生於

[  
h\gg M/\eta.  
]

也就是當 (h) 遠大於 (M) 時，條件 (|h/n|<\eta) 可能在尺度上快速振盪並覆蓋很多。這個區間覆蓋方法不夠。

但直觀上 (h/n) 隨 (n) 變化很快，fractional parts 應更均勻，不是更壞。要證則需要 exponential sum 或 van der Corput 類估計：

[  
\sum_{n\in\mathcal N_M} e(h/n).  
]

這是另一個 Kloosterman-like sum，但一維半素數版本。

## 6. 又回到指數和？

對角高頻需要控制

[  
\sum_{p,q} e\left(\frac{h}{pq}\right).  
]

這是 bilinear exponential sum with reciprocal phase. 形式上比 prime Kloosterman (e(a\bar p/q)) 不同，但也屬於 reciprocal exponential sums。

如果 (q) 固定，作 (p) 變量，phase 是 (h/(pq))，不是 modulo (q) inverse，而是真實 reciprocal。這是分析估計，可能可由 van der Corput/Robert–Sargos 類處理。

但也許 local CLT 文獻正是處理這種 characteristic function，不必我們從頭估。

因此你的提醒再次有效：分類後應統一為 characteristic function decay，並尋找現有 weighted Bernoulli local CLT 的假設，而不是自己分情況證。

## 7. 統一方案：把主弧工具作為黑箱

現在最穩健的研究設計是：

1. 將對角主弧 local CLT 明確作為一個 **weighted Bernoulli local CLT lemma**。
    
2. 核對現有工具能否覆蓋權重  
    [  
    a_n=1/(pq)  
    ]  
    with (n) in our dyadic block graph.
    
3. 若不能，單獨研究 characteristic function  
    [  
    \prod_{pq}(1-\theta+\theta e(h/(pq))).  
    ]
    

這比繼續手工分類好。

所以分類的探索價值是：它告訴我們 local CLT 黑箱需要處理哪些隱患，例如 (h=L/q) 這類高頻週期。但正式證明應該由一個統一的 aperiodicity condition 處理。

---

## 8. 可採用的 local CLT 條件

一個常見充分條件是：

設整數權重 (a_i) span 為 (1)，Bernoulli 變量 (X_i)，方差 (\sigma_L^2=\operatorname{Var}\sum a_iX_i)。若

[  
\sup_{\delta\le |t|\le \pi}  
\prod_i |1-\theta+\theta e(t a_i)|  
\to0  
]

足夠快，且最大權重 (o(\sigma_L))，則 lattice local CLT 成立。

在我們的歸一化變量中，這個條件正是：

[  
\sum_i |t a_i/(2\pi)|^2  
]

有下界。

所以要證的不是許多分類，而是一條 aperiodicity estimate：

[  
\boxed{  
\forall h\notin \text{small arc},\quad  
\sum_{pq\in\mathcal E}\left|\frac{h}{pq}\right|^2  
\text{ large}.  
}  
]

這是統一目標。

---

## 9. 是否能由隨機稀疏化改善？

如果 deterministic aperiodicity 難，可以隨機選候選邊集 (\mathcal E) 本身。隨機半素數權重通常更不可能有週期共振。比如從 Irving-compatible graph 中先隨機保留邊，使總質量仍約 (2/b)，再對保留邊做 Bernoulli 選取。

但隨機稀疏化會減少質量與次弧衰減；不過我們有富餘。它可能讓 aperiodicity 更易證：對固定 (h)，若原圖有不少好邊，隨機保留後仍有；對所有 (h) union bound 可能困難，因 (h) 多達 (L)，但可用 net/entropy or high-level probabilistic method。

這是一個備選方向。

---

## 10. 我目前自衡後的決策

我不應再繼續細分 (h)。應把所有對角高頻統一為 local CLT 的 aperiodicity condition，然後查或證這個 condition。

因此下一步應是：

[  
\boxed{  
\text{明確陳述一個 Weighted Bernoulli Lattice Local CLT 黑箱，列出其假設，然後逐項對我們的 }pq\text{ 權重核對。}  
}  
]

這樣若現有工具能直接套，我們就走通很大一段；若不能，也能清楚看到缺哪個假設。

所以，繼續時我會不再新增分類，而是寫出這個 local CLT black-box lemma，然後把我們已知的：

[  
a_{\max}=o(\sigma),\quad \mathrm{span}=1,  
]

以及待證的

[  
\text{aperiodicity estimate}  
]

放進同一框架。
