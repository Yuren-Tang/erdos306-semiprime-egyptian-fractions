## 1. 基本設定

令 (P) 是一組大素數頂點，(G=(P,E)) 是我們的 dyadic prime block graph。對每個頂點 (p\in P)，給一個 residue

[  
a_p\in\mathbb F_p.  
]

對邊 (pq\in E)，定義 CRT glue：

[  
H_{pq}(a)  
\in \mathbb Z/pq\mathbb Z  
]

由

[  
H_{pq}\equiv a_p\pmod p,\qquad  
H_{pq}\equiv a_q\pmod q  
]

決定。取對稱代表

[  
H_{pq}^{*}\in\left(-\frac{pq}{2},\frac{pq}{2}\right].  
]

邊相位為

[  
\phi_{pq}(a)=\frac{H_{pq}^{*}}{pq}\pmod1.  
]

Fourier 大係數意味很多邊有

[  
\left|\frac{H_{pq}^{*}}{pq}\right|  
]

很小。

## 2. 要證的協議命題

我們需要如下型態的定理。

**CRT interval agreement lemma，粗略版。**  
給定一個足夠稠密、尺度擴張良好的 dyadic prime graph (G=(P,E))。若 residue assignment

[  
a=(a_p)_{p\in P}  
]

滿足

[  
\sum_{pq\in E}  
\left|  
\frac{H_{pq}^{*}}{pq}  
\right|^2  
\le R,  
]

則存在一個整數 (h)，使

[  
a_p\equiv h\pmod p  
]

對大多數 (p\in P) 成立，且剩下的例外頂點數／權重由 (R) 控制。

更強、對 Fourier 求和更有用的版本是：

滿足上式的 assignments (a) 的數量至多

[  
\exp(O(R))\cdot |\mathfrak M|,  
]

其中 (\mathfrak M) 是對角主弧族，即

[  
a_p\equiv h\pmod p  
]

且 (h) 屬於小頻率範圍。

這句話的意思是：

[  
\boxed{  
\text{低能量 character 幾乎只能是主弧 character。}  
}  
]

## 3. 為什麼這正好夠用？

Fourier 係數有估計

[  
|\widehat\mu(a)|  
\le  
\exp\left(  
-c\sum_{pq\in E}  
\left|  
\frac{H_{pq}^{*}}{pq}  
\right|^2  
\right).  
]

若低能量 assignments 很少，高能量 assignments 又有指數衰減，則

[  
\sum_{a\notin\mathfrak M}|\widehat\mu(a)|  
]

可控。

而主弧 (\mathfrak M) 由一維 weighted Bernoulli local CLT 處理，給出正主項。

所以整個 Fourier 證明可閉合為：

[  
\text{主弧 local CLT}  
+  
\text{CRT interval agreement}  
+  
\text{Irving 分散}  
\Longrightarrow  
\text{存在表示}.  
]

## 4. 這個 lemma 與 Irving 的關係

Irving 定理主要保證：在 dyadic prime blocks 中，對固定模數 (q)，素數逆元

[  
p^{-1}\pmod q  
]

不會集中在短弧內。這對局部約束很重要。

在 agreement lemma 中，固定一個頂點 (q)，它的許多鄰居 (p) 對 (a_q) 施加條件：

[  
\left|  
\frac{a_q p^{-1}}q+\beta_p  
\right|\le\delta.  
]

這是許多不同斜率 (p^{-1}) 的模區間交集。Irving 或其推論正可用來保證這些斜率足夠分散，不會讓很多不同條件異常重合。

所以 Irving 不只是用來做簡單「非對角衰減」，也可作為 agreement lemma 中的局部隨機性來源。

## 5. 更可攻的局部版本

先不要證整張 graph。先證一個 block-pair 版本：

令

[  
P\subset[X,2X],\qquad Q\subset[Y,2Y]  
]

是兩個 Irving-good prime blocks，且 (X,Y) 在 Irving 範圍內。

若 residues

[  
a_p\in\mathbb F_p,\quad b_q\in\mathbb F_q  
]

使得對至少 (1-\varepsilon) 比例的 pairs ((p,q)\in P\times Q)，

[  
\left|  
\frac{a_p q^{-1}}p+  
\frac{b_q p^{-1}}q  
\right|\le\delta,  
]

則存在一個整數 (h) 使

[  
a_p\equiv h\pmod p  
]

對大多數 (p\in P)，且

[  
b_q\equiv h\pmod q  
]

對大多數 (q\in Q)。

這是二部原子版。

若二部原子版成立，整個 dyadic graph 版本可用 graph connectivity / expansion 拼起來：相鄰 blocks 各自得到 (h_{ij})，overlap consistency 迫使這些 (h_{ij}) 相同，最後得到全局 (h)。

## 6. 二部原子版如何證？

可用「測試鄰居」法。

固定 (p\in P)。對每個 (q\in Q)，條件限制 (a_p) 落在一個密度約 (2\delta) 的模區間：

[  
a_p\in A_{q,b_q}\subset\mathbb F_p.  
]

若 (p) 與許多 (q)'s 的邊都通過，則

[  
a_p\in \bigcap_{q\in Q_p}A_{q,b_q}.  
]

若取 (t\sim C\log p) 個測試鄰居 (q_1,\dots,q_t)，並且這些斜率 (q_i^{-1}\pmod p) 足夠分散，則對任意 shifts (b_{q_i})，交集

[  
\bigcap_i A_{q_i,b_{q_i}}  
]

應非常小。

這表示：(a_p) 幾乎由少量鄰居的 residues 決定。

然後在整個二部圖中，許多 (p)'s 被同一小測試集 (Q_0) 控制；這會壓縮 (P)-側 residues 的自由度。再反過來用 (P)-側測試集控制 (Q)-側 residues。反覆一次，就可逼出全局 agreement。

這是標準 agreement/entropy 壓縮的輪廓。

## 7. 為什麼不是空話？

因為模區間交集有清楚的 Fourier 形式：

# [  
#\left{  
a:  
\left|  
\frac{a r_i}{p}+\beta_i  
\right|\le\delta  
\quad\forall i  
\right}

p\sum_{\sum \xi_i r_i=0}  
\prod_i \widehat{1_{I_i}}(\xi_i).  
]

只要 (r_i=q_i^{-1}\pmod p) 沒有太多小係數線性關係，交集就接近隨機大小

[  
(2\delta)^t p.  
]

這是可檢驗的有限域事實。可以通過隨機選 (q_i)、large sieve、或 prime inverse 分散估計來證。

因此 agreement lemma 並非神秘命題；它分解成：

1. 模區間交集估計；
    
2. expander/agreement propagation。
    

這兩個都是有既有工具味道的問題。
