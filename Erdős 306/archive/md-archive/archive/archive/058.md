是，這個新階段應該檢索，而且檢索後可以更清楚地定位：我們不是在找一個單一現成定理直接砸下來，而是在把問題拆成幾個已有理論相鄰的模塊。

目前找到的相關工具大致分三類：

第一，**weighted Bernoulli local limit theorem**。Giuliano–Weber 明確研究 weighted sums of Bernoulli variables，並用 characteristic function 方法處理數論隨機模型中的 local limit theorem；這正對應我們的主弧 local CLT。([arXiv](https://arxiv.org/abs/1502.05939?utm_source=chatgpt.com "Local Limit Theorems in some Random models from Number Theory"))

第二，**agreement testing / direct product testing**。Dinur–Filmus–Harsha 的 agreement tests 論文研究「局部片段平均相容是否推出存在全局對象」這類 local-to-global 現象；這與我們的 CRT interval agreement lemma 在精神上很接近。([arXiv](https://arxiv.org/abs/1711.09426?utm_source=chatgpt.com "Agreement tests on graphs and hypergraphs")) 另外 direct product testing 的一般化也提供了「測試圖具備何種擴張性即可推出全局 agreement」的語言。([arXiv](https://arxiv.org/abs/1901.06220?utm_source=chatgpt.com "Towards a General Direct Product Testing Theorem"))

第三，**characteristic function / Littlewood–Offord 型反集中**。這類工具不一定直接給我們要的 CRT 版本，但它提供了「大 Fourier 係數意味着結構／低熵」的標準思想。

所以現在應該繼續，但要把工作方式改成：**不要幻想已有文獻直接證 Erdős 問題；而是把我們的 CRT agreement lemma 對齊 agreement testing 與 inverse Littlewood–Offord 的語言。**

## 1. 現在的核心 lemma 應該這樣寫

我們把它叫作：

[  
\boxed{\text{CRT interval agreement lemma}}  
]

一個可研究版本是：

設 (G=(P,E)) 是 dyadic prime block graph。每個頂點 (p\in P) 有 residue

[  
a_p\in\mathbb F_p.  
]

對邊 (pq)，令 (H_{pq}^{*}) 是 CRT glue 的對稱代表：

[  
H_{pq}^{_}\equiv a_p\pmod p,\qquad  
H_{pq}^{_}\equiv a_q\pmod q,  
]

[  
H_{pq}^{*}\in(-pq/2,pq/2].  
]

定義能量

[  
\mathcal Q(a)=  
\sum_{pq\in E}  
\left(\frac{H_{pq}^{*}}{pq}\right)^2  
]

或用距離符號寫作

[  
\mathcal Q(a)=  
\sum_{pq\in E}  
\left|  
\frac{H_{pq}^{*}}{pq}  
\right|^2.  
]

我們想證：

[  
\mathcal Q(a)\text{ 小}  
\quad\Longrightarrow\quad  
a_p\equiv h\pmod p  
]

對大多數 (p) 成立，其中 (h) 是某個小整數；或者至少證滿足 (\mathcal Q(a)\le R) 的 assignments (a) 數量很少，少到足以求和。

這正是 agreement testing 的 local-to-global 形式：很多邊上局部 CRT 相容，推出存在一個全局 (h)。

---

## 2. 為什麼 agreement testing 的工具有關？

Agreement testing 一般研究這樣的問題：你在許多局部集合上看到一些局部資料；若局部資料在重疊處經常一致，是否存在一個全局資料解釋它們。Dinur–Filmus–Harsha 的摘要正是說這種 local-to-global phenomenon 是 agreement tests 的核心，並研究圖與超圖上的 agreement theorem。([arXiv](https://arxiv.org/abs/1711.09426?utm_source=chatgpt.com "Agreement tests on graphs and hypergraphs"))

我們這裏的「局部資料」不是集合限制，而是邊 (pq) 上的 CRT glue；「一致」不是完全相等，而是 (H_{pq}^{*}) 落在短區間內。這不是文獻中的標準形式，但結構相似：

[  
\text{邊上局部相容}  
\Rightarrow  
\text{頂點 residues 來自全局 }h.  
]

所以現成 theorem 未必直接能套，但它給我們應該使用的證明語言：測試圖擴張性、局部約束、全局解碼、majority/entropy decoding。

## 3. 模區間交集是 agreement lemma 的原子步

固定頂點 (p)。若它與許多鄰居 (q_i) 的邊都通過，即

[  
\left|  
\frac{a_p q_i^{-1}}p+  
\frac{a_{q_i}p^{-1}}{q_i}  
\right|  
\le\delta,  
]

那麼每個 (q_i) 都把 (a_p) 限制在 (\mathbb F_p) 中一個密度約 (2\delta) 的模區間裏。

所以我們需要證：

若 (q_i^{-1}\pmod p) 足夠分散，則許多這類模區間的交集很小。

形式是：

[  
#\left{  
a\in\mathbb F_p:  
\left|  
\frac{a q_i^{-1}}p+\beta_i  
\right|\le\delta  
\quad (1\le i\le t)  
\right}  
]

應遠小於 (p)，最好在 (t\asymp C\log p) 時變成 (O(1)) 或至少 (p^\varepsilon)。

這一點可用 Fourier 展開：

# [  
#A

p  
\sum_{\sum \xi_i q_i^{-1}=0}  
\prod_i \widehat{1_{I_i}}(\xi_i).  
]

所以關鍵是 (q_i^{-1}) 沒有太多小係數線性關係。這可望由隨機選測試鄰居、large sieve，或 Irving 型 prime reciprocal estimates 支持。

## 4. 現在不是完全轉向，而是形成「三工具架構」

目前證明計畫可以穩定為三個模塊：

[  
\boxed{  
\text{Irving prime Kloosterman}  
}  
]

負責素數逆元分散，尤其保證 dyadic block graph 中可選出 good slopes。

[  
\boxed{  
\text{weighted Bernoulli local CLT}  
}  
]

負責小對角主弧；Giuliano–Weber 類結果正與此相關。([arXiv](https://arxiv.org/abs/1502.05939?utm_source=chatgpt.com "Local Limit Theorems in some Random models from Number Theory"))

[  
\boxed{  
\text{CRT interval agreement / entropy compression}  
}  
]

負責高頻與非主弧 characters：大 Fourier 係數若存在，必須來自全局小 (h)，否則低熵、可求和。

這三者合起來，才是目前最可信的完整路線。

## 5. 下一步：先證弱版模區間交集

我們先不證整個 CRT agreement theorem，而證一個可用的弱版：

**弱模區間交集引理。**  
對每個大素數 (p)，可以從某個 Irving-compatible 鄰接素數塊中選出

[  
q_1,\dots,q_t,\qquad t=C\log p,  
]

使得對所有 shifts (\beta_i)，

[  
#\left{  
a\in\mathbb F_p:  
\left|  
\frac{a q_i^{-1}}p+\beta_i  
\right|\le\delta  
\quad\forall i  
\right}  
\le p^{\varepsilon}.  
]

這比要求交集 (\le1) 弱，但已能大幅壓縮單頂點 entropy。

證明方向：

1. 隨機選 (q_i)，希望 (q_i^{-1}) 像隨機斜率；
    
2. 對固定 shifts，用二階矩或 Chernoff 控制交集；
    
3. 對所有 shifts 用離散化與 union bound；
    
4. prime 條件由 Irving 或素數在模 (p) 的分佈處理。
    

---

## 6. 為什麼這可能夠？

若每個頂點 residue 從 (p) 個可能壓到 (p^\varepsilon) 個，單次還不夠；但在 expander-like block graph 上反覆傳播，entropy 可繼續下降。Agreement testing 文獻正是處理這種「局部壓縮如何推出全局一致」的問題。([arXiv](https://arxiv.org/abs/1711.09426?utm_source=chatgpt.com "Agreement tests on graphs and hypergraphs"))

最終想得到：

[  
#{a:\mathcal Q(a)\le R}  
\le  
|\mathfrak M|\exp(O(R)+o(\log L)),  
]

其中 (\mathfrak M) 是小主弧族。這樣 Fourier 求和就有希望閉合。
