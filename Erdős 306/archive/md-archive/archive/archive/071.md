好。現在看完整二部圖

[  
P\times Q.  
]

先取最理想的情形：每個 (p\in P) 連到所有 (q\in Q)。我們研究低能 assignments：

[  
a_p\in\mathbb F_p,\qquad b_q\in\mathbb F_q,  
]

使

# [  
Q(a,b)

\sum_{p\in P,q\in Q}  
\left|  
\frac{a_pq^{-1}}p+  
\frac{b_qp^{-1}}q  
\right|^2  
]

很小。

目標是證明：低能 assignments 基本來自一個全局整數 (h)：

[  
a_p\equiv h\pmod p,\qquad b_q\equiv h\pmod q.  
]

---

## 1. 固定 (b_Q)，看每個 (a_p)

先固定 (Q)-側 residues

[  
b=(b_q)_{q\in Q}.  
]

對某個 (p\in P)，能量中與 (p) 有關的部分是

# [  
Q_p(a_p;b)

\sum_{q\in Q}  
\left|  
\frac{a_pq^{-1}}p+  
\frac{b_qp^{-1}}q  
\right|^2.  
]

若

[  
Q_p(a_p;b)  
]

小，則對多數 (q\in Q),

[  
\frac{a_pq^{-1}}p+  
\frac{b_qp^{-1}}q  
]

接近整數。

對固定 (b)，這是對 (a_p) 的大量模區間約束。若 (Q) 足夠大且 (q^{-1}\bmod p) 分散，那麼符合小 (Q_p) 的 (a_p) 應該很少。

因此對固定 (b)，低能 assignments 的數量可估為

[  
\prod_{p\in P} L_p(b),  
]

其中

# [  
L_p(b)

#{a_p\in\mathbb F_p: Q_p(a_p;b)\le R_p}.  
]

如果大多數 (L_p(b)) 很小，則總 entropy 被壓縮。

---

## 2. 什麼時候 (L_p(b)) 不小？

若存在一個整數 (h)，使

[  
b_q\equiv h\pmod q  
]

對大多數 (q)，則取

[  
a_p\equiv h\pmod p  
]

會讓所有相位變成

[  
\frac{h}{pq},  
]

很小。此時 (L_p(b)) 至少含有這個 (a_p)，甚至含有少量附近的 (h')。

這是主弧，應當保留。

反過來，若 (b_q) 不接近同一個 (h)，則不同 (q) 對 (a_p) 給出的模區間位置應該彼此不協調。那麼 (L_p(b)) 應很小。

所以完整二部 agreement 可用這個二分：

[  
\boxed{  
Q\text{-側已接近全局 }h  
\Rightarrow P\text{-側也被迫接近 }h;  
}  
]

[  
\boxed{  
Q\text{-側不接近任何 }h  
\Rightarrow P\text{-側 entropy 大幅下降。}  
}  
]

這是 local-to-global 的核心。

---

## 3. 可以定義 (Q)-側的「候選 (h) 集」

給定 (b=(b_q))，定義

[  
H_Q(B)=  
\left{  
h\in\mathbb Z:  
\sum_{q\in Q}  
\left|  
\frac{h-b_q}{q}  
\right|^2  
\le B  
\right}.  
]

直觀上，這些 (h) 是能解釋 (Q)-側 residues 的全局整數。

若 (H_Q(B)) 很小，則 (b_Q) 有低 complexity。  
若 (H_Q(B)) 為空或很小，那麼 (P)-側可選 (a_p) 應被強烈限制。

理想的不等式：

[  
\log #{a_P:Q(a,b)\le R}  
\le  
\log |H_Q(B)| + C R.  
]

然後再對 (b_Q) 做同樣分析，最終得到全局 (h)。

---

## 4. 一個更直接的二部熵遞推

設

# [  
N_{P,Q}(R)

#{(a,b):Q(a,b)\le R}.  
]

想證

[  
N_{P,Q}(R)  
\le  
\sum_h e^{C R_h},  
]

其中 (h) 是全局解釋。

可以用條件熵：

令 ((A,B)) 在低能集合上均勻。則

[  
H(A,B)=H(B)+H(A\mid B).  
]

若給定 (B)，低能約束使

[  
H(A\mid B)  
\le  
H(h\mid B)+C R.  
]

如果 (B) 本身不接近任何 (h)，則 (H(A\mid B)) 很小，但 (H(B)) 可能仍大。對稱地也有

[  
H(B\mid A)  
]

小。兩邊合起來應迫使共同低維參數 (h) 存在。

這就是 agreement testing 中常見的互信息增益：

[  
A\text{ and }B\text{ are highly constrained across many edges}  
\Rightarrow  
\text{both determined by common latent variable }h.  
]

---

## 5. 直接用兩個 (p) 比較 (b_q)

換一個角度。若兩個 (P)-側頂點

[  
p_1,p_2  
]

都與同一批 (q)'s 形成小相位，則對每個 (q),

[  
H_{p_1q}^{_}\equiv H_{p_2q}^{_}\equiv b_q\pmod q.  
]

如果 (H_{p_1q}^{_},H_{p_2q}^{_}) 在絕對值上都不太大，便可推出它們相等，從而兩個 (p) 共享同一局部 (h_q)。再對很多 (q) 交叉，應得到全局 (h)。

對完整二部圖，選許多 (p_1,p_2,q) 可放大這種一致性。這就是四環／多環 agreement。

但絕對小問題仍在，所以需以能量分層處理：

- 邊 glue 很小者提供 equality propagation；
    
- glue 較大者支付能量；
    
- 若支付能量太多，就不在 (N(R)) 內。
    

這可以轉成 entropy-energy bound。

---

## 6. 完整二部中的一個可證弱版

我們可以先證非常低能版本：

假設所有邊滿足

[  
|H_{pq}^{*}|<M  
]

其中

[  
M<\min(P,Q)/2.  
]

則對共享頂點的任意兩條邊，glue 值相等。因完整二部圖連通，所有邊 glue 等於同一個整數 (h)，從而

[  
a_p\equiv h\pmod p,\quad b_q\equiv h\pmod q.  
]

這個命題完全成立。

證明：

若 (p q_1) 與 (p q_2) 共享 (p)，則

[  
H_{pq_1}^{_}\equiv H_{pq_2}^{_}\pmod p.  
]

二者絕對值均 (<p/2)，故相等。  
連通傳播，全部等於同一 (h)。

這是 exact/very-low-energy agreement。

---

## 7. 中能版本：bad edges 計數

若不是所有邊都有 (|H_{pq}^{*}|<M)，令 bad edges 為

[  
|H_{pq}^{*}|\ge M.  
]

每條 bad edge 的能量至少

[  
\left(\frac{M}{pq}\right)^2.  
]

在 (p\sim X,q\sim Y) 的 block 中，這約

[  
\frac{M^2}{X^2Y^2}.  
]

若總能量 (\le R)，bad edges 數至多

[  
\ll R\frac{X^2Y^2}{M^2}.  
]

若取

[  
M\asymp X  
]

（假設 (X\le Y)），則 bad edges 數至多

[  
\ll RY^2.  
]

完整二部圖總邊數約

[  
|P||Q|\asymp \frac{XY}{\log X\log Y}.  
]

這個 bad edge 界是否小，要看 (R)。若 (R) 是主弧尺度，可能可以；若 (Y) 大，未必。

所以 very-low agreement 只處理極低能，不夠全部。

---

## 8. 需要多尺度正規化

用固定 (M<\min(p,q)/2) 太嚴格。主弧 (h) 可大於底層 (p)，所以不能要求所有 glues 小於 (p/2)。

但在高層邊上，(\min(p,q)) 很大，主弧 (h) 會小於它。於是可以利用高層作 agreement backbone。

這提示完整 dyadic graph 比單一二部 block更有利：低層可能不能直接傳播，但它連到更高層，高層能識別共同 (h)，再反向約束低層。

所以單一二部 block可能不是最自然的原子；「低層—高層—低層」三層結構才是 agreement 原子。

---

## 9. 三層 agreement gadget

取

[  
P_k \longleftrightarrow P_{3k/2} \longleftrightarrow P_k'  
]

兩個低層副本或同一低層的兩批頂點，中間高層素數 (q) 大於主弧 (H).

如果兩條低—高邊共享高頂點 (q)，且 glues 絕對值 (<q/2)，則它們相等。這把不同低頂點的 local h 同步。

因此高層頂點充當「比較器」：它可以比較多個低層 residues 是否來自同一 (h)。

這是比單一二部模型更好的 agreement gadget。

我們的 dyadic graph正好有上層 (3k/2)，所以結構自然存在。

---

## 10. 新的 atomic agreement gadget

對每個底層 block (P_k)，選上層 block

[  
Q_k=P_{\lceil3k/2\rceil}.  
]

讓多個底層頂點連到共同的上層測試頂點。若 assignment 在這些邊上低能，則上層頂點迫使其鄰居 glues 相等，從而同步底層。

這需要測試圖不是每個底層頂點選完全獨立上層鄰居，而要有 overlap：同一上層 (q) 被多個底層 (p) 使用。這是 agreement testing 的設計：測試不是獨立私有，而是有交疊，才能傳播一致性。

這是重要修正。

---

## 11. 測試圖設計更新

原先每個 (p) 選 (C\log p) 個 (q)；但若每個 (q) 只連一個 (p)，無法 agreement propagation。應讓上層 q 有 degree 多個低層 p。

設計成 bipartite expander：

- 左側 (P_k)，右側 (P_{3k/2});
    
- 左度 (d_L\asymp Ck);
    
- 右度也至少 (d_R\asymp Ck) 或更大;
    
- graph 是 sampler/expander.
    

這樣右側頂點比較多個左側 glues，左側又連多個右側，形成 consistency network。

這更像真正的 agreement test graph。

---

## 12. 目前結論

單一 complete bipartite block 給了一個有用的非常低能 agreement，但中能情況需要三層／共享高頂點來傳播 common (h)。這不是壞事，因為 dyadic graph本來就是多尺度。

因此 atomic object 應改為：

[  
\boxed{  
P_k \leftrightarrow P_{3k/2}  
]

的 bipartite expander gadget, with shared high vertices.  
[  
}  
]

它同時提供：

1. local list-decoding for each low vertex;
    
2. glue comparison at high vertices;
    
3. propagation of global (h).
    

下一步應分析這個三層/二部 expander gadget 的 low-energy states。
