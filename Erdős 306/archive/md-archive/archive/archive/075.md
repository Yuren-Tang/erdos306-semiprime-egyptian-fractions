好。現在證「相鄰尺度 label 合併」。

我們已有這樣的局部結論：對每個 upward block pair

[  
P_k \times P_{3k/2},  
]

若其能量低，則存在一個整數 label

[  
h_k,\qquad |h_k|\le B_k,  
]

使得大多數

[  
p\in P_k  
]

與大多數

[  
q\in P_{3k/2}  
]

都滿足

[  
a_p\equiv h_k\pmod p,  
\qquad  
a_q\equiv h_k\pmod q.  
]

其中我們取

[  
B_k=X_k^{1+\eta},  
\qquad X_k=2^k,  
\qquad \frac14<\eta<\frac12.  
]

現在要證：若相鄰兩個 upward block pairs 都 good，則它們的 labels 必須一致。

## 1. 相鄰兩個 labels

令

[  
Y=X_k^{3/2}.  
]

第一個 good block pair 是

[  
P_k\leftrightarrow P_{3k/2},  
]

給 label

[  
h_k,\qquad |h_k|\le X_k^{1+\eta}.  
]

第二個 good block pair 是

[  
P_{3k/2}\leftrightarrow P_{9k/4},  
]

給 label

[  
h_{3k/2},\qquad |h_{3k/2}|\le Y^{1+\eta}=X_k^{\frac32(1+\eta)}.  
]

兩個 labels 都解釋中間 block

[  
P_{3k/2}.  
]

因此，對大多數

[  
q\in P_{3k/2},  
]

同時有

[  
a_q\equiv h_k\pmod q,  
]

和

[  
a_q\equiv h_{3k/2}\pmod q.  
]

所以

[  
h_k\equiv h_{3k/2}\pmod q  
]

對許多 (q\in P_{3k/2}) 成立。

## 2. 多個中間素數強迫 equality

令

[  
D=h_{3k/2}-h_k.  
]

若 (D\ne0)，且它被 (s) 個不同的

[  
q\in P_{3k/2}  
]

整除，則

[  
\prod_{i=1}^{s}q_i \mid D.  
]

但

[  
|D|  
\le  
|h_{3k/2}|+|h_k|  
\ll  
X_k^{\frac32(1+\eta)}.  
]

而每個

[  
q_i\asymp Y=X_k^{3/2}.  
]

所以只要

[  
Y^s>X_k^{\frac32(1+\eta)},  
]

就必有

[  
D=0.  
]

因為

# [  
Y^s

X_k^{\frac32s},  
]

條件是

[  
\frac32s>\frac32(1+\eta),  
]

即

[  
s>1+\eta.  
]

由於

[  
\eta<1,  
]

取

[  
s=2  
]

已足夠。

所以只要兩個 labels 在中間 block 上共同解釋至少兩個素數，就必有

[  
h_k=h_{3k/2}.  
]

這非常強。

---

## 3. 大多數共同解釋自然給出至少兩個素數

每個 good block pair 只可能有少數例外頂點。假設第一個 pair 在中間 block (P_{3k/2}) 上解釋比例至少

[  
1-\epsilon,  
]

第二個 pair 也至少

[  
1-\epsilon.  
]

則共同被兩者解釋的中間素數比例至少

[  
1-2\epsilon.  
]

只要

[  
|P_{3k/2}|  
]

很大，這當然至少有兩個素數。

因此 labels 必須合併：

[  
h_k=h_{3k/2}.  
]

---

## 4. 沿尺度圖傳播

因此，只要一串 upward block pairs 都 agreement-good：

[  
P_k\leftrightarrow P_{3k/2},  
]

[  
P_{3k/2}\leftrightarrow P_{(3/2)^2k},  
]

[  
P_{(3/2)^2k}\leftrightarrow P_{(3/2)^3k},  
]

等等，則所有 labels 相等：

[  
h_k=h_{3k/2}=h_{(3/2)^2k}=\cdots=h.  
]

所以 agreement-good block pairs 在尺度圖的連通分量上產生一個共同全局 (h)。

## 5. 如果某些尺度 pair 不 good？

那該 block pair 付出能量至少

# [  
T_k  
\asymp  
\frac{B_k^2}{X_k\cdot X_k^{3/2}\cdot k^2}

\frac{X_k^{-1/2+2\eta}}{k^2}.  
]

因為

[  
\eta>\frac14,  
]

所以

[  
-1/2+2\eta>0,  
]

故

[  
T_k\to\infty  
]

隨 (k) 增長。

這意味著：在大尺度上，打斷 agreement 傳播非常昂貴。Fourier 權重

[  
e^{-cQ(a)}  
]

會強烈懲罰這類 assignment。

## 6. 這給出一個很乾淨的圖像

對每個 upward comparison pair

[  
k\to \frac32 k,  
]

有二選一：

1. **good**：產生 label (h_k)，並與鄰接 good pair 的 label 合併；
    
2. **bad**：支付能量
    
    [  
    \gg T_k.  
    ]
    

所以整個尺度圖上，非對角自由度只能存在於被 bad pairs 切開的 components 中。每切開一處，要付能量 (T_k)。

這正是我們想要的：

[  
\text{多自由度} \Longrightarrow \text{多 cuts} \Longrightarrow \text{多能量}.  
]

也就是

[  
\log N(R)\le \log N_{\mathrm{diag}}(R)+C R  
]

的組合骨架。

## 7. 尺度圖上的 entropy accounting

假設在尺度 range

[  
k_0\le k\le K  
]

中，我們使用 upward edges

[  
k\to \frac32 k.  
]

若所有這些 comparison pairs 都 good，則所有 blocks 共享一個 label (h)，因此 character 接近 diagonal。

若有 (m) 個 bad comparison pairs，尺度圖被切成最多 (m+1) 個 components，每個 component 可有自己的 label (h_j)。

這增加的 entropy 大約是

[  
\sum_{j}\log B_{component}  
]

或者粗略地

[  
O(m\log B_{\max}).  
]

但每個 bad pair 在尺度 (k) 付能量至少 (T_k), 而 (T_k) 可以比 (\log B_{\max}) 大，若尺度足夠高。對低尺度少數例外，可手動吸收進常數或把 (k_0) 取大。

所以能量可支付新增 labels 的 entropy。

這就是 energy-entropy closure。

## 8. 低尺度問題

若 (k) 接近 (k_0)，則

# [  
T_k

\frac{X_k^{-1/2+2\eta}}{k^2}  
]

雖然趨於無窮，但可能不夠大以支付所有 entropy。解法：

選

[  
k_0  
]

非常大。這是允許的，因為我們可以避開有限多個小素數，從任意大尺度開始建構。

只要

[  
T_{k_0}\gg \log B_{\max}  
]

或至少大於所需常數，就能閉合。

而 (T_{k_0}) 是 (X_{k_0}^{2\eta-1/2}/k_0^2)，可隨 (k_0\to\infty) 任意大。

因此低尺度問題可通過把起始尺度取大解決。

## 9. 這對原問題允許嗎？

允許。原題只要求存在某些不同 semiprime 分母，沒有上界。給定 (b)，我們可避開所有小素數與已有素數，從極大 (k_0) 開始取 blocks。素數倒數和發散很慢，但仍可通過取足夠長尺度 range 達到所需質量。

所以把 (k_0) 取很大沒有原理障礙。

## 11. 下一個必查點：這些 upward comparison pairs 是否足夠參與候選質量？

我們用 (k\to1.5k) 的 pairs 作 agreement backbone。這些 pairs 的 mass 為

[  
u_k u_{1.5k}  
\asymp  
\frac1{k\cdot1.5k}  
\asymp  
\frac1{k^2}.  
]

沿 (k) 求和：

[  
\sum_k\frac1{k^2}  
]

收斂。也就是 agreement backbone 本身不提供發散質量。

但沒關係：候選圖仍包括所有 compatible pairs

[  
\ell\in[2k/3,3k/2],  
]

它們提供發散質量

[  
\sum_k\frac1k.  
]

Agreement backbone 只用來控制 characters，不必承擔全部質量。Fourier coefficient product 包含 backbone edges，所以它可提供次弧衰減；主質量由完整 compatible graph提供。

這分工合理。

## 12. 但要檢查 Bernoulli 概率

若所有候選 edges 都以同一 (\theta\sim1/2) 選取，backbone edges 也在其中，因此 Fourier 能量 includes them.

Backbone mass finite且可能小，但 Fourier 次弧衰減按邊數/phase 能量，不按質量。邊很多，且 bad cutoff chosen gives huge energy threshold。Good.

所以 backbone 可控制 Fourier characters，即使其 reciprocal mass 不發散。
