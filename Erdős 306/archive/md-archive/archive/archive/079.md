好。現在回到 **full graph 主弧 local CLT**。前面 agreement/Poincaré 部分的任務，是把一般 character 壓回「對角族」

[  
a_p\equiv h\pmod p.  
]

一旦如此，Fourier 係數變成一維問題：

# [  
\Phi(h)

\prod_{pq\in E_{\mathrm{full}}}  
\left(  
1-\theta+\theta e\left(\frac{h}{pq}\right)  
\right).  
]

現在要確認這個一維 weighted Bernoulli sum 能給出目標命中。

---

## 1. Full graph 的候選邊

取 dyadic levels

[  
k_0\le k\le K.  
]

每層素數塊

[  
P_k={p:2^k\le p<2^{k+1}}.  
]

Full graph 邊為所有 compatible block pairs：

# [  
E_{\mathrm{full}}

\bigcup_{\frac23\ell\le k\le \frac32\ell}  
P_k\times P_\ell,  
]

同層時排除 (p=q)。

它的 reciprocal mass 是

# [  
W

\sum_{pq\in E_{\mathrm{full}}}\frac1{pq}.  
]

由

[  
\sum_{p\in P_k}\frac1p\asymp \frac1k  
]

得

[  
W  
\asymp  
\sum_{k=k_0}^{K}  
\sum_{\ell\asymp k}  
\frac1{k\ell}  
\asymp  
\sum_{k=k_0}^{K}\frac1k  
\asymp  
\log\frac K{k_0}.  
]

所以給定 (b)，可選 (K) 使

[  
W\asymp \frac{2}{b}.  
]

然後取

[  
\theta=\frac{1/b}{W}.  
]

例如若

[  
W=2/b,  
]

則

[  
\theta=\frac12.  
]

這很好；(\theta) 遠離 (0,1)，Fourier 衰減穩定。

---

## 2. 隨機和與目標

令

[  
X_{pq}\sim \operatorname{Bernoulli}(\theta)  
]

獨立，考慮

[  
S=\sum_{pq\in E_{\mathrm{full}}}X_{pq}\frac1{pq}.  
]

則

# [  
\mathbb E S

# \theta W

\frac1b.  
]

我們要證

[  
\mathbb P\left(S=\frac1b\right)>0.  
]

整數化：令

[  
L=\prod_{p\in P}p.  
]

則

[  
S=\frac1b  
]

等價於

# [  
\sum_{pq\in E_{\mathrm{full}}}  
X_{pq}\frac{L}{pq}

\frac Lb.  
]

因為 (b) 的素因子也放入 (P) 或單獨放入底層固定頂點集合，所以

[  
b\mid L.  
]

---

## 3. 方差

實數和的方差：

# [  
\sigma^2

\theta(1-\theta)  
\sum_{pq\in E_{\mathrm{full}}}  
\frac1{p^2q^2}.  
]

對一層：

# [  
\sum_{p\in P_k}\frac1{p^2}  
\asymp  
\frac{#P_k}{2^{2k}}  
\asymp  
\frac{2^k/k}{2^{2k}}

\frac1{k2^k}.  
]

所以

[  
\sum_{k,\ell\asymp k}  
\frac1{k2^k}\frac1{\ell2^\ell}  
]

由最低尺度 (k_0) 主導。粗略地

[  
\sigma^2\asymp \frac{1}{k_0^2,2^{2k_0}},  
]

故

[  
\sigma\asymp \frac1{k_0,2^{k_0}}.  
]

這與之前估計一致。

---

## 4. 最大權重

最大單項來自最低尺度：

# [  
a_{\max}

\max_{pq}\frac1{pq}  
\asymp  
2^{-2k_0}.  
]

因此

# [  
\frac{a_{\max}}{\sigma}  
\asymp  
\frac{2^{-2k_0}}{1/(k_0 2^{k_0})}

\frac{k_0}{2^{k_0}}  
\to0.  
]

這是 local CLT 的關鍵 Lindeberg 條件：

[  
a_{\max}=o(\sigma).  
]

非常好。

---

## 5. Span

整數權重是

[  
w_{pq}=\frac{L}{pq}.  
]

若每個頂點素數 (r\in P) 至少出現在一條候選邊中，則存在邊 (rs)，使

[  
w_{rs}=\frac{L}{rs}  
]

不被 (r) 整除。因此沒有任何素數 (r\mid L) 整除所有權重。

所以

[  
\gcd_{pq\in E_{\mathrm{full}}}w_{pq}=1.  
]

格點 span 是 (1)。這消除 lattice obstruction。

---

## 6. 主弧 Gaussian

對角 Fourier 係數是

# [  
\Phi(h)

\prod_{pq}  
\left(1-\theta+\theta e(h/pq)\right).  
]

對小 (h)，展開：

# [  
\log\Phi(h)

## 2\pi i h\mu

2\pi^2h^2\sigma^2  
+  
O(h^3\sum a_{pq}^3),  
]

其中

[  
a_{pq}=\frac1{pq}.  
]

因為

[  
a_{\max}=o(\sigma),  
]

在

[  
|h|\lesssim c/a_{\max}  
]

甚至相當寬的範圍內，三次項可控；在主要 Gaussian 範圍

[  
|h|\lesssim 1/\sigma  
]

當然可控。

所以主弧貢獻近似

[  
\sum_h  
e^{-2\pi^2h^2\sigma^2}  
]

，尺度約為

[  
1/\sigma.  
]

Fourier 反演除以 (L) 後，目標點概率應約

[  
\frac{1}{\sigma L}.  
]

只要錯誤項比這小，就有正概率。

## 7. 對角中高頻

原先最難的是對角中高頻 (h)。現在 agreement/Poincaré 其實不處理對角 (h)，因為對角 (h) 本來就是合法的近主弧族。它們由一維 characteristic function 控制。

對於

[  
|h|\gg 1/\sigma,  
]

Gaussian 項

[  
e^{-c h^2\sigma^2}  
]

已經很小。若 (h) 還在小相位區

[  
|h|a_{\max}\ll1,  
]

則二次展開直接給衰減。

若

[  
|h|a_{\max}\gtrsim1,  
]

進入真正高頻。這需要 local CLT 的 aperiodicity condition。

但是注意：在整數化 lattice 上，對角高頻 (h) 對應的是 ordinary one-dimensional weighted Bernoulli sum。這部分可以交給 weighted Bernoulli local CLT 黑箱，只要其假設包含 aperiodicity。

我們已經有：

- span (1);
    
- max weight (o(\sigma));
    
- weights 由大量不同 (L/(pq)) 組成；
    
- full graph 多尺度。
    

仍需核對黑箱所需的高頻 characteristic bound。

---

## 8. 高頻 aperiodicity 是否可由 agreement 控制？

有趣的是，對角高頻 (h) 也可以放回 general character agreement 框架：對角 (a_p=h\bmod p)。如果 (h) 很大，會不會被 agreement 判為多 component label？其實它仍是單一 label (h)，只是 (h) 超過某些 block cutoffs (B_k)，因此 backbone block 會把它視為 bad 並收取能量。

這個能量正是

# [  
Q_{\mathrm{bb}}(h)

h^2\sum_{pq\in E_{\mathrm{bb}}}\frac1{p^2q^2}  
]

在小相位範圍內，或更一般地給 characteristic decay。

所以 agreement energy framework 也能懲罰大 (h)。主弧只保留能量小的 (h)，即

[  
|h|\lesssim1/\sigma.  
]

因此對角高頻有兩種處理：

1. 用 weighted Bernoulli local CLT 黑箱；
    
2. 或用 backbone/full graph energy 直接估計。
    

第一種更標準；第二種可作補充。

---

## 9. Local CLT 黑箱需要什麼？

我們可將需求表述為：

**Weighted lattice Bernoulli LCLT.**  
設整數權重

[  
w_i=L/(p_iq_i)  
]

，Bernoulli 參數 (\theta\in[c,1-c])。若

1. (\gcd(w_i)=1);
    
2. (\max |w_i|=o(\sigma_L))，其中
    

[  
\sigma_L^2=\operatorname{Var}\sum X_iw_i;  
]

3. characteristic function 在非小頻區有足夠衰減；
    

則

[  
\mathbb P\left(\sum X_iw_i=m\right)  
\sim  
\frac1{\sqrt{2\pi}\sigma_L}  
\exp\left(-\frac{(m-\mu_L)^2}{2\sigma_L^2}\right).  
]

其中

[  
\sigma_L=L\sigma.  
]

前兩條已證。第三條，即 aperiodicity，現在可由 agreement/Poincaré + diagonal energy bound 輔助證明。

所以不必完全依賴黑箱；我們的次弧控制其實就是 characteristic function decay。

Fourier 反演：

\sum_{a\in\widehat{G}}  
\widehat\mu(a)  
e(-a\cdot L/b).

把 characters 分為：

[

貢獻

[  
\asymp \frac1{\sigma L}.  
]

2. **Other diagonal frequencies:**
    

[  
a_p\equiv h\pmod p,\qquad |h|\gg1/\sigma.

由 one-dimensional Bernoulli characteristic decay 控制。

3. **Non-diagonal characters:**
    

由 backbone agreement + Irving correction 控制，總貢獻

[

]

若三者成立，則概率正，故存在表示

[  
\frac1b=\sum_{pq\in E_0}\frac1{pq}.  
]

## 11. 從 (1/b) 到 (a/b)

原題是

[  
\frac ab.  
]

若能對任意 finite forbidden prime set (F) 表示

[  
\frac1b  
]

為互異 semiprime reciprocals，且避開 (F)，則重複 (a) 次即可。

第 (j) 次把前面用過的素數加入 forbidden set，重新在更高尺度構造一個 disjoint representation：

# [  
\frac1b

\sum_{pq\in E_j}\frac1{pq}.  
]

合併得

# [  
\frac ab

\sum_{j=1}^{a}  
\sum_{pq\in E_j}  
\frac1{pq}.  
]

所有 semiprimes 互異，且每個分母仍是兩個不同素數之積。

所以核心只需證 (1/b)，且可避開任意 finite forbidden set。

## 12. Squarefree (b) 的角色

若 (b) squarefree，則把 (b) 的 prime factors 放入 (L) 後，

[  
L/b  
]

是整數，而且 CRT 坐標清楚。若 (b) 不 squarefree，(L) 是 squarefree product of primes，不能整除 (b) 的平方部分。因此題目要求 (b) squarefree 正是為了 lattice target 合法。

這與整個 Fourier/CRT 模型完全吻合。

## 13. 目前剩餘真正缺口

現在剩下的主要技術缺口是兩個：

### 缺口 A：Irving-good block construction

需要嚴格從 Irving 平均定理中抽出足夠多 good primes/blocks，使 vertex correction lemma 對所有 relevant (q) 和 (d\ne0) 成立。

這是解析數論工作，但看起來與 Irving 定理相當吻合。

### 缺口 B：完整 Fourier 次弧求和

Backbone agreement + vertex correction 給出結構性衰減，但需寫成嚴格求和：

[  
\sum_{\text{non-major }a}|\widehat\mu(a)|  
=o(1/\sigma).  
]

這需要精細 book-keeping，但概念上已清楚。

Local CLT 主弧則相對標準。
