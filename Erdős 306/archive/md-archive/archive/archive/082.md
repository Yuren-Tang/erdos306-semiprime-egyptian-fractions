# 0. 目標定理

要證：

給定

[  
\frac ab\in\mathbb Q_{>0},  
\qquad b\ \text{squarefree},  
]

存在互異整數

[  
1<n_1<\cdots<n_k,  
]

每個

[  
n_i=p_iq_i  
]

為兩個不同素數之積，使得

[  
\frac ab=\sum_{i=1}^k\frac1{n_i}.  
]

如前所說，只需證：

> 對任意 squarefree (b) 和任意有限 forbidden prime set (F)，存在一組互異 semiprime denominators (pq)，所有 (p,q\notin F)，使  
> [  
> \frac1b=\sum\frac1{pq}.  
> ]

然後重複 (a) 次即可得到 (a/b)。

# 1. 隨機模型構造

取極大起始尺度

[  
k_0  
]

避開所有 forbidden primes 與 (b) 的有限干擾。取 dyadic prime blocks

[  
P_k={p:2^k\le p<2^{k+1}}  
]

for

[  
k_0\le k\le K.  
]

令 full compatible graph 為

# [  
E_{\mathrm{full}}

\bigcup_{\frac23\ell\le k\le \frac32\ell}  
P_k\times P_\ell,  
]

排除 (p=q)。

其 reciprocal mass

[  
W=\sum_{pq\in E_{\mathrm{full}}}\frac1{pq}  
]

滿足

[  
W\asymp \log\frac K{k_0}.  
]

選 (K) 使

[  
W>\frac1b  
]

且例如

[  
W\asymp \frac2b.  
]

令

[  
\theta=\frac{1/b}{W}\in(c,1-c).  
]

對每條邊 (pq\in E_{\mathrm{full}})，獨立選

[  
X_{pq}\sim\operatorname{Bernoulli}(\theta).  
]

考慮

[  
S=\sum_{pq\in E_{\mathrm{full}}}X_{pq}\frac1{pq}.  
]

則

[  
\mathbb E S=\frac1b.  
]

令

[  
L=b\prod_{p\in P}p  
]

或把 (b) 的素因子也納入固定部分，使 (b\mid L)。整數化後：

[  
S=\frac1b  
]

等價於

# [  
\sum_{pq}X_{pq}\frac{L}{pq}

\frac Lb.  
]

# 2. Fourier 反演

對 group

[  
G=\mathbb Z/L\mathbb Z  
]

或等價的 squarefree CRT group 做 Fourier 反演。characters 可視為 residue assignment

[  
a=(a_p)_{p\in P},  
\qquad a_p\in\mathbb F_p.  
]

對邊 (pq)，相位為

# [  
\phi_{pq}(a)

\frac{a_pq^{-1}}p+  
\frac{a_qp^{-1}}q.  
]

Fourier 係數：

# [  
\widehat\mu(a)

\prod_{pq\in E_{\mathrm{full}}}  
\left(  
1-\theta+\theta e(\phi_{pq}(a))  
\right).  
]

有基本能量估計：

[  
|\widehat\mu(a)|  
\le  
\exp(-c_\theta Q(a)),  
]

其中

[  
Q(a)=  
\sum_{pq\in E_{\mathrm{full}}}  
|\phi_{pq}(a)|^2.  
]

目標是證：

# [  
\mathbb P(S=1/b)

\frac1L  
\sum_a  
\widehat\mu(a)e(-a\cdot L/b)

> 0.  
> ]

# 3. Major arc

Major arc 是對角 characters：

[  
a_p\equiv h\pmod p  
]

且

[  
|h|\lesssim 1/\sigma,  
]

其中

# [  
\sigma^2

\theta(1-\theta)\sum_{pq\in E_{\mathrm{full}}}\frac1{p^2q^2}.  
]

我們已有估計：

[  
\sigma\asymp \frac1{k_0 2^{k_0}},  
]

最大權重

[  
a_{\max}\asymp 2^{-2k_0},  
]

且

[  
a_{\max}=o(\sigma).  
]

主弧 Gaussian 貢獻：

[  
\sum_{|h|\lesssim1/\sigma}\widehat\mu(h)e(-h/b)  
\asymp  
\frac1\sigma.  
]

所以概率主項約

[  
\frac1{\sigma L}.  
]

這一部分看起來標準。

# 4. Minor arc 要證的核心命題

我們需要：

# [  
\sum_{a\notin\mathfrak M}  
|\widehat\mu(a)|

o(1/\sigma).  
]

這將由下列四個 lemma 組成。

---

## Lemma A：Dense block CRT agreement

取兩個 blocks

[  
P\subset[X,2X],  
\qquad  
Q\subset[Y,2Y],  
]

且有 cutoff (B) 滿足

[  
B<Y/2,\qquad X^2>2B.  
]

給 residues

[  
a_p\in\mathbb F_p,\quad b_q\in\mathbb F_q.  
]

令

[  
H_{pq}^{*}  
]

為 CRT glue 的對稱代表。若 strong-good 邊

[  
|H_{pq}^{*}|\le B  
]

在完整二部圖 (P\times Q) 中密度至少

[  
1-\varepsilon,  
]

則存在整數

[  
|h|\le B  
]

使得除 (O(\varepsilon)) 比例頂點外，

[  
a_p\equiv h\pmod p,\qquad b_q\equiv h\pmod q.  
]

這個 lemma 幾乎已證。證明只用：

- 同一高點 (q) 上，bounded glues 同餘 mod (q)，故相等；
    
- 兩個高點共享兩個低鄰居，則 label 差被兩個低素數整除，因 product (>2B)，故相等；
    
- density pruning 把例外比例控制為 (O(\varepsilon))。
    

這是目前最穩的一塊。

---

## Lemma B：Backbone cut cost

選一個固定

[  
1<\gamma<3/2  
]

例如

[  
\gamma=4/3.  
]

取

[  
B_k=X_k^{1+\eta},  
\qquad X_k=2^k,  
]

其中

[  
\frac{\gamma-1}{2}<\eta<\gamma-1.  
]

例如

[  
\eta=1/4.  
]

Backbone blocks：

[  
P_k\times P_{\gamma k}.  
]

若這個 block pair 不是 agreement-good，則其 Fourier energy 至少

[  
T_k  
\asymp  
\frac{X_k^{1-\gamma+2\eta}}{k^2}.  
]

因為

[  
1-\gamma+2\eta>0,  
]

所以

[  
T_k\to\infty.  
]

Thus every bad scale cut costs large energy.

這一塊基本是計算，需把 constants 和 bad-density threshold 寫嚴。

---

## Lemma C：Label merging across scales

若兩個 adjacent backbone pairs 都 agreement-good：

[  
P_k\leftrightarrow P_{\gamma k},  
]

[  
P_{\gamma k}\leftrightarrow P_{\gamma^2k},  
]

得到 labels

[  
h_k,\quad h_{\gamma k},  
]

則它們必相等。

因為它們都解釋 (P_{\gamma k}) 中大多數素數，所以

[  
h_k\equiv h_{\gamma k}\pmod q  
]

對許多

[  
q\in P_{\gamma k}  
]

成立。若二者不等，其差被至少兩個這樣的 (q) 整除。但

[  
|h_{\gamma k}-h_k|  
\le  
B_{\gamma k}+B_k  
\ll  
X_k^{\gamma(1+\eta)},  
]

而兩個中間素數乘積約

[  
X_k^{2\gamma}.  
]

只要

[  
2\gamma>\gamma(1+\eta),  
]

即

[  
\eta<1,  
]

就矛盾。成立。

這一塊也很穩。

---

## Lemma D：Vertex correction via Irving

若一個 block (P\subset[X,2X]) 已由 label (h) 控制：

[  
a_p\equiv h\pmod p  
]

對大多數 (p\in P)，而相鄰 vertex (q\sim Q) 選錯 residue：

[  
a_q\not\equiv h\pmod q,  
]

則 incident energy

[  
\sum_{p\in P}|\phi_{pq}(a)|^2  
\gg |P|.  
]

證明：

令

[  
d=a_q-h\not\equiv0\pmod q.  
]

則

[  
\phi_{pq}(a)  
\equiv  
\frac{h}{pq}  
+  
\frac{d p^{-1}}q.  
]

Irving 應給：

[  
#\left{  
p\in P:  
\left|  
\frac{d p^{-1}}q  
\right|>\delta  
\right}  
\gg |P|  
]

uniformly in (d\ne0), for all good (q) after discarding a small bad set. Then incident energy follows.

這是第一個真正依賴外部深工具的 lemma。需要從 Irving exponential sum bound + Erdős–Turán 推出。此處仍需嚴格查證。

---

# 5. Minor arc summation from A–D

Assuming A–D:

- Bad backbone cuts cost (\sum T_k).
    
- Good components get a common label (h_C).
    
- Wrong vertices in a component cost incident energy (\gg |P|), paying their residue entropy.
    
- Multiple component labels are paid by cuts.
    
- If no cuts and no wrong vertices, the character is diagonal (h).
    

Thus any non-major character either:

1. is diagonal high-frequency, handled by one-dimensional local CLT / Gaussian tail;
    
2. has bad cuts, paid by (T_k);
    
3. has wrong vertices, paid by Irving incident energy.
    

Because (k_0) can be chosen arbitrarily large, all energy payments dominate the combinatorial entropy.

This yields

[  
\sum_{a\notin\mathfrak M}|\widehat\mu(a)|  
=o(1/\sigma).  
]

# 6. What to do next?

The most urgent lemma to actually close is **Lemma D**. A–C are elementary and nearly done. The remaining global summation is bookkeeping, but D decides whether the whole route can be made unconditional.

So next I would focus on D:

1. State Irving’s available bound precisely.
    
2. Convert exponential sum bound into interval nonconcentration.
    
3. Show bad moduli can be discarded without damaging graph mass/agreement.
    
4. Verify the scale (\gamma=4/3) lies safely inside Irving’s range.
    

That is the next mathematically decisive step.
