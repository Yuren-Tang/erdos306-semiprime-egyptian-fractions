好。現在把 A–D 合成一個正式的 **Minor Arc Proposition**。

# Minor Arc Proposition 目標版

設 (E_{\mathrm{full}}) 是完整 compatible dyadic prime graph，並已刪去 Irving-bad primes，保留 good blocks。令

# [  
\widehat\mu(a)

\prod_{pq\in E_{\mathrm{full}}}  
\left(  
1-\theta+\theta e(\phi_{pq}(a))  
\right).  
]

令 major arc (\mathfrak M) 為小對角 characters：

[  
a_p\equiv h\pmod p,\qquad |h|\le C/\sigma.  
]

則對充分大的起始尺度 (k_0)，有

[  
\sum_{a\notin\mathfrak M}|\widehat\mu(a)|  
=o(1/\sigma).  
]

這就是 Fourier 反演中需要的 minor arc bound。

---

# 1. 先把 character 結構化

對一個任意 character

[  
a=(a_p)_{p\in P},  
]

看 backbone scale chain：

[  
k,\quad \gamma k,\quad \gamma^2k,\quad\ldots  
]

其中取

[  
\gamma=\frac43,\qquad \eta=\frac14.  
]

對每個 backbone block pair

[  
P_k\times P_{\gamma k},  
]

取 cutoff

[  
B_k=X_k^{1+\eta}=X_k^{5/4}.  
]

稱邊 strong-good 若

[  
|H_{pq}^{*}|\le B_k.  
]

若該 block pair 的 bad edge density 很小，Lemma A 給出 label

[  
h_k,\qquad |h_k|\le B_k,  
]

解釋大多數 (P_k) 與 (P_{\gamma k}) 的頂點。

若該 block pair bad density 不小，Lemma B 說它付出能量至少

[  
T_k\asymp \frac{X_k^{1/6}}{k^2}.  
]

因為

[  
T_k\to\infty,  
]

且 (k_0) 可任意大，bad cuts 很昂貴。

---

# 2. Good block labels 合併

若相鄰兩個 backbone block pairs 都 good：

[  
P_k\times P_{\gamma k},  
\qquad  
P_{\gamma k}\times P_{\gamma^2 k},  
]

它們給 labels

[  
h_k,\quad h_{\gamma k}.  
]

二者都解釋中間 block (P_{\gamma k}) 的大多數頂點，所以對至少兩個中間素數 (q),

[  
h_k\equiv h_{\gamma k}\pmod q.  
]

但

[  
|h_{\gamma k}-h_k|  
\le B_{\gamma k}+B_k  
\ll X_k^{\gamma(1+\eta)}.  
]

而兩個 (q\sim X_k^\gamma) 的乘積約

[  
X_k^{2\gamma}.  
]

因為

[  
2\gamma>\gamma(1+\eta),  
]

即 (\eta<1)，所以必有

[  
h_{\gamma k}=h_k.  
]

因此：在刪去 bad cuts 後，每個 backbone 連通分量有一個共同 label

[  
h_C.  
]

---

# 3. Wrong vertices 被 Irving correction 殺掉

現在看一個 labelled component (C)，label 為

[  
h_C.  
]

若某 good vertex (q\in C) 沒有滿足

[  
a_q\equiv h_C\pmod q,  
]

則取一個相鄰 labelled block (P_x) 與 (q) 相連。對大多數 (p\in P_x),

[  
a_p\equiv h_C\pmod p.  
]

令

[  
d=a_q-h_C\not\equiv0\pmod q.  
]

則

[  
\phi_{pq}(a)  
\equiv  
\frac{h_C}{pq}  
+  
\frac{d\bar p}{q}  
\pmod1.  
]

由 Irving Theorem 1 推出的 Lemma D，

[  
\sum_{p\in P_x}  
\left|  
\frac{d\bar p}{q}  
\right|^2  
\gg |P_x|.  
]

又因

[  
\frac{h_C}{pq}  
]

在此尺度下很小，所以

[  
\sum_{p\in P_x}|\phi_{pq}(a)|^2  
\gg |P_x|.  
]

因此一個 wrong vertex 付出能量至少

[  
\gg |P_x|\asymp \frac{x}{\log x}.  
]

這遠大於它的 residue entropy

[  
\log q.  
]

所以 wrong vertices 的總自由度完全由 energy 支付。

---

# 4. 因此所有非對角自由度都要付費

一個 character 若不是全局 diagonal，只有三種原因：

### 第一，scale graph 有 bad cuts

每個 bad cut (k) 付出

[  
T_k\asymp \frac{X_k^{1/6}}{k^2}.  
]

這可支付新增 component label 的 entropy，因為 label entropy 至多

[  
O(k),  
]

而對充分大 (k_0),

[  
T_k\gg k.  
]

### 第二，component 內有 wrong vertices

每個 wrong vertex (q) 付出

[  
\gg |P_x|  
]

incident energy，遠大於它的 (\log q) entropy。

### 第三，它其實是 diagonal，但 (h) 不在 major arc

這是 one-dimensional high-frequency case，由 diagonal characteristic function 控制：

[  
\Phi(h)=  
\prod_{pq\in E_{\mathrm{full}}}  
(1-\theta+\theta e(h/pq)).  
]

若

[  
|h|>C/\sigma,  
]

則 Gaussian decay 或 weighted Bernoulli local CLT 的 aperiodicity 給出

[  
\sum_{|h|>C/\sigma}|\Phi(h)|=o(1/\sigma).  
]

---

# 5. 熵—能量求和

把 character 按以下資料分類：

1. bad cuts 集合 (\mathcal B);
    
2. 每個 component 的 label (h_C);
    
3. wrong vertices 集合 (W);
    
4. wrong vertices 的 residues.
    

對固定分類，Fourier 係數總量受到

[  
\exp\left(  
-c\sum_{k\in\mathcal B}T_k  
-c\sum_{q\in W}E_q  
\right)  
]

控制。

而相應的 entropy 至多為：

[  
\exp\left(  
O\left(\sum_{k\in\mathcal B} k\right)  
+  
O\left(\sum_{q\in W}\log q\right)  
\right).  
]

因為

[  
T_k\gg k,  
]

且

[  
E_q\gg \log q,  
]

所以取 (k_0) 充分大後，能量壓過 entropy。

因此所有非-piecewise-diagonal 的 character 總貢獻收斂且可令其小於

[  
o(1/\sigma).  
]

剩下 piecewise diagonal，但若有多個 components，必有 bad cuts，其 label entropy 已由 bad cuts 支付。若無 bad cuts，則只有一個 global label (h)，即真正 diagonal。

所以最終 minor arcs 只剩 diagonal high-frequency，而它由一維 local CLT/characteristic decay 處理。

---

# 6. Proposition 結論

因此，在 A–D 與 one-dimensional diagonal tail bound 成立的前提下，

[  
\sum_{a\notin\mathfrak M}|\widehat\mu(a)|  
=o(1/\sigma).  
]

這是我們需要的 Minor Arc Proposition。

---

# 7. 還剩哪裏不夠嚴？

這個 Proposition 的概念已經相當穩，但若要成文，還要補三個技術細節：

### (i) 防止 incident energy 重複計數

多個 wrong vertices 可能用到同一批 incident edges。解法是為每個 scale/side 選 disjoint correction directions，或用 bounded overlap。因為 dyadic graph degree in scale 是 (O(k)) 但可控，這應能處理。

### (ii) 對角高頻 tail

這需要正式選擇：  
要麼引用 weighted lattice Bernoulli local CLT；  
要麼用 multiscale energy 自己證 diagonal tail。  
這一塊還沒有完全寫死。

### (iii) full graph 與 pruned good graph 的一致性

Irving-bad primes 刪去後，full graph mass、span、variance、local CLT 都要重新核對。因刪去比例 (o(1))，應無本質問題，但要寫。
