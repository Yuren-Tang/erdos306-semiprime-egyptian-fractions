好。現在把剛才的想法整理成一個 **Aperiodicity Proposition**，也就是把「高頻」正式納入 agreement 框架。

# Aperiodicity Proposition 目標版

令 (\mathfrak M) 為小對角主弧：

# [  
\mathfrak M

\left{  
a:\exists h\in\mathbb Z,\ |h|\le C/\sigma,\  
a_p\equiv h\pmod p\ \forall p  
\right}.  
]

則要證：

# [  
\sum_{a\notin\mathfrak M}  
|\widehat\mu(a)|

o(1/\sigma).  
]

這裏不再把「對角高頻」單獨當成神秘的 lattice obstruction；它分兩類處理：

1. **普通大整數對角 (h)**：由 Gaussian 衰減；
    
2. **模 (L) 意義下的高頻繞回**：其實不是小整數 label，會被 backbone agreement 檢出並付能量。
    

---

## 1. Character 的兩種對角性

在 squarefree CRT group 中，一個 character 是 residue assignment：

[  
a=(a_p)_{p\in P}.  
]

若存在整數 (h) 使

[  
a_p\equiv h\pmod p  
\quad\forall p,  
]

則它是 diagonal character。

但這個 (h) 是 modulo

[  
L_P=\prod_{p\in P}p  
]

唯一的。也就是說，若

[  
h  
]

很大但 modulo (L_P) 等於小整數 (h_0)，那它不是新 character，而就是 (h_0)。

所以真正的「小主弧」是：

[  
h\in(-L_P/2,L_P/2],  
\qquad |h|\le C/\sigma.  
]

若 diagonal character 的最小代表 (|h|>C/\sigma)，它就是 diagonal tail。

---

## 2. Diagonal tail 第一段：無繞回 Gaussian

若

[  
C/\sigma<|h|\ll a_{\max}^{-1},  
]

其中

[  
a_{\max}=\max_{pq}\frac1{pq}\asymp X_0^{-2},  
]

則對所有邊

[  
\left|\frac{h}{pq}\right|=\frac{|h|}{pq}.  
]

因此

# [  
Q(h)

# \sum_{pq}  
\left(\frac{h}{pq}\right)^2

h^2\sum_{pq}\frac1{p^2q^2}  
\asymp h^2\sigma^2.  
]

所以

[  
|\Phi(h)|  
\le e^{-c h^2\sigma^2}.  
]

取 (C) 大，得

[  
\sum_{C/\sigma<|h|\ll a_{\max}^{-1}}  
|\Phi(h)|  
=o(1/\sigma).  
]

這部分乾淨。

---

## 3. Diagonal tail 第二段：大到產生繞回

若

[  
|h|\gtrsim a_{\max}^{-1}\asymp X_0^2,  
]

則不能再用全局小相位展開。

但此時若它還想在 backbone 上不付能量，就必須在許多 block pair 上看起來像某個小 label。

具體地，在 backbone block

[  
P_k\times P_{\gamma k}  
]

上，cutoff 是

[  
B_k=X_k^{1+\eta}.  
]

如果該 block agreement-good，Lemma A 產生一個小 label

[  
h_k,\qquad |h_k|\le B_k,  
]

並使大多數 vertices 滿足

[  
h\equiv h_k\pmod p.  
]

因為 character 是 diagonal (h)，這表示

[  
h-h_k  
]

被該 block 中大多數 primes 整除。若 block 含有足夠多 primes，則除非 (h\equiv h_k\pmod{M_k}) for a huge product (M_k)，這不可能。

若所有 backbone blocks 都 good 並且 labels merge，得到一個 global small label

[  
h_0  
]

使

[  
h\equiv h_0\pmod p  
\quad\forall p\in P  
]

除少數 paid exceptions 外。若無 exceptions，則

[  
h\equiv h_0\pmod{L_P}.  
]

作為 character，這就是小 label (h_0)，若 (|h_0|\le C/\sigma) 則在 major arc；若 (|h_0|>C/\sigma)，回到 ordinary diagonal Gaussian tail。

所以任何真正的 high-frequency wrap character 若不等於小 label，必須有：

- bad cuts；
    
- wrong vertices；
    
- 或 large label Gaussian energy。
    

這些都已被 Minor Arc Proposition 的能量—熵付款處理。

---

## 4. 這消除了 lattice high-frequency 黑箱

因此我們不必另引一個完整的 weighted Bernoulli LCLT aperiodicity theorem。其實我們自己建立了 aperiodicity：

[  
\boxed{  
\text{若 Fourier character 在高頻仍大，則它必須局部看起來像小 CRT label；}  
}  
]

[  
\boxed{  
\text{dense agreement 迫使這些局部 labels 合併；}  
}  
]

[  
\boxed{  
\text{最後只能是小 diagonal major arc，否則付能量。}  
}  
]

這就是 aperiodicity。

# Aperiodicity Proposition 的證明骨架

給任意 (a\notin\mathfrak M)。

### Step 1. Backbone decomposition

檢查每個 backbone block (k\to\gamma k)。

若 block bad，記入 bad cuts，付能量 (T_k)。

若 block good，得到 label (h_k).

### Step 2. Merge labels

相鄰 good blocks 的 labels 合併，所以每個 good component 有 label (h_C).

### Step 3. Correct vertices

若 component 內某 good vertex 不服從

[  
a_p\equiv h_C\pmod p,  
]

由 Irving correction 付 incident energy (\gg |P_{\mathrm{neighbor}}|).

### Step 4. If no paid defects remain

若沒有 bad cuts、沒有 wrong vertices，則所有 vertices 服從同一 label (h).  
若

[  
|h|\le C/\sigma,  
]

則 (a\in\mathfrak M)，矛盾。  
若

[  
|h|>C/\sigma,  
]

則由 Gaussian diagonal tail付能量。

所以 (a\notin\mathfrak M) 必定付出某種能量。

### Step 5. Sum over all possibilities

分類數量由：

- bad cuts entropy；
    
- labels entropy；
    
- wrong vertices entropy；
    

給出，全部由相應能量支付。

故總和

[  
\sum_{a\notin\mathfrak M}|\widehat\mu(a)|  
]

比主弧尺度

[  
1/\sigma  
]

小。

---

## 5. 現在最需嚴化的一個點

上面 Step 5 仍然是「付款語言」。若寫成正式證明，需要建立一個明確不等式，例如：

[  
#{a: Q(a)\le R,\ a\notin\mathfrak M}  
\le  
\frac1\sigma \exp(\kappa R)  
]

with

[  
\kappa<c.  
]

或者更細：

[  
\sum_{a\notin\mathfrak M}e^{-cQ(a)}  
\le  
o(1/\sigma).  
]

這就是最後的 bookkeeping lemma。

但現在它已經是純組合/計數問題，不再含新解析數論難點。

---

## 6. 接下來應做什麼？

下一步最合理是寫一個 **Energy–Entropy Bookkeeping Lemma**：

假設有：

- bad cut costs (T_k\gg k);
    
- wrong vertex costs (E_q\gg\log q);
    
- labels have entropy (O(k));
    
- cuts and wrong vertices have bounded overlap;
    

則推出

[  
\sum_{\text{minor structures}}  
e^{-c\text{energy}}\cdot e^{\text{entropy}}  
=o(1/\sigma).  
]

這是最後的形式化環節。完成它後，就能回到主定理拼接。
