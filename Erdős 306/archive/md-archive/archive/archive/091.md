好。現在處理 **Major Arc Gaussian Lemma**。這一塊要證的是：小對角 characters 的 Fourier 和給出一個正的、量級正確的主項。

# Major Arc Gaussian Lemma

令

[  
S=\sum_{pq\in E_{\mathrm{full}}}X_{pq}\frac1{pq},  
\qquad  
X_{pq}\sim \operatorname{Bernoulli}(\theta)  
]

獨立，且

[  
\mathbb E S=\frac1b.  
]

令

[  
a_{pq}=\frac1{pq}.  
]

方差為

# [  
\sigma^2

\theta(1-\theta)\sum_{pq\in E_{\mathrm{full}}}a_{pq}^2.  
]

取 major arc

[  
\mathfrak M=  
{h\in\mathbb Z:\ |h|\le H},  
\qquad  
H=C/\sigma  
]

其中 (C) 是一個先固定的大常數。

對角 Fourier 係數為

# [  
\Phi(h)

\prod_{pq\in E_{\mathrm{full}}}  
\left(1-\theta+\theta e(h a_{pq})\right).  
]

Fourier 反演中，目標是

[  
S=\frac1b.  
]

因為均值正好是 (1/b)，線性相位會抵消。因此主弧貢獻應近似

[  
\sum_{|h|\le H}  
\exp(-2\pi^2\sigma^2 h^2).  
]

這個和量級是

[  
\asymp \frac1\sigma.  
]

---

## 1. 方差與最大項

我們已估計：

[  
\sigma^2  
\asymp  
\frac1{k_0^2 2^{2k_0}},  
]

所以

[  
\sigma  
\asymp  
\frac1{k_0 2^{k_0}}.  
]

最大單項來自最低尺度：

# [  
a_{\max}

\max_{pq}a_{pq}  
\asymp  
2^{-2k_0}.  
]

因此

# [  
\frac{a_{\max}}{\sigma}  
\asymp  
\frac{2^{-2k_0}}{1/(k_0 2^{k_0})}

\frac{k_0}{2^{k_0}}  
\to0.  
]

這是主弧展開所需的核心小量。

---

## 2. 單個 Bernoulli factor 的展開

對小 (t)，有

# [  
1-\theta+\theta e(t)

## \exp\left(  
2\pi i\theta t

2\pi^2\theta(1-\theta)t^2  
+  
O_\theta(|t|^3)  
\right).  
]

這裏

[  
t=h a_{pq}.  
]

若

[  
|h|\le C/\sigma,  
]

則

# [  
|h|a_{\max}  
\le  
\frac{C a_{\max}}{\sigma}

o(1).  
]

所以所有 factors 都可一致展開。

因此

# [  
\log\Phi(h)

## 2\pi i\theta h\sum a_{pq}

2\pi^2\theta(1-\theta)h^2\sum a_{pq}^2  
+  
O\left(  
|h|^3\sum a_{pq}^3  
\right).  
]

即

# [  
\log\Phi(h)

## 2\pi i h\mathbb E S

2\pi^2\sigma^2h^2  
+  
O\left(  
|h|^3\sum a_{pq}^3  
\right).  
]

---

## 3. 三次項檢查

需要證

[  
|h|^3\sum a_{pq}^3=o(1)  
]

uniformly for

[  
|h|\le C/\sigma.  
]

由

[  
\sum a_{pq}^3  
\le  
a_{\max}\sum a_{pq}^2.  
]

而

[  
\sum a_{pq}^2  
\asymp \sigma^2  
]

因為 (\theta(1-\theta)\asymp1)。所以

[  
\sum a_{pq}^3  
\ll  
a_{\max}\sigma^2.  
]

若

[  
|h|\le C/\sigma,  
]

則

# [  
|h|^3\sum a_{pq}^3  
\ll  
\frac{C^3}{\sigma^3}\cdot a_{\max}\sigma^2

# C^3\frac{a_{\max}}{\sigma}

o(1).  
]

所以三次誤差確實可忽略。

---

## 4. 目標相位抵消

Fourier 反演中對目標 (1/b) 的相位是

[  
e(-h/b).  
]

主弧項為

[  
\Phi(h)e(-h/b).  
]

因為我們選

[  
\mathbb E S=\frac1b,  
]

而

[  
\Phi(h)  
]

的線性項是

[  
e(h\mathbb E S)=e(h/b),  
]

所以

# [  
\Phi(h)e(-h/b)

\exp(-2\pi^2\sigma^2h^2+o(1)).  
]

這是正實數主項，沒有振盪抵消。

---

## 5. 主弧求和

因此

# [  
\sum_{|h|\le C/\sigma}  
\Phi(h)e(-h/b)

(1+o(1))  
\sum_{|h|\le C/\sigma}  
e^{-2\pi^2\sigma^2h^2}.  
]

令

[  
u=\sigma h.  
]

則這是 Riemann sum：

[  
\sum_{|h|\le C/\sigma}  
e^{-2\pi^2\sigma^2h^2}  
\sim  
\frac1\sigma  
\int_{-C}^{C}  
e^{-2\pi^2u^2},du.  
]

取 (C) 固定大，該積分為正常數。因此主弧貢獻

[  
\asymp \frac1\sigma.  
]

更精確地：

[  
\sum_{|h|\le C/\sigma}  
\Phi(h)e(-h/b)  
\ge  
c_C\frac1\sigma  
]

for large (k_0), with (c_C>0).

---

## 6. 為什麼只求 (|h|\le C/\sigma) 足夠？

主弧只需提供正主項；尾部由 minor arc / diagonal tail 控制。選 (C) 足夠大，可使 Gaussian tail

[  
\sum_{|h|>C/\sigma}  
e^{-c\sigma^2h^2}  
]

很小於 (1/\sigma)。而真正非 Gaussian 的高頻由 aperiodicity proposition 控制。

所以主弧 lemma 完成。

# Major Arc Lemma 結論

在我們的 full compatible graph 與 Bernoulli 參數選擇下，

# [  
\sum_{a\in\mathfrak M}  
\widehat\mu(a)e(-a\cdot L/b)

c_0\frac1\sigma+o(1/\sigma)  
]

其中 (c_0>0)。

配合 minor arc bound

[  
\sum_{a\notin\mathfrak M}  
|\widehat\mu(a)|  
=o(1/\sigma),  
]

得到

[  
\sum_a  
\widehat\mu(a)e(-a\cdot L/b)

> 0.  
> ]

除以 (L) 後，

[  
\mathbb P(S=1/b)>0.  
]

所以存在所需的 semiprime reciprocal subset representing (1/b)。
