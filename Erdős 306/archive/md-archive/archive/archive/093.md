# 題目

可暫名：

**Representing positive rationals by reciprocals of distinct semiprimes**

或更精確：

**A CRT–Fourier approach to an Erdős-type semiprime reciprocal problem**

---

# Theorem

Let

[  
\frac ab\in\mathbb Q_{>0}  
]

with (b) squarefree. Then there exist distinct integers

[  
1<n_1<\cdots<n_k  
]

such that each (n_i) is the product of two distinct primes and

[  
\frac ab=\sum_{i=1}^k\frac1{n_i}.  
]

We prove first the case (a=1), with an arbitrary finite forbidden prime set (F); the general case follows by repetition.

---

# Section 1. Dyadic prime blocks and compatible graph

Define

[  
P_k={p:\ 2^k\le p<2^{k+1},\ p\text{ prime}}.  
]

After Irving-pruning later, write (P_k^\ast\subset P_k).

Define the compatible edge set

# [  
E_{\mathrm{full}}

\bigcup_{\frac23\ell\le k\le\frac32\ell}  
P_k^\ast\times P_\ell^\ast.  
]

Main lemma:

[  
\sum_{pq\in E_{\mathrm{full}}}\frac1{pq}  
\asymp  
\sum_{k=k_0}^{K}\frac1k  
\asymp  
\log\frac K{k_0}.  
]

Therefore choose (K) and (\theta\in[c,1-c]) so that

# [  
\theta\sum_{pq\in E_{\mathrm{full}}}\frac1{pq}

\frac1b  
]

up to the negligible lattice-fixing edges.

**Dependency:** only PNT/Mertens in dyadic intervals.

**Status:** stable.

---

# Section 2. Lattice setup and span

Let

[  
L=\prod_{p\in P^\ast}p  
]

initially. Because (b\mid L) is needed, include prime factors of (b). But if a prime (r\mid b) appears only in (L) and not in any denominator (pq), the span breaks.

So add finitely many lattice-fixing edges

[  
r s_r,\qquad r\mid b,  
]

with (s_r) a very large new prime, chosen so that

[  
\frac1{r s_r}=o(\sigma)  
]

and their total variance is negligible.

Then define (L) as the product of all primes appearing in the variable graph plus all (r\mid b) and all (s_r).

Prove:

[  
\gcd\left{\frac L{pq}: pq\in E\right}=1.  
]

Also

[  
\frac Lb\in\mathbb Z.  
]

Hence

[  
\sum X_{pq}\frac1{pq}=\frac1b  
]

is equivalent to

[  
\sum X_{pq}\frac L{pq}=\frac Lb.  
]

**Dependency:** elementary.

**Status:** stable, but needs careful wording because lattice-fixing edges may have small primes (r\mid b), so they should not be used in asymptotic Fourier control except as negligible factors.

# Section 3. Irving pruning

Use Irving’s Theorem 1.

Irving defines

[  
S_q(a;x)=  
\sum_{\substack{p\sim x\(p,q)=1}}  
e\left(\frac{a\bar p}{q}\right)  
]

and proves

[  
\sum_{q\sim Q}\max_{(a,q)=1}|S_q(a;x)|  
\ll_\epsilon  
\left(  
Q^{5/4}x^{5/8}  
+  
Qx^{9/10}  
+  
Q^{7/6}x^{13/18}  
\right)Q^\epsilon  
]

for

[  
Q^{2/3}\le x\le Q^{3/2}.  
]

Choose

[  
Q=x^{4/3}.  
]

Then the bound is

[  
o(Q|P_x|)  
]

after accounting for logs.

Therefore for all but (o(Q/\log Q)) prime (q\sim Q),

[  
\max_{a\ne0}  
\left|  
\sum_{p\sim x}e_q(a\bar p)  
\right|  
=o(|P_x|).  
]

Using Erdős–Turán, obtain for good (q):

[  
\sum_{p\sim x}  
\left|  
\frac{d\bar p}{q}  
\right|^2  
\gg |P_x|  
\quad(d\ne0).  
]

Prune bad (q)'s in every relevant block. Since only (O(1)) neighbouring scales are required per block, the union of bad sets is still (o(|P_k|)).

**Dependency:** Irving theorem + Erdős–Turán.

**Status:** good. This is the principal external input.

# Section 4. Random model and Fourier transform

Let

[  
X_{pq}\sim\operatorname{Bernoulli}(\theta_{pq})  
]

independent. For main edges, (\theta_{pq}=\theta\in[c,1-c]). For finite lattice-fixing edges, choose harmless parameters or include them in expectation adjustment.

Let

[  
\mu  
]

be the distribution of

[  
\sum X_{pq}\frac L{pq}  
\quad\text{in }\mathbb Z/L\mathbb Z.  
]

Characters of the squarefree CRT part are residue assignments

[  
a=(a_p),\qquad a_p\in\mathbb F_p.  
]

For edge (pq),

# [  
\phi_{pq}(a)

\frac{a_p\bar q}{p}+\frac{a_q\bar p}{q}.  
]

Then

# [  
\widehat\mu(a)

\prod_{pq}  
\left(  
1-\theta_{pq}+\theta_{pq}e(\phi_{pq}(a))  
\right).  
]

And

[  
|\widehat\mu(a)|  
\le  
\exp\left(  
-c\sum_{pq}|\phi_{pq}(a)|^2  
\right).  
]

**Dependency:** elementary Fourier/CRT.

**Status:** stable, but the presence of the finite (b)-fixing edges needs notational care. They can be separated as a finite product.

# Section 5. Major arc Gaussian

Define

# [  
\sigma^2

\sum_{pq}\theta_{pq}(1-\theta_{pq})\frac1{p^2q^2}.  
]

Prove:

[  
\sigma\asymp \frac1{k_0 2^{k_0}},  
]

and

[  
a_{\max}:=\max\frac1{pq}  
\asymp2^{-2k_0}  
]

so

[  
a_{\max}=o(\sigma).  
]

For diagonal characters

[  
a_p\equiv h\pmod p,  
]

with

[  
|h|\le C/\sigma,  
]

expand:

# [  
\widehat\mu(h)e(-h/b)

\exp(-2\pi^2\sigma^2h^2+o(1)).  
]

Thus

[  
\sum_{|h|\le C/\sigma}  
\widehat\mu(h)e(-h/b)  
\asymp \frac1\sigma.  
]

**Dependency:** Section 1, Fourier setup.

**Status:** stable.

---

# Section 6. Dense CRT agreement

Fix blocks

[  
P\subset[X,2X],  
\qquad Q\subset[Y,2Y],  
]

and cutoff (B) with

[  
B<Y/2,\qquad X^2>2B.  
]

For residues (a_p,b_q), define CRT glue (H_{pq}^{*}).

If

[  
|H_{pq}^{*}|\le B  
]

for at least (1-\varepsilon) proportion of pairs ((p,q)), then there exists (|h|\le B) such that except (O(\varepsilon)) of vertices,

[  
a_p\equiv h\pmod p,\qquad b_q\equiv h\pmod q.  
]

Proof:

1. Good high vertices have a unique label because (B<Y/2).
    
2. Two good high vertices share many good low neighbours, at least two, so labels agree because (X^2>2B).
    
3. Low vertices linked to good high vertices inherit the label.
    
4. Degree pruning with fixed threshold gives (O(\varepsilon)), not (O(\sqrt\varepsilon)).
    

**Dependency:** elementary CRT and density.

**Status:** stable.

---

# Section 7. Backbone and cut costs

Choose fixed

[  
\gamma=\frac43,\qquad \eta=\frac14.  
]

Backbone links:

[  
P_k^\ast\leftrightarrow P_{\gamma k}^\ast.  
]

Cutoff:

[  
B_k=X_k^{1+\eta}=X_k^{5/4}.  
]

Conditions hold:

[  
B_k<X_k^\gamma,  
\qquad  
X_k^2>2B_k.  
]

If a backbone block pair is not agreement-good, it has bad edge density bounded below by a fixed constant, so its energy is at least

# [  
T_k\asymp  
\frac{X_k^{1-\gamma+2\eta}}{k^2}

\frac{X_k^{1/6}}{k^2}.  
]

Thus

[  
T_k\gg k  
]

for (k\ge k_0) large.

Adjacent good labels merge by product of two middle primes.

**Dependency:** Sections 6 and scale arithmetic.

**Status:** stable.

# Section 8. Vertex correction

Suppose component label (h) controls a lower block (P_x), and (q\sim x^{4/3}) is a good prime. If

[  
a_q\not\equiv h\pmod q,  
]

then with

[  
d=a_q-h  
]

one has

[  
\phi_{pq}(a)  
\equiv  
\frac{h}{pq}+\frac{d\bar p}{q}.  
]

Because

[  
|h|\le x^{5/4},  
\qquad pq\asymp x^{7/3},  
]

the perturbation (h/(pq)) is negligible. Irving non-concentration gives

[  
\sum_{p\in P_x}|\phi_{pq}(a)|^2  
\gg |P_x|.  
]

This cost dominates the entropy (\log q).

**Dependency:** Section 3.

**Status:** stable conditional on clean pruning setup.

# Section 9. Minor arc energy–entropy proposition

Define major arc:

[  
\mathfrak M={a:\exists h,\ |h|\le C/\sigma,\ a_p\equiv h\pmod p\ \forall p}.  
]

Prove:

[  
\sum_{a\notin\mathfrak M}|\widehat\mu(a)|  
=o(1/\sigma).  
]

Proof outline:

For any (a\notin\mathfrak M),

- bad backbone cuts cost (\sum T_k);
    
- after deleting bad cuts, good components get labels;
    
- labels merge inside components;
    
- wrong vertices cost (\gg |P_x|);
    
- remaining global label with (|h|>C/\sigma) has Gaussian tail.
    

Bookkeeping:

[  
\sum_{\mathcal B}  
\exp\left(  
O\left(\sum_{k\in\mathcal B}k\right)  
-c\sum_{k\in\mathcal B}T_k  
\right)  
=1+o(1),  
]

and

[  
\prod_q  
\left(1+e^{\log q-cE_q}\right)=1+o(1).  
]

Then diagonal tail gives (o(1/\sigma)).

**Dependency:** Sections 6–8.

**Status:** conceptually strong, but this is still the section most needing rigorous writing.

# Section 10. Completion for (1/b)

Fourier inversion:

# [  
\mathbb P\left(S=\frac1b\right)

\frac1L  
\sum_a  
\widehat\mu(a)e(-aL/b).  
]

Major arc gives

[  
\gg\frac1{\sigma L}.  
]

Minor arc gives

[  
o\left(\frac1{\sigma L}\right).  
]

Thus probability positive. Therefore there is a subset of candidate semiprime denominators with sum (1/b).

**Dependency:** Sections 5 and 9.

**Status:** stable.

---

# Section 11. From (1/b) to (a/b)

Given finite forbidden set (F), construction can start at arbitrarily large (k_0) and avoid (F). After one representation of (1/b), add all used primes to (F) and repeat (a) times.

This yields disjoint semiprime denominators and

[  
\frac ab=\sum\frac1{pq}.  
]

**Dependency:** construction avoids finite sets.

**Status:** stable.
