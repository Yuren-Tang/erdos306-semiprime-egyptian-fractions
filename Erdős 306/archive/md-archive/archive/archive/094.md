接下來最該做的，不是再開新路線，而是**攻 Section 9：Minor arc energy–entropy proposition**。這是目前唯一仍可能藏有實質漏洞的核心段落。其他部分多半是可寫清的 lemma；Section 9 是「所有部件是否真的合成一個總估計」的地方。

我建議下一步按以下順序推：

---

## 1. 先把 Section 9 改成一個精確命題

不要再說「能量支付熵」，而要寫成類似：

[  
\sum_{a\notin\mathfrak M}  
\exp(-cQ(a))  
\le o(1/\sigma).  
]

或者先證較粗但夠用的版本：

[  
#{a\notin\mathfrak M:Q(a)\le R}  
\le  
\frac1\sigma \exp(\kappa R)  
]

並且

[  
\kappa<c.  
]

這樣再按能量層求和：

[  
\sum_R #{R\le Q(a)<R+1}e^{-cR}  
]

即可閉合。

這一步會迫使我們明確定義：

[  
Q(a),\quad \mathfrak M,\quad \text{bad cuts},\quad \text{wrong vertices},\quad \text{component labels}.  
]

---

## 2. 明確設計 backbone scale chain

先不要用所有 dyadic levels；為了 bookkeeping 清楚，可以選一條幾何尺度鏈：

[  
k_j=\lfloor \gamma^j k_0\rfloor,  
\qquad \gamma=\frac43.  
]

Backbone 只用

[  
P_{k_j}\times P_{k_{j+1}}.  
]

Full graph 仍用所有 compatible pairs 來提供質量和主弧，但 minor arc proof 只依賴這條 backbone。

這樣 bad cuts 就是一維鏈上的 cuts，計數最乾淨：

[  
\mathcal B\subset{j}.  
]

如果用所有 dyadic levels，尺度圖不是線而是稠密有向圖，雖然更強，但 bookkeeping 會亂。先證鏈版最穩。

---

## 3. 對每個 character 做 canonical decomposition

給定 (a)，對每條 backbone link (j\to j+1) 判斷：

- 若 bad edge density (\ge\varepsilon_0)，標為 bad cut；
    
- 否則標為 good link，並由 dense agreement 得 label (h_j)。
    

然後：

- good links 串成 components；
    
- Lemma C 使同一 component 內 labels 合併成 (h_C)；
    
- component 內不服從 (h_C) 的 vertices 是 wrong vertices。
    

這要寫成一個 deterministic algorithm。只有這樣，後面求和時每個 character 都被唯一或有限重地歸類。

---

## 4. 建立兩個付款不等式

第一個：

[  
Q(a)\ge c\sum_{j\in\mathcal B}T_j  
]

其中

[  
T_j\asymp X_{k_j}^{1/6}/k_j^2.  
]

第二個：

[  
Q(a)\ge c\sum_{q\in W}E_q,  
\qquad  
E_q\gg |P_{\text{neighbour}(q)}|.  
]

但二者可能用到同一部分能量，所以正式寫法應為：

[  
Q(a)\ge c\left(  
\sum_{j\in\mathcal B}T_j  
+  
\sum_{q\in W}E_q  
+  
Q_{\mathrm{diag}}(h_{\mathrm{global}})  
\right)  
]

或至少用分區邊集避免重疊。

這裏要做「edge allocation」：  
把 full graph 的邊分成三類，分別供 bad cuts、wrong vertices、diagonal tail 使用。若不分，容易雙重計數。

---

## 5. 計算 entropy

對固定 bad cut 集合 (\mathcal B)：

- components 數 (\le|\mathcal B|+1)；
    
- 每個 component label 有至多 (\ll B_j) 種；
    
- label entropy (\ll\sum_{j\in\mathcal B}k_j+O(\log(1/\sigma)))。
    

wrong vertices：

[  
\prod_{q\in W} q  
]

種 residue choice，entropy

[  
\sum_{q\in W}\log q.  
]

然後用

[  
T_j\gg k_j,  
\qquad  
E_q\gg\log q  
]

吸收。

---

## 6. 最後才回到主定理

一旦 Section 9 的 proposition 完成，整篇 proof 就基本能串起來。  
所以接下來我會做第一步：**寫出精確版 Minor Arc Proposition，並給出 canonical decomposition。**

---

# 下一步開始：精確版 Minor Arc Proposition

令 scale chain

[  
k_j=\lfloor \gamma^j k_0\rfloor,  
\qquad j=0,\dots,J,  
]

其中

[  
\gamma=\frac43.  
]

令

[  
X_j=2^{k_j},  
\qquad  
P_j=P_{k_j}^\ast.  
]

Backbone links:

[  
\mathcal E_j=P_j\times P_{j+1}.  
]

Cutoff:

[  
B_j=X_j^{5/4}.  
]

對 character (a)，邊 (pq\in\mathcal E_j) 是 (j)-good 若

[  
|H_{pq}^{*}|\le B_j.  
]

若 bad density in (\mathcal E_j) 至少固定 (\varepsilon_0)，把 (j) 放入 bad cut set (\mathcal B(a))。否則 (\mathcal E_j) 是 good link，由 dense agreement 得一個 label (h_j).

Good links 的 labels 沿鏈合併；每個 connected interval component (I) 得 label (h_I).

若 (p\in P_j) 位於 component (I)，但

[  
a_p\not\equiv h_I\pmod p,  
]

則 (p) 是 wrong vertex。

這就是 canonical decomposition。

下一步要證：

[  
Q(a)\ge  
c\sum_{j\in\mathcal B(a)}T_j  
+  
c\sum_{p\in W(a)}E_p  
+  
c,Q_{\mathrm{diag}}(h_{\text{labels}})  
]

以適當避免重疊。這就是下一個要推的具體不等式。
