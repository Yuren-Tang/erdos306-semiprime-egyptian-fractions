是，這裏**還有真難度**。而且要分清：

[  
\boxed{\text{不是普通書寫細節；是 minor arc 結構控制中的實體熵問題。}}  
]

但也不是已經看到反例。更準確地說：我們發現「單一 cutoff + 一條 backbone chain」這個簡化版本不夠強，可能讓某些尺度塊在 bookkeeping 裏暫時變成自由，產生巨大 entropy。這不是說原路線錯了，而是說 Section 9 不能用過粗模型收束。

---

## 1. 問題到底在哪裏？

之前我們說：

- bad cut 付能量 (T_k)；
    
- bad cut 允許多一個 component label；
    
- (T_k\gg k)，足以支付 label entropy。
    

這句話只對「每個 component 仍有一個 label」時成立。

但如果某個整層 (P_k) 被兩邊 bad cuts 隔開，而我們沒有證明它自身仍接近某個 label，那麼它的自由度不是

[  
O(k)  
]

而是

[  
\sum_{p\in P_k}\log p\asymp X_k.  
]

這巨大得多。單個 cut cost

[  
T_k\asymp X_k^{1/6}/k^2  
]

完全付不起

[  
X_k.  
]

所以問題是：

[  
\boxed{  
\text{低能 character 是否必然在每個大塊上仍有某種 label？}  
}  
]

如果有，熵小；如果沒有，必須證它付出大量能量。

這是實體問題。

---

## 2. 為什麼我仍覺得不是致命反例？

因為一整層任意亂選 residues，面對 full compatible graph 的大量邊，應該會造成**常數級相位能量**，不是只造成 cutoff 級小能量。也就是說，真正的 energy 會很大，只是我們用

[  
|H|>B\Rightarrow \text{energy}\ge (B/pq)^2  
]

這個下界太粗，粗到付不起熵。

所以漏洞更像：

[  
\boxed{  
\text{我們的下界太弱，不是低能態真的有那麼多自由度。}  
}  
]

如果 residues 無結構，Irving 分散會使 incident phases 大量分散，能量足以殺掉它們。這正是 vertex correction 的精神；只是我們此前只在「已有 label」時使用它。現在需要更強版本：**若沒有 label，也要付能量。**

---

## 3. 所以接下來要證什麼？

我們需要把 Section 9 的核心命題升級成：

[  
\boxed{  
\text{每個尺度塊要麼有一個 label，要麼付出接近其原始熵的能量。}  
}  
]

具體可寫成一個 **block coercivity lemma**：

給定一個 block (P_k) 和若干相鄰 good blocks。對 residues ((a_p)_{p\in P_k})，若不存在整數 (h) 使大多數

[  
a_p\equiv h\pmod p,  
]

則 full graph 中從 (P_k) 到相鄰 blocks 的能量至少

[  
\gg X_k  
]

或至少足以支付

[  
\sum_{p\in P_k}\log p\asymp X_k.  
]

這樣 raw entropy 就被殺掉。

---

## 4. 如何證 block coercivity？

大概有兩種路。

### 路線 A：以「相鄰 labelled block」為基準

如果 (P_k) 的某個相鄰 block 已有 label (h)，那麼每個不滿足

[  
a_p\equiv h\pmod p  
]

的 (p\in P_k) 都由 Irving vertex correction 付能量

[  
\gg |P_{\text{neighbour}}|.  
]

若 (P_k) 中有很多 wrong vertices，總能量巨大，足以支付它們 entropy。

這能處理「已有鄰居 label」的情形。

### 路線 B：兩個相鄰 block 互相迫使 label

如果兩側都沒有預先 label，那就要用 dense block agreement 的反面：

在完整二部圖

[  
P_k\times P_\ell  
]

中，若邊相位能量很低，則 glues 多數集中在某個範圍，並由 dense agreement 產生共同 label。若不能產生 label，則能量必大。

這其實是 **agreement lemma 的能量版本**，不是固定 cutoff 版本。

---

## 5. 單一 cutoff 為什麼不夠？

因為一個 block 可能不是 (B=X^{5/4})-labelled，但也不是亂的。它可能由一個較大的 label

[  
h\sim X^{1.8}  
]

控制。這時：

- 它不通過小 cutoff agreement；
    
- 但它仍只有一個 label，entropy 小；
    
- 它的能量由 Gaussian (h^2/(pq)^2) 支付。
    

所以正確的說法不是「good/bad」二分，而是多層：

[  
\boxed{  
\text{要麼有小 label；要麼有較大 label並付較大 Gaussian energy；要麼無 label並付巨大分散能量。}  
}  
]

這就是我剛才說的 adaptive cutoff / multi-threshold labels。

---

## 6. 下一步應該改造 Section 9

把 Section 9 從「bad cut」模型改成「label scale」模型。

對每個 block component，不只記一個 label (h)，還記它的大小級別：

[  
|h|\le B.  
]

允許 (B) 變大。若 (B) 變大，label entropy 增加

[  
\log B,  
]

但對角型能量也增加：

[  
h^2\sum_{pq\in \text{incident edges}}\frac1{p^2q^2}.  
]

若沒有任何 (B) 可使大多數 glues 受控，則 residues 無共同 label，由 Irving 分散給出大量 incident energy。

也就是把 minor arc 結構改成：

[  
\boxed{  
\text{low energy states are labelled states, with energy depending on label size.}  
}  
]

這比之前的「fixed cutoff labelled states」更正確。

---

## 7. 這是技術問題還是實體問題？

兩者都有，但偏實體。

- **實體部分**：必須證「低能態必有 label」。這不是單純 bookkeeping。
    
- **技術部分**：一旦有這個 coercivity / adaptive agreement lemma，後面的熵求和就是技術 bookkeeping。
    

目前真正要攻的是：

[  
\boxed{  
\text{Adaptive dense CRT agreement / coercivity lemma.}  
}  
]

## 8. 我建議下一步

我們不要立刻處理全圖。先證一個二部 block 版本：

令

[  
P\subset[X,2X],\quad Q\subset[Y,2Y]  
]

完整二部。給 residues (a_p,b_q)。令 energy

[  
Q_{P,Q}(a,b)=  
\sum_{p,q}  
\left(\frac{H_{pq}^{*}}{pq}\right)^2.  
]

證明一個二選一：

1. 存在整數 (h)，使大多數 (p,q) 滿足  
    [  
    a_p\equiv h\pmod p,\quad b_q\equiv h\pmod q,  
    ]  
    且 energy 至少像  
    [  
    h^2\sum_{p,q}\frac1{p^2q^2};  
    ]
    
2. 若不存在這樣的 (h)，則  
    [  
    Q_{P,Q}(a,b)  
    ]  
    至少大到能支付 raw entropy of the non-labelled residues.
    

這是目前最該推的 lemma。若它成立，Section 9 的熵漏洞就補上了。

## 9. 先推一個弱版

先不求完整 entropy，只證：

若

[  
Q_{P,Q}(a,b)  
]

很小，例如比某個固定比例的 (|P||Q|) 小很多，則必存在一個 label (h) 解釋大多數頂點。

這是 energy-to-agreement 版本。證明思路：

小平方能量 (\Rightarrow) 對多數邊

[  
|H_{pq}^{*}|\le B  
]

其中 (B) 選為能量均方尺度乘 (pq)。

然後套 dense agreement lemma 得 label (|h|\le B)。

也就是：

若

[  
Q_{P,Q}\le R,  
]

則 bad edge density for cutoff

[  
B=\lambda XY\sqrt{R/(|P||Q|)}  
]

小。Dense agreement 給 label

[  
|h|\le B.  
]

這很好！它正是 adaptive cutoff。

---

## 10. 這其實很有希望

設

# [  
\overline E

\frac{Q_{P,Q}}{|P||Q|}  
]

是平均平方相位。取

[  
B=10XY\sqrt{\overline E}.  
]

則 Markov 給：

[  
#{|H_{pq}^{*}|>B}\le 0.01|P||Q|.  
]

如果 (B<Y/2) 且 (X^2>2B)，dense agreement 立即給 label (|h|\le B)。

若 (B\ge Y/2) 或 (B\ge X^2/2)，那表示

[  
\overline E  
]

已經不小，因為

[  
B\sim XY\sqrt{\overline E}.  
]

例如 (B\ge Y/2) gives

[  
\sqrt{\overline E}\gg 1/X,  
\quad  
\overline E\gg 1/X^2.  
]

Total energy

[  
Q_{P,Q}\gg |P||Q|/X^2.  
]

Depending on sizes, this may be enough to pay entropy? Need check, but it is much stronger than fixed cutoff.

This is the right adaptive lemma.

## 11. 下一步具體推 adaptive block lemma

We now formulate:

**Adaptive dense block agreement lemma.**  
Let (P\sim X,Q\sim Y), (Y>X). If

[  
Q_{P,Q}(a,b)=R,  
]

then either:

- (R) is large enough to be directly useful; or
    
- there exists (|h|\le CXY\sqrt{R/(|P||Q|)}) explaining all but (O(\epsilon)) vertices.
    

This converts energy itself into label size.

Then global labels can have sizes determined by energy, and entropy (\log |h|) can be paid by (R).

This is exactly what we need.

So yes, we continue by proving this adaptive block lemma.
