# Three-level counting lemma：目標

三層：

[  
P_0\sim X,\qquad P_1\sim Y=X^{4/3},\qquad P_2\sim Z=Y^{4/3}=X^{16/9}.  
]

只看兩組完整二部邊：

[  
P_0\times P_1,\qquad P_1\times P_2.  
]

定義能量

[  
E(a,b,c)=E_{01}+E_{12},  
]

其中

[  
E_{01}=\sum_{p\in P_0,q\in P_1}  
\left(\frac{H_{pq}^{*}}{pq}\right)^2,  
]

[  
E_{12}=\sum_{q\in P_1,r\in P_2}  
\left(\frac{H_{qr}^{*}}{qr}\right)^2.  
]

我們想證一個形如：

# [  
N_3(R)

#{(a,b,c):E(a,b,c)\le R}  
]

滿足

[  
\boxed{  
N_3(R)  
\le  
\sum_{h}  
\exp(CR)\cdot  
\mathbf 1_{\text{diagonal energy of }h\lesssim R}  
}  
]

粗略說：

[  
\log N_3(R)  
\le  
\log N_{\mathrm{diag}}(R)+CR.  
]

這就是三層版 energy–entropy inequality。

---

## 1. 先由 (E_{01}) 產生 label (h_{01})

固定一個 energy budget (R)。若

[  
E_{01}\le R,  
]

adaptive dense agreement 給出：

除非 (E_{01}) 已經進入 energetic case，否則存在 label

[  
h_{01}  
]

解釋大多數 (P_0\cup P_1)，且

[  
|h_{01}|  
\ll  
XY\sqrt{\frac{E_{01}}{|P_0||P_1|}}.  
]

若 (E_{01}) energetic，它本身就消耗一大塊 (R)。在 counting lemma 裏，可把這種情況納入 (e^{CR}) 的誤差：energy 越大，可容許的 entropy 越大。

所以低能 assignments 可先按 (h_{01}) 分類。

同理，由 (E_{12}) 得 label

[  
h_{12}  
]

with

[  
|h_{12}|  
\ll  
YZ\sqrt{\frac{E_{12}}{|P_1||P_2|}}.  
]

---

## 2. labels 不相等時的計數

若

[  
h_{01}\ne h_{12},  
]

但二者都解釋大多數 (P_1)，則對很多

[  
q\in P_1  
]

有

[  
h_{01}\equiv h_{12}\pmod q.  
]

令

[  
D=h_{12}-h_{01}\ne0.  
]

如果共同解釋的中層 primes 至少 (m) 個，則

[  
\prod_{i=1}^m q_i\mid D.  
]

因此

[  
|D|\ge Y^m.  
]

但 (h_{01},h_{12}) 的大小由能量控制。若 (E_{01}+E_{12}\le R)，則

[  
|h_{01}|\ll  
XY\sqrt{\frac R{|P_0||P_1|}},  
]

[  
|h_{12}|\ll  
YZ\sqrt{\frac R{|P_1||P_2|}}.  
]

估算：

[  
|P_0||P_1|\asymp \frac{XY}{(\log X)^2},  
]

所以

# [  
|h_{01}|  
\ll  
XY\sqrt{\frac{R(\log X)^2}{XY}}

(\log X)\sqrt{RXY}.  
]

同理

[  
|h_{12}|  
\ll  
(\log X)\sqrt{RYZ}.  
]

後者較大。若 (D\ne0)，須有

[  
Y^m  
\ll  
(\log X)\sqrt{RYZ}.  
]

這給出

# [  
R\gg  
\frac{Y^{2m}}{YZ(\log X)^2}

\frac{Y^{2m-1}}{Z(\log X)^2}.  
]

取 (m=2)，則

[  
R\gg  
\frac{Y^3}{Z(\log X)^2}.  
]

前面算過：

# [  
\frac{Y^3}{Z}

X^{20/9}.  
]

這是巨大能量。

所以在低能範圍內，兩個 labels 必相等。若它們不等，所需能量巨大，足以被 (e^{-cR}) 殺掉並支付所有相關 label entropy。

這是 counting lemma 的核心。

---

## 3. labels 相等時的計數

若

[  
h_{01}=h_{12}=h,  
]

則三層大多數 vertices 都服從同一 (h)。

剩下的 exceptions：

- (P_0) wrong vertices 用 (P_1) correction；
    
- (P_1) wrong vertices 用 (P_0) 或 (P_2) correction；
    
- (P_2) wrong vertices 用 (P_1) correction。
    

每個 wrong vertex (v) 的 residue choices entropy 是 (\log v)，但 Irving correction energy 是

[  
\gg |P_{\text{adjacent}}|,  
]

遠大於 (\log v)。所以 exceptions contribute at most

[  
e^{O(E_{\mathrm{wrong}})}  
]

with a small constant, absorbed by (e^{CR}).

Thus once common label (h) is fixed, the number of assignments with energy (\le R) is at most

[  
\exp(CR)  
]

times the number of possible (h)'s satisfying the diagonal energy bound.

---

## 4. diagonal energy of (h)

If all three levels follow (h), then energy is approximately

# [  
E_{\mathrm{diag}}(h)

h^2  
\left(  
\sum_{p\in P_0,q\in P_1}\frac1{p^2q^2}  
+  
\sum_{q\in P_1,r\in P_2}\frac1{q^2r^2}  
\right)  
]

as long as (h) is below products; if (h) is larger, dense agreement modulo many primes forces either (h) to be represented by a small congruent label or pays energy. In the low-energy counting range, we can treat (h) by its minimal representative.

The first sum dominates:

[  
\sum_{p\in P_0}\frac1{p^2}  
\sum_{q\in P_1}\frac1{q^2}  
\asymp  
\frac1{X\log X}\cdot \frac1{Y\log Y}.  
]

So

[  
E_{\mathrm{diag}}(h)  
\asymp  
\frac{h^2}{XY(\log X)^2}.  
]

Thus

[  
E_{\mathrm{diag}}(h)\le R  
]

allows

[  
|h|\ll (\log X)\sqrt{RXY}.  
]

This matches the adaptive label bound.

Hence the count of possible (h)'s is

[  
\ll 1+(\log X)\sqrt{RXY}.  
]

But in global Fourier summation this label count becomes the diagonal Gaussian sum, which is exactly the major arc/tail contribution.

---

## 5. 三層 counting lemma 形式

So we can state:

**Three-level counting lemma.**  
For the three-level system above, for all (R\ge1),

[  
N_3(R)  
\le  
\sum_{h\in\mathbb Z}  
\mathbf 1_{E_{\mathrm{diag}}(h)\le C R}  
\exp(CR).  
]

Equivalently,

[  
N_3(R)  
\le  
\exp(CR)  
\left(1+\sqrt{\frac R{\sigma_3^2}}\right),  
]

where

[  
\sigma_3^2  
\asymp  
\sum_{P_0P_1\cup P_1P_2}\frac1{p^2q^2}.  
]

This is exactly the desired local energy–entropy inequality.

## 6. Does this prove global Section 9?

Not immediately, but it gives the local building block.

For the full scale range, cover the scale graph by overlapping triples:

[  
(P_j,P_{j+1},P_{j+2}).  
]

If each triple obeys the counting lemma, one can try to combine them by a Shearer-type entropy inequality or by inductive gluing:

- first triple gives label (h);
    
- next triple sharing two levels must use same (h), unless energy paid;
    
- continue.
    

This is a standard local-to-global pattern.

The major advantage is that the local counting lemma already prevents an isolated middle level from having raw entropy. That was the leak.

---

## 7. Remaining global gluing issue

When combining triples, must avoid multiplying the diagonal label count at every triple. Since overlapping triples force same (h), the global diagonal count should appear only once.

If a triple introduces a new label inconsistent with previous, the shared levels force a large label difference or energy, as in three-level lemma.

Thus induction should work:

Assume first (j) levels are controlled by label (h). Add level (j+1). Apply three-level lemma to ((j-1,j,j+1)). It either uses same (h), or pays energy; wrong vertices corrected. Thus label remains global.

This is promising.

---

## 8. What is still difficult?

The three-level counting lemma proof above is still schematic at two points:

1. It assumes adaptive dense agreement can be applied cleanly at energy (R), with exceptions counted by correction. This is plausible but must be written.
    
2. It treats large modular representatives of (h) as ordinary integers; one must carefully work in a symmetric interval and use product-of-primes to eliminate wrap-around.
    

These are technical but manageable.
