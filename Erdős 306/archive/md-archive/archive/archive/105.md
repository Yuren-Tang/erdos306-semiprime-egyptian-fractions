好。現在先做 **Weighted CRT list-agreement theorem 的 exact version**。  
這是最基本的測試：如果所有局部資料完全相容，是否必然來自一個全局整數 (h)？

---

# 1. Exact CRT agreement：最簡形式

令 (P) 是一組互異素數。給每個

[  
p\in P  
]

一個 residue

[  
a_p\in\mathbb F_p.  
]

若存在整數 (h) 使

[  
a_p\equiv h\pmod p  
\quad\forall p\in P,  
]

則這組 residues 當然是全局一致的。由中國剩餘定理，對任意 residues 其實總存在唯一

[  
h\bmod L,\qquad L=\prod_{p\in P}p,  
]

使

[  
h\equiv a_p\pmod p.  
]

所以在 exact CRT 意義下，「存在全局 (h\bmod L)」是永遠成立的。這提醒我們：真正有內容的不是「存在 (h\bmod L)」，而是：

[  
\boxed{  
\text{是否存在一個小的 ordinary integer }h  
\text{ 解釋 residues。}  
}  
]

Fourier 主弧需要的是小 (h)，不是任意 (h\bmod L)。

---

# 2. 邊約束的 exact 意義

對邊 (pq)，令

[  
H_{pq}^{*}\in(-pq/2,pq/2]  
]

是滿足

[  
H_{pq}^{_}\equiv a_p\pmod p,  
\qquad  
H_{pq}^{_}\equiv a_q\pmod q  
]

的 CRT 對稱代表。

若 exact zero edge energy，即

[  
H_{pq}^{*}=0,  
]

則立刻

[  
a_p=0,\qquad a_q=0.  
]

所以如果 graph 連通且每條邊都有 (H_{pq}^{*}=0)，則

[  
a_p=0  
\quad\forall p.  
]

這是零能剛性。

但主弧不是 (H_{pq}^{*}=0)，而是

[  
H_{pq}^{*}=h  
]

for a common small (h)。所以 exact agreement 的正確命題應該是：

若所有邊的 CRT glue 都等於某個邊 label (H_e)，且這些 (H_e)'s 在重疊頂點上相容、又足夠小，則它們來自同一個 (h)。

---

# 3. Exact edge-label theorem

設圖 (G=(P,E))。對每條邊 (e=pq)，有一個 integer edge label

[  
H_{pq}  
]

滿足

[  
H_{pq}\equiv a_p\pmod p,\qquad  
H_{pq}\equiv a_q\pmod q.  
]

假設還有大小界：

[  
|H_{pq}|\le B.  
]

若兩條邊 (pq_1,pq_2) 共享頂點 (p)，則

[  
H_{pq_1}\equiv H_{pq_2}\pmod p.  
]

因此若

[  
|H_{pq_1}-H_{pq_2}|<p,  
]

則

[  
H_{pq_1}=H_{pq_2}.  
]

更一般地，若沿着一條路徑，edge labels 的差被許多共享頂點 primes 整除，而這些 primes 的乘積超過 (2B)，則 edge labels 必相等。

所以 exact theorem 是：

> 若 (G) 中任意兩條邊可由一條 edge-chain 連接，且沿路共享頂點 primes 的乘積總能超過 (2B)，則所有 edge labels (H_{pq}) 相等於同一個整數 (h)，並且所有 vertex residues 滿足  
> [  
> a_p\equiv h\pmod p.  
> ]

這是我們 dense block agreement 的 exact 核心。

---

# 4. 三層 exact agreement

現在看三層：

[  
P_0\leftrightarrow P_1\leftrightarrow P_2.  
]

假設每條邊有 edge label

[  
H_{pq}  
]

且

[  
|H_{pq}|\le B  
]

並與 endpoint residues 相容。

若 (B) 小於：

- 高層 primes (P_1) 的大小；
    
- 或中層若干 primes 的乘積；
    

則：

1. 在 (P_0P_1) pair 中，所有 edge labels 相等於某個 (h_{01});
    
2. 在 (P_1P_2) pair 中，所有 edge labels 相等於某個 (h_{12});
    
3. 中層 (P_1) 同時滿足  
    [  
    h_{01}\equiv h_{12}\pmod q  
    ]  
    for all (q\in P_1);
    
4. 若 product of enough (q)'s exceeds  
    [  
    |h_{01}-h_{12}|,  
    ]  
    則  
    [  
    h_{01}=h_{12}.  
    ]
    

因此 exact 三層 agreement 成立。

---

# 5. What exact version teaches us

Exact version 告訴我們兩件事：

第一，沒有 algebraic obstruction。  
只要 glues 有共同小界，local agreement 的確推出 global ordinary (h)。

第二，困難完全在 approximate / energy 版：

[  
\text{glues 不是全都小於同一 }B,  
]

而是平方平均小；且 (B) 應該隨能量、尺度、label size 改變。

所以 exact version 支持路線，沒有發現反例。

---

# 6. Approximate version 的正確形式

由 exact version 可知，approximate version 應該是：

若大多數邊的 edge labels (H_{pq}^{*}) 落在某個短 interval

[  
[-B,B],  
]

則存在一個 ordinary label (h) with (|h|\le B)，解釋大多數頂點。

這就是 dense block agreement lemma。

若不同邊只在不同大小 (B) 下好，則要分 energy levels：

[  
B=2^rB_0.  
]

每個 energy level 產生一個可能 label scale。這就是 list-agreement 的來源：每個 vertex 可能暫時屬於不同 label scale/list。

---

# 7. 下一個 exact-list 測試

我們應再測一個稍複雜的 exact-list 情形：

每個 block pair 不是有單一 edge label，而是有少數 labels

[  
h_1,\dots,h_s  
]

使每條好邊的 (H_{pq}) 屬於其中之一。問是否可能存在 (s>1) 的穩定局部結構？

這相當於「玻璃態」檢查。

如果 (s>1)，那麼 vertices 被分成 label clusters：

[  
P=P^{(1)}\cup\cdots\cup P^{(s)}.  
]

在 complete bipartite block 中，若一個 high vertex (q) 連到兩個 labels (h_i,h_j)，則

[  
h_i\equiv h_j\pmod q.  
]

若 (q) 足夠大或多個 primes product 足夠大，則 (h_i=h_j).  
所以不同 labels 必須幾乎分離在 graph 上，形成 cut。

但 complete bipartite graph 沒有便宜 cut：不同 clusters 之間有大量 cross edges，這些 cross edges cannot be good for both labels. Hence they pay energy.

這是 list-collapse mechanism.

---

# 8. Exact list-collapse lemma

在完整二部圖 (P\times Q) 中，假設 vertices are assigned labels from a finite set (\mathcal L), and an edge is good only if the endpoint labels agree. If good edge density is (1-\varepsilon), then all but (O(\varepsilon)) vertices have the same label.

這只是 dense graph majority statement.

Our CRT version adds condition: two labels may be congruent mod a vertex prime. But if label differences are smaller than product of a few primes, congruence implies equality. Large label differences are energy-paid.

So exact list-collapse holds in bounded label range.

---

# 9. This suggests a proof scheme for low-energy version

Use dyadic energy levels:

For each scale pair, sort edges by size of glue:

[  
2^rB_0<|H_{pq}^{*}|\le 2^{r+1}B_0.  
]

For small (r), dense agreement gives dominant label.  
If multiple labels appear, complete bipartite density collapses list to one label unless cross edges have larger glue, i.e. move to higher energy level. Thus list size growth costs energy.

This is exactly a multiscale list-agreement proof.

---

# 10. Is this manageable?

It is not short, but now the structure is crisp:

1. **Exact agreement:** true.
    
2. **Bounded approximate agreement:** dense block lemma, true.
    
3. **List collapse:** true in dense graph with bounded labels.
    
4. **Unbounded labels:** handled by energy levels; larger labels cost quadratic energy.
    
5. **Global gluing:** overlapping triples.
    

So the remaining hard proof is a multiscale induction over energy levels. That is substantial but no longer mysterious.

# 11. Next concrete step

We should formulate **bounded list-collapse lemma** precisely, because it is the bridge from dense agreement to list-agreement.

Then we can attempt to stack dyadic (B)-levels.

Bounded list-collapse lemma:

Let (P,Q) be complete bipartite. Suppose each vertex (v) has a label (\lambda(v)\in[-B,B]\cap\mathbb Z). Suppose for at least (1-\varepsilon) edges (pq),

[  
\lambda(p)\equiv \lambda(q)\pmod p  
\quad\text{and}\quad  
\lambda(p)\equiv \lambda(q)\pmod q  
]

or equivalently endpoint residues agree with same edge glue. If (X^2>2B) and (Y>B), then all but (O(\varepsilon)) vertices have the same label.

This is essentially immediate by the earlier product-of-primes argument.
