好。現在先證 **CRT list collapse lemma**。  
這一步的角色是：即使我們暫時允許多個候選 global labels

[  
\mathcal L={h_1,\dots,h_s},  
]

也要證在稠密 CRT 測試圖中，若多數邊低能，這些 labels 不能長期共存；最後必塌縮成一個主 label，除非 cross edges 支付能量。

# CRT list collapse lemma：基本模型

取兩個 prime blocks

[  
P\subset[X,2X],\qquad Q\subset[Y,2Y],  
]

完整二部圖

[  
P\times Q.  
]

給一組候選 labels

[  
\mathcal L={h_1,\dots,h_s}\subset\mathbb Z.  
]

每個頂點被分配到一個 label：

[  
\tau(p)\in{1,\dots,s},\qquad p\in P,  
]

[  
\tau(q)\in{1,\dots,s},\qquad q\in Q.  
]

其 residue 由該 label 給出：

[  
a_p\equiv h_{\tau(p)}\pmod p,  
]

[  
b_q\equiv h_{\tau(q)}\pmod q.  
]

對邊 (pq)，若

[  
\tau(p)=\tau(q),  
]

則它是同 label edge，CRT glue 就是該 (h_i) modulo (pq)，若 (h_i) 在可見範圍內，能量小。

若

[  
\tau(p)\ne\tau(q),  
]

則 endpoint labels 不同。這種 cross edge 應該產生能量，除非

[  
h_i\equiv h_j\pmod p  
\quad\text{and}\quad  
h_i\equiv h_j\pmod q,  
]

也就是

[  
pq\mid h_i-h_j.  
]

對多數 (p,q)，這不可能。

---

## 1. Cross edge 的基本能量

令

[  
D_{ij}=h_i-h_j\ne0.  
]

若 (p\nmid D_{ij})，則

[  
h_i\not\equiv h_j\pmod p.  
]

對 cross edge (p\in\mathcal C_i), (q\in\mathcal C_j)，CRT glue (H_{pq}^{*}) 是滿足

[  
H_{pq}^{_}\equiv h_i\pmod p,  
\qquad  
H_{pq}^{_}\equiv h_j\pmod q  
]

的整數。

如果這個 glue 很小，比如

[  
|H_{pq}^{*}|\le B,  
]

那麼它給出一個小整數同時接近兩個不同 labels 的模 restrictions。

對固定 (i,j)，若許多 cross edges 都有小 glue，則 dense agreement lemma 會推出 (h_i=h_j) 在普通整數意義下，或至少 (D_{ij}) 被大量 primes 整除。否則 cross edges 必多數 high energy。

所以 list collapse 的基本形式是：

[  
\boxed{  
\text{兩個不同 labels 的兩個大 clusters 之間，cross energy 很大。}  
}  
]

---

## 2. 精確 bounded version

先假設 labels 有共同大小界：

[  
|h_i|\le B  
\quad\forall i,  
]

且

[  
B<Y/2,\qquad X^2>2B.  
]

若一條 cross edge (pq) has small glue

[  
|H_{pq}^{*}|\le B,  
]

則由高點 (q) 的 bounded uniqueness，對固定 (q)，所有 incident small glues with the same (q) must equal (h_j) if (q) belongs to label (j). More directly:

若 (p) label (i), (q) label (j)，小 glue (H) 滿足

[  
H\equiv h_i\pmod p,  
\qquad  
H\equiv h_j\pmod q,  
\qquad |H|,|h_j|\le B<q/2.  
]

若同一 (q) 還有一個 (p') from label (j), then the good edge (p'q) has glue (h_j). Comparing glues modulo (q) gives

[  
H=h_j.  
]

Then

[  
h_j\equiv h_i\pmod p.  
]

若這對很多 (p\in P_i) 成立，product of such (p)'s divides (h_i-h_j). Since

[  
|h_i-h_j|\le2B,  
]

兩個 such (p)'s already force equality if (X^2>2B).

Thus if both clusters are non-negligible, many cross edges cannot be small unless (h_i=h_j).

---

## 3. Dense bipartite collapse estimate

設 label (i) 在 (P)-側比例為

[  
\alpha_i=\frac{|P_i|}{|P|},  
]

在 (Q)-側比例為

[  
\beta_i=\frac{|Q_i|}{|Q|}.  
]

同 label edges 的 density 是

[  
\sum_i\alpha_i\beta_i.  
]

Cross edges density 是

[  
1-\sum_i\alpha_i\beta_i.  
]

若多個 labels 真正共存，例如沒有某一 label 同時佔據 (P,Q) 的絕大多數，則 cross density 有正下界。

對 bounded labels，cross edges 幾乎都必 bad，因為小 glue would force labels equal. Therefore total bad edge density is at least roughly cross density.

如果 low energy implies bad edge density small, then cross density must be small. Hence there exists one (i) with

[  
\alpha_i,\beta_i\approx1.  
]

這就是 collapse。

---

## 4. 定量 lemma

**Bounded CRT list collapse.**  
Assume (|h_i|\le B), (B<Y/2), (X^2>2B). Suppose for at least (1-\varepsilon) of edges (pq\in P\times Q), the CRT glue has

[  
|H_{pq}^{*}|\le B.  
]

Then all but (O(\varepsilon)) vertices in (P\cup Q) are assigned to a single label (h_i).

Proof sketch:

1. From the set of small-glue edges, apply dense block agreement lemma directly to the actual residues (a_p,b_q).
    
2. It returns a single (h\in[-B,B]) explaining all but (O(\varepsilon)) vertices.
    
3. But every non-exception vertex already has residue from some (h_i). If  
    [  
    h_i\equiv h\pmod p  
    ]  
    for many (p), and (|h_i-h|\le2B), product of two primes forces (h_i=h).
    
4. Thus the corresponding label is unique; other labels occupy only (O(\varepsilon)) vertices.
    

This is even simpler: list collapse is a corollary of dense block agreement.

---

## 5. Unbounded labels：按 (H)-level 處理

若 labels 不 bounded, decompose by size:

[  
\mathcal L_r={h_i:2^r\le |h_i|<2^{r+1}}.  
]

For a given level (H=2^r), choose blocks high enough so that

[  
H<Y/2,\qquad H<X^2/2.  
]

Then bounded list collapse applies to labels in this level.

If two labels are in very different size levels, an edge between them has glue scale at least comparable to the larger label unless modulo coincidences occur; such coincidences require divisibility by many primes and are rare/energy-paid.

Thus list collapse can be run level by level, with large (H) handled at high scales.

---

## 6. Cross-level labels

Suppose

[  
|h_i|\ll H,\qquad |h_j|\asymp H  
]

with (H) visible at scale (P,Q). If an edge between clusters (i,j) has glue (|H_{pq}^{*}|\le C H), dense agreement at cutoff (CH) still applies. It will force both labels to equal the same (h) if many such edges exist. If not, cross edges have glue (>CH), paying energy at scale (H^2/(p^2q^2)).

So different size levels also cannot coexist cheaply.

---

# 7. What this buys us

We now know:

[  
\boxed{  
\text{Given a visible label scale }H,\text{ complete bipartite agreement collapses any list at that scale to one label.}  
}  
]

This is exactly the local list-collapse half of the central theorem.

The only remaining nontrivial part is **list decoding**:

How do we know a low-energy assignment can be covered by a small list of visible labels at some scale?

But adaptive cutoff gives a route:

- For each energy layer (R), define (H\sim XY\sqrt{R/(|P||Q|)}).
    
- Most edges have glue (\le H).
    
- Dense agreement gives one label, not merely a list, once (H) is visible.
    
- If (H) invisible at current scale, go to higher scale where (H) visible.
    

Thus list decoding may be avoidable: once visible, dense agreement directly gives one label. The main difficulty remains ensuring some visible scale exists without top entropy; but list collapse itself is fine.

---

## 8. Shift in final strategy

This suggests a possible finite-scale method:

For each (H)-level, restrict attention to blocks with primes above (H). On the subgraph of sufficiently high blocks, low-energy edges yield one label by dense agreement. Then use Irving correction to pull lower blocks to that label. This was our anchor idea, but now framed by (H)-levels.

Top entropy issue persists only for labels (H) larger than all available prime scales. But such labels correspond to diagonal frequencies with very high energy on low edges, and should be killed directly by Fourier energy. Need quantify.

If (H) exceeds top scale, then for low edges (pq), (|h/(pq)|) should behave aperiodically; maybe dense agreement no longer needed. This is exactly high-frequency diagonal tail. It might be controllable by divisibility/aperiodicity, but not yet fully.

So final remaining gap: label levels above available visibility.

This can be solved by choosing top scale high enough relative to the major/minor transition. For frequencies above top visibility, full graph high-frequency should be negligible; may require a separate diagonal aperiodicity argument.

---

# 9. Summary of current position

We have now solidified:

- bounded list collapse: yes;
    
- visible adaptive agreement: yes;
    
- Irving correction: yes;
    
- major arc: yes.
    

Still difficult:

- finite-scale all-energy agreement, especially labels/frequencies above visible top scale;
    
- global counting without top entropy leak.
    

This is the same central theorem, now more sharply isolated.
