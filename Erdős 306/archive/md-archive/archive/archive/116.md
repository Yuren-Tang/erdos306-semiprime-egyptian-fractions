好。現在證 **diagonal small-glue collapse lemma**。這是 diagonal aperiodicity 的最小局部版本。

# Diagonal small-glue collapse lemma

取兩個 prime blocks：

[  
P\subset[X,2X],\qquad Q\subset[Y,2Y],  
]

其中 (Y\ge X)。令

[  
h\in\mathbb Z/L\mathbb Z  
]

是一個 diagonal character。對邊 (pq)，其 glue 就是

# [  
H_{pq}^{*}

(h\bmod pq)_{\mathrm{sym}}  
\in(-pq/2,pq/2].  
]

假設有 cutoff (B)，滿足

[  
B<Y/2,\qquad X^2>2B.  
]

若對至少

[  
1-\varepsilon  
]

比例的邊 (pq\in P\times Q)，有

[  
|H_{pq}^{*}|\le B,  
]

則存在一個整數

[  
h_0,\qquad |h_0|\le B,  
]

使得除了 (O(\varepsilon)) 比例外，

[  
h\equiv h_0\pmod p  
\quad(p\in P),  
]

以及

[  
h\equiv h_0\pmod q  
\quad(q\in Q).  
]

---

## 1. 這其實是 dense agreement 的對角特例

對每個頂點定義 residue：

[  
a_p\equiv h\pmod p,  
\qquad  
a_q\equiv h\pmod q.  
]

對邊 (pq)，CRT glue 正是

[  
H_{pq}^{*}.  
]

所以假設「多數 (H_{pq}^{*}) 小於 (B)」正是 dense block agreement lemma 的前提。

因此直接得到某個

[  
|h_0|\le B  
]

解釋大多數 (P,Q) 頂點。

這個 (h_0) 的含義就是：

[  
h\equiv h_0\pmod p  
]

對大多數 (p\in P\cup Q)。

Lemma 證畢。

---

# 2. 這個 lemma 對 diagonal high-frequency 的意義

若 (h) 是真正高頻，即它不等於任何小整數 modulo 大量 primes，那麼在任何可見 block pair 中，不可能有多數邊使

[  
|h\bmod pq|_{\mathrm{sym}}\le B.  
]

換言之，對每個可見 pair，要麼：

1. (h) 在該 pair 的大多數頂點上等於某個小 (h_0)；
    
2. 或者相當多邊有  
    [  
    |h\bmod pq|_{\mathrm{sym}}>B,  
    ]  
    從而貢獻能量至少  
    [  
    \left(\frac{B}{XY}\right)^2  
    ]  
    each.
    

這給出 diagonal high-frequency 的基本懲罰機制。

---

# 3. 跨 block 黏合

現在若在兩個相鄰可見 block pairs 上得到小 labels：

[  
h_0^{(1)},\qquad h_0^{(2)},  
]

而它們都解釋同一個中間 block (Q)，則對多數 (q\in Q)

[  
h_0^{(1)}\equiv h\equiv h_0^{(2)}\pmod q.  
]

所以

[  
q\mid h_0^{(1)}-h_0^{(2)}  
]

對許多 (q) 成立。

若二者大小都 (\le B)，且取兩個共同 (q) 的乘積超過 (2B)，則

[  
h_0^{(1)}=h_0^{(2)}.  
]

所以 local small labels 在重疊 blocks 上自動合併。

這就是 diagonal 版的 local-to-global。

---

# 4. Diagonal aperiodicity 的可見範圍

對某個 energy layer 或 cutoff (B)，只要存在足夠多的 block pairs 可見，即

[  
B<\min(Y/2,X^2/2),  
]

那麼 (h) 若在這些 pairs 上有很多 small-glue edges，就必須等於同一個小 (h_0) modulo 大量 primes。

若這些 primes 的 product 接近整個 (L)，則 (h) 作為 character 就是小 (h_0)，屬於 major arc。

若不是，剩下未覆蓋 primes 的 incident edges 應該提供能量，或者需要更多 blocks 覆蓋。

---

# 5. 由 small-glue collapse 到能量下界

固定可見 pair (P\times Q)。若 (h) 不是該 pair 上的小 label，則 small-glue density 不能太高。也就是存在 (\delta>0)，使至少 (\delta |P||Q|) 條邊滿足

[  
|H_{pq}^{*}|>B.  
]

於是該 pair 的 diagonal energy

# [  
E_{P,Q}(h)

\sum_{p,q}  
\left|\frac{h}{pq}\right|^2  
]

至少

[  
\gg  
|P||Q|\left(\frac{B}{XY}\right)^2.  
]

這正是 fixed cutoff energy lower bound。

選 (B) 為該 pair 可見範圍內最大值，例如

[  
B\asymp Y  
]

或

[  
B\asymp X^2  
]

較小者，可得到 strongest lower bound。

對 (Y=X^{4/3})，最大可見 (B\asymp Y)。則

# [  
E_{P,Q}(h)  
\gg  
|P||Q|\frac{Y^2}{X^2Y^2}

# \frac{|P||Q|}{X^2}  
\asymp  
\frac{Y}{X(\log X)^2}

\frac{X^{1/3}}{(\log X)^2}.  
]

這是我們反覆見到的可見閾值。

所以每個可見 pair 若不能把 (h) 解釋為小 label，就給出至少

[  
X^{1/3-o(1)}  
]

能量。

---

# 6. 對 diagonal high-frequency 是否足夠？

若 (h) 不是 major arc，沿許多尺度 pairs 它要麼：

- 被解釋為同一個小 (h_0)，最終成為 major arc；
    
- 或在許多 pairs 上支付 (X^{1/3-o(1)}) 能量。
    

對中低頻，這足夠。對非常高頻，仍需確保存在足夠多可見 pairs 覆蓋其 small-glue 可能性。

也就是說，diagonal aperiodicity 現在歸約為：

[  
\boxed{  
\text{若 }h\notin\mathfrak M,  
\text{則在某個可見 pair 上 small-glue collapse 失敗。}  
}  
]

或者更強：

[  
\boxed{  
\text{若 small-glue collapse 在所有可見 pairs 上成功，則 }h\in\mathfrak M.  
}  
]

這是一個純 CRT 覆蓋命題。

---

# 7. CRT 覆蓋命題

設有很多 block pairs，每個成功 pair 給出一個小 label (h_{k,\ell})。重疊使它們合併為一個 (h_0)。若可見 pairs 覆蓋所有 prime blocks，則

[  
h\equiv h_0\pmod p  
]

對大多數 (p) in all blocks.

若其實對所有或幾乎所有 p 成立，則 (h) 在 CRT group 中等於 (h_0) up to exceptions.  
例外 primes 若存在，edges incident to them會使 small-glue collapse fail or pay energy.

因此如果 no energy paid，必有

[  
h\equiv h_0\pmod L.  
]

若 (|h_0|\le C/\sigma)，則 major arc；若 (|h_0|>C/\sigma)，則普通 Gaussian tail付能量。

所以 diagonal high-frequency 可由：

1. local small-glue collapse；
    
2. block overlap merging；
    
3. exception edge energy；
    

三步處理。

---

# 8. 還剩什麼沒有證？

關鍵還是「所有可見 pairs 成功」是否足以覆蓋所有 blocks，尤其最高 blocks / above-top label。

若某個 top block沒有更高可見 pair，但它和低一層 pair仍可見 for cutoffs up to its prime size. 如果 h 在 top block residues 不等於 (h_0), then edges to lower labelled block produce small-glue failure or energy. This is analogous to vertex correction but diagonal; likely easier.

因此 top boundary may be manageable in diagonal case.

---

# 9. Diagonal top correction

Suppose lower block (P) is labelled by (h_0), top vertex (q) has

[  
h\not\equiv h_0\pmod q.  
]

Then for edge (pq), diagonal phase difference from (h_0) is

[  
\frac{(h-h_0)\bar p}{q}  
]

mod 1, exactly as vertex correction. Since (d=h-h_0\not\equiv0\pmod q), Irving correction applies and gives large energy over p.

So top exceptions are handled by same Irving correction.

Great. Thus diagonal high-frequency can be controlled by local label plus correction.

This mirrors the general case.

---

# 10. Summary

Diagonal aperiodicity is not fully written, but now its proof path is clean:

1. Choose a visible pair at some scale.
    
2. If small-glue density high, get small label (h_0).
    
3. Propagate (h_0) through graph using overlap and Irving correction.
    
4. If h differs from (h_0) at any vertex, correction energy.
    
5. If h equals (h_0) everywhere, it is small diagonal; if outside major arc, Gaussian tail.
    

This is essentially the same architecture as the general CRT Poincaré theorem, but diagonal version is easier because the candidate residues are automatically globally linked.
