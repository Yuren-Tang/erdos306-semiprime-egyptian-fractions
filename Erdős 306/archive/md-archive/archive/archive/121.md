好。現在處理 Peierls 框架中的局部核心：**two-phase boundary energy lemma**。

這個 lemma 要回答：

若一側 block 服從 label (h)，另一側 block 服從 label (h'\ne h)，那麼跨邊能量有多大？

---

# Two-phase boundary energy lemma

取兩個 prime blocks：

[  
P\subset[X,2X],\qquad Q\subset[Y,2Y].  
]

假設 (P)-側 residues 來自 label (h)：

[  
a_p\equiv h\pmod p,\qquad p\in P,  
]

而 (Q)-側 residues 來自 label (h')：

[  
a_q\equiv h'\pmod q,\qquad q\in Q.  
]

令

[  
D=h'-h.  
]

若 (D=0)，兩側是同一相。若 (D\ne0)，跨邊相位可寫成：

# [  
\phi_{pq}

\frac{h}{pq}  
+  
\frac{D\bar p}{q}  
\pmod1.  
]

因為若兩端都是 (h)，相位是 (h/(pq))；把 (Q)-側改成 (h')，多出的就是

[  
D\bar p/q.  
]

因此 cross energy 是

# [  
E_{P,Q}(h,h')

\sum_{p\in P,q\in Q}  
\left|  
\frac{h}{pq}  
+  
\frac{D\bar p}{q}  
\right|^2.  
]

---

## 1. 若 (D\not\equiv0\pmod q)，Irving 給出能量

固定 (q\in Q)。若

[  
D\not\equiv0\pmod q,  
]

則由 Irving inverse-residue dispersion：

[  
\sum_{p\in P}  
\left|  
\frac{D\bar p}{q}  
\right|^2  
\gg |P|  
]

provided (P,Q) are in Irving range, i.e.

[  
Q^{2/3}\le X\le Q^{3/2}  
]

in notation (q\sim Q). Our compatible scale choice ensures this.

若 (h/(pq)) 是小擾動，則

[  
\sum_{p\in P}  
\left|  
\frac{h}{pq}  
+  
\frac{D\bar p}{q}  
\right|^2  
\gg |P|.  
]

若 (h/(pq)) 不是小擾動，則 diagonal part itself contributes energy. 所以一般可寫成穩健形式：

[  
\sum_{p\in P}  
\left|  
\frac{h}{pq}  
+  
\frac{D\bar p}{q}  
\right|^2  
+  
\sum_{p\in P}  
\left|\frac{h}{pq}\right|^2  
\gg |P|.  
]

這表示：cross edge either pays boundary energy, or base phase (h) already pays diagonal energy.

在 Peierls expansion 中，(Q_{\mathrm{diag}}(h)) 是基態能量，故 cross excess energy 可從這裏抽出。

---

## 2. 例外 (q)：若 (D\equiv0\pmod q)

如果

[  
q\mid D,  
]

那麼 (h) 和 (h') 在 (q)-頂點不可區分：

[  
h\equiv h'\pmod q.  
]

這些 (q) 不形成 boundary cost。

但能有多少？

令

[  
Q_D={q\in Q:q\mid D}.  
]

若 (Q_D) 佔 (Q) 中正比例，那麼 (D) 被大量 primes (q\sim Y) 整除，因此

[  
|D|  
\ge  
\prod_{q\in Q_D}q.  
]

這極大。若 (h,h') 都在某個 bounded label scale，例如

[  
|h|,|h'|\le B  
]

且

[  
\prod_{q\in Q_D}q>2B,  
]

則只能

[  
D=0.  
]

所以在 bounded regime 中，除非 (h=h')，不能有大量 (q\mid D).

在 unbounded regime 中，若 (D) 巨大，則 labels 本身高頻，diagonal energy / label entropy會處理。這是 multiscale label-level 的內容。

---

## 3. Boundary lower bound

因此，若 (D\ne0) 且 (D) 不被大多數 (q\in Q) 整除，則對至少正比例的 (q\in Q),

[  
D\not\equiv0\pmod q.  
]

每個這樣的 (q) 貢獻

[  
\gg |P|  
]

cross energy。故

[  
E_{P,Q}(h,h')  
+  
E_{P,Q}^{\mathrm{diag}}(h)  
\gg  
|P||Q|.  
]

更精確地，若只有 (\rho |Q|) 個 (q) 不整除 (D)，則

[  
E_{P,Q}(h,h')+E_{P,Q}^{\mathrm{diag}}(h)  
\gg  
\rho |P||Q|.  
]

這是很強的：不是 (X^{1/3}) 那種 cutoff 級能量，而是常數相位級能量

[  
\asymp |P||Q|.  
]

這足以壓 raw entropy。

---

## 4. 這是重要突破

之前用 small-glue cutoff 得到的 boundary cost 只有

[  
X^{1/3}/k^2.  
]

現在用 Irving dispersion 對 cross-label difference (D\bar p/q)，得到的是

[  
|P||Q|\asymp \frac{XY}{k\ell},  
]

這巨大得多，足以支付整塊熵。

這解釋了為什麼 Peierls 方法比 cutoff bad-edge 方法強：  
畴壁不是「相位略大於 cutoff」，而是不同 labels 在跨邊上造成 inverse residues 分散，給常數級能量。

---

## 5. Peierls boundary condition

所以 two-phase boundary lemma 可表述為：

**Lemma.**  
Let (P,Q) be Irving-compatible prime blocks. Suppose (P) is in phase (h), (Q) in phase (h'). Then either:

1. (h\equiv h'\pmod q) for almost all (q\in Q), hence phases are indistinguishable on (Q); or
    
2. the cross energy across (P\times Q) is  
    [  
    \gg |P||Q|  
    ]  
    after subtracting the diagonal background.
    

In bounded label regimes, case 1 implies (h=h').  
In general, case 1 means the labels agree on (Q), so there is no real contour across that side.

This is exactly the Peierls boundary estimate.

---

## 6. Symmetry

We used dispersion in (p) modulo (q). If instead many (q\mid D) but few (p\mid D), use the other direction:

[  
\frac{D\bar q}{p}.  
]

Irving applies because scale compatibility is symmetric within the range:

[  
X^{2/3}\le Y\le X^{3/2}  
]

or equivalent. Thus boundary energy can be obtained from whichever side has many primes not dividing (D).

If (D) is divisible by almost all primes on both sides, then (h,h') agree on both blocks; no boundary there.

---

# 7. Consequence for contours

Consider a cluster (C) labelled (h') surrounded by exterior label (h). Along any boundary block pair (P\times Q):

- if (h,h') differ on a positive fraction of boundary vertices, boundary energy is (\gg |P||Q|);
    
- if they agree modulo almost all boundary primes, then the two phases are actually the same on those vertices, so the boundary can be merged.
    

Thus a genuine contour has enormous energy proportional to complete bipartite boundary size.

This should dominate:

- choosing the contour;
    
- choosing (h');
    
- choosing internal residues.
    

This is the Peierls condition.

---

## 8. Remaining issue: subtracting diagonal background rigorously

We need compare

[  
Q(a)  
]

to

[  
Q_{\mathrm{diag}}(h)  
]

for exterior label (h). Cross edge energy estimate included

[  
E_{P,Q}^{\mathrm{diag}}(h)  
]

as possible alternative. For partition function, we can use inequality like:

[  
|x+y|^2  
\ge  
c|y|^2 - C|x|^2.  
]

Thus

## [  
E_{cross}(h,h')  
\ge  
c\sum_{q:D\not\equiv0}  
\sum_p\left|\frac{D\bar p}{q}\right|^2

C E_{\mathrm{diag}}(h).  
]

When summing over all boundaries, subtracting (E_{\mathrm{diag}}(h)) multiple times could be an issue. But in Peierls expansion, diagonal background is fixed and can absorb a bounded multiple if boundary edge sets are chosen disjoint. Choose disjoint boundary stars so each diagonal edge is subtracted O(1) times.

This is technical but manageable.

---

# 9. This greatly strengthens the route

Now we have:

- single-site defect cost: (\gg |P|);
    
- two-phase boundary cost: (\gg |P||Q|) for genuine phase difference;
    
- bounded list collapse: multiple labels cannot coexist cheaply;
    
- diagonal high-frequency: if (h') is a global phase, counted in diagonal sum; if it forms local cluster, boundary cost kills it.
    

This is exactly the Peierls picture.

The hard theorem is still nontrivial, but we now have the right local energy estimate.
