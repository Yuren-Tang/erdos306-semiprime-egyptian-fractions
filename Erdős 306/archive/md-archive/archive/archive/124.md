# 最終歸約圖

原命題：

[  
\frac ab  
]

其中 (b) squarefree，要表示為互異 semiprime reciprocals 之和。

我們已把它歸約到下列結構：

[  
\boxed{  
\text{Mass/Span}  
+  
\text{Major Arc}  
+  
\text{Irving Correction}  
+  
\text{Peierls Domination}  
+  
\text{Diagonal Aperiodicity}  
}  
]

其中真正未閉合的是最後兩個，尤其 diagonal aperiodicity 與 Peierls contour 的完全形式化。

# 1. Mass / lattice setup

這部分已穩。

取大量 dyadic prime blocks，構造 compatible semiprime graph：

[  
E={pq:\ p,q\in\mathcal P,\ \text{scales compatible}}.  
]

總 reciprocal mass：

[  
W=\sum_{pq\in E}\frac1{pq}  
\asymp \log(K/k_0),  
]

可調到

[  
\theta W=\frac1b  
]

with

[  
\theta\in[c,1-c].  
]

整數化用

[  
L=\prod_{p\in\mathcal P}p  
]

加上 (b) 的素因子。span 問題用 finitely many lattice-fixing edges

[  
r s_r,\qquad r\mid b  
]

修正。這些邊可取極大，不影響主項。

結論：

[  
\mathbb E S=1/b,  
]

且 lattice span 為 1。

# 2. Major arc

對小 diagonal characters

[  
a_p\equiv h\pmod p,\qquad |h|\le C/\sigma,  
]

Fourier 係數展開：

# [  
\widehat\mu(h)e(-h/b)

\exp(-2\pi^2\sigma^2h^2+o(1)).  
]

方差：

[  
\sigma^2\asymp \frac1{k_0^2 2^{2k_0}},  
]

最大權重：

[  
a_{\max}\asymp 2^{-2k_0}=o(\sigma).  
]

三次項：

[  
|h|^3\sum a_{pq}^3  
\ll  
\frac{a_{\max}}{\sigma}  
=o(1).  
]

所以主弧貢獻：

[  
\sum_{|h|\le C/\sigma}  
\widehat\mu(h)e(-h/b)  
\asymp \frac1\sigma  
]

且為正。

這部分也穩。

# 3. Irving correction

Irving theorem 給：

[  
\sum_{q\sim Q}\max_{(a,q)=1}  
\left|  
\sum_{p\sim x}e_q(a\bar p)  
\right|  
\ll  
Q^{5/4}x^{5/8}  
+  
Qx^{9/10}  
+  
Q^{7/6}x^{13/18}  
]

在

[  
Q^{2/3}\le x\le Q^{3/2}  
]

內成立。取

[  
Q=x^{4/3}  
]

有 power saving。

因此對大多數 good (q),

[  
\max_{a\ne0}  
\left|  
\sum_{p\sim x}e_q(a\bar p)  
\right|  
=o(|P_x|).  
]

由 Erdős–Turán 得：

[  
\sum_{p\sim x}  
\left|  
\frac{d\bar p}{q}  
\right|^2  
\gg |P_x|  
\quad(d\ne0).  
]

所以若一個頂點不服從 label (h)，其 incident energy 至少

[  
\gg |P_x|,  
]

遠大於 residue entropy

[  
\log q.  
]

這是單點 defect cost，也是 Peierls 論證的局部核心。這部分也相當穩。

# 4. Dense bounded CRT agreement / list collapse

若在完整二部圖

[  
P\subset[X,2X],\quad Q\subset[Y,2Y]  
]

中，多數 CRT glues 有

[  
|H_{pq}^{*}|\le B,  
]

且

[  
B<Y/2,\qquad X^2>2B,  
]

則存在單一

[  
|h|\le B  
]

解釋大多數頂點：

[  
a_p\equiv h\pmod p,\qquad a_q\equiv h\pmod q.  
]

這也給出 bounded list-collapse：在可見 label 尺度 (B) 內，多個 labels 不能在 dense complete bipartite graph 中共存；它們會塌縮為同一個 label，否則 cross edges 付能量。

這部分也穩。

# 5. Two-phase boundary energy

若 (P) 服從 (h)，(Q) 服從 (h')，令

[  
D=h'-h.  
]

跨邊相位為

[  
\phi_{pq}  
\equiv  
\frac{h}{pq}  
+  
\frac{D\bar p}{q}.  
]

若

[  
D\not\equiv0\pmod q  
]

對正比例 (q\in Q) 成立，Irving 給出每個這樣的 (q) 成本

[  
\gg |P|.  
]

因此 boundary block pair energy 為

[  
\gg |P||Q|.  
]

若 (D\equiv0\pmod q) 對大多數 (q) 成立，則兩個 labels 在 (Q) 上不可區分，應視為同一相在該 boundary 上合併；若 label 有 bounded size，甚至強迫 (h=h')。

這是 Peierls contour 的邊界成本。這個局部估計也相當有力。

# 6. Peierls domination：未完全形式化，但方向明確

目標命題：

[  
\sum_a e^{-cQ(a)}  
\le  
C\sum_h e^{-c'Q_{\mathrm{diag}}(h)}.  
]

意義：

所有非對角激發，包括：

- 單點 defect；
    
- 小 defect 集；
    
- 大 droplet；
    
- 多相共存；
    
- nested contours；
    

都由 defect/boundary energy 支付，其 entropy 被壓住。

局部依據：

- single-site defect cost (\gg |P|);
    
- two-phase boundary cost (\gg |P||Q|);
    
- contour choices entropy (\ll) boundary energy;
    
- internal disorder 遞歸分解成 smaller contours 或 single-site defects.
    

這是很合理的 Peierls 型定理，但目前還是 sketch，需要正式的 canonical contour decomposition 和 nested contour bookkeeping。

# 7. Diagonal aperiodicity：仍是核心硬骨頭

即使 Peierls domination 成立，還要證：

# [  
\sum_{h\notin\mathfrak M}  
e^{-cQ_{\mathrm{diag}}(h)}

o(1/\sigma).  
]

其中

[  
Q_{\mathrm{diag}}(h)=  
\sum_{pq\in E}  
\left|\frac{h}{pq}\right|^2.  
]

小 (h) 區間已由 Gaussian tail 控制：

[  
Q_{\mathrm{diag}}(h)\asymp h^2\sigma^2.  
]

高頻 (h) 的困難是：

[  
h\bmod pq  
]

可能因 CRT 繞回而對許多 (pq) 小。

我們有可行思路：

- 若在某可見 block pair 上多數 (h\bmod pq) 小，dense agreement 得小代表 (m)；
    
- 若 (h) 不服從 (m) at some prime，Irving correction付能量；
    
- 若所有 primes 服從 (m)，則 (h=m\bmod L)，回到 ordinary diagonal；
    
- 若沒有 small-glue seed，則多個 block pairs 給大能量。
    

但這還沒有寫成完整高頻求和定理。這是目前最硬、最需要專攻的部分。

# 8. 若兩大 lemma 成立，原定理如何完成？

Assume:

### Lemma P：Peierls domination

[  
\sum_a e^{-cQ(a)}  
\le  
C\sum_h e^{-c'Q_{\mathrm{diag}}(h)}.  
]

### Lemma D：Diagonal aperiodicity

# [  
\sum_{h\notin\mathfrak M}  
e^{-c'Q_{\mathrm{diag}}(h)}

o(1/\sigma).  
]

Then minor arc follows:

# [  
\sum_{a\notin\mathfrak M}|\widehat\mu(a)|  
\le  
\sum_{a\notin\mathfrak M}e^{-cQ(a)}

o(1/\sigma).  
]

Together with major arc:

[  
\sum_{a\in\mathfrak M}  
\widehat\mu(a)e(-aL/b)  
\asymp  
1/\sigma,  
]

Fourier inversion gives

[  
\mathbb P(S=1/b)>0.  
]

Then repeat (a) times avoiding used primes to get (a/b).

# 10. 下一步最應該做什麼？

我建議下一步專攻 **diagonal aperiodicity**。原因：

1. 它是 Peierls domination 之後仍必需的；
    
2. 它是一維子問題，較容易孤立；
    
3. 若它失敗，整條 Fourier route 會嚴重受阻；
    
4. 若它成功，剩下 non-diagonal 可靠 Peierls contour machinery。
    

具體下一步是嘗試證一個高頻 bound：

[  
\forall h\notin\mathfrak M,\quad  
Q_{\mathrm{diag}}(h)\gg \Psi(h)  
]

足以使

[  
\sum_h e^{-cQ_{\mathrm{diag}}(h)}  
]

收斂到 (O(1/\sigma))，並在主弧外為 (o(1/\sigma))。

我會接着專攻這個。
