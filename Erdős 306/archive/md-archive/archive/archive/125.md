我們現在攻 **diagonal aperiodicity**。上一輪說它是硬骨頭，但已經看出一條有力路徑：不要直接研究整除，而把 diagonal (h) 看成頂點 residues

[  
h_p=h\bmod p.  
]

若這些 residues 在某些 blocks 中不接近同一 ordinary label (m)，則跨邊能量應該大；若接近同一 (m)，則可以用 Irving correction 傳播；若全體都接近 (m)，則 (h=m\bmod L)，回到普通對角 Gaussian。

現在先證一個更強的局部命題。

---

# 1. Diagonal residue spread lemma

取兩個 Irving-compatible prime blocks

[  
P\subset[X,2X],\qquad Q\subset[Y,2Y],  
]

例如

[  
Y=X^{4/3}.  
]

固定 diagonal (h\bmod L)。對每個 (q\in Q)，看

[  
d_q=h-m\pmod q  
]

相對某個 candidate label (m)。

若對正比例的 (q\in Q) 有

[  
d_q\ne0,  
]

且 (P) 中大多數 primes 滿足

[  
h\equiv m\pmod p,  
]

則

[  
E_{P,Q}(h)  
+  
E_{P,Q}(m)  
\gg  
|P|\cdot  
#{q\in Q:h\not\equiv m\pmod q}.  
]

更直白地說：

[  
\boxed{  
Q\text{ 中每一個不服從 }m\text{ 的 }q  
\text{ 都要付 } \gg |P|\text{ 能量。}  
}  
]

這其實是 Irving correction 的 diagonal 版本。

---

## 2. 證明

對 (p\in P) 且

[  
h\equiv m\pmod p,  
]

以及 (q\in Q) with

[  
d=h-m\not\equiv0\pmod q,  
]

有

[  
\frac{h}{pq}  
\equiv  
\frac{m}{pq}  
+  
\frac{d\bar p}{q}  
\pmod1.  
]

Irving dispersion 給

[  
\sum_{p\in P}  
\left|  
\frac{d\bar p}{q}  
\right|^2  
\gg |P|.  
]

利用

[  
|x+y|^2+|x|^2\gg |y|^2  
]

這種基本不等式，得

[  
\sum_{p\in P}  
\left|  
\frac{h}{pq}  
\right|^2  
+  
\sum_{p\in P}  
\left|  
\frac{m}{pq}  
\right|^2  
\gg |P|.  
]

若 (m) 本身在這些邊上能量小，則 (h) 付能量；若 (m) 能量大，則 (m) 在 diagonal tail 中被壓住。無論如何，不能免費。

這個 lemma 很強，因為成本是

[  
|P|\asymp \frac{X}{\log X},  
]

遠大於 (\log q)。

---

# 3. 用它建立「若有一個 seed label，則全圖可控」

假設我們在某個 block pair 找到一個 label (m)，使

[  
h\equiv m\pmod p  
]

對該 pair 的大多數 vertices 成立。

那麼沿 compatible scale graph 向外傳播：

- 若下一層 (Q) 中很多 vertices 不服從 (m)，每個 wrong vertex 成本 (\gg |P|)；
    
- 若 wrong vertices 很少，該層也被 (m) 控制；
    
- 繼續傳播。
    

因此一個 seed label (m) 一旦存在，就會擴張成全局 label，除非支付巨大能量。

這一步比之前的 tower anchor 更好，因為它不要求所有尺度自己可見；只要找到一個 seed，就能靠 Irving correction 傳播。

---

# 4. 所以 diagonal aperiodicity 的核心只剩「seed 存在」

現在問題變成：

若 (E(h)) 不太大，是否必有某個 visible block pair 產生 seed label (m)？

這可以用 contrapositive：

若沒有任何 seed label，則在每個 visible block pair 中，small-glue density 不高。於是每個 visible pair 有大量邊滿足

[  
|h\bmod pq|_{\mathrm{sym}}>B,  
]

給出能量。

之前我們擔心這個 cutoff 能量太弱。但現在注意：只要沒有 seed，我們可以在**很多不同 cutoff levels** 上失敗，而不是只取一個最大可見 cutoff。

這會累積成接近常數級能量，而不只是 (X^{1/3})。

---

# 5. 多 cutoff 疊加：關鍵補強

對一個 block pair (P,Q)，考慮 dyadic thresholds

[  
B_0,2B_0,4B_0,\dots,B_{\max},  
]

其中

[  
B_{\max}\asymp \min(Y,X^2).  
]

若在某個 threshold (B) 上，多數邊有

[  
|H_{pq}^{*}|\le B,  
]

則 dense agreement 給 seed label。

若沒有 seed，則對每個 (B) 都有正比例邊滿足

[  
|H_{pq}^{*}|>B.  
]

用 layer-cake 公式：

[  
\sum_{pq}\left(\frac{|H_{pq}^{*}|}{pq}\right)^2  
]

不只是由最大 (B) 的一層給貢獻，而是由所有 thresholds 累積：

[  
E_{P,Q}(h)  
\asymp  
\int_0^{B_{\max}}  
2t\cdot  
#{pq:|H_{pq}^{*}|>t}  
\frac{dt}{(XY)^2}.  
]

若對所有 (t\le B_{\max}) 都有正比例 tail，則

[  
E_{P,Q}(h)  
\gg  
|P||Q|  
\frac{B_{\max}^2}{X^2Y^2}.  
]

這仍是最大 cutoff 級，對 (B_{\max}=Y) 給

[  
\frac{|P||Q|}{X^2}.  
]

但如果 (B_{\max}) is actually (pq/2), then it would give constant energy. The visibility restriction stops at (Y), not (XY). 所以單 pair 仍有限。

但 across scales, (B_{\max}) increases. Thus no seed across many scales gives cumulative energy.

這是正確方向。

---

# 6. 累積可見能量 across scales

在尺度 (X_j\to X_{j+1}=X_j^{4/3})，最大可見 cutoff 約

[  
B_{\max,j}\asymp X_{j+1}.  
]

該 pair 若無 seed，能量至少

# [  
E_j\gg  
|P_j||P_{j+1}|  
\frac{X_{j+1}^2}{X_j^2X_{j+1}^2}

\frac{|P_j||P_{j+1}|}{X_j^2}.  
]

估算：

# [  
|P_j||P_{j+1}|  
\asymp  
\frac{X_jX_{j+1}}{k_j^2}

\frac{X_j^{7/3}}{k_j^2}.  
]

所以

[  
E_j\gg  
\frac{X_j^{1/3}}{k_j^2}.  
]

這就是 (V_j).

如果沒有 seed at any scale up to (J)，則

[  
E(h)\gg \sum_{j\le J} \frac{X_j^{1/3}}{k_j^2}.  
]

因 (X_j) grows fast, dominated by top:

[  
\asymp \frac{X_J^{1/3}}{k_J^2}.  
]

這正是 visibility threshold. So no-seed up to high scale forces energy at least top threshold.

若 (h) has energy (R), choose (J) with top threshold (\gg R). Then no-seed impossible. Therefore seed exists.

This is the anchoring argument again, but now for diagonal and with no raw entropy issue because we only prove existence for given (h), not count all h.

Good! For each (h) with (E(h)\le R), choose (J) high enough so threshold (>R); then seed exists below (J).

The earlier top entropy problem was about counting all h; but for proving structural statement for a fixed h with energy R, this works.

Then counting follows via m + exceptions. Nice.

# 13. Next concrete attack: divisor counting partition bound

We need bound

[  
Z=\sum_{h\bmod L}e^{-cE(h)}.  
]

Use level sets of small edges. For a given h, define graph

[  
G_h(\delta)={pq:|h/(pq)|<\delta}.  
]

If (E(h)) small, (G_h(\delta)) dense for some (\delta). Dense subgraph implies by dense agreement h equals small m on many vertices. If not dense, energy large.

Thus low-energy h has dense agreement structure. We can encode h by:

- small m on a large vertex set S;
    
- arbitrary residues on complement T.
    

Edges incident to T cost via Irving/dense correction, so T small in entropy-energy sense. This may actually handle all R, because h arbitrary on top primes corresponds to large T; incident edges to S cost huge.

This is same as m+exceptions, but seed S may be obtained from any dense small-edge graph, not requiring top visibility. For high R, S may be smaller, T larger, but energy pays T. Counting may work.

So formulate: for each h, choose maximal set S on which h equals some small m. If S not large, then many edges have no small-glue agreement, causing energy proportional to boundary between S and T. This Peierls again.

Thus diagonal can be handled by Peierls with phases m and defects T. No need top visibility separately.

We are back to Peierls, but now diagonal pure phases included. The final theorem is indeed Peierls + diagonal small m.

This suggests our Peierls domination may actually include diagonal aperiodicity if pure phases restricted to small m; high h becomes a phase with huge defect boundary relative to small m.

But if all vertices follow high h, that's a pure phase high h, no boundary. Then need diagonal energy of high h. So only global high h remains.

For global high h, choose m=nearest small? all vertices are defects, no exterior. Peierls cannot help. This is true diagonal tail.

Thus high global h remains.

# 14. For global high h, can Gaussian energy on low edges suffice?

If h as ordinary representative >C/σ but <lowest product, yes. If h wraps, global high h may not have ordinary small representation. But as a pure phase, h is just an element mod L. Its energy may be small only if it has small residues modulo many pq, which would imply it equals small m modulo many primes; if all vertices follow h, that's still true with m? If h equals small m modulo many primes but not all, then not pure small globally; exceptions are primes where h differs. Those exceptions have boundary energy. So pure high h can be reinterpreted as small phase m plus defects unless no large set of primes shares a small m.

If no large set shares small m, energy large. Thus Peierls can also handle global high h by choosing dominant small m if exists. If no dominant small m, disorder energy large. This is full CRT Poincaré again.

So diagonal tail is not separate after all if Peierls theorem uses small phases only and allows defects covering all system.

This is conceptually unified.

# 15. The real final lemma

We should state one final theorem:

**CRT Peierls–Poincaré theorem.**  
Let (\mathfrak M) be small diagonal labels. Then

[  
\sum_{a\notin\mathfrak M}e^{-cQ(a)}  
=o(1/\sigma).  
]

Its proof uses phases only from small labels (|m|\le C/\sigma). Any character not close to these phases has defects/contours/disorder paying energy.

This avoids separate diagonal tail.

But proving it is exactly the hard part. At least now the conceptual framework is settled.
