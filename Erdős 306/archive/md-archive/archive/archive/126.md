好。現在把路線統一成一個更強、也更乾淨的命題：**small-phase Peierls theorem**。

此前我們把問題拆成 Peierls domination 和 diagonal aperiodicity，後來發現 diagonal high-frequency 也可視為「不靠近任何小相 (m) 的缺陷態」。所以現在乾脆只承認「小對角 (m)」是真正基態；其餘一律當作缺陷、畴壁或無序態，由能量支付。

# Small-phase Peierls theorem

令主弧小相集合為

# [  
\mathcal M

{m\in\mathbb Z:\ |m|\le C/\sigma}.  
]

對每個 (m\in\mathcal M)，定義相 (m)：

[  
a_p\equiv m\pmod p  
\quad\forall p.  
]

定義距離／缺陷：

[  
D_m(a)={p:\ a_p\not\equiv m\pmod p}.  
]

我們想證：

[  
\boxed{  
\sum_{a\notin\mathfrak M}e^{-cQ(a)}  
=o(1/\sigma).  
}  
]

更強地，希望有：

[  
\sum_a e^{-cQ(a)}  
\le  
(1+o(1))  
\sum_{m\in\mathcal M}e^{-c'Q_{\mathrm{diag}}(m)}  
+  
o(1/\sigma),  
]

其中非小相的所有東西都被 Peierls cost 壓掉。

---

## 1. 三種非主弧情形

任一 character (a) 若不在主弧，對小相集合 (\mathcal M) 來說只有三種可能：

### A. 接近某個小相 (m)，但有少量 defects

即大多數 (p) 滿足

[  
a_p\equiv m\pmod p,  
]

少數不滿足。  
這由 single-site Irving correction 處理。

### B. 有兩個或多個小相區域共存

例如一片區域服從 (m)，另一片服從 (m'\ne m)。  
這由 two-phase boundary energy 處理。

### C. 不接近任何小相

即在任何大 block pair 上都不能由某個 (|m|\le C/\sigma) 解釋。  
這是「無序態」。它應由 dense agreement contrapositive 處理：若能量低，必產生某個 small glue label；若沒有小相，則大量邊能量大。

這三類涵蓋了 diagonal high-frequency：若 (h\bmod p) 在多數 primes 上等於某個小 (m)，就是 A；若不同區域等於不同小 (m)，就是 B；若不接近任何小 (m)，就是 C。

---

# 2. A 類：單相小缺陷

固定 (m\in\mathcal M)。若一個 vertex (q) 不服從 (m)，而相鄰 block (P) 多數服從 (m)，則

[  
d=a_q-m\not\equiv0\pmod q.  
]

對 (p\in P),

[  
\phi_{pq}(a)  
\equiv  
\frac{m}{pq}  
+  
\frac{d\bar p}{q}.  
]

Irving correction 給

[  
\sum_{p\in P}|\phi_{pq}(a)|^2  
+  
\sum_{p\in P}\left|\frac{m}{pq}\right|^2  
\gg |P|.  
]

因 (m\in\mathcal M)，主相能量受控，所以 defect 額外成本是

[  
\gg |P|.  
]

而 defect 的熵是

[  
\log q+O(\log |P_j|)\asymp k_j.  
]

由於

[  
|P|\asymp \frac{X}{k}\gg k,  
]

缺陷被壓住。

---

# 3. B 類：兩小相畛域

若 (P) 區域服從 (m)，(Q) 區域服從 (m'\ne m)，令

[  
D=m'-m.  
]

對跨邊

[  
pq\in P\times Q,  
]

有

[  
\phi_{pq}  
\equiv  
\frac{m}{pq}  
+  
\frac{D\bar p}{q}.  
]

若

[  
D\not\equiv0\pmod q  
]

對正比例 (q\in Q) 成立，Irving 給

[  
\sum_{q}  
\sum_{p}  
|\phi_{pq}|^2  
+  
E_{\mathrm{diag}}(m)  
\gg  
|P||Q|.  
]

若 (D\equiv0\pmod q) 對大量 (q) 成立，由於

[  
|D|\le 2C/\sigma  
]

而 block 中任取足夠多 primes 的乘積會超過 (2C/\sigma)，則迫使

[  
D=0,  
]

即

[  
m=m',  
]

矛盾。  
所以兩個不同小相之間有巨大 boundary cost：

[  
\gg |P||Q|.  
]

這遠遠大於 contour 選擇熵與 label 熵。

# 4. C 類：無序態

這是最微妙的一類。要證：

若某個區域既不服從任何小 (m)，也不能分解成少數小相區域，則內部能量大。

可用 bounded dense agreement 的逆否命題。

在一個 visible block pair (P\times Q) 上，取 cutoff

[  
B=C/\sigma  
]

或稍大常數倍。若至少 (1-\varepsilon) 的邊有

[  
|H_{pq}^{*}|\le B,  
]

則 dense agreement 產生一個

[  
|m|\le B  
]

解釋大多數頂點，即存在小相。  
若無小相，則至少正比例邊有

[  
|H_{pq}^{*}|>B.  
]

於是該 pair 能量至少

[  
\gg  
|P||Q|  
\left(\frac{B}{XY}\right)^2.  
]

這個下界如果 (B=C/\sigma) 可能很強，尤其在低層邊上。

讓我們估算最低尺度：

[  
X\sim Y\sim X_0,  
\qquad  
B\sim 1/\sigma\sim k_0X_0.  
]

邊 denominator

[  
XY\sim X_0^2.  
]

所以相位尺度

[  
\frac{B}{XY}\sim \frac{k_0}{X_0}.  
]

邊數約

[  
|P||Q|\sim \frac{X_0^2}{k_0^2}.  
]

能量下界約

[  
\frac{X_0^2}{k_0^2}\cdot\frac{k_0^2}{X_0^2}  
\asymp1.  
]

單 pair 只給常數。但有許多 low-compatible pairs; summing across enough pairs gives large cost. More importantly, disorder on vertices creates Irving-type constant energy when compared to any local majority label. This part still needs a precise multi-pair argument.

所以 C 類仍是核心，但已比「所有高頻」窄：它是「沒有任何小相局部 majority」的真正無序態。

---

# 5. Small-phase Peierls theorem 的正式版本

可以寫成：

**Theorem SP.**  
For every (\varepsilon>0), for (k_0) sufficiently large,

[  
\sum_{a\notin \mathfrak M}e^{-cQ(a)}  
\le  
o(1/\sigma).  
]

Moreover every (a\notin\mathfrak M) admits a decomposition into polymers of three types:

1. single-site defects relative to some (m\in\mathcal M);
    
2. contours separating two small phases (m\ne m');
    
3. disorder polymers containing no small-phase majority;
    

and the total entropy of polymers with cost (T) is at most

[  
e^{\varepsilon T},  
]

while their energy cost is at least (T).

A and B 類的 local cost 已清楚。  
C 類需要一個 disorder polymer lower bound。

# 6. Disorder polymer lower bound

這是下一個要攻的局部命題：

給一個 connected block region (U)。若對每個小相 (m\in\mathcal M)，都有正比例 vertices 不服從 (m)，且 (U) 不能分割成少數 small-phase droplets，則

[  
Q_U(a)\gg \mathrm{Entropy}(U).  
]

如何證？

使用 majority decoding：

1. 在每個 block pair 上，若邊能量低，dense agreement 產生 local small label (m_{ij}\in\mathcal M).
    
2. 若 local labels 不一致，形成 B 類 contour.
    
3. 若某 pair 沒有 local small label，則該 pair 本身為 disorder edge，付能量.
    
4. 因 compatible graph 在 block 層面連通且擴張，不能有大量 disorder without paying enough energy.
    

這正是 agreement-testing reverse union bound 的位置。

---

# 7. 現在最窄的剩餘命題

所以我們把硬點進一步縮小成：

[  
\boxed{  
\text{Disorder polymer lower bound / local majority decoding.}  
}  
]

也就是：

若一個區域沒有小相 majority，又沒有清晰的兩相 contour，則它的內部 CRT energy 足以支付其熵。

這比之前的「整個 minor arc」小得多，也與 agreement testing 文獻完全吻合。

---

# 8. 對信心的評價

這條路現在是合理的：

- 單缺陷：Irving，強；
    
- 兩相界面：Irving，強；
    
- bounded agreement：初等，強；
    
- 無序態：需要 agreement-testing 型 lemma，但現在形式清楚。
    

所以不是沒信心，而是知道最後需要一個真正的 local majority / disorder lemma。這很正常；硬問題不會完全沒有核心。

在一個完整二部 block pair (P\times Q) 中，若不存在任何小 (m\in\mathcal M) 解釋至少 (1-\eta) 的兩側頂點，則證該 pair 的 energy 或 cross-phase decomposition cost 足以支付自由度。
