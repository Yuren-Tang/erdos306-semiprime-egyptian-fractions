好。現在專攻 **local disorder lemma**。這是 small-phase Peierls theorem 中最後比較硬的一塊。

我們先只看一個完整二部 block pair：

[  
P\subset[X,2X],\qquad Q\subset[Y,2Y],  
]

並只考慮邊

[  
P\times Q.  
]

給 residues

[  
a_p\in\mathbb F_p,\qquad b_q\in\mathbb F_q.  
]

定義 CRT glue：

[  
H_{pq}^{*}\in(-pq/2,pq/2],  
]

滿足

[  
H_{pq}^{_}\equiv a_p\pmod p,\qquad  
H_{pq}^{_}\equiv b_q\pmod q.  
]

局部能量：

# [  
Q_{P,Q}(a,b)

\sum_{p,q}  
\left(\frac{H_{pq}^{*}}{pq}\right)^2.  
]

---

# 1. 小相集合

主弧小相是

[  
\mathcal M={m\in\mathbb Z:\ |m|\le M},  
]

其中

[  
M\asymp C/\sigma.  
]

在最低尺度附近，

[  
M\asymp k_0X_0.  
]

若 (P,Q) 的尺度足夠大，使

[  
M<Y/2,\qquad M<X^2/2,  
]

則小相在該 block pair 上可見。

這是 local disorder lemma 的前提。

---

# 2. 若能量低，必有 bounded label

假設

[  
Q_{P,Q}(a,b)  
\le R.  
]

令 adaptive cutoff

[  
B_R  
\asymp  
XY\sqrt{\frac{R}{|P||Q|}}.  
]

若

[  
B_R\le M  
]

且 (M) 可見，則多數邊有

[  
|H_{pq}^{*}|\le M.  
]

由 dense agreement lemma 得到某個

[  
|m|\le M  
]

解釋大多數 (P\cup Q)。

所以在可見小相尺度內：

[  
\boxed{  
\text{低能 } \Rightarrow \text{小相 majority}.  
}  
]

因此若沒有小相 majority，就必有

[  
B_R>M,  
]

即

[  
R\gtrsim  
|P||Q|\left(\frac{M}{XY}\right)^2.  
]

這給出 disorder energy lower bound：

[  
Q_{P,Q}(a,b)  
\gg  
|P||Q|\frac{M^2}{X^2Y^2}.  
]

---

# 3. 這個 lower bound 是否足夠？

估算最低尺度 (X\sim Y\sim X_0)：

[  
|P||Q|\asymp \frac{X_0^2}{k_0^2},  
]

[  
M\asymp k_0X_0.  
]

因此

[  
|P||Q|\frac{M^2}{X_0^4}  
\asymp  
\frac{X_0^2}{k_0^2}  
\cdot  
\frac{k_0^2X_0^2}{X_0^4}  
\asymp1.  
]

單一最低 pair 只給常數能量，不足以支付整塊無序熵。

但注意：若一個區域真的無小相 majority，它不只在一個 pair 上失敗；在許多 compatible pairs 上都失敗。累加後能量增大。

對固定 block (P_k)，compatible partners (\ell\asymp k) 有 (\asymp k) 個。每個 pair 給常數級或更高成本，總計可達 (\asymp k)。這仍未必付整個 block raw entropy (\asymp X_k)。

所以單靠「無小相 majority implies (B_R>M)」仍不夠。這是之前反覆遇到的問題。

---

# 4. 為什麼實際無序應付更多能量？

因為上面的 lower bound 是極弱的：它只說多數邊不是落在 ([-M,M])，但它們可能剛剛超出 (M)。真正隨機無序時，(|H_{pq}^{*}|) 通常是 (pq) 級，給常數相位能量，總能量

[  
\asymp |P||Q|,  
]

足以支付 raw entropy。

因此要證 local disorder lemma，不能只用單 cutoff (M)。要用 **multi-threshold energy decomposition**。

---

# 5. Multi-threshold disorder bound

考慮所有 cutoff：

[  
M,2M,4M,\dots,XY.  
]

如果在某個 cutoff (B) 下，多數邊滿足

[  
|H_{pq}^{*}|\le B,  
]

且 (B) 可見，那麼 dense agreement 給一個 label (|h|\le B).  
若 (|h|\le M)，這是小相；若 (|h|>M)，這是一個大相，需要作為「非主相 droplet」處理，其 diagonal/base energy 會在主弧外被壓住或形成 contour。

若沒有任何 cutoff 給 label，則對所有 (B) 都有正比例邊超過 (B)，layer-cake gives

[  
Q_{P,Q}\gg |P||Q|.  
]

這足以支付 raw entropy。

因此 local disorder 有三分法：

1. 小相 majority：歸入 phase；
    
2. 大 label majority：歸入 non-small phase / contour；
    
3. 無任何 label：常數級無序能量，支付 raw entropy。
    

這是正確的 local lemma。

---

# 6. Local disorder lemma：正式版本

**Lemma.**  
Fix a visible maximum cutoff (B_{\max}\asymp c\min(Y,X^2)). For a block pair (P\times Q), one of the following holds:

### Case I: labelled

There exists an integer (h), (|h|\le B_{\max}), such that all but (O(\eta)) vertices in (P\cup Q) satisfy

[  
a_p\equiv h\pmod p,\qquad b_q\equiv h\pmod q.  
]

### Case II: disorder energy

[  
Q_{P,Q}(a,b)  
\gg_\eta |P||Q|\left(\frac{B_{\max}}{XY}\right)^2.  
]

This still gives only (V_j), not constant, because (B_{\max}<XY). But if no label at all up to (B_{\max}), that is the best visible scale gives.

If (B_{\max}) is (Y), this is (X^{1/3}). Not raw entropy. However if label requires (B>B_{\max}), then label is too large for this pair; it should be detected at higher scales. Thus local lemma cannot alone pay raw entropy; it creates a large label phase.

So the correct statement is not raw disorder energy, but **label-or-pass-up**:

- either labelled at this scale;
    
- or energy (V_j) and unresolved large label, to be handled at higher scale.
    

This is the multiscale recursion.

---

# 7. Multiscale disorder recursion

At each scale pair:

- if local glues concentrate below visible (B_{\max}), get a label;
    
- if not, pay (V_j);
    
- if (V_j) insufficient, move to higher scale where larger labels become visible.
    

In Peierls language, a high-label phase is not disorder; it is another phase. It becomes visible at sufficiently high scale, unless it is global high phase. Global high phase is diagonal tail.

So C 類「無序」真正只應指：即使在 all available relevant scales, no label emerges. Then cumulative (V_j) over scales may be large; but for finite top still issue. Yet global high phase is diagonal tail.

Thus the local disorder lemma alone cannot close; it feeds into phase decomposition across label scales.

---

# 8. 這告訴我們：必須允許大相

Small-phase-only Peierls theorem太嚴格；大 (h) 不能全部立即當缺陷，否則本地看不見。應該允許所有 diagonal (h) 作為 phases，先做 Peierls domination over all phases：

[  
\sum_a e^{-cQ(a)}  
\le  
C\sum_{h\bmod L}e^{-c'Q_{\mathrm{diag}}(h)}.  
]

然後另證 diagonal tail。

這回到上一版分工。這其實更正確。

也就是：

- Non-diagonal disorder can be controlled by allowing arbitrary phase (h);
    
- Diagonal high-frequency is one-dimensional separate problem.
    

Small-phase-only Peierls把兩個問題混在一起，反而困難。

---

# 9. 所以我們回到更穩的二段方案

二段方案是：

### Part A: Peierls domination over all diagonal phases

[  
\sum_a e^{-cQ(a)}  
\le  
C\sum_{h\bmod L}e^{-c'Q_{\mathrm{diag}}(h)}.  
]

這裏 local disorder lemma 允許大 label；因此高 label phase 不算無序，而是 diagonal phase。這避免 visible scale問題。

### Part B: Diagonal aperiodicity

[  
\sum_{h\notin\mathfrak M}e^{-c'Q_{\mathrm{diag}}(h)}  
=o(1/\sigma).  
]

這是獨立的一維難點。

這分工最乾淨。

---

# 10. Local disorder 在 Part A 中如何解決？

因為允許任意 (h\bmod L)，每個低能 local block pair都可以產生某個 label (h) at whatever scale; if label too large for current pair, pass to higher scales. But as a phase variable, it will be counted in diagonal sum, not as entropy defect.

Thus local disorder only needs to show low-energy configurations are close to some diagonal phase, not necessarily small. That is easier because label size no longer constrained by (M).

Still need multiscale agreement, but top high labels are allowed as phases, so no need to kill them in Peierls.

This is why Peierls domination over all phases is the right theorem.

---

# 11. 下一步應做什麼？

We should focus on proving Part A, because it no longer fights diagonal high frequencies. The local disorder lemma for Part A becomes:

If a region has low internal energy, it is close to some diagonal phase (h\bmod L). Since (h) may be arbitrary modulo L, exact CRT guarantees a global h always exists for vertex residues, but low edge energy should imply consistency of the same h across regions. This is agreement theorem, but without smallness requirement; easier in counting because h is counted on RHS.

Thus Part A may be reachable with Peierls/list-agreement over arbitrary phases.

Then Part B diagonal aperiodicity remains.

I think this is the correct stable decomposition.

---

We have learned that small-phase-only Peierls is too ambitious locally. The more robust path is:

[  
\boxed{  
\text{First dominate by all diagonal phases; then handle diagonal high-frequency separately.}  
}  
]

This is not retreat; it is correct separation of variables.
