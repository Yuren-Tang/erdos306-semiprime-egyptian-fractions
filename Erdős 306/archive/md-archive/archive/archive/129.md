好。現在寫 **modular phase agreement lemma**。這是剛才修正後的關鍵：不要把 local phase 強行看成小整數，而要看成一個 CRT residue class。

# Modular phase agreement lemma

取一個 prime block pair

[  
P\subset[X,2X],\qquad Q\subset[Y,2Y].  
]

給 residues

[  
a_p\in\mathbb F_p,\qquad b_q\in\mathbb F_q.  
]

對每條邊 (pq)，CRT 自動給出唯一 residue class

[  
H_{pq}\pmod{pq}  
]

使

[  
H_{pq}\equiv a_p\pmod p,\qquad H_{pq}\equiv b_q\pmod q.  
]

以前我們要求其對稱代表

[  
H_{pq}^{*}  
]

小，從而得到一個小整數 label。  
現在改為模相位語言：

若許多邊的 CRT classes (H_{pq}\pmod{pq}) 在重疊頂點上彼此相容，則它們應該來自一個共同 CRT phase

[  
h\pmod{M},  
]

其中

[  
M=\prod_{p\in P'\cup Q'}p  
]

for large subsets (P'\subset P,\ Q'\subset Q).

也就是：

[  
a_p\equiv h\pmod p  
\quad(p\in P'),  
]

[  
b_q\equiv h\pmod q  
\quad(q\in Q').  
]

這裏 (h) 不必小。它可以是任意 residue modulo (M)。

---

## 1. Exact modular agreement 是平凡但重要的

若選定大多數 vertices (P',Q')，那麼任意 residues

[  
(a_p)_{p\in P'},(b_q)_{q\in Q'}  
]

由 CRT 自動決定唯一

[  
h\pmod{M}.  
]

所以「存在 modular phase」本身沒有內容。

真正有內容的是：  
若 block pair 低能，則 residues 在很多 vertices 上應該由**少數**或**一個**這樣的 phase 描述，而不是每個 vertex 都自由。換言之，我們需要將 vertex-level entropy 壓縮成 phase-level entropy。

對一個完整 block，任意 residues 都可拼成一個 (h\bmod M)，其 entropy 等於原始 vertex entropy。  
所以 modular phase alone 不減熵。

這提醒我們：只允許 arbitrary modular phase 還不夠；Peierls domination RHS 也包含所有 global (h\bmod L)，其 entropy 是 (L)。真正要證的是：低能 configurations 的 entropy 不超過 global diagonal phases 的 entropy，加上可付費 defects。這是合理的。

---

## 2. Local-to-global 的非平凡處

若不同區域有不同 modular phases：

[  
h_A\pmod{M_A},\qquad h_B\pmod{M_B},  
]

且兩區域重疊或有大量跨邊，那麼：

- 若它們在共同 primes 上給同樣 residues，就可以合併成一個更大的 phase modulo (M_AM_B)；
    
- 若它們在某些 boundary primes 上不同，跨邊由 Irving dispersion 付巨大能量。
    

因此 modular Peierls 的作用不是降低單個 region 的 entropy，而是防止多個 independent CRT phases 共存。  
最後整個低能 configuration 只能有一個 global phase

[  
h\bmod L  
]

加少量 defects。

這正是 Peierls domination over all diagonal phases。

---

# 3. Modular two-phase boundary lemma

令區域 (A) 有 phase

[  
h_A\pmod{M_A},  
]

區域 (B) 有 phase

[  
h_B\pmod{M_B}.  
]

考慮 boundary block pair

[  
P\subset A,\qquad Q\subset B.  
]

對 (p\in P),

[  
a_p\equiv h_A\pmod p.  
]

對 (q\in Q),

[  
a_q\equiv h_B\pmod q.  
]

若存在正比例 (q\in Q) 使

[  
h_A\not\equiv h_B\pmod q,  
]

則令

[  
D_q=h_B-h_A\not\equiv0\pmod q.  
]

跨邊相位：

[  
\phi_{pq}  
\equiv  
\frac{h_A}{pq}  
+  
\frac{(h_B-h_A)\bar p}{q}  
\pmod1.  
]

這裏 (h_A) 可取任意代表；第一項是背景 diagonal phase，第二項是差異項。

Irving dispersion 對每個這樣的 (q) 給出：

[  
\sum_{p\in P}  
\left|  
\frac{(h_B-h_A)\bar p}{q}  
\right|^2  
\gg |P|.  
]

所以 boundary energy 巨大：

[  
\gg |P|\cdot  
#{q\in Q:h_A\not\equiv h_B\pmod q}.  
]

若對大多數 (q\in Q) 都有

[  
h_A\equiv h_B\pmod q,  
]

則兩個 phases 在 (Q) 上相容，應把 (Q) 併入同一 global phase，而不是視為 contour。

這是 modular 版的 two-phase boundary lemma。它不要求 (h_A,h_B) 小。

---

## 4. Phase merging rule

定義一個等價／合併規則：

若兩個 local phases 在交界 block 的大多數 primes 上同餘，則合併它們；新的 phase 是 CRT 合併：

[  
h_{A\cup B}\pmod{\operatorname{lcm}(M_A,M_B)}  
]

或更準確地，modulo product of all primes in the merged region.

若不同餘的 boundary primes 比例正，則不合併，並收取 boundary cost。

這樣，一個低能 configuration 經過反覆合併後，會得到：

1. 一個主 global phase (h\bmod M_{\mathrm{main}});
    
2. 若干 contour regions，其 phases 在 boundary 上 genuinely differ and pay energy;
    
3. 少量 single-vertex defects。
    

若 no boundary cost，所有 phases 合併成一個 global

[  
h\bmod L,  
]

即 diagonal character。

---

# 5. Modular Peierls domination 的核心

因此可望證：

[  
\sum_a e^{-cQ(a)}  
\le  
C\sum_{h\bmod L}e^{-c'Q_{\mathrm{diag}}(h)}.  
]

證明思路：

- 每個 assignment (a) 先分解為 local phases；
    
- 相容 phases 合併；
    
- 不相容界面付 Irving boundary energy；
    
- defects 付 single-site Irving energy；
    
- 合併完成後剩下的自由度正是一個 global diagonal phase (h\bmod L)，這正由右側求和計數。
    

所以 arbitrary large phase entropy 不再是問題：它被右側 diagonal sum 完整承擔。

---

# 6. 剩餘的 local disorder 問題

若一個區域內 residues 完全混亂，沒有可識別 local phase，怎麼辦？

在 modular language 下，每個 connected region 其實總有 CRT phase modulo product of its vertices。  
但若這個 region 與外部在 boundary 上不相容，boundary pays；若相容，合併。  
所以「無 local phase」不再是基本概念；所有自由 residues 都可視為一個 modular phase on that region.

這是重大簡化。

代價是：一個小 region 的 phase entropy 很大，但如果它不合併到 global phase，boundary energy必須支付它；如果它合併，就成為 global phase的一部分，不產生額外 entropy。

這正是 Peierls 的本質。

---

# 7. 最小 contour：一個 arbitrary-residue vertex cluster

假設 (C) 是一組 vertices，內部 residues任意。CRT 給它一個 phase

[  
h_C\bmod M_C.  
]

外部 phase

[  
h_0\bmod M_0.  
]

若在 boundary primes 上 (h_C) 與 (h_0) 不同，boundary Irving energy支付。  
若相同，則 CRT 合併，(C) 不是 defect。

所以 even arbitrary cluster is either merged or paid.

這解決了「無序態」的 conceptual problem。  
我們不需要證它有 small label；它有 modular label，並由 merging/boundary 決定是否付費。

---

# 8. 這是否完全解決非對角 Peierls？

概念上是的。剩下是 formal bookkeeping：

1. 如何選 contour clusters？
    
2. 如何避免每個 arbitrary subset 都有 enormous phase entropy？
    
3. 如何用 boundary cost支付不合併 cluster 的 phase entropy？
    

第三點需要檢查。

一個 cluster (C) 的 phase entropy是

[  
\log M_C=\sum_{p\in C}\log p.  
]

Boundary cost若 cluster 很小：single-site Irving cost per vertex (\gg |P_{\mathrm{adj}}|)，足以支付 (\log p).  
若 cluster 很大 whole blocks：boundary cost (\gg |P||Q|), 通常也足以支付 (\sum_{p\in C}\log p)? Need check.

Suppose (C) is an entire high block (Q) of size (|Q|\sim Y/\log Y), entropy (\sim Y). Boundary to adjacent (P) cost (|P||Q|\sim XY/(\log X\log Y)), much larger than (Y) if (X/\log X\to\infty. Good.

So boundary cost pays phase entropy of cluster. Great. This is the key advantage of constant boundary energy.

---

# 9. Contour entropy now works

For any cluster (C), its phase entropy:

[  
H(C)=\sum_{p\in C}\log p.  
]

Boundary cost roughly:

[  
\mathrm{Cost}(\partial C)  
\gg  
\sum_{q\in C_{\partial}}  
|P_{\mathrm{outside}}|  
]

or block version

[  
\gg |P||Q|.  
]

For a whole block cluster, boundary cost is at least adjacent block size times cluster boundary vertices, hence usually

[  
\gg H(C)  
]

by a factor tending to infinity.

This pays:

- choice of (C);
    
- phase residues inside (C);
    
- nested subclusters.
    

Thus modular Peierls domination is now much more plausible.

---

# 10. Formal local cluster cost

Let (C) be a set of vertices in a block (Q), boundary to an exterior labelled block (P). If (C)'s phase differs from exterior on every (q\in C), then each (q) costs

[  
\gg |P|.  
]

Total cost

[  
\gg |C||P|.  
]

Cluster phase entropy

[  
\sum_{q\in C}\log q  
\asymp |C|\log Y.  
]

Since

[  
|P|\gg\log Y,  
]

cost pays entropy.

For multi-block cluster, sum over boundary layers. Compatible graph expansion should ensure each cluster has enough boundary unless it is whole system. If whole system, its phase is global diagonal and counted on RHS.

Thus Peierls domination reduces to graph expansion of compatible block graph, which is plausible.

---

# 11. This is a substantial clarification

The repeated entropy problem was not solved by small labels; it is solved by:

[  
\boxed{  
\text{allow arbitrary modular phases locally, but charge every independent phase by its boundary.}  
}  
]

Then the only uncharged phase is the single global diagonal (h\bmod L), exactly the RHS.

This is the right conceptual resolution.
