好。現在正式寫 **Modular Peierls domination theorem**。這是目前處理 non-diagonal 部分最穩的框架。

# Modular Peierls domination theorem

令

[  
\mathcal P=\bigcup_k P_k^\ast  
]

為 pruning 後的 prime vertices，令

[  
L=\prod_{p\in\mathcal P}p.  
]

對任意 character assignment

[  
a=(a_p)_{p\in\mathcal P},  
\qquad a_p\in\mathbb F_p,  
]

定義能量

# [  
Q(a)

\sum_{pq\in E}  
|\phi_{pq}(a)|^2.  
]

Diagonal phases 是

[  
a^{(h)}_p\equiv h\pmod p,  
\qquad h\in\mathbb Z/L\mathbb Z.  
]

其能量為

[  
Q_{\mathrm{diag}}(h)=Q(a^{(h)}).  
]

目標命題：

[  
\boxed{  
\sum_{a}e^{-cQ(a)}  
\le  
C  
\sum_{h\bmod L}e^{-c'Q_{\mathrm{diag}}(h)}.  
}  
]

這表示所有非對角自由度的 partition function 被全體 diagonal phase 的 partition function 支配。

---

## 1. 為什麼這個命題合理？

因為任意一團 vertices (C) 的 residues 都可以用 CRT 合成一個 local modular phase：

[  
h_C\bmod M_C,  
\qquad  
M_C=\prod_{p\in C}p.  
]

所以局部「無序」不再是無 phase，而是「有自己的局部 phase」。

若這個局部 phase 與外部 phase 相容，它會併入外部，形成更大的 phase。  
若不相容，它與外部之間的 boundary edges 付巨大能量。

因此所有配置都可理解為：

[  
\text{一個全局 diagonal phase}  
+  
\text{若干不相容局部 phase droplets}.  
]

而 droplets 的熵由 boundary energy 支付。

---

# 2. 局部 cluster 的 phase entropy

設 (C\subset\mathcal P) 是一個 cluster。它的 local phase 有

[  
M_C=\prod_{p\in C}p  
]

種可能，phase entropy 是

[  
\log M_C=\sum_{p\in C}\log p.  
]

這正好等於逐頂點 residue entropy。  
所以「把 residues 合成 local phase」本身沒有省熵；省熵來自下一步：如果這個 local phase 不併入全局 phase，它必須付 boundary cost。

---

# 3. Boundary cost 支付 local phase entropy

假設 cluster (C) 與外部 (C^c) 有一個 boundary block pair：

[  
P\subset C^c,\qquad Q\subset C.  
]

外部 phase 是 (h_0)，cluster phase 是 (h_C)。  
若對 (q\in Q)

[  
h_C\not\equiv h_0\pmod q,  
]

則對跨邊 (pq)：

[  
\phi_{pq}  
\equiv  
\frac{h_0}{pq}  
+  
\frac{(h_C-h_0)\bar p}{q}  
\pmod1.  
]

Irving 給出每個 such (q) 的成本：

[  
\sum_{p\in P}  
|\phi_{pq}|^2  
+  
\sum_{p\in P}  
\left|\frac{h_0}{pq}\right|^2  
\gg |P|.  
]

所以若 (Q) 中有 (r) 個 genuinely different vertices，boundary cost 至少

[  
\gg r|P|.  
]

而這些 vertices 的 phase entropy 是

[  
\sum_{q\in Q'}\log q  
\asymp r\log Y.  
]

因為

[  
|P|\gg \log Y,  
]

boundary cost 壓倒 phase entropy。

對整塊 (Q) 而言：

[  
\text{Cost}\gg |Q||P|,  
]

而

[  
\text{Entropy}\asymp |Q|\log Y.  
]

仍然有

[  
|Q||P|\gg |Q|\log Y.  
]

所以一個不合併的 cluster 很昂貴。

---

# 4. 合併規則

若 cluster phase (h_C) 與外部 phase (h_0) 在 boundary 的大多數 primes 上相同：

[  
h_C\equiv h_0\pmod q  
\quad\text{for most }q\in Q,  
]

那麼它們在這些 vertices 上不可區分。  
這些 vertices 應該被併入外部 phase，而不是當作 droplet。

若整個 boundary 都相容，CRT 直接把兩個 phases 合併成一個更大的 phase：

[  
h_{C\cup C^c}\pmod{M_CM_{C^c}}.  
]

因此，只有 genuinely incompatible boundary 才形成 contour；相容者自動合併。

---

# 5. Contour decomposition

對任意 assignment (a)，可以想像以下過程：

1. 起初每個 vertex (p) 自成一個 cluster，phase 為  
    [  
    a_p\bmod p.  
    ]
    
2. 若兩個相鄰 clusters 在其 boundary primes 上相容，合併。
    
3. 若不相容，該 boundary 留作 contour，並記錄 boundary cost。
    
4. 重複，直到所有相容 components 合併完。
    

最後得到若干 maximal phase clusters。  
若只剩一個 cluster，就是一個 global diagonal phase (h\bmod L)。  
若有多個 clusters，它們之間的 boundaries 都 genuinely incompatible，付 Irving boundary cost。

這是 modular Peierls decomposition。

---

# 6. Energy inequality

希望證：

[  
Q(a)  
\ge  
c,Q_{\mathrm{diag}}(h_{\mathrm{global}})  
+  
c\sum_{\gamma\in\Gamma}\mathrm{Cost}(\gamma)  
]

其中：

- (h_{\mathrm{global}}) 是某個選定的外部/global phase；
    
- (\Gamma) 是 contour family；
    
- (\mathrm{Cost}(\gamma)) 是 boundary Irving cost；
    
- 若多個 global-size clusters存在，可任取一個作 exterior，其餘作 droplets；或者把所有 pure phases 的選擇計入右側 diagonal sum。
    

更實際的 partition function proof 不一定需要逐點 inequality；只要每個 contour 的 Boltzmann weight 有

[  
w(\gamma)\le e^{-c\mathrm{Cost}(\gamma)}  
]

並且 entropy

[  
#{\gamma:\mathrm{Cost}=T}\le e^{\varepsilon T}.  
]

---

# 7. Entropy bound

Contour family 的資料包括：

- cluster vertex set；
    
- local phase residues；
    
- nested subclusters；
    
- boundary choices.
    

對一個 cluster (C)，local phase entropy：

[  
\sum_{p\in C}\log p.  
]

若 (C) 不合併到外部，boundary cost 至少：

[  
\sum_{q\in\partial C}|P_{\mathrm{outside}}|.  
]

在 compatible graph 中，每個 boundary vertex 有很多 outside neighbours unless (C) is whole system. 因此：

[  
\mathrm{Cost}(\partial C)  
\gg  
\sum_{p\in C}\log p  
+  
\log(\text{number of possible }C)  
]

for all proper clusters, if (k_0) large.

若 (C) 是 whole system，它沒有 boundary，正是 global diagonal phase，計入 RHS：

[  
\sum_{h\bmod L}e^{-c'Q_{\mathrm{diag}}(h)}.  
]

這就是 Peierls entropy mechanism.

---

# 8. 最危險的 cluster：整個高 block

檢查一下最危險情形：(C=Q) 是一整個高 prime block。

Entropy:

# [  
\log M_C

\sum_{q\in Q}\log q  
\asymp |Q|\log Y  
\asymp Y.  
]

Boundary to adjacent block (P) cost:

[  
|P||Q|  
\asymp  
\frac{X}{\log X}\frac{Y}{\log Y}.  
]

若 (X\to\infty)，則

[  
|P||Q|  
\gg Y  
]

provided

[  
\frac{X}{\log X\log Y}\to\infty.  
]

這當然成立。  
所以 even a whole block droplet is paid.

若 cluster 是一串 many blocks，boundary may be smaller relative to volume if it is an interval in scale graph. But full compatible graph is dense in scale; an interval cluster has two boundary sides, each with complete bipartite pairs to adjacent outside scales. Boundary cost roughly outside block size times boundary block size, still enough to pay at least boundary layer entropy. Interior phase entropy? If entire interval shares one phase, its phase entropy is only one CRT residue modulo product of all interior primes, i.e. (\sum_{\text{interior}}\log p). Boundary cost only sees boundary layers, not whole volume. Is that enough?

This is subtle.

For a large interval droplet with a different phase (h'), interior phase entropy may be huge, but the choice of (h') is a single residue modulo all interior primes. Boundary constraints only restrict (h') on boundary primes; residues on deep interior primes do not affect boundary. Could this entropy leak?

But if droplet is internally a single phase (h'), its residues on deep interior primes are determined by (h'). There are indeed (\prod_{\text{interior}}p) possible h'. Boundary energy depends only on h' modulo boundary primes. Many h' differing only on deep interior primes have same boundary cost. Are they counted on RHS diagonal sum? Not if droplet is not whole system.

This is serious: large droplet internal phase entropy may exceed boundary cost. In usual finite-spin Peierls, phase choices are finite; here phase space grows with volume. That is exactly our recurrent entropy issue.

How to fix? A local phase on a droplet with arbitrary CRT residue has huge entropy. Boundary alone cannot pay volume phase entropy if phase space is continuous/large. Unless internal edges' diagonal energy (Q_{\mathrm{diag}}(h')) inside droplet pays it. Yes: a high arbitrary phase (h') inside droplet has internal diagonal energy on droplet edges. The droplet weight includes

[  
e^{-cQ_{\mathrm{diag},C}(h')}.  
]

So phase entropy is not free; it is governed by diagonal partition function on the droplet.

Thus Peierls domination must be recursive with diagonal partition functions of clusters, not just boundary cost.

This is another key insight.

---

# 9. Recursive domination

For a cluster (C) with internal phase (h_C), its contribution is bounded by

# [  
Z_{\mathrm{diag}}(C)

\sum_{h_C\bmod M_C}  
e^{-cQ_{\mathrm{diag},C}(h_C)}  
]

times boundary penalty.

If diagonal partition function on a proper cluster is controlled by its Gaussian scale, perhaps boundary penalty handles the rest.

But this means Peierls domination over all phases for whole graph may require diagonal aperiodicity on all subclusters. Still one-dimensional.

The huge internal phase entropy is exactly diagonal entropy. We cannot pay it by boundary; we must include it in a diagonal sum.

If cluster is proper, do we allow independent diagonal phase sums for each cluster? Then partition function could factor into many diagonal sums, too large unless boundary penalties suppress multiple clusters.

This is like Pirogov-Sinai with infinite/large phase space; phase entropy per volume matters.

So diagonal aperiodicity remains central even inside Peierls.

---

# 10. Stable conclusion

The recurring issue is definitely core: large phase space of diagonal (h). Peierls controls non-diagonal deviations, but the diagonal phase space is huge and must be controlled by diagonal energy at every cluster scale.

Therefore the final hard theorem is essentially:

[  
\boxed{  
\text{Diagonal partition functions on all relevant subgraphs are small enough.}  
}  
]

If diagonal aperiodicity holds robustly for every connected cluster/subgraph, Peierls works.

Thus diagonal aperiodicity is the central bottleneck.

---

# 11. Where to focus now?

We should stop expanding Peierls until diagonal partition function is understood. The exact bottleneck:

For any substantial semiprime graph (G),

# [  
Z_{\mathrm{diag}}(G)

\sum_{h\bmod L_G}  
\exp\left(  
-c\sum_{pq\in E(G)}  
\left|\frac{h}{pq}\right|^2  
\right)  
]

should be of order

[  
\frac1{\sigma_G}  
]

where

[  
\sigma_G^2=\sum_{pq\in E(G)}\frac1{p^2q^2},  
]

not of order (L_G).

This is exactly weighted Bernoulli aperiodicity/local CLT for the semiprime reciprocal weights.

So yes: the same issue is the core. The right next move is to attack this diagonal partition function theorem directly.

---

# 12. Next theorem to prove

**Semiprime reciprocal diagonal partition theorem.**  
For compatible semiprime graphs (G),

[  
\sum_{h\bmod L_G}  
e^{-cE_G(h)}  
\ll  
\frac1{\sigma_G},  
]

and outside (|h|\le C/\sigma_G),

[  
\sum_{h\notin[-C/\sigma_G,C/\sigma_G]}  
e^{-cE_G(h)}  
=o(1/\sigma_G).  
]

This is the precise diagonal aperiodicity theorem.

Everything else can be built around it.

Next we should focus entirely on this theorem.
