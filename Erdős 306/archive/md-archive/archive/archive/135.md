好。我們現在把 **no-seed theorem** 改寫成一個真正的 **agreement-test statement**。這是下一個正確落點。

---

# 1. 測試系統

頂點：

[  
V=\mathcal P  
]

即所有選入的 primes。

每個頂點 (p) 帶一個 residue：

[  
a_p\in\mathbb F_p.  
]

邊測試：

[  
pq\in E.  
]

對邊 (pq)，由 CRT 得唯一對稱代表：

[  
H_{pq}^{*}\in(-pq/2,pq/2]  
]

滿足

[  
H_{pq}^{_}\equiv a_p\pmod p,  
\qquad  
H_{pq}^{_}\equiv a_q\pmod q.  
]

在 threshold (B_{pq}) 下，稱邊 (pq) **pass** 若

[  
|H_{pq}^{*}|\le B_{pq}.  
]

能量為

[  
Q(a)=\sum_{pq\in E}  
\left(\frac{H_{pq}^{*}}{pq}\right)^2.  
]

所以能量就是所有測試失敗程度的平方和。

---

# 2. Ordinary global object

普通 agreement testing 的 global object 是某個全局 assignment。  
但我們要的 global object 不是任意 assignment，而是 ordinary integer label：

[  
m\in\mathbb Z  
]

使

[  
a_p\equiv m\pmod p  
]

對大多數 (p) 成立。

這是我們的測試比普通 agreement 更強的地方：邊不只是要求兩端「可由某個 CRT class 解釋」——那永遠成立；它要求該 CRT class 有小 ordinary representative。

因此 pass 邊攜帶的是「兩端像同一個小整數」。

---

# 3. 單二部 agreement lemma

對一個完整二部 block pair

[  
P\subset[X,2X],  
\qquad  
Q\subset[Y,2Y],  
]

若

[  
B<Y/2,\qquad X^2>2B,  
]

且 pass 邊密度至少

[  
1-\eta,  
]

則存在 ordinary label

[  
|m|\le B  
]

使大多數 (P\cup Q) 頂點滿足

[  
a_p\equiv m\pmod p.  
]

這是 local agreement lemma。它已基本可靠。

---

# 4. 多尺度 agreement theorem：想要的形式

現在全圖上要證：

**Multiscale CRT agreement theorem.**  
Fix an energy layer (R). Suppose (Q(a)\le R). Then either:

1. 存在 ordinary label (m) with  
    [  
    m^2\sigma^2\lesssim R  
    ]  
    such that all but energy-paid exceptional vertices satisfy  
    [  
    a_p\equiv m\pmod p;  
    ]
    
2. 或者 (a) 屬於 no-seed class，而 no-seed class 的總數滿足  
    [  
    #\mathcal N_R\le e^{\varepsilon R}.  
    ]
    

若此 theorem 成立，則立刻得到：

[  
#{a:Q(a)\le R}  
\le  
e^{\varepsilon R}  
#{m:m^2\sigma^2\lesssim R}.  
]

再能量分層：

[  
\sum_a e^{-cQ(a)}  
\ll  
\sum_R e^{-cR}e^{\varepsilon R}\frac{\sqrt R}{\sigma}  
\ll  
\frac1\sigma.  
]

主弧外也由 Gaussian tail 得

[  
o(1/\sigma).  
]

這就是整個 minor arc 的核心。

---

# 5. 怎樣從 pass 邊得到 seed？

對固定 (R)，每個 block pair 有自然 cutoff：

# [  
B_{P,Q}(R)

C,pq_{\mathrm{scale}}  
\sqrt{\frac{R_{P,Q}}{|P||Q|}}  
]

更粗地取

[  
B_{P,Q}(R)  
\asymp  
XY\sqrt{\frac{R}{|P||Q|}}.  
]

若

[  
Q_{P,Q}(a)\le R_{P,Q},  
]

Markov 給出大多數邊 pass at cutoff (B_{P,Q})。  
若此 (B_{P,Q}) 可見，local agreement 給 seed (m)。

所以 no-seed 意味著：對所有可見 block pairs，要麼局部能量過高，要麼 pass 邊不夠密。兩者都應該貢獻能量。

---

# 6. 這裏需要 reverse union bound

問題是：不能簡單說「每個 pair 失敗，所以總能量大」，因為 failures 可能高度重疊。

Agreement testing 中的 reverse union bound 正是處理這種情況：

若許多局部測試 individually pass，但不存在 global object，則 inconsistent overlaps 會造成額外 failures；反過來，若 global object 不存在，總 failure probability 有下界。

我們需要 CRT 版：

[  
\boxed{  
\text{若沒有 ordinary seed }m,  
\text{則 pass 邊密度在多尺度平均上不能太高。}  
}  
]

等價地：

[  
\boxed{  
\text{若 pass 邊密度在多尺度上很高，則存在 ordinary seed }m.  
}  
]

這就是 multiscale CRT agreement theorem 的核心。

---

# 7. 三層 reverse union bound

先看三層：

[  
P_0\leftrightarrow P_1\leftrightarrow P_2.  
]

若 (P_0P_1) pass 邊多，得到 label (m_{01}).  
若 (P_1P_2) pass 邊多，得到 label (m_{12}).  
兩個 label 都解釋 (P_1) 多數頂點，所以

[  
m_{01}=m_{12}  
]

除非差值被許多 primes 整除；在 bounded visible range 這不可能。  
所以三層 high pass rate implies a common seed.

這是 reverse union bound 的最小模型。

---

# 8. 多尺度版本

多尺度版本沿 overlapping triples 或 block graph propagation：

- 每個 high-pass pair 產生 local label；
    
- 重疊 pairs 的 labels 必合併；
    
- 若合併失敗，重疊 block 上出現大量 non-pass 或 boundary energy；
    
- 因 block graph connected，高 pass density over a connected family forces one global ordinary label.
    

所以 no-seed theorem 可由一個 block-level agreement theorem 推出。

---

# 9. 剩餘難點：不同 cutoff levels

不同 block pairs 的 cutoff (B) 不同。  
Local labels (m_{ij}) 可能有不同大小界。

合併時需要比較：

[  
|m_{ij}-m_{jk}|  
]

是否小於重疊 primes 的乘積。

若 labels 很大，不能用少數 primes 比較。但大 label 對應大的 energy layer：

[  
m^2\sigma^2\lesssim R.  
]

所以在能量層 (R)，只需乘積超過

[  
O(\sqrt R/\sigma).  
]

重疊 block 裏有很多 primes，取少數即可使乘積超過這個值，除非 (R) 極大。極大 (R) 的 contributions should be killed by energy, but counting still needs care. This is a technical but finite problem.

Thus label merging can be made energy-layer dependent.

---

# 10. Agreement theorem 的精確版本

可以寫：

For every (R), define a block pair (P_iP_j) to be (R)-good if:

1. local energy (Q_{ij}(a)\le \rho_{ij}R);
    
2. adaptive cutoff  
    [  
    B_{ij}(R)  
    ]  
    is visible;
    
3. pass density at (B_{ij}(R)) is at least (1-\eta).
    

Then:

If the (R)-good block-pair graph contains a connected chain covering enough scales, then all its labels merge into a single (m) with

[  
m^2\sigma^2\lesssim R.  
]

Vertices outside this good component are exceptions and must be incident to many bad/high-energy pairs; their entropy is paid by energy.

This is the operational statement.

---

# 11. What remains to prove after this?

Two things:

1. **Good component existence**:  
    If no large (R)-good component exists, then many block pairs are bad, hence total energy is large enough to pay entropy.
    
2. **Exception charging**:  
    Vertices not covered by good component have enough incident bad edges or wrong-label correction energy.
    

These are graph expansion / entropy bookkeeping, not number-theoretic mysteries. The number theory is in local agreement and Irving, already in hand.

This is a much better place to be.

# 12. Next step

I will now formulate the **block-level good-component lemma**:

Given an energy layer (R), define good block pairs as above. Then either:

- there is a giant good component producing a global ordinary label (m); or
    
- the set of bad block pairs has total energy cost large enough to dominate the entropy of configurations without such a label.
    

This is the coarse combinatorial heart of no-seed theorem.
