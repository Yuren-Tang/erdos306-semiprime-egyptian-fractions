你記得沒錯；我補查了 Poincaré / log-Sobolev / Gibbs mixing 那條線。結論是：**這條線有概念啟發，但暫時不像能直接給我們所需的 theorem。**

查到的 Poincaré / log-Sobolev / spectral independence 文獻，多數是在研究已知 Gibbs measure 的混合時間、entropy factorisation、spectral independence 等；例如 spectral independence 與 entropy factorisation 可推出某些 spin systems 的 modified log-Sobolev 常數與快速混合。這對「能量壓熵」很相似，但它通常要求 spin system 已有良好的 uniqueness / influence control，且 spin space、相互作用形式比我們的 CRT 測試規整得多。([arXiv](https://arxiv.org/abs/2103.07459?utm_source=chatgpt.com "On Mixing of Markov Chains: Coupling, Spectral Independence, and Entropy Factorization"))

另一類 Poincaré / log-Sobolev for Gibbs measures 的結果依賴凸性或特殊可解結構；這和我們的離散 CRT residue space、非凸多相能量不太匹配。([arXiv](https://arxiv.org/abs/1805.00708?utm_source=chatgpt.com "On Poincare and logarithmic Sobolev inequalities for a class of singular Gibbs measures"))

相較之下，**agreement testing** 仍然更貼近。Dinur–Filmus–Harsha 的 agreement tests 正是 local-to-global 問題：局部 restrictions 在重疊處平均相容，推出存在 global function 解釋大多數局部資料；而且 reverse union bound 是其證明中的關鍵步驟。這幾乎正對我們的 no-seed theorem。([arXiv](https://arxiv.org/abs/1711.09426?utm_source=chatgpt.com "Agreement tests on graphs and hypergraphs"))

所以裁決如下：

[  
\boxed{  
\text{Poincaré/log-Sobolev 是背景語言；agreement testing 才是應借鑑的主工具。}  
}  
]

但我們需要的不是一般 agreement theorem，而是 **CRT-smallness agreement theorem**：邊測試不只是「兩端相容」，而是「兩端 CRT glue 有小 ordinary representative」。這個小性質才會把 global object 從任意 CRT assignment 收縮成 ordinary integer (m)。

# 回到推導：block-level good-component lemma

現在定義尺度 block graph。每個 block 是

[  
P_k^\ast.  
]

若兩個尺度 compatible，就有完整二部邊

[  
P_k^\ast\times P_\ell^\ast.  
]

固定能量層 (R)。對 block pair ((k,\ell))，定義局部能量

# [  
Q_{k\ell}(a)

\sum_{p\in P_k,q\in P_\ell}  
|\phi_{pq}(a)|^2.  
]

取 adaptive cutoff

[  
B_{k\ell}(R)  
\asymp  
X_kX_\ell  
\sqrt{  
\frac{R_{k\ell}}{|P_k||P_\ell|}  
},  
]

其中 (R_{k\ell}) 是分配給該 pair 的能量份額。實際證明中可按 dyadic energy layer 或平均份額選取；現在先概念化。

稱 ((k,\ell)) 為 (R)-good，若：

1. (Q_{k\ell}\le R_{k\ell});
    
2. (B_{k\ell}(R)) 可見：  
    [  
    B_{k\ell}<c\min(X_\ell,X_k^2);  
    ]
    
3. 在 cutoff (B_{k\ell}) 下 pass 邊密度 (\ge1-\eta)。
    

則 local CRT agreement 給出一個 ordinary label

[  
m_{k\ell}  
]

使大多數

[  
P_k\cup P_\ell  
]

服從

[  
a_p\equiv m_{k\ell}\pmod p.  
]

---

## 1. Good pairs 的 labels 會合併

若兩個 good pairs 重疊一個 block：

[  
(k,\ell),\qquad (\ell,r),  
]

它們給出 labels

[  
m_{k\ell},\qquad m_{\ell r}.  
]

兩者都解釋 (P_\ell) 的大多數 primes。故對許多

[  
p\in P_\ell  
]

有

[  
m_{k\ell}\equiv m_{\ell r}\pmod p.  
]

若

[  
m_{k\ell}\ne m_{\ell r},  
]

則差值被許多 (p) 整除。只要取足夠多這樣的 primes，其乘積超過能量層 (R) 所允許的 label size，就得

[  
m_{k\ell}=m_{\ell r}.  
]

若 label size 已大到不可比較，則 (m^2\sigma^2) 已相應變大，會進入高 (R) 層；在 counting bound 中由 (m)-Gaussian 數量支付。

所以在同一 (R)-good connected component 中，所有 local labels 合併成一個

[  
m_C.  
]

---

## 2. 大 good component 給 seed

若 (R)-good block graph 有一個大 connected component (C)，那麼它給出 ordinary seed label

[  
m_C.  
]

在 component 內，大多數 vertices 服從 (m_C)。

對 component 外但相鄰的 block，若某 vertex 不服從 (m_C)，則用 component 邊界中的 labelled block 做 Irving correction，得到每個 wrong vertex 成本

[  
\gg |P_{\mathrm{adj}}|.  
]

若整個外部 block 都不服從，那麼它與 component 的 boundary block pair 多數邊不是 pass，或形成另一 label contour，總之付 boundary energy。

因此有大 good component時，configuration 可以由：

- label (m_C);
    
- energy-paid exceptions/contours;
    

編碼。這給

[  
e^{\varepsilon R}#{m:m^2\sigma^2\lesssim R}  
]

型 bound。

---

## 3. 若沒有大 good component

那就是 no-seed 情形。要證它的總數小，或至少其能量成本大。

這裏需要 block graph expansion / reverse union bound：

- 若很多 block pairs good，則 good components 不應全都小，除非 labels 在 overlaps 上不合併；但不合併會造成額外 bad edges/energy。
    
- 若 good pairs 很少，則很多 block pairs bad。Bad 代表局部能量高或 pass density低，兩者都貢獻 (Q(a))。
    
- 若有許多小 good components，每個 component 的 boundary 有 bad pairs；boundary energy支付 component label/vertex entropy。
    

這就是 Peierls contour 在 block graph 上的版本。

所以 good-component lemma 應為：

[  
\boxed{  
\text{No giant good component}  
\Rightarrow  
\text{bad-pair boundary cost dominates entropy.}  
}  
]

---

## 4. 這裏最需要 agreement testing 的地方

在 ordinary graph expansion 中，「沒有 giant component」可由 boundary edges 多來處理。  
但我們還要處理 labels：不同 good components可能有不同 (m)。若它們相鄰而 labels不同，則 boundary energy很大；若 labels相同，它們應合併。

這正是 agreement testing 的 local-to-global 合併問題。DFH reverse union bound 的作用就是：局部 agreement 高時，重疊依賴不能被當成獨立漏掉；它會強迫 global majority-decoded object。([arXiv](https://arxiv.org/abs/1711.09426?utm_source=chatgpt.com "Agreement tests on graphs and hypergraphs"))

我們的版本要把「global object」換成 ordinary (m)，這由 CRT smallness補上。

---

## 5. 結論：Poincaré 工具線的裁決

- Poincaré/log-Sobolev/spectral independence：有用作「能量壓熵」語言，但不直接解。
    
- Agreement testing / reverse union bound：最貼近 no-seed theorem。
    
- Weighted Bernoulli LCLT：貼近 Fourier 高頻表述，但仍需我們自己的 CRT aperiodicity/agreement 證明。
    
- Inverse Littlewood–Offord：可能可作備用工具，用於若高頻集中則推出權重結構；但目前不如 agreement 路線直接。
    

所以我會沿 good-component / reverse-union agreement 路線繼續，而不是改投一般 Poincaré 工具。
