好。現在正式寫 **good-component counting lemma**。這是 no-seed theorem 的組合核心。

# Good-component counting lemma

固定能量層

[  
R.  
]

考慮 block graph：

[  
\mathcal B={k_0,\dots,K},  
]

每個頂點 (k) 對應 prime block

[  
P_k^\ast.  
]

若尺度 (k,\ell) compatible，則有 block edge

[  
(k,\ell)  
]

代表完整二部 prime graph

[  
P_k^\ast\times P_\ell^\ast.  
]

對 assignment (a)，定義局部能量

# [  
Q_{k\ell}(a)

\sum_{p\in P_k^\ast,q\in P_\ell^\ast}  
|\phi_{pq}(a)|^2.  
]

---

## 1. (R)-good block edge

稱 block edge ((k,\ell)) 為 (R)-good，若存在 cutoff (B_{k\ell}) 滿足：

[  
B_{k\ell}<c\min(X_\ell,X_k^2)  
]

且在 (P_kP_\ell) 中，至少

[  
1-\eta  
]

比例的 prime edges 有

[  
|H_{pq}^{*}|\le B_{k\ell}.  
]

同時 (B_{k\ell}) 不可任意大，而要由能量層控制：

[  
|P_k||P_\ell|  
\left(\frac{B_{k\ell}}{X_kX_\ell}\right)^2  
\lesssim R.  
]

這樣 local agreement 給出一個 ordinary label

[  
m_{k\ell},\qquad |m_{k\ell}|\le B_{k\ell},  
]

使大多數

[  
P_k\cup P_\ell  
]

服從

[  
a_p\equiv m_{k\ell}\pmod p.  
]

---

## 2. Good edges 合成 good components

令

[  
\mathcal G_R(a)  
]

是由 (R)-good block edges 組成的 graph。

若兩條 good edges 重疊一個 block，例如

[  
(k,\ell),\quad(\ell,r),  
]

則它們的 labels

[  
m_{k\ell},\quad m_{\ell r}  
]

都解釋 (P_\ell) 的大多數 vertices。若它們不等，則差值被許多 (p\in P_\ell) 整除。

在能量層 (R) 中，labels 的大小至多約

[  
\sqrt R/\sigma  
]

級別；而 (P_\ell) 中取少數 primes 的乘積即可超過這個量，除非 (R) 極大。對極大 (R)，能量權重本身壓住。故在 relevant range 中：

[  
m_{k\ell}=m_{\ell r}.  
]

因此每個 good connected component (C) 有一個共同 label

[  
m_C.  
]

---

## 3. Giant good component

若存在一個 good component (C) 覆蓋大多數 block mass，例如覆蓋所有除了一組 boundary-small blocks 外的 blocks，則 (m_C) 是 ordinary seed。

對 (C) 內大多數 vertices：

[  
a_p\equiv m_C\pmod p.  
]

對 (C) 外的相鄰 block，若某 vertex 不服從 (m_C)，Irving correction 給出成本

[  
\gg |P_{\mathrm{adj}}|.  
]

因此 outside defects 的 entropy 被能量支付。

這一情形給出計數：

[  
#{a:Q(a)\le R,\text{ has giant good component}}  
\le  
e^{\varepsilon R}  
#{m:m^2\sigma^2\lesssim R}.  
]

這是好情形。

---

## 4. 沒有 giant good component

若沒有 giant good component，則 block graph 被分成許多 good components，或 good edges 很少。

這時考慮 good components 的 boundary：

# [  
\partial C

{(k,\ell):k\in C,\ell\notin C,\ (k,\ell)\text{ compatible}}.  
]

對 boundary edge ((k,\ell))，若它不是 good，則原因之一是：

1. pass density 不足：很多 prime edges 有大 glue；
    
2. 局部能量太高；
    
3. cutoff 不可見；這表示該 pair 無法在當前 scale 看見相關 label，但 then adjacent higher/lower scales must handle it; in block-level formulation, this edge is treated as unresolved and charged at another scale.
    

若 boundary edge good，則它應把 (C) 和外部合併，除非 labels 不一致；labels 不一致又會在重疊 blocks / boundary edges 上產生 Irving boundary energy。

因此每個 genuine boundary edge carries energy cost.

---

## 5. Boundary cost 支付 component entropy

一個 good component (C) 的自由資料包括：

- component shape；
    
- component label (m_C);
    
- exceptional vertices inside；
    
- local residues in bad parts.
    

如果 component 不合併到 giant seed，則其 boundary bad cost要支付這些資料。

對 component label (m_C) 的 entropy：

[  
\log |m_C|  
\lesssim \log(\sqrt R/\sigma),  
]

這遠小於單個 boundary block pair cost，只要 boundary pair在非極低尺度。

對 component shape entropy：

[  
O(|\partial C|\log K),  
]

也被 boundary block pair costs支付。

對 exceptional vertices，Irving single-site cost支付。

因此若 no giant component，總 entropy由 boundary bad energy支付：

[  
#{\text{no-giant configurations with cost }\le R}  
\le  
e^{\varepsilon R}.  
]

這就是 no-seed class bound。

---

# 6. Good-component counting lemma：正式表述

**Lemma.**  
For every (\varepsilon>0), if (k_0) is sufficiently large, then for each energy layer (R),

[  
#{a:Q(a)\le R}  
\le  
e^{\varepsilon R}  
\left(  
1+#{m\in\mathbb Z:m^2\sigma^2\le C R}  
\right).  
]

更精確地，若 (a) 沒有 giant good component producing such an (m)，則它屬於 no-seed class，其 cardinality (\le e^{\varepsilon R}).

這正是我們需要的分層計數。

---

# 7. 還沒有完全證明的點

這個 lemma 現在還有三個技術缺口：

### (i) Cutoff 不可見的 boundary edges

若 edge 不可見，不應直接算 bad cost；可能需要更高 scale。  
這要求 good-component graph 是多尺度的，而不是只看相鄰固定尺度。解法是：對每個 (R)，只在可見尺度族中建 graph；不可見處由 higher visible edges覆蓋。

### (ii) Label 大小與合併

需要嚴格證在能量層 (R) 下，local labels 的大小可由

[  
\sqrt R/\sigma  
]

控制，並且重疊 block 的 prime product足以比較。這是可寫的，但要細節。

### (iii) Boundary expansion of block graph

需要明確選擇 block graph，使任何非 giant component有足夠 boundary edges。  
Compatible graph 在尺度上很稠密，應可保證；但要避免 overcount energy，需要取 bounded-overlap 子圖，例如固定幾條 scale chains 或 expander-like skeleton。

---

# 8. 這個 lemma 是否可信？

比早先版本可信得多，因為它準確抓住了：

- local agreement gives labels；
    
- labels merge through overlap；
    
- no giant component creates boundary；
    
- boundary/bad edges pay entropy。
    

但它仍需要一個嚴格的 block-level expansion/agreement proof。這正是 agreement testing 工具能幫的地方。

---

# 9. 立即後果

若 Good-component counting lemma 成立，則

# [  
Z

\sum_a e^{-cQ(a)}  
]

按 (R) 分層：

[  
Z  
\le  
\sum_{R\ge0}  
e^{-cR}  
e^{\varepsilon R}  
\left(1+\frac{\sqrt R}{\sigma}\right)  
\ll  
\frac1\sigma.  
]

主弧外，排除

[  
|m|\le C/\sigma  
]

後，Gaussian tail給

[  
o(1/\sigma).  
]

所以它正是最後需要的核心 lemma。

---

# 10. 下一步

下一步要處理缺口 (iii)：設計一個 bounded-overlap block skeleton，使：

1. 它仍足以傳播 labels；
    
2. 每個非 giant component 有很多 boundary；
    
3. 每條 prime edge能量只被計算 (O(1)) 次。
    

這樣 good-component lemma 可以變成嚴格組合命題。
