好。現在分析 **chain transfer model**。這可能是目前最有希望把多尺度問題真正落地的形式。

---

# 1. 取最近鄰尺度鏈

令 blocks 為

[  
P_{k_0},P_{k_0+1},\dots,P_K.  
]

只保留 skeleton edges：

[  
P_k\times P_{k+1}.  
]

定義 skeleton energy：

# [  
Q_{\mathrm{ch}}(a)

\sum_{k=k_0}^{K-1}Q_{k,k+1}(a),  
]

其中

# [  
Q_{k,k+1}(a)

\sum_{p\in P_k,\ q\in P_{k+1}}  
|\phi_{pq}(a)|^2.  
]

因為

[  
Q_{\mathrm{full}}(a)\ge Q_{\mathrm{ch}}(a),  
]

所以

[  
e^{-cQ_{\mathrm{full}}(a)}  
\le  
e^{-cQ_{\mathrm{ch}}(a)}.  
]

因此若能證

# [  
Z_{\mathrm{ch}}

\sum_a e^{-cQ_{\mathrm{ch}}(a)}  
\ll \frac1{\sigma},  
]

則 full graph partition bound 自動成立。

這是重要簡化。

---

# 2. 鏈式 transfer kernel

對相鄰 blocks (P_k,P_{k+1})，定義 kernel：

# [  
K_k(u,v)

\exp(-cQ_{k,k+1}(u,v)),  
]

其中

[  
u=(a_p)_{p\in P_k},  
\qquad  
v=(a_q)_{q\in P_{k+1}}.  
]

整體 partition function 是：

# [  
Z_{\mathrm{ch}}

\sum_{u_{k_0},\dots,u_K}  
\prod_{k=k_0}^{K-1}  
K_k(u_k,u_{k+1}).  
]

這是一個一維巨大 spin space 的 transfer matrix model。

我們要證它的有效自由度不是

[  
\prod_{p\in\mathcal P}p,  
]

而只有 ordinary labels (m) 的 Gaussian family：

[  
\asymp 1/\sigma.  
]

---

# 3. Kernel 的物理意義

對固定 (u\in P_k)，哪些 (v\in P_{k+1}) 讓

[  
K_k(u,v)  
]

大？

如果 (u) 近似某個 ordinary label (m)，即

[  
a_p\equiv m\pmod p  
]

對大多數 (p\in P_k)，那麼 (K_k(u,v)) 大的 (v) 也必須近似同一 (m)，否則 Irving correction 會讓

[  
Q_{k,k+1}  
]

大。

所以 transfer kernel 應該具有「label-preserving」性：

[  
m\longrightarrow m.  
]

非 label 方向會被強烈壓縮。

---

# 4. 單步 transfer lemma：想要的形式

設 (F_k(u)) 是到第 (k) 層的 partial partition function。希望歸納證：

[  
F_k(u)  
\le  
\sum_m W_k(m),\mathbf 1[u\approx m]  
+  
\text{small error},  
]

其中

[  
u\approx m  
]

表示 (P_k) 中大多數 vertices 服從 (m)，例外由能量支付。

然後

# [  
F_{k+1}(v)

\sum_u F_k(u)K_k(u,v).  
]

若 (u\approx m) 而 (v\not\approx m)，則 kernel 付 Irving energy；若 (v\approx m)，則 label (m) 傳播到下一層，權重乘上 diagonal contribution：

[  
\exp(-cQ_{k,k+1}^{\mathrm{diag}}(m)).  
]

所以

[  
W_{k+1}(m)  
\approx  
W_k(m)  
\exp(-cQ_{k,k+1}^{\mathrm{diag}}(m)).  
]

最後得到：

[  
Z_{\mathrm{ch}}  
\ll  
\sum_m  
\exp\left(  
-c\sum_k Q_{k,k+1}^{\mathrm{diag}}(m)  
\right).  
]

這就是 ordinary-label Gaussian sum。

---

# 5. 單步 lemma 的關鍵估計

固定 (m) and a labelled block (u\approx m). We need bound:

[  
\sum_{v}  
\exp(-cQ_{k,k+1}(u,v))  
\le  
C  
\exp(-c'Q_{k,k+1}^{\mathrm{diag}}(m)).  
]

up to (e^{\varepsilon R})-type constants.

這就是從一個 labelled block 到下一 block 的 transfer bound。

如果這成立 uniformly, then chain proof closes.

---

## 5.1 證明思路

令 (P=P_k), (Q=P_{k+1}). Assume (u) is exactly labelled by (m) first:

[  
a_p\equiv m\pmod p  
\quad(p\in P).  
]

對每個 (q\in Q)，其 residue (b_q\in\mathbb F_q).

若

[  
b_q\equiv m\pmod q,  
]

則 q is good and contributes diagonal energy:

[  
\sum_{p\in P}\left|\frac{m}{pq}\right|^2.  
]

若

[  
b_q\ne m\pmod q,  
]

令

[  
d_q=b_q-m\ne0.  
]

Then

[  
\phi_{pq}  
\equiv  
\frac{m}{pq}  
+  
\frac{d_q\bar p}{q}.  
]

Irving gives

[  
\sum_{p\in P}  
|\phi_{pq}|^2  
+  
\sum_{p\in P}  
\left|\frac{m}{pq}\right|^2  
\gg |P|.  
]

Thus wrong (q) costs (\gg |P|). Therefore summing over all possible (b_q):

[  
\sum_{b_q\in\mathbb F_q}  
\exp\left(  
-c\sum_{p\in P}|\phi_{pq}|^2  
\right)  
]

is bounded by

[  
\exp\left(-c'\sum_{p\in P}\left|\frac{m}{pq}\right|^2\right)  
\left(1+q e^{-c''|P|}\right).  
]

Since

[  
q e^{-c''|P|}=o(1),  
]

the contribution per q is essentially the good residue only.

Multiplying over (q\in Q):

[  
\sum_v e^{-cQ_{P,Q}(m,v)}  
\le  
\exp(-c'Q_{P,Q}^{\mathrm{diag}}(m))  
\prod_{q\in Q}(1+o(1)).  
]

Because (|Q|o(1)\to0) if pruning/goodness strong enough, or at least bounded, we get:

[  
\sum_v e^{-cQ_{P,Q}(m,v)}  
\le  
C\exp(-c'Q_{P,Q}^{\mathrm{diag}}(m)).  
]

This is excellent.

---

# 6. This is a breakthrough: transfer from labelled block is easy

The kernel factorizes over (q) once (P)-side is fixed labelled, because (Q)-side residues are independent in the sum. Each wrong (q) is killed by Irving.

So the real difficulty is only the initial condition and maintaining approximate labelledness. But transfer lemma itself strongly pulls next block to same (m).

---

# 7. Initial block problem

At (k_0), no previous labelled block. The initial sum over

[  
u_{k_0}  
]

has huge entropy. Need start the chain somewhere with a label.

Possible fix: start with two blocks and use two-sided transfer?  
Partition function of first edge:

[  
\sum_{u_0,u_1}e^{-cQ_{0,1}(u_0,u_1)}  
]

may still be large if no anchor.

But we can start by summing over ordinary labels (m) on (P_{k_0}):

Need show

[  
\sum_{u_0}  
1  
]

is bounded by labels plus defects? Not true without energy.

Thus chain transfer needs a starting anchor. In statistical mechanics one fixes boundary condition or sums over phases. Here no boundary.

We can handle by starting with a **pair** and diagonalising the first kernel: need bound

[  
\sum_{u_0,u_1}e^{-cQ_{0,1}(u_0,u_1)}  
\ll  
\sum_m e^{-cQ_{0,1}^{\mathrm{diag}}(m)}.  
]

But this is the two-layer partition bound, which we suspected may fail.

So chain transfer with one boundary condition works, but free boundary still has initial entropy.

Need eliminate initial entropy by adding a left boundary condition or using a cycle.

---

# 8. Use a cycle skeleton instead of chain?

If we make block skeleton a cycle, no free boundary. But scales are ordered; can connect top back to bottom? Are they Irving-compatible? Top and bottom scales may be far apart, not compatible. Could add bridging chain downward? Not possible if range large.

Alternatively, choose multiple chains so every block has both left and right constraints. Still endpoints remain at bottom/top. Boundary entropy of endpoints might be handled by lattice fixing or by major label sum? Need anchor one block to ordinary label.

The true system has no external boundary; global ordinary label m is the zero mode. We need sum over m, not over arbitrary initial block residues. Thus first block entropy should collapse through interactions with later blocks. Transfer from left alone doesn't see that until after summing u0. Need two-sided transfer or spectral analysis.

---

# 9. Transfer operator spectral viewpoint

The product of kernels has dominant eigenvectors indexed by ordinary labels (m). The partition function with free boundary is trace/inner product with all-ones; arbitrary initial entropy is suppressed because only label modes survive through many transfers.

So instead of induction from fixed labelled boundary, prove each kernel is a contraction onto label subspace. Then product kills orthogonal directions.

This is a spectral gap / Poincaré statement for transfer matrices.

This is promising but harder.

---

# 10. Finite-chain workaround: start in middle and propagate both directions

Choose a middle block (P_{k__}). Sum over possible residues (u_{k__}). If (u_{k__}) arbitrary, no label. But interactions to both left and right may force it to be labelled. We can condition on (u_{k__}), propagate left and right; if (u_{k_*}) not labelled, both sides kernels may suppress it. Need prove a two-sided block marginal bound:

[  
\sum_{\text{left,right}}  
e^{-Q_{\mathrm{chain}}}  
\le  
\sum_m W(m)\mathbf 1[u_{k_*}\approx m]  
+  
\text{small}.  
]

This is again local Poincaré on central block.

---

# 11. Add a small anchoring gadget?

Could deliberately include a few extra semiprime edges that force one block to ordinary label? For example lattice-fixing edges involving fixed primes? But representation problem cannot impose boundary condition on Fourier characters. Random variables cannot fix characters.

However, one could include denominators involving a fixed small prime? The phase then depends on residue modulo that prime and may anchor. But ordinary label m still free; fixed prime edges maybe reduce initial entropy. Yet denominator set must remain squarefree semiprimes; perhaps include a hub block? A complete bipartite hub to many scales could anchor. This is full graph again.

Maybe use a low-scale hub block connected to all comparable scales? Compatibility only limited range; hub cannot connect to far top.

---

# 12. Need spectral transfer theorem

Thus the clean chain route requires showing transfer matrix product has small partition despite free boundaries. This is a genuine 1D spin chain with long/large alphabet but strong ferromagnetic kernels. Standard method: Perron-Frobenius largest eigenvalue. Need bound top singular/eigenvalue of each kernel.

For kernel (K_k(u,v)), what is its operator norm from (\ell^2) to (\ell^2)? If small enough, product partition bound follows.

Maybe use Hilbert-Schmidt:

# [  
|K_k|_{HS}^2

\sum_{u,v}e^{-2cQ_{k,k+1}(u,v)}.  
]

This is exactly two-layer partition at stronger c; may be large.

But operator norm may be much smaller than HS; label modes dominate.

Need analyze kernel spectrum. Hard, but possible with group Fourier diagonalization? The state spaces are finite abelian groups:

[  
G_k=\prod_{p\in P_k}\mathbb F_p.  
]

Kernel (K_k(u,v)) depends on CRT bilinear? Actually (Q_{k,k+1}(u,v)) is sum of functions of pairs ((u_p,v_q)). This is like bipartite spin interaction. It is translation-invariant under adding ordinary label m? Not full group translation.

Maybe use Fourier transform over (G_k) to diagonalize? Kernel factors over complete bipartite interactions; not simple.

# 13. This is again deep

The labelled-boundary transfer lemma is strong and useful. Free-boundary partition needs showing the system spontaneously selects a label without boundary; this is exactly the global Poincaré theorem. No free lunch.

But transfer formulation narrows the problem to spectral gap of kernels / contraction to label subspace. This may be a useful tool target.

---

# 14. What can we salvage?

We can use transfer lemma in Peierls proof once a seed exists. So seed/no-seed remains central.

The initial entropy problem is no-seed again.

Thus no-seed theorem is unavoidable.

# 15. Next move

Given all this, perhaps formulate no-seed theorem as a **spectral gap of the transfer chain**:

Let (L_m) be label subspace functions supported near ordinary labels. Then transfer operator product satisfies

[  
\mathbf 1^\top K_{k_0}\cdots K_{K-1}\mathbf 1  
\ll  
\sum_m e^{-c m^2\sigma^2}.  
]

This is equivalent but maybe easier to compare with known Markov chain tools.

However for our current exploration, better to state the labelled-boundary transfer lemma as a proved component, and mark free-boundary/no-seed as the only unresolved part.
