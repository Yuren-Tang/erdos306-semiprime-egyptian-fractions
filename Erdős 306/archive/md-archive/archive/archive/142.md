好。現在攻 **free-boundary chain Poincaré lemma**。

鏈模型是：

[  
P_{k_0},P_{k_0+1},\dots,P_K,  
]

能量

# [  
Q_{\mathrm{ch}}(a)

\sum_{k=k_0}^{K-1}Q_{k,k+1}(a).  
]

要證：

# [  
Z_{\mathrm{ch}}

\sum_{a}  
e^{-cQ_{\mathrm{ch}}(a)}  
\ll  
\sum_{m\in\mathbb Z}  
e^{-c'Q_{\mathrm{ch}}^{\mathrm{diag}}(m)}  
\asymp  
\frac1{\sigma_{\mathrm{ch}}}.  
]

上一節已證「若某一側 block 固定為 ordinary label (m)，下一層幾乎只能跟著 (m)」。現在問題是：**自由邊界如何自發產生這個 (m)**。

---

# 1. 為何 free boundary 難？

若第一層 (P_{k_0}) 完全自由，其 residue 數量是

[  
\prod_{p\in P_{k_0}}p  
]

巨大。若沒有邊，partition function 爆炸。

邊 (P_{k_0}P_{k_0+1}) 會壓縮它，但單條 transition 未必足夠。真正的壓縮來自整條鏈：

[  
P_{k_0}\leftrightarrow P_{k_0+1}\leftrightarrow\cdots\leftrightarrow P_K.  
]

若某個初始 residue configuration 不是 ordinary label，則它很難同時與後續多層保持低能。這就是 free-boundary Poincaré。

---

# 2. 嘗試：用第二層反推第一層

兩層 partition

[  
\sum_{u_0,u_1}e^{-cQ_{0,1}(u_0,u_1)}  
]

可能偏大，因為 (u_0,u_1) 可以共同形成一個 local CRT pattern，不一定是 ordinary label。

但三層：

[  
u_0,u_1,u_2  
]

更強。若 (Q_{0,1}) 低，得到 local label (m_{01})；若 (Q_{1,2}) 低，得到 (m_{12})。兩者在 (P_1) 上合併。  
若沒有 label，至少某一 pair 付能量。

所以 free-boundary 問題至少需要三層才能開始壓縮。

---

# 3. 三層 transfer block

因此把鏈切成 overlapping triples：

[  
(P_j,P_{j+1},P_{j+2}).  
]

希望證一個三層 partition inequality：

# [  
Z_{j,j+1,j+2}

\sum_{u_j,u_{j+1},u_{j+2}}  
e^{-c(Q_{j,j+1}+Q_{j+1,j+2})}  
]

不應該像三個 blocks 的 raw entropy，而應該像 ordinary labels：

[  
\ll  
\sum_m  
e^{-c'(Q_{j,j+1}^{\mathrm{diag}}(m)+Q_{j+1,j+2}^{\mathrm{diag}}(m))}.  
]

若這個三層 inequality 成立，就可以用它作為局部 transfer block，沿鏈拼接。

但我們前面已看到：固定三層對所有 label sizes 可能仍不夠。對 free boundary，三層只能壓可見尺度內的 ordinary labels；更大尺度還需要更多層。

所以應改成 **multi-step block**。

---

# 4. 多步 block：visibility window

對能量層 (R)，需要能看見 labels up to

[  
|m|\lesssim \sqrt R/\sigma.  
]

若從最低尺度 (X_0) 開始，block (P_k) 能看見 label size roughly

[  
X_k.  
]

所以需要走到

[  
X_k\gtrsim \sqrt R/\sigma.  
]

這給一個 visibility window length

[  
k-k_0\asymp \log_2(\sqrt R/\sigma X_0^{-1}).  
]

對 major labels (1/\sigma\sim k_0X_0)，只需

[  
O(\log k_0)  
]

層。對更高 (R)，window 更長，但 Gaussian tail也更強。

所以 free-boundary chain proof可以是 energy-layer dependent：

- 對能量層 (R)，取最短 initial window (W_R) 足以看見 labels of size (\sqrt R/\sigma);
    
- 證該 window 的 partition bound已壓縮到 ordinary label (m);
    
- 之後用 labelled-boundary transfer lemma向右傳播。
    

這避免需要一開始就整條鏈 spectral analysis。

---

# 5. Initial window lemma

我們需要：

**Initial window lemma.**  
令

[  
I_R=[k_0,k_R]  
]

使

[  
X_{k_R}\gg \sqrt R/\sigma.  
]

則

[  
\sum_{(u_k)_{k\in I_R}:Q_{I_R}\le R}  
1  
\le  
e^{\varepsilon R}  
#{m:m^2\sigma^2\lesssim R}.  
]

或者 partition form：

[  
Z_{I_R}  
\le  
e^{\varepsilon R}  
\sum_m e^{-cQ_{I_R}^{\mathrm{diag}}(m)}.  
]

這個 lemma 說：在足夠長的 initial window 內，自由邊界的 raw entropy 被多尺度 agreement 壓成 ordinary label entropy。

這其實就是 no-seed theorem，但只在一個 finite window 中。比全鏈稍簡單。

---

# 6. 如何證 initial window lemma？

在 (I_R) 內，最高 block 已能看見 all relevant labels. 若整個 window 低能，則：

1. 在高端某個 pair 上，adaptive agreement產生 seed (m);
    
2. 因 (X_{k_R}\gg \sqrt R/\sigma)，label 合併可比較；
    
3. 用 Irving correction 往低端傳播；
    
4. 若某處無法傳播，付能量。
    

這是之前 anchor argument，但現在它只用於 initial window，且 (R) 固定。  
它不試圖處理所有高於 window 的 entropy，因為 window 之外會由 labelled-boundary transfer處理。

這看起來可行。

---

# 7. 高 (R) 問題如何處理？

若 (R) 很大，需要 (k_R) 很高。只要整條鏈 (K) 足夠高，對 relevant (R) 可做。  
但 (R) 可以無限到 raw entropy。對 very large (R)，partition weight (e^{-cR}) 很小，但 number of configurations huge。需要 cutoff (R_{\max}).

Initial window lemma 本身如果成立 for all (R\le R_{\max})，剩下 (R>R_{\max}) 可用 trivial bound。要讓 trivial bound可用，需要

[  
R_{\max}\gg \log|\mathcal A|.  
]

這又是 top issue。

但 maybe we don't need trivial bound: for (R) beyond top visibility, take (k_R=K), initial window is whole chain. Then initial window lemma becomes full theorem. If it holds for whole chain at all R, done. So no escape.

However, for large R, (#{m:m^2\sigma^2\lesssim R}) is also large, eventually up to (L)? Ordinary m modulo L count up to (L), but Gaussian sum with (e^{-cR}) still okay. For (R) near (\log|\mathcal A|), label count (\sqrt R/\sigma) is much smaller than raw entropy. Need theorem, not trivial.

So initial window lemma for whole chain remains core. But energy-layer visibility helps for all (R) up to top label size (X_K). For labels beyond top scale, if (m>!X_K), can still ordinary m but not visible; however diagonal energy on low edges may be huge unless wrap. Arbitrary CRT entropy again. Hmm.

This indicates again: finite chain top must itself be large enough to see relevant (m), or theorem must handle beyond-top via different argument.

---

# 8. Maybe choose (K) so top scale exceeds (L)? impossible.

Since (L) is product of primes up to (K), top scale never exceeds possible CRT labels.  
So visibility-by-scale cannot cover all ordinary representatives.

But ordinary labels beyond (X_K) are not needed individually: modulo L, any (m) has representative up to (L/2). There are huge. To show they high energy is exactly aperiodicity.

Thus top issue is unavoidable even for chain.

---

# 9. What has the transfer model gained?

It gave a very solid lemma: labelled boundary propagates.  
But free-boundary still equivalent to proving existence of a seed or aperiodicity.  
So not full solution, but valuable component.

At this point, the true unresolved theorem is:

**Seed generation theorem.**  
In a free chain, configurations with no ordinary seed have exponentially small total weight.

This is the same no-seed theorem, now in a chain.

This seems the cleanest final bottleneck.

---

# 10. Can seed generation be attacked by two-sided transfer?

Maybe yes. Consider a long window. Even if no seed at top, arbitrary CRT residues create local phases. But if no ordinary seed anywhere, each transition must carry significant energy. Maybe total energy grows linearly with window length. Entropy also grows with blocks. Need compare constants.

For random residues, per edge-block energy (\asymp |P||Q|), enormous; entropy per block (\asymp |P|k). Since (|P||Q|\gg |P|k), random no-seed is killed. The dangerous configurations are structured to keep transition energies low without ordinary seed. Agreement theorem should rule them out.

Thus seed generation theorem is plausible.

---

# 11. Final current state

We have:

- labelled-boundary transfer lemma: solid.
    
- free-boundary seed generation theorem: central unresolved.
    
- agreement testing seems the right external tool.
    

Next step should be to state seed generation theorem for a chain and attempt proof by contradiction:

Assume a configuration on a long chain has low energy but no ordinary seed.  
Then each adjacent pair either gives local label or has high energy.  
If local labels exist along a path, they merge to ordinary seed.  
If labels break often, break edges pay energy.  
If no local labels over long segment, many high-energy edges pay entropy.

This is now clear. Let's write it.
