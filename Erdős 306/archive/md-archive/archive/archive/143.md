好。現在寫 **chain seed generation theorem**。這是 free-boundary chain 的核心：沒有外部固定 label 時，鏈自己必須產生一個 ordinary seed，否則要付出足夠能量。

---

# Chain seed generation theorem

考慮最近鄰 block chain：

[  
P_0,P_1,\dots,P_N.  
]

邊只取

[  
P_i\times P_{i+1}.  
]

能量：

# [  
Q_{\mathrm{ch}}(a)

\sum_{i=0}^{N-1}Q_i(a),  
]

其中

# [  
Q_i(a)

\sum_{p\in P_i,q\in P_{i+1}}  
|\phi_{pq}(a)|^2.  
]

目標是證明：

> 若 (Q_{\mathrm{ch}}(a)\le R)，則除非 (a) 屬於能量可支付的 no-seed 類，否則存在 ordinary label (m)，使一個長 block interval 中大多數 vertices 滿足  
> [  
> a_p\equiv m\pmod p.  
> ]  
> 一旦 seed 存在，labelled-boundary transfer lemma 會把它向兩側傳播。

---

## 1. 每條鏈邊的二分

對每條相鄰 block edge

[  
i=(P_i,P_{i+1}),  
]

取一個 cutoff (B_i)。如果：

1. (B_i) 在該尺度可見；
    
2. 至少 (1-\eta) 的 prime edges 有  
    [  
    |H_{pq}^{*}|\le B_i;  
    ]
    

則稱 (i) 為 **label edge**。由 local agreement 得到 ordinary label

[  
m_i,\qquad |m_i|\le B_i,  
]

解釋大多數

[  
P_i\cup P_{i+1}.  
]

否則稱 (i) 為 **break edge**。break edge 表示：

- either pass density低，故大量 edge glues 大；
    
- or local energy high；
    
- or cutoff 不可見，需要在更高尺度處理。
    

在 chain seed theorem 裏，我們先只處理可見 cutoff；不可見情況透過選擇足夠高的 window 解決。

---

## 2. label edges 連成 interval 時，labels 合併

若兩條相鄰 label edges：

[  
(P_{i-1},P_i),\qquad(P_i,P_{i+1})  
]

給出 labels

[  
m_{i-1},\qquad m_i,  
]

則它們都解釋 (P_i) 的大多數 primes。

所以對很多 (p\in P_i),

[  
m_{i-1}\equiv m_i\pmod p.  
]

若

[  
m_{i-1}\ne m_i,  
]

則

[  
m_{i-1}-m_i  
]

被很多 primes (p\in P_i) 整除。取足夠多共同 primes，其乘積超過

[  
|m_{i-1}-m_i|,  
]

即得

[  
m_{i-1}=m_i.  
]

在能量層 (R) 中，只需比較 labels up to size

[  
\lesssim \sqrt R/\sigma.  
]

只要 (P_i) 的 prime product足夠大，這可保證。對不夠高的 (P_i)，可等到較高 visibility window 再做 seed。

因此一段連續 label edges 產生一個共同 ordinary label：

[  
m_I.  
]

---

## 3. break edges 的能量成本

若 edge (i) 不是 label edge，在可見 cutoff (B_i) 下 pass density不足，則至少 (\eta |P_i||P_{i+1}|) 條 prime edges 滿足

[  
|H_{pq}^{*}|>B_i.  
]

因此

[  
Q_i(a)  
\gg  
\eta |P_i||P_{i+1}|  
\left(\frac{B_i}{X_iX_{i+1}}\right)^2.  
]

若選 (B_i) 接近該尺度最大可見 cutoff，得到

[  
Q_i(a)\gg V_i,  
]

其中

[  
V_i  
\asymp  
|P_i||P_{i+1}|  
\left(\frac{B_{i,\max}}{X_iX_{i+1}}\right)^2.  
]

若 (X_i\sim X_{i+1}\sim X)，且 (B_{\max}\asymp X)，則

[  
V_i\asymp  
\frac{|P_i||P_{i+1}|}{X^2}  
\asymp  
\frac1{(\log X)^2}.  
]

這太小。若用 comparable dyadic blocks，最大可見 (B\asymp X)，單 break edge成本只有 log 級以下常數，甚至趨零。這比之前 (4/3) 尺度還弱。

這說明：nearest-neighbour comparable blocks對 small-glue cutoff 能量太弱。

---

## 4. 為什麼 Irving correction仍強？

如果已有 label (m)，wrong vertex 成本是

[  
\gg |P_i|.  
]

但沒有 label時，單純 pass density failure在 (B\asymp X) 只給很弱能量，因為相位門檻是

[  
B/(X^2)\asymp1/X.  
]

所以 seed generation 不能靠最大可見 cutoff failure在 comparable scales。  
這是重要問題。

使用比例 (Y=X^{4/3}) 時，最大可見 (B\asymp Y)，成本

[  
V_i\asymp X^{1/3}/(\log X)^2,  
]

較強，但仍不如 raw entropy；不過比 comparable scales好。

因此 nearest-neighbour skeleton對 no-seed不夠強。需保留尺度跳躍 (X\to X^{4/3}) 或更高比率來獲得可見 cutoff能量。

---

# 5. 回到 (4/3)-jump chain

選 chain：

[  
X_{i+1}=X_i^{4/3}.  
]

則 break edge成本

[  
V_i\asymp X_i^{1/3}/(\log X_i)^2.  
]

這隨尺度增長，能為 no-seed 提供越來越大成本。

Labelled-boundary transfer仍成立，因 Irving正好適用。

所以 skeleton應是 (4/3)-jump chain，而不是 nearest-neighbour chain。  
但為覆蓋所有 dyadic blocks，需要多條 offset chains；或者只在這些 (4/3)-scale blocks上建控制 skeleton，其他 blocks由相鄰控制。這是技術設計問題。

---

## 6. Seed generation on (4/3)-chain

在 (4/3)-chain中：

- 若有長段 label edges，labels合併成 seed (m)；
    
- 若沒有長段，break edges多；
    
- 每個高尺度 break edge成本 (V_i) 增長；
    
- 所以 no-seed configurations必有總能量至少約最高 break scale。
    

對能量層 (R)，選 chain長度到 (V_i\gg R)。若 (Q\le R)，則不可能所有高尺度 edges都是break；必有 label edge。連續 label edges再合併成 seed。

這是 anchor/window argument，但在 chain內自洽。

---

## 7. Still the top issue

若能量 (R) 超過最高 (V_K)，chain不能保證 seed。但在 partition sum中，這類高能仍需控制。  
這正是全局 no-seed高頻問題。

不過現在若用多條 (4/3)-chains覆蓋到 very high (K)，可以讓 (V_K)非常大。仍不可能超過 raw entropy，但 perhaps high (R) with no seed has more break edges and energy near raw entropy; our lower bound is weak but maybe enough with refined layer-cake.

At this stage, seed theorem for all (R) still hard. But for the significant range (R\lesssim V_K), it works.

---

# 8. 這一輪給出的重要修正

Nearest-neighbour chain對 labelled transfer很好，但對 seed generation弱。  
(4/3)-jump chain對 seed generation更好，因為 visible cutoff成本有 power saving。

所以 skeleton應該混合：

- (4/3)-jump edges用於產生 seed / no-seed energy；
    
- comparable/nearest edges可輔助覆蓋與傳播；
    
- full graph最終包含更多邊。
    

This is a sensible architecture.

---

# 9. Chain seed theorem：修正版

**Theorem, energy-limited version.**  
On a (4/3)-jump chain up to scale (X_J), if

[  
R\ll V_J\asymp X_J^{1/4}/(\log X_J)^2  
]

（具體指數依 indexing），then every (a) with

[  
Q_{\mathrm{chain}}(a)\le R  
]

has an ordinary seed label (m) with

[  
m^2\sigma^2\lesssim R  
]

on some high-scale block pair. This seed propagates along the chain, and exceptions are energy-paid.

Thus for (R\le V_J),

[  
#{a:Q\le R}  
\le  
e^{\varepsilon R}  
#{m:m^2\sigma^2\lesssim R}.  
]

This is a partial but real result.

---

# 10. How to handle (R>V_J)?

Three options:

1. Take (J) so high that (V_J) exceeds all (R) contributing non-negligibly. We saw raw entropy prevents trivial, but maybe with partial counting it suffices.
    
2. Develop stronger no-seed energy lower bound using multi-threshold beyond visible cutoff.
    
3. Use external weighted LCLT/agreement theorem.
    

This remains the unresolved frontier.
