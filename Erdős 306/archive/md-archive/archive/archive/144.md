好。現在看剛才留下的高能尾部：

[  
R>V_J  
]

是否能不用完整 no-seed theorem，而用 (L^2)、Parseval 或碰撞概率來處理。

---

# 1. Minor arc 原本需要 (L^1) 控制

Fourier inversion中，誤差通常用

[  
\sum_{\chi\notin\mathfrak M}|\widehat\mu(\chi)|  
]

控制。若

[  
|\widehat\mu(\chi)|\le e^{-cQ(\chi)},  
]

就需要

[  
\sum_{\chi\notin\mathfrak M}e^{-cQ(\chi)}  
=o(1/\sigma).  
]

這是 (L^1) partition bound。

對高能 (Q(\chi)>V_J)，粗略有

[  
e^{-cQ(\chi)}\le e^{-cV_J},  
]

但 character 數量巨大：

[  
|\widehat G|=\prod_{p\in\mathcal P}p,  
]

所以 raw bound

[  
|\widehat G|e^{-cV_J}  
]

完全不夠。這是反覆出現的 top entropy 問題。

---

# 2. Parseval / (L^2) 能否幫忙？

Parseval 給

# [  
\sum_{\chi}|\widehat\mu(\chi)|^2

|G|\sum_x \mu(x)^2.  
]

這等於碰撞概率乘以群大小。

若分布已經很平滑，這可能很小。但我們正在證 local CLT，不能先假設。

用 Cauchy–Schwarz：

[  
\sum_{\chi\in T}|\widehat\mu(\chi)|  
\le  
|T|^{1/2}  
\left(\sum_{\chi\in T}|\widehat\mu(\chi)|^2\right)^{1/2}.  
]

若 (T) 很大，(|T|^{1/2}) 仍巨大。除非 (L^2) 極小，否則不夠。

而

[  
\sum_{\chi}|\widehat\mu(\chi)|^2  
]

本身通常約

[  
|G|\cdot \frac1{\sigma L}  
]

之類，未必能壓過 (|T|^{1/2})。更本質地，(L^2) 控制平均平方，不能排除一大片中等大小 Fourier coefficients 對 (L^1) 的貢獻。

所以單純 Parseval 不足。

---

# 3. 高能 tail 若有 pointwise (e^{-cV_J})，用 (L^2) 插值？

可以用

[  
\sum_T |\hat\mu|  
\le  
(\sup_T|\hat\mu|)^{1/2}  
\sum_T |\hat\mu|^{1/2}  
]

之類插值，但仍需要某種 counting / entropy bound。  
若只知道

[  
\sup_T|\hat\mu|\le e^{-cV_J},  
]

則最直接仍要乘 character 數量。除非 (V_J\gg\log|G|)，而這不成立。

因此高能 tail 不能靠簡單 (L^2) 或 sup bound 關掉。

---

# 4. 可否用 smoothing / 多次卷積？

有時 local CLT 證明會把分布分成兩部分：

[  
\mu=\mu_1*\mu_2.  
]

用 (\mu_1) 給主弧 Gaussian，用 (\mu_2) 作 smoothing，使高頻消失。  
這相當於讓一部分 denominators 提供強 aperiodicity。

在我們情形中，可以考慮：

- 低尺度 semiprimes 提供方差與主弧；
    
- 高尺度 semiprimes 作 smoothing / anti-concentration。
    

若高尺度 smoothing set 的 Fourier transform 在所有非主弧頻率上很小，則高能 tail可由它單獨壓住。

這可能是一條新路線。

---

# 5. Smoothing block idea

選一組專門的 smoothing edges (E_{\mathrm{sm}})，與 mass-carrying edges 分開。  
要求：

[  
\forall \chi\notin\mathfrak M,  
\qquad  
|\widehat\mu_{\mathrm{sm}}(\chi)|  
\le  
\exp(-A)  
]

其中

[  
A\gg \log(1/\sigma)  
]

甚至足以支配剩餘 Fourier (L^1)。

如果 smoothing set 能做到這點，而其 reciprocal mass很小，不影響期望太多，則原證明大幅簡化。

這是常見做法：用一部分隨機變量平滑格點分布。

問題：要對所有非主弧 CRT characters 平滑，仍是同一 aperiodicity問題，但或許可設計 smoothing edges 更密、更有結構，例如使用 complete bipartite blocks at carefully chosen scales.

---

# 6. 能否用一個 smoothing tower？

如果 smoothing tower 專門設計得很高、很密，它可以檢測 arbitrary CRT residues。  
但加入 smoothing primes也增加 group (L) 和 character entropy。若 smoothing variables使用的 primes也進入 denominator，確實增加 entropy。

不過 smoothing目的不是靠 raw entropy，而是提供強 pointwise decay。若每個新 smoothing block同時引入 entropy和更多 smoothing，可能仍可 balance。這又變成 Poincaré，但也許更可控，因為 smoothing blocks可選成 expander-like complete graph。

---

# 7. Alternative: use complete graph within one scale?

Earlier semiprimes (pq) with both primes in same large block (P). Consider complete graph on (P) itself. For arbitrary residues (a_p), energy

[  
\sum_{p<q\in P}|\phi_{pq}|^2.  
]

Could this single-scale complete graph have a strong Poincaré theorem to ordinary labels? Visibility cutoff limited by (p), but labels relevant at that scale. It may kill arbitrary residues within that block very strongly.  
If every scale block has an internal complete graph, then each block independently collapses to an ordinary label (m_k). Then cross-scale edges only need merge (m_k)'s.

This is promising.

Previously we focused on bipartite between scales. But include same-scale semiprimes (p q) with (p,q\in P_k). They are valid distinct primes. Internal complete graph may provide much stronger local agreement and remove block raw entropy.

Let's test.

# 8. Same-scale complete block local Poincaré

For one block (P\sim X), energy:

[  
Q_P(a)=\sum_{p<q\in P}|\phi_{pq}(a)|^2.  
]

Question:

[  
Z_P=\sum_{a_p\in\mathbb F_p}e^{-cQ_P(a)}  
\stackrel{?}{\ll}  
#{m: |m|\lesssim X}  
]

or roughly (X), not (\prod p).

If (a) is not close to ordinary (m) visible at scale (X), dense agreement on complete graph should give huge energy.  
Because complete graph within one block has many triangles / overlaps. This is stronger than bipartite pair. It may prove local Poincaré for labels up to size (X).

Then arbitrary residues inside each block collapse to a block label (m_k) with (|m_k|\lesssim X_k) or pay huge energy. This kills raw entropy locally.

This is exactly what we need! Large labels beyond (X_k) will be represented at higher blocks, but per block labels are ordinary within scale.

# 9. Why same-scale helps

In a bipartite (P\times Q), a whole (Q)-side cluster can carry local CRT entropy if not anchored.  
In a complete graph (P) itself, every pair of vertices tests small ordinary glue. If many tests pass, all vertices share an ordinary label. If not, many pair tests fail. Since there are (|P|^2) edges, disorder energy can be (|P|^2), enough to pay block entropy (\sim X).

At maximum visible (B\sim X), energy per failed edge threshold:

[  
(B/X^2)^2\sim 1/X^2.  
]

Number of edges (|P|^2\sim X^2/\log^2 X). Cost (\sim1/\log^2 X), still weak if only just above cutoff. But Irving/two-phase for wrong residues against a labelled majority gives (|P|) per wrong vertex, strong. Need seed; complete graph may produce seed by graph agreement.

If no seed, random disorder gives constant per edge. But cutoff lower bound weak. Yet complete graph agreement may yield stronger Peierls: multiple labels create cross energy via Irving? For same-scale, difference term (D\bar p/q) with p,q same scale still Irving-compatible. Boundary between two label clusters has cost (|A||B|), huge. Good.

Thus same-scale block has a low-temperature Potts model whose phases are ordinary labels visible at that scale; contour cost pays entropy. This is more plausible local theorem.

5. Global partition reduces to ordinary label chain (m_k), with ferromagnetic interactions forcing (m_k) equal or paying energy. This is much lower-dimensional.
    

This is a significant improvement.

# 11. Is local block partition bound true?

For a complete graph on primes (P\sim X), ordinary labels (m) visible up to (X). But there are labels (m) beyond X with same residues? If (|m|>X), within block they may not be visible; however CRT residues arbitrary could correspond to a large (m). Local block energy only sees (m\bmod pq); if (m) large but (m\bmod pq) small for many pairs, then it behaves like small label on those vertices. Otherwise energy large. The local partition bound should reduce to (O(X)) visible labels plus defects, not all CRT residues.

This is still a nontrivial local Poincaré, but finite single block complete graph may be provable by Peierls because phase space of ordinary visible labels is only (O(X)), and arbitrary high residues create defects/contours.

Maybe easier than global multiscale.

# 12. Need check semiprime denominators within same block

Original construction allowed compatible pairs; same-scale (p,q\in P_k) are squarefree semiprimes and reciprocal mass contribution per block:

[  
\sum_{p<q\in P_k}\frac1{pq}  
\asymp \frac12\left(\sum_{p\in P_k}\frac1p\right)^2  
\asymp \frac1{k^2}.

Summing over k gives finite (\sum1/k^2), not enough mass alone, but they can be added as control edges. Cross-scale edges still provide mass (\sum 1/(k\ell)) over compatible range giving log growth. Adding same-scale edges is allowed and helpful.

They also contribute to variance; fine.

# 13. Local block theorem as new target

**Single-block CRT Poincaré theorem.**  
For (P\sim X),

[  
Q_P(a)=\sum_{p<q\in P}|\phi_{pq}(a)|^2.  
]

Then

[  
Z_P=\sum_{a\in\prod_{p\in P}\mathbb F_p}e^{-cQ_P(a)}  
\ll X  
]

or more precisely

[  
\ll 1/\sigma_P  
]

where

[  
\sigma_P^2=\sum_{p<q}\frac1{p^2q^2}\asymp \frac1{X^2(\log X)^2?}  
]

Compute:

[  
\sum_{p\in P}1/p^2\asymp |P|/X^2\asymp 1/(X\log X).  
]

For pairs,

[  
\sigma_P^2\asymp \left(\sum 1/p^2\right)^2\asymp \frac1{X^2\log^2 X}.  
]

So

[  
1/\sigma_P\asymp X\log X.  
]

Visible labels up to (X) count (X), with log factor acceptable. Good.

If true, each block raw entropy collapses to (X\log X), not (\exp(X)).

This is huge.

# 14. Can we prove single-block theorem with Irving?

Take a candidate label m from majority. Wrong vertex q costs

[  
\sum_{p\in P}|(a_q-m)\bar p/q + m/(pq)|^2\gg |P|  
]

using same-scale Irving. Since (|P|\gg\log q), defects paid.

If two clusters labels m,m', boundary between them costs (|A||B|). This pays cluster entropy.

No top scale issue inside block because phase labels only need be visible up to (X); high CRT residues without visible label behave as disorder and should create many cross energies. Complete graph gives enough boundary.

This is plausible.

# 15. Great: reduce global problem

If every block is locally collapsed to a label (m_k), then global partition roughly:

[  
\sum_{(m_k)}  
\exp\left(  
-c\sum_k Q_{P_k}^{\mathrm{diag}}(m_k)  
-c\sum_{k\sim\ell}Q_{k\ell}^{\mathrm{cross}}(m_k,m_\ell)  
\right).  
]

Cross energy between block labels (m_k,m_\ell) forces equality unless paid. This is now an ordinary integer spin chain, not huge CRT alphabet. Much easier.

Then Peierls over integer labels should collapse all (m_k) to one global (m), with Gaussian energy.

This may be the missing structural move.
