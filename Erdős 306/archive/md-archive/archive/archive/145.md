好。現在攻 **single-block CRT Poincaré theorem**。這一步若成立，整個問題會大幅降維：每個 prime block 先自己塌縮成一個 ordinary label，然後全局只剩 block labels 之間的合併問題。

# 1. 單 block 模型

取一個 prime block

[  
P\subset[X,2X].  
]

在同一 block 內取所有 semiprime edges：

[  
{pq:p,q\in P,\ p<q}.  
]

對 residue assignment

[  
a=(a_p)_{p\in P},\qquad a_p\in\mathbb F_p,  
]

定義同尺度能量

# [  
Q_P(a)

\sum_{\substack{p,q\in P\p<q}}  
|\phi_{pq}(a)|^2.  
]

其中

# [  
\phi_{pq}(a)

\frac{a_p\bar q}{p}  
+  
\frac{a_q\bar p}{q}  
\pmod1.  
]

目標是證：

[  
Z_P  
:=  
\sum_{a\in\prod_{p\in P}\mathbb F_p}  
e^{-cQ_P(a)}  
\ll  
\frac1{\sigma_P},  
]

其中

# [  
\sigma_P^2

\sum_{p<q}\frac1{p^2q^2}  
\asymp  
\frac1{X^2(\log X)^2}.  
]

所以

[  
\frac1{\sigma_P}\asymp X\log X.  
]

直觀上，(Z_P) 應該只有 ordinary labels

[  
m,\qquad |m|\lesssim X\log X  
]

的高斯量級，而不是

[  
\prod_{p\in P}p  
]

的巨大 CRT 熵。

---

# 2. 為何 single-block 可能成立？

同一 block 是 complete graph。若 residues 不服從同一 ordinary label，則有大量 cross pairs。

如果一大群 vertices 服從 label (m)，另一群服從 (m'\ne m)，跨邊相位為

[  
\frac{m}{pq}  
+  
\frac{(m'-m)\bar p}{q}  
\pmod1.  
]

對固定 (q)，只要

[  
m'-m\not\equiv0\pmod q,  
]

Irving dispersion 給

[  
\sum_{p}  
\left|  
\frac{(m'-m)\bar p}{q}  
\right|^2  
\gg |P|.  
]

所以兩相邊界成本約

[  
\gg |A||B|.  
]

這非常大，足以支付 cluster entropy。

若一個 vertex 不服從 majority label (m)，它與 majority set 的 incident edges 也給

[  
\gg |A|  
]

成本，足以支付該 vertex residue entropy。

因此 single-block complete graph 應該有強 Peierls 性。

# 3. Exact zero-energy check

若

[  
Q_P(a)=0,  
]

則對每條 (pq),

[  
\phi_{pq}=0.  
]

即 CRT glue 為 (0)，所以

[  
a_p=0,\qquad a_q=0.  
]

因 complete graph 連通，所有

[  
a_p=0.  
]

若考慮非零 ordinary label (m)，其 energy 是

# [  
Q_P^{\mathrm{diag}}(m)

\sum_{p<q}  
\left|\frac{m}{pq}\right|^2.  
]

小 (m) 有小但非零 energy。  
所以 ground state 是 (m=0)，低能態是 small ordinary (m)。

這與期望一致。

# 4. 單 block partition theorem 的更精確形式

我們希望證：

[  
#{a:Q_P(a)\le R}  
\le  
e^{\varepsilon R}  
#{m\in\mathbb Z:  
Q_P^{\mathrm{diag}}(m)\le C R}.  
]

因為

[  
Q_P^{\mathrm{diag}}(m)  
\asymp  
m^2\sigma_P^2  
]

在可見範圍內，所以右側約為

[  
e^{\varepsilon R}\frac{\sqrt R}{\sigma_P}.  
]

然後

[  
Z_P  
\le  
\sum_R e^{-cR}  
e^{\varepsilon R}  
\frac{\sqrt R}{\sigma_P}  
\ll  
\frac1{\sigma_P}.  
]

這是最理想的 local Poincaré counting。

# 5. 如何證：majority-label encoding

對一個低能 assignment (a)，嘗試找 ordinary label (m)。

取 cutoff

# [  
B_R

X^2\sqrt{\frac{R}{|P|^2}}  
\asymp  
X^2\frac{\sqrt R}{|P|}  
\asymp  
X\log X\sqrt R.  
]

如果

[  
Q_P(a)\le R,  
]

則由 Markov，多數 edges 有

[  
|H_{pq}^{*}|\le B_R.  
]

若

[  
B_R<X/2,  
]

即

[  
X\log X\sqrt R<X,  
]

也就是

[  
R\ll \frac1{\log^2 X},  
]

這只適用於極小 (R)，太弱。  
所以單純 dense agreement at adaptive cutoff 在同尺度 block 中不夠。

但 partition 主要 (R) 不是這麼小；(R\sim1) 已經重要。此時 (B_R\sim X\log X)，超過 visible (X).

所以 single-block 不能靠「多數 edge glue 小於可見 (B)」直接得到 label。

---

# 6. 這是問題嗎？

不一定。對 ordinary label (m\sim X\log X)，它本來就超過 prime size (X)，但 diagonal energy仍約 (1):

[  
m^2\sigma_P^2  
\sim  
(X^2\log^2X)\cdot\frac1{X^2\log^2X}  
\sim1.  
]

所以 local block partition 的 natural label range 是

[  
|m|\lesssim X\log X,  
]

大於單個 prime (p\sim X)。一個 block 內無法用單個 prime直接唯一識別這麼大的 (m)。

但用多個 primes 的乘積可以識別。若 (m) 與 (m') 在兩個 primes 上同餘，且

[  
p_1p_2>|m-m'|,  
]

則相等。因

[  
p_1p_2\sim X^2  
]

大於

[  
X\log X  
]

，所以兩三個 primes足夠。

因此 single-block agreement 應使用 **multi-overlap within complete graph**，不是單個高點 visible condition。

這是好消息。

# 7. Improved single-block agreement

若多數 edges (pq) have CRT glue near some label scale (B\le X^2), then edge labels can be compared via shared vertices and products of primes.

在 complete graph 中，edge (pq) 和 (pr) share (p)，只給 mod (p) comparison；不夠時，沿 triangles / multiple shared vertices 可用 product of several primes。

具體：若 edge glues (H_{pq},H_{pr}) both bounded by (B), then

[  
H_{pq}\equiv H_{pr}\pmod p.  
]

若 (B<X/2)，立刻相等；若 (B<X^2/2)，一個 prime不夠。但如果再比較 across many shared vertices，差值被多個 primes整除，乘積超過 (B)，則相等。

Complete graph 高連通，能提供許多 shared primes。  
所以 label visibility in a block is not (X), but roughly product of a few primes, potentially (X^t). For (B\sim X\log X), two primes suffice.

Thus single-block theorem may indeed hold.

---

# 8. General visibility by (r)-connectivity

For label scale (B), need

[  
X^r>B  
]

for some small integer (r). In complete graph, if pass edge graph has high density, any two edge labels are connected through many overlaps; along a path, differences divisible by products of shared vertices. With (r) independent overlaps, product (X^r).

For (B\le X\log X), (r=2) suffices.  
For energy layer (R), label scale

[  
B_R\sim X\log X\sqrt R.  
]

Need

[  
X^r> B_R.  
]

For (R) up to huge, (r) grows slowly. Complete graph can provide arbitrary finite (r) as long as pass graph dense enough.

This is a strong mechanism absent from bipartite two-layer.

# 9. Single-block multioverlap agreement lemma

**Lemma.**  
Let (P\sim X). Suppose a dense set of edges (E_{\mathrm{pass}}\subset\binom P2) has CRT glues

[  
|H_{pq}^{*}|\le B.  
]

If the pass graph has enough common-neighbour expansion so that edge labels are connected by overlap chains whose shared-prime products exceed (B), then there exists ordinary (m), (|m|\le B), explaining most vertices.

For complete graph with pass density (1-\eta), this holds for any

[  
B\le \exp(c\eta |P|\log X)  
]

essentially enormous, because one can find many shared vertices. In particular it covers all relevant (B) up to (L_P^{c}).

This is the missing visibility upgrade.

Excellent.

---

# 10. Consequence

Single-block local agreement can handle label scales much larger than (X), not just (X), because complete graph gives many overlaps. Therefore adaptive cutoff (B_R\sim X\log X\sqrt R) is visible for all (R) up to extremely large.

Thus single-block partition theorem becomes plausible.

---

# 11. Proof sketch of multioverlap agreement

Take two regular vertices (p,p'). For many common neighbours (q), edges (pq) and (p'q) pass. Their glues (H_{pq},H_{p'q}) satisfy:

[  
H_{pq}\equiv a_q\equiv H_{p'q}\pmod q.  
]

So

[  
q\mid H_{pq}-H_{p'q}.  
]

If this holds for many (q), and (|H_{pq}|,|H_{p'q}|\le B), product of a few such (q)'s exceeds (2B), forcing

[  
H_{pq}=H_{p'q}.  
]

Thus all regular vertices share one label (m). Then vertex residues follow.

This is simple! We only need multiple common neighbours.

Dense pass graph ensures many common neighbours.

# 12. Great: single-block agreement is strong

This removes the earlier visibility barrier.  
For complete graph, local labels can be as large as product of many primes, because common-neighbour products compare them.

Therefore single-block Poincaré may be provable fully.
