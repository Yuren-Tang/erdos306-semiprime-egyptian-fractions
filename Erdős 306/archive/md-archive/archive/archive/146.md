是，總體徵兆現在比前面好。前面反覆卡住，是因為我們一直只靠「一條邊」或「一個二部 block pair」去看 label；那時可見尺度最多大約是一個 prime 或一個高層 prime，所以大 label／高頻總是逃走。

現在的新點是：

[  
\boxed{  
\text{同一 prime block 內的 complete graph 有大量三角形與共同鄰點。}  
}  
]

這讓我們比較兩條 edge glue 時，不只得到「差值被一個 prime 整除」，而是能讓差值被許多共同鄰點 primes 整除。於是可見尺度從

[  
X  
]

提升到

[  
X^r  
]

乃至

[  
\exp(c|P|\log X),  
]

這正好打到此前反覆逃逸的高頻／大 label 問題。這不是枝節修補，而是可能改變局面的機制。

現在繼續證這個核心局部 lemma。

---

# Large-(B) single-block dense agreement lemma

設

[  
P\subset[X,2X]  
]

是一個 prime block，(|P|=N)。給每個

[  
p\in P  
]

一個 residue

[  
a_p\in\mathbb F_p.  
]

對邊 (pq)，令

[  
H_{pq}^{*}\in(-pq/2,pq/2]  
]

為 CRT glue：

[  
H_{pq}^{_}\equiv a_p\pmod p,\qquad  
H_{pq}^{_}\equiv a_q\pmod q.  
]

假設有一個邊集合

[  
E_{\mathrm{good}}\subset \binom P2  
]

密度至少

[  
1-\eta,  
]

且對每條 good edge，

[  
|H_{pq}^{*}|\le B.  
]

若

[  
B<X^r/10  
]

而 good graph 有足夠多長度 (r) 的共同鄰點鏈，則存在 ordinary integer

[  
m,\qquad |m|\le B,  
]

使得除了 (O_r(\eta)N) 個 vertices 外，

[  
a_p\equiv m\pmod p.  
]

對 complete graph 的高密度子圖，只要 (r) 固定且 (\eta) 足夠小，這個共同鄰點條件成立。更強地，可以取

[  
r\asymp cN  
]

於是可見範圍達到

[  
B\le X^{cN}.  
]

---

## 1. regular vertices

稱 (p\in P) 為 regular，若它在 good graph 中的 degree 至少

[  
(1-\sqrt\eta)N.  
]

由邊密度 (1-\eta)，非 regular vertices 至多

[  
O(\sqrt\eta)N.  
]

以下只看 regular vertices。

---

## 2. 比較兩個 regular vertices

取兩個 regular vertices

[  
p,p'\in P.  
]

它們的共同 good neighbours 至少

[  
(1-2\sqrt\eta)N.  
]

任取共同 good neighbour (q)。則邊 (pq) 和 (p'q) 都 good，所以有兩個 bounded glues：

[  
H_{pq}^{_},\qquad H_{p'q}^{_},  
]

且

[  
|H_{pq}^{_}|,\ |H_{p'q}^{_}|\le B.  
]

因為二者在 (q) 端都等於

[  
a_q\pmod q,  
]

所以

[  
H_{pq}^{_}\equiv H_{p'q}^{_}\pmod q.  
]

即

[  
q\mid H_{pq}^{_}-H_{p'q}^{_}.  
]

若我們能找到 (r) 個共同 good neighbours

[  
q_1,\dots,q_r  
]

使同一對 glues 可比較，則其差值被

[  
q_1\cdots q_r  
]

整除。由於每個 (q_i\sim X)，若

[  
X^r>2B,  
]

則

[  
H_{pq}^{_}=H_{p'q}^{_}.  
]

不過這裏有一個小心點：(H_{pq}^{*}) 依賴於 (q)，不是固定的 (H_p)。所以不能直接用不同 (q_i) 比較同一個差值。需要稍微改寫。

---

## 3. 正確比較：固定一個 base vertex

選一個 regular base vertex

[  
p_0.  
]

對每個 regular (p)，若 (p_0p) 是 good edge，定義

[  
m_p:=H_{p_0p}^{*}.  
]

它滿足

[  
m_p\equiv a_{p_0}\pmod{p_0},  
\qquad  
m_p\equiv a_p\pmod p,  
]

且

[  
|m_p|\le B.  
]

我們要證大多數 (m_p) 相等。

取兩個 regular vertices (p,p')，且 (p_0p,p_0p') good。  
找共同 good neighbours (q) 同時與 (p,p') 連 good edges。對三角形 (p_0,p,q)，有：

[  
m_p=H_{p_0p}^{*},  
]

而

[  
H_{pq}^{*}  
]

與 (m_p) 在 (p) 端同餘：

[  
H_{pq}^{*}\equiv a_p\equiv m_p\pmod p.  
]

同時 (H_{pq}^{_}) 與 (H_{p_0q}^{_}=m_q) 在 (q) 端同餘：

[  
H_{pq}^{*}\equiv a_q\equiv m_q\pmod q.  
]

這仍只給局部關係。更直接的方法是用 good triangle (p_0,p,q)：三條邊 glues

[  
H_{p_0p},\ H_{p_0q},\ H_{pq}  
]

都 bounded by (B)，且：

[  
H_{p_0p}\equiv H_{p_0q}\pmod{p_0},  
]

[  
H_{p_0p}\equiv H_{pq}\pmod p,  
]

[  
H_{p_0q}\equiv H_{pq}\pmod q.  
]

這是一個 bounded CRT 2-cocycle。若 (B) 小於產品 (p_0pq) 的足夠部分，三者應被迫等於同一 (m). 但一個三角形只給三個 congruences，產品約 (X^3)。所以若

[  
B<X^3  
]

，一個 good triangle already forces equality? Let's check.

Let

[  
A=H_{p_0p},\quad B_1=H_{p_0q},\quad C=H_{pq}.  
]

We know:

[  
A-B_1\equiv0\pmod{p_0},  
]

[  
A-C\equiv0\pmod p,  
]

[  
B_1-C\equiv0\pmod q.  
]

This does not imply (A=B_1=C) merely by product divisibility, because each difference has only one prime divisor. Need combine them:

[  
(A-B_1)+(B_1-C)+(C-A)=0.  
]

Let

[  
A-B_1=p_0u,\quad B_1-C=qv,\quad C-A=pw.  
]

Then

[  
p_0u+qv+pw=0.  
]

Bounds:

[  
|u|,|v|,|w|\le 2B/X.  
]

There may be nontrivial small solutions if (B/X) large. So triangle alone not enough.

Need many triangles to force consistency statistically.

---

## 4. Alternative: majority decoding by vertex labels

For each vertex (p), define a list of candidate labels:

[  
L_p={H_{pq}^{*}: q\sim p,\ pq\text{ good}}.  
]

All candidates in (L_p) are congruent to

[  
a_p\pmod p  
]

and bounded by (B).

Thus (L_p) is contained in one residue class mod (p) inside ([-B,B]), so

[  
|L_p|\lesssim B/p\asymp B/X.  
]

If (B\sim X\log X\sqrt R), this list size is about (\log X\sqrt R), manageable.

Good edge (pq) has label (H_{pq}) belonging to both (L_p) and (L_q).

Thus we have a graph whose edges are labelled by integers, and incident edge labels at a vertex lie in a short arithmetic progression modulo (p).

Need show dense edge-labelled graph with such modular constraints has a dominant common label, or else many labels/crossings causing energy. This is list agreement.

This is exactly where list agreement enters.

---

## 5. For partition, list size may be acceptable

At scale (B_R), per-vertex list size

[  
\ll B_R/X.  
]

For relevant (R), this is much smaller than (p). Even if lists have size (L_s), entropy per vertex becomes (\log L_s), not (\log X). Boundary energy may pay it.

So single-block theorem could be proved via list decoding:

1. Low energy gives each vertex a short list (L_p) of candidate ordinary labels.
    
2. Dense good edges imply lists agree on many edges.
    
3. List agreement collapse gives a short global list of labels.
    
4. Complete graph / cross energy collapses list to one dominant label, or pays energy.
    
5. Sum over global labels and small lists.
    

This is plausible and matches known list agreement expansion.

---

## 6. Large-(B) dense agreement may be list, not unique

So the earlier hope “many common neighbours force single label immediately” was too quick. For (B>X), a vertex may support multiple labels congruent mod (p). But the list size is only (B/p). The correct theorem is:

**Single-block list agreement lemma.**  
If most edges have (|H_{pq}|\le B), then there is a short list

[  
\mathcal L={m_1,\dots,m_s},\qquad s\ll B/X  
]

such that most vertices (p) satisfy

[  
a_p\equiv m_j\pmod p  
]

for some (m_j\in\mathcal L).

Then complete graph cross edges force one label to dominate unless multiple labels are separated into clusters with boundary energy.

For partition bound, list size entropy

[  
\log s\ll \log(B/X)  
]

is small.

This is likely the right local theorem.

---

# 7. Does this suffice for (Z_P\ll X\log X)?

For (R\sim1),

[  
B_R\sim X\log X,  
]

so

[  
s\ll \log X.  
]

A global list of (\log X) labels is fine. Cross-label boundary in complete graph costs (|A||B|), so only one label can dominate in low energy. Thus single-block partition bound should hold.

For larger (R), (s\ll \log X\sqrt R), still entropy (\log s) is tiny compared with energy layer (R), except perhaps very small R where s small anyway.

Thus this is promising.

---

# 8. Single-block proof architecture

For each energy layer (R):

1. Set  
    [  
    B_R=CX^2\sqrt{R}/N\asymp CX\log X\sqrt R.  
    ]
    
2. Mark good edges with (|H_{pq}|\le B_R). Most edges good.
    
3. For each vertex (p), candidate list  
    [  
    L_p={n\in[-B_R,B_R]: n\equiv a_p\pmod p}  
    ]  
    has size  
    [  
    \ll B_R/X\ll \log X\sqrt R.  
    ]
    
4. Every good edge (pq) witnesses  
    [  
    L_p\cap L_q\ne\varnothing  
    ]  
    namely (H_{pq}).
    
5. Dense graph with pairwise list intersections implies there is a small global list (\mathcal L) covering most vertices. In fact choose a regular vertex (p_0); its list (L_{p_0}) has size (s), and most neighbours (q) have (L_q\cap L_{p_0}\ne\emptyset), so (q) is covered by (\mathcal L=L_{p_0}). Thus global list size (s).
    
6. Partition vertices according to which (m\in\mathcal L) they choose. If more than one large class, cross edges between different labels have Irving energy (\gg) product of class sizes, unless labels congruent modulo many primes, which for ordinary distinct labels in ([-B_R,B_R]) can happen only on few primes.
    
7. Therefore one (m) dominates; exceptions paid.
    

This is actually quite clean.

---

# 9. Key list-cover step

Let (p_0) be regular. Define

[  
\mathcal L=L_{p_0}.  
]

For any good edge (p_0q), (H_{p_0q}\in L_{p_0}\cap L_q). Thus (q) is covered by (\mathcal L). Since (p_0) has degree ((1-o(1))N), most vertices are covered by (\mathcal L).

Excellent. No complex agreement theorem needed for single block.

Then list-collapse by complete graph.

This is a substantial simplification.

---

# 10. Next: prove list-collapse inside block

Given (\mathcal L) of size (s\ll B/X), most vertices choose at least one compatible label. Assign each covered vertex one label (m(p)\in\mathcal L) with

[  
a_p\equiv m(p)\pmod p.  
]

If two labels (m\ne m') both have large vertex classes (A,B), then for cross edge (pq), p in A, q in B, phase difference contains

[  
D=m'-m.  
]

For most q in B, (D\not\equiv0\pmod q), since (|D|\le2B) and (D) cannot be divisible by many primes (q\sim X) unless product (\le2B). Number of such q divisors is at most

[  
O(\log B/\log X).  
]

For (B=B_R), this is (O(1+\log R/\log X)), small relative to (|B|) for large block. Thus almost all cross edges cost Irving. Hence two large classes impossible at low energy.

So one label dominates.

---

# 11. This essentially proves single-block counting

Covered exceptions:

- vertices not adjacent good to (p_0): (O(\eta N)), paid by bad edge energy / entropy.
    
- vertices in minority labels: paid by cross energy.
    
- residues not in list: paid by bad edges to p0 or majority.
    

Thus

[  
#{a:Q_P(a)\le R}  
\le  
e^{\varepsilon R}  
#{m:m^2\sigma_P^2\le CR}.  
]

This is exactly local Poincaré.
