好。現在把局部 single-block 壓縮接到全局。

前一步得到的圖像是：

每個 prime block (P_k) 內，由同尺度 complete graph energy

[  
Q_{P_k}^{\mathrm{int}}  
]

把巨大 residue assignment

[  
(a_p)_{p\in P_k}  
]

壓縮成一個 ordinary block label

[  
m_k,  
]

外加少量 energy-paid exceptions。

現在全局剩下的主要問題是：不同 blocks 的 labels

[  
m_k  
]

是否必須相同？

答案應該是是的：跨 block semiprime edges 會把它們合併。

---

# 1. 全局 block-label energy

假設每個 block (P_k) 幾乎服從 label (m_k)：

[  
a_p\equiv m_k\pmod p  
\qquad(p\in P_k).  
]

對兩個 compatible blocks (P_k,P_\ell)，跨邊 (pq) 的相位為

[  
\phi_{pq}  
\equiv  
\frac{m_k}{pq}  
+  
\frac{(m_\ell-m_k)\bar p}{q}  
\pmod1,  
]

其中 (p\in P_k,\ q\in P_\ell)。

若

[  
m_k=m_\ell,  
]

這就是 ordinary diagonal label (m_k) 的貢獻。

若

[  
m_k\ne m_\ell,  
]

令

[  
D=m_\ell-m_k\ne0.  
]

對大多數 (q\in P_\ell)，若

[  
D\not\equiv0\pmod q,  
]

Irving dispersion 給：

[  
\sum_{p\in P_k}  
\left|  
\frac{D\bar p}{q}  
\right|^2  
\gg |P_k|.  
]

因此跨 block energy 至少

[  
\gg |P_k||P_\ell|  
]

或至少帶 polylog 損失的巨大量。

所以不同 block labels 之間有強 ferromagnetic coupling。

---

# 2. 若 (D) 被很多 (q) 整除？

若

[  
D\equiv0\pmod q  
]

對許多

[  
q\in P_\ell  
]

成立，則 (D) 被許多 primes 除盡。  
但

[  
|D|\le |m_k|+|m_\ell|.  
]

在能量層 (R) 中，single-block counting 給

[  
m_k^2\sigma_k^2\lesssim R,  
\qquad  
m_\ell^2\sigma_\ell^2\lesssim R.  
]

所以

[  
|m_k|\lesssim \frac{\sqrt R}{\sigma_k}  
\asymp \sqrt R,X_k\log X_k,  
]

[  
|m_\ell|\lesssim \sqrt R,X_\ell\log X_\ell.  
]

取 block 中少數 primes，其乘積很快超過 (|D|)。  
因此若 (D) 被足夠多 block primes 整除，則只能

[  
D=0.  
]

所以 genuine difference labels cannot hide by divisibility, except for a small divisor-exception set, whose entropy/energy can be controlled.

---

# 3. Block-label chain collapse

現在考慮 block-level graph，頂點是 (k)，label 是 (m_k)。  
跨 edge ((k,\ell)) 的 cost 大約：

[  
\mathcal E_{k\ell}(m_k,m_\ell)  
\approx  
\begin{cases}  
Q_{k\ell}^{\mathrm{diag}}(m_k),&m_k=m_\ell,\  
\gg |P_k||P_\ell|/\log^C,&m_k\ne m_\ell.  
\end{cases}  
]

這是一個 ordinary integer spin model，且不同 labels 的邊界成本巨大。

因此 Peierls/chain argument now becomes easy：

- 若所有 (m_k) 相同，得到全局 ordinary label (m)；
    
- 若 labels 分成多個 regions，regions之間的 boundary block edges 有巨大成本；
    
- 這個成本支付選擇不同 (m_k)、選擇 region、以及 local exceptions 的 entropy。
    

最終 block labels collapse to one global label

[  
m.  
]

---

# 4. 全局 counting theorem

把 single-block theorem 與 block-label collapse 合起來，應得到：

[  
#{a:Q(a)\le R}  
\le  
e^{\varepsilon R}  
#{m:m^2\sigma^2\le C R}.  
]

其中

[  
\sigma^2=\sum_{pq\in E_{\mathrm{full}}}\frac1{p^2q^2}.  
]

具體分解：

1. 每個 block (P_k)：  
    [  
    a|_{P_k}  
    \leadsto  
    m_k  
    ]  
    with exceptions paid.
    
2. Internal block energies give:  
    [  
    \prod_k  
    e^{\varepsilon R_k}  
    #{m_k:m_k^2\sigma_k^2\le CR_k}.  
    ]
    
3. Cross-block energies force:  
    [  
    m_k=m  
    ]  
    for all k, except block-label defects paid by boundary energy.
    
4. The remaining label (m) has global diagonal energy  
    [  
    m^2\sigma^2\lesssim R.  
    ]
    

Thus:

[  
#{a:Q(a)\le R}  
\le  
e^{\varepsilon R}  
\frac{\sqrt R}{\sigma}.  
]

This is exactly the desired global Poincaré counting.

---

# 5. 為什麼這比之前好？

之前我們直接在跨尺度 graph 上找 seed，總被高頻／top entropy 卡住。  
現在先加入同尺度 complete graph，單 block 內部就把 arbitrary CRT residue 熵壓掉。於是跨尺度階段不再面對巨大 alphabet，只面對 ordinary integer labels (m_k)。

這是一個真正的降維：

[  
\prod_{p\in P_k}\mathbb F_p  
\quad\longrightarrow\quad  
m_k\in\mathbb Z.  
]

所以 top entropy 不再以 raw CRT 形式出現。

---

# 6. 檢查是否允許加入同尺度 edges

原問題要求 denominators 是互異 semiprimes，沒有禁止兩個 primes 同尺度。  
同尺度 edges (pq) 是合法 squarefree semiprimes。

加入這些 edges會：

- 增加 reciprocal mass：  
    [  
    \sum_k \sum_{p<q\in P_k}\frac1{pq}  
    \asymp  
    \sum_k \frac1{k^2},  
    ]  
    convergent，不會破壞 mass tuning；
    
- 增加 variance，但可控；
    
- 強化 minor arc energy；
    
- 不影響 major arc Gaussian 形式，只是把 (\sigma^2) 加上同尺度貢獻。
    

所以這是合法且有益的修改。

# 7. 剩餘的技術核心列表

現在剩下三個具體 lemma：

## Lemma I：single-block CRT list-Helly / counting

[  
#{a_{P_k}:Q_{P_k}^{int}(a)\le R}  
\le  
e^{\varepsilon R}  
\frac{\sqrt R}{\sigma_k}.  
]

核心估計：cross-label divisor-energy lemma。

## Lemma II：block-label cross-energy

對 ordinary labels (m_k\ne m_\ell),

[  
E_{k\ell}(m_k,m_\ell)  
\gg  
\frac{|P_k||P_\ell|}{\log^C}  
]

或足以支付 block-label entropy，除 divisor exceptions.

## Lemma III：block-label Peierls / chain collapse

普通 integer label model with strong ferromagnetic cross costs has partition function

[  
\ll \sum_m e^{-c m^2\sigma^2}  
\asymp1/\sigma.  
]

這三個都比原始問題小很多，而且各自有明確證明路線。
