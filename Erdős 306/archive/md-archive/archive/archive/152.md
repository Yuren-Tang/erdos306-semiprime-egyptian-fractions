對，這次的徵兆確實好得多。原因可以很簡單地說：

以前的路線是：

[  
\text{跨尺度找 seed}\quad\Rightarrow\quad\text{seed 不夠高}\quad\Rightarrow\quad\text{再加高層}\quad\Rightarrow\quad\text{高層又有新熵}.  
]

這就是「層變塔」爆炸。

現在的新路線是：

[  
\boxed{  
\text{先在每個同尺度 block 內，用 complete graph 把 raw CRT 熵壓成 ordinary label。}  
}  
]

也就是先做局部降維：

[  
(a_p)_{p\in P_k}  
\quad\leadsto\quad  
m_k.  
]

然後全局只剩一串 ordinary integer labels：

[  
m_{k_0},m_{k_0+1},\dots,m_K.  
]

這時再用跨尺度邊迫使

[  
m_k=m_\ell  
]

就容易得多。因為我們不再面對

[  
\prod_{p\in P_k}\mathbb F_p  
]

那種巨大字母表，而是面對普通整數 label。這是本質性的降維，不是修補。

現在繼續 Lemma II。

---

# Lemma II：block-label cross-energy

取兩個 prime blocks

[  
P\subset[X,2X],\qquad Q\subset[Y,2Y],  
]

尺度 compatible。假設 (P) 服從 ordinary label (m)，(Q) 服從 ordinary label (m')：

[  
a_p\equiv m\pmod p,\qquad p\in P,  
]

[  
a_q\equiv m'\pmod q,\qquad q\in Q.  
]

令

[  
D=m'-m.  
]

若

[  
D=0,  
]

則兩 blocks 屬同一 label。若

[  
D\ne0,  
]

則跨邊相位為

[  
\phi_{pq}  
\equiv  
\frac{m}{pq}  
+  
\frac{D\bar p}{q}  
\pmod1.  
]

我們要證：除非 (D) 被大量 (q\in Q) 整除，跨邊能量很大。

---

## 1. Irving 直接給下界

對固定 (q\in Q)，若

[  
D\not\equiv0\pmod q,  
]

則 Irving-good pruning 給：

[  
\sum_{p\in P}  
\left|  
\frac{D\bar p}{q}  
\right|^2  
\gg |P|.  
]

由基本不等式

[  
|x+y|^2+|x|^2\gg |y|^2  
]

得

[  
\sum_{p\in P}  
\left|  
\frac{m}{pq}  
+  
\frac{D\bar p}{q}  
\right|^2  
+  
\sum_{p\in P}  
\left|  
\frac{m}{pq}  
\right|^2  
\gg |P|.  
]

因此，要麼 cross edge 自身付能量，要麼 diagonal background (m) 已付能量。放在 partition counting 中，這兩者都可接受。

若對正比例的 (q\in Q) 有

[  
q\nmid D,  
]

則

[  
Q_{P,Q}(m,m')  
+  
Q_{P,Q}^{\mathrm{diag}}(m)  
\gg  
|P||Q|.  
]

所以 genuine label mismatch 的 boundary cost 是 complete-bipartite 級別，非常大。

---

## 2. 整除例外

若 (q\mid D)，則 (m) 與 (m') 在該 (q)-頂點不可區分：

[  
m\equiv m'\pmod q.  
]

但 (D\ne0) 能被多少個 (q\sim Y) 整除？

有粗界：

[  
#{q\in Q:q\mid D}  
\le  
\frac{\log |D|}{\log Y}.  
]

在能量層 (R) 中，block labels 由 single-block counting 給出：

[  
|m|  
\lesssim \frac{\sqrt R}{\sigma_P},  
\qquad  
|m'|  
\lesssim \frac{\sqrt R}{\sigma_Q}.  
]

所以

[  
|D|  
\lesssim  
\sqrt R  
\left(  
\frac1{\sigma_P}  
+  
\frac1{\sigma_Q}  
\right).  
]

這使

[  
\frac{\log |D|}{\log Y}  
]

遠小於 (|Q|)；尤其當 (Q) 是大 block 時，整除例外只佔極小比例。

因此對幾乎所有 (q\in Q),

[  
q\nmid D.  
]

所以 mismatch energy 下界成立。

---

## 3. Lemma II 的正式形式

**Block-label cross-energy lemma.**  
對 compatible blocks (P,Q)，ordinary labels (m\ne m')，有

[  
Q_{P,Q}(m,m')  
+  
Q_{P,Q}^{\mathrm{diag}}(m)  
\gg  
|P|  
\left(  
|Q|-  
O\left(\frac{\log |m-m'|}{\log Y}\right)  
\right).  
]

尤其，在所有 relevant energy layers 中，

[  
Q_{P,Q}(m,m')  
+  
Q_{P,Q}^{\mathrm{diag}}(m)  
\gg  
|P||Q|.  
]

若要 symmetric 版本，也可加上

[  
Q_{P,Q}^{\mathrm{diag}}(m')  
]

吸收另一側背景。

---

# 4. 這個 lemma 對 block labels 的意義

一旦每個 block 壓成 label (m_k)，跨尺度邊讓不同 labels 幾乎不可能相鄰。

若

[  
m_k\ne m_{k+1},  
]

則該 block boundary cost 約

[  
|P_k||P_{k+1}|.  
]

而選擇一個新的 label (m_{k+1}) 的 entropy 只有

[  
\log |m_{k+1}|.  
]

選擇一段 label-droplet 的形狀，在 block chain 上也只有

[  
O(\log K)  
]

級別。相比

[  
|P_k||P_{k+1}|  
]

完全可以忽略。

所以 block labels 的 Peierls collapse 很容易。

---

# 5. Block-label Peierls theorem

現在可寫第三個 lemma：

設每個 block (k) 有 ordinary label (m_k)，能量為

# [  
\mathcal Q(m_\bullet)

\sum_k Q_{P_k}^{\mathrm{diag}}(m_k)  
+  
\sum_{(k,\ell)}  
Q_{k\ell}^{\mathrm{cross}}(m_k,m_\ell).  
]

則

[  
\sum_{(m_k)}  
e^{-c\mathcal Q(m_\bullet)}  
\ll  
\sum_{m\in\mathbb Z}  
e^{-c'Q^{\mathrm{diag}}_{\mathrm{global}}(m)}.  
]

證明：

- 若所有 (m_k=m)，就是 global label (m)；
    
- 若有一段 blocks label 不同於外部，邊界跨邊付  
    [  
    \gg |P_k||P_\ell|  
    ]  
    的成本；
    
- 這成本支付該段 label 選擇與位置選擇；
    
- nested droplets 也一樣；
    
- 因此 partition function dominated by pure phases.
    

這是普通一維 Peierls，比先前巨大 CRT-spin Peierls簡單得多。

---

# 6. 全局證明骨架現在很清楚

加入兩類 edges：

1. **internal edges**：每個 block 內 complete graph；
    
2. **cross edges**：不同 compatible blocks 之間，用於 mass 與 label merging。
    

總 energy：

[  
Q(a)=Q_{\mathrm{int}}(a)+Q_{\mathrm{cross}}(a).  
]

步驟：

### Step 1: block-local compression

由 Lemma I：

[  
\sum_{a|_{P_k}}e^{-cQ_{P_k}^{int}(a)}  
\ll  
\sum_{m_k}e^{-c'Q_{P_k}^{diag}(m_k)}.  
]

更精確地，以 counting/conditional 方式，把每個 block residue assignment 編碼為 (m_k)+exceptions，exceptions energy-paid。

### Step 2: cross-block collapse

由 Lemma II：

[  
m_k\ne m_\ell  
\Rightarrow  
\text{cross energy huge}.  
]

### Step 3: block-label Peierls

所以所有 (m_k) 合併為一個 global ordinary (m)，得到

[  
Z  
\ll  
\sum_m e^{-c Q_{\mathrm{diag}}(m)}  
\asymp  
1/\sigma.  
]

### Step 4: minor arc

主弧外 (|m|>C/\sigma) 的 Gaussian tail 是

[  
o(1/\sigma).  
]

因此 Fourier minor arc閉合。

---

# 7. 目前最關鍵仍是 Lemma I

Lemma II 很穩，基本由 Irving + divisor count 解決。  
Block-label Peierls 也穩。  
真正需要細寫的是 Lemma I，即 single-block counting theorem，核心為：

- base-list reduction；
    
- CRT list-Helly / energy list-collapse；
    
- cross-label divisor-energy bound；
    
- exceptions entropy payment。
    

但這已是一個局部問題，沒有層變塔爆炸。

所以是的：我們似乎真的找到了避開層變塔爆炸的正確結構。
