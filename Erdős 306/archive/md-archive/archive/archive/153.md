# 主定理的證明骨架

目標：給定有理數

[  
\frac ab  
]

構造一組互異 squarefree semiprime denominators

[  
n=pq  
]

使

[  
\frac ab=\sum_{n\in S}\frac1n.  
]

Fourier-probabilistic 路線是：構造 Bernoulli 隨機和

[  
X=\sum_{pq\in E}\xi_{pq}\frac1{pq},  
]

使

[  
\mathbb E X=\frac1b  
]

並證

[  
\mathbb P(X=1/b)>0.  
]

然後重複 (a) 次，避開已用 primes，得到 (a/b)。

---

# I. 結構設計

選一批 prime blocks

[  
P_k\subset[2^k,2^{k+1}]  
]

經過 Irving-good pruning。邊集合 (E) 包含兩類 semiprime edges：

### 1. 同尺度 internal edges

[  
p q,\qquad p,q\in P_k,\ p<q.  
]

這些邊主要用於 minor arc 控制：在每個 block 內把 arbitrary CRT residues 壓縮為 ordinary label。

### 2. 跨尺度 cross edges

[  
p q,\qquad p\in P_k,\ q\in P_\ell  
]

其中 (k,\ell) compatible，例如尺度相近或滿足 Irving 範圍。這些邊有兩個作用：

- 提供足夠 reciprocal mass；
    
- 迫使不同 block labels 合併成同一 global label。
    

同尺度 edges 的 reciprocal mass 約為

[  
\sum_k \frac1{k^2},  
]

是收斂的；不會破壞 mass tuning。跨尺度 edges 負責提供可調的主質量。

---

# II. 主弧

ordinary global label 是

[  
a_p\equiv m\pmod p  
\quad\forall p.  
]

對小 (m)，Fourier 相位為

[  
\frac{m}{pq}.  
]

能量近似

# [  
Q_{\mathrm{diag}}(m)

\sum_{pq\in E}  
\left|\frac{m}{pq}\right|^2  
\sim  
m^2\sigma^2,  
]

其中

[  
\sigma^2=\sum_{pq\in E}\frac1{p^2q^2}.  
]

主弧範圍：

[  
|m|\le C/\sigma.  
]

在此範圍內 characteristic function 有 Gaussian 展開，主弧貢獻：

[  
\asymp \frac1\sigma.  
]

這部分此前已經相當穩。

---

# III. Minor arc 的核心 partition theorem

需要證：

[  
\sum_{a\in\prod_p\mathbb F_p}  
e^{-cQ(a)}  
\ll  
\frac1\sigma,  
]

且主弧外：

[  
\sum_{a\notin\mathfrak M}e^{-cQ(a)}  
=o(1/\sigma).  
]

這是整個小弧的核心。現在的新結構是用三個引理完成它。

---

# Lemma 1：single-block counting theorem

對單個 block

[  
P=P_k\subset[X,2X],  
]

定義 internal energy

[  
Q_P(a)=  
\sum_{p<q\in P}|\phi_{pq}(a)|^2.  
]

令

[  
\sigma_P^2=  
\sum_{p<q\in P}\frac1{p^2q^2}  
\asymp  
\frac1{X^2(\log X)^2}.  
]

要證：

[  
#{a_P:Q_P(a_P)\le R}  
\le  
e^{\varepsilon R}  
#{m:m^2\sigma_P^2\le C R}.  
]

等價地：

[  
#{a_P:Q_P(a_P)\le R}  
\le  
e^{\varepsilon R}\frac{\sqrt R}{\sigma_P}.  
]

從而

[  
\sum_{a_P}e^{-cQ_P(a_P)}  
\ll  
\frac1{\sigma_P}.  
]

這個引理的意義是：

[  
\prod_{p\in P}\mathbb F_p  
]

這個巨大 CRT 字母表，在同尺度 complete graph 的能量下，塌縮成 ordinary label (m_k)。

---

## Lemma 1 的證明部件

### 1. Base-list reduction

若

[  
Q_P(a)\le R,  
]

取

[  
B_R\asymp X\log X\sqrt R.  
]

多數邊有 bounded CRT glue：

[  
|H_{pq}^{*}|\le B_R.  
]

選 regular base vertex (p_0)，定義

# [  
\mathcal L=L_{p_0}

{n\in[-B_R,B_R]:n\equiv a_{p_0}\pmod {p_0}}.  
]

則

[  
|\mathcal L|\ll1+\log X\sqrt R.  
]

大多數 vertices (q) 被 (\mathcal L) 覆蓋，即存在 (m\in\mathcal L) 使

[  
a_q\equiv m\pmod q.  
]

### 2. Cross-label divisor-energy lemma

若兩個不同 labels

[  
m\ne m'  
]

分別覆蓋 subsets

[  
A,B\subset P,  
]

則跨邊 energy 有下界：

[  
E(A,B;m,m')  
\gg  
|A||B|/(\log X)^C  
]

至少在 (A,B) 非微小時。證明用 divisor incidence：

若 cross glue (|H_{pq}|\le T)，則存在 (n) with

[  
|n|\le T,\quad p\mid n-m,\quad q\mid n-m'.  
]

因此 small cross edges 數量由

[  
\sum_{|n|\le T}  
\omega_A(n-m)\omega_B(n-m')  
]

控制，而每個整數在 ([X,2X]) 中的 prime divisors 數很少。Layer-cake 給 energy lower bound。

### 3. Dominant label

如果沒有一個 label (m\in\mathcal L) 覆蓋絕大多數 vertices，則有 substantial cross-label pairs，由上一步產生巨大 energy，與 (Q_P(a)\le R) 矛盾，或這些 vertices 作為 exceptions 由 energy 支付。

因此低能 assignment 必有 dominant (m)，例外 residues 的 entropy 由 internal cross energy支付。

這完成 Lemma 1。

---

# Lemma 2：cross-block label energy

假設 block (P_k) 服從 label (m_k)，block (P_\ell) 服從 label (m_\ell)。

若

[  
m_k\ne m_\ell,  
]

則對跨邊 (p\in P_k,q\in P_\ell)：

[  
\phi_{pq}  
\equiv  
\frac{m_k}{pq}  
+  
\frac{(m_\ell-m_k)\bar p}{q}.  
]

Irving-good 性質給：

[  
Q_{k\ell}(m_k,m_\ell)  
+  
Q_{k\ell}^{\mathrm{diag}}(m_k)  
\gg  
|P_k||P_\ell|  
]

除非 (m_\ell-m_k) 被大量 (q\in P_\ell) 整除；但 divisor bound 控制這類例外。

所以不同 block labels 的 boundary cost 巨大。

---

# Lemma 3：block-label Peierls collapse

經 Lemma 1 後，整個系統可視為 block labels

[  
(m_k)_k  
]

加上 energy-paid exceptions。

Block-label energy 形如：

# [  
\mathcal Q(m_\bullet)

\sum_k Q_{P_k}^{\mathrm{diag}}(m_k)  
+  
\sum_{k\ell}Q_{k\ell}^{\mathrm{cross}}(m_k,m_\ell).  
]

Lemma 2 說：

[  
m_k\ne m_\ell  
]

的 boundary cost 遠大於選擇 label-droplet 的 entropy。

所以 Peierls collapse 給：

[  
\sum_{(m_k)}  
e^{-c\mathcal Q(m_\bullet)}  
\ll  
\sum_{m\in\mathbb Z}  
e^{-c'Q_{\mathrm{diag}}(m)}.  
]

右側是 global ordinary label (m) 的 Gaussian sum：

[  
\sum_m e^{-c'm^2\sigma^2}  
\asymp  
1/\sigma.  
]

主弧外尾部：

[  
\sum_{|m|>C/\sigma}e^{-c'm^2\sigma^2}  
=o(1/\sigma).  
]

# IV. Fourier 完成

Fourier inversion：

# [  
\mathbb P(X=1/b)

\frac1L  
\sum_{\chi}  
\widehat\mu(\chi)e(-\chi(1/b)).  
]

主弧貢獻：

[  
\frac1L\cdot c_0/\sigma.  
]

小弧絕對值：

[  
\frac1L\cdot o(1/\sigma).  
]

因此

[  
\mathbb P(X=1/b)>0.  
]

所以存在一個 subset (S\subset E) 使

[  
\sum_{pq\in S}\frac1{pq}=\frac1b.  
]

再以 disjoint prime blocks 重複 (a) 次，得

[  
\frac ab.  
]
