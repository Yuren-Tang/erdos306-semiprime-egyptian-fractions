好。現在細寫 **Lemma 1：single-block counting theorem** 的核心證明。重點是兩件事：

[  
\text{小 representative 的 incidence 上界}  
\quad\Longrightarrow\quad  
\text{cross-label energy 下界}  
]

以及

[  
\text{cross-label energy}  
\quad\Longrightarrow\quad  
\text{minority labels 的 entropy 被支付。}  
]

---

# 1. 設定回顧

取

[  
P\subset[X,2X],\qquad N=|P|\asymp \frac X{\log X}.  
]

對每個

[  
p\in P  
]

有 residue

[  
a_p\in\mathbb F_p.  
]

同尺度能量：

[  
Q_P(a)=  
\sum_{p<q}  
\left(\frac{|H_{pq}^{*}|}{pq}\right)^2.  
]

因

[  
pq\asymp X^2,  
]

可粗略寫作

[  
Q_P(a)\asymp  
\frac1{X^4}  
\sum_{p<q}|H_{pq}^{*}|^2.  
]

令

[  
Q_P(a)\le R.  
]

取

[  
B_R=A X\log X\sqrt R.  
]

多數邊滿足

[  
|H_{pq}^{*}|\le B_R.  
]

---

# 2. Base-list reduction

選 regular base vertex

[  
p_0.  
]

定義

# [  
\mathcal L

{n\in[-B_R,B_R]:n\equiv a_{p_0}\pmod{p_0}}.  
]

則

[  
|\mathcal L|  
\ll  
1+\frac{B_R}{X}  
\ll  
1+\log X\sqrt R.  
]

大多數 (q\in P) 被 (\mathcal L) 覆蓋，即存在

[  
m\in\mathcal L  
]

使

[  
a_q\equiv m\pmod q.  
]

把被覆蓋 vertices 任意指派到某個 (m\in\mathcal L)，得到 classes

[  
C_m.  
]

若存在

[  
m_*  
]

使

[  
|C_{m_*}|\ge (1-\rho)N,  
]

則 dominant label 已得。接下來只需處理沒有 dominant label 的情形。

---

# 3. Cross-label incidence bound

取兩個不同 labels

[  
m\ne m',  
\qquad  
D=m'-m.  
]

取兩個 classes

[  
A=C_m,\qquad B=C_{m'}.  
]

對

[  
p\in A,\quad q\in B  
]

的 cross CRT representative (H_{pq}^{*}) 滿足：

[  
H_{pq}^{_}\equiv m\pmod p,  
\qquad  
H_{pq}^{_}\equiv m'\pmod q.  
]

令

# [  
N_{A,B}(T)

#{(p,q)\in A\times B: |H_{pq}^{*}|\le T}.  
]

若 (|H_{pq}^{*}|\le T)，存在 (|n|\le T) 使

[  
p\mid n-m,\qquad q\mid n-m'.  
]

故

[  
N_{A,B}(T)  
\le  
\sum_{|n|\le T}  
\omega_A(n-m)\omega_B(n-m')  
+  
\mathrm{Spec},  
]

其中 special terms 只來自 (n=m) 或 (n=m')，由

[  
\omega_A(D),\omega_B(D)  
]

控制。

對非零 (t)，有

[  
\omega_A(t)\le \omega_X(t)  
\ll  
1+\frac{\log |t|}{\log X}.  
]

因此若

[  
T\le X^2,\qquad |m|,|m'|\le B_R,  
]

則

[  
N_{A,B}(T)  
\ll  
T L^2+(|A|+|B|)L,  
]

其中

[  
L=1+\frac{\log(B_R+T+2)}{\log X}.  
]

在我們關心的範圍，(L) 是 polylog 或至多 (O(1+\log R/\log X))。

---

# 4. Incidence bound 推出 energy lower bound

設

[  
M=|A||B|.  
]

取

[  
T_*=  
c\frac{M}{L^2}  
]

並截斷於 (X^2)：

[  
T_*=\min\left(c\frac{M}{L^2},,cX^2\right).  
]

若 special terms 不大於 (M/10)，則由 incidence bound 可取 (c) 小，使

[  
N_{A,B}(T_*)\le \frac12 M.  
]

所以至少 (M/2) 條 cross edges 滿足

[  
|H_{pq}^{_}|>T__.  
]

因此

[  
E(A,B;m,m')  
\gg  
M\left(\frac{T_*}{X^2}\right)^2.  
]

即

[  
E(A,B;m,m')  
\gg  
M\min\left(  
1,\frac{M^2}{X^4L^4}  
\right).  
]

這是核心 cross-energy 下界。

---

# 5. special terms 的處理

若 special terms 大，表示

[  
D=m'-m  
]

有許多 prime divisors in (A) or (B)。

但

[  
0<|D|\le 2B_R.  
]

所以

[  
\omega_P(D)  
\ll  
1+\frac{\log B_R}{\log X}  
\ll L.  
]

因此 special terms 最多約

[  
(|A|+|B|)L.  
]

只要

[  
|A||B|\gg (|A|+|B|)L,  
]

即兩側都不是極小，special terms可忽略。  
若其中一側很小，該 label class 本身會被當作 exception，其 entropy稍後由 majority-cross energy支付。

所以 special terms 不會破壞大 class 的能量下界。

---

# 6. 從 cross-energy 到 dominant class

現在假設沒有 dominant label。令 classes (C_m) 按大小 dyadic 分層。

總 covered vertices 至少

[  
(1-O(\sqrt\eta))N.  
]

若沒有 dominant class，則存在不少 cross mass：

# [  
\sum_{m\ne m'} |C_m||C_{m'}|

N_{\mathrm{cov}}^2-\sum_m|C_m|^2  
\gg \rho N^2  
]

其中 (\rho) 衡量「非 dominant」程度。

把 cross-energy 下界加總，得到一個正的 lower bound。粗略地，如果每個 substantial class 大小至少 (s_0)，則每對 substantial classes給

[  
|C_m||C_{m'}|  
\min\left(1,  
\frac{|C_m|^2|C_{m'}|^2}{X^4L^4}  
\right).  
]

若 classes 都很小，則 label 數很多；但 list size

[  
|\mathcal L|\ll \log X\sqrt R  
]

限制了 class 數。  
因此最大 class 至少

[  
\gg \frac{N}{|\mathcal L|}.  
]

代入得到 cross energy 至少大約

[  
\gg  
\frac{N^2}{|\mathcal L|^? \log^C X}  
]

某個正量。這在絕大多數 relevant (R) 下遠大於 (R)，迫使 dominant class存在。

更實用地，我們不需要最優估計；只需結論：

若沒有 dominant class，則

[  
Q_P(a)\ge c,\mathrm{Entropy}(\text{multi-label assignment}).  
]

這可由 dyadic class decomposition 與上面的 cross-energy lower bound完成。

---

# 7. Exception entropy payment

一旦有 dominant class

[  
G=C_m,\qquad |G|\ge(1-\rho)N,  
]

其餘 vertices

[  
D=P\setminus G  
]

是 exceptions。

對 (q\in D)，它的 residue不等於 (m\pmod q)。與 majority class (G) 的 cross energy用同一 cross-label divisor-energy lemma，或若 (D) 中 label不明，直接按其 assigned label處理。

每個 exception 的平均成本至少

[  
\gg \frac{|G|}{(\log X)^C}  
\gg \frac{X}{(\log X)^{C+1}}.  
]

而選擇並賦 residue 的 entropy為

[  
\ll |D|\log X+\log\binom N{|D|}.  
]

因

[  
\frac{X}{(\log X)^{C+1}}\gg \log X,  
]

只要 (X) 足夠大，exceptions 的 entropy完全由 energy支付。

所以計數時可寫：

[  
\text{exception contribution}\le e^{\varepsilon R}.  
]

---

# 8. Dominant label 數量

dominant label (m) 滿足

[  
|m|\le B_R  
\asymp X\log X\sqrt R.  
]

而

[  
1/\sigma_P\asymp X\log X.  
]

所以 label 數量

[  
#{m:|m|\le B_R}  
\ll  
\frac{\sqrt R}{\sigma_P}.  
]

因此

[  
#{a:Q_P(a)\le R}  
\le  
e^{\varepsilon R}  
\frac{\sqrt R}{\sigma_P}.  
]

這就是 single-block counting theorem。

---

# 9. 技術缺口是否可接受？

現在還有一個需要細寫的部分：

[  
\text{dyadic class decomposition}  
\quad+\quad  
\text{cross-energy lower bound}  
\quad\Rightarrow  
\text{dominant class or energy pays entropy}.  
]

但它已是純粹的 combinatorial bookkeeping，不再涉及深層 CRT 或高頻逃逸。  
核心算術估計已經明確：

[  
N_{A,B}(T)  
\ll  
T L^2+(|A|+|B|)L.  
]

這是初等 divisor bound。

所以 Lemma 1 的證明基本可望完成。
