好。現在證 **Lemma 3：block-label Peierls collapse**。

這一步比前面的 single-block counting 簡單得多，因為巨大 CRT residue 空間已經被壓縮成普通整數 labels：

[  
P_k \leadsto m_k.  
]

現在要證所有 (m_k) 會合併成一個 global ordinary label (m)。

---

# 1. Block-label model

每個 block (P_k) 有 label

[  
m_k\in\mathbb Z.  
]

其 internal diagonal energy 為

# [  
Q_k^{\mathrm{diag}}(m_k)

\sum_{p<q\in P_k}  
\left|\frac{m_k}{pq}\right|^2.  
]

兩個 compatible blocks (P_k,P_\ell) 之間的 cross energy 為

# [  
Q_{k\ell}(m_k,m_\ell)

\sum_{p\in P_k,q\in P_\ell}  
|\phi_{pq}(m_k,m_\ell)|^2.  
]

其中

[  
\phi_{pq}(m_k,m_\ell)  
\equiv  
\frac{m_k}{pq}  
+  
\frac{(m_\ell-m_k)\bar p}{q}  
\pmod1.  
]

總 block-label energy：

# [  
\mathcal Q(m_\bullet)

\sum_k Q_k^{\mathrm{diag}}(m_k)  
+  
\sum_{k\ell}Q_{k\ell}(m_k,m_\ell).  
]

要證：

[  
\sum_{(m_k)}e^{-c\mathcal Q(m_\bullet)}  
\ll  
\sum_m e^{-c'Q_{\mathrm{global}}^{\mathrm{diag}}(m)}.  
]

右側就是所有 blocks 同一 label (m) 的 pure phase。

---

# 2. Cross-label cost

若

[  
m_k\ne m_\ell,  
]

令

[  
D=m_\ell-m_k\ne0.  
]

對固定 (q\in P_\ell)，若

[  
q\nmid D,  
]

Irving-good 給：

[  
\sum_{p\in P_k}  
\left|  
\frac{D\bar p}{q}  
\right|^2  
\gg |P_k|.  
]

因此

[  
Q_{k\ell}(m_k,m_\ell)  
+  
Q_{k\ell}^{\mathrm{diag}}(m_k)  
\gg  
|P_k|  
\cdot  
#{q\in P_\ell:q\nmid D}.  
]

而

[  
#{q\in P_\ell:q\mid D}  
\le  
\frac{\log |D|}{\log X_\ell}+O(1).  
]

在能量層 (R) 中，labels 的大小由 internal counting 控制：

[  
|m_k|\lesssim \frac{\sqrt R}{\sigma_k},  
\qquad  
|m_\ell|\lesssim \frac{\sqrt R}{\sigma_\ell}.  
]

所以 (\log|D|) 遠小於 (|P_\ell|\log X_\ell)。因此

[  
#{q:q\nmid D}  
\sim |P_\ell|  
]

除極小例外。故

[  
Q_{k\ell}(m_k,m_\ell)  
+  
Q_{k\ell}^{\mathrm{diag}}(m_k)  
\gg  
|P_k||P_\ell|.  
]

這是巨大邊界成本。

---

# 3. Block-label droplets

考慮 block index graph (\mathcal B)，其邊是 compatible block pairs。  
若 labels 不全相同，則存在某個 label region / droplet：

[  
C={k:m_k=m'}  
]

外部 label 為 (m\ne m')。

其 boundary

# [  
\partial C

{(k,\ell):k\in C,\ell\notin C}  
]

每條 genuine boundary edge付成本

[  
\gg |P_k||P_\ell|.  
]

選擇 droplet 位置的 entropy 至多

[  
\exp(O(|\partial C|\log K))  
]

或者在 chain skeleton 中只是 interval choices，熵更小。  
選擇 droplet label (m') 的 entropy是

[  
\log |m'|.  
]

但如果 (m') 很大，internal diagonal energy

[  
\sum_{k\in C}Q_k^{\mathrm{diag}}(m')  
]

已經很大；如果 (m') 不大，boundary cost輕鬆支付其 label choice。

因此 droplet 權重可估：

[  
w(C,m')  
\le  
\exp\left(  
-c\sum_{(k,\ell)\in\partial C}|P_k||P_\ell|  
\right)  
\cdot  
e^{-c\sum_{k\in C}Q_k^{\mathrm{diag}}(m')}.  
]

這對所有 proper droplets 可求和。

---

# 4. 一維 chain 版本更簡單

若我們只取一條 block skeleton chain，boundary of a droplet interval 至多兩條邊。  
但每條 boundary edge cost 已經是

[  
\gg |P_k||P_{k+1}|,  
]

巨大。選擇 interval 有 (O(K^2)) 種，遠小於 boundary cost。

Nested intervals 也可用標準 Peierls bound：

[  
\sum_{\Gamma}  
\prod_{\gamma\in\Gamma}e^{-c\mathrm{Cost}(\gamma)}  
\le C.  
]

所以 chain 上已足夠。

若使用 fuller compatible graph，boundary更大，只會更好。

---

# 5. Collapse to one global label

因此 partition function over block labels satisfies：

[  
\sum_{(m_k)}  
e^{-c\mathcal Q(m_\bullet)}  
\le  
C  
\sum_m  
\exp\left(  
-c'  
\sum_k Q_k^{\mathrm{diag}}(m)  
-c'  
\sum_{k\ell}Q_{k\ell}^{\mathrm{diag}}(m)  
\right).  
]

右側 exponent 正是 global diagonal energy：

# [  
Q_{\mathrm{diag}}(m)

\sum_{pq\in E}  
\left|\frac{m}{pq}\right|^2.  
]

所以

[  
\sum_{(m_k)}  
e^{-c\mathcal Q(m_\bullet)}  
\le  
C  
\sum_m e^{-c'Q_{\mathrm{diag}}(m)}.  
]

---

# 6. Gaussian evaluation

對 ordinary (m)，只要在主有效範圍內無繞回：

[  
Q_{\mathrm{diag}}(m)  
\asymp m^2\sigma^2.  
]

對大 (m)，若有繞回，block-local internal energy仍已確保不會有大量異常低能 ordinary labels；更直接地，對 global ordinary (m) 的 Fourier 主尾可用同樣的 one-dimensional Gaussian/aperiodicity estimate。此時 ordinary (m) 是真正一維，不再有 CRT entropy問題；即使需要分段處理，也只是整數和：

[  
\sum_m e^{-cQ_{\mathrm{diag}}(m)}.  
]

在小到中等 (m)：

[  
Q_{\mathrm{diag}}(m)\asymp m^2\sigma^2.  
]

因此

[  
\sum_m e^{-cQ_{\mathrm{diag}}(m)}  
\ll  
\frac1\sigma.  
]

主弧外：

[  
\sum_{|m|>C/\sigma}e^{-cQ_{\mathrm{diag}}(m)}  
=o(1/\sigma).  
]

這裏剩下的 ordinary (m) high-frequency 問題遠小於原本 arbitrary CRT high-frequency，因為它是一維整數求和，不帶巨大 residue entropy。

---

# 7. 是否還有 ordinary (m) 的 top wrap 問題？

要小心：ordinary (m) modulo (L) 仍可能很大，並在某些 (pq) 上 wrap，使

[  
m\bmod pq  
]

小。  
但現在只是在求

[  
\sum_{m\in\mathbb Z/L\mathbb Z}  
e^{-cQ_{\mathrm{diag}}(m)}.  
]

這仍可能有一維 aperiodicity問題。不過與之前不同，它沒有每個 block 的 arbitrary residue entropy；可用 divisor counting處理。

若 (m\bmod pq) 小 for many same-scale pairs (p,q\in P_k)，single-block list-Helly with (a_p=m\bmod p) implies (m) agrees with a small ordinary representative on most of (P_k). But since (m) itself is ordinary, this means

[  
m\equiv m_k\pmod{\prod_{p\in G}p}.  
]

若 product 大於 (|m-m_k|)，則 (m=m_k).  
否則 (m) has large size and diagonal energy across blocks grows.

This can be handled by the same single-block theorem specialised to diagonal labels. It is much easier than arbitrary assignment.

So ordinary high wrap is not invisible, but needs a small one-dimensional lemma. It should be manageable.

---

# 8. 完成後的 global counting

Combining Lemma 1 and Lemma 3 yields:

[  
#{a:Q(a)\le R}  
\le  
e^{\varepsilon R}  
#{m:Q_{\mathrm{diag}}(m)\le CR}.  
]

Then one-dimensional estimate:

[  
#{m:Q_{\mathrm{diag}}(m)\le CR}  
\ll \frac{\sqrt R}{\sigma}.  
]

Therefore

[  
#{a:Q(a)\le R}  
\le  
e^{\varepsilon R}\frac{\sqrt R}{\sigma}.  
]

And hence

[  
\sum_a e^{-cQ(a)}  
\ll  
\frac1\sigma.  
]

Minor arc outside (|m|\le C/\sigma) gives (o(1/\sigma)).

---

# 9. Remaining final small lemma

The last technical lemma is now:

**Ordinary diagonal counting lemma.**

[  
#{m\bmod L:Q_{\mathrm{diag}}(m)\le R}  
\ll  
\frac{\sqrt R}{\sigma}.  
]

This is one-dimensional and should follow from internal complete blocks plus divisor/product rigidity.

Compared with the original problem, this is much smaller.
