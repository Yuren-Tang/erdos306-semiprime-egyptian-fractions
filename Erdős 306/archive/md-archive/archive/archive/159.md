# Cross-label divisor-energy lemma

## 設定

令

[  
P\subset[X,2X]  
]

是一個 prime block。取兩個 subsets

[  
A,B\subset P.  
]

取兩個不同 ordinary labels

[  
m\ne m',  
\qquad |m|,|m'|\le B_0.  
]

對

[  
p\in A,\qquad q\in B  
]

令

[  
H_{pq}=H_{pq}^{m,m'}  
]

為對稱代表，滿足

[  
H_{pq}\equiv m\pmod p,  
\qquad  
H_{pq}\equiv m'\pmod q,  
]

且

[  
|H_{pq}|\le pq/2.  
]

我們要估計

# [  
E(A,B;m,m')

\sum_{p\in A,q\in B}  
\left(\frac{H_{pq}}{pq}\right)^2.  
]

因 (p,q\sim X)，有

[  
pq\asymp X^2,  
]

所以

[  
E(A,B;m,m')  
\asymp  
\frac1{X^4}  
\sum_{p\in A,q\in B}H_{pq}^2.  
]

---

# 1. 小 representative 的 incidence 上界

定義

[  
N(T)=#{(p,q)\in A\times B: |H_{pq}|\le T}.  
]

若

[  
|H_{pq}|\le T,  
]

則存在整數 (n)，(|n|\le T)，使

[  
n\equiv m\pmod p,\qquad n\equiv m'\pmod q.  
]

因此

[  
p\mid n-m,\qquad q\mid n-m'.  
]

所以

[  
N(T)  
\le  
\sum_{|n|\le T}  
\omega_A(n-m)\omega_B(n-m'),  
]

其中

[  
\omega_A(t)=#{p\in A:p\mid t}.  
]

對非零 (t)，有粗略 divisor bound：

[  
\omega_A(t)\le  
1+\frac{\log |t|}{\log X}.  
]

更準確地，若 (t\ne0)，因每個 (p\in[X,2X]) 都至少為 (X)，故

[  
X^{\omega_A(t)}\le |t|,  
]

所以

[  
\omega_A(t)\le \frac{\log |t|}{\log X}.  
]

若 (t=0)，則所有 (p\in A) 都整除 (t)，這是 special case。

因此，除 (n=m) 或 (n=m') 外，若

[  
|n|\le T,\qquad |m|,|m'|\le B_0,  
]

則

[  
|n-m|,\ |n-m'|\le T+B_0.  
]

令

[  
L_X=1+\frac{\log(T+B_0+2)}{\log X}.  
]

則

[  
\omega_A(n-m),\omega_B(n-m')\ll L_X.  
]

所以 ordinary part 給：

[  
\sum_{\substack{|n|\le T\ n\ne m,m'}}  
\omega_A(n-m)\omega_B(n-m')  
\ll  
T L_X^2.  
]

special terms：

- 若 (n=m)，則 (p\mid0) 對所有 (p\in A) 成立，但還需  
    [  
    q\mid m-m'=-D.  
    ]  
    所以 contribution 至多  
    [  
    |A|\omega_B(D).  
    ]
    
- 若 (n=m')，則 contribution 至多  
    [  
    |B|\omega_A(D).  
    ]
    

其中

[  
D=m'-m\ne0.  
]

又

[  
|D|\le2B_0,  
]

故

[  
\omega_A(D),\omega_B(D)  
\ll  
1+\frac{\log B_0}{\log X}  
\ll L_X.  
]

因此

[  
\boxed{  
N(T)  
\ll  
T L_X^2+(|A|+|B|)L_X.  
}  
]

這是 incidence 上界。

---

# 2. 由 incidence 上界取 quantile

令

[  
M=|A||B|.  
]

我們希望找到一個 (T_*)，使

[  
N(T_*)\le \frac M2.  
]

若能做到，則至少 (M/2) 條 cross edges 有

[  
|H_{pq}|>T_*.  
]

由上界

[  
N(T)\ll T L_X^2+(|A|+|B|)L_X,  
]

只要

[  
(|A|+|B|)L_X\le cM  
]

並取

[  
T_*=  
c\frac{M}{L_X^2},  
]

就有

[  
N(T_*)\le M/2.  
]

若

[  
(|A|+|B|)L_X>cM,  
]

則表示至少一側很小，例如

[  
\min(|A|,|B|)\ll L_X.  
]

這種情形在 single-block counting 中會被當作 small exception class；它的 entropy 可以直接支付，不需要強 cross-energy。

所以對 substantial classes，我們可取

[  
T_*\asymp \min\left(\frac{M}{L_X^2},,X^2\right).  
]

---

# 3. Quantile 推出平方能量下界

至少 (M/2) 條邊滿足

[  
|H_{pq}|>T_*.  
]

因此

[  
\sum_{p,q}H_{pq}^2  
\ge  
\frac M2 T_*^2.  
]

除以 (X^4)，得

[  
E(A,B;m,m')  
\gg  
M\left(\frac{T_*}{X^2}\right)^2.  
]

所以

[  
\boxed{  
E(A,B;m,m')  
\gg  
M\min\left(  
1,  
\frac{M^2}{X^4L_X^4}  
\right).  
}  
]

這就是 divisor-energy lemma 的基本形式。

---

# 4. 檢查量級

若

[  
|A|,|B|\asymp N\asymp\frac X{\log X},  
]

則

[  
M\asymp \frac{X^2}{\log^2 X}.  
]

因此

[  
\frac{M^2}{X^4L_X^4}  
\asymp  
\frac1{\log^4X,L_X^4}.  
]

所以

[  
E(A,B;m,m')  
\gg  
\frac{X^2}{\log^{6+O(1)}X}.  
]

這遠大於描述一整個 block 的 entropy

[  
\asymp X.  
]

因為

[  
\frac{X^2}{\log^{C}X}\gg X.  
]

所以兩個 substantial labels 不可能在低能層共存。

---

# 5. 小 class 情形

如果某個 label class (A) 很小，divisor-energy lower bound可能不強。但小 class 本身 entropy 也小。

設

[  
|A|=t,\qquad B=G  
]

為 dominant 或 large background class，(|G|\asymp N)。

則

[  
M=tN.  
]

若 (t) 不太小，剛才 energy lower bound給：

[  
E(A,G)\gg  
tN\min\left(1,\frac{t^2N^2}{X^4L^4}\right).  
]

這對很小 (t) 可能弱。  
但當 (t) 很小時，可以用更直接的 Irving / full-majority correction：每個 exception vertex (q\in A) 與整個 majority (G) 相連。若 (G) 不是任意 subset而是幾乎整個 (P)，則可以用 pruned full-block Irving，因為 (G) 缺少的只是少量點；刪掉少量點不破壞 lower bound：

[  
\sum_{p\in G}  
\left|  
\frac{(m_q-m)\bar p}{q}  
\right|^2  
\gg |P|  
]

只要 (G) 是 (1-o(1)) dense subset。  
所以每個 exception 成本

[  
\gg N,  
]

足以支付

[  
\log X  
]

entropy。

因此：

- substantial non-dominant classes：用 divisor-energy；
    
- small exceptions after a dominant class exists：用 majority Irving correction。
    

這分工解決小 class 問題。

---

# 6. 回到 single-block counting

對 (Q_P(a)\le R)：

1. 得到短 base list (\mathcal L)；
    
2. 若沒有 dominant class，則至少兩個 substantial classes coexist；
    
3. divisor-energy lemma給出巨大 energy，超過 (R) 或支付 multi-label entropy；
    
4. 因而低能 configurations 只有 dominant label (m) + exceptions；
    
5. exceptions 由 majority Irving correction支付；
    
6. label choices 為  
    [  
    O(B_R)=O(X\log X\sqrt R)=O(\sqrt R/\sigma_P).  
    ]
    

因此

[  
#{a:Q_P(a)\le R}  
\le  
e^{\varepsilon R}  
\frac{\sqrt R}{\sigma_P}.  
]

---

# 7. 這個證明目前最需嚴格化的點

最主要是第 2–3 步的 combinatorial bookkeeping：

從「無 dominant class」推出「有 substantial cross-label pairs enough to trigger divisor-energy」。  
這可通過 dyadic decomposition 寫成標準形式：

- 令 class sizes (t_j)；
    
- 若最大 class (< (1-\rho)N)，則  
    [  
    \sum_{i<j}t_it_j\gg \rho N^2.  
    ]
    
- 去掉 (t_j\le L_X) 的 tiny classes；tiny classes總 entropy直接處理。
    
- 剩下 substantial cross pairs套 divisor-energy。
    
- 若 substantial part不足，則大部分 non-dominant mass在 tiny classes，但 tiny classes 的總頂點數可作 exceptions，與 dominant class cross，用 majority correction付款。
    

這是可寫的，不再有概念障礙。
