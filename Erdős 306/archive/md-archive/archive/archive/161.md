# Global partition theorem

設全部 primes 分成 blocks：

[  
\mathcal P=\bigsqcup_k P_k.  
]

邊集合 (E) 分為：

[  
E=E_{\mathrm{int}}\sqcup E_{\mathrm{cross}},  
]

其中：

[  
E_{\mathrm{int}}=\bigcup_k \binom{P_k}{2}  
]

是同尺度 internal complete graph；

[  
E_{\mathrm{cross}}  
]

是跨尺度 semiprime edges，用於質量與 label 合併。

對 residue assignment

[  
a=(a_p)_{p\in\mathcal P}  
]

定義總能量：

[  
Q(a)=Q_{\mathrm{int}}(a)+Q_{\mathrm{cross}}(a).  
]

我們要證：

[  
\boxed{  
\sum_a e^{-cQ(a)}  
\ll  
\frac1\sigma,  
}  
]

其中

[  
\sigma^2=\sum_{pq\in E}\frac1{p^2q^2}.  
]

更強地，能量層 counting 為：

[  
\boxed{  
#{a:Q(a)\le R}  
\le  
e^{\varepsilon R}  
\left(1+\frac{\sqrt R}{\sigma}\right).  
}  
]

---

# 1. 先按 block internal energy 分層

對每個 block (P_k)，令

[  
Q_k(a)=Q_{P_k}^{\mathrm{int}}(a|_{P_k}).  
]

若總能量

[  
Q(a)\le R,  
]

則

[  
\sum_k Q_k(a)\le R.  
]

對每個 (k)，套 single-block counting theorem：

給定內部能量層 (R_k)，block (P_k) 的 residues 可由：

- 一個 ordinary block label (m_k)；
    
- 少量 exceptions；
    

編碼，且計數代價為：

[  
e^{\varepsilon R_k}  
\left(1+\frac{\sqrt{R_k}}{\sigma_k}\right),  
]

其中

[  
\sigma_k^2=  
\sum_{p<q\in P_k}\frac1{p^2q^2}.  
]

但如果直接把所有 block 的 label choices 相乘，會得到太大；下一步 cross energy 將強制這些 (m_k) 合併。

所以應該把全局 counting 寫成：

[  
#{a:Q(a)\le R}  
\le  
e^{\varepsilon R}  
\sum_{\substack{(m_k)\ \mathcal Q(m_\bullet)\le CR}}  
1,  
]

其中 (\mathcal Q(m_\bullet)) 是由 block labels 產生的 effective label energy：

# [  
\mathcal Q(m_\bullet)

\sum_k Q_k^{\mathrm{diag}}(m_k)  
+  
\sum_{k\ell}Q_{k\ell}^{\mathrm{cross}}(m_k,m_\ell).  
]

這一步的意思是：block 內 exceptions 的 entropy 已由 (e^{\varepsilon R}) 支付，剩下的自由度是 block labels (m_k)。

---

# 2. Effective block-label counting

現在要證：

[  
#{(m_k):\mathcal Q(m_\bullet)\le R}  
\le  
e^{\varepsilon R}  
\left(1+\frac{\sqrt R}{\sigma}\right).  
]

這是 block-label Peierls counting。

若所有

[  
m_k=m  
]

相同，則

[  
\mathcal Q(m_\bullet)=Q_{\mathrm{diag}}(m)  
]

且

[  
Q_{\mathrm{diag}}(m)\asymp m^2\sigma^2  
]

在主 range 中。所以 pure labels 數量為

[  
\ll 1+\frac{\sqrt R}{\sigma}.  
]

若不是全相同，則存在 label boundary。

---

# 3. Boundary cost between different block labels

若 compatible blocks (P_k,P_\ell) labels 不同：

[  
m_k\ne m_\ell,  
]

則由 Lemma II：

[  
Q_{k\ell}^{\mathrm{cross}}(m_k,m_\ell)  
+  
Q_{k\ell}^{\mathrm{diag}}(m_k)  
\gg  
|P_k||P_\ell|  
]

up to harmless divisor exceptions.

因此一條 genuine label boundary 的成本是：

[  
\mathrm{Cost}(k,\ell)\asymp |P_k||P_\ell|.  
]

這通常遠大於：

- 選擇 boundary 位置的 entropy；
    
- 選擇新 label 的 entropy；
    
- 選擇一段 droplet 的 entropy。
    

---

# 4. Peierls encoding of label configurations

任取一個 reference global label (m)，例如佔據最低 internal energy或最大 block mass的 label。

定義 defect blocks：

[  
D={k:m_k\ne m}.  
]

若 (D=\varnothing)，是 pure phase。

若 (D\ne\varnothing)，則 (D) 分解為 connected components / intervals：

[  
C_1,C_2,\dots.  
]

每個 component 的 boundary 至少有一條 cross block edge 連到外部 label (m)，除非它是整個 block graph；若是整個 graph，則其 label應被視為新的 global label，而不是 defect。

每個 proper component付 boundary cost：

[  
\gg \sum_{(k,\ell)\in\partial C}|P_k||P_\ell|.  
]

該 component 內部 label 可以是某個 (m')，或者再有 nested labels；nested structure 由同樣 boundary cost 遞歸支付。

因 block graph 可以選為 bounded-degree skeleton 加 full graph補強，component shape entropy 至多：

[  
\exp(O(|\partial C|\log K)).  
]

而每個 boundary edge成本

[  
|P_k||P_\ell|\to\infty  
]

遠大於 (\log K)。

Label choice entropy：

若 (m') 很大，component internal diagonal energy已大；若 (m') 不大，其可選數量的 logarithm也遠小於 boundary cost。更精確地，用 energy-layer counting：

[  
#{m':Q_C^{\mathrm{diag}}(m')\le S}  
\ll  
1+\frac{\sqrt S}{\sigma_C},  
]

而 boundary factor (e^{-c\partial C}) 使所有 proper components 可求和。

所以 Peierls expansion converges：

[  
\sum_{\text{proper droplets}}  
e^{-c\mathrm{Cost}}  
\cdot  
\text{entropy}  
\le C.  
]

---

# 5. Result of block-label Peierls

因此：

[  
\sum_{(m_k)}e^{-c\mathcal Q(m_\bullet)}  
\le  
C  
\sum_m e^{-c'Q_{\mathrm{diag}}(m)}.  
]

Equivalent counting form：

[  
#{(m_k):\mathcal Q(m_\bullet)\le R}  
\le  
e^{\varepsilon R}  
#{m:Q_{\mathrm{diag}}(m)\le CR}.  
]

再由 ordinary diagonal counting：

[  
#{m:Q_{\mathrm{diag}}(m)\le CR}  
\ll  
1+\frac{\sqrt R}{\sigma}.  
]

得：

[  
#{(m_k):\mathcal Q(m_\bullet)\le R}  
\le  
e^{\varepsilon R}  
\left(1+\frac{\sqrt R}{\sigma}\right).  
]

# 6. Combine with block-local compression

回到 original residues (a)。每個 block 的 deviations from (m_k) 已由 single-block theorem 支付，給出總 factor：

[  
e^{\varepsilon \sum_k R_k}  
\le  
e^{\varepsilon R}.  
]

block labels 的 counting 給：

[  
e^{\varepsilon R}  
\left(1+\frac{\sqrt R}{\sigma}\right).  
]

所以整體：

[  
\boxed{  
#{a:Q(a)\le R}  
\le  
e^{\varepsilon R}  
\left(1+\frac{\sqrt R}{\sigma}\right).  
}  
]

這就是 global counting theorem。

# 7. Partition bound

由 dyadic / integer energy layers：

# [  
\sum_a e^{-cQ(a)}

\sum_{j\ge0}  
\sum_{\substack{a\j\le Q(a)<j+1}}  
e^{-cQ(a)}  
]

[  
\le  
\sum_{j\ge0}  
e^{-cj}  
#{a:Q(a)\le j+1}.  
]

代入 counting：

[  
\le  
\sum_{j\ge0}  
e^{-cj}  
e^{\varepsilon(j+1)}  
\left(1+\frac{\sqrt{j+1}}{\sigma}\right).  
]

取

[  
\varepsilon<c/2,  
]

則

[  
\sum_a e^{-cQ(a)}  
\ll  
1+\frac1\sigma.  
]

因 (\sigma\to0)，得

[  
\boxed{  
\sum_a e^{-cQ(a)}  
\ll  
\frac1\sigma.  
}  
]

# 8. Minor arc outside main labels

主弧為：

[  
\mathfrak M={a:a_p\equiv m\pmod p,\ |m|\le C/\sigma}.  
]

對非主弧 assignment，global counting theorem 給 dominant global label (m) 或 energy-paid deviations。

若 dominant global label滿足

[  
|m|>C/\sigma,  
]

則 Gaussian diagonal energy

[  
Q_{\mathrm{diag}}(m)\gg C^2.  
]

取 (C\to\infty) slowly 或先固定大 (C)，可令 tail

[  
\sum_{|m|>C/\sigma}  
e^{-cQ_{\mathrm{diag}}(m)}  
=o(1/\sigma).  
]

Deviations around such (m) are paid by Peierls/exceptions and do not change order.

因此：

[  
\boxed{  
\sum_{a\notin\mathfrak M}e^{-cQ(a)}  
=o(1/\sigma).  
}  
]

這是 Fourier 小弧需要的估計。

---

# 9. 檢查不重複計費

這裏最需小心的是：internal energy與cross energy分別使用，不能同一份能量支付兩次。

我們的分工是：

- (Q_{\mathrm{int}})：用於每個 block 產生 (m_k) 並支付 block-level exceptions。
    
- (Q_{\mathrm{cross}})：用於合併 (m_k) 並支付 cross-block droplets。
    

因

[  
Q=Q_{\mathrm{int}}+Q_{\mathrm{cross}},  
]

兩者 disjoint，沒有重複計費。

在 single-block theorem內部，exception payment使用 internal edges；在 block-label Peierls中，boundary payment使用 cross edges。這是乾淨的。
