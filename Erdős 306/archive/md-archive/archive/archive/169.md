# 修正版定理

**Theorem.**  
設

[  
b  
]

為 squarefree 正整數。則對任意正整數 (a)，存在有限個互異 squarefree semiprimes

[  
n_i=p_iq_i  
]

使

[  
\frac ab=\sum_i\frac1{n_i}.  
]

這是目前能成立的正確版本。若 (b) 非 squarefree，則此命題本身不可能成立，因為 squarefree denominators 的有限和之既約分母必 squarefree。

# 1. Lattice gadget 的目的

令

[  
b=\prod_{r\in B}r  
]

squarefree。

主體大素數集合記為

[  
\mathcal P_{\mathrm{big}}.  
]

為了讓

[  
1/b  
]

落在 denominator lattice 中，我們需要所有 (r\mid b) 出現在 lcm 中。因此令總 prime set

[  
\mathcal P=B\sqcup \mathcal P_{\mathrm{big}}.  
]

總 lcm 為

[  
L=\prod_{p\in\mathcal P}p.  
]

則

[  
b\mid L,  
]

目標整數為

[  
L/b.  
]

若 edge set (E) 的 weights

[  
w_{pq}=L/(pq)  
]

生成 full lattice，即

[  
\gcd_{pq\in E}w_{pq}=1,  
]

則沒有 gcd obstruction。

---

# 2. 如何保證 gcd 為 1？

對每個 prime

[  
r\in\mathcal P  
]

只要存在某條 edge

[  
rs\in E  
]

含有 (r)，則

[  
w_{rs}=L/(rs)  
]

不含因子 (r)。

所以所有 weights 的 gcd 不含 (r)。若每個 (r\in\mathcal P) 都出現在至少一條 edge，則

[  
\gcd_{pq\in E}w_{pq}=1.  
]

主體大 blocks 的 internal/cross edges自然覆蓋所有大 primes。  
對小 primes (r\mid b)，加入 gadget edges：

[  
r s_r  
]

其中 (s_r\in\mathcal P_{\mathrm{big}}) 是未用或專門保留的大 prime。

這樣每個 (r\mid b) 也出現在 edge 中。故 gcd obstruction 消失。

---

# 3. gadget edges 對分析的影響

Gadget edges 數量有限，至多

[  
\omega(b)  
]

或 (O_b(1))。它們的 reciprocals 是

[  
\frac1{rs_r}  
]

其中 (s_r) 可取極大，因此總 mass 可以做得任意小：

[  
\sum_{r\mid b}\frac1{rs_r}\ll \varepsilon.  
]

它們對 variance 也很小：

[  
\sum_{r\mid b}\frac1{r^2s_r^2}\ll\varepsilon.  
]

所以它們不改變主弧 Gaussian scale，也不影響 minor arc 的核心 estimates。

Fourier 端，它們只增加有限個因子：

[  
1-\theta+\theta e(h/(rs_r)).  
]

這些因子模長 (\le1)，所以對小弧 upper bound 不會造成麻煩；它們可以完全忽略於 damping estimate 中。

# 4. 是否需要 gadget edges 參與隨機選擇？

有兩種方式。

## 方式 A：把 gadget edges 放入 Bernoulli 模型

令 gadget edges 也有 Bernoulli 參數

[  
\theta_g\in[c,1-c].  
]

它們參與 Fourier inversion，但因數量有限，只影響常數。  
Mass tuning 時把它們的期望一起計入：

# [  
\mathbb E X

# \sum_{e\in E_{\mathrm{main}}}\frac{\theta_e}{e}  
+  
\sum_{g\in E_{\mathrm{gad}}}\frac{\theta_g}{g}

\frac1b.  
]

這最簡單。

## 方式 B：先枚舉 gadget choices

Gadget subset 有有限多種選擇。對每一種選擇，目標變成

[  
\frac1b-\sum_{g\in G}\frac1g.  
]

再用主體大 blocks 表示剩餘。  
這樣可以把 gadget 從 Fourier 分析中剝離，但記號稍繁。

方式 A 足夠。

# 5. 小 primes 與 CRT character space

包含小 primes (r\mid b) 後，character assignment 多了 residues

[  
a_r\in\mathbb F_r.  
]

這些小 primes 不在大 block Poincaré 中。會不會產生額外 minor arc entropy？

因為它們只有有限個，所以總 character 數多一個常數因子：

[  
\prod_{r\mid b}r=b.  
]

對固定 (b)，這是常數。  
即使 (b) 不是固定而是輸入參數，也可把依賴寫成 (O_b(1))，或者選大 blocks使主弧量 (1/\sigma) 遠大於這個有限因子。

更嚴格地，可以 condition on small-prime residues；對每個 small residue pattern，大-prime minor arc estimate一致成立，最後乘以 (b) 個 patterns。只要選 (\sigma) 足夠小，使主弧

[  
1/\sigma  
]

遠大於 (b)-dependent constants即可。

這沒有本質問題。

# 6. Full lattice span statement

**Lemma.**  
Assume (b) is squarefree. Let (\mathcal P\supseteq{r:r\mid b}), (L=\prod_{p\in\mathcal P}p). If every (p\in\mathcal P) is incident to at least one edge (pq\in E), then

[  
\gcd\left{\frac{L}{pq}:pq\in E\right}=1.  
]

**Proof.**  
Let (r\in\mathcal P). Since (r) is incident to some edge (rs\in E),

[  
\frac{L}{rs}  
]

is not divisible by (r). Hence (r) does not divide the gcd of all weights. Since every prime divisor of (L) is in (\mathcal P), no prime divides the gcd. Thus gcd is 1.

Therefore the support of (Y=LX) generates the full additive lattice in (\mathbb Z), and Fourier inversion on (\mathbb Z/L\mathbb Z) has no hidden period.

# 7. Reachability of (L/b)

gcd 1 ensures the group generated by weights is all of (\mathbb Z/L\mathbb Z). But Bernoulli variables are (0/1), so the set of subset sums is not automatically all residues. Fourier positivity is stronger: if we prove

[  
\mathbb P(Y=L/b)>0,  
]

then reachability follows.  
The lattice span lemma only ensures no Fourier obstruction forces this probability to be zero.

So span condition is exactly what we need.

# 8. Final theorem now consistent

With (b) squarefree:

1. (b\mid L), so (L/b) is an integer.
    
2. Gadget edges ensure full lattice span.
    
3. Main big-prime edges supply mass, variance, and minor arc damping.
    
4. Fourier positivity gives a subset sum equal to (1/b).
    

Thus lattice issue is resolved.
