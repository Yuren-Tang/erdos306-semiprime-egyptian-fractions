# Irving-good pruning lemma：所需形式

令

[  
P\subset[X,2X],\qquad Q\subset[Y,2Y]  
]

為兩個 prime blocks，且尺度 compatible，例如

[  
X\asymp Y  
]

或更一般處在 Irving 估計可用的範圍內。

我們需要一個 pruning 後的版本：

存在子集

[  
P^\ast\subset P,\qquad Q^\ast\subset Q  
]

滿足

[  
|P^\ast|=(1-o(1))|P|,  
\qquad  
|Q^\ast|=(1-o(1))|Q|,  
]

使得對每個

[  
q\in Q^\ast  
]

與每個非零

[  
d\in\mathbb F_q^\times  
]

都有

[  
\sum_{p\in P^\ast}  
\left|  
\frac{d\bar p}{q}  
\right|^2  
\gg |P^\ast|.  
]

同理也需要對調方向：

對每個

[  
p\in P^\ast,\quad d\in\mathbb F_p^\times  
]

有

[  
\sum_{q\in Q^\ast}  
\left|  
\frac{d\bar q}{p}  
\right|^2  
\gg |Q^\ast|.  
]

這就是我們在 single-block exceptions、cross-block label mismatch、labelled-boundary transfer 中使用的核心估計。

---

# 1. 為什麼這個估計足夠？

若某一側已有 label (m)，另一個 vertex (q) 不服從它，即

[  
a_q\not\equiv m\pmod q,  
]

令

[  
d=a_q-m\ne0\pmod q.  
]

對 (p\in P^\ast)，相位為

# [  
\phi_{pq}

\frac{m}{pq}  
+  
\frac{d\bar p}{q}  
\pmod1.  
]

由不等式

[  
|x+y|^2+|x|^2\gg |y|^2  
]

得

[  
\sum_{p\in P^\ast}|\phi_{pq}|^2  
+  
\sum_{p\in P^\ast}  
\left|  
\frac{m}{pq}  
\right|^2  
\gg  
\sum_{p\in P^\ast}  
\left|  
\frac{d\bar p}{q}  
\right|^2  
\gg |P^\ast|.  
]

所以一個 wrong vertex 成本為

[  
\gg |P^\ast|.  
]

這正是我們所有 Peierls / exception payment 的基石。

---

# 2. 同尺度 internal block 也需要此估計

對同一 block

[  
P\subset[X,2X],  
]

internal edges 是

[  
p q,\qquad p,q\in P.  
]

我們需要在 pruning 後：

對每個

[  
q\in P^\ast,\quad d\ne0\pmod q,  
]

有

[  
\sum_{\substack{p\in P^\ast\p\ne q}}  
\left|  
\frac{d\bar p}{q}  
\right|^2  
\gg |P^\ast|.  
]

這只是上面 (P=Q) 的同尺度版本，去掉 (p=q) 一個點不影響。

這保證 single-block theorem 中 majority Irving correction 成立。

---

# 3. 對 subsets 的使用如何合法？

前面有時對 majority subset (G\subset P^\ast) 使用 Irving。嚴格來說，Irving-good lemma 給的是對整個 (P^\ast) 的求和：

[  
\sum_{p\in P^\ast}\left|\frac{d\bar p}{q}\right|^2\gg |P^\ast|.  
]

若 (G) 是 (1-\rho) dense subset：

[  
|P^\ast\setminus G|\le \rho |P^\ast|,  
]

則

## [  
\sum_{p\in G}\left|\frac{d\bar p}{q}\right|^2  
\ge  
\sum_{p\in P^\ast}\left|\frac{d\bar p}{q}\right|^2

|P^\ast\setminus G|\cdot\frac14.  
]

因每項最多 (1/4)。若 Irving 常數為 (c_0|P^\ast|)，取

[  
\rho<c_0/2  
]

即可得

[  
\sum_{p\in G}\left|\frac{d\bar p}{q}\right|^2  
\gg |P^\ast|.  
]

所以我們只能在 (G) 是 sufficiently dense majority 時直接使用 Irving。  
對 arbitrary subsets，我們沒有假設 Irving，這正是我們引入 divisor-energy lemma 的原因。

---

# 4. 如何 pruning？

形式上可以這樣寫：

對每個 block pair ((P_k,P_\ell))，Irving 給出平均意義的 good estimate：

除 (o(|Q|)) 個 (q\in Q) 外，對所有 (d\ne0\pmod q) 有

[  
\sum_{p\in P}  
\left|  
\frac{d\bar p}{q}  
\right|^2  
\gg |P|.  
]

對所有需要的 block pairs 取 union bound，刪去所有在某個 pair 中 bad 的 primes。  
因 block graph 的 degree 可取 bounded 或 (O(\log K))，而每個 pair 的 bad set比例 (o(1))，最終每個 block 只刪去

[  
o(|P_k|)  
]

個 primes。

得到 pruned block：

[  
P_k^\ast.  
]

之後所有 internal / cross arguments 都在 (P_k^\ast) 上進行。

---

# 5. 若 block graph 很密怎麼辦？

若使用 full compatible graph，某個 block 可能與很多 blocks相連，union bound 可能累積太大。  
解法有兩種：

1. **只取 bounded-degree skeleton 作控制邊**  
    用 skeleton cross edges 來做 label merging；full graph 可保留作 mass，但小弧控制只需 skeleton + internal complete graph。  
    這樣每個 block只需對 (O(1)) 或 (O(\log K)) 個 pairs pruning。
    
2. **分批 pruning**  
    若 full graph degree較大，選參數使每 pair bad比例遠小於 degree(^{-1})，仍可 union bound。這需要更強 Irving uniformity，不如 skeleton乾淨。
    

最穩妥的方案是：

[  
\boxed{  
\text{internal complete edges + bounded-degree cross skeleton for control；額外 cross edges只用於 mass。}  
}  
]

若額外 mass edges 不參與小弧控制，會不會影響 Fourier？  
它們只會增加 energy (Q)，因此小弧 upper bound更強。即使我們在 partition theorem 中忽略它們，也有

[  
Q_{\mathrm{full}}\ge Q_{\mathrm{control}}.  
]

所以控制 skeleton 足夠。

---

# 6. Control edge set

定義

# [  
E_{\mathrm{ctrl}}

E_{\mathrm{int}}\cup E_{\mathrm{skel}}.  
]

其中：

- (E_{\mathrm{int}})：每個 block 的 internal complete graph；
    
- (E_{\mathrm{skel}})：bounded-degree compatible block graph，例如每個 block連向若干相鄰或 (4/3)-jump blocks。
    

Mass edge set

[  
E_{\mathrm{mass}}  
]

可以更大，用來調整期望與方差。

總 edge set：

[  
E=E_{\mathrm{ctrl}}\cup E_{\mathrm{mass}}\cup E_{\mathrm{gad}}.  
]

小弧中：

[  
Q_E(a)\ge Q_{\mathrm{ctrl}}(a).  
]

所以 partition bound 只需證：

[  
\sum_a e^{-cQ_{\mathrm{ctrl}}(a)}  
\ll1/\sigma_{\mathrm{ctrl}}?  
]

但 Fourier 主弧 scale 用的是總 variance

[  
\sigma_E^2.  
]

要小心：如果 mass edges大幅增加 (\sigma_E)，而 control edges 的 (\sigma_{\mathrm{ctrl}}) 太小，則 minor arc bound (1/\sigma_{\mathrm{ctrl}}) 可能不夠比主弧 (1/\sigma_E) 小。

因此需要確保 control edges 貢獻的 variance與總 variance同階：

[  
\sigma_{\mathrm{ctrl}}\asymp \sigma_E.  
]

這可以通過讓 mass edges也包含足夠的 control-like skeleton edges，或選擇 mass edges不使 variance遠超 internal/control variance。

注意 reciprocal mass和variance不同：mass edges可多但在較大尺度，對 mass貢獻可顯著，對 variance貢獻可能小。  
所以可安排：

- control edges控制 (\sigma)；
    
- mass edges在更高尺度調 mass，對 (\sigma) 影響較小。
    

這需要 bookkeeping，但可行。

# 7. Irving-good pruning lemma：最終表述

**Lemma.**  
For each finite bounded-degree family of compatible prime block pairs ((P_k,P_\ell)), after removing (o(|P_k|)) primes from each block, the remaining (P_k^\ast) satisfy:

For every control pair ((k,\ell)), every (q\in P_\ell^\ast), and every (d\in\mathbb F_q^\times),

[  
\sum_{p\in P_k^\ast}  
\left|  
\frac{d\bar p}{q}  
\right|^2  
\ge c|P_k^\ast|.  
]

And symmetrically in (k,\ell).  
Also for internal pairs (P_k^\ast) with itself,

[  
\sum_{\substack{p\in P_k^\ast\p\ne q}}  
\left|  
\frac{d\bar p}{q}  
\right|^2  
\ge c|P_k^\ast|.  
]

This is the only external equidistribution input used.
