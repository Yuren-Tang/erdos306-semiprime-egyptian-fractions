# 一、已基本證成，可整理成正式引理

## 1. Cross-label divisor-energy lemma

這個我們其實已經證得比較完整了。

命題是：給定兩組 primes

[  
A,B\subset[X,2X],  
]

兩個不同 labels

[  
m\ne m',  
]

對 cross CRT representatives (H_{pq}^{m,m'})，有小 representative 計數：

# [  
N(T)

#{(p,q): |H_{pq}^{m,m'}|\le T}  
\ll  
T L_X^2+(|A|+|B|)L_X,  
]

其中

[  
L_X=1+\frac{\log(B_0+X^2+2)}{\log X}.  
]

再由 quantile argument 得：

[  
\sum_{p\in A,q\in B}  
\left(\frac{H_{pq}^{m,m'}}{pq}\right)^2  
\gg  
M\min\left(1,\frac{M^2}{X^4L_X^4}\right),  
\qquad M=|A||B|.  
]

這一引理已經有完整證明：  
小 (H) 對應到

[  
p\mid n-m,\qquad q\mid n-m',  
]

然後用 dyadic prime divisor counting。special terms (n=m,m') 也已處理。

這個可以直接整理成正式 Lemma。

## 2. Lattice span lemma

這個也已經證完。

若

[  
b  
]

squarefree，取

[  
L=\prod_{p\in\mathcal P}p  
]

且 (b\mid L)。若每個 (p\in\mathcal P) 都 incident to some edge (pq\in E)，則

[  
\gcd_{pq\in E}\frac{L}{pq}=1.  
]

證明很短：對每個 prime (r\in\mathcal P)，取 edge (rs)，則 (L/(rs)) 不含 (r)，所以 (r) 不整除全體 gcd。

這可直接收正文。

## 3. Squarefree denominator obstruction

這也已經完全清楚。

若所有 denominators 都 squarefree，則 lcm squarefree；有限和的既約分母整除 squarefree lcm，所以也 squarefree。故任意 (b) 的版本不可能，必須改成 squarefree (b)。

這應放在定理前或 remark 中。

## 4. Mass/variance architecture

這部分邏輯已清楚：

- internal edges 的 mass：
    

[  
\sum_k \frac1{k^2}  
]

收斂；

- internal variance：
    

[  
\sum_k \frac1{2^{2k}k^2}  
]

由最低尺度主導；

- 高尺度 cross mass edges 可提供 mass 而幾乎不增加 variance；
    
- 安排
    

[  
\sigma_{\mathrm{ctrl}}\asymp\sigma_{\mathrm{total}}.  
]

這部分需要寫得稍微嚴格一些，但沒有核心難點。

# 二、已有證明骨架，但正式稿需補細節

## 1. Single-block counting theorem

這是最重要的核心引理。骨架已經很完整：

[  
#{a:Q_P(a)\le R}  
\le  
e^{\varepsilon R}  
\left(1+\frac{\sqrt R}{\sigma_P}\right).  
]

已完成的主要步驟：

1. low energy (\Rightarrow) dense good graph；
    
2. regular base vertex (\Rightarrow) short base list；
    
3. vertices 被 base list 覆蓋；
    
4. 若有 dominant label，exceptions 由 Irving correction 付費；
    
5. 若無 dominant label，用 cross-label divisor-energy 壓多 label 共存；
    
6. dominant label 數量是
    

# [  
O(X\log X\sqrt R)

O(\sqrt R/\sigma_P).  
]

還需補寫的地方是第 5 步的 dyadic bookkeeping。不是新想法，而是要正式寫：

- 按 class size 分層；
    
- tiny classes 作 exceptions；
    
- substantial classes 用 cross-label divisor-energy；
    
- cross energy 如何支付 label assignment entropy；
    
- sum over dyadic layers。
    

這是目前最值得花力氣寫成正式文字的地方。

## 2. Block-label Peierls collapse

骨架清楚：

single-block theorem 把每個 block 壓成 ordinary label (m_k)。  
若

[  
m_k\ne m_\ell,  
]

cross-block Irving 給巨大 boundary cost：

[  
\gg |P_k||P_\ell|.  
]

所以 label droplets 的 entropy 被 boundary energy 支付，最後所有 (m_k) collapse to one global (m)。

需補正式細節：

- 選 bounded-degree skeleton；
    
- 定義 connected components / droplets；
    
- boundary cost 下界；
    
- droplet labels 的 counting；
    
- Peierls sum convergence。
    

這比 single-block bookkeeping 簡單。

## 3. Ordinary diagonal counting

骨架也清楚：

[  
m\bmod L  
]

若低能，則在 each internal block 上接近某個 local representative (m_k)；cross-block energy 迫使 (m_k) 合併；product rigidity 將 (m) 限制到 global ordinary (m_0) 加 energy-paid exceptional primes。

需補：

- 如何從 internal block theorem 的 diagonal specialization 得 (m_k)；
    
- exception primes 的 encoding；
    
- product rigidity 的精確形式；
    
- 最終 counting：
    

[  
#{m:Q_{\mathrm{diag}}(m)\le R}  
\le  
e^{\varepsilon R}  
\left(1+\frac{\sqrt R}{\sigma}\right).  
]

這也不是新障礙。

# 三、仍是外部輸入，需引用或查明精確形式

## Irving-good pruning lemma

我們使用的形式是：

對 compatible prime blocks (P,Q)，刪去少量 bad primes 後，對所有

[  
q\in Q^\ast,\qquad d\in\mathbb F_q^\times  
]

有

[  
\sum_{p\in P^\ast}  
\left|  
\frac{d\bar p}{q}  
\right|^2  
\gg |P^\ast|.  
]

以及同尺度版本。

這個我們沒有自證，依賴 Irving 類結果。正式稿中必須：

1. 查清 Irving 原定理精確敘述；
    
2. 確認它給的是 pointwise-in-(d) 還是 average-in-(d)；
    
3. 確認 pruning 後是否能 uniform over all (d\ne0)；
    
4. 確認尺度範圍 (X,Y)；
    
5. 若只能得到 weaker form，要調整 single-block / cross-block proof。
    

這是目前唯一真正需要外部核驗的點。

# 四、現在應做什麼？

不必全部再寫一遍。最合理的順序是：

1. **先固定外部 Irving lemma 的精確可用版本。**

3. **再寫 Block-label Peierls + ordinary diagonal counting。**
    
4. **最後合成 theorem proof。**
    

所以我建議下一步不是重複全部，而是先做 Irving lemma 的「依賴檢查」。也就是明確列出我們需要的最弱版本，然後看能否由已知 Irving 結果推出。

# 下一步：最弱 Irving 輸入

我們真正需要的其實不一定是最強 uniform 版本。最弱可用版本如下：

對每個 block (P\subset[X,2X])，經 pruning 後，對每個 (q\in P^\ast) 和 (d\ne0\pmod q)，

[  
\sum_{p\in P^\ast}  
\left|  
\frac{d\bar p}{q}  
\right|^2  
\ge c|P^\ast|.

對 cross-block (P,Q) 亦同。

這只用於：

- majority correction；
