---

# Multi-label entropy lemma

設 (\mathcal L) 是短 label list，

[  
|\mathcal L|=s,  
]

頂點集 (V\subset P) 被分成 classes

[  
V=\bigsqcup_{m\in\mathcal L} C_m.  
]

若沒有 dominant class：

[  
\max_m |C_m|<(1-\rho)|V|,  
]

則由不同 labels 間的 cross-label energy 可得：

[  
E_{\mathrm{cross}}({C_m})  
\ge  
\mathrm{Ent}({C_m})/!C  
]

或者更實用地：

[  
#{\text{multi-label configurations with energy }\le S}  
\le e^{\varepsilon S}.  
]

一旦這個 lemma 寫實，single-block theorem 就真正完整。

---

# 這個 lemma 怎麼證？

分三步。

## 1. 去掉 tiny classes

令

[  
L_X=1+\frac{\log(B+X^2+2)}{\log X}.  
]

稱 class tiny 若

[  
|C_m|<C_0L_X.  
]

tiny classes 中的頂點，要麼總數少，可作 exceptions；要麼總數多，則與任何大 class 的 cross energy 足以支付它們。這裏用 majority Irving correction。

## 2. substantial classes 之間有大量 cross mass

對 substantial classes，

[  
|C_m|\ge C_0L_X.  
]

若沒有 dominant label，則 substantial classes 之間的 cross pair 總量為正比例級，或者 tiny vertices 已經佔據非 dominant mass、回到 exception case。

所以有

[  
\sum_{\substack{m\ne m'\ C_m,C_{m'}\ \mathrm{sub}}}  
|C_m||C_{m'}|  
\gg_\rho N^2  
]

在真正的 multi-label case 中成立。

## 3. 用 cross-label divisor-energy lemma 逐 dyadic 層求和

對每一對 substantial classes，

[  
A=C_m,\qquad B=C_{m'},  
]

已有

[  
E(A,B;m,m')  
\gg  
M\min\left(1,\frac{M^2}{X^4L_X^4}\right),  
\qquad M=|A||B|.  
]

按 class size dyadic 分層：

[  
2^u\le |C_m|<2^{u+1}.  
]

在每一層中，class 數與頂點總數受控，cross energy 下界可和該層的配置熵比較。配置熵大致是：

[  
\ll N\log s+N  
]

而 substantial cross energy在非 dominant 情形下至少是

[  
\gg \frac{X^2}{(\log X)^C}  
]

或在較小層中仍逐頂點支付

[  
\gg \frac{N}{(\log X)^C}  
]

級別；這遠大於單頂點 entropy

[  
O(\log X+\log s).  
]

因此多 label 狀態總數可吸收到

[  
e^{\varepsilon R}  
]

裏。

所以接下來不是「再寫一遍 single-block theorem」，而是只正式寫這個 **Multi-label entropy lemma**。寫完後，single-block theorem 就可以用一句話引用：

> By base-list reduction and the multi-label entropy lemma, every low-energy block assignment has a dominant ordinary label, up to energy-paid exceptions.
