好。下面把 **Multi-label entropy lemma** 寫成可放入正文的版本。這個引理的作用是：在 single-block proof 中，base-list reduction 之後，若沒有一個 dominant label，則多 label 共存本身會造成足夠大的 cross energy；否則非 dominant 部分只能作為 energy-paid exceptions。

---

# Multi-label entropy lemma

令

[  
P\subset[X,2X],\qquad N=|P|\asymp X/\log X.  
]

設有一個短 label list

[  
\mathcal L\subset[-B,B]\cap\mathbb Z,\qquad |\mathcal L|=s,  
]

其中在 single-block 應用中

[  
s\ll 1+\log X\sqrt R.  
]

設大多數 vertices 已被指派到 labels：

[  
V=\bigsqcup_{m\in\mathcal L}C_m\subset P,  
]

且

[  
|V|\ge (1-\delta)N.  
]

對 (m\ne m')，定義 cross-label energy

# [  
E(C_m,C_{m'};m,m')

\sum_{p\in C_m,\ q\in C_{m'}}  
\left(\frac{H_{pq}^{m,m'}}{pq}\right)^2.  
]

其中 (H_{pq}^{m,m'}) 是滿足

[  
H_{pq}^{m,m'}\equiv m\pmod p,\qquad  
H_{pq}^{m,m'}\equiv m'\pmod q  
]

的對稱代表。

令

# [  
E_{\mathrm{ml}}

\sum_{m<m'}E(C_m,C_{m'};m,m').  
]

則對任意固定 (\rho>0)，若沒有 dominant class：

[  
\max_m |C_m|<(1-\rho)N,  
]

則以下二者至少有一個成立：

1. (E_{\mathrm{ml}}) 已大到足以支付整個 multi-label assignment 的 encoding entropy；
    
2. 所有非最大類 vertices 可作為 exception set，其選擇與 residues 由 majority-Irving correction 的能量支付。
    

在 counting form 中，可寫為：

[  
#{\text{multi-label assignments with total energy}\le S}  
\le e^{\varepsilon S}  
]

up to the eventual dominant label choice.

---

# 證明

## 1. Tiny / substantial classes

令

[  
L_X=1+\frac{\log(B+X^2+2)}{\log X}.  
]

取常數 (C_0) 充分大。稱 class (C_m) 為 substantial 若

[  
|C_m|\ge C_0L_X,  
]

否則稱 tiny。

記 substantial part 為

[  
V_{\mathrm{sub}}=\bigcup_{|C_m|\ge C_0L_X}C_m.  
]

tiny part 為

[  
V_{\mathrm{tiny}}=V\setminus V_{\mathrm{sub}}.  
]

我們分情形。

---

## 2. Substantial part 本身無 dominant

假設 substantial part 承載正比例質量，而且 substantial classes 中沒有一個接近全部 (V)。也就是存在常數 (c_\rho>0) 使

[  
\sum_{\substack{m\ne m'\ C_m,C_{m'}\ \mathrm{sub}}}  
|C_m||C_{m'}|  
\ge c_\rho N^2.  
]

對每一對 substantial classes

[  
A=C_m,\qquad B'=C_{m'},  
]

cross-label divisor-energy lemma 給

[  
E(A,B';m,m')  
\gg  
M\min\left(1,\frac{M^2}{X^4L_X^4}\right),  
\qquad M=|A||B'|.  
]

因 substantial classes 至少有大小 (C_0L_X)，special terms 已被該引理吸收。

現在按大小 dyadic 分層。令

# [  
\mathcal C_u

{m:2^u\le |C_m|<2^{u+1}}.  
]

對一對 dyadic 層 (u,v)，令

[  
T_u=\sum_{m\in\mathcal C_u}|C_m|.  
]

若這兩層之間 cross mass 是

[  
\asymp T_uT_v,  
]

則對此 dyadic pair 的總 cross energy至少

[  
\gg  
T_uT_v  
\min\left(  
1,\frac{2^{2u}2^{2v}}{X^4L_X^4}  
\right),  
]

up to harmless dyadic logarithmic factors.

另一方面，描述這些 classes 的 entropy至多為

[  
O!\left(  
(T_u+T_v)(\log s+\log X)  
\right)  
]

加上選擇 class sizes 與 vertices 的 binomial entropy；後者也被同一量控制：

[  
\log\binom{N}{T_u+T_v}  
\ll (T_u+T_v)\log X.  
]

對 substantial classes 而言，若 (T_uT_v) 是正比例級 cross mass，則能量下界至少為每個 involved vertex 提供

[  
\gg \frac{N}{(\log X)^C}  
]

級別的成本，或更大。因

# [  
\frac{N}{(\log X)^C}

\frac{X}{(\log X)^{C+1}}  
\gg  
\log X+\log s  
]

對 (X) 足夠大成立，故這些 multi-label configurations 的 entropy 被 cross energy 支付。

形式上，對每個 dyadic layer 可寫成：

[  
#{\text{assignments in this layer with energy }S}  
\le  
\exp(\varepsilon S),  
]

再對 (O((\log N)^2)) 個 dyadic pairs 求和，吸收進 (e^{2\varepsilon S})。

所以 substantial multi-label case contributes only an energy-paid factor。

---

## 3. Tiny part dominates the non-majority mass

剩下的情形是：除了某個最大 class (G=C_{m_*}) 外，其餘 substantial classes 總量不大；非 majority 質量主要由 tiny classes 承載。

令

[  
D=P\setminus G  
]

為 exception set。若

[  
|G|\ge(1-\rho)N,  
]

則這已是 dominant case。

若 (|G|) 只是正比例大小但不是 dominant，則 (D) 的 vertices 與 (G) 之間產生 energy。對每個

[  
q\in D  
]

若

[  
a_q\not\equiv m_*\pmod q,  
]

由 Irving-good majority correction：

[  
\sum_{p\in G}  
|\phi_{pq}(a)|^2  
+  
\sum_{p\in G}  
\left|\frac{m_*}{pq}\right|^2  
\gg |G|  
\gg N.  
]

若 (G) 不是 (1-o(1)) dense，而只是 (cN) 大小，則可先用 substantial cross-label energy處理；在真正需要 majority correction 的情形，我們已在 dominant附近，即 (G) 是 dense subset。這正是 single-block proof 中的分流。

因此，每個 exception vertex 的成本至少

[  
\gg N  
]

或至少

[  
\gg N/(\log X)^C,  
]

視使用 Irving correction 或 divisor-energy而定。兩者都遠大於單 vertex entropy：

[  
O(\log X+\log s).  
]

所以 exception set 的選擇與 residues 可由能量支付：

[  
\sum_{D}  
\exp\bigl(O(|D|(\log X+\log s))-c|D|N/(\log X)^C\bigr)  
\le 1+o(1).  
]

故 tiny-class case 也只給 (e^{\varepsilon S}) 因子。

---

## 4. 結論

若沒有 dominant label，則：

- either substantial labels coexist, and cross-label divisor-energy pays their entropy;
    
- or all non-majority vertices are exceptions, and majority correction pays them.
    

因此 multi-label configurations of energy (\le S) contribute at most

[  
e^{\varepsilon S}.  
]

這證明 Multi-label entropy lemma。

---

# 如何嵌入 single-block theorem

在 single-block theorem 中，base-list reduction 給

[  
\mathcal L,\qquad |\mathcal L|\ll 1+\log X\sqrt R.  
]

若有 dominant label (m)，則：

[  
|m|\le B=A X\log X\sqrt R  
]

所以 label choices 為

[  
\ll B+1  
\ll 1+\frac{\sqrt R}{\sigma_P}.  
]

exceptions 由 energy支付。

若無 dominant label，Multi-label entropy lemma 說整個配置族只給 (e^{\varepsilon R}) energy-paid factor，不產生新的主自由度。

故

[  
#{a:Q_P(a)\le R}  
\le  
e^{\varepsilon R}  
\left(1+\frac{\sqrt R}{\sigma_P}\right).  
]

這就是 single-block counting theorem。
