好。現在寫 **Multi-label entropy lemma 的 dyadic 詳證**。我會把估計寫得稍保守；不追求最佳常數，只要足以證明 energy dominates entropy。

# Dyadic multi-label estimate

## 0. 設定

令

[  
P\subset[X,2X],\qquad N=|P|\asymp \frac X{\log X}.  
]

經 base-list reduction 後，covered vertices 被分成 label classes：

[  
V=\bigsqcup_{m\in\mathcal L}C_m,  
]

其中

[  
|\mathcal L|=s\ll 1+\log X\sqrt R.  
]

在 single-block proof 中，我們只需處理

[  
s\le X^{o(1)}  
]

或至多 (s\le X^{1/10}) 之類的範圍；若 (R) 大到讓 (s) 成為 (X^c) 級，則能量層本身已足夠大，可用粗熵吸收。正式寫時可把 (R) 分成中低能與高能兩段。

令

[  
L_X=1+\frac{\log(B+X^2+2)}{\log X}.  
]

substantial class 定義為：

[  
|C_m|\ge C_0L_X.  
]

對兩個 substantial classes

[  
A=C_m,\qquad B=C_{m'},\quad m\ne m',  
]

cross-label divisor-energy lemma 給

[  
E(A,B)  
\gg  
M\min\left(1,\frac{M^2}{X^4L_X^4}\right),  
\qquad M=|A||B|.  
]

---

# 1. Dyadic 分層

對整數 (u)，定義

[  
\mathcal C_u={m\in\mathcal L:2^u\le |C_m|<2^{u+1}}.  
]

令

[  
n_u=|\mathcal C_u|,  
\qquad  
T_u=\sum_{m\in\mathcal C_u}|C_m|.  
]

則

[  
n_u2^u\le T_u< n_u2^{u+1}.  
]

只考慮 substantial 層：

[  
2^u\ge C_0L_X.  
]

若兩個 dyadic layers (u,v) 之間有 cross mass

# [  
\mathcal M_{u,v}

\sum_{\substack{m\in\mathcal C_u,\ m'\in\mathcal C_v\m\ne m'}}  
|C_m||C_{m'}|,  
]

那麼

[  
\mathcal M_{u,v}\asymp T_uT_v  
]

除非 (u=v) 且同一 label pairs 被排除；這只造成 harmless diagonal subtraction。對 (u=v)，若該層內 class 數至少 (2)，亦有

[  
\mathcal M_{u,u}  
\asymp T_u^2-\sum_{m\in\mathcal C_u}|C_m|^2.  
]

在 non-dominant substantial case，總 cross mass有正比例：

[  
\sum_{u,v}\mathcal M_{u,v}\gg_\rho N^2.  
]

dyadic 層數只有

[  
O(\log N).  
]

故存在一對 dyadic layers ((u,v)) 使

[  
\mathcal M_{u,v}  
\gg_\rho \frac{N^2}{(\log N)^2}.  
]

這一對稱為 active dyadic pair。

---

# 2. Active pair 的能量下界

對 active pair 中任意 class pair，有

[  
|C_m||C_{m'}|\asymp 2^{u+v}.  
]

因此每對 class 的 energy 下界為

[  
\gg  
2^{u+v}  
\min\left(  
1,  
\frac{2^{2u+2v}}{X^4L_X^4}  
\right).  
]

active pair 中 class-pair 數約為

[  
\frac{\mathcal M_{u,v}}{2^{u+v}}.  
]

所以 active pair 的總 cross energy

[  
E_{u,v}  
\gg  
\mathcal M_{u,v}  
\min\left(  
1,  
\frac{2^{2u+2v}}{X^4L_X^4}  
\right).  
]

代入 active lower bound：

[  
E_{u,v}  
\gg_\rho  
\frac{N^2}{(\log N)^2}  
\min\left(  
1,  
\frac{2^{2u+2v}}{X^4L_X^4}  
\right).  
]

這是第一種寫法。

但這個式子在 (2^{u+v}) 很小時可能弱。此時每個 class小，但 substantial lower bound仍是 (C_0L_X)，且 class 數多。更好的熵比較應按 vertex-per-energy 來做。

---

# 3. 能量對熵的比較：兩個區間

令

[  
U=2^u,\qquad V=2^v.  
]

每個 class pair 的 cross energy至少

[  
\gg  
UV\min\left(1,\frac{U^2V^2}{X^4L_X^4}\right).  
]

## 情形 I：(UV\ge X^2L_X^2)

則每對 class 的 energy

[  
\gg UV.  
]

active pair 中總 energy

[  
E_{u,v}\gg \mathcal M_{u,v}  
\gg \frac{N^2}{(\log N)^2}.  
]

而 active pair 涉及的 vertices 至多 (N)，其配置熵至多

[  
O(N(\log X+\log s)).  
]

因

[  
\frac{N^2}{(\log N)^2}  
\asymp  
\frac{X^2}{(\log X)^4}  
\gg  
X  
\gg  
N(\log X+\log s)  
]

在 (s\le X^{o(1)}) 下成立。  
所以 energy 壓過 entropy。

## 情形 II：(UV< X^2L_X^2)

此時 energy lower bound為

[  
E_{u,v}  
\gg  
\mathcal M_{u,v}  
\frac{U^2V^2}{X^4L_X^4}.  
]

這看似可能弱。但注意：

[  
U,V\ge C_0L_X.  
]

若 (U,V) 都非常小，則 classes 很多。這時 active pair 的 cross mass (\mathcal M_{u,v}\gg N^2/\log^2N) 迫使

[  
T_u,T_v  
]

都很大，而 class 數

[  
n_u\asymp T_u/U,\qquad n_v\asymp T_v/V  
]

很大。配置這麼多 small classes 的 label assignments 乍看熵大；但 label list size (s) 限制 class 數：

[  
n_u+n_v\le s.  
]

因此

[  
T_u\le sU,\qquad T_v\le sV.  
]

由 active mass

[  
T_uT_v\gg \frac{N^2}{(\log N)^2}  
]

得

[  
s^2UV\gg \frac{N^2}{(\log N)^2}.  
]

所以

[  
UV\gg \frac{N^2}{s^2(\log N)^2}.  
]

代入情形 II 的 energy 下界：

[  
E_{u,v}  
\gg  
\frac{N^2}{(\log N)^2}  
\cdot  
\frac{1}{X^4L_X^4}  
\left(  
\frac{N^2}{s^2(\log N)^2}  
\right)^2.  
]

即

[  
E_{u,v}  
\gg  
\frac{N^6}{X^4L_X^4s^4(\log N)^6}.  
]

用

[  
N\asymp X/\log X  
]

得

[  
E_{u,v}  
\gg  
\frac{X^2}{s^4(\log X)^{12}L_X^4}  
]

粗略地。

只要

[  
s\le X^{1/8}/(\log X)^C,  
]

這仍然

[  
\gg X^{3/2-o(1)}  
]

遠大於配置熵 (O(X))。

這覆蓋中低能範圍。

---

# 4. 高能範圍的處理

若

[  
s> X^{1/8}/(\log X)^C,  
]

因

[  
s\ll \log X\sqrt R,  
]

可推出

[  
R\gg X^{1/4}/(\log X)^C.  
]

這還未必大過 (X)，所以不能直接用 raw entropy (e^{O(X)}) 吸收。若要完全嚴格，需要稍微優化上面指數，或採用另一個策略：不用按 base-list size (s) 粗估，而是把 label range 本身的 energy納入。

這裏出現一個真實技術點：dyadic 粗估可能不夠強到覆蓋所有 (R)。

不過在這個高 (s) 範圍，label interval

[  
[-B,B]  
]

已大，而不同 labels (m,m') 的差值分布會使 diagonal energy本身變大；multi-label configurations 的 labels不是免費的。正式證明中應加入 label energy項：

[  
\sum_m |C_m|, m^2/X^4  
]

或 internal diagonal contribution。這會補足高 (R) 範圍。

另一個更簡單的處理是把 single-block counting 按 (R) 分兩段：

- (R\le X^{\alpha})：用上面 dyadic argument；
    
- (R>X^{\alpha})：用 crude bound，但選 (\alpha>1) 才能壓 raw entropy；目前 (s) 閾值只給 (\alpha=1/4)，不夠。
    

所以還需要改進。

---

# 5. 改進：用 majority-exception 而非 arbitrary multi-class entropy

其實不必讓 dyadic energy支付整個 arbitrary partition 熵。  
我們只需證：低能 configuration 必有 dominant label。若沒有 dominant label，cross mass為 (\gg N^2)。即使 class sizes小，cross-label energy總量應該更強，因為很多 class pairs總和時不能只用最壞的 (UV) lower bound；small classes多時，label differences眾多，divisor-energy bound累積更好。

前面的 dyadic lower bound太粗，丟掉了 label-pair 數與不同 labels 的 arithmetic separation。Cross-label divisor-energy lemma對每對 labels給能量；若 labels很多，總能量應該近似所有 cross pairs 的平均，應達到

[  
\gg N^2/\log^C X  
]

而不該退化到 (X^2/s^4)。

要證這一點，需要一個 stronger summed incidence lemma:

[  
\sum_{m\ne m'}E(C_m,C_{m'};m,m')  
\gg  
\frac{1}{\log^C X}  
\sum_{m\ne m'}|C_m||C_{m'}|.  
]

如果這成立，問題立即解決。

能否由我們的 divisor-incidence 推出？可能可以，因為 special small-representative counts summed over many label pairs still controlled by (n)-divisor counts plus label multiplicities. 但這是另一個更強的 incidence lemma。

---

# 6. 誠實結論

我們剛才試圖把 dyadic bookkeeping 完全補嚴，發現粗版本在高 (s)/中高能範圍不夠漂亮。這不是整體崩潰，但表示：

[  
\boxed{  
\text{Multi-label entropy lemma 還不能算完全證完。}  
}  
]

需要補一個 stronger summed cross-label energy lemma，避免能量下界在很多小 classes時退化。

這是目前真正剩下的技術核心。

---

# 7. 下一步應做什麼？

應該改攻：

**Summed cross-label energy lemma.**

命題形式：

給短 label list (\mathcal L)，classes (C_m)，則

[  
\sum_{m\ne m'}  
\sum_{p\in C_m,q\in C_{m'}}  
\left(\frac{H_{pq}^{m,m'}}{pq}\right)^2  
\gg  
\frac{1}{(\log X)^C}  
\sum_{m\ne m'}|C_m||C_{m'}|  
]

除 tiny/exception cases。

若這個成立，無 dominant label時 cross mass (\gg N^2)，故能量 (\gg N^2/\log^C X\gg X)，足以支付 entropy。

這比 dyadic pairwise pigeonhole更強，也更符合實際。

所以，接下來應改寫並證這個 summed incidence-energy lemma。
