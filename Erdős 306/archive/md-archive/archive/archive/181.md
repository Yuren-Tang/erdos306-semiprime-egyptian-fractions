對，技術細節常常如此：一開始看似只差「整理」，一寫到熵與能量逐層比較，就會暴露哪個估計其實不夠強。這不壞，因為它幫我們把真正剩餘難點定位得更準。

現在要改攻這個更合適的命題：

[  
\boxed{  
\text{Summed cross-label energy lemma}  
}  
]

它比「逐一 class-pair 下界再 pigeonhole」更自然。因為 multi-label 情形中真正需要控制的是所有不同 labels 之間的總 cross energy，而不是某一對 dyadic layers。

---

# 1. 命題形式

設

[  
P\subset[X,2X],\qquad |P|=N.  
]

有一個 label list

[  
\mathcal L\subset[-B,B],  
]

以及 disjoint classes

[  
P_0=\bigsqcup_{m\in\mathcal L} C_m  
\subset P.  
]

對 (m\ne m')，(p\in C_m), (q\in C_{m'})，令

[  
H_{pq}^{m,m'}  
]

為滿足

[  
H_{pq}^{m,m'}\equiv m\pmod p,\qquad  
H_{pq}^{m,m'}\equiv m'\pmod q  
]

的對稱代表。

定義總 cross energy：

# [  
E_{\mathrm{cross}}

\sum_{m\ne m'}  
\sum_{p\in C_m,\ q\in C_{m'}}  
\left(\frac{H_{pq}^{m,m'}}{pq}\right)^2.  
]

希望證：

[  
\boxed{  
E_{\mathrm{cross}}  
\gg  
\frac{1}{(\log X)^C}  
\sum_{m\ne m'}|C_m||C_{m'}|  
}  
]

除去 tiny classes 或 divisor-special exceptions。

若此成立，則無 dominant label 時

[  
\sum_{m\ne m'}|C_m||C_{m'}|\gg N^2,  
]

故

[  
E_{\mathrm{cross}}\gg \frac{N^2}{(\log X)^C}\gg X  
]

足以支付整個 block 的配置熵

[  
O(N\log X)=O(X).  
]

這就乾淨地封住 multi-label entropy。

---

# 2. 為什麼逐 pair lemma 不夠？

逐 pair lemma 給：

[  
E(C_m,C_{m'})  
\gg  
M\min\left(1,\frac{M^2}{X^4L_X^4}\right),  
\qquad M=|C_m||C_{m'}|.  
]

若很多 classes 都很小，則每個 (M) 小，逐 pair 下界退化；加總後可能損失 (s^4)。但實際上，很多不同 labels 的存在應該讓小 CRT representative 的總數更難集中，不應該退化那麼多。

所以要直接估計所有不同 labels 的 small representative 總數。

---

# 3. Summed small-representative counting

對 (T\le X^2/2)，定義

# [  
N_{\mathrm{cross}}(T)

#{(m,m',p,q):  
m\ne m',\ p\in C_m,\ q\in C_{m'},\ |H_{pq}^{m,m'}|\le T}.  
]

若 (|H|\le T)，則存在 (|n|\le T) 使

[  
p\mid n-m,\qquad q\mid n-m'.  
]

因此

[  
N_{\mathrm{cross}}(T)  
\le  
\sum_{|n|\le T}  
\left(  
\sum_m \omega_{C_m}(n-m)  
\right)^2  
]

粗略地；更精確要去掉 (m=m') 與 (p=q)，但 upper bound 可先如此。

令

[  
F(n)=\sum_{m\in\mathcal L}\omega_{C_m}(n-m).  
]

則

[  
N_{\mathrm{cross}}(T)\le \sum_{|n|\le T}F(n)^2.  
]

現在我們需要 bound (F(n))。

對固定 (n)，每個 counted pair ((m,p)) 滿足

[  
p\mid n-m,\qquad p\in C_m.  
]

也就是

[  
m\equiv n\pmod p.  
]

但 (m\in[-B,B])，(p\sim X)。

對固定 prime (p)，有多少 (m\in\mathcal L) 滿足

[  
m\equiv n\pmod p?  
]

至多

[  
1+\frac{B}{X}.  
]

這可能是 (s) 級別，不能直接好。

另一方向：對固定 (m)，(\omega_{C_m}(n-m)\le L_X)。所以

[  
F(n)\le sL_X.  
]

太粗，會給 (N(T)\le Ts^2L_X^2)，仍有 (s^2) 損失。

需要利用 classes sizes：(\omega_{C_m}(n-m)) 不能同時對太多 (m) 大，除非 (n-m) 有很多 prime factors across many distinct (m)。但 (m) varies, so divisor bound alone does not prevent this.

這裏出現新問題：summed incidence lemma 不像逐 pair 那樣立即由單整數 divisor bound得到。

---

# 4. 可否利用 base-list structure？

重要：(\mathcal L) 不是任意短 list；它是 base vertex (p_0) 的 residue list：

[  
\mathcal L={m\in[-B,B]:m\equiv a_{p_0}\pmod{p_0}}.  
]

所以所有 labels in (\mathcal L) 同餘 modulo (p_0)。因此任意

[  
m-m'  
]

都是 (p_0) 的倍數。

這看似使 labels 更相關，但也給結構：labels 是一條 arithmetic progression

[  
m=m_0+t p_0,  
\qquad |t|\ll B/X.  
]

也就是 (s\asymp B/X)。

Can this help summed incidence?  
For fixed (n,p), condition (m\equiv n\pmod p) and (m\equiv m_0\pmod{p_0}) gives a CRT class modulo (pp_0). Within ([-B,B]), number of such (m) is about

[  
1+\frac{B}{X^2}.  
]

This is much smaller than (B/X).

This is crucial.

Because labels all lie in one residue class modulo (p_0), adding the base congruence improves list incidence by one more prime factor.

---

# 5. Improved bound for (F(n))

For fixed (n) and prime (p\sim X), labels (m\in\mathcal L) with

[  
m\equiv n\pmod p  
]

satisfy simultaneous congruences:

[  
m\equiv a_{p_0}\pmod{p_0},  
\qquad  
m\equiv n\pmod p.  
]

Since (p\ne p_0), modulo (pp_0\sim X^2) there is one class. Thus count in ([-B,B]) is

[  
\ll 1+\frac{B}{X^2}.  
]

In single-block application,

[  
B\asymp X\log X\sqrt R.  
]

For moderate (R), (B<X^2), this is (O(1)). For larger (R), it grows but slower.

Now

# [  
F(n)

# \sum_m\omega_{C_m}(n-m)

\sum_{p\in P}  
#{m\in\mathcal L:p\in C_m,\ p\mid n-m}.  
]

Ignoring (p\in C_m) only increases:

[  
F(n)  
\le  
\sum_{p\in P}  
#{m\in\mathcal L:m\equiv n\pmod p}.  
]

Hence

[  
F(n)  
\ll  
N\left(1+\frac{B}{X^2}\right).  
]

This is too large: (N\sim X/\log X). Squaring gives (T N^2), not useful.

But this bound ignores classes; for each (p), since (p) belongs to exactly one class (C_m), the condition (p\in C_m) determines (m=m(p)). Then

[  
F(n)=#{p\in P_0: m(p)\equiv n\pmod p}.  
]

Ah! Because classes are disjoint assignment: each vertex (p) has only one assigned label (m(p)). Then

# [  
F(n)

#{p\in P_0:p\mid n-m(p)}.  
]

This is at most (N), but need better average square over (n).

Summed cross small count becomes:

[  
N_{\mathrm{cross}}(T)  
\le  
\sum_{|n|\le T} F(n)^2,  
]

where each (p) contributes to an arithmetic progression

[  
n\equiv m(p)\pmod p.  
]

So this is an incidence problem between interval ([-T,T]) and many residue classes modulo primes.

We need bound

[  
\sum_{|n|\le T} F(n)^2.  
]

Expanding:

# [  
\sum_n F(n)^2

\sum_{p,q}  
#{n: |n|\le T,\ n\equiv m(p)\pmod p,\ n\equiv m(q)\pmod q}.  
]

But this is exactly counting small representatives again. This seems circular, but maybe average is easy:

For (p\ne q), simultaneous congruences modulo (pq\sim X^2) have

[  
\ll 1+\frac{T}{X^2}  
]

solutions (n).

Therefore

[  
\sum_{|n|\le T}F(n)^2  
\ll  
N T/X  
?  
]

Let's compute.

Diagonal (p=q): for each p,

[  
#{n:|n|\le T,n\equiv m(p)\mod p}\ll 1+T/X.  
]

Total diagonal:

[  
\ll N(1+T/X).  
]

Off-diagonal (p\ne q):

one CRT class modulo (pq), so

[  
\ll 1+T/X^2  
]

solutions per pair. Total:

[  
\ll N^2(1+T/X^2).  
]

Thus

[  
N_{\mathrm{cross}}(T)  
\ll  
N^2 + N^2T/X^2 + NT/X.  
]

This is not enough for small T because of (N^2) constant term. It says at threshold (T=0) there can be (N^2) solutions, false? For each pair p,q, there may be one representative in ([-T,T]) only if the CRT representative happens in that interval. The bound (1+T/X^2) is too crude; the "1" kills everything. Divisor bound removed this "1" by summing over n not pairs. Need combine.

For arbitrary residues, indeed each pair has one representative somewhere in ([-X^2/2,X^2/2]). For small T, probability should be T/X^2, but deterministic bound can't say better than 1 per pair. Need arithmetic structure or Kloosterman/large sieve.

This is essentially an incidence distribution problem of CRT representatives for arbitrary assigned labels (m(p)). Maybe Irving can help here too: if labels differ, phases distribute unless structured. But (m(p)) arbitrary from list.

# 6. Alternative: use energy minimisation argument

Maybe we don't need summed incidence. If many small classes, then for most vertices, they differ from any chosen majority label. Once choose maximum class (G), if (G) is not dense, Irving majority not directly works. But if (G) size is, say (N/s), not enough.

However, single-block internal complete graph itself might force some dense class via graph agreement theorem; maybe the appropriate tool is not incidence-energy but agreement theorem on dense graph with lists. We circled back.

But base-list (\mathcal L) is one-dimensional AP. Could use graph density: most original edges were good. Good edge between different assigned labels may have a third common label, meaning assignments arbitrary caused loss. We assigned each vertex arbitrarily; maybe choose assignment to maximize largest class. Then if cross edge between C_m,C_m' is good, the endpoint lists intersect at some label n. This could reassign one endpoint? A global agreement theorem may show a large common label. This may avoid energy counting.

This suggests the right formal statement is a list-agreement lemma for AP lists, not energy of arbitrary assigned labels.

# 7. Current honest status

The "summed cross-label energy" route is harder than expected because arbitrary assignment to labels destroys simple divisor summation.

But the original good graph density gives more information than the assigned labels. We should use list intersection graph directly:

- each vertex (p) has list (L_p);
    
- good edges are pairs with (L_p\cap L_q\ne\emptyset);
    
- good graph is dense;
    
- base list (L_{p_0}) covers most vertices;
    
- need prove one (m\in L_{p_0}) covers most vertices.
    

This is a dense graph/list agreement theorem. The assigned classes approach loses information because vertices can have multiple possible labels.

So next should switch to proving:

**AP-list agreement lemma.**  
Let (L_p={n\in[-B,B]:n\equiv a_p\pmod p}). If the intersection graph is (1-\eta) dense and all (L_p) meet base list (\mathcal L) for most p, then some (m\in\mathcal L) belongs to (L_p) for (1-O(\eta')) fraction of p, or bad edges have energy/entropy.

This might be true by dependent random choice / agreement theorem.
