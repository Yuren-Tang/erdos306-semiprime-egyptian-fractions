# 1. Defective triangle 的算術形式

固定 base prime

[  
p_0\sim X.  
]

對 (p,q\sim X)，若三條 pairwise intersections 都存在，則有

[  
u\in L_{p_0}\cap L_p,  
\qquad  
v\in L_{p_0}\cap L_q,  
\qquad  
w\in L_p\cap L_q,  
]

其中

[  
|u|,|v|,|w|\le B.  
]

若三者沒有共同交點，即不是同一個 (m)，則令

[  
v-u=\alpha p_0,  
]

[  
w-u=\beta p,  
]

[  
w-v=\gamma q.  
]

因

[  
(v-u)+(w-v)-(w-u)=0,  
]

得到

[  
\beta p-\gamma q=\alpha p_0.  
]

並且

[  
|\alpha|,|\beta|,|\gamma|  
\ll \frac{B}{X}.  
]

設

[  
K:=C\frac{B}{X}.  
]

那麼 defective pair ((p,q)) 對應某組非平凡整數

[  
|\alpha|,|\beta|,|\gamma|\le K  
]

滿足

[  
\beta p-\gamma q=\alpha p_0.  
]

# 2. 對固定 ((\alpha,\beta,\gamma)) 的解數

固定

[  
\alpha,\beta,\gamma.  
]

若

[  
\gamma\ne0,  
]

則

[  
q=\frac{\beta p-\alpha p_0}{\gamma}.  
]

所以對每個 (p)，至多一個 (q)。因此解數至多

[  
O(N).  
]

更強地，要 (q) 為 prime 且 (q\sim X)，還有同餘與大小限制；但粗略 (O(N)) 足夠作第一步。

係數組數是

[  
O(K^3).  
]

所以 defective pairs 數粗界：

[  
\ll K^3 N.  
]

這要小於總 pair 數

[  
N^2  
]

需

[  
K^3\ll N.  
]

也就是

[  
\left(\frac{B}{X}\right)^3  
\ll \frac X{\log X},  
]

即

[  
B\ll X^{4/3}/(\log X)^{1/3}.  
]

在

[  
B\asymp X\log X\sqrt R  
]

下，要求

[  
R\ll X^{2/3}/(\log X)^{8/3}.  
]

這覆蓋不小範圍，但仍不是全部。

# 3. 改進：固定 (\beta,\gamma)，(\alpha) 由 (p,q) 決定

式子

[  
\beta p-\gamma q=\alpha p_0  
]

等價於

[  
\beta p\equiv \gamma q\pmod {p_0}.  
]

對固定 (\beta,\gamma)，若 (\gamma\ne0)，則

[  
q\equiv \beta\gamma^{-1}p\pmod {p_0}.  
]

因 (p,q,p_0\sim X)，且 (q\in[X,2X])，對每個 (p) 只有 (O(1)) 個整數 (q) in the interval satisfying this congruence。  
所以固定 ((\beta,\gamma)) 解數仍 (O(N))，但不用乘 (\alpha)。

係數 ((\beta,\gamma)) 有

[  
O(K^2)  
]

種。故 defective pairs 數

[  
\ll K^2N.  
]

這要求

[  
K^2\ll N,  
]

即

[  
B\ll X^{3/2}/(\log X)^{1/2}.  
]

代入

[  
B=X\log X\sqrt R,  
]

得到

[  
R\ll X/(\log X)^3.  
]

這已經接近 raw entropy threshold (X)，非常好。  
在 (R\ll X/\log^C X) 的主要範圍，defective pairs 稀少。

# 4. 更精確的 defective pair bound

我們可表述：

**Lemma, triangle defect count.**  
若

[  
K=B/X,  
]

則對固定 regular base vertex (p_0)，base-defective pairs ((p,q)) 數至多

[  
O(K^2N+K N+N)  
]

up to logarithmic factors.

證明：

對 defective triangle，存在 ((\beta,\gamma)) with (|\beta|,|\gamma|\le K), not both zero, satisfying

[  
\beta p\equiv \gamma q\pmod {p_0}.  
]

對固定 ((\beta,\gamma))，此 congruence 對每個 (p) 給 (O(1)) 個 (q\sim X)，因 modulus (p_0\sim X) and interval length (X).  
故 (O(N)) pairs per ((\beta,\gamma)).  
總 (O(K^2N)).

特殊情況：

- (\gamma=0): then (\beta p=\alpha p_0). Since (p,p_0) primes and (p\ne p_0), impossible unless coefficients degenerate.
    
- (\beta=0): similarly.
    
- (\beta,\gamma\ne0) main case.
    

Thus bound holds.

# 5. 用 defect count 推 AP-list agreement

Recall:

- good graph density (1-\eta);
    
- base (p_0) regular, so most vertices adjacent to base;
    
- base neighbours are covered by base labels (m\in\mathcal L).
    

若沒有 dominant base label，則 base-neighbour set splits among labels with no large (C_m).  
Consider pairs (p,q) among base-neighbours with different base labels.

Such pairs are numerous:

[  
\gg_\rho N^2.  
]

For a pair (p,q) with different base labels, if (pq) is good, then triangle (p_0,p,q) is defective unless (L_{p_0}\cap L_p\cap L_q\ne\emptyset). But if (p,q) have different chosen base labels, they might still share another base label; to avoid assignment issue, choose labels by majority? Better define base cover sets non-disjoint.

We need a clean argument:

Let

[  
C_m={p:m\in L_p\cap L_{p_0}}.  
]

If no (m) has (|C_m|\ge(1-\rho)N), then pairs ((p,q)) not sharing any base label are numerous? Not necessarily: sets (C_m) may overlap. But how large can overlaps be?

For (p), (L_p\cap L_{p_0}) size at most

[  
1+\frac{B}{pp_0}  
\asymp 1+\frac{B}{X^2}.  
]

If (B<X^{3/2}) or (B<X^2), this is (O(1)). More generally small.

In the main range (B\ll X^{3/2}), each vertex has (O(1)) base labels. If no (C_m) dominant, then many pairs share no base label.

Indeed total number of pairs sharing a base label is

[  
\sum_m |C_m|^2.  
]

If no (|C_m|) dominant and each vertex belongs to at most (r) sets, then (\sum |C_m|\le rN), but (\sum |C_m|^2) could still be large if one (C_m) large. Without dominant, bounded overlap implies many pairs do not share base label.

So pairs with no common base label are (\gg N^2).

Since good graph is dense, many of these pairs (p,q) are good. For such a good pair:

- (L_p\cap L_q\ne\emptyset);
    
- both p,q meet base list;
    
- but (L_p\cap L_q\cap L_{p_0}=\emptyset) because they share no base label.
    

Therefore they are defective.

But defect count says defective pairs (\ll K^2N).

Thus if

[  
K^2N=o(N^2),  
\quad\text{i.e.}\quad K^2=o(N),  
]

contradiction. Hence a dominant (m) exists.

This gives AP-list agreement for

[  
B/X=K\ll \sqrt N\asymp \sqrt{X/\log X},  
]

i.e.

[  
B\ll X^{3/2}/\sqrt{\log X}.  
]

As above, this covers

[  
R\ll X/(\log X)^3.  
]

# 6. 高能範圍如何處理？

For single-block partition, raw block entropy is

[  
\log\prod_{p\in P}p\asymp X.  
]

If (R\gg X), trivial (e^{-cR}) kills entropy.  
Our AP-list agreement works up to

[  
R\ll X/(\log X)^3.  
]

There remains a logarithmic gap:

[  
X/(\log X)^3 \lesssim R \lesssim X.  
]

This is much smaller than previous power gaps. It may be closable by:

1. improving defect count by logarithms using primality/sieving;
    
2. choosing (B) cutoff with larger (A) or multi-base vertices;
    
3. using two base vertices to extend range.
    

Multi-base idea likely closes log gap: choose two base vertices (p_0,p_1). Then labels visible modulo (p_0p_1), reducing overlap and defect counts; threshold becomes (B\ll X^2/\mathrm{polylog}), enough up to (R\ll X^2/\ldots), certainly beyond raw entropy threshold.

# 7. Two-base amplification

Instead of one base (p_0), choose a small set of bases

[  
S={p_0,p_1}  
]

forming good edge and with many common good neighbours.

A vertex (p) is covered by a joint base label (m) if

[  
m\in L_p\cap L_{p_0}\cap L_{p_1}.  
]

Since (m) must satisfy congruences modulo (p_0p_1\sim X^2), the number of joint base labels per vertex in ([-B,B]) is

[  
\ll 1+\frac{B}{X^3}  
]

or perhaps (1+B/(p p_0 p_1)), very small for (B\ll X^3).

Defective configurations now lead to coefficient equations with more base moduli, and defect count improves to threshold roughly (B/X\ll N) or (B\ll X^2/\log X), enough.

This is promising and natural: use (r) base vertices to raise visibility. We only need (r=2) to pass (R\sim X), because (B\sim X\log X\sqrt X=X^{3/2}\log X), already below (X^2).

Actually one base threshold was (B\ll X^{3/2}/\sqrt{\log X}), missing by log^{3/2}. Two bases certainly closes.
