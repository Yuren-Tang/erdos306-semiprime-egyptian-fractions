# Two-base AP-list agreement lemma

## 1. 設定

令

[  
P\subset[X,2X],\qquad N=|P|\asymp X/\log X.  
]

每個 (p\in P) 有 list

[  
L_p={n\in[-B,B]\cap\mathbb Z:n\equiv a_p\pmod p}.  
]

假設

[  
B<X^2/100.  
]

定義 good edge：

[  
pq\ \text{good}  
\quad\Longleftrightarrow\quad  
L_p\cap L_q\ne\varnothing.  
]

假設 good graph 密度至少

[  
1-\eta.  
]

則存在

[  
m\in[-B,B]  
]

使

[  
m\in L_p  
]

對至少

[  
(1-O(\sqrt\eta))N  
]

個 (p\in P) 成立。

---

# 2. 選兩個 base vertices

因 good graph 密度 (1-\eta)，可選一條 good edge

[  
p_0p_1  
]

且 (p_0,p_1) 都是 regular vertices，其共同 good neighbourhood

[  
G={p\in P: p_0p\text{ good and }p_1p\text{ good}}  
]

滿足

[  
|G|\ge (1-O(\sqrt\eta))N.  
]

因 (p_0p_1) good，有唯一

[  
m_{01}\in L_{p_0}\cap L_{p_1}  
]

只要 (B<X^2/100)，因兩 lists 的交點至多一個。

---

# 3. 對共同鄰點的 two-base label

對

[  
p\in G,  
]

因 (p_0p) good，有唯一

[  
u_p\in L_{p_0}\cap L_p.  
]

因 (p_1p) good，有唯一

[  
v_p\in L_{p_1}\cap L_p.  
]

我們要證

[  
u_p=v_p=m_{01}  
]

對大多數 (p) 成立，甚至對所有 (p\in G) 成立，除非某些 triangle defect 出現。

若

[  
u_p=v_p,  
]

則此共同值同時屬於

[  
L_{p_0}\cap L_{p_1}\cap L_p.  
]

由 (L_{p_0}\cap L_{p_1}) 唯一性，它必為

[  
m_{01}.  
]

所以 (p) 服從 global base label (m_{01})。

因此唯一壞情形是

[  
u_p\ne v_p.  
]

---

# 4. 但 (u_p\ne v_p) 不可能

[

[

\qquad  
m_{01}\equiv a_{p_1}\pmod{p_1}.  
]

Now compare (u_p) and (v_p). We know:

[  
u_p-v_p\equiv0\pmod p.  
]

But do we know divisibility by (p_0p_1)? Not directly. We know:

[  
u_p-m_{01}\equiv0\pmod{p_0},  
]

[  
v_p-m_{01}\equiv0\pmod{p_1}.  
]

Thus

# [  
u_p-v_p

]

is not necessarily divisible by (p_0) or (p_1). So this direct claim is false.

]

Since (u_p,v_p\in L_p),

[
