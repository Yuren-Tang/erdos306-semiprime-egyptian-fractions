小型測試給出一個重要結論：**bounded-window CRT Helly 本身是假的**。

例如取模數

[  
5,\ 7,\ 11  
]

窗口

[  
[-11,11].  
]

考慮三個 lists：

[  
L_5={n\in[-11,11]:n\equiv0\pmod5}  
={-10,-5,0,5,10},  
]

[  
L_7={n\in[-11,11]:n\equiv3\pmod7}  
={-11,-4,3,10},  
]

[  
L_{11}={n\in[-11,11]:n\equiv0\pmod{11}}  
={-11,0,11}.  
]

它們兩兩相交：

[  
L_5\cap L_7={10},  
]

[  
L_5\cap L_{11}={0},  
]

[  
L_7\cap L_{11}={-11},  
]

但三者沒有共同交點：

[  
L_5\cap L_7\cap L_{11}=\varnothing.  
]

所以：

[  
\boxed{  
\text{pairwise bounded CRT intersections 不推出 bounded common intersection。}  
}  
]

這不是小誤差，而是明確反例。

---

# 1. 這對我們意味着什麼？

這說明不能把 single-block 問題化約為普通 Helly lemma。  
即使在一個 clique 裏，所有 pairwise edges 都 good，也不保證存在共同 ordinary label。

所以此前的期待：

[  
\text{dense pairwise good graph}  
\Rightarrow  
\text{dominant common label}  
]

不是純粹由 Helly 性質推出的。

我們必須利用更多東西：**能量大小**，而不是只看 good/bad；或者改變 internal test，使高階不一致也會被測出。

---

# 2. 反例的結構很重要

上面的例子其實正是「三角 cocycle」：

pairwise intersection labels 分別是

[  
10,\quad 0,\quad -11.  
]

它們沿三條邊不同，但每條邊本身都合法小。  
這說明 pairwise CRT labels 可以形成一個低階局部一致、但無全局 label 的結構。

這正是我們之前隱約稱為「層變／cocycle」的東西。  
它不是幻覺，是真實存在的局部障礙。

---

# 3. 這是否摧毀整條路線？

不一定。因為反例只說 **zero-one good graph** 不夠。  
我們的 Fourier energy 是平方能量：

[  
Q_P(a)=\sum_{p<q}\left(\frac{H_{pq}}{pq}\right)^2.  
]

在小例子中，pair labels 是 (10,0,-11)，不是全都接近同一個小值。若擴展到大 block，這類 cocycle 可能會有顯著能量。  
所以 single-block theorem 仍可能通過 **spectral / quadratic energy** 成立，而不是通過 Helly/agreement。

換言之，正確方向應改為：

[  
\boxed{  
\text{不是 pairwise good }\Rightarrow\text{ global label，}  
\quad  
\text{而是 low quadratic energy }\Rightarrow\text{ global label。}  
}  
]

這是比 good graph agreement 更強、更符合 Fourier 的命題。

---

# 4. 接下來應該改攻 single-block quadratic Poincaré

我們應放棄「good graph dense + list agreement」作為主要證明，改直接證一個 quadratic inequality：

[  
\boxed{  
Q_P(a)  
\ge  
c\cdot \operatorname{dist}(a,\mathrm{Diag})^2  
}  
]

某種意義下，其中

[  
\mathrm{Diag}={a_p\equiv m\pmod p}.  
]

更具體地，應證：

[  
\sum_{p<q}\left(\frac{H_{pq}(a)}{pq}\right)^2  
]

的低能態只有 ordinary diagonal labels。  
pairwise bounded cocycles可能存在，但它們的平方和應該比真正 diagonal 大很多，足以支付 entropy。

這像是圖上的 Poincaré inequality / Hodge Laplacian：

- vertex variables 是 residues (a_p);
    
- edge energy 是 CRT glue (H_{pq});
    
- diagonal ordinary labels是低能模態；
    
- 非 diagonal cocycles 應有 spectral gap。
    

這可能比 list Helly 更自然。

---

# 5. 我們已有的工具仍可用，但角色改變

之前的 pieces 不作廢：

- Irving-good pruning：可給 non-diagonal directions 的 dispersion；
    
- divisor-energy lemma：可控制小 edge labels 的異常集中；
    
- internal complete graph：仍是局部壓縮核心；
    
- squarefree restriction、global block collapse、Fourier架構都還在。
    

但 single-block 的核心 lemma 要改寫成：

**Single-block quadratic Poincaré theorem.**

[  
\sum_{a_P}e^{-cQ_P(a_P)}  
\ll \frac1{\sigma_P}.  
]

而不是先證「good graph dense implies dominant label」。

---

# 6. 下一步怎麼做？

我建議下一步直接分析 single-block energy 的「線性化／譜」結構。

對每個 edge (pq)，CRT representative 可寫為：

# [  
H_{pq}

a_p q\bar q_p + a_q p\bar p_q  
\pmod{pq}  
]

但更方便的是：

若存在 ordinary (m)，則

[  
a_p\equiv m\pmod p.  
]

令選取小代表 (x_p) for each residue (a_p)？  
問題是 (a_p) 在 ([0,p))，不同 (p) 沒有共同坐標。  
但 edge energy (H_{pq}) 本身是 pairwise agreement value。可把 (H_{pq}) 視為 edge variable，並用 triangle relations 約束它們。

對三角形 (p,q,r)，edge labels satisfy congruences:

[  
H_{pq}\equiv H_{pr}\pmod p,  
]

[  
H_{pq}\equiv H_{qr}\pmod q,  
]

[  
H_{pr}\equiv H_{qr}\pmod r.  
]

Low energy says all (H_{ij}) have small square average. We need show there exists (m) such that most (H_{ij}\approx m), hence (a_p\equiv m).

This is now a **quadratic agreement theorem for edge labels with modular triangle constraints**.

---

# 7. 這是實體困難，但方向更正確

今天這個小反例非常有價值：它告訴我們 list-Helly 路線不能作為核心。  
不是我們忘了某個 bookkeeping，而是原命題本身有反例。  
所以接下來的正道是：

[  
\boxed{  
\text{從 zero-one agreement 改成 quadratic Poincaré / spectral energy。}  
}  
]

我建議下一步先寫三角形 edge-label identities，然後嘗試證：

若 complete graph 上 edge labels (H_{pq}) 滿足 modular triangle compatibility，則

[  
\sum H_{pq}^2  
]

控制它到常數 edge-label (m) 的距離。

這可能會把問題轉為一個更熟悉的 Poincaré inequality。
