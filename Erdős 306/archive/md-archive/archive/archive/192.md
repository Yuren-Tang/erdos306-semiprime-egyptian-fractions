對，這個例子非常有用。它告訴我們：**good graph / Helly / list agreement** 不是正確核心；正確核心應是：

[  
\boxed{  
\text{pairwise bounded intersection 可以有 cocycle defect，}  
\quad  
\text{但 quadratic energy 應懲罰這些 defect。}  
}  
]

所以現在改道：不再試圖從

[  
L_p\cap L_q\ne\varnothing  
]

推出共同 (m)，而是直接研究

[  
Q_P(a)=\sum_{p<q}\left(\frac{H_{pq}}{pq}\right)^2.  
]

# 1. Edge-label formulation

對每對 (p,q\in P)，令

[  
H_{pq}\in(-pq/2,pq/2]  
]

滿足

[  
H_{pq}\equiv a_p\pmod p,\qquad  
H_{pq}\equiv a_q\pmod q.  
]

若 (a_p\equiv m\pmod p) 對所有 (p)，則

[  
H_{pq}=m  
]

只要 (|m|<pq/2)。  
所以 diagonal ordinary label 對應於 complete graph 上的**常數 edge label**：

[  
H_{pq}\equiv m.  
]

非 diagonal assignment 對應於一組 edge labels (H_{pq})，滿足 triangle congruence constraints。

---

# 2. 三角形約束

對三個 primes

[  
p,q,r  
]

有：

[  
H_{pq}\equiv H_{pr}\pmod p,  
]

因二者都 congruent to (a_p\pmod p)。

同理：

[  
H_{pq}\equiv H_{qr}\pmod q,  
]

[  
H_{pr}\equiv H_{qr}\pmod r.  
]

因此存在整數

[  
\alpha,\beta,\gamma  
]

使

[  
H_{pr}-H_{pq}=\alpha p,  
]

[  
H_{qr}-H_{pq}=\beta q,  
]

[  
H_{qr}-H_{pr}=\gamma r.  
]

相加得到

[  
\beta q-\alpha p=\gamma r.  
]

這就是剛才小反例背後的 cocycle equation。

---

# 3. Quadratic energy 對三角 defect 的懲罰

若三條 edge labels 都接近同一個 (m)，則三角形近似 diagonal。

若不是，則至少一個差值

[  
|H_{pr}-H_{pq}|,\quad |H_{qr}-H_{pq}|,\quad |H_{qr}-H_{pr}|  
]

較大。由於差值是 (p,q,r) 的倍數，只要不為零，就至少約 (X)。

所以對一個 nontrivial triangle defect，三條 edge labels 不可能都極小且彼此接近；它們至少要在尺度 (X) 上分開。

這意味著：cocycle defect 的最小平方能量不是 (O(1))，而是至少

[  
\gg X^2  
]

在未除以 (p^2q^2) 的 (H^2) 尺度上；換到 normalized energy，每條 edge 貢獻約

[  
\frac{X^2}{X^4}=\frac1{X^2}.  
]

單個三角形成本小，但若 defect 在 complete graph 中大面積存在，總成本會大。

---

# 4. 用 edge-label variance 取代 list agreement

定義 edge-label 平均或中位數：

[  
\mu=\operatorname{median}{H_{pq}}.  
]

若

[  
\sum_{p<q}H_{pq}^2  
]

很小，那所有 (H_{pq}) 多數靠近 (0)。但 diagonal label (m) 不一定是 0；在 counting 層 (R) 中，(m) 可有大小約

[  
X\log X\sqrt R.  
]

所以更合適是對每個候選 (m) 看 centered labels：

[  
H_{pq}-m.  
]

目標是證存在 (m) 使

[  
\sum_{p<q}(H_{pq}-m)^2  
]

小，並且 (m) 解釋大多數 vertices。

但 (H_{pq}-m) 若不是同時 divisible conditions，不能任意。  
A natural (m) 是 edge labels 的 majority/median。若 (H_{pq}) 在 complete graph 中低能且 triangle constraints 強，應該集中到一個 cluster。

# 5. 可能的 Poincaré inequality

我們想證一種形式：

[  
\inf_{m\in\mathbb Z}  
\sum_{p<q}\frac{(H_{pq}-m)^2}{X^4}  
\le  
C Q_P(a)  
]

或者更強：

[  
Q_P(a)  
\ge  
c\cdot Q_P^{\mathrm{diag}}(m)  
+  
c\cdot \mathrm{Defect}(a,m).  
]

但左邊本來就是

[  
\sum H_{pq}^2/X^4.  
]

若 (m\ne0)，(\sum(H_{pq}-m)^2) 未必由 (\sum H_{pq}^2) 控制。  
然而 partition counting 允許所有 (m)，所以我們可以 decompose by (m) chosen as mean of (H_{pq}).

Then the number of possible (m) is Gaussian scale, and deviations cost energy.

This suggests proving:

**Edge-label clustering lemma.**  
For any assignment (a), there exists an integer (m) such that either:

1. (a_p\equiv m\pmod p) for most (p), with exceptions energy-paid; or
    
2. the edge-label variance  
    [  
    \sum_{p<q}(H_{pq}-\bar H)^2  
    ]  
    is large enough to pay entropy.
    

This avoids false Helly.

# 6. How to prove clustering?

Use triangle constraints.

For each vertex (p), incident edge labels

[  
{H_{pq}:q\ne p}  
]

all lie in one residue class modulo (p).  
If two incident edge labels differ, their difference is a nonzero multiple of (p), hence at least (p\sim X).

Therefore, inside the interval where edge labels have size (\le B<X^2), each vertex’s incident labels form a separated set with spacing (X).

If global energy is low, most incident labels at a vertex cannot occupy many different separated values; otherwise vertex contributes large incident energy.

Thus for most (p), incident edge labels cluster around one value (m_p). Since edges are shared, (m_p) and (m_q) must agree for most adjacent pairs. Then ordinary graph agreement on scalar labels (m_p) gives a dominant (m).

This is promising.

---

# 7. Vertex-wise incident energy

Define for each vertex (p):

[  
E_p=\sum_{q\ne p}H_{pq}^2.  
]

Total unnormalised energy:

[  
\sum_p E_p=2\sum_{p<q}H_{pq}^2.  
]

Low normalized energy (Q_P(a)\le R) means

[  
\sum_{p<q}H_{pq}^2  
\ll R X^4.  
]

Average per vertex:

[  
E_p^{\mathrm{avg}}\ll \frac{R X^4}{N}.  
]

For diagonal label (m), each (H_{pq}=m), so

[  
E_p\sim N m^2.  
]

Thus (m^2\sim E_p/N). Fine.

Now if for a fixed (p), incident labels take two values differing by at least (X), this adds variance at least proportional to (X^2) times the smaller cluster size.

But energy around large mean m may hide this; use variance:

[  
\sum_q (H_{pq}-\mu_p)^2.  
]

Since values are separated by (X), if no dominant incident value, variance (\gg N X^2).

Summing over many vertices gives normalized cost:

# [  
\frac{N\cdot N X^2}{X^4}

\frac{N^2}{X^2}  
\asymp \frac1{\log^2 X}  
]

not huge. Wait per bad vertex cost normalized (N X^2/X^4=N/X^2=1/(X\log X)). Too small to pay vertex entropy. Need many defects maybe (N) -> total (1/\log^2X), not enough. But energy (H^2) includes large mean m; variance not in Q directly if centered.

So this route may not pay entropy.

# 8. But non-diagonal differences trigger Irving dispersion

If (m_p\ne m_q), then for a third vertex r, edge constraints produce phases involving ((m_p-m_q)\bar r), and Irving gives cost (\gg N). This was cross-block style. Need local version: vertex labels (m_p) as incident cluster centers; mismatches across many vertices cost Irving.

Could work but complex.
