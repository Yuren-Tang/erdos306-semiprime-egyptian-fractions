# Bounded cocycle rigidity lemma

## 1. 設定

令

[  
P\subset[X,2X]  
]

為 primes。對每對

[  
p,q\in P  
]

有整數 edge label

[  
H_{pq}=H_{qp}  
]

滿足 CRT compatibility：

[  
H_{pq}\equiv a_p\pmod p,  
\qquad  
H_{pq}\equiv a_q\pmod q.  
]

等價地，對任意三點

[  
p,q,r  
]

有 triangle congruences：

[  
H_{pq}\equiv H_{pr}\pmod p,  
]

[  
H_{pq}\equiv H_{qr}\pmod q,  
]

[  
H_{pr}\equiv H_{qr}\pmod r.  
]

假設在某個 vertex subset

[  
S\subset P  
]

上，所有 edge labels 都有一致小界：

[  
|H_{pq}|\le C X  
\qquad(p,q\in S).  
]

其中 (C) 固定，且 (X) 足夠大。

---

## 2. 結論

則存在一個整數

[  
m  
]

以及一個 exceptional subset

[  
S_{\mathrm{bad}}\subset S  
]

滿足

[  
|S_{\mathrm{bad}}|\le O_C(1),  
]

使得對所有

[  
p,q\in S\setminus S_{\mathrm{bad}},  
]

有

[  
H_{pq}=m.  
]

換言之，一個大 clique 上若所有 edge labels 都只有 (O(X)) 大小，那麼除有限多點外，它必為 ordinary diagonal label。

---

# Proof

取任意一條 base edge

[  
p_0q_0\subset S,  
]

令

[  
m=H_{p_0q_0}.  
]

對任意

[  
r\in S\setminus{p_0,q_0},  
]

由 triangle congruences：

[  
H_{p_0r}\equiv m\pmod {p_0},  
]

[  
H_{q_0r}\equiv m\pmod {q_0}.  
]

因此存在整數

[  
\alpha_r,\beta_r  
]

使

[  
H_{p_0r}=m+\alpha_r p_0,  
]

[  
H_{q_0r}=m+\beta_r q_0.  
]

因

[  
|H_{p_0r}|,\ |H_{q_0r}|,\ |m|\le CX  
]

且

[  
p_0,q_0\asymp X,  
]

故

[  
|\alpha_r|,\ |\beta_r|\le C'  
]

其中 (C') 只依賴於 (C)。

又 (H_{p_0r}) 與 (H_{q_0r}) 都 congruent to (a_r\pmod r)，所以

[  
m+\alpha_r p_0\equiv m+\beta_r q_0\pmod r.  
]

即

[  
r\mid \alpha_r p_0-\beta_r q_0.  
]

若

[  
\alpha_r=\beta_r=0,  
]

則

[  
H_{p_0r}=H_{q_0r}=m.  
]

若不然，則 ((\alpha_r,\beta_r)) 是一個非零 bounded integer pair。  
對固定非零 pair ((\alpha,\beta))，整數

[  
D_{\alpha,\beta}=\alpha p_0-\beta q_0  
]

大小為

[  
|D_{\alpha,\beta}|\ll_C X.  
]

一個 prime

[  
r\in[X,2X]  
]

若整除 (D_{\alpha,\beta})，則除非 (D_{\alpha,\beta}=0)，這樣的 (r) 至多 (O_C(1)) 個；實際上因 (|D_{\alpha,\beta}|\ll_C X)，最多一個或有限個。

若

[  
D_{\alpha,\beta}=0,  
]

則

[  
\alpha p_0=\beta q_0.  
]

因 (p_0,q_0) 是 distinct primes，且 (|\alpha|,|\beta|=O_C(1))，當 (X>C') 時不可能，除非

[  
\alpha=\beta=0.  
]

所以非零 pairs ((\alpha,\beta)) 只產生 (O_C(1)) 個 possible (r)。  
非零 bounded pairs總數也是 (O_C(1))。故所有 bad (r)，即

[  
(\alpha_r,\beta_r)\ne(0,0),  
]

總數為

[  
O_C(1).  
]

令這些 (r) 連同 (p_0,q_0) 中必要者之外為 exceptional set。對所有其他

[  
r  
]

有

[  
H_{p_0r}=H_{q_0r}=m.  
]

現在取兩個 good vertices

[  
r,s\in S\setminus S_{\mathrm{bad}}.  
]

由 congruence：

[  
H_{rs}\equiv H_{p_0r}=m\pmod r,  
]

以及

[  
H_{rs}\equiv H_{p_0s}=m\pmod s.  
]

所以

[  
H_{rs}\equiv m\pmod {rs}.  
]

但

[  
|H_{rs}|,\ |m|\le CX,  
]

而

[  
rs\asymp X^2.  
]

對 (X) 足夠大，

[  
|H_{rs}-m|<rs.  
]

故

[  
H_{rs}=m.  
]

引理證畢。

# 4. 從 clique rigidity 到 dense graph

現在 single-block energy不會保證所有 edges 都有

[  
|H_{pq}|\le CX.  
]

但可以選 threshold

[  
T=CX  
]

定義 small-edge graph：

[  
G_T={{p,q}: |H_{pq}|\le T}.  
]

若 (G_T) 很稠密，則存在一條 edge (p_0q_0) 其 common neighbourhood 在 (G_T) 中大小

[  
(1-O(\eta))N.  
]

在這個 common neighbourhood 加上 (p_0,q_0) 上，所有與 bases 的 edges small，但 common neighbourhood內部 edges未必全 small。  
要使用 clique rigidity，需要 large clique，而 dense graph 只保證固定大小 clique，不保證線性 clique。

所以要稍作修改：我們不需要所有內部 edges small，只需對 common neighbourhood中多數 (r,s)，edge (rs) small。Dense graph可給：選 regular edge，其 common neighbourhood induced subgraph仍 dense。

Then apply rigidity on many small triangles? The lemma requires clique, but perhaps we can use it on many fixed-size cliques to show most vertices align.

Simpler route:

Choose base edge (p_0q_0) with many common neighbours (R) such that:

[  
|H_{p_0r}|,|H_{q_0r}|\le CX.  
]

For each (r\in R), the previous proof already shows either (r) is one of (O_C(1)) exceptional possible residues or

[  
H_{p_0r}=H_{q_0r}=m.  
]

Indeed that part did not use edges among different (r,s)!  
Because for each (r), if (H_{p_0r},H_{q_0r}\le CX), then

[  
r\mid \alpha p_0-\beta q_0  
]

with bounded (\alpha,\beta); only (O_C(1)) r can be bad.

Excellent. So dense graph version is easier.

Thus:

If there is a base edge (p_0q_0) such that many (r) have both anchor edges small, then all but (O_C(1)) of those (r) satisfy

[  
a_r\equiv m\pmod r.  
]

Hence we get a large diagonal core.

# 5. Energy dichotomy

Let

[  
T=CX.  
]

If the graph of (T)-small edges has many edges, choose a regular edge (p_0q_0) with large common (T)-small neighbourhood. Then bounded rigidity gives large diagonal core.

If the graph of (T)-small edges does not have many edges, then many edges have

[  
|H_{pq}|>CX.  
]

Each such edge contributes at least

# [  
\frac{C^2X^2}{X^4}

]

to (Q_P).  
If there are (\gg N^2) such edges, then

[  
Q_P\gg \frac{N^2}{X^2}  
\asymp \frac1{\log^2 X}.  
]

This is not huge, but for partition maybe not enough. Need choose threshold bigger?

If choose

]

then low energy ensures most edges T-small. But bounded rigidity requires (T=O(X)).
