看 small-edge graph，則可從一條 regular small base edge 得到一個 core，其外例外數約

[

]

若

[  
K^2\ll N,  
]

core 是幾乎全體，直接完成。  
log gap 發生在

[  
K^2

可能不是 (o(N))，但只要它仍小於某個固定比例，例如

[  
K^2\le c_0N,

我們仍得到一個 **positive-density core**：

[  
|G|\ge cN.

一旦有 positive-density core (G)，對任何 wrong vertex (r)，Irving dispersion over (G) 應給

[

\left|  
\frac{(a_r-m)\bar p}{r}  
\right|^2  
\gg |G|  
\gg N.  
]

這不是再產生下一層 core，而是直接給每個例外 vertex 成本

[  
\gg N.

而一個 vertex 的 entropy 只有

[  
O(\log X).

所以例外立即被支付。這就是一次 bootstrap，不是塔。

# 2. 關鍵變成：能否得到 positive-density core？

我們之前要求

[  
K^2=o(N)  
]

是為了得到 almost-all core。其實只需

[  
K^2\le cN  
]

即可。

因

[  
K=\frac{T}{X}.  
]

若用 Markov cutoff

[  
T\sim X\log X\sqrt R,  
]

則

[  
K^2\sim R\log^2X.  
]

positive-density core 要求

[  
R\log^2X\le c\frac X{\log X},  
]

即

[  
R\le c\frac X{\log^3X}.  
]

這還是原門檻。  
所以如果只用 Markov cutoff，不能直接推到 (R\sim X)。

但我們不必取 (T) 等於 Markov cutoff。可以取較小 cutoff

[  
T=KX  
]

with

[  
K^2=cN.  
]

也就是

[  
K\asymp\sqrt N\asymp \sqrt{X/\log X}.  
]

若 small-edge graph at this cutoff 不夠稠密，則很多 edges 有

[  
|H_{pq}|>KX,  
]

每條貢獻

[  
\frac{K^2X^2}{X^4}=\frac{K^2}{X^2}  
\asymp  
\frac{1}{X\log X}.  
]

若正比例 (N^2) 條邊如此，總能量

# [  
\gg  
N^2\cdot \frac1{X\log X}  
\asymp  
\frac{X^2/\log^2X}{X\log X}

\frac X{\log^3X}.  
]

仍然只是低能門檻。  
所以到這裏確實還有 log gap。

# 3. Bootstrap 不能單獨閉合 log gap

這點要誠實：一次 bootstrap 只要有 positive-density core就很強；但 bounded rigidity 用 cutoff (KX) 得到 positive-density core的條件本身仍是

[  
R\lesssim X/\log^3X.  
]

所以它不能直接處理

[  
X/\log^3X\lesssim R\lesssim X.  
]

除非我們能強化 bounded rigidity 的壞點數：

[  
O(K^2)\quad\leadsto\quad O(K^2/\log^3X)  
]

或用多 base / sieve / average divisor saving 取得 log 因子。

\ll K^2.

至少節省幾個 (\log)。

這是一個普通的 divisor/sieve incidence 問題，不是層變塔。

# 5. 可以怎樣拿 log saving？

對固定 base primes (p_0,q_0)，考慮集合

[  
D_{\alpha,\beta}=\alpha p_0-\beta q_0,  
\qquad |\alpha|,|\beta|\le K.  
]

我們要數

[  
r\in[X,2X]  
]

使 (r\mid D_{\alpha,\beta}) for some pair.

等價於存在 ((\alpha,\beta)) satisfying

[  
\alpha p_0\equiv \beta q_0\pmod r.  
]

若 (r\nmid p_0q_0)，則

[  
\alpha\equiv \beta q_0\bar p_0\pmod r.  
]

因 (|\alpha|,|\beta|\le K).  
如果

[  
K<r  
]

，這是在一個 (K\times K) box 中數一條 modular line。對固定 (r)，解數約

[  
\frac{K^2}{r}+O(K)  
]

粗略。  
當

[  
K\le \sqrt r  
]

時，這約 (O(K))，不夠；但我們只要判斷是否至少一解。

對隨機 (r)，有解概率約

[  
\min(1,K^2/r).  
]

所以 expected bad r 數約

# [  
\frac{X}{\log X}\cdot \frac{K^2}{X}

\frac{K^2}{\log X}.  
]

這自然給一個 (\log) saving。  
要 (\log^3)，可能需要：

- 多個 independent base edges，給 probability ((K^2/X)^s);
    
- 或用 higher moment / pair correlations.
    

所以多 base edge不是塔，而是固定次數 amplification。用三個 independent base edges，bad probability roughly

[  
(K^2/X)^3  
]

，可能足夠。
