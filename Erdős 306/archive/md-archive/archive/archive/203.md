好。我們現在把 bad-neighbour incidence 再往前推一步。剛才的粗界

[  
|\mathcal B(p_0,q_0;K)|\ll K^2  
]

確實太粗；但它對**任意固定 base edge**是自然粗界。若允許在許多候選 base edges 中選一條「平均好的」，可以拿到至少一個 (\log) 節省。

這不是塔，而是一次 averaging。

---

# 1. Bad-neighbour set

固定 base edge

[  
p_0q_0,  
]

定義

# [  
\mathcal B(p_0,q_0;K)

{r\in P:\exists |\alpha|,|\beta|\le K,\ r\mid \alpha p_0-\beta q_0,\ (\alpha,\beta)\ne(0,0)}.  
]

粗界：

[  
|\mathcal B(p_0,q_0;K)|\ll K^2.  
]

因為每個 ((\alpha,\beta)) 給一個整數

[  
D_{\alpha,\beta}=\alpha p_0-\beta q_0,  
]

而 (r\sim X) 若整除它，可能性有限。

---

# 2. 對 base edge 平均

現在讓

[  
p_0,q_0  
]

在 block (P) 中變動。考慮三元組

[  
(p_0,q_0,r)  
]

以及係數 (\alpha,\beta) 滿足

[  
\alpha p_0\equiv \beta q_0\pmod r.  
]

對固定 (r) 和固定 ((\alpha,\beta))，若 (\alpha,\beta\ne0)，此條件等價於

[  
p_0\equiv \alpha^{-1}\beta q_0\pmod r.  
]

因

[  
p_0,q_0,r\sim X,  
]

模數 (r\sim X)，在長度 (\asymp X) 的 dyadic interval 中，對每個 (q_0) 只有 (O(1)) 個可能的整數 (p_0)，而 prime restriction 平均給約 (1/\log X) 的密度。更粗但足夠的平均啟發是：

對隨機 base pair ((p_0,q_0))，固定 (r) 被某個 ((\alpha,\beta)) 捕獲的概率約

[  
\min\left(1,\frac{K^2}{X}\right).  
]

當

[  
K^2\ll X  
]

時，平均 bad (r) 數約

# [  
N\frac{K^2}{X}  
\asymp  
\frac{X}{\log X}\frac{K^2}{X}

\frac{K^2}{\log X}.  
]

所以應有一條 base edge滿足

[  
|\mathcal B(p_0,q_0;K)|  
\ll  
\frac{K^2}{\log X}  
]

至少在 (K^2\ll X) 範圍內。

這把門檻從

[  
K^2\ll N\asymp X/\log X  
]

推到

[  
K^2/\log X\ll X/\log X,  
]

即

[  
K^2\ll X.  
]

換回

[  
K^2\sim R\log^2X,  
]

得到

[  
R\ll X/\log^2X.  
]

這比原來

[  
X/\log^3X  
]

前進了一個 (\log)。

---

# 3. 但一個 (\log) 還不夠

我們需要至少推到

[  
R\sim X  
]

附近，或者至少推到一個能被粗熵壓住的常數倍 (X)。  
單次 base-edge averaging只能自然給一個 (\log) 節省。

要再推，需要更強機制：

1. **多 base averaging**，但當 (K^2>X) 時概率飽和，不能直接給冪次提升；
    
2. **使用 energy distribution 而非單 cutoff**；
    
3. **高能層用粗 entropy + Fourier 常數**；
    
4. **證真正的 CRT Poincaré，而不是靠 bad-neighbour counting。**
    

---

# 4. 高能層能否用粗 entropy？

Raw number of assignments 是

[  
\prod_{p\in P}p  
]

其 log 約為

[  
\sum_{p\in P}\log p\asymp X.  
]

所以最粗的 partition contribution at energy (R) 是

[  
\exp(X-cR).  
]

若 (R\ge C X) 且 (C>1/c)，就能壓住。  
因此真正剩下的範圍是

[  
X/\log^2X\lesssim R\lesssim C X.  
]

仍然是一個 substantial logarithmic-to-constant gap。

---

# 5. 這說明什麼？

bad-neighbour incidence 能推低能 theorem，但不太可能單獨證完整 theta bound。  
它是 single-block CRT Poincaré 的一個低能工具，不是終局工具。

完整中能區需要更 global 的 spectral/theta argument。

這和我們目前的經驗一致：

- Helly 路線錯；
    
- bounded rigidity gives low-energy compression；
    
- bad-neighbour averaging gives one log improvement；
    
- remaining middle energy wants genuine spectral gap / code theta bound。
    

---

# 6. 現在最穩健的整理

我們可以把 local results 分成：

### Lemma A：bounded cocycle rigidity

若一條 base edge有很多 (KX)-small common neighbours，則除

[  
O(K^2)  
]

或平均

[  
O(K^2/\log X)  
]

外，它們都服從 same label。

### Lemma B：low-energy single-block compression

由 Lemma A 得：

[  
R\ll X/\log^2X  
]

範圍內，低能 assignments 有 dominant label，exceptions energy-paid。

### Lemma C：full single-block CRT Poincaré

仍未證。它需處理

[  
X/\log^2X\lesssim R\lesssim X.  
]

因此 conditional proof 的假設可弱化為：只需補 middle-energy CRT Poincaré。

---

# 7. 接下來應做什麼？

我建議現在不要再嘗試用同一個 incidence trick硬推。它已經給出自然極限。  
下一步應該研究 middle-energy 的 spectral/theta結構，尤其是：

[  
\sum_a e^{-cQ_P(a)}  
]

能否用某種逐點 counting 之外的方法直接估計，比如：

- Poisson summation on CRT lattice；
    
- Gaussian comparison for finite abelian groups；
    
- entropy tensorisation；
    
- log-Sobolev / Poincaré inequality for the CRT code;
    
- comparison with random residues.
    

這些才可能處理 (R\sim X/\log^2X) 到 (X) 的區間。

換言之，我們現在應把 single-block local theorem分成「低能已基本可控」與「中能 spectral theta bound」兩部分，然後攻後者。
