好。現在攻 **middle-energy theta bound**。我們先不要再用「找 core」的語言，而是直接看單塊 partition function：

[  
Z_P(c)=  
\sum_{a\in\prod_{p\in P}\mathbb F_p}  
e^{-cQ_P(a)}.  
]

目標是

[  
Z_P(c)\ll \frac1{\sigma_P}\asymp X\log X.  
]

這個目標非常強：原始狀態數是

[  
\prod_{p\in P}p\approx e^X,  
]

但 Gaussian weight 要把它壓到只有 (X\log X) 級。這說明中能區不能只靠粗能量層，必須利用乘積／局部測試結構。

# 1. 重新寫 partition function

對每條邊 (pq)，有

[  
H_{pq}(a)  
]

是 (a_p,a_q) 的 CRT glue。  
若把

# [  
K_{pq}(u,v)

\exp\left(  
-c\left(\frac{H_{pq}(u,v)}{pq}\right)^2  
\right),  
]

其中

[  
u\in\mathbb F_p,\quad v\in\mathbb F_q,  
]

則

# [  
Z_P(c)

\sum_{(a_p)}  
\prod_{p<q}K_{pq}(a_p,a_q).  
]

這是 complete graph 上的 pair-interaction partition function。  
Diagonal labels (a_p=m\bmod p) 是 ferromagnetic phases；非 diagonal states 應被 pair potentials 壓掉。

這提示可用統計力學／熵方法：不是逐一計數能量層，而是直接證 complete graph interaction 有強同步化。

# 2. Pair kernel 的基本性質

對固定 (p,q)，(K_{pq}(u,v)) 大，當且僅當 CRT representative (H_{pq}(u,v)) 小。

對固定 (u)，隨 (v) 變動，(H_{pq}(u,v)) 在 ((-pq/2,pq/2]) 中等距取值，基本均勻。  
因此

[  
\sum_{v\in\mathbb F_q}K_{pq}(u,v)  
\approx  
\sum_{h\equiv u\pmod p}  
e^{-c h^2/(p^2q^2)}.  
]

因 (h) 間距為 (p)，範圍長 (pq)，Gaussian 寬度約 (pq)，所以這個 row sum 大約是 (q) 的常數比例，不給強壓縮。

但多邊 complete graph 的交互會強烈同步。  
可以嘗試用 **transfer operator / second eigenvalue**。

# 3. Kernel spectral gap

把 (K_{pq}) 視作從 (\ell^2(\mathbb F_q)) 到 (\ell^2(\mathbb F_p)) 的矩陣。  
對角 ordinary modes 對應於 additive characters?

若對每個 pair kernel 有一個大 singular value 對應「constant/global label」，其餘 singular values 小，則 complete graph product partition 可控。

但 kernel 不是同一空間上的卷積，因模數不同。  
可用 CRT 將 (K_{pq}) 表示為：

[  
K_{pq}(u,v)=F_{pq}(h),  
]

其中 (h) 是 CRT combination of (u,v).  
其 Fourier transform in residues:

# [  
\widehat K_{pq}(\xi,\eta)

\sum_{u,v}K_{pq}(u,v)e_p(-\xi u)e_q(-\eta v).  
]

經 CRT 變換，((u,v)\leftrightarrow h\bmod pq)，所以

# [  
\widehat K_{pq}(\xi,\eta)

\sum_{h\bmod pq}  
F_{pq}(h)  
e_p(-\xi h)e_q(-\eta h).  
]

合併相位：

[  
e\left(  
-h\left(\frac{\xi}{p}+\frac{\eta}{q}\right)  
\right).  
]

所以 Fourier coefficient 是一維 Gaussian theta 的 Fourier coefficient at frequency

[  
\frac{\xi q+\eta p}{pq}.  
]

它大當且僅當

[  
\xi q+\eta p\equiv 0 \pmod{pq}  
]

或頻率很小。  
因 (\xi\in\mathbb F_p,\eta\in\mathbb F_q)，唯一 exact zero 是

[  
\xi=0,\eta=0.  
]

Nonzero frequencies should be exponentially or at least strongly damped.

This is promising.

# 4. Fourier expansion of pair kernels

Let

[  
F_{pq}(h)=e^{-c|h/(pq)|^2}.  
]

Then

[  
K_{pq}(u,v)=F_{pq}(h(u,v)).  
]

On group

[  
G_{pq}=\mathbb Z/(pq)\mathbb Z\simeq \mathbb F_p\times\mathbb F_q.  
]

Fourier expansion:

# [  
F_{pq}(h)

\sum_{\chi\bmod pq}\widehat F_{pq}(\chi)\chi(h).  
]

Because (F) is a smooth bump on the cyclic group, its Fourier coefficients decay like

[  
\widehat F_{pq}(t)  
\approx  
\frac1{pq}  
\sum_h e^{-c|h/(pq)|^2}e(-th/(pq)).  
]

For (t=0), coefficient (\asymp1).  
For (t\ne0), coefficient decays roughly

[  
e^{-c't^2}  
]

after Poisson, because the Gaussian has width comparable to the full circle. Actually (F(h)=e^{-c|h/M|^2}) on (h/M\in[-1/2,1/2]) is not narrow; Fourier coefficients decay polynomial/exponential in t depending smoothness at boundary. Still (|\widehat F(t)|<\rho<1) for all (t\ne0).

Crucial: uniform spectral gap

[  
|\widehat F(t)|\le \rho\widehat F(0)  
\quad(t\ne0)  
]

with (\rho<1) depending only on c.

If true, each edge interaction suppresses nonzero pair-frequency by factor (\rho).

# 6. Compare with diagonal mixture

The desired bound

[  
Z_P\ll X\log X  
]

suggests Gibbs measure is close to mixture over diagonal m.  
Could prove by sequential summation:

Order primes (p_1,\dots,p_N). Sum over (a_{p_N}) conditional on previous residues. The conditional weight is

[  
\prod_{i<N} K_{p_i p_N}(a_{p_i},a_{p_N}).  
]

If previous residues are not close to a common (m), this product is very small for every (a_{p_N}). If they are close to common (m), then (a_{p_N}=m\bmod p_N) dominates.

This is again agreement but in partition form. Could be made rigorous with kernel spectral gap.

# 7. A possible operator inequality

For any nonnegative weights (w_p(u)), define pair interaction operator. Need inequality:

[  
\sum_{v\in\mathbb F_q}  
\prod_{p\in S} K_{pq}(a_p,v)  
\le  
C\left(  
\text{diagonal contribution}  
+  
e^{-c|S|}\prod?  
\right)  
]

If (S) is moderately large, the product over many p acts like a likelihood for v. It should be large only if many (a_p) are compatible with the same integer (m).

This is a multi-anchor decoding but in weighted form. It might avoid hard zero-one agreement.

For fixed (v), each (p) prefers (m) satisfying (m\equiv a_p\pmod p) and (m\equiv v\pmod q). Product is a Gaussian in those CRT reps. If (a_p)'s incoherent, product small.

This seems promising but still technical.

# 8. What can be proven now?

We can state a **single-site contraction lemma** as target:

Let (S\subset P), (|S|=M). For any residues (a_p), define

[  
W_q(v)=\prod_{p\in S}  
\exp\left(-c\left(\frac{H_{pq}(a_p,v)}{pq}\right)^2\right).  
]

Then

[  
\sum_{v\in\mathbb F_q} W_q(v)  
\le  
C\left(1+\frac{X}{\sqrt M}\right)  
\exp(-c'\operatorname{Defect}(a_S)).  
]

Or more simply, summing over a new vertex reduces entropy by a factor unless previous residues are diagonal.

If such contraction iterates, partition bound follows.

But this is essentially another form of CRT Poincaré.

# 9. Honest assessment

Middle-energy theta bound is now the true hard problem. We have:

- false Helly ruled out;
    
- low-energy rigidity partial theorem;
    
- conditional global theorem.
    

Further progress likely needs a new analytic inequality for the pair kernel. This may be nontrivial but plausible.

At this point, a sane research plan is:

1. Formalize single-block CRT Poincaré as the central conjecture/theorem.
    
2. Prove the low-energy version via bounded rigidity.
    
3. Develop or seek a kernel/operator proof for the full theta bound.
    
4. Meanwhile write conditional global result.
    

If we continue now, the most concrete thing is to derive the pair-kernel Fourier coefficients explicitly. That may reveal a usable spectral gap. I suggest doing that next.
