好。現在正式計算 pair-kernel Fourier transform。

---

# 1. Pair kernel

固定兩個 primes

[  
p,q\sim X,\qquad (p,q)=1.  
]

對

[  
u\in\mathbb F_p,\qquad v\in\mathbb F_q  
]

令

[  
H_{pq}(u,v)\in(-pq/2,pq/2]  
]

為 CRT representative：

[  
H_{pq}\equiv u\pmod p,\qquad H_{pq}\equiv v\pmod q.  
]

定義 pair kernel：

# [  
K_{pq}(u,v)

\exp\left(  
-c\left(\frac{H_{pq}(u,v)}{pq}\right)^2  
\right).  
]

令

[  
M=pq.  
]

CRT 給出雙射：

[  
\mathbb F_p\times\mathbb F_q  
\simeq  
\mathbb Z/M\mathbb Z.  
]

在這個同一化下，

[  
K_{pq}(u,v)=F_M(h),  
]

其中

[  
F_M(h)=\exp\left(-c\left(\frac{\tilde h}{M}\right)^2\right),  
]

(\tilde h\in(-M/2,M/2]) 是 (h\bmod M) 的對稱代表。

---

# 2. Fourier coefficient

對

[  
\xi\in\mathbb F_p,\qquad \eta\in\mathbb F_q,  
]

定義

# [  
\widehat K_{pq}(\xi,\eta)

\sum_{u\bmod p}\sum_{v\bmod q}  
K_{pq}(u,v)  
e_p(-\xi u)e_q(-\eta v).  
]

由 CRT，(u\equiv h\pmod p), (v\equiv h\pmod q)，所以

# [  
\widehat K_{pq}(\xi,\eta)

\sum_{h\bmod M}  
F_M(h)  
e\left(  
-h\left(\frac{\xi}{p}+\frac{\eta}{q}\right)  
\right).  
]

合併頻率：

# [  
\frac{\xi}{p}+\frac{\eta}{q}

# \frac{\xi q+\eta p}{pq}

\frac{t}{M},  
]

其中

[  
t\equiv \xi q+\eta p\pmod M.  
]

因此

# [  
\widehat K_{pq}(\xi,\eta)

\widehat F_M(t),  
]

其中

# [  
\widehat F_M(t)

\sum_{h\bmod M}  
F_M(h)e\left(-\frac{th}{M}\right).  
]

所以 pair kernel 的雙變量 Fourier transform 完全化成一維 cyclic Gaussian transform。

---

# 3. 頻率零點

若

[  
t\equiv0\pmod M,  
]

即

[  
\xi q+\eta p\equiv0\pmod{pq}.  
]

Modulo (p) 得

[  
\xi q\equiv0\pmod p.  
]

因 (q\not\equiv0\pmod p)，故

[  
\xi=0.  
]

Modulo (q) 同理得

[  
\eta=0.  
]

所以唯一零頻率是

[  
(\xi,\eta)=(0,0).  
]

這很好：每條 pair kernel 的唯一完全不衰減 mode 是常數 mode。

---

# 4. 非零頻率衰減

我們需要估計

[  
\frac{|\widehat F_M(t)|}{\widehat F_M(0)}  
\qquad(t\ne0).  
]

注意

[  
F_M(h)=f(h/M),  
\qquad  
f(x)=e^{-cx^2},\quad x\in[-1/2,1/2].  
]

所以當 (M\to\infty) 時，

[  
\frac1M\widehat F_M(t)  
\to  
\int_{-1/2}^{1/2}  
e^{-cx^2}e(-tx),dx.  
]

而

[  
\frac1M\widehat F_M(0)  
\to  
\int_{-1/2}^{1/2}e^{-cx^2},dx.  
]

因此對固定非零 (t)，

[  
\frac{|\widehat F_M(t)|}{\widehat F_M(0)}  
\to  
\rho_t  
:=  
\frac{  
\left|\int_{-1/2}^{1/2}e^{-cx^2}e(-tx),dx\right|  
}{  
\int_{-1/2}^{1/2}e^{-cx^2},dx  
}.  
]

對每個 (t\ne0)，有

[  
\rho_t<1.  
]

而且因 (f) 平滑，(\rho_t\to0) as (|t|\to\infty)。  
所以存在

[  
\rho=\rho(c)<1  
]

使得對所有

[  
t\ne0  
]

和所有足夠大 (M)，

[  
|\widehat F_M(t)|  
\le  
\rho,\widehat F_M(0).  
]

這是單邊 kernel 的 uniform spectral gap。

---

# 5. 這是否足以證 single-block theta bound？

還不直接。原因是 (Z_P) 是 complete graph 上所有 pair kernels 的乘積：

[  
Z_P=  
\sum_{(a_p)}  
\prod_{p<q}K_{pq}(a_p,a_q).  
]

每條邊有 spectral gap，但它們作用在不同模數空間上。要將 pairwise gap tensorize 成 global partition bound，需要一個全圖 inequality。

但現在有一個非常有希望的表述：

將每個 (K_{pq}) 正規化：

# [  
\widetilde K_{pq}

\frac{K_{pq}}{\widehat F_{pq}(0)}.  
]

其 Fourier transform滿足：

[  
\widehat{\widetilde K}_{pq}(0,0)=1,  
]

[  
|\widehat{\widetilde K}_{pq}(\xi,\eta)|  
\le \rho  
\quad  
\text{for }(\xi,\eta)\ne(0,0).  
]

這表示每條 edge interaction 是一個「帶 uniform spectral gap 的同步 kernel」。

若能證 complete graph 上這類同步 kernels 的 partition function滿足

[  
Z_P  
\ll  
\text{diagonal contribution},  
]

則 single-block Poincaré完成。

---

# 6. 但有一個微妙點

pair kernel 的 zero Fourier mode 是常數 mode，而我們想保留的 diagonal modes 是

[  
a_p\equiv m\pmod p.  
]

這些不是 Fourier constant modes；它們是 primal-space 的 diagonal codewords。  
所以 pair-kernel Fourier spectral gap本身可能壓向「uniform randomness」，而不是直接解釋 diagonal synchronisation。

換言之，這個 kernel在 Fourier side 的唯一 zero mode是常數；但 partition function 的大貢獻在 primal side 是 diagonal strings。這是典型低溫 ferromagnetic模型：Fourier 高溫展開未必是最自然的。

這提醒我們：pair Fourier gap有用，但不能單獨說明 diagonal dominance。可能需要低溫 cluster expansion / Peierls，而不是純高溫 Fourier。

---

# 7. 新理解

pair kernel

[  
K_{pq}(u,v)  
]

不是「鼓勵 (u=v)」的普通 Potts kernel，而是鼓勵 ((u,v)) 來自某個小整數 (m) 的 CRT projection。  
也就是每條邊都有 (O(X^2)) possible pairs，但高權重集中在一條「small CRT diagonal band」：

[  
(u,v)=(m\bmod p,m\bmod q),  
\qquad |m|\ll pq.  
]

Full graph中，若所有邊都選的 (m) 近似一致，就得到 global diagonal label。  
如果 edge-wise (m)'s 不一致，就形成 cocycle defects，應由 bounded rigidity/Peierls壓住。

所以 Fourier kernel分析提供了單邊 smoothness，但主同步仍需低溫 geometric argument。

---

# 8. 有用的導出：pair kernel 的 row sums

雖然 Fourier gap不直接完成，但可用來估計 row sums / convolution。

對固定 (u),

# [  
\sum_{v\in\mathbb F_q}K_{pq}(u,v)

\sum_{\substack{h\bmod M\h\equiv u\pmod p}}  
F_M(h).  
]

這是一條 step (p) 的 Gaussian sum，約為

[  
\frac1p\sum_{h\bmod M}F_M(h)  
\asymp q.  
]

更精確地，均勻於 (u)：

# [  
\sum_vK_{pq}(u,v)

A_{pq}+O(\rho^q?)  
]

其實由 Poisson modulo p可得非常均勻。  
這說單條邊不偏好某個 residue；同步來自多邊共同約束。

---

# 9. 下一個合理工具：low-temperature cluster expansion

Given pair kernel high weight when (H_{pq}) small.  
Define edge label

[  
m_{pq}=H_{pq}.  
]

Diagonal phase: all (m_{pq}=m).  
Defect: for triangle, labels not equal.

Bounded cocycle rigidity says defects with small labels cannot percolate: any large small-edge cluster is almost constant.  
This is exactly Peierls contour logic. Middle energy may be handled by cluster expansion over contours of large edge labels and finite cocycle defects.

Thus perhaps the correct final proof is not Fourier spectral but **Peierls over edge-label clusters**, with bounded rigidity as contour lemma.  
The log gap came from single cutoff; cluster expansion over all contours/scales may close it.

This is again a kind of low-temperature expansion, but not tower if formulated as contours.

---

# 10. 結論

Pair-kernel Fourier transform is clean:

# [  
\widehat K_{pq}(\xi,\eta)

\sum_{h\bmod pq}  
e^{-c(h/pq)^2}  
e\left(  
-h\frac{\xi q+\eta p}{pq}  
\right),  
]

唯一 exact zero frequency is ((0,0)), and nonzero frequencies have uniform Fourier decay.

However, this does not directly prove diagonal dominance because diagonal labels are primal low-temperature phases, not Fourier zero modes.

So the best current synthesis is:

- **low-energy / local rigidity**: bounded cocycle rigidity;
    
- **middle energy**: likely needs Peierls/cluster expansion over edge-label defects, not naive layer counting;
    
- **Fourier kernel**: useful for operator estimates but not a complete shortcut.
    

Next I would try to formulate a Peierls contour model for edge-label deviations from a chosen phase (m).
