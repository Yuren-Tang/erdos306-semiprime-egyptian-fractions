好，現在我把「原命題」抽象成一個可操作模型，然後判斷它落在哪一級。

設 (\alpha=\pi)。我們關心的是有理數 (p/q) 對 (\pi) 的逼近誤差

[  
\left|\pi-\frac pq\right|.  
]

對任意無理數 (\alpha)，連分數收斂子 (p_n/q_n) 有兩個根本性質：

其一，收斂子是最佳逼近的一種。粗略說，若分母不超過 (q_n)，則沒有比 (p_n/q_n) 更好的逼近。更精確地說，收斂子控制了「最佳逼近」的骨架。

其二，誤差大小由下一個 partial quotient (a_{n+1}) 控制：

[  
\frac{1}{(a_{n+1}+2)q_n^2}  
<  
\left|\alpha-\frac{p_n}{q_n}\right|  
<  
\frac{1}{a_{n+1}q_n^2}.  
]

所以對 (\pi) 而言，問題真正變成：

[  
a_{n+1} \text{ 是否會相對於 } q_n \text{ 異常巨大？}  
]

若 (a_{n+1}) 普通大小，那誤差就是約 (1/q_n^2)。若 (a_{n+1}) 很大，例如 (a_{n+1}\approx q_n^\delta)，則誤差變成約

[  
\frac{1}{q_n^{2+\delta}}.  
]

這正是 irrationality exponent 中額外多出的那一截。

現在來看幾種可能命題。

若原命題只是說：

[  
\left|\pi-\frac pq\right|<\frac{1}{q^2}  
]

有無窮多解，這很容易，因爲連分數收斂子給出比 (1/q^2) 同階的逼近。這對任何無理數都成立，不是 π 的特殊性。

若原命題說：

[  
\left|\pi-\frac pq\right|<\frac{1}{q^{2+\varepsilon}}  
]

對某個固定 (\varepsilon>0) 有無窮多解，那這等價於說 (\mu(\pi)\geq 2+\varepsilon)。目前沒有理由相信它真；一般預期是假的，但不能對所有 (\varepsilon) 證明。

若原命題反過來說：

[  
\left|\pi-\frac pq\right|>\frac{C}{q^{2+\varepsilon}}  
]

對所有充分大的 (q) 成立，這相當於 (\mu(\pi)\leq 2+\varepsilon)。若要對任意 (\varepsilon>0) 都成立，就是 (\mu(\pi)=2)。這是自然猜想，但未證。

若原命題只要求存在某個很大的 (K)，使

[  
\left|\pi-\frac pq\right|>\frac{C}{q^K}  
]

對所有 (p/q) 成立，那這已知可由 π 的 irrationality measure 給出。也就是說，我們能證明 π 不可能被「過分離譜地」逼近，但不能證明它只具有「普通程度」的逼近。

這裏的分水嶺很清楚：

[  
K\gg 2  
]

屬於可證區域；

[  
K=2+\varepsilon  
]

屬於深水區；

[  
K=2  
]

本身不可能作爲統一排除界，因爲連分數收斂子會反覆達到 (1/q^2) 級別。

更準確地說，對任意無理數 (\alpha)，不能指望存在正的常數 (C)，使所有 (p/q) 都滿足

[  
\left|\alpha-\frac pq\right|>\frac{C}{q^2},  
]

因爲這只對 badly approximable numbers 成立，即連分數 partial quotients 有界的數。π 是否 badly approximable 也未知，而且一般不相信。若 π 的 partial quotients 無界，那麼對任意 (C>0)，都會有收斂子滿足

[  
\left|\pi-\frac{p_n}{q_n}\right|<\frac{C}{q_n^2}.  
]

所以我們還要區分兩件事：

第一，(\mu(\pi)=2)：表示 π 沒有 (q^{-2-\varepsilon}) 級別的無窮多異常逼近。

第二，π badly approximable：表示 π 連 (C/q^2) 中的常數 (C) 都能統一控制。

第二件事比第一件事強得多，而且更不可信。對「隨機實數」而言，(\mu=2) 幾乎處處成立，但 badly approximable 的數測度為零。換言之，通常預期 π 應該滿足 (\mu(\pi)=2)，但不應該 badly approximable。

這一點對原問題很要緊。因爲若原命題想要的是：

[  
\left|\pi-\frac pq\right|>\frac{C}{q^2}  
]

這不是 (\mu(\pi)=2)，而是更強的 bounded partial quotients 命題。這幾乎肯定不是合適目標。

比較合理的形式應該是：

[  
\forall \varepsilon>0,\quad  
\left|\pi-\frac pq\right|>\frac{1}{q^{2+\varepsilon}}  
]

對充分大的 (q) 成立。

這纔是「π 的有理逼近沒有超過正常極限」的正確表述。

現在我們可以把後續工作切成兩條。

一條是定理路線：利用已知 irrationality measure，得到某個具體但較弱的結論。例如若已知 (\mu(\pi)<M)，則對任意 (K>M)，不等式

[  
\left|\pi-\frac pq\right|<\frac{1}{q^K}  
]

只有有限多解。這條路能產生真正定理，但指數很差。

另一條是猜想路線：假設 (\mu(\pi)=2)，則得到最自然、最鋒利的結論。這條路不能稱爲證明，但可以把原問題歸入標準未解問題。

我現在傾向於選擇第一條作爲嚴格主線，第二條作爲解釋性旁線。也就是：

我們先證明一個弱但無條件的命題；然後說明若採用標準猜想 (\mu(\pi)=2)，則可升級到原本想要的鋒利版本。這樣不會把未證猜想偷換成結論，也不會因爲不能解決 (\mu(\pi)=2) 就停步。
