現在把「弱但無條件」的主線寫成正式命題。

設 (\mu_0) 是一個已知的 π 的 irrationality measure 上界，意思是存在常數 (C>0)，使對所有整數 (p,q)，(q\geq1)，都有

[  
\left|\pi-\frac pq\right|>\frac{C}{q^{\mu_0}}.  
]

這裏不用急着填入最佳數值；數值會隨文獻改進而變，結構上只要知道目前確有某個有限的 (\mu_0)。於是立刻得到：

**無條件弱結論。**  
對任意 (K>\mu_0)，不等式

[  
\left|\pi-\frac pq\right|<\frac{1}{q^K}  
]

只有有限多個有理數解 (p/q)。

證明很短。由 irrationality measure 上界，有

[  
\left|\pi-\frac pq\right|>\frac{C}{q^{\mu_0}}.  
]

若同時有

[  
\left|\pi-\frac pq\right|<\frac{1}{q^K},  
]

則

[  
\frac{C}{q^{\mu_0}}<\frac{1}{q^K},  
]

所以

[  
C<q^{\mu_0-K}=\frac{1}{q^{K-\mu_0}}.  
]

當 (q\to\infty) 時，右邊趨於 (0)，不可能永遠大於固定正數 (C)。故 (q) 有上界，解只有有限多個。

這雖然很弱，但它是貨真價實的定理。它說：π 不能被有理數「超多項式地好」或「任意高次地好」地逼近。π 不是 Liouville 數；它的有理逼近能力有有限上限。

可是這個命題距離我們真正想要的「正常逼近」還很遠。真正自然的猜想版本是：

[  
\forall \varepsilon>0,\quad  
\left|\pi-\frac pq\right|<\frac{1}{q^{2+\varepsilon}}  
]

只有有限多解。

這等價於

[  
\mu(\pi)=2.  
]

現在可以清楚看到：原問題若要求排除 (q^{-3})、(q^{-4})、(q^{-10}) 這類逼近，可能已有已知理論覆蓋一部分；若要求排除 (q^{-2.01})，那基本就撞上 (\mu(\pi)=2) 這個未解核心。

接着，我們需要把連分數殼層和這個命題完全對接。令

[  
\pi=[a_0;a_1,a_2,a_3,\dots],  
]

其收斂子為

[  
\frac{p_n}{q_n}.  
]

有基本公式：

[  
q_{n+1}=a_{n+1}q_n+q_{n-1}.  
]

因此 (a_{n+1}) 大，就意味著下一個分母 (q_{n+1}) 相對 (q_n) 有一次大跳躍。又有估計：

[  
\left|\pi-\frac{p_n}{q_n}\right|  
\asymp  
\frac{1}{q_nq_{n+1}}.  
]

由於

[  
q_{n+1}\asymp a_{n+1}q_n,  
]

所以

[  
\left|\pi-\frac{p_n}{q_n}\right|  
\asymp  
\frac{1}{a_{n+1}q_n^2}.  
]

現在假設某次 (a_{n+1}) 有大小

[  
a_{n+1}\approx q_n^\delta.  
]

則

[  
\left|\pi-\frac{p_n}{q_n}\right|  
\approx  
\frac{1}{q_n^{2+\delta}}.  
]

這說明 (\mu(\pi)>2) 的唯一來源，從連分數角度看，就是 partial quotients (a_{n+1}) 相對於既有分母 (q_n) 有冪次級增長。若 (a_{n+1}) 只是偶爾很大但不達到 (q_n^\delta) 這種級別，則仍不會破壞 (\mu(\pi)=2)。

所以我們得到一個非常乾淨的等價猜想：

[  
\mu(\pi)=2  
]

等價於

[  
\forall \varepsilon>0,\quad a_{n+1}<q_n^\varepsilon  
]

對充分大的 (n) 成立。

這句話比原來的有理逼近表述更有操作感。它不是說 (a_n) 必須有界，也不是說 (a_n) 不能很大；它只說 (a_{n+1}) 不能相對於 (q_n) 呈現固定冪次級的大。

例如，如果某個 partial quotient 是 (10^{20})，這在數值上很大；但若此時 (q_n) 已經是 (10^{10^{6}})，那麼它對 irrationality exponent 幾乎沒有影響。真正危險的是：

[  
a_{n+1}>q_n^{0.01}  
]

這樣的事件無窮多次出現。

這也是為甚麼「看 π 的連分數中有沒有巨大項」本身不夠。巨大與否不是絕對尺度，而是相對於 (q_n) 的尺度。若不把 (a_{n+1}) 與 (q_n) 比較，只看 (a_{n+1}) 的十進制大小，很容易誤判。

現在我們可以把路線再精煉成三個層級：

**一、可證層：有限 irrationality measure。**

存在某個 (\mu_0)，使

[  
\left|\pi-\frac pq\right|>\frac{C}{q^{\mu_0}}.  
]

這給出弱排除定理。

**二、自然猜想層：(\mu(\pi)=2)。**

等價於

[  
a_{n+1}<q_n^\varepsilon  
]

最終成立，對任意 (\varepsilon>0)。

**三、過強且不宜採用的層：π badly approximable。**

即存在 (C>0)，使

[  
\left|\pi-\frac pq\right|>\frac{C}{q^2}  
]

對所有 (p/q) 成立。這等價於 π 的 partial quotients 有界。這不只是未證，而且按一般隨機模型看很不可信。

所以，如果我們要把原問題歸納成一個合理的猜想，最好的形式不是：

[  
\left|\pi-\frac pq\right|>\frac{C}{q^2},  
]

而是：

[  
\forall \varepsilon>0,\quad  
\left|\pi-\frac pq\right|>\frac{1}{q^{2+\varepsilon}}  
]

最終成立。

前者要求 π 的連分數項有界；後者只要求它們不以 (q_n^\delta) 的級別爆發。後者才是「π 如一般數」的正確數學翻譯。

下一步若繼續推進，我會把這個連分數條件轉成一個概率模型：假設 π 的連分數 partial quotients 像 Gauss–Kuzmin 分布下的典型實數那樣行爲，則可以解釋為甚麼 (\mu(\pi)=2) 是正確預期，而 badly approximable 幾乎不應期待。這不會給出證明，但能清楚說明「我們到底在相信甚麼」。
