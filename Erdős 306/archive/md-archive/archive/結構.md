# **0. Introduction**

這一節要先說清楚限制：

若所有分母 n_i 都 squarefree，則

\operatorname{lcm}(n_i)

squarefree，因此有限和

\sum_i \frac1{n_i}

的既約分母必 squarefree。故 squarefree denominator 是必要條件，不是技術假設。

主定理：

\boxed{ b\ \text{squarefree} \quad\Longrightarrow\quad \forall a\ge1,\ \frac ab=\sum_i\frac1{p_iq_i}, }

其中 p_i,q_i 為不同 primes，p_iq_i 互異且 squarefree semiprime。

Introduction 還應簡述方法：

1. 用 Bernoulli 隨機子集；
2. Fourier inversion；
3. 主弧 ordinary labels m 給 Gaussian；
4. 小弧核心難點是 CRT residue entropy；
5. 新機制：同尺度 complete graph 先將每個 prime block 壓縮成 ordinary block label；
6. 跨尺度 skeleton 再將 block labels 合併成 single global label。

這裏可以明確說：同尺度 internal edges 是為 minor arc control 而加入的，不是為 mass tuning。

# **1. Preliminaries and notation**

這節放基本記號。

Prime blocks：

P_k\subset[2^k,2^{k+1}]

或一般

P\subset[X,2X].

CRT representative：

對 residues

a_p\in\mathbb F_p,\quad a_q\in\mathbb F_q,

令

H_{pq}(a)\in(-pq/2,pq/2]

滿足

H_{pq}(a)\equiv a_p\pmod p,\qquad H_{pq}(a)\equiv a_q\pmod q.

相位：

\phi_{pq}(a)=\frac{H_{pq}(a)}{pq}\pmod1.

能量：

Q(a)=\sum_{pq\in E}\|\phi_{pq}(a)\|^2.

Bernoulli Fourier bound：

|1-\theta+\theta e(x)| \le \exp(-c_\theta\|x\|^2),

其中 \theta\in[c,1-c]。

這給

|\widehat\mu(a)|\le e^{-cQ(a)}.

# **2. Kloosterman sums over primes and pruning**

這節引用 Irving 文稿。

定義

S_q(a;x)=\sum_{\substack{p\sim x\\(p,q)=1}}e_q(a\overline p).

引用 Irving Theorem 1：

\sum_{q\sim Q}\max_{(a,q)=1}|S_q(a;x)| \ll_\varepsilon \left( Q^{5/4}x^{5/8} +Qx^{9/10} +Q^{7/6}x^{13/18} \right)Q^\varepsilon.

然後推出 pruning lemma：

**Lemma 2.1, Irving-good pruning.**  
對 compatible blocks

Q^{2/3+\delta}\le x\le Q^{3/2},

刪去 (o(|Q|)) 個 q 後，對每個保留 q 與每個 d\ne0\pmod q，

\sum_{p\in P} \left\|\frac{d\overline p}{q}\right\|^2 \gg |P|.

證明：

- 用 Markov 從平均 Kloosterman bound 得到  
    \max_a |S_q(a;x)|\le \delta_0|P|  
    for most q;
- 用 Erdős–Turán / finite Fourier approximation of \|t\|^2 得到二次距離下界；
- bounded-degree skeleton 上 union bound。

這一節是外部輸入正式化的地方。

# **3. The cross-label divisor-energy lemma**

這節放我們已基本證完的自證算術引理。

**Lemma 3.1.**  
令

A,B\subset[X,2X]

為 prime subsets，m\ne m'，|m|,|m'|\le B_0。  
令 H_{pq}^{m,m'} 是

H\equiv m\pmod p,\qquad H\equiv m'\pmod q

的對稱代表。

對

N(T)=\#\{(p,q): |H_{pq}^{m,m'}|\le T\}

有

N(T)\ll T L_X^2+(|A|+|B|)L_X,

其中

L_X=1+\frac{\log(B_0+X^2+2)}{\log X}.

證明：

若 |H|\le T，存在 |n|\le T 使

p\mid n-m,\qquad q\mid n-m'.

用

\omega_{[X,2X]}(t)\le 1+\frac{\log |t|}{\log X}

處理；special cases n=m,m' 用 m-m' 的 prime divisors 控制。

**Corollary 3.2.**  
若

M=|A||B|

且 \min(|A|,|B|)\gg L_X，則

\sum_{p\in A,q\in B} \left(\frac{H_{pq}^{m,m'}}{pq}\right)^2 \gg M\min\left(1,\frac{M^2}{X^4L_X^4}\right).

證明用 quantile：

取

T_*=c\min(M/L_X^2,X^2).

使 N(T_*)\le M/2，故至少半數邊有 |H|>T_*。

# **4. Single-block Poincaré / counting theorem**

這是文章核心之一。

定義 internal block energy：

Q_P(a)=\sum_{p<q\in P} \left(\frac{H_{pq}(a)}{pq}\right)^2.

\sigma_P^2=\sum_{p<q\in P}\frac1{p^2q^2} \asymp \frac1{X^2(\log X)^2}.

**Theorem 4.1, Single-block counting.**

\#\{a_P:Q_P(a_P)\le R\} \le e^{\varepsilon R} \left(1+\frac{\sqrt R}{\sigma_P}\right).

Proof structure:

1. Set  
    B=A X\log X\sqrt R.
2. Low energy implies good graph density 1-\eta:  
    |H_{pq}|\le B  
    on most edges.
3. Pick regular base vertex p_0.
4. Define base list:  
    \mathcal L=\{n\in[-B,B]:n\equiv a_{p_0}\pmod{p_0}\}.  
    Then  
    |\mathcal L|\ll1+\log X\sqrt R.
5. Most vertices are covered by \mathcal L.
6. Partition covered vertices into label classes C_m.
7. If dominant label exists, exceptions are paid by Irving-good majority correction.
8. If no dominant label, use dyadic decomposition of class sizes plus cross-label divisor-energy lemma to show multi-label configurations are energy-paid.
9. Dominant label count:  
    |m|\le B\asymp \sqrt R/\sigma_P.

這一節仍是最需要正式細寫的部分，尤其第 8 步。可把 dyadic bookkeeping 做成 Lemma 4.2：

**Lemma 4.2, Energy pays multi-label entropy.**  
Given a short label list \mathcal L, if no label class has size (1-\rho)N, then either the configuration has internal energy \gg S, sufficient to pay its encoding entropy, or all non-majority vertices form an exception set paid by majority Irving.

這樣主 theorem 不會太臃腫。

# **5. Block labels and cross-block energy**

這節從 local labels 走向 global labels。

假設 blocks P_k,P_\ell 分別服從 ordinary labels m_k,m_\ell。

**Lemma 5.1, Cross-block label mismatch cost.**  
若

m_k\ne m_\ell,

則

Q_{k\ell}(m_k,m_\ell)+Q_{k\ell}^{\mathrm{diag}}(m_k) \gg |P_k||P_\ell|

除 m_\ell-m_k 被異常多 primes 整除的 negligible cases。

證明：

令

D=m_\ell-m_k.

對 q\in P_\ell 且 q\nmid D，Irving-good gives

\sum_{p\in P_k} \left\|\frac{D\bar p}{q}\right\|^2 \gg |P_k|.

整除例外由

\#\{q\in P_\ell:q\mid D\} \le \frac{\log|D|}{\log X_\ell}

控制。

# **6. Block-label Peierls collapse**

這節證全局 partition theorem 的 structural part。

Define effective block-label energy:

\mathcal Q(m_\bullet)= \sum_k Q_k^{\mathrm{diag}}(m_k) + \sum_{(k,\ell)\in E_{\mathrm{skel}}} Q_{k\ell}(m_k,m_\ell).

**Theorem 6.1, Block-label collapse.**

\sum_{(m_k)}e^{-c\mathcal Q(m_\bullet)} \ll \sum_m e^{-c'Q_{\mathrm{diag}}(m)}.

Proof:

- use skeleton graph connected and bounded degree;
- regions where m_k\ne m are droplets;
- each proper droplet has boundary edge with cost \gg |P_k||P_\ell|;
- boundary energy dominates:
    - location entropy;
    - label entropy;
    - nested droplets;
- Peierls expansion converges.

可先在 chain skeleton 上寫，最簡潔。Full graph不需要參與。

# **7. Ordinary diagonal counting**

這節處理 m\bmod L wrap-around。

**Lemma 7.1.**

\#\{m\bmod L:Q_{\mathrm{diag}}(m)\le R\} \le e^{\varepsilon R} \left(1+\frac{\sqrt R}{\sigma}\right).

Proof:

- apply single-block theorem to diagonal residues a_p=m\bmod p;
- obtain local representatives m_k;
- cross-block collapse forces m_k=m_0;
- product rigidity:  
    m\equiv m_0\pmod{\prod_{p\in S}p}  
    for most primes;
- exceptional primes are energy-paid;
- count m_0 by Gaussian scale.

This lemma also gives:

\sum_{m\bmod L}e^{-cQ_{\mathrm{diag}}(m)} \ll1/\sigma.

---

# **8. Global partition theorem**

Now prove:

\#\{a:Q_{\mathrm{ctrl}}(a)\le R\} \le e^{\varepsilon R} \left(1+\frac{\sqrt R}{\sigma_{\mathrm{ctrl}}}\right).

Proof:

1. Apply single-block theorem to each block to encode a|_{P_k} by m_k + exceptions.
2. Sum exception entropy into e^{\varepsilon R}.
3. Apply block-label collapse to (m_k).
4. Apply ordinary diagonal counting.

Then:

\sum_a e^{-cQ_{\mathrm{ctrl}}(a)} \ll1/\sigma_{\mathrm{ctrl}}.

And outside main arc:

\sum_{a\notin\mathfrak M}e^{-cQ_{\mathrm{ctrl}}(a)} =o(1/\sigma_{\mathrm{ctrl}}).

# **9. Edge construction, mass, variance, lattice**

This section constructs E.

Include:

1. internal edges for all used blocks;
2. bounded-degree skeleton;
3. high-scale mass edges;
4. gadget edges rs_r for r\mid b.

Prove:

\sum_{e\in E}\theta_e/e=1/b.

\sigma_E\asymp\sigma_{\mathrm{ctrl}}.

\gcd\{L/e:e\in E\}=1.

Also note:

b\text{ squarefree}

is necessary and sufficient for b\mid L with squarefree denominators.
